# 学習指導要領との対応

本プロジェクトは、文部科学省「**小学校学習指導要領（平成29年告示）**」および
同解説に準拠して教材を編集します。

> **注意**: 学習指導要領本文は文部科学省告示の著作物です。本プロジェクトでは
> 領域名・項目名・指導事項のコードや要旨の言い換え（パラフレーズ）にとどめ、
> 本文の全文転載は行いません。本文を参照したい場合は、文部科学省の公式ページ
> （ https://www.mext.go.jp/a_menu/shotou/new-cs/ ）をご覧ください。

## 国語科の構造

平成29年告示・小学校学習指導要領「国語」は、各学年で次の2領域・5区分に
指導事項を配置しています。

### 〔知識及び技能〕

- (1) 言葉の特徴や使い方に関する事項
- (2) 情報の扱い方に関する事項
- (3) 我が国の言語文化に関する事項

### 〔思考力, 判断力, 表現力等〕

- A 話すこと・聞くこと
- B 書くこと
- C 読むこと

教材の frontmatter `curriculum_items` には、各教材で意識した指導事項を
次のフォーマットで記載します：

```yaml
curriculum_items:
  - code: "C(1)エ"
    label: "登場人物の気持ちの変化や性格、情景について、場面の移り変わりと結び付けて具体的に想像する"
```

## コードの読み方

`C(1)エ` の意味：

- `C` = 領域記号（A 話す・聞く / B 書く / C 読む）
- `(1)` = 指導事項の小項目番号
- `エ` = 五十音インデックス（ア・イ・ウ・エ・オ...）

学年は frontmatter `grade` で別途指定されるため、コード自体には学年情報を
含めません。

## 第3学年 国語 主要指導事項（要旨パラフレーズ）

> 以下は文部科学省告示の本文ではなく、編集者による要旨の言い換えです。
> 正確な指導事項は文部科学省の公式資料を必ず参照してください。

### C 読むこと（文学的な文章を中心に）

| コード | 要旨 |
|---|---|
| C(1)ア | 段落相互の関係に着目しながら、考えと事例の関係などを捉える |
| C(1)イ | 登場人物の行動や気持ちなどについて、叙述を基に捉える |
| C(1)ウ | 目的を意識して、中心となる語や文を見つけて要約する |
| C(1)エ | 登場人物の気持ちの変化や性格、情景について、場面の移り変わりと結び付けて具体的に想像する |
| C(1)オ | 文章を読んで感じたことや考えたことを共有し、一人一人の感じ方などに違いがあることに気付く |

### 〔知識及び技能〕第3学年 関連項目

| コード | 要旨 |
|---|---|
| (1)ク | 文章全体の構成や内容の大体を意識した音読 |
| (3)エ | 易しい文語調の短歌や俳句を音読・暗唱する（古典との出会い） |

## マッピングのガイドライン

- 1つの教材に複数の指導事項を紐付けてよい
- 「中心となる指導事項（メイン）」と「補助的に触れる指導事項（サブ）」を区別したい場合は、
  frontmatter の `curriculum_items` の **先頭** をメインとする
- 教材ごとに、想定する **発問（questions）** が指導事項のどれを意識しているかを
  内部的に確認する（コメントやレビュー時の議論に役立つ）
