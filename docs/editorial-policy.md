# 編集方針

このドキュメントは、project_procopios における教材編集の原則と手順を定めます。
新しい教材を追加する人、既存教材を改訂する人は、編集前に必ず一読してください。

## 1. 基本理念

- **公益優先**: 利用者（子どもと教師）にとっての分かりやすさ・学びやすさを最優先とする
- **原典への敬意**: 原文の文学的価値を不必要に損なわない
- **編集の透明性**: 改変の方針を明示し、編集過程を可視化する
- **オープン保証**: 全ての成果物が自由に再利用・改変・再配布されうる状態を維持する

## 2. 原典素材の採用基準

採用できる素材は、以下を **すべて** 満たすものに限ります。

1. 著者の没年が、明らかに著作権満了している（日本では原則として没後70年が経過、
   ただし戦時加算や2018年改正前後の経過措置に注意）
2. 原文または底本のテキストが、信頼できる出典（青空文庫、国立国会図書館
   デジタルコレクション「インターネット公開」資料、確実にパブリックドメインで
   確認できる紙資料）から入手できる
3. 現代の人権感覚と相容れない差別的表現については、教材化の可否を編集会議で
   判断する（注釈を付して残す／改稿する／不採用とする、のいずれか）

### 採用してはいけないもの

- 「館内限定」「送信サービス限定」など、国立国会図書館でインターネット公開
  されていない資料
- 編纂者・挿絵作者の著作権が別途残っている資料（教科書には注意）
- 学習指導要領の本文転載（項目名・領域構造の引用にとどめる）
- 検定済教科書（現行・廃版を問わず）の本文

## 3. 編集の段階

### Step 1: 原文の整序

- 旧字体 → 新字体（例: 駈 → 駆、廻 → 回、舊 → 旧）
- 明らかな誤植・OCRエラーの修正（修正箇所は編集ログに記録）
- 句読点の補い（原文に句読点がない明治期テキストの場合のみ）

### Step 2: 仮名遣い

- 旧仮名遣い → 現代仮名遣い（例: ゐる → いる、けふ → きょう、思ひました → 思いました）
- 「ハ・ヲ・ヘ」助詞は現代表記の「は・を・へ」へ
- 旧字熟語の現代語化は最小限にとどめる（例: 仰有った → おっしゃった は許容、
  美しき → 美しい は **しない**：作品の文体を壊すため）

### Step 3: ルビ（フリガナ）の追加

対象は以下の漢字：

- 該当学年で未習の漢字
- 学年配当表にあっても、文脈で読み方が分かりにくい熟字訓・特殊な読み
- 原典で既にルビが振られている全ての箇所（残す）

参考: 学習指導要領「学年別漢字配当表」

### Step 4: 語注の作成

以下の語句について「語句の意味」セクションに簡潔な説明を添える：

- 現代の児童には馴染みのない物品名（白銅貨、シャッポ、風呂敷 など）
- 動植物の固有名（樅、家鴨 など）
- 古風な言い回し（仰有った、跳んで など）
- 比喩的表現（牡丹色、真綿のように など）

語注の長さは1〜2文を原則とし、それ以上の補足は別途「解説」セクションに分ける。

### Step 5: 学習指導要領との対応

frontmatter の `curriculum_items` に、教材が想定する指導事項のコードとラベルを
記載する。文部科学省告示の本文をそのまま転載してはならない（要約・パラフレーズに
とどめる）。

詳しくは [`curriculum-mapping.md`](./curriculum-mapping.md) を参照。

### Step 6: 発問（考えてみよう）の作成

指導要領の指導事項を意識しつつ、以下のバランスで3〜5問程度を作成する：

- 場面・登場人物・情景についての事実的読み取り
- 登場人物の気持ちや心情の変化についての解釈
- 自分自身の経験や考えに引きつけた応答

正解を一つに絞らない、開かれた発問を心がける。

## 4. 差別的表現への対応

戦前・戦中の教材には、現代の人権感覚と合わない表現が含まれる場合があります。
編集の選択肢は以下の3つで、教材ごとに明示します。

| 選択肢 | 適用条件 | 表示方法 |
|---|---|---|
| **そのまま掲載＋注釈** | 文学的・歴史的価値が高く、当時の社会背景を学ぶ機会となる場合 | 該当箇所に注釈を付け、編集方針セクションで理由を説明 |
| **改稿** | 教材本体の価値を損なわず、表現を置き換えられる場合 | 改稿前後を編集方針セクションで対比 |
| **不採用** | 上記2つで対応できない場合 | 教材自体を採用しない |

## 5. 編集に使うコメント記号

教材本文の HTML には、以下の方針で編集マーキングを残してもよい（最終公開時には
不要なものは除去すること）：

```html
<!-- 原文: 駈け廻った → 駆け回った (旧字体修正, 2026-05-24) -->
```

## 6. ライセンス表記

各教材ページ末尾には、以下の3点を必ず明記すること：

1. **原文の権利状態**: 例「パブリックドメイン」
2. **編集物のライセンス**: 「CC BY 4.0」
3. **出典**: 入力底本、デジタル化作業者へのクレジット

## 7. 改訂手順

既存教材の改訂は、以下の手順で行います：

1. プルリクエストを作成
2. PR説明欄に変更理由・該当方針を明記
3. レビュー（最低1名）の承認を得てマージ
4. 大きな改訂の場合は、ページ末尾に改訂履歴を残す
