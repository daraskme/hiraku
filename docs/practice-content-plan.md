# 基礎練習コンテンツ拡充計画

読み物教材を中心にした現在の「ひらく」に、反復練習できる基礎教材を追加するための方針です。

## 追加する三本柱

### 漢字ドリル

- 対象: 小1〜小6配当漢字、中学校で扱う常用漢字
- 単位: 学年・テーマごとのセット
- 内容: 漢字、読み、音訓、画数、意味、熟語、短文づくり
- 入口: `/practice/kanji`
- 印刷: `/practice/print/kanji/<slug>` で書き取り欄つきA4プリントを出す

今後の拡張候補:

- 読みだけを隠すフラッシュカード
- 学年別の未習漢字フィルタ

### 算数・数学問題

- 対象: 小学校算数、中学校数学
- 単位: 単元ごとの問題セット
- 内容: 問題、答え、解説、難易度、ヒント
- 入口: `/practice/math`
- 印刷: `/practice/print/math/<slug>` で解答欄つきA4プリントを出す

今後の拡張候補:

- 自動採点できる短答問題
- 途中式の段階ヒント
- 図形・数直線・面積図などの図解
- 間違えた問題だけを復習する localStorage 保存

### 英語の単語・熟語・イディオム

- 対象: 小学校英語、中学英語、高校基礎英語
- 単位: 用途・品詞・場面ごとのセット
- 内容: term、意味、品詞、発音目安、核イメージ、意味の枝、例文、小テスト
- 入口: `/practice/english`
- 印刷: `/practice/print/english/<slug>` で単語表・例文欄・概念挿絵つきA4プリントを出す

今後の拡張候補:

- 多義語を「訳語の暗記」ではなく「核イメージからの展開」として見せる
- `spring` のように、バネ・春・泉などを一枚の挿絵でつなぐ
- 英和・和英切り替え
- スペリング練習
- 例文穴埋め
- イディオムとコロケーションの分離

## データ配置

- 漢字: `src/content/kanjiPractice/*.md`
- 算数・数学: `src/content/mathPractice/*.md`
- 英語: `src/content/englishPractice/*.md`

印刷用プリントの一覧は `/practice/print`、個別プリントは `/practice/print/<kind>/<slug>` に置きます。
各プリントは画面上で「問題だけ」「解答つき」「省スペース」を切り替え、同じページから印刷またはPDF保存できます。

いずれも `src/content.config.ts` で schema を定義しています。読み物教材とは別 collection にしているため、既存の `/library`、RSS、教材検索、読了進捗に混ざりません。

## 挿絵・図解の拡充方針

読み物教材の `cover_image` は全件整備済みです。次は次の情報を増やします。

- `illustrations` に scene2 以降を追加し、本文中の場面と対応させる
- caption と alt を充実させ、教材としての読み取りにも使えるようにする
- 算数・数学では生成絵よりも、面積図、数直線、比例グラフ、展開図などの図解を優先する
- 画像生成プロンプトの要旨と時代考証メモを残す
- 英語では単語の中心感覚を表す「概念挿絵」を追加し、同じ絵から複数の意味へ分岐させる

## 最初の量産目標

1. 小1漢字 80字を 8〜10 セットに分割
2. 小4算数の「大きな数」「わり算」「角」「面積」を各20問
3. 中1英語の基本動詞・名詞・形容詞・前置詞を合計200語
4. 中1数学の正負の数・文字式・一次方程式を各30問
