# Claude Codeに「日本の教材サイトを作って」と頼み続けたら、212教材＋挿絵フルカバーのオープンソース学習プラットフォームができた

## 「あったらいいな」を、本当に作ってみた

ある日、ふと思った。

> 「日本中の **子どもも大人も**、無料で・登録なしで、小学1年生レベルから大学入試レベルまでの名作を、全部読めるサイトって、なんで無いんだろう？」

青空文庫はある。教科書もある。Wikipedia もある。でも、それを **「教材」 として、挿絵付きで・縦書きで・解説と問いつきで・スマホでもタブレットでも快適に読めるサイト** は、無かった。

子どもが「学校で習った話の続きを読みたい」と思ったとき。
大人が「久しぶりに古典を読み直したい」「教養を学び直したい」「子どもと一緒に読みたい」と思ったとき。

その全部の人が、何のハードルもなく学べる場所が欲しい。

なら、作ろう。

そう決めて、Claude Code (Anthropic の AI コーディングエージェント) に「教材サイト作って。子どもにも大人にも届くやつ」と頼み続けた結果がこちらです。

**🌸 ひらく — 子どもも大人も学べる、日本の教材オープンソース・プラットフォーム**
👉 https://hiraku-5cb.pages.dev/

## 何ができるか

ただ「読める」だけじゃない。

- **縦書きモード** ─ 古典は縦書きで読むのが一番美しい
- **リーダーモード** ─ 没入感のある読書体験
- **挿絵を背景に表示** ─ スクロール追従で本文の世界観に浸れる
- **クイズモード** ─ 各教材にインタラクティブな問題
- **フラッシュカード/SRS** ─ 単語暗記もできる
- **TTS 読み上げ** ─ 音声で耳から学ぶ
- **AI チューター** ─ Cloudflare Workers AI で質問できる
- **学習進捗・ブックマーク・お気に入り** ─ Google ログインで端末間同期
- **Pagefind 全文検索** ─ JLPT 別フィルタも
- **印刷用ワークシート PDF** ─ 教室でも使える
- **OG 画像自動生成** ─ SNS シェアでも映える
- **PWA** ─ オフラインでも読める
- **i18n 対応** ─ 英語版ランディング・中国語版サンプル
- **アクセシビリティ** ─ WCAG 2.2 AA 基本監査済み

## どうやって作ったのか

- **Claude Code** で全コーディング (Astro + TypeScript + Cloudflare Pages)
- **教材本文** は青空文庫等の PD 作品 → 編集部の試訳と注釈を付与
- **挿絵** は ChatGPT の DALL-E / GPT Image-1 で生成し、`cwebp` で WebP 化
- **ホスティング** は Cloudflare Pages (無料枠で世界配信)
- **CI/CD** は GitHub Actions
- **検索** は Pagefind (静的サイトで動く全文検索エンジン)
- **同期** は Cloudflare KV + Google OAuth
- **AI チューター** は Cloudflare Workers AI

ドメインを取って、Cloudflare アカウントを設定して、Google OAuth クライアントを作って ─ そういう細かい設定全部含めても、AI と二人三脚なら数日で立ち上がる時代になりました。

**個人開発のハードルが、ここまで下がっていることを、ぜひ体感してほしい。**
