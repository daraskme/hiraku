# 行動規範（Code of Conduct）

「ひらく」プロジェクトは、すべての貢献者にとって、
ハラスメントのない、開かれた、敬意のあるコミュニティを目指します。

## 私たちの誓い

私たちは、参加者・貢献者・メンテナとして、以下を誓います：

- 年齢・体格・障害の有無・人種・性別・性自認・性的指向・国籍・宗教・
  経験レベル・教育・経済状況にかかわらず、すべての人にとって
  ハラスメントのない参加体験を提供すること
- 多様な視点と経験を歓迎すること
- 建設的なフィードバックを尊重をもって与え合うこと

## 期待される行動の例

- 思いやりのある言葉づかいをする
- 異なる視点・経験を尊重する
- 建設的な批評を快く受け入れる
- コミュニティ全体にとって何が最善かを考える
- 他者への共感を示す

## 受け入れられない行動の例

- 性的な言葉や図像を使用すること、性的な誘いをかけること
- 揶揄・侮辱的なコメント、政治的攻撃、個人攻撃
- 公的または私的なハラスメント
- 他者の住所などの個人情報を本人の許可なく公開すること
- 職場としてふさわしくない行為

## 適用範囲

この行動規範は、プロジェクトの公式空間
（GitHub リポジトリ、Issue、プルリクエスト、Discussions、メール、
オンライン/オフラインのイベントなど）全体に適用されます。

## 違反の報告

不適切な行動を目撃または受けた場合は、メンテナまでメールで連絡してください
（メールアドレスは README に記載）。すべての報告は内容を精査し、必要に応じた
対応をいたします。報告者の身元は秘密に保ちます。

## 参照

この行動規範は [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
を簡略化して採用したものです。
