## 葛 飾 応 為 ─ 北 斎 の 娘 が 描 い た 闇 の 光

### 1. 江 戸 ・ 北 斎 工 房 の 末 娘

時 は 江 戸 後 期、 1800 年 頃。 江 戸 で「葛 飾 北 斎」 と い う 浮 世 絵 師 が 40 歳 前 後 の 働 き 盛 り で あ っ た。 そ の 北 斎 に、 三 番 目 の 娘 が 生 ま れ た。 名 は「栄 (え い)」。

栄 は 父 の 工 房 で 育 ち、 物 心 つ く こ ろ に は 父 と 同 じ よ う に 絵 筆 を 取 っ て い た。 父 北 斎 は、 仕 事 に 集 中 す る と 周 り が 見 え な く な る 性 格 で、 娘 を 呼 ぶ と き「お ー い、 お い」 と 大 声 で 叫 ん だ。 そ れ が ま ま、 栄 の 雅 号「応 為 (お う い)」 と な っ た と 伝 え ら れ る。

20 歳 を 過 ぎ た 頃、 応 為 は 一 度 絵 師 の 男 性 と 結 婚 し た が、 折 り 合 い が 悪 く 数 年 で 離 婚。 そ の あ と は 父 北 斎 と 共 に 暮 ら し な が ら、 父 の 弟 子 兼 共 同 制 作 者 と し て 一 生 を 過 ご し た。

***

### 2. 父 北 斎 の「美 人 画 は 応 為 に 及 ば ず」

応 為 は、 父 北 斎 と 同 じ よ う に 様 々 な 主 題 (風 景・人 物・動 植 物) を 描 い た が、 と く に「美 人 画」 で 才 能 を 発 揮 し た。

父 北 斎 は 晩 年、 自 ら 美 人 画 を 描 く こ と が あ っ た が、 そ の 多 く に 応 為 が 関 与 し て い た と さ れ る (北 斎 と の 共 同 制 作)。 そ し て、 北 斎 自 ら「美 人 画 は 応 為 に 及 ば ず (お え い に 及 ば な い)」 と 認 め た と、 江 戸 後 期 の 浮 世 絵 評 論 集 が 伝 え て い る。

世 界 的 に 有 名 な 葛 飾 北 斎 が、 自 ら の 娘 を「自 分 を 超 え る」 と 評 価 し た 美 人 画 の 才 能。 そ れ が 応 為 で あ っ た。

***

### 3. 『夜 桜 美 人 図』 ─ 闇 の 中 の 光

応 為 の 代 表 作 が、 1840 年 代 頃 の『夜 桜 美 人 図』 で あ る。

縦 約 1 メ ー ト ル の 肉 筆 (= 版 画 で な く 紙 に 直 接 筆 で 描 い た 一 点 物) 浮 世 絵。 構 図 は こ う で あ る。

桜 の 木 の 下、 闇 の 中 に 一 人 の 若 い 女 性 が 立 っ て い る。 女 性 は 紙 灯 籠 を 手 に 持 ち、 そ の 淡 い 光 が 女 性 の 顔・着 物・足 元 の 桜 の 花 び ら を、 黄 色 い グ ラ デ ー シ ョ ン で 浮 か び 上 が ら せ る。 背 景 は 真 っ 黒 な 闇。 桜 の 花 は 闇 と 光 の 境 界 で 微 か に 色 を 帯 び る。

これ は、 当 時 の 日 本 浮 世 絵 と し て は 革 命 的 な 表 現 で あ っ た。

***

### 4. 西 洋 絵 画 と 日 本 浮 世 絵 の 出 会 い

そ れ ま で の 日 本 浮 世 絵 は、 「明 る い 平 面 的 表 現」 が 主 流 で あ っ た。 美 人 や 役 者 を、 一 様 な 光 の 中 で、 線 と 色 で 描 い た。 「光 と 影」 の コ ン ト ラ ス ト は、 ほ と ん ど 使 わ れ な か っ た。

一 方、 西 洋 絵 画 で は ル ネ サ ン ス 以 降、 「明 暗 法 (キ ア ロ ス ク ー ロ)」 が 発 達 し て お り、 レ ン ブ ラ ン ト (1606-1669)『夜 警』 等 で「闇 の 中 の 光」 が 表 現 さ れ て き た。

江 戸 後 期 (1800 年 代)、 司 馬 江 漢 (1747-1818) や 葛 飾 北 斎 ら が、 オ ラ ン ダ 経 由 で 入 っ て き た 西 洋 絵 画 の「明 暗 法」 を 部 分 的 に 取 り 入 れ 始 め た。 そ し て、 そ の 完 成 度 を 一 気 に 引 き 上 げ た の が、 北 斎 の 娘 応 為 で あ っ た。

『夜 桜 美 人 図』 の「紙 灯 籠 の 黄 色 い 光、 闇 の 黒、 女 性 の 淡 い 色」 と い う 三 階 調 の コ ン ト ラ ス ト は、 日 本 美 術 史 上 ほ と ん ど 前 例 の な い 革 新 で あ っ た。

***

### 5. 100 年 の 沈 黙 と 1980 年 代 か ら の 再 発 見

し か し、 応 為 の 名 前 は、 父 北 斎 と 比 べ る と、 没 後 ほ と ん ど 知 ら れ な か っ た。 1849 年 北 斎 没、 そ し て 1857 年 頃 応 為 自 身 も 没 し て か ら、 約 100 年。

理 由 は い く つ か あ る。 (1) 応 為 が 父 と の 共 同 制 作 を 多 く 行 い、 単 独 名 義 の 作 品 が 少 な か っ た こ と。 (2) 当 時 の 日 本 で「女 性 画 家」 と し て の 評 価 体 系 が 整 っ て い な か っ た こ と。 (3) 美 術 史 学 そ の も の が 個 人 名 義 中 心 で 工 房 の 共 同 制 作 を 見 落 と し が ち で あ っ た こ と。

し か し 1980 年 代、 美 術 史 家 が 北 斎 工 房 の 共 同 制 作 を 再 調 査 す る 中 で、 『夜 桜 美 人 図』 や『関 羽 図』 等 が 応 為 の 単 独 制 作 と 確 認 さ れ、 さ ら に 北 斎 晩 年 美 人 画 の 多 く に 応 為 が 関 与 し て い た こ と が 学 界 で 認 め ら れ た。

1990 年 代 以 降、 応 為 は「日 本 美 術 史 上 最 も 重 要 な 女 性 画 家」 と 評 価 さ れ る よ う に な っ た。

***

### 6. 2 1 世 紀 の 応 為

2 0 1 5 年、 NHK が 朝 井 ま か て 原 作 の 小 説『眩 (く ら ら)』 を 元 に し た 同 名 ド ラ マ を 放 送 (主 演: 宮 崎 あ お い)。 応 為 の 生 涯 が 全 国 ネ ッ ト で 紹 介 さ れ た。

2 0 1 7 年、 ア ニ メ 映 画『百 日 紅 ~Miss HOKUSAI~』 (杉 浦 日 向 子 原 作、 原 恵 一 監 督) が 公 開、 応 為 の 物 語 が 国 際 的 に 知 ら れ た。 こ の 映 画 は ア ヌ シ ー 国 際 ア ニ メ 映 画 祭 ・ ア ニ メ ー シ ョ ン 神 戸 賞 等 で 受 章。

200 年 の 時 を 経 て、 葛 飾 応 為 の 名 が、 父 北 斎 と 並 ぶ よ う に 立 ち 戻 っ て き た。

***

### 7. 結 び

『夜 桜 美 人 図』 の 闇 の 中 の 光 は、 200 年 経 っ た 今 も、 観 る 者 を 立 ち 止 ま ら せ る。

闇 の 中 か ら 浮 か び 上 が る 一 人 の 女 性。 そ の 顔 は、 江 戸 後 期 の 庶 民 の 美 人 で あ り、 同 時 に、 父 の 工 房 で 一 生 を 描 き 続 け た 一 人 の 女 性 画 家 ─ 葛 飾 応 為 自 身 の 姿 で あ る か も し れ な い。
