## 喜 多 川 歌 麿『婦 女 人 相 十 品』 ─ 浮 世 絵 美 人 画 の 完 成

### 1. 寛 政 期 浮 世 絵 の 黄 金 時 代

寛 政 4 年 (1792 年) か ら 寛 政 5 年 (1793 年) に か け て、 江 戸 の 出 版 業 者・蔦 屋 重 三 郎 か ら、 二 人 の 浮 世 絵 師 の 革 命 的 な 連 作 が 出 版 さ れ た。

一 人 は、 美 人 を 描 く 喜 多 川 歌 麿。 一 人 は、 役 者 を 描 く 東 洲 斎 写 楽。

二 人 と も「大 首 絵 (お お く び え)」 と い う 新 し い 表 現 形 式 を 完 成 さ せ た。 そ れ ま で の 浮 世 絵 は、 美 人 や 役 者 を「全 身 図」 で 描 く こ と が 多 か っ た が、 大 首 絵 で は「顔 と 胸 か ら 上」 を 大 写 し に 描 く。

こ れ は、 浮 世 絵 史 上 の「ク ロ ー ズ ア ッ プ 革 命」 で あ っ た。

***

### 2. 『婦 女 人 相 十 品』 ─ 10 種 の 江 戸 美 人

喜 多 川 歌 麿 が 描 い た『婦 女 人 相 十 品』 は、 江 戸 の 様 々 な 階 層・性 格 の 美 人 を 10 種 類 描 い た 連 作 で あ る。

そ の 中 の 代 表 的 な 一 枚 が「ポ ッ ピ ン を 吹 く 娘」(玉 屋 内 鳰 鳥 図) で あ る。 ガ ラ ス の 玩 具「ポ ッ ピ ン」 を 唇 に 当 て て 吹 こ う と し て い る 若 い 娘。 大 きな 黒 い 瞳、 開 け か け の 口、 そ し て 黒 雲 母 (く ろ き ら) の 光 沢 を 帯 び た 背 景。 一 瞬 の 表 情 が、 永 遠 に 切 り 取 ら れ て い る。

他 に「文 を 読 む 女」「煙 草 の 火 を つ け る 女」「鏡 を 見 る 女」 な ど、 江 戸 の 日 常 の 美 人 が 10 種、 一 人 ひ と り 違 う 性 格・年 齢・職 業 で 描 か れ た。

***

### 3. 黒 雲 母 と い う 技 法

『婦 女 人 相 十 品』 の も う 一 つ の 革 新 は、 「黒 雲 母 摺 り (く ろ き ら ず り)」 と い う 版 画 技 法 で あ っ た。

雲 母 の 粉 を 黒 い 染 料 と 混 ぜ て、 美 人 の 背 景 全 体 に 摺 る。 す る と、 背 景 が キ ラ キ ラ と 光 沢 を 持 つ 黒 と な り、 美 人 の 顔・着 物 の 色 を 一 層 際 立 た せ る。

こ の 技 法 は、 当 時 の 高 級 浮 世 絵 の 印 で あ っ た。 一 枚 一 枚 が 手 摺 り の 多 色 木 版 で、 蔦 屋 が 当 時 の 最 高 の 彫 師・摺 師 を 動 員 し て 制 作 し た 結 晶 で あ っ た。

***

### 4. 制 約 と 創 造 ─ 寛 政 の 改 革

逆 説 的 だ が、 こ の 寛 政 期 浮 世 絵 の 黄 金 時 代 は、 実 は 厳 し い 文 化 統 制 の 中 か ら 生 ま れ た。

1787 年 か ら 松 平 定 信 が「寛 政 の 改 革」 を 推 進 し、 享 楽 文 化 を 弾 圧 し た。 浮 世 絵 の 多 色 摺 り の 色 数、 美 人 画 の 派 手 さ、 役 者 絵 の 役 者 名 表 示 ─ こ う し た 要 素 が 法 令 で 制 限 さ れ た。

し か し、 こ の 制 約 の 中 で、 浮 世 絵 師 た ち は 新 し い 工 夫 を 生 ん だ。 役 者 名 を 出 せ な け れ ば、 表 情 と 衣 装 だ け で 役 者 を 特 定 さ せ る 工 夫 が さ れ た。 色 数 を 減 ら せ ば、 線 と 構 図 で 勝 負 す る 工 夫 が さ れ た。

制 約 が、 創 造 を 生 ん だ。 こ れ が 寛 政 期 浮 世 絵 の 不 思 議 な 美 し さ で あ る。

***

### 5. ジ ャ ポ ニ ス ム ─ ヨ ー ロ ッ パ へ の 衝 撃

100 年 後、 開 国 と と も に 浮 世 絵 が ヨ ー ロ ッ パ に 流 出 し た。 当 初 は 陶 器 の 包 装 紙 と し て、 安 物 と し て 渡 っ た。

し か し、 そ れ を 拾 っ た パ リ の 画 家 た ち が、 浮 世 絵 の 革 命 性 に 気 づ い た。 ゴ ッ ホ、 ロ ー ト レ ッ ク、 モ ネ、 ド ガ、 マ ネ、 ク リ ム ト ─ 印 象 派・ア ー ル ヌ ー ヴ ォ ー の 巨 匠 た ち が、 こ ぞ っ て 浮 世 絵 を 模 写・コ レ ク シ ョ ン し た。

特 に ト ゥ ー ル ー ズ・ロ ー ト レ ッ ク の ム ー ラ ン・ル ー ジ ュ の 踊 り 子 の ポ ス タ ー は、 喜 多 川 歌 麿 の 大 首 絵 の 直 接 的 影 響 を 受 け て い る。 平 面 的 な 構 図、 単 純 化 さ れ た 線、 「顔 を 大 き く 描 く」 と い う 大 胆 さ ─ 全 て、 歌 麿 か ら 学 ん だ も の で あ る。

これ を「ジ ャ ポ ニ ス ム (Japonisme)」 と 呼 ぶ。 寛 政 期 の 江 戸 の 美 人 画 が、 1 世 紀 後 の パ リ で、 印 象 派 を 生 ん だ。 美 術 が 国 境 と 時 代 を 超 え て 影 響 し 合 う、 美 し い 例 で あ る。

***

### 6. 1804 年 の 手 鎖 と 早 す ぎ た 死

し か し、 歌 麿 の 晩 年 は 不 遇 で あ っ た。 1804 年、 太 閤 秀 吉 の 五 妻 を 描 い た 浮 世 絵 が「武 家 の 名 誉 を 損 ね る」 と し て 幕 府 の 怒 り を 買 い、 手 鎖 50 日 の 刑 を 受 け た。 蔦 屋 重 三 郎 は 既 に 1797 年 に 没 し て お り、 歌 麿 は 心 痛 か ら 体 を 崩 し、 1806 年、 54 歳 で 没 し た。

『婦 女 人 相 十 品』 の 10 種 の 江 戸 美 人 は、 し か し、 200 年 を 超 え て 私 た ち の 心 に 残 っ て い る。 ボ ス ト ン 美 術 館、 大 英 博 物 館、 東 京 国 立 博 物 館 ─ 世 界 中 の 美 術 館 で 今 も 観 賞 で き る。
