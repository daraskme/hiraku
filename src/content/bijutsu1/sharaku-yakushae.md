<h3>10 か 月 で 消 え た 謎 の 絵 師</h3>

<p>
寛 政 6 年 (1794) 5 月。 江 戸 の 版 元・蔦 屋 重 三 郎 (つ た や じ ゅ う ざ ぶ ろ う) が、
今 ま で 誰 も 見 た こ と の な い 役 者 絵 を 28 図 一 気 に 出 版 し た。
作 者 の 名 は 「東 洲 斎 写 楽 (と う し ゅ う さ い し ゃ ら く)」。 これ ま で 全 く 無 名 の 絵 師 だ っ た。
</p>

<p>
顔 と 胸 だ け を 大 写 し に し た 「大 首 絵 (お お く び え)」。
役 者 の 個 性 を 極 端 に 誇 張 し、 鋭 い 目・突 き 出 た 鼻・歪 ん だ 口 で、
役 の 心 情 を 抉 る よ う に 描 く。 江 戸 庶 民 は 驚 嘆 し た。
</p>

<p>
し か し、 寛 政 7 年 (1795) 1 月、 約 145 図 を 発 表 し た と こ ろ で、 写 楽 は
突 然 姿 を 消 し た。 そ れ 以 降 の 作 品 も、 本 人 の 記 録 も、 一 切 残 っ て い な い。
</p>

<h3>「三 代 目 大 谷 鬼 次 の 江 戸 兵 衛」 ─ 世 界 が 知 る 一 枚</h3>

<p>
写 楽 の 最 も 有 名 な 作 品 は、 『三 代 目 大 谷 鬼 次 (お お た に お に じ) の 江 戸 兵 衛 (え ど べ え)』。
寛 政 6 年 5 月 初 演 の 歌 舞 伎『恋 女 房 染 分 手 綱 (こ い に ょ う ぼ う そ め わ け た づ な)』 の
悪 役・江 戸 兵 衛 を 演 じ る 大 谷 鬼 次 を 描 い た。
</p>

<p>
鋭 く 吊 り 上 が っ た 目。 大 き く 開 い た 口。 そ し て 何 よ り、 こ ち ら に
突 き 出 さ れ た 両 手 ─ 親 指 と 人 差 し 指 を 広 げ、 今 に も 何 か を 奪 い 取 ろ う と
す る 構 え。 「悪 さ」 が 顔 と 手 の 動 き か ら 立 ち 上 が る。
</p>

<p>
ル ー ヴ ル 美 術 館 ・ ボ ス ト ン 美 術 館 ・ 大 英 博 物 館 ─ 世 界 中 の 美 術 館 が 所 蔵 す る、
浮 世 絵 を 代 表 す る 一 枚。
</p>

<h3>デ フ ォ ル メ ─ 100 年 を 先 取 り し た 表 現</h3>

<p>
写 楽 の 表 現 は、 当 時 と し て は 革 命 的 だ っ た。 役 者 を 「美 し く」 描 く の が
役 者 絵 の 常 識 だ っ た 時 代 に、 写 楽 は 役 者 の 「醜 さ」 や 「特 徴」 を
あ え て 強 調 し た。 デ フ ォ ル メ (誇 張) で あ る。
</p>

<p>
こ の 表 現 は、 100 年 後 の 西 洋 美 術 ─ ピ カ ソ の キ ュ ビ ス ム、 ム ン ク の 表 現 主 義 ─
を 先 取 り す る と も 評 価 さ れ る。 19 世 紀 末 ヨ ー ロ ッ パ で 写 楽 が 発 見 さ れ た 時、
画 家 た ち は 衝 撃 を 受 け た。
</p>

<h3>「写 楽 と は 誰 だ っ た の か」 ─ 200 年 の 謎</h3>

<p>
写 楽 の 正 体 を め ぐ っ て は、 200 年 以 上 諸 説 が 飛 び 交 う。 阿 波 (徳 島)
藩 主・蜂 須 賀 (は ち す か) 家 の 能 役 者・斎 藤 十 郎 兵 衛 (さ い と う じ ゅ う ろ べ え)
説 が 現 在 最 有 力 だ が、 確 定 は し て い な い。
</p>

<p>
歌 麿 や 北 斎 と 同 じ 人 物 だ っ た と す る 説、 複 数 の 絵 師 の 共 同 ペ ン ネ ー ム
説、 海 外 か ら の 帰 化 人 説 ─ 今 も 「日 本 美 術 史 最 大 の 謎」 と し て
研 究 さ れ 続 け る。
</p>

<h3>結 び ─ 君 の 友 達 を 写 楽 風 に</h3>

<p>
浮 世 絵 を 学 ぶ 楽 し み の 一 つ は、 「個 性 を 見 る 眼」 が 育 つ こ と。
広 重 が 風 景 を 静 か に 観 察 す る な ら、 写 楽 は 人 物 の 性 格 を 鋭 く 見 抜 く。
</p>

<p>
今 度、 友 達 や 家 族 の 顔 を 見 た 時、 写 楽 の 眼 を 借 り て み よ う。
そ の 人 の「一 番 目 立 つ 特 徴」 は 何 だ ろ う か?
そ れ を 誇 張 し て 一 枚 に 描 い た ら、 ど ん な 「写 楽 風 ポ ー ト レ ー ト」 が 生 ま れ る だ ろ う か?
</p>
