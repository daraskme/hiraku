<h3>「写 生」 ─ 目 の 前 の 自 然 を、 そ の ま ま 描 く</h3>

<p>
江 戸 中 期、 京 都。 一 人 の 画 家 が 京 都 の 街 ・ 山 ・ 鳥 ・ 花 ・ 虫 を、 毎 日 ス ケ ッ チ し て い ま し た。
名 前 は 円 山 応 挙 (ま る や ま お う き ょ)、 1733–1795。
</p>

<p>
応 挙 が 確 立 し た の は、 「写 生 (し ゃ せ い)」 ─ つ ま り「目 の 前 の も の を そ の ま ま 描 く」 と い う
技 法 で し た。
</p>

<p>
今 で は 当 た り 前 に 見 え る か も し れ ま せ ん。 し か し、 江 戸 中 期 の 日 本 で は、 そ う で は あ り ま せ ん で し た。
</p>

<h3>江 戸 中 期 ま で の 日 本 絵 画 ─ 「型」 を 描 く 伝 統</h3>

<p>
江 戸 中 期 ま で、 日 本 の 絵 画 は 主 に「型 (か た)」 を 描 い て き ま し た ─
</p>

<ul>
  <li><strong>水 墨 画</strong> ─ 中 国 の「山 水 の 型」 を 学 ぶ (本 サ イ ト bijutsu1/sesshu-sansui 既 出)</li>
  <li><strong>狩 野 派</strong> ─ 中 国 風 の「鳥 獣 ・ 草 花 の 型」 を 受 け 継 ぐ</li>
  <li><strong>土 佐 派</strong> ─ 平 安 期 大 和 絵 の「型」 を 守 る</li>
</ul>

<p>
ど の 派 も、 「過 去 の 名 作 の 型 を 模 倣 し、 完 璧 に 描 く」 こ と が 修 行 で し た。 自 然 を 直 接 見 て、
そ の ま ま 描 く ─ そ う い う 発 想 は 主 流 で は あ り ま せ ん で し た。
</p>

<h3>応 挙 の 革 命 ─ 「型」 で な く「実 物」 を 描 く</h3>

<p>
円 山 応 挙 は、 こ の 伝 統 を 革 命 し ま し た。 彼 は ─
</p>

<ul>
  <li>毎 日 京 都 の 街 ・ 山 を 歩 き、 ス ケ ッ チ し た</li>
  <li>「写 生 帖」 (ス ケ ッ チ ブ ッ ク) を 1000 点 以 上 残 し た</li>
  <li>蝶 ・ 鳥 ・ 魚 ・ 花 を 細 部 ま で 観 察 し 描 い た</li>
  <li>長 崎 経 由 で 入 っ た 西 洋 銅 版 画 ・ 解 剖 学 図 を 学 ん だ</li>
  <li>中 国 の「形 を 写 す」 思 想 (写 真 ・ 写 意) を 取 り 入 れ た</li>
</ul>

<p>
そ し て、 西 洋・中 国・日 本 の 技 法 を 融 合 し て、 「日 本 的 な 写 生」 を 創 始 し ま し た。
</p>

<h3>代 表 作『四 季 草 花 図』 ─ 草 花 の 美 し さ</h3>

<p>
応 挙 の 代 表 作 の 一 つ『四 季 草 花 図 (し き そ う か ず)』。 春・夏・秋・冬 の 草 花 を、 一 つ 一 つ
丁 寧 に 写 生 し た 連 作。 桜・菖 蒲・菊・梅 ─ そ れ ぞ れ の 季 節 の 花 を、 実 物 を 観 察 し て 描 い た。
</p>

<p>
中 国 風 の「型」 で 描 か れ た 花 と は 違 い、 「こ の 花 は こ の 京 都 の 庭 で 咲 い た」 と い う 個 別 性 が
感 じ ら れ ま す。 こ れ が 応 挙 の 革 命 で し た。
</p>

<h3>代 表 作『松 に 孔 雀 図』 ─ 大 画 面 の 名 作</h3>

<p>
京 都 の 大 乗 寺 ・ 客 殿 を 飾 る 大 障 壁 画 と し て 描 か れ た『松 に 孔 雀 図』。 巨 大 な 松 の 木 に 孔 雀
(く じ ゃ く) が 止 ま る、 ダ イ ナ ミ ッ ク な 構 図。
</p>

<p>
驚 く べ き は、 孔 雀 の 羽 の 一 つ 一 つ が 写 生 で 描 か れ て い る こ と。 当 時、 京 都 の 寺 院 で 孔 雀 を
飼 う こ と は ま れ で し た が、 応 挙 は 京 都 の 動 物 園 (江 戸 期 の「見 世 物 小 屋」 等) で 孔 雀 を
何 度 も 観 察 し、 細 部 ま で 描 き ま し た。
</p>

<h3>西 洋 画 法 と の 融 合 ─ 遠 近 法 と 陰 影</h3>

<p>
応 挙 が 取 り 入 れ た 西 洋 画 法 ─
</p>

<ul>
  <li><strong>遠 近 法</strong> ─ 遠 い 物 を 小 さ く、 近 い 物 を 大 き く 描 く</li>
  <li><strong>陰 影 法</strong> ─ 光 が 当 た る 方 と 影 の 方 で 色 を 変 え る</li>
  <li><strong>解 剖 学 的 写 実</strong> ─ 動 物 の 骨 格 ・ 筋 肉 を 意 識</li>
</ul>

<p>
こ れ ら 全 て を、 日 本 風 の 紙 (絹 本 ・ 紙 本)・墨・絵 具 で 表 現。 「日 本 風 の 写 実」 を 創 り 出 し ま し た。
</p>

<h3>弟 子 達 ─ 円 山 派 の 拡 大</h3>

<p>
応 挙 に は 多 く の 弟 子 が 集 ま り、 円 山 派 (ま る や ま は) を 形 成 し ま し た ─
</p>

<ul>
  <li><strong>呉 春 (ご し ゅ ん、 1752–1811)</strong> ─ 応 挙 の 弟 子 で、 後 に 池 大 雅 (い け の た い が) 風 の 文 人 画 と 融 合 し て「四 条 派」 を 開 く</li>
  <li><strong>長 沢 蘆 雪 (な が さ わ ろ せ つ、 1754–1799)</strong> ─ 応 挙 の 弟 子 で、 独 創 的・奇 想 的 な 作 風 で「奇 想 派」 と も 呼 ば れ る</li>
  <li><strong>森 徹 山 (も り て つ ざ ん)</strong> ─ 円 山 派 の 主 要 後 継 者</li>
</ul>

<p>
円 山 派 は 江 戸 後 期 ま で 続 き、 上 方 (京 都・大 阪) を 中 心 に 大 き な 影 響 を 与 え ま し た。
</p>

<h3>結 び ─ 「見 る こ と」 か ら 始 め よ う</h3>

<p>
円 山 応 挙 が 教 え る の は、 「絵 を 描 く 前 に、 ま ず 見 る こ と」 で す。 1000 点 以 上 の ス ケ ッ チ
ノ ー ト ─ そ れ は 全 て「見 た こ と の 記 録」 で し た。
</p>

<p>
君 も 写 生 を 試 し て み よ う。 校 庭 の 木 ・ 花 ・ 友 達 の 顔 ─ 何 で も 良 い、 一 枚 ス ケ ッ チ し て み よ う。
そ の 一 枚 か ら、 応 挙 の 250 年 前 の 道 が 始 ま る。
</p>
