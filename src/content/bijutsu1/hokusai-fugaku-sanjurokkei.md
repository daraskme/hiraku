<h3>「神 奈 川 沖 浪 裏」 ─ 世 界 で 最 も 有 名 な 日 本 美 術</h3>

<p>
大 き な 波 が、 三 艘 の 漁 船 を 飲 み 込 ま ん と し て い る。<br>
波 頭 が 鋭 い 爪 の よ う に 砕 け、 飛 沫 が 白 く 飛 ぶ。<br>
そ の 波 の 谷 間 の 奥 に、 小 さ く、 し か し 確 か な 存 在 感 で、 富 士 山 が 見 え る。
</p>

<p>
こ れ が、 葛 飾 北 斎 『<strong>富 嶽 三 十 六 景 ・ 神 奈 川 沖 浪 裏</strong>』(か な が わ お き な み う ら)。<br>
英 語 で 「The Great Wave off Kanagawa」 と 呼 ば れ、 世 界 で 最 も 有 名 な 日 本 美 術 作 品 の 一 つ で あ る。
</p>

<h3>70 歳 を 超 え て の 大 仕 事</h3>

<p>
こ の 作 品 を 描 い た 葛 飾 北 斎 (1760–1849) は ─ 何 と <strong>71 歳</strong> だ っ た。
</p>

<p>
1831 年、 北 斎 71 歳 の と き、『富 嶽 三 十 六 景』 の 出 版 が 始 ま っ た。 全 46 図 (本 編 36 + 追 加 10) の 大 連 作 は、 1834 年 ま で 続 い た。
</p>

<p>
北 斎 は 90 歳 で 没 す る 直 前 ま で、 絵 を 描 き 続 け た。<br>
死 の 直 前、 こ う 語 っ た と 伝 え ら れ る:
</p>

<blockquote>
「あ と 5 年 命 を 借 す こ と が で き れ ば、 真 の 画 工 に な れ る の だ が」
</blockquote>

<p>
90 歳 で も、 「ま だ 上 達 し て い な い」 と。<br>
こ れ が 「画 狂 老 人 卍 (が き ょ う ろ う じ ん ま ん じ)」 と 自 称 し た 北 斎 の 生 涯 で あ る。
</p>

<h3>富 士 山 と い う 主 題</h3>

<p>
『富 嶽 三 十 六 景』 は、 様 々 な 場 所 か ら 見 た 富 士 山 を 描 い た 連 作:
</p>

<ul>
<li>「神 奈 川 沖 浪 裏」 ─ 海 か ら 見 た 富 士</li>
<li>「凱 風 快 晴 (が い ふ う か い せ い)」 ─ 通 称「赤 富 士」</li>
<li>「山 下 白 雨 (さ ん か は く う)」 ─ 雷 雨 の 中 の 富 士</li>
<li>「東 海 道 程 ヶ 谷」 ─ 街 道 か ら 見 た 富 士</li>
<li>「五 百 ら か ん 寺 さ ざ ゐ ど う」 ─ 高 い 寺 か ら 見 た 富 士</li>
</ul>

<p>
ど の 場 所 か ら 見 て も、 富 士 山 は 同 じ「富 士 山」 で あ る。<br>
だ が、 季 節・時 間・天 気・視 点 に よ っ て、 そ の 姿 は 無 限 に 変 化 す る。<br>
北 斎 は、 こ の「変 わ ら ぬ も の の 変 化」 を、 46 通 り に 描 い た。
</p>

<h3>ベ ル リ ン ・ ブ ル ー の 革 命</h3>

<p>
『富 嶽 三 十 六 景』 の 衝 撃 を 強 め た の は、 「<strong>ベ ル リ ン ・ ブ ル ー</strong>」 と い う 鮮 や か な 青 い 顔 料 だ っ た。
</p>

<p>
こ の 顔 料 は、 1704 年 に ド イ ツ で 発 明 さ れ た 化 学 顔 料 で、 江 戸 後 期 に オ ラ ン ダ 経 由 で 日 本 に 輸 入 さ れ 始 め た。<br>
従 来 の 日 本 の 顔 料 で は 出 せ な か っ た、 強 く 鮮 や か な 青。<br>
北 斎 は こ れ を 大 胆 に 使 い、 海・空・富 士 山 の 雪 を 表 現 し た。
</p>

<h3>ジ ャ ポ ニ ス ム ─ ヨ ー ロ ッ パ を 揺 さ ぶ る</h3>

<p>
明 治 維 新 (1868) で 日 本 が 開 国 す る と、 浮 世 絵 が 海 を 渡 っ て ヨ ー ロ ッ パ に 渡 っ た。
</p>

<p>
最 初 は ─ 「日 本 か ら 輸 入 さ れ た 陶 磁 器 を 包 む 包 装 紙 と し て 偶 然 渡 っ た」 と 伝 え ら れ る。<br>
だ が そ の 鮮 や か な 色・大 胆 な 構 図・遠 近 法 の な い 平 面 表 現 が、 ヨ ー ロ ッ パ の 画 家 た ち を 衝 撃 さ せ た。
</p>

<p>
こ う し て 始 ま っ た の が「<strong>ジ ャ ポ ニ ス ム</strong>」(Japonisme):
</p>

<ul>
<li><strong>ク ロ ー ド ・ モ ネ</strong> ─ 自 宅 に 浮 世 絵 を 多 数 飾 り、 庭 の 「日 本 風 太 鼓 橋」 を 作 る</li>
<li><strong>ヴ ィ ン セ ン ト ・ ヴ ァ ン ・ ゴ ッ ホ</strong> ─ 浮 世 絵 を 油 絵 で 模 写 (「タ ン ギ ー 爺 さ ん の 肖 像」 の 背 景 に も 浮 世 絵)</li>
<li><strong>エ ド ガ ー ・ ド ガ</strong> ─ 構 図 と 平 面 表 現 を 学 ぶ</li>
<li><strong>ク ロ ー ド ・ ド ビ ュ ッ シ ー</strong> ─ 交 響 詩『海』(1905) の 楽 譜 表 紙 に「神 奈 川 沖 浪 裏」</li>
</ul>

<p>
印 象 派 の 革 命 ─ 19 世 紀 末 ヨ ー ロ ッ パ 美 術 の 最 大 の 変 革 ─ の 一 因 は、 日 本 の 浮 世 絵 だ っ た の だ。
</p>

<h3>「ザ・グ レ ー ト ・ ウ ェ ー ブ」 の 世 界 制 覇</h3>

<p>
「神 奈 川 沖 浪 裏」 (英 語 名 The Great Wave) は、 21 世 紀 で も 世 界 中 で 愛 さ れ て い る。
</p>

<ul>
<li>大 英 博 物 館 ・ メ ト ロ ポ リ タ ン 美 術 館 ・ ル ー ブ ル 等 に 所 蔵</li>
<li>iPhone の 絵 文 字「🌊」(波) は こ の 絵 を モ デ ル と さ れ る</li>
<li>2024 年 発 行 の 新 千 円 札 の 裏 面 デ ザ イ ン</li>
<li>欧 米 で の 美 術 展 で 常 に 集 客 ト ッ プ ク ラ ス</li>
</ul>

<p>
71 歳 の 江 戸 の 画 師 が 描 い た 一 枚 の 木 版 画 が、 200 年 を 超 え て、 世 界 の 「日 本 文 化 の 象 徴」 と な っ た。
</p>

<h3>君 自 身 で 観 て み よ う</h3>

<p>
『富 嶽 三 十 六 景』 全 46 図 は、 国 立 国 会 図 書 館 デ ジ タ ル コ レ ク シ ョ ン や、 様 々 な 美 術 館 の ウ ェ ブ サ イ ト で 無 料 で 鑑 賞 で き る。
</p>

<p>
46 図 を 一 つ ず つ 見 て、 「自 分 が 一 番 好 き な 富 嶽」 を 探 し て み よ う。<br>
そ し て、 「な ぜ そ れ が 好 き か」 を 言 葉 に し て み よ う。
</p>

<p>
北 斎 が 71 歳 で 描 い た こ の 連 作 が、 今 も 君 の 心 に 何 か を 伝 え る ─ そ れ こ そ が、 真 の 芸 術 の 力 で あ る。
</p>
