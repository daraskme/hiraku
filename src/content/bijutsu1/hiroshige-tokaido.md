<h3>江 戸 か ら 京 都 ま で ─ 53 の 宿 場 を 描 く</h3>

<p>
天 保 4 年 (1833)、 歌 川 広 重 36 歳。 浮 世 絵 師 と し て 駆 け 出 し だ っ た 広 重 が、
人 生 を 変 え る 一 枚 を 出 版 し た ─ 『東 海 道 五 十 三 次』 で あ る。
</p>

<p>
東 海 道 ─ 江 戸 か ら 京 都 ま で の 約 492 km の 街 道。 53 の 宿 場 が 整 備 さ れ、
大 名 行 列・商 人・お 伊 勢 参 り の 庶 民 で 賑 わ う 江 戸 期 最 大 の 街 道 だ っ た。
</p>

<p>
広 重 は、 起 点 の 日 本 橋 か ら 終 点 の 京 三 条 大 橋 ま で、 55 図 で 旅 路 の 風 景 を 描 い た。
雨・雪・月・桜・夕 暮 れ ─ 同 じ 道 で も、 季 節 と 時 間 で 異 な る 表 情 を 見 せ る。
</p>

<h3>「庄 野 ─ 白 雨」 ─ 雨 の 線 を 描 く 革 新</h3>

<p>
全 55 図 の 中 で 最 も 有 名 な の が 「庄 野 ─ 白 雨」 (し ょ う の ─ は く う) で あ る。
庄 野 宿 (現 三 重 県 鈴 鹿 市) で 突 然 の 夕 立 (白 雨) に 襲 わ れ た 旅 人 た ち が、
傘 を さ し て 坂 道 を 走 る 場 面。
</p>

<p>
当 時、 木 版 画 で 「雨 の 線」 を 描 く こ と は 異 例 だ っ た。 広 重 は 細 か な 斜 め の 線 で
夕 立 を 表 現。 動 的 で 臨 場 感 の あ る 構 図 は、 後 の 西 洋 画 家 を 驚 か せ た。
</p>

<h3>広 重 ブ ル ー ─ ゴ ッ ホ を 魅 了 し た 藍</h3>

<p>
広 重 の も う 一 つ の 革 新 は、 鮮 や か な 藍 色 ─ 後 に 「広 重 ブ ル ー」 と 呼 ば れ る 色 調 だ。
オ ラ ン ダ 経 由 で 輸 入 さ れ た 化 学 染 料 「ベ ロ 藍」 (プ ル シ ア ン ブ ル ー) を 多 用 し、
深 い 青 で 空 と 水 を 描 い た。
</p>

<p>
こ の 色 が、 後 に フ ィ ン セ ン ト・フ ァ ン・ゴ ッ ホ (1853–1890) を 魅 了 す る。
ゴ ッ ホ は 広 重 の 『名 所 江 戸 百 景』 の 「大 は し あ た け の 夕 立」 を、
『日 本 趣 味・雨 の 橋』 (1887) と し て 油 絵 で 模 写 し た。
</p>

<h3>ジ ャ ポ ニ ス ム ─ 浮 世 絵 が 西 洋 を 動 か す</h3>

<p>
19 世 紀 後 半、 開 国 し た 日 本 か ら 大 量 の 浮 世 絵 が ヨ ー ロ ッ パ に 流 れ 込 ん だ。
モ ネ・ド ガ・マ ネ・ド ビ ュ ッ シ ー ら 印 象 派 の 巨 匠 た ち は、 浮 世 絵 の 大 胆 な 構 図・
平 面 的 な 色 面・斜 め の 構 図 に 衝 撃 を 受 け た。
</p>

<p>
こ の 現 象 を 「ジ ャ ポ ニ ス ム」 (日 本 趣 味) と い う。 西 洋 美 術 の 革 新 を 牽 引 し た の は、
日 本 の 庶 民 文 化 か ら 生 ま れ た 浮 世 絵 だ っ た の だ。
</p>

<h3>旅 の 美 学 ─ 広 重 が 教 え る こ と</h3>

<p>
広 重 の 風 景 画 は、 単 な る 観 光 案 内 で は な い。 雨 ・ 夕 暮 れ ・ 雪 の 中 を 歩 く 旅 人 の
姿 を 通 し て、 「旅 は 苦 し く も 美 し い」 と い う 江 戸 期 の 美 学 を 伝 え る。
</p>

<p>
君 が 学 校 か ら 帰 る 道、 通 学 路 の 朝 の 光、 雨 の 日 の 校 庭 ─ 全 て が 「広 重 の 視 点」 で
見 れ ば、 一 枚 の 絵 に な る。
</p>

<h3>結 び ─ 一 度 旅 し て、 一 度 描 い て み よ う</h3>

<p>
浮 世 絵 を 鑑 賞 す る こ と は、 江 戸 期 の 旅 を 追 体 験 す る こ と。 そ し て、 自 分 の 街 を
新 し い 目 で 見 る こ と で も あ る。 広 重 の 視 点 を 借 り て、 君 の 日 常 を 一 枚 の 絵 に
し て み よ う。
</p>
