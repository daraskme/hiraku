<h3>本 文 (『木 綿 以 前 の 事』 柳 田 國 男・抜 粋)</h3>

<blockquote>
<p>
私 た ち 日 本 人 が、 今 「衣 服」 と 言 う と き、 ま ず 思 い 浮 か べ る の は、 木 綿 か、 あ る い は 化 学 繊 維 で あ る。<br>
だ が こ れ は、 歴 史 的 に は ご く 最 近 の こ と で あ る。
</p>

<p>
日 本 人 の 衣 生 活 が、 革 命 的 に 変 わ っ た の は、 木 綿 (も め ん) が 普 及 し た <strong>江 戸 中 期</strong> 以 降 で あ る。<br>
そ れ 以 前、 つ ま り 古 代 か ら 中 世 末 期 ま で の 日 本 人 は、 麻 (あ さ) ・ 葛 (く ず) ・ 藤 (ふ じ) ・ 苧 (か ら む し) な ど、 山 野 に 生 え る 植 物 の 繊 維 か ら 衣 服 を 作 っ て い た。
</p>

<p>
こ れ ら の 繊 維 は、 木 綿 と 比 べ る と、 は る か に <strong>ご わ ご わ し て、 硬 く、 着 心 地 が 悪 い</strong>。<br>
夏 は 風 通 し が 良 く 涼 し い と い う 利 点 は あ る が、 冬 は 全 く 暖 か く な い。
</p>

<p>
庶 民 は こ う し た 服 を、 何 年 も、 時 に は 何 十 年 も 着 続 け た。<br>
洗 う の も 一 苦 労、 着 替 え る 服 も そ う 多 く な い ─ そ ん な 暮 ら し が、 何 千 年 も 続 い て き た の で あ る。
</p>

<p>
木 綿 が イ ン ド か ら 中 国 を 経 て 日 本 に 伝 来 し た の は、 鎌 倉 ~ 室 町 時 代。 だ が 当 初 は 高 級 品 で、 一 般 庶 民 に は 手 が 届 か な か っ た。
</p>

<p>
江 戸 時 代 に 入 り、 全 国 で 綿 花 (め ん か) の 栽 培 が 普 及 し、 18 世 紀 (1700 年 代) に は、 つ い に 庶 民 も 木 綿 の 服 を 着 ら れ る よ う に な っ た。<br>
こ の 変 化 は、 日 本 人 の 生 活 を <strong>根 本 か ら 変 え た</strong>。
</p>

<ul>
<li>柔 ら か い 木 綿 の 服 で、 体 が 楽 に な っ た</li>
<li>洗 濯 が し や す く な り、 清 潔 度 が 上 が っ た</li>
<li>冬 の 防 寒 着 と し て 綿 入 れ が 普 及 し、 寒 さ で 死 ぬ 人 が 減 っ た</li>
<li>子 ど も の 服 を 何 着 か 用 意 で き る よ う に な っ た</li>
</ul>

<p>
こ の 「木 綿 革 命」 こ そ、 江 戸 時 代 の 庶 民 の 生 活 を、 静 か に だ が 確 実 に 豊 か に し た 大 き な 出 来 事 で あ っ た。
</p>

<p>
私 た ち は 今、 「服 を 着 る」 こ と を 当 た り 前 と 思 っ て い る。<br>
だ が、 千 年 前 の 日 本 人 か ら 見 れ ば、 こ ん な に 柔 ら か く て 暖 か い 服 を、 子 ど も か ら 大 人 ま で が 毎 日 着 替 え て 暮 ら し て い る 私 た ち の 生 活 は、 ま さ に 「夢 の よ う」 で あ る に 違 い な い。
</p>

<p>
「当 た り 前」 と 思 っ て い る も の が、 実 は 長 い 歴 史 の 結 果 で あ る ─ こ う し た こ と を 教 え て く れ る の が、 民 俗 学 と い う 学 問 で あ る。
</p>
</blockquote>

<h3>解 説 ─ 「木 綿 以 前」 と 「木 綿 以 後」</h3>

<p>
柳 田 國 男 が こ の 本 を 書 い た の は 1939 年 (昭 和 14 年)。<br>
当 時 の 日 本 で は、 木 綿 は す で に 一 般 的 で、 さ ら に 化 学 繊 維 (人 絹 ・ ナ イ ロ ン) も 普 及 し 始 め て い た。
</p>

<p>
柳 田 は こ う 言 う ─ <strong>「便 利 に な っ た 今 こ そ、 そ の 前 の こ と を 思 い 出 そ う」</strong> と。

<h3>「便 利 さ」 の 影 の 部 分</h3>

<p>
木 綿 革 命 ─ そ し て そ の 後 の 化 学 繊 維 革 命 ─ は、 確 か に 私 た ち の 暮 ら し を 楽 に し た。<br>
だ が、 そ れ と 引 き 換 え に、 失 わ れ た も の も あ る。
</p>

<ul>
<li><strong>「物 を 大 切 に す る」 文 化</strong> ─ 安 く 買 え る か ら、 す ぐ 捨 て る。 江 戸 時 代 の 「一 着 を 何 十 年」 と は 真 逆。</li>
<li><strong>地 元 の 素 材 の 利 用</strong> ─ 山 野 の 葛 や 藤 か ら 服 を 作 る 技 術 は、 ほ ぼ 失 わ れ た。</li>
<li><strong>環 境 へ の 負 荷</strong> ─ 大 量 生 産 ・ 大 量 廃 棄 が、 地 球 環 境 を 圧 迫 し て い る。</li>
<li><strong>体 と の 関 係</strong> ─ 自 然 素 材 か ら、 化 学 繊 維 へ。 ア レ ル ギ ー の 増 加 と も 関 係 す る と さ れ る。</li>
</ul>

<p>
柳 田 國 男 が 1939 年 に 教 え た こ と ─ 「便 利 に な っ た 今 こ そ、 そ の 前 の こ と を 思 い 出 そ う」 ─ は、 環 境 問 題 が 深 刻 な 2026 年 の 私 た ち に も、 重 い メ ッ セ ー ジ で あ る。
</p>

<h3>民 俗 学 と い う 学 問</h3>

<p>
柳 田 國 男 (1875–1962) は、 日 本 民 俗 学 の 創 始 者 で あ る。
</p>

<p>
彼 は、 教 科 書 に 書 か れ た 「政 治 史」「戦 争 史」 で は な く、 「<strong>庶 民 の 暮 ら し の 歴 史</strong>」 を 研 究 し た。
</p>

<ul>
<li>遠 野 の 昔 話 を 記 録 し た 『遠 野 物 語』 (1910)</li>
<li>明 治 ・ 大 正 の 庶 民 生 活 を 描 い た 『明 治 大 正 史 世 相 篇』 (1931)</li>
<li>そ し て 衣 生 活 を 描 い た 『木 綿 以 前 の 事』 (1939)</li>
</ul>

<p>
彼 の 学 問 は、 「<strong>名 も な い 人 々 の 暮 ら し こ そ、 本 当 の 歴 史 で あ る</strong>」 と い う 信 念 に 立 つ。
</p>

<h3>家 で 「お じ い ち ゃ ん ・ お ば あ ち ゃ ん の 服」 を 聞 こ う</h3>

<p>
今 度、 家 族 と 集 ま る 機 会 が あ っ た ら、 祖 父 母 や 大 叔 父・大 叔 母 に、 こ う 聞 い て み よ う:
</p>

<ol>
<li>子 ど も の 頃、 ど ん な 服 を 着 て い た?</li>
<li>服 は 何 着 持 っ て い た?</li>
<li>服 は ど こ で 買 っ た? (家 で 作 っ た?)</li>
<li>洗 濯 は ど う し て い た?</li>
<li>季 節 で 服 を ど う 変 え た?</li>
</ol>

<p>
答 え を 聞 く と、 君 自 身 の 暮 ら し と の 違 い に 驚 く は ず だ。<br>
そ し て、 「100 年 で こ ん な に 変 わ っ た」 と い う 実 感 が、 君 の 中 に 残 る。<br>
こ れ こ そ、 柳 田 國 男 が 私 た ち に 伝 え た い、 民 俗 学 の 第 一 歩 で あ る。
</p>
