<h3>本 文 (与 謝 野 晶 子 の 育 児 論 ・ 編 集 部 編)</h3>

<blockquote>
<p>
私 は 12 人 の 子 を 産 み、 そ の う ち 11 人 を 育 て た。 そ し て そ の 育 児 の 合 間 に、 詩 を 書 き、 評 論 を 書 き、 古 典 を 現 代 語 訳 し て き た。
</p>

<p>
育 児 の 経 験 か ら、 私 は こ う 思 う ─
</p>

<p>
<strong>子 ど も を 育 て る こ と は、 一 人 の 「家 庭 の 中 の 仕 事」 で は な い</strong>。<br>
そ れ は、 未 来 の 国 民 を 育 て る、 一 つ の <strong>「国 を 育 て る 仕 事」</strong> な の で あ る。
</p>

<p>
だ か ら こ そ、 育 児 は 母 親 だ け の 仕 事 で あ っ て は な ら な い。<br>
父 親 も、 同 じ よ う に、 こ の 大 仕 事 に 参 加 す る べ き で あ る。<br>
そ し て、 国 家 も 社 会 も、 育 児 を す る 親 を 支 え る 義 務 が あ る。
</p>

<p>
わ が 国 で は、 ま だ 「育 児 は 主 婦 の 仕 事」 と い う 古 い 考 え が 根 強 い。<br>
父 親 が 子 ど も の 世 話 を す る と、 「妻 を 助 け て あ げ て い る」 と 周 り か ら 言 わ れ る。<br>
だ が、 父 親 が 自 分 の 子 を 育 て る の は、 「助 け る」 こ と で は な く、 「当 然 の 義 務」 で あ る。
</p>

<p>
家 庭 と 社 会 は、 切 り 離 さ れ た も の で は な い。<br>
家 庭 で 子 を 育 て る こ と は、 そ の ま ま 社 会 を 育 て る こ と。<br>
家 庭 が 健 全 で あ れ ば、 社 会 も 健 全 に な る。 家 庭 が 弱 け れ ば、 社 会 も 弱 く な る。
</p>

<p>
だ か ら 私 は、 す べ て の 大 人 に こ う 言 い た い ─<br>
「<strong>家 庭 の 中 の 仕 事 を、 軽 ん じ て は な ら な い</strong>」 と。
</p>

<p>
食 事 を 作 り、 子 ど も を 抱 き、 衣 服 を 縫 い、 病 気 を 看 取 り、 子 を 寝 か せ る ─ こ れ ら す べ て が、 国 を 作 る 仕 事 な の で あ る。
</p>
</blockquote>

<h3>解 説 ─ 与 謝 野 晶 子 の 多 才 さ</h3>

<p>
与 謝 野 晶 子 (1878–1942) は、 詩 人 で あ り、 評 論 家 で あ り、 翻 訳 者 で あ り、 そ し て <strong>12 人 の 子 の 母</strong> で あ っ た。
</p>

<p>
彼 女 の 仕 事 は 驚 異 的 で あ る:
</p>

<ul>
<li><strong>詩 集</strong> ─ 約 75 冊 (『み だ れ 髪』『恋 衣』 な ど)</li>
<li><strong>評 論</strong> ─ 多 数 の 雑 誌 連 載・単 行 本</li>
<li><strong>翻 訳</strong> ─ 『新 訳 源 氏 物 語』 (1912–1913、 全 4 巻)、 後 に『新 新 訳 源 氏 物 語』 (1938–1939、 全 6 巻)</li>
<li><strong>育 児</strong> ─ 12 人 出 産、 11 人 を 育 て た</li>
<li><strong>社 会 運 動</strong> ─ 女 性 解 放 運 動・平 和 運 動・教 育 改 革</li>
</ul>

<p>
こ れ ら す べ て を、 一 人 の 人 間 が、 64 年 の 生 涯 で 成 し 遂 げ た。<br>
そ し て 晶 子 自 身、 こ の 仕 事 量 を 「特 別 な こ と」 と は し な か っ た。 育 児 と 仕 事 は、 別 物 で は な く、 同 じ 一 つ の「<strong>生 き る こ と</strong>」 だ っ た の だ。
</p>

<h3>「母 性 保 護 論 争」 (1918–1919)</h3>

<p>
1918 年、 与 謝 野 晶 子 と 平 塚 ら い て う の 間 で、 大 論 争 が 起 こ っ た。
</p>

<p>
<strong>平 塚 ら い て う の 主 張</strong>: 「母 親 は 子 ど も を 育 て る 大 仕 事 を す る か ら、 国 家 が 経 済 的 に 母 親 を 保 護 す べ し」
</p>

<p>
<strong>与 謝 野 晶 子 の 反 論</strong>: 「い や、 そ れ は 女 性 を 国 家 の 保 護 対 象 に す る こ と。 む し ろ 女 性 は 自 分 で 仕 事 を し、 経 済 的 に 自 立 す べ し」
</p>

<p>
こ の 論 争 は 「<strong>母 性 保 護 論 争</strong>」 と 呼 ば れ、 現 代 の 「育 児 休 業 制 度」 「ベ ー シ ッ ク イ ン カ ム」 「働 く 母 親 の 支 援」 と い う テ ー マ の 出 発 点 と な っ た。
</p>

<h3>108 年 後 の 私 た ち へ</h3>

<p>
1918 年 か ら 108 年 ─ 日 本 社 会 は 確 か に 変 わ っ た:
</p>

<ul>
<li>1947 ─ 男 女 平 等 を 定 め た 日 本 国 憲 法</li>
<li>1985 ─ 男 女 雇 用 機 会 均 等 法</li>
<li>1991 ─ 育 児 休 業 法</li>
<li>1999 ─ 男 女 共 同 参 画 社 会 基 本 法</li>
<li>2022 ─ 男 性 育 児 休 業 取 得 率 約 17%</li>
</ul>

<p>
だ が、 与 謝 野 晶 子 が 100 年 前 に 訴 え た 「<strong>父 親 も 共 に 育 て る</strong>」 は、 ま だ 完 全 に は 実 現 し て い な い。<br>
そ れ は、 今 を 生 き る 私 た ち が、 引 き 続 き 取 り 組 む べ き 課 題 で あ る。
</p>

<p>
あ な た の 家 庭 で の 「家 事 ・ 育 児 分 担」 を、 一 度 振 り 返 っ て み て ほ し い。<br>
そ し て、 100 年 前 の 与 謝 野 晶 子 が 投 げ か け た 問 い ─ 「家 庭 と 社 会 は 一 つ で あ る」 ─ を、 自 分 自 身 の 言 葉 で 考 え 直 し て み て ほ し い。
</p>
