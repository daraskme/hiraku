<h3>本文（村井弦斎『食道楽』冬の巻 より抜粋・現代仮名遣い）</h3>

<blockquote>
<p>
冬 の 食卓 の 中心 は、鍋 (なべ) で ある べし。
</p>

<p>
夏 は 冷たく、淡白 に、味 を 濃く ── と 説いた。<br>
冬 は、これ と 全く 逆 で あって、<strong>温 (あたた) かく、こく が あり、家族 で 同じ 鍋 を 分かち 合う</strong> こと が 大切 で ある。
</p>

<p>
寒さ は、まず 体 を 冷やす。 冷えた 体 を 温める に は、外 から の 暖房 だけ で は 足り ない。 体 の 内側 から、温かい 食物 で 温める こと が 必要 で ある。<br>
そこ で、冬 の 食卓 で は、必ず 一品、湯気 の 立つ もの を 中心 に 据 えよ。
</p>

<p>
最 も 望ましい の は、囲炉裏 (いろり) または 火鉢 (ひばち) の 上 に 鍋 を かけ、家族 全員 が その 周り に 集まって、それぞれ の 椀 に 取り 分けて 食べる こと で ある。
</p>

<p>
湯豆腐 ─ 京 の 都 (みやこ) の 雅 (みやび) な 一品。 昆布 だし の 湯 に、豆腐 を 入れる だけ。 醤油 と 薬味 で 食す。<br>
水炊き ─ 鶏 の 肉 と 野菜 を、水 だけ で 煮る。 ポン酢 で。<br>
寄せ鍋 ─ 旬 の 魚 ・ 野菜 ・ 豆腐 を、何 でも 入れる。 家 の 個性 が 出る。<br>
ちゃんこ ─ 力士 が 食べる 大鍋 料理。 たんぱく 質 が 豊富。
</p>

<p>
これ ら すべて に 共通 する の は、<strong>「同じ 鍋 を、家族 で 食べる」</strong> という こと で ある。<br>
各自 が バラバラ の 皿 に 取り 分け られた もの を 食べる の で は ない。 中央 の 一つ の 鍋 から、それぞれ が 自分 の 椀 に 取って 食べる。<br>
これ こそ が、鍋 料理 の 本質 で あ る。
</p>

<p>
食 は、栄養 を 補給 する だけ の 行為 で は ない。<br>
それ は、家族 が 同じ 場 に 集まり、同じ もの を 食べる こと に より、互い の 結びつき を 確かめ 合う 儀式 で も ある。<br>
そして 冬 の 鍋 こそ、その 儀式 に 最も 相応 (ふさわ) しい 形 で ある。
</p>

<p>
寒い 夜、家族 が 鍋 を 囲み、湯気 の 中 から 互い の 顔 を 見つつ、笑い 合い、語り 合う ── これ ほど 幸せ な 食卓 が、他 に あ ろう か。
</p>
</blockquote>

<h3>解説 ─ 「鍋 を 囲む」 という 食 文化</h3>

<p>
日本 の 鍋 料理 は、世界 の 食 文化 の 中 で も、独特 の 位置 を 占める。
</p>

<p>
多くの 国 で は、鍋 で 煮 た 料理 は、調理 後 に 個別 の 皿 に 取り 分けて、食卓 に 出す。 客 は、自分 の 皿 の 中 で 食べる。<br>
だが 日本 で は、<strong>調理 中 の 鍋 そのもの を 食卓 の 中央 に 置き</strong>、家族 が それぞれ の 椀 (わん) で 取って 食べる。
</p>

<p>
これ は、世界 史 上、極めて 興味 深い 文化 で あ る。
</p>

<ul>
<li><strong>中国 の 火鍋 (ホー・グォ)</strong> ─ 同じ く 中央 の 鍋 を 囲む。日本 の 鍋 文化 の 源流 と も 言える。</li>
<li><strong>フランス の フォンデュ</strong> ─ チーズ や ブイヨン の 鍋 を 中央 に 置き、串 で 取って 食べる。</li>
<li><strong>韓国 の チゲ</strong> ─ 個別 の 鍋 で 出される こと が 多い (各自 用)。</li>
</ul>

<p>
これら の 「囲む 料理」 に 共通 する の は、<strong>「同じ 鍋 を 共有 する」</strong> こと で ある。<br>
食事 は 単 なる 栄養 摂取 で は なく、人 と 人 の 関係 を 確認 する 場 で ある ─ そう いう 哲学 が、鍋 料理 文化 の 根底 に ある。
</p>

<h3>冬 の 食材 と 体 の 温め</h3>

<p>
東洋 医学 で は、食材 を 「<strong>陽性 (ようせい)</strong>」 と 「<strong>陰性 (いんせい)</strong>」 に 分ける 考え 方 が ある。
</p>

<p>
体 を 温める 「陽性 食材」:
</p>
<ul>
<li><strong>根 菜</strong> ─ 大根、人参、ごぼう、れんこん</li>
<li><strong>葉 菜</strong> ─ 白菜、春菊、ねぎ、にら</li>
<li><strong>たんぱく 質</strong> ─ 鶏 肉、鴨、鮭、たら、豆腐</li>
<li><strong>香辛料</strong> ─ 生姜、にんにく、唐辛子</li>
</ul>

<p>
これら の 食材 は、冬 の 鍋 に 入れる と、体 の 内側 から 体温 を 高める。 冷蔵 庫 や セントラル ヒー ティング が なかった 明治 時代、こうした 食材 の 選択 と 調理 は、生 きる ため の 必須 知識 だった。
</p>

<h3>現代 の 「個食」 「孤食」 問題</h3>

<p>
120 年 後 の 日本 で は、弦斎 が 描いた 「家族 で 鍋 を 囲む 食卓」 は、急速 に 減って いる。
</p>

<table>
<thead>
<tr><th>用語</th><th>意味</th></tr>
</thead>
<tbody>
<tr><td><strong>共食 (きょうしょく)</strong></td><td>家族・友人 と 同じ 食事 を 共 に する こと</td></tr>
<tr><td><strong>個食 (こしょく)</strong></td><td>同じ 食卓 でも、家族 が それぞれ 別 の もの を 食べる</td></tr>
<tr><td><strong>孤食 (こしょく)</strong></td><td>一人 で 食事 を する</td></tr>
</tbody>
</table>

<p>
内閣府 の 「<strong>食育 に 関する 意識 調査</strong>」 (令和 4 年) に よれ ば:
</p>
<ul>
<li>朝食 を 「家族 と 一緒 に 食べる」 児童 ─ 約 49%</li>
<li>夕食 を 「家族 と 一緒 に 食べる」 児童 ─ 約 70%</li>
<li>「ほとんど 毎日 一人 で 食事 する」 児童 ─ 約 11%</li>
</ul>

<p>
明治 期 の 鍋 を 囲む 食卓 と、現代 の 個食 文化。<br>
便利 さ ・ 多様 性 を 得た 代わり に、<strong>家族 が 「同じ もの を 食べる」 という 単純 な 儀式</strong> を、私 たち は 失い つつ ある の かも しれない。
</p>

<h3>あなた の 家 の 「鍋 の 夜」</h3>

<p>
今夜、あるい は 今 週末、家族 と 一緒 に 鍋 を 囲んで み よう。
</p>

<ol>
<li>鍋 を 何 に する か、家族 で 話し 合う。</li>
<li>買い物 を 一緒 に 行く。</li>
<li>下 ご しらえ (野菜 を 切る、肉 を 切る) を 分担。</li>
<li>テーブル の 中央 に 鍋 を 置き、家族 で 囲む。</li>
<li>湯気 の 中 で、その 日 の 話 を する。</li>
</ol>

<p>
弦斎 が 100 年 以上 前 に 描いた 「<strong>幸せ な 食卓</strong>」 は、君 の 家 で も、今夜、再現 できる。<br>
それ こそ が、家庭科 の 学び の 真 の 目的 で あ る。
</p>
