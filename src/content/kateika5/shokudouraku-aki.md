  - term: 実りの秋 (みのり の あき)

  - term: 新米 (しんまい)
    meaning: その年に収穫されたばかりの米。明治の家庭では、新米が出ると家族で祝った。
  - term: 米櫃 (こめびつ)
    meaning: 米を保存する木製の箱。湿気を防ぐため、蓋がきっちり閉まる作り。明治の家には必ず大きな米櫃があった。
  - term: 梅干し (うめぼし)
    meaning: 完熟した梅を塩漬けにし、紫蘇 (しそ) で色付けし、土用 (7 月) の日光で干したもの。1 年以上経つほど美味しくなる。家庭の常備食の代表。
  - term: 味噌 (みそ)
    meaning: 大豆・米麹・塩を発酵させた調味料。明治の家庭では、秋に「味噌仕込み」を行い、樽 (たる) で 1 年以上熟成させた。各家庭で味が違う「手前味噌」が普通だった。
  - term: 漬物 (つけもの)
    meaning: 野菜を塩・糠 (ぬか)・酢などに漬けて保存する食品。秋に多く仕込まれる「冬野菜の漬物」(白菜・大根) が、冬の食卓を支えた。
  - term: 干し柿 (ほしがき)
    meaning: 渋柿の皮をむき、軒下に吊るして乾燥させた保存食。明治の秋の風物詩。
  - term: 干物 (ひもの)
    meaning: 魚を内臓を取って塩水に浸し、天日干しにした保存食。秋刀魚 (さんま)・鯵 (あじ)・鰯 (いわし) などが多い。
  - term: 発酵 (はっこう)
    meaning: 微生物 (酵母・乳酸菌・麹菌) の働きで、食材が変化する現象。味噌・醤油・漬物・酒など、日本の伝統食の多くが発酵食品。
  - term: 食料自給 (しょくりょう じきゅう)
    meaning: 自分の家で食べるものを、自分の家や地域で生産・保存すること。明治の農家では、ほぼ全ての食料を自給していた。

<h3>本文（村井弦斎『食道楽』秋の巻 より抜粋・現代仮名遣い）</h3>

<blockquote>
<p>
秋 は、実 (みのり) の 季節 で あり、同時 に、冬 を 越す 備え の 季節 で ある。
</p>

<p>
新米 が 米櫃 (こめびつ) に 満ち、芋 が 蔵 に 積まれる。<br>
柿 (かき) は 軒下 に 吊るされ、干 し柿 と なる。<br>
秋 刀 魚 (さんま) は 内臓 を 取って 塩水 に 浸し、天日 で 干す。<br>
これら は、冬 の 食卓 を 支える もの で ある。
</p>

<p>
そして、特 に 主婦 が 心 を 配る べき は、味噌 (みそ) 仕込み と、漬物 の 仕込み で ある。<br>
味噌 は、秋 に 仕込めば、翌年 の 春 や 夏 まで に 熟成 する。各 家庭 で 麹 (こうじ) の 量 や 塩 の 配分 を 変えれば、その 家 ならで は の 「手前 味噌」 が 生まれる。<br>
漬物 は、白菜 や 大根 を 塩 で 漬けて、樽 (たる) に 重し を 載せる。これ も 数 週間 から 数 ヶ月 で 熟成 し、冬 中 の 副菜 と なる。
</p>

<p>
この よう に、秋 に 集中 して 保存食 を 仕込む の は、冬 が 来 る 前 の 自然 の 慣わ し で ある。<br>
食 は、その 季節 の 中 に だけ 完結 する もの で は なく、四季 全体 を 通じて 計画 さ れる もの だ。
</p>

<p>
さて、近代 化 に 伴って、生活 は どんどん 便利 に なって いる。氷 を 入れた 「氷箱 (こおり ばこ)」 ─ 冷蔵 庫 の 原型 ─ も、東京 の 一部 で 普及 し 始め た。<br>
しかし、「便利」 が 「知恵」 を 失わせる の は、ある まじき こと だ。
</p>

<p>
保存食 の 文化 は、単 に 「冬 を 越す ため の もの」 で は ない。<br>
それ は、<strong>食 を 大切 に する 心</strong>、<strong>家族 と 食卓 を 共有 する 心</strong>、そして <strong>四季 を 通じて 生きる 知恵</strong> で ある。
</p>

<p>
主婦 たる 者、秋 の 仕込み を 怠る べから ず。
</p>
</blockquote>

<h3>明治 の 秋 ─ 「冬 を 越す」 という 必要</h3>

<p>
明治 の 日本 の 大多数 は、農村 で 暮らして いた。<br>
冬 が 来れば、畑 から 収穫 で きる もの は 激減 する。雪 に 閉ざされる 地方 で は、4 ヶ月、5 ヶ月 も 新鮮 な 野菜 が 手 に 入らない。
</p>

<p>
だから 秋 ─ 収穫 期 の 終わり に ─ <strong>冬 の 4 ヶ月 分 の 食料</strong> を 保存 食 として 仕込んで おか ねば ならなかった。
</p>

<ul>
<li><strong>米</strong> ─ 米櫃 (こめびつ) で 1 年 分</li>
<li><strong>芋 (さつまいも・じゃがいも・里芋)</strong> ─ 蔵 (くら) で 数ヶ月</li>
<li><strong>柿</strong> ─ 干し柿 で 半年 以上</li>
<li><strong>秋 刀 魚</strong> ─ 干物 で 数ヶ月</li>
<li><strong>梅 (うめ)</strong> ─ 梅干し で 数 年</li>
<li><strong>大豆</strong> ─ 味噌 で 1 年 以上</li>
<li><strong>白菜 ・ 大根</strong> ─ 漬物 で 冬 中</li>
</ul>

<p>
これら は、現代 の 言葉 で 言えば 「<strong>食料 安全 保障</strong>」 の 草 の 根 版 で あった。
</p>

<h3>三 つ の 保存 原理</h3>

<p>
明治 の 保存 食 は、すべて 三 つ の 原理 の うち、どれ か (または 組合せ) を 使って いる。
</p>

<table>
<thead>
<tr><th>原理</th><th>科学 的 仕組み</th><th>代表 例</th></tr>
</thead>
<tbody>
<tr><td><strong>塩漬け</strong></td><td>高い 塩 濃度 で、腐敗 菌 が 増殖 できなく する (浸透圧)</td><td>梅干し・漬物・塩鮭</td></tr>
<tr><td><strong>乾燥</strong></td><td>水分 を 飛ばし、微生物 が 活動 できなく する</td><td>干物・干し柿・するめ</td></tr>
<tr><td><strong>発酵</strong></td><td>有益 な 微生物 (麹菌・乳酸菌) を 増やし、腐敗 菌 を 抑える</td><td>味噌・醤油・漬物・甘酒</td></tr>
</tbody>
</table>

<p>
これ ら は、現代 の 食品 科学 で 言う <strong>「水分 活性 を 下げる」</strong> という 共通 原理 で 説明 できる。<br>
微生物 は 水 が ないと 増えられない。塩 で 水 を 奪い、乾燥 で 水 を 飛ばし、発酵 で 水 の 化学 状態 を 変える ─ どれ も、食 を 微生物 から 守る 知恵 だ。
</p>

<h3>「手前 味噌」 の 多様性</h3>

<p>
明治 の 各 家庭 は、自分 の 家 で 味噌 を 仕込んで いた。麹 の 量、塩 の 配分、大豆 の 種類、熟成 期間 ─ どれも 家 ごと に 異なる。<br>
その 結果、<strong>家 ごと に 味 が 違う</strong>。 自慢 の 「手前 味噌」 を 客 に 振る舞う の が、当時 の 家庭 の 大きな 楽しみ だった。
</p>

<p>
この 多様 性 は、現代 で 言えば 「<strong>テロワール</strong>」 ─ ワイン の 産地 に よる 個性 ─ と 同じ 概念 で ある。<br>
食 は、画一 (がいいつ) で は ない。家 ごと、地域 ごと の 個性 が、食 を 豊か に する。
</p>

<h3>現代 へ の 接続 ─ フード ロス と 持続可能 な 食</h3>

<p>
2020 年代 の 日本 で、最大 の 食 の 問題 は <strong>フード ロス (food loss)</strong> で ある。<br>
年間 約 472 万 トン (令和 4 年・農水省 推計) の 食品 が、まだ 食べられる の に 捨て られて いる。
</p>

<p>
明治 の 「保存 食」 の 文化 は、この 問題 へ の 一つ の 答え で ある。
</p>

<ul>
<li><strong>季節 に 取れた もの</strong> を、<strong>その 季節 に 食べきる</strong>。</li>
<li><strong>余った もの</strong> は、<strong>保存 食</strong> として 加工 し、次 の 季節 に 持ち越す。</li>
<li><strong>家 ごと</strong> に、<strong>自分 たち の 食</strong> を 計画 する。</li>
</ul>

<p>
弦斎 が 「主婦 たる 者、秋 の 仕込み を 怠る べから ず」 と 書いた 心 は、120 年 経った 今 こそ、私 たち 全員 に 向けられて いる の かも しれない。
</p>

<h3>あなた の 家 の 「秋 の 仕込み」 は?</h3>

<p>
家 に 帰って、家族 と 話し て みよう ─
</p>
<ul>
<li>今、家 に ある 保存 食 を 全部 数えて み よう。</li>
<li>その うち、どれ が 「家 で 作った」 もの? どれ が 「買って きた」 もの?</li>
<li>家 で 作る なら、何 か ら 始められる? (梅干し が おすすめ)</li>
</ul>

<p>
明治 の 知恵 は、君 の 家 の 台所 で 再生 できる の だ。
</p>
