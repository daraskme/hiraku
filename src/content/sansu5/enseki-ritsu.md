<h3>円 周 率 と は 何 か</h3>

<p>
ま ず、 自 分 で 確 か め て み よ う。
</p>

<ol>
<li>コ ン パ ス で 円 を 描 く (例: 半 径 5 cm の 円)</li>
<li>紐 (ひ も) を 円 の 周 り に そ っ て 巻 き つ け、 周 の 長 さ を 計 る</li>
<li>そ の 円 の 「直 径」 (= 半 径 × 2) を 計 る</li>
<li><strong>周 の 長 さ ÷ 直 径</strong> を 計 算 す る</li>
</ol>

<p>
ど ん な 大 き さ の 円 で も、 こ の 計 算 を す る と、 だ い た い 同 じ 数 に な る ─ <strong>約 3.14</strong>。<br>
こ れ が 「円 周 率」 (え ん し ゅ う り つ) で あ る。 ギ リ シ ャ 文 字 で 「π (パ イ)」 と 書 く。
</p>

<h3>江 戸 時 代 の 円 周 率</h3>

<p>
円 周 率 を 計 算 し よ う と し た 人 類 の 歴 史 は、 4000 年 以 上 前 ま で 遡 る。<br>
古 代 エ ジ プ ト・古 代 バ ビ ロ ニ ア・古 代 ギ リ シ ャ ─ そ し て 古 代 中 国 ・ 江 戸 日 本 で も、 多 く の 数 学 者 が 円 周 率 の 計 算 に 挑 戦 し て き た。
</p>

<p>
日 本 の 江 戸 時 代 の 動 き を 見 て み よ う:
</p>

</thead>

<p>
特 に、 関 孝 和 (せ き た か か ず) ・ 建 部 賢 弘 (た け べ か た ひ ろ) の 業 績 は、 当 時 の 世 界 最 高 水 準 で あ っ た。
</p>

<h3>関 孝 和 ─ 和 算 史 上 最 大 の 天 才</h3>

<p>
<strong>関 孝 和</strong> (1642 頃 –1708) は、 江 戸 前 期 の 数 学 者。 「和 算 史 上 最 大 の 天 才」 と 呼 ば れ る 人 物 で あ る。
</p>

<p>
彼 は、 ヨ ー ロ ッ パ の 数 学 を <strong>一 切 学 ぶ 機 会 が な か っ た</strong>。<br>
鎖 国 (1639–1853) の 真 っ た だ 中、 西 洋 の 数 学 書 は 日 本 に 入 っ て こ な か っ た か ら だ。
</p>

<p>
そ れ に も か か わ ら ず、 関 孝 和 は:
</p>

<ul>
<li>円 周 率 を 11 桁 ま で 正 確 に 計 算 (西 洋 の ア ル キ メ デ ス・ヴ ァ ン ・ ロ ー モ ン 等 の 結 果 と 一 致)</li>
<li>方 程 式 の 解 法 を 体 系 化</li>
<li><strong>行 列 式</strong> (デ ィ タ ー ミ ナ ン ト) を 発 見 ─ ヨ ー ロ ッ パ の ラ イ プ ニ ッ ツ (1693) に 約 10 年 先 行 し た と 言 わ れ る</li>
<li>「点 竄 術」(て ん ざ ん じ ゅ つ) と い う 代 数 の 体 系 を 確 立</li>
</ul>

<p>
ヨ ー ロ ッ パ と 接 触 し な か っ た 関 孝 和 が、 ヨ ー ロ ッ パ と 同 じ 答 え に た ど り 着 い た ─ こ れ は 何 を 意 味 す る か?<br>
そ れ は、 「<strong>数 学 は 世 界 共 通 の 言 葉</strong>」 と い う こ と で あ る。
</p>

<h3>建 部 賢 弘 ─ 弟 子 が 師 を 超 え る</h3>

<p>
関 孝 和 の 弟 子 <strong>建 部 賢 弘</strong> (1664–1739) は、 師 を も 超 え る 仕 事 を 残 し た。
</p>

<p>
1722 年、 彼 は 著 書 『綴 術 算 経』(て つ じ ゅ つ さ ん け い) で、 円 周 率 を <strong>小 数 点 以 下 42 桁</strong> ま で 正 確 に 計 算 し た。
</p>

<blockquote>
π = 3.14159 26535 89793 23846 26433 83279 50288 4...
</blockquote>

<p>
当 時、 ヨ ー ロ ッ パ で も こ の 桁 数 ま で 計 算 し た 例 は ご く 少 な か っ た。<br>
鎖 国 の 中 の 日 本 で、 世 界 最 高 水 準 の 数 学 が 営 ま れ て い た の だ。
</p>

<h3>「無 限 に 続 く 数」 と い う 不 思 議</h3>

<p>
円 周 率 は、 い く ら 計 算 し て も <strong>終 わ ら な い 数</strong> で あ る。<br>
そ し て、 数 字 の パ タ ー ン が <strong>繰 り 返 さ れ な い</strong>。<br>
こ う し た 数 を 「<strong>無 理 数</strong>」 (む り す う) と 呼 ぶ。
</p>

<p>
小 学 校 で 「π ≒ 3.14」 と 習 う と き、 実 は そ の 背 後 に は、 こ う し た 不 思 議 な 「無 限」 が 隠 れ て い る。
</p>

<p>
2022 年 ─ Google の 技 術 者 が、 コ ン ピ ュ ー タ で 円 周 率 を <strong>100 兆 桁</strong> ま で 計 算 し た (世 界 記 録)。<br>
そ れ で も、 ま だ 「終 わ り」 は 見 え な い。
</p>

<h3>な ぜ 円 周 率 を 計 算 す る の か?</h3>

<p>
「3.14 で 十 分 じ ゃ な い か」 と 思 う か も し れ な い。<br>
だ が、 実 際 の 工 学・物 理 学 で は、 桁 数 が 必 要 に な る 場 面 が あ る:
</p>

<ul>
<li><strong>NASA の 宇 宙 船 軌 道 計 算</strong> ─ 約 16 桁 で 十 分</li>
<li><strong>地 球 の 大 き さ を 1 cm の 精 度 で 計 算</strong> ─ 約 10 桁 必 要</li>
<li><strong>観 測 可 能 な 宇 宙 全 体 を 原 子 の 大 き さ で 計 算</strong> ─ 約 40 桁 必 要</li>
</ul>

<p>
建 部 賢 弘 が 1722 年 に 求 め た 42 桁 は、 実 用 上 ほ ぼ 完 璧 な 精 度 で あ る。<br>
そ れ 以 上 の 桁 数 は、 純 粋 に 「数 学 の 限 界 へ の 挑 戦」 と し て 計 算 さ れ て い る。
</p>

<h3>3 月 14 日 ─ π の 日</h3>

<p>
3.14 ─ こ の 数 字 か ら、 「<strong>3 月 14 日 = π (パ イ) の 日</strong>」 と 呼 ば れ、 世 界 中 の 数 学 者 や 数 学 好 き が 祝 う 日 と な っ た。
</p>

<p>
こ の 日 は、 ア ル ベ ル ト ・ ア イ ン シ ュ タ イ ン の 誕 生 日 で も あ る (1879 年 3 月 14 日)。<br>
偶 然 だ が、 数 学 と 物 理 の 二 大 天 才 が こ の 日 で 結 ば れ て い る の は、 面 白 い。
</p>

<h3>自 分 で 計 算 し て み よ う</h3>

<p>
今 度、 家 に あ る 円 形 の も の (お 皿、 時 計、 コ ッ プ の 底) を 使 っ て、 自 分 で π を 求 め て み よ う。
</p>

<ol>
<li>そ の 円 の 周 り に 紐 を 巻 き、 周 の 長 さ を 計 る</li>
<li>定 規 で 直 径 を 計 る</li>
<li>計 算: 周 ÷ 直 径 = ?</li>
<li>3.14 に 近 い 数 が 出 た か な?</li>
</ol>

<p>
紐 が 完 璧 に ぴ っ た り 巻 け な か っ た り、 計 測 に 誤 差 が あ っ た り す る か ら、 ぴ っ た り 3.14 に は な ら な い だ ろ う。<br>
で も、 3.1 か ら 3.2 の 間 に 入 る は ず だ。
</p>

<p>
関 孝 和 が 400 年 前 に 11 桁 ま で 求 め た こ の 不 思 議 な 数 ─ 君 自 身 の 手 で、 そ の 一 端 に 触 れ て み よ う。
</p>
