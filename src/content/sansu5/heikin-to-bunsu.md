<h3>江 戸 の 庶 民 算 数 ─ 米 を 公 平 に 分 け る</h3>

<p>
江 戸 期 (17–19 世 紀)、 普 通 の 人 々 の 日 常 に 算 数 が あ り ま し た。 一 番 多 い 場 面 は ─
「米 を ど う 分 け る か」 で す。
</p>

<p>
お 父 さ ん が「5 升 の 米 を、 3 人 の 子 ど も で 公 平 に 分 け な さ い」 と 言 い ま す。 ど う 分 け ま す か?
</p>

<p>
これ が、 平 均 ・ 分 数 ・ 割 り 算 の 入 り 口 に な り ま す。 答 え は ─ 5 ÷ 3 = 1 + 2/3 = 1 升 6 合 (お よ そ)
ず つ で す。
</p>

<h3>平 均 ─ 「公 平 に 分 け る」 の 数 字</h3>

<p>
平 均 と は、 「合 計 を 個 数 で 割 っ た 値」 で す。
</p>

<p style="text-align: center; font-size: 1.2em;">
<strong>平 均 = 合 計 ÷ 個 数</strong>
</p>

<p><strong>例 1</strong> ─ 5 人 の 子 ど も の 身 長 が、 130・132・128・135・130 cm。 平 均 身 長 は?</p>

<ol>
  <li>合 計: 130 + 132 + 128 + 135 + 130 = 655 cm</li>
  <li>平 均: 655 ÷ 5 = 131 cm</li>
</ol>

<p><strong>例 2</strong> ─ 1 週 間 の 給 食 残 量 (g): 50, 30, 100, 20, 0。 平 均 は?</p>

<ol>
  <li>合 計: 50 + 30 + 100 + 20 + 0 = 200 g</li>
  <li>平 均: 200 ÷ 5 = 40 g</li>
</ol>

<p>
平 均 は、 「ば ら ば ら の 数 字 を 一 つ の 代 表 値 で 表 す」 便 利 な 道 具 で す。
</p>

<h3>分 数 ─ 「全 体 を 等 し く 分 け る」</h3>

<p>
分 数 と は、 「全 体 を 等 し く い く つ か に 分 け た い く つ 分」 を 表 す 数 で す。
</p>

<ul>
  <li>1 個 の リ ン ゴ を 2 等 分 し た 1 つ = 1/2 (二 分 の 一)</li>
  <li>1 m の 布 を 4 等 分 し た 3 つ 分 = 3/4 (四 分 の 三)</li>
  <li>1 時 間 (60 分) を 12 等 分 し た 1 つ 分 = 5 分 = 1/12 時 間</li>
</ul>

<p>
分 母 (下) = い く つ に 等 分 し た か。 分 子 (上) = そ の う ち い く つ 取 る か。
</p>

<h3>異 分 母 分 数 の 加 減 ─ 通 分 が 鍵</h3>

<p>
小 5 で 一 番 難 し い の は、 分 母 の 違 う 分 数 の 加 減 で す。 例 え ば ─
</p>

<p style="text-align: center; font-size: 1.2em;">
<strong>1/2 + 1/3 = ?</strong>
</p>

<p>
そ の ま ま「1+1=2、 2+3=5」 と し て「2/5」 と し て は い け ま せ ん! 分 母 が 違 う 場 合、 ま ず 通 分 し ま す。
</p>

<p><strong>通 分 の 手 順</strong> ─</p>

<ol>
  <li>共 通 の 分 母 を 見 つ け る (2 と 3 の 最 小 公 倍 数 = 6)</li>
  <li>分 子 と 分 母 を 同 じ 数 で 増 や す: 1/2 = 3/6、 1/3 = 2/6</li>
  <li>通 分 し た ら 加 え る: 3/6 + 2/6 = 5/6</li>
</ol>

<p>
答 え は <strong>5/6</strong> で す。
</p>

<h3>江 戸 算 数 の「米 分 配」 を 分 数 で 解 く</h3>

<p>
冒 頭 の 問 題 に 戻 り ま し ょ う ─ 「5 升 の 米 を 3 人 で 等 し く 分 け る」。
</p>

<ol>
  <li>5 ÷ 3 = 1 余 り 2 = 1 + 2/3 升</li>
  <li>1 升 = 10 合 で す か ら、 2/3 升 = 2/3 × 10 = 20/3 = 6.67 合</li>
  <li>つ ま り、 1 升 6 合 + 残 り 2/3 合 ≈ 1 升 6.7 合</li>
</ol>

<p>
小 さ な 単 位 (合) で 表 す と、 1 人 あ た り 約 16.67 合。 茶 碗 1 杯 ≈ 1 合 と 考 え る と、 約 17 杯 分。
3 人 合 計 50 杯 分。
</p>

<h3>平 均 と 分 数 の 関 係</h3>

<p>
実 は、 平 均 と 分 数 は 深 く 関 係 し て い ま す。 「5 升 を 3 人 で 分 け る」 と は、 「平 均 5/3 升」 で す。
</p>

<p>
平 均 を 求 め る = 個 数 で 割 る = 分 数 で 表 せ る 場 合 が 多 い。 こ の 関 係 を 理 解 す る と、 算 数 の
全 体 像 が 見 え て き ま す。
</p>

<h3>江 戸 期 か ら 受 け 継 ぐ「公 平 さ の 数 字」</h3>

<p>
江 戸 期 の 庶 民 が「公 平 に 分 け る」 為 に 学 ん だ 算 数 が、 現 代 で も 君 の 教 科 書 で 学 ば れ て い ま す。
時 代 が 変 わ っ て も、 「公 平 」 と「分 け 方」 の 重 要 性 は 変 わ り ま せ ん。
</p>

<p>
ク ラ ス で お や つ を 分 け る 時、 部 活 で 当 番 を 決 め る 時、 家 族 で 仕 事 を 分 担 す る 時 ─ 君 の 周 り で
「平 均」「分 数」「割 り 算」 が 使 わ れ て い ま す。 ぜ ひ 意 識 し て み て く だ さ い。
</p>

<h3>結 び ─ 算 数 を 楽 し む 心</h3>

<p>
『塵 劫 記』 が 400 年 前 か ら 教 え る こ と は、 「算 数 は 苦 し い も の で は な く、 楽 し い 道 具 で あ る」
と い う こ と。 公 平 に 分 け る、 平 均 で 全 体 を 知 る、 分 数 で 比 較 す る ─ 全 て 日 常 を 便 利 に す る 道 具 で す。
</p>

<p>
小 5 で 平 均・分 数 を マ ス タ ー し て、 中 学・高 校 の 数 学 へ の 第 一 歩 を 踏 み 出 し ま し ょ う。
</p>
