<h3>算 額 と は 何 か</h3>

<p>
江 戸 時 代 か ら 明 治 期 に か け て、 日 本 で は 「<strong>算 額</strong>」 と い う 不 思 議 な 文 化 が あ っ た。
</p>

<p>
算 額 と は、 数 学 の 問 題 と そ の 解 法 を 絵 馬 (え ま) に 描 い て、 神 社 や 寺 院 に 奉 納 す る も の で あ る。
</p>

<p>
今 か ら 340 年 以 上 前、 1683 年 (天 和 3 年) ─ 栃 木 県 佐 野 市 の 星 宮 (ほ し の み や) 神 社 に、 現 存 す る 最 古 の 算 額 が 奉 納 さ れ た。<br>
そ の 後、 江 戸 時 代 を 通 し て 全 国 に 広 ま り、 明 治 期 ま で に 約 <strong>2000 面 以 上</strong> の 算 額 が 奉 納 さ れ た と 推 定 さ れ る。 現 存 す る の は 約 <strong>900 面</strong>。
</p>

<h3>な ぜ 神 社 に 数 学 を 奉 納 し た の か</h3>

<p>
算 額 が 奉 納 さ れ た 理 由 は、 い く つ か 考 え ら れ る。
</p>

<ol>
<li>
  <strong>感 謝 の 奉 納</strong> ─ 「神 様 の お か げ で、 こ の 難 し い 問 題 が 解 け ま し た」 と お 礼 を 言 う。
</li>
<li>
  <strong>挑 戦 状 の 奉 納</strong> ─ 「こ の 問 題 を 解 け る 人 が い ま す か?」 と 全 国 の 数 学 愛 好 家 に 挑 戦 す る。
</li>
<li>
  <strong>新 発 見 の 報 告</strong> ─ 「私 は 新 し い 数 学 を 発 見 し ま し た。 神 様 に 報 告 し ま す」。
</li>
<li>
  <strong>学 び の 公 開</strong> ─ 「こ う い う 解 き 方 が あ り ま す。 皆 さ ん の 参 考 に」 と 共 有 す る。
</li>
</ol>

<p>
特 に 「挑 戦 状」 と し て の 算 額 は、 各 地 の 数 学 愛 好 家 が 相 互 に 競 い 合 う <strong>「数 学 コ ン テ ス ト」</strong> の よ う な 機 能 を 果 た し て い た。
</p>

<h3>世 界 史 上 稀 有 な「庶 民 の 数 学 文 化」</h3>

<p>
同 時 代 の ヨ ー ロ ッ パ で、 数 学 を 学 ぶ の は 大 学 教 授・聖 職 者・貴 族 ─ つ ま り <strong>エ リ ー ト 層</strong> だ け で あ っ た。
</p>

<p>
だ が 江 戸 時 代 の 日 本 で は ─ 武 士 ・ 商 人 ・ 農 民 が、 寺 子 屋 (て ら こ や) で 「読 み・書 き・算 盤 (そ ろ ば ん)」 を 学 び、 そ し て さ ら に 高 度 な 数 学 を も 楽 し ん だ。
</p>

<p>
庶 民 が 数 学 を 趣 味 と し て 楽 し み、 神 社 に 奉 納 す る ─ こ の 文 化 は <strong>世 界 史 上 で も 極 め て 珍 し い</strong>。
</p>

<p>
明 治 時 代 に 来 日 し た 西 洋 の 数 学 者 た ち は、 各 地 の 神 社 に 算 額 が 奉 納 さ れ て い る の を 見 て、 「な ぜ 庶 民 が こ ん な に 高 度 な 数 学 を 楽 し む の か」 と 驚 い た と 伝 え ら れ る。
</p>

<h3>算 額 で よ く 出 題 さ れ た 問 題</h3>

<p>
算 額 の 問 題 は、 多 く が <strong>円 と 多 角 形 の 図 形 問 題</strong> で あ っ た。
</p>

<h4>例: 「円 の 中 の 三 つ の 円」 問 題</h4>

<blockquote>
<p>
大 き な 円 の 中 に、 同 じ 大 き さ の 3 つ の 小 円 を、 互 い に 接 し、 か つ 大 円 に も 接 す る よ う に 描 く。<br>
大 円 の 半 径 が 6 cm の と き、 小 円 の 半 径 は い く ら か?
</p>
</blockquote>

<p>
こ の 種 の 問 題 が、 算 額 で 何 千 と 出 題 さ れ た。<br>
答 え に は 三 角 関 数・無 理 数・微 積 分 の 萌 芽 ま で 含 ま れ る こ と が あ り、 当 時 の 世 界 最 高 水 準 の 数 学 が、 神 社 の 絵 馬 に 描 か れ て い た の で あ る。
</p>

<h4>な ぜ 図 形 問 題 が 好 ま れ た か</h4>

<ul>
<li><strong>視 覚 的 で 美 し い</strong> ─ 絵 馬 と し て 神 社 に 奉 納 す る の に ふ さ わ し い</li>
<li><strong>答 え が 一 つ に 定 ま る</strong> ─ 解 け た か 解 け な い か が 明 確</li>
<li><strong>計 算 だ け で な く 発 想 力 が 要 る</strong> ─ 図 を 工 夫 す る 楽 し み</li>
<li><strong>誰 で も 取 り 組 め る</strong> ─ 数 字 が 読 め な く て も 図 を 見 て 考 え ら れ る</li>
</ul>

<h3>関 孝 和 と 算 額 文 化</h3>

<p>
和 算 史 上 最 大 の 天 才 ・ <strong>関 孝 和</strong> (本 サ イ ト sansu5/enseki-ritsu で 既 習) も、 算 額 文 化 と 関 わ り が あ っ た。
</p>

<p>
関 自 身 は 直 接 算 額 を 奉 納 し た 記 録 は な い が、 彼 の 弟 子 た ち は 算 額 を 奉 納 し た。<br>
そ し て、 関 の 「点 竄 術 (て ん ざ ん じ ゅ つ)」 と い う 計 算 法 が、 算 額 を 解 く 強 力 な 道 具 と し て、 全 国 に 広 ま っ て い っ た。
</p>

<p>
1683 年 (関 孝 和 41 歳 頃) ─ 最 古 の 算 額 が 奉 納 さ れ た 年 と、 関 孝 和 の 活 躍 期 は 重 な る。<br>
和 算 の 黄 金 期 と、 算 額 文 化 の 出 発 は、 ほ ぼ 同 時 で あ っ た。
</p>

<h3>現 代 の 「算 額」 文 化</h3>

<p>
算 額 文 化 は、 明 治 時 代 に 西 洋 数 学 が 普 及 す る と と も に、 一 度 衰 退 し た。<br>
だ が 21 世 紀 に 入 っ て、 各 地 で <strong>「算 額 文 化 の 復 興」</strong> 運 動 が 起 こ っ て い る。
</p>

<ul>
<li>全 国 各 地 の 神 社 で 「現 代 の 算 額」 を 奉 納 す る イ ベ ン ト</li>
<li>数 学 オ リ ン ピ ッ ク と 算 額 を 結 び つ け る 動 き</li>
<li>「日 本 数 学 史 学 会」 に よ る 算 額 調 査</li>
<li>小 中 高 校 で 算 額 を 教 材 と し て 用 い る 学 校 の 増 加</li>
</ul>

<p>
江 戸 時 代 の 庶 民 が 数 学 を 楽 し ん だ 文 化 を、 21 世 紀 の 私 た ち が 受 け 継 ぐ ─ そ ん な 動 き が、 静 か に 進 ん で い る。
</p>

<h3>あ な た 自 身 の 「算 額」 を 作 っ て み よ う</h3>

<p>
ぜ ひ、 自 分 で 「現 代 の 算 額」 を 作 っ て み よ う。
</p>
