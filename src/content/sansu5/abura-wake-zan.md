<h3>問題 (『塵劫記』 より、現代語化)</h3>

<blockquote>
<p>
ここ に、油 が 10 升、ぴったり 入った 大桶 が ある。<br>
そして、空 の 桶 が 二つ。 一 つ は 7 升 入り、 もう 一 つ は 3 升 入り。
</p>

<p>
この 三 つ の 桶 だけ を 使って、10 升 の 油 を、ちょうど <strong>5 升 と 5 升 の 二 つ</strong> に 分けよ。<br>
ただし、油 は 一滴 も こぼ さず、他 の 容器 も 使わ ない こと。
</p>
</blockquote>

<h3>条件 を 整理 しよう</h3>

<ul>
<li>桶 は 3 つ ─ (10 升 入り、7 升 入り、3 升 入り)。</li>
<li>最初 の 状態 ─ (10, 0, 0)。</li>
<li>目標 の 状態 ─ (5, 5, 0)。</li>
<li>許される 操作 ─ 「ある 桶 から 別 の 桶 へ、油 を 移す」 だけ。</li>
<li>桶 は <strong>「ちょうど 空」</strong> か <strong>「ちょうど いっぱい」</strong> に なる まで しか 量を 計れない。</li>
</ul>

<h3>なぜ 江戸 の 人 は こんな パズル を 楽しん だ の か</h3>

<p>
『塵劫記』 が 出版 された 寛永 4 年 (1627) ─ それ は 江戸 幕府 が 成立 して まだ 24 年。日本 が ようやく 平和 を 取り戻し、町人 文化 が 花開き 始めた 時期 で あった。
</p>

<p>
当時 の 庶民 は、識字率 が 急速 に 上昇 し、寺子屋 (てらこや) で 「読み・書き・算盤 (そろばん)」 を 学んだ。<br>
『塵劫記』 は その 算盤 教育 の 教科書 と して 大 流行 した。 鶴亀算・旅人算・鼠算・油分け算 ─ こうした 「面白 算術」 が、寺子屋 の 生徒 たち を 夢中 に させた。
</p>

<p>
さらに、江戸 期 に 発達 した <strong>「算額 (さん がく)」</strong> 文化 ─ 神社 の 絵馬 に 数学 の 難問 を 奉納 する 習慣 ─ も、こうした 数学 パズル 文化 の 延長 線 上 に ある。<br>
日本人 は、世界 で も 例外 的 に、<strong>庶民 が 数学 を 楽しむ</strong> 文化 を 持って いた の だ。
</p>

<h3>現代 の 「水汲み 問題」 と</h3>

<p>
この 形 の パズル は、現代 で は 「<strong>水汲み 問題</strong>」 (water pouring puzzle) と 呼ばれて、コンピュータ 科学 の 探索 アルゴリズム の 教材 と して も 使われる。
</p>

<ul>
<li>1995 年 の 映画 『<strong>ダイ・ハード 3</strong>』 で、主人公 が 5 ガロン と 3 ガロン の 容器 で 4 ガロン を 量る 場面 が 有名。</li>
<li>コンピュータ で 全 探索 を かければ、最短 解 を 機械 的 に 求められる。</li>
<li>容量 が <strong>互いに 素</strong> (最大公約数 が 1) ならば、ほぼ あらゆる 量 が 作れる ─ という 数学 的 結果 が ある。</li>
</ul>

<p>
1627 年 の 京都 の 油 桶 が、1995 年 の ハリウッド 映画 と、そして 21 世紀 の コンピュータ 教育 と、こうして 繋がって いる ─ それ が 数学 の 楽しさ で ある。
</p>

<h3>挑戦 ─ 別 の サイズ で やってみよう</h3>

<p>
さあ、君 も 別 の 数字 で やって みよう。
</p>

<ol>
<li><strong>6 升 入り と 4 升 入り</strong> で、10 升 の 油 を 5 升 ずつ に。── これ は 解 が ない! (理由 を 考え よう)</li>
<li><strong>8 升 入り と 5 升 入り</strong> で、10 升 の 油 を 5 升 ずつ に。── 何 手 で 解ける か?</li>
<li><strong>6 升 入り と 5 升 入り</strong> で、11 升 の 油 を 1 升 だけ 取り出せる か?</li>
</ol>

<p>
解 が ある か 否か を、状態 表 と 「最大 公約数」 の 性質 から 考えて み よう。
</p>
