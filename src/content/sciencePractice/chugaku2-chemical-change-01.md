problems:
  - prompt: 物質が別の物質に変わる変化を何といいますか。
    answer: 化学変化
    explanation: 化学変化では、もとの物質とは性質の違う別の物質ができます。
    difficulty: basic
    type: term
    hints:
      - 状態変化とは違います。
  - prompt: 水を電気分解すると、主に何と何に分かれますか。
    answer: 水素と酸素
    explanation: 水は水素と酸素からできているため、電気分解で水素と酸素に分かれます。
    difficulty: basic
    type: chemical-change
    hints:
      - 水の化学式は H2O です。
  - prompt: 鉄が酸素と結びつく変化を何といいますか。
    answer: 酸化
    explanation: 物質が酸素と結びつく化学変化を酸化といいます。
    difficulty: basic
    type: term
    hints:
      - さびにも関係します。
  - prompt: 酸化銅から酸素を取り除いて銅に戻すような変化を何といいますか。
    answer: 還元
    explanation: 酸化物から酸素を取り除く変化を還元といいます。
    difficulty: standard
    type: term
    hints:
      - 酸化の反対として考えます。
  - prompt: 化学変化の前後で、全体の質量はどうなりますか。
    answer: 変わらない
    explanation: 閉じた容器の中では、反応前後で物質全体の質量は保存されます。
    difficulty: standard
    type: law
    hints:
      - 質量保存の法則です。
  - prompt: 炭酸水素ナトリウムを加熱すると、気体が発生します。この気体を石灰水に通すと白くにごりました。発生した気体は何ですか。
    answer: 二酸化炭素
    explanation: 二酸化炭素は石灰水を白くにごらせます。
    difficulty: standard
    type: experiment
    hints:
      - 石灰水で調べる代表的な気体です。
  - prompt: 化学反応式で、左辺と右辺の原子の種類と数をそろえる理由を書きましょう。
    answer: 化学変化で原子はなくならず、新しく生まれもしないから
    explanation: 原子の組み合わせは変わりますが、原子の種類と数は保存されます。
    difficulty: challenge
    type: explanation
    hints:
      - 質量保存にもつながります。
  - prompt: 鉄と硫黄を加熱してできる硫化鉄は、鉄と硫黄を混ぜただけのものと性質が同じですか。
    answer: 同じではない
    explanation: 硫化鉄は化学変化でできた別の物質なので、鉄や硫黄とは性質が異なります。
    difficulty: challenge
    type: concept
    hints:
      - 磁石につくかどうかなどが変わります。
