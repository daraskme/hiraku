problems:
  - prompt: 気温をはかるとき、温度計は日なたと日かげのどちらに置くのがよいですか。
    answer: 日かげ
    explanation: 直射日光が当たると温度計そのものがあたためられ、空気の温度より高く出ることがあります。
    difficulty: basic
    type: observation
    hints:
      - 空気の温度をはかります。
  - prompt: 春から夏にかけて、ふつう気温はどう変わりますか。
    answer: 高くなる
    explanation: 太陽の光が強くなり、昼の時間も長くなるため、気温が上がりやすくなります。
    difficulty: basic
    type: explanation
    hints:
      - 夏は暑い季節です。
  - prompt: 秋から冬にかけて、植物のようすにはどんな変化が見られますか。
    answer: 葉の色が変わる、葉が落ちる、成長がゆっくりになるなど
    explanation: 気温が下がり、日照時間も短くなるため、植物の活動が変化します。
    difficulty: basic
    type: observation
    hints:
      - 校庭の木を思い出しましょう。
  - prompt: 同じ場所で気温をくらべるとき、毎日同じ時刻にはかるのはなぜですか。
    answer: 時刻による気温の違いを小さくするため
    explanation: 朝、昼、夕方では気温が違うので、時刻をそろえると季節による変化を比べやすくなります。
    difficulty: standard
    type: explanation
    hints:
      - 比べる条件をそろえます。
  - prompt: 昆虫が冬にあまり見られなくなる理由を一つ書きましょう。
    answer: 気温が低くなり、活動しにくくなるから
    explanation: 多くの昆虫は寒い時期に活動を弱めたり、卵やさなぎなどで冬をこしたりします。
    difficulty: standard
    type: explanation
    hints:
      - 生き物の活動と温度を結びつけます。
  - prompt: 気温の記録をグラフにすると、何がわかりやすくなりますか。
    answer: 気温の上がり下がりや季節ごとの変化
    explanation: 数字を並べるだけより、線や棒で表すと変化の流れが見えやすくなります。
    difficulty: standard
    type: data
    hints:
      - 変化を目で見ます。
  - prompt: 夏の昼、地面近くがとても暑くなる理由を説明しましょう。
    answer: 太陽の光で地面があたためられ、地面の近くの空気もあたたまるから
    explanation: 地面は日光を受けて熱をもち、その熱が近くの空気に伝わります。
    difficulty: challenge
    type: explanation
    hints:
      - まず地面があたたまります。
  - prompt: 季節の変化を調べる観察で、気温以外に記録するとよいものを一つ書きましょう。
    answer: 天気、植物の葉のようす、花の有無、昆虫の数など
    explanation: 気温と生き物のようすを合わせて見ると、季節の変化を考えやすくなります。
    difficulty: challenge
    type: observation
    hints:
      - 生き物や天気も記録できます。
