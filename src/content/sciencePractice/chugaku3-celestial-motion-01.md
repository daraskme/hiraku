tags:

problems:
  - prompt: 地球が地軸を中心に1日に1回回転する運動を何といいますか。
    answer: 自転
    explanation: 地球の自転によって、太陽や星が東から西へ動くように見えます。
    difficulty: basic
    type: term
    hints:
      - 昼と夜に関係します。
  - prompt: 地球が太陽のまわりを1年に1回回る運動を何といいますか。
    answer: 公転
    explanation: 地球は太陽のまわりを公転しており、季節の変化にも関係します。
    difficulty: basic
    type: term
    hints:
      - 1年に関係します。
  - prompt: 太陽が東からのぼり西へ沈むように見えるのは、地球の何が原因ですか。
    answer: 地球の自転
    explanation: 地球が西から東へ自転しているため、太陽は東から西へ動くように見えます。
    difficulty: basic
    type: explanation
    hints:
      - 見かけの動きです。
  - prompt: 北の空で、ほとんど動かないように見える星を何といいますか。
    answer: 北極星
    explanation: 北極星は地球の地軸の延長方向近くにあるため、位置がほとんど変わらないように見えます。
    difficulty: standard
    type: term
    hints:
      - 方角を知る目印になります。
  - prompt: 季節によって見える星座が変わる主な理由は何ですか。
    answer: 地球が太陽のまわりを公転しているから
    explanation: 地球の位置が季節ごとに変わるため、夜に見える宇宙の方向も変わります。
    difficulty: standard
    type: explanation
    hints:
      - 1年の動きを考えます。
  - prompt: 夏に太陽の南中高度が高くなると、地表はどうなりやすいですか。
    answer: あたたまりやすくなる
    explanation: 太陽の光が地面により集中して当たるため、気温が高くなりやすくなります。
    difficulty: standard
    type: explanation
    hints:
      - 光の当たり方を考えます。
  - prompt: 同じ星を1時間後に観察すると、どちらの方角へ動いたように見えますか。
    answer: 西
    explanation: 地球の自転により、星は東から西へ動くように見えます。
    difficulty: challenge
    type: observation
    hints:
      - 太陽の見かけの動きと同じ向きです。
  - prompt: 日本で冬の昼が夏より短い理由を、地軸の傾きと関係づけて説明しましょう。
    answer: 冬は北半球が太陽から少しそむく向きになり、太陽が出ている時間が短くなるから
    explanation: 地軸が傾いたまま公転するため、季節によって太陽の通り道と昼の長さが変わります。
    difficulty: challenge
    type: explanation
    hints:
      - 北半球と太陽の向きを考えます。
