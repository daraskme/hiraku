school_stage: shogaku

subject: 理科
unit: 磁石
order: 1
description: 磁石につくもの、N極・S極、引き合う・しりぞけ合う性質を確認する基礎プリント。
tags:
  - 小3
  - 理科
  - 磁石
problems:
  - prompt: 磁石につきやすいものは、主にどんな材料でできていますか。
    answer: 鉄
    explanation: 磁石は鉄でできたものを引きつけやすいです。
    difficulty: basic
    type: term
    hints:
      - くぎやクリップを思い出しましょう。
  - prompt: 磁石のはしにある、力が強くはたらくところを何といいますか。
    answer: 極
    explanation: 磁石にはN極とS極があります。
    difficulty: basic
    type: term
    hints:
      - N極、S極という言い方をします。
  - prompt: N極とS極を近づけると、どうなりますか。
    answer: 引き合う
    explanation: ちがう極どうしは引き合います。
    difficulty: basic
    type: concept
    hints:
      - ちがう名前の極です。
  - prompt: N極とN極を近づけると、どうなりますか。
    answer: しりぞけ合う
    explanation: 同じ極どうしはしりぞけ合います。
    difficulty: basic
    type: concept
    hints:
      - 同じ名前の極です。
  - prompt: 磁石の力は、紙を一枚はさんでもはたらきますか。
    answer: はたらく
    explanation: 紙やうすい板をはさんでも、磁石の力がはたらくことがあります。
    difficulty: standard
    type: experiment
    hints:
      - クリップを紙の下に置いて試す実験です。
  - prompt: 方位磁針の針は、何の力を利用していますか。
    answer: 磁石の力
    explanation: 方位磁針の針は小さな磁石で、地球の磁石の力を受けて向きを示します。
    difficulty: standard
    type: concept
    hints:
      - 方位磁針の針も磁石です。
  - prompt: アルミニウムの缶は、ふつう磁石につきますか。
    answer: つかない
    explanation: アルミニウムは鉄ではないため、ふつうの磁石にはつきません。
    difficulty: challenge
    type: material
    hints:
      - 鉄かどうかを考えます。
  - prompt: 磁石を使って、鉄でできたものを調べる方法を一つ書きましょう。
    answer: 磁石を近づけて、引きつけられるか調べる
    explanation: 引きつけられれば、鉄をふくむ可能性があります。
    difficulty: challenge
    type: experiment
    hints:
      - 近づける、つく、つかないを観察します。
---
