<h3>現代語訳</h3>

<p>
まだ 前髪 を あげたばかり の 少女 が、林檎 の 木 の もと に 立って いた とき、<br>
その 髪 に さした 花櫛 が 美しく、── 「ああ、この 子 こそ 花 の 君 だ」 と、私 は 心 に 思った。
</p>

<p>
彼女 が、やさしく 白い 手 を 伸ばして、林檎 を 私 に 渡して くれた。<br>
その うす紅 (くれない) 色 の 秋 の 実 を 通して、私 は 初めて 「人 を 恋する」 という こと を 知った。
</p>

<p>
私 の 思わず こぼれた ため息 が、その 髪 に かかった とき、<br>
私 は、楽しい 恋 の 盃 (さかずき) を、君 (彼女) の やさしさ の おかげ で、汲んで 飲む こと が できた の だった。
</p>

<p>
林檎畑 の 木 の 下 に、いつしか 自然 に できた 細い 道。<br>
彼女 は そっと たずねる ──「これは、だれが ふみ始めた、思い出 の 道 でしょうか」 と。<br>
そう たずねる 君 の 声 こそ、何より 恋しい もの な の だ。
</p>

<p>
この 4 連 は、たった 1 つ の 恋 の 始まり と 確認 を、見事 に 物語って いる。<br>
林檎 の 木 の 下 で 出会い、林檎 を 受け取り、ため息 を こぼし、そして 「あなた も 私 を 想って いた のね」 と 二人 の 想い が ぴたり と 合う ─ その 一連 の 流れ が、たった 16 行 に 収められて いる。
</p>

<h3>日本 近代詩 の 出発点</h3>

<p>
1897 年 (明治 30 年) 8 月、島崎藤村 は 第一詩集 『若菜集』 を 春陽堂 から 刊行 した。<br>
その 巻頭 を 飾る の が、この 「初恋」 で ある。
</p>

<p>
この 詩集 は、日本 の 詩 の 歴史 で、極めて 大きな 意味 を 持って いる。
</p>

<ul>
<li>それまで の 漢詩 や 和歌 とは 異なる、<strong>七五調 の 新しい 詩形</strong>。</li>
<li>西洋 ロマン 主義 の 影響 を 受けた、<strong>個人 の 感情</strong> を うたう 詩。</li>
<li>古典 の 用語 を 残しつつ、<strong>近代 的 な 主題</strong> (初恋・自我・自然) を 扱う。</li>
</ul>

<p>
こうして 藤村 の 『若菜集』 は、<strong>日本 近代詩 の 出発点</strong> として、後 の 北原白秋・三木露風・萩原朔太郎 など、多くの 詩人 たち に 影響 を 与えた。
</p>

<h3>明治 の 言葉 で 読む 「初恋」</h3>

<p>
明治 30 年 (1897 年) ── いま から 130 年 ちかく まえ。<br>
だが、林檎 の 木 の 下 で 起こった この 小さな 物語 は、いま 読んでも 心 を 打つ。
</p>

<p>
それは、「初恋」 という 経験 が、時代 を 超えて 普遍 的 だから だ。<br>
言葉 は 古くても、心 の 動き は、今 の 君 の もの と 同じ。<br>
ぜひ、声 に 出して 読み、七五調 の リズム の なか に、明治 の 少年・少女 の こころ を 感じて ほしい。
</p>
