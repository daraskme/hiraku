<h3>本 文 (抜 粋・現 代 仮 名 遣 い)</h3>

<blockquote>
<p>
私 (大 津) は、 こ こ 数 年、 一 つ の 草 稿 を 書 い て い ま す。 タ イ ト ル は 「忘 れ え ぬ 人 々」 と 言 い ま す。
</p>

<p>
こ の 「忘 れ え ぬ 人 々」 と は、 ど う い う 人 々 か。<br>
私 が 言 う 「忘 れ え ぬ 人」 と は、 親 兄 弟 で は な い。 知 人 ・ 朋 友 (ほ う ゆ う) で も な い。 妻 子 ・ 親 戚 で も な い。
</p>

<p>
そ れ は、 旅 の 途 中 で、 ほ ん の 一 瞬 す れ 違 っ た だ け の、 <strong>名 も 知 ら ぬ 人 々</strong> な の で す。<br>
け れ ど、 不 思 議 な こ と に、 そ の 一 瞬 の 出 会 い が、 何 年 経 っ て も 心 に 残 り、 ふ と し た 瞬 間 に 思 い 出 さ れ る ─ そ ん な 人 々 で す。
</p>

<p>
た と え ば、 こ う い う 人 々 で す。
</p>

<p>
あ る 春 の 日、 私 が 瀬 戸 内 海 を 船 で 渡 っ て い た 時 の こ と。<br>
小 さ な 島 の 浜 辺 で、 一 人 の 男 が、 一 心 不 乱 に 何 か を 拾 い 集 め て い ま し た。<br>
近 づ い て み る と、 そ れ は 流 木 を 集 め て い る の で し た。 寒 い 季 節 の 暖 房 用 で し ょ う か。<br>
彼 は 私 が 通 り 過 ぎ る の も 気 づ か ず、 た だ 黙 々 と 仕 事 を し て い ま し た。
</p>

<p>
そ の 男 と は、 一 言 も 交 わ さ ず、 名 前 も 聞 か ず、 顔 も よ く 覚 え て い ま せ ん。<br>
で も、 そ の 後 ろ 姿 の 寂 し さ と、 黙 々 と 働 く 姿 は、 何 年 経 っ て も 私 の 心 か ら 離 れ な い の で す。
</p>

<p>
ま た、 別 の 旅 の 時。<br>
信 州 の 山 道 で、 一 人 の 琵 琶 法 師 (び わ ほ う し) が 歩 い て い ま し た。<br>
盲 目 (も う も く) で あ り な が ら、 杖 を つ い て 一 人 で 山 を 越 え て ゆ く。<br>
私 が 「ど こ へ 行 か れ る の で す か」 と 聞 い て も、 彼 は 静 か に 微 笑 む だ け で 答 え ま せ ん で し た。
</p>

<p>
こ の 琵 琶 法 師 も、 私 の 「忘 れ え ぬ 人 々」 の 一 人 で す。
</p>

<p>
親 ・ 兄 弟 ・ 友 人 と い っ た 「親 し い 人 」 は、 当 然 忘 れ ら れ な い。 そ れ は 当 た り 前 で す。<br>
だ が、 こ の 「す れ 違 っ た だ け の 人 」 が、 な ぜ 私 の 心 に こ ん な に 深 く 残 る の か。
</p>

<p>
そ れ は、 思 う に、 <strong>そ の 一 瞬 に、 私 と そ の 人 が、 同 じ 一 つ の 世 界 を 共 有 し た</strong> か ら で は な い で し ょ う か。<br>
浜 辺 で 流 木 を 拾 う 男 も、 山 を 越 え る 琵 琶 法 師 も、 そ の 時 そ こ で、 自 分 の 人 生 を 一 心 に 生 き て い た。<br>
そ し て、 そ の 姿 を、 私 が た ま た ま 見 た。
</p>

<p>
そ う し た 「一 瞬 の 共 有」 が、 か え っ て 親 し い 人 よ り も、 深 い 印 象 を 残 す こ と が あ る の で す。
</p>

<p>
だ か ら 私 は、 こ う し た 名 も 知 ら ぬ 人 々 を、 一 人 ず つ 草 稿 に 書 き 留 め て い ま す。<br>
そ れ が、 私 の 「忘 れ え ぬ 人 々」 で す。
</p>
</blockquote>

<h3>解 説 ─ 「忘 れ え ぬ 人」 と い う 不 思 議</h3>

<p>
国 木 田 独 歩 が こ の 短 編 で 描 い た の は、 不 思 議 な 「忘 れ え ぬ 人」 と い う 概 念 で あ る。
</p>

<p>
普 通、「忘 れ ら れ な い 人」 と 言 え ば ─
</p>

<ul>
<li>家 族 (親・兄 弟・子 ど も)</li>
<li>恩 師 ・ 恩 人</li>
<li>初 恋 の 人</li>
<li>親 友</li>
</ul>

<p>
こ れ ら の 人 々 が 思 い 浮 か ぶ だ ろ う。<br>
独 歩 は こ う 言 う:「い や、 こ う い う 親 し い 人 を 私 は 言 わ な い。 私 が 言 う 『忘 れ え ぬ 人』 は ─ <strong>旅 で す れ 違 っ た、 名 も 知 ら ぬ 人 々</strong> な の だ」 と。
</p>

<h3>な ぜ「す れ 違 っ た 人」 が 忘 れ ら れ な い の か</h3>

<p>
こ の 不 思 議 な 現 象 に は、 い く つ か の 理 由 が 考 え ら れ る。
</p>

<ol>
<li>
  <strong>純 粋 な 印 象</strong> ─ 親 し い 人 と は 多 く の 情 報 (性 格・歴 史) が 結 び つ く が、 一 瞬 の 出 会 い に は そ う し た 情 報 が な く、 純 粋 な 「印 象」 だ け が 残 る。
</li>
<li>
  <strong>未 完 結 の 物 語</strong> ─ 何 を し て い た 人 か、 ど こ か ら 来 て ど こ へ 行 く の か、 何 も 分 か ら な い。 そ の 「謎」 が 心 に 引 っ か か る。
</li>
<li>
  <strong>一 瞬 の 真 剣 さ</strong> ─ 一 心 に 仕 事 を す る 姿、 黙 々 と 歩 く 姿 ─ そ こ に 「人 が 生 き て い る」 と い う 純 粋 な 事 実 が 凝 縮 さ れ る。
</li>
<li>
  <strong>自 分 と の 重 な り</strong> ─ そ の 人 を 見 て、 私 も ま た 「あ あ、 私 も こ う し て 一 人 で 生 き て い る の だ」 と 感 じ る 瞬 間 が あ る。
</li>
</ol>

<h3>君 自 身 の 「忘 れ え ぬ 人」 は</h3>

<p>
ち ょ っ と 思 い 出 し て み よ う。
</p>

<ul>
<li>電 車 の 中 で、 一 心 に 本 を 読 ん で い た お じ い さ ん</li>
<li>夕 方 の 公 園 で、 一 人 で ジ ョ ギ ン グ を し て い た 女 性</li>
<li>遠 い 旅 先 で、 道 を 教 え て く れ た 親 切 な 人</li>
<li>祭 り の 中 で、 す れ 違 い ざ ま に 笑 顔 を 向 け た 知 ら な い 子 ど も</li>
</ul>

<p>
こ う し た 「す れ 違 っ た だ け の 人 」 が、 君 の 心 に も、 い つ ま で も 残 っ て い る か も し れ な い。<br>
そ れ こ そ が、 君 自 身 の 「忘 れ え ぬ 人 々」 で あ る。
</p>

<h3>「忘 れ え ぬ 人 々」 と「忘 れ て は な ら ぬ 人 々」</h3>

<p>
作 中、 大 津 は 興 味 深 い こ と を 言 う:<br>
「<strong>『忘 れ て は な ら ぬ 人』 と 『忘 れ え ぬ 人』 は 違 う</strong>」 と。
</p>

<ul>
<li><strong>忘 れ て は な ら ぬ 人</strong> ─ 親・恩 師・恩 人 な ど、「忘 れ る と 申 し 訳 が 立 た な い」 人</li>
<li><strong>忘 れ え ぬ 人</strong> ─ 何 の 義 理 も な い の に、 ど う し て も 心 に 残 る 人</li>
</ul>

<p>
前 者 は <strong>道 徳 的 な 義 務</strong> か ら 覚 え て お く 人 で あ る。<br>
後 者 は、 <strong>自 然 に、 ど う し て も 心 が 引 っ か か る</strong> 人 で あ る。
</p>

<p>
独 歩 が 描 き た か っ た の は、 こ の 後 者 ─ 「心 が 勝 手 に 覚 え て い る 人」 と い う 不 思 議 な 存 在 で あ っ た。
</p>

<h3>明 治 31 年、 国 木 田 独 歩 27 歳</h3>

<p>
こ の 短 編 が 書 か れ た の は、 明 治 31 年 (1898) 4 月。 独 歩 は ま だ 27 歳 で あ っ た。<br>
同 じ 年 に、 「武 蔵 野」 (本 サ イ ト kokugo6 で 既 習) も 発 表 し て い る。
</p>

<p>
独 歩 は こ の 後 も 「春 の 鳥」(本 サ イ ト chugaku1 で 既 習)、「牛 肉 と 馬 鈴 薯」 な ど 多 く の 名 短 編 を 残 す が、 37 歳 と い う 若 さ で 結 核 で 早 世 し た。
</p>

<p>
独 歩 が 短 い 生 涯 で 描 い た の は、 ど の 作 品 も <strong>「日 常 の 中 に あ る 静 か な 真 実」</strong> で あ っ た。<br>
名 も な い 雑 木 林、 名 も 知 ら ぬ 少 年、 名 も 知 ら ぬ す れ 違 っ た 人 ─ そ う し た 「目 立 た な い も の」 を、 真 摯 に 見 つ め 続 け た。
</p>

<h3>全 文 を 読 ん で み よ う</h3>

<p>
「忘 れ え ぬ 人 々」 の 全 文 は、 青 空 文 庫 で 無 料 で 読 め る。<br>
一 度 通 読 し て、 大 津 が 出 会 っ た 「忘 れ え ぬ 人 々」 を、 一 人 ず つ 心 に 留 め て み て ほ し い。
</p>

<p>
そ し て、 君 自 身 の 「忘 れ え ぬ 人」 も、 そ っ と 心 に 思 い 浮 か べ て み て ほ し い。
</p>
