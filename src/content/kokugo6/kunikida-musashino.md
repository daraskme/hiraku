<h3>本文 (『武蔵野』 国木田独歩・冒頭・現代仮名遣い)</h3>

<blockquote>
<p>
「武 蔵 野 (む さ し の) の 俤 (お も か げ) は 今 わ ず か に 入 間 郡 (い る ま ご お り) に 残 れ り」 と、 自 分 は、 友 人 と 武 蔵 野 を 語 る と き よ く こ の 句 を 引 く。
</p>

<p>
そ し て こ の 句 を 引 く と、 自 分 の 心 に、 何 と も 言 え ぬ 美 し い 落 葉 林 の 風 景 が 浮 か ん で く る。
</p>

<p>
昔 の 武 蔵 野 は、 萱 (か や) の 原 で あ っ た と い う。 今 は 一 面 の 林 で あ る。 林 と い う も 、 ま だ 林 に な ら ぬ も の も あ る、 一 度 切 ら れ て 後 に 生 え て き た 木 が、 ま だ 五 六 尺 か ら 一 丈 ぐ ら い に し か な ら ぬ も の も 多 い。
</p>

<p>
こ う し て 主 に 楢 (な ら) の 類 で、 冬 は 葉 が す っ か り 落 ち て し ま う 林 で あ る。 自 分 は こ の 落 葉 林 が 好 き で あ る。 と い う よ り、 自 分 が 武 蔵 野 で 最 も 好 き な も の は 落 葉 林 で あ る、 と 言 っ た ほ う が 適 切 で あ ろ う。
</p>

<p>
昔 の 詩 人 ・ 歌 人 ・ 文 人 は、 こ の 落 葉 林 の 美 を 知 ら な か っ た。 彼 ら は、 桜 を 詠 い、 紅 葉 を 詠 い、 月 を 詠 ん だ。 だ が、 こ の 雑 木 林 (ぞ う き ば や し) の、 冬 の 落 葉 し た 寂 し い 美 を、 誰 も 詠 ま な か っ た。
</p>

<p>
私 は こ こ で、 こ の 落 葉 林 の 美 を 語 ろ う と 思 う。
</p>
</blockquote>

<h3>解説 ─ 「ま だ 誰 も 美 し い と 言 っ て い な い」 風 景 を 発 見 す る</h3>

<p>
国 木 田 独 歩 の 「武 蔵 野」 が 書 か れ た 明 治 31 年 (1898)、 日 本 文 学 史 上 で 革 命 的 な こ と が 起 こ っ た。
</p>

<p>
そ れ ま で の 日 本 文 学 は、 自 然 を 描 く 時、 「<strong>典 型 的 な 美 し い も の</strong>」 だ け を 描 い た:
</p>

<ul>
<li>桜 (春)</li>
<li>月 (秋)</li>
<li>紅 葉 (秋)</li>
<li>富 士 山 (す べ て の 季 節)</li>
<li>松 (永 遠)</li>
</ul>

<p>
こ れ ら は、 千 年 以 上、 和 歌 ・ 俳 句 ・ 漢 詩 で 何 万 回 と 詠 ま れ て き た。<br>
で は、 「<strong>雑 木 林</strong>」 は ど う か。 武 蔵 野 の 楢 や 椚 の 林 は、 ま だ 誰 も 「美 し い」 と 言 っ て い な か っ た。<br>
そ れ は 「典 型 的 な 美」 で は な か っ た か ら だ。
</p>

<p>
だ が 国 木 田 独 歩 は、 こ う 宣 言 し た: <strong>「私 は、 こ の 雑 木 林 の 美 を 語 ろ う」</strong> と。
</p>

<h3>な ぜ こ れ が 「画 期 的」 か</h3>

<p>
明 治 30 年 代 ─ 日 本 は 急 速 に 西 洋 化 し て い た。<br>
そ の 中 で、 多 く の 知 識 人 は 「西 洋 の よ う な 美」 を 求 め、 自 分 の 周 り の 自 然 を 見 失 い つ つ あ っ た。
</p>

<p>
独 歩 は そ の 流 れ に 抗 (あ ら が) い、 「<strong>身 の 回 り の、 名 も な い 雑 木 林 こ そ、 美 し い</strong>」 と 言 っ た。<br>
こ れ は、 言 わ ば 「自 分 の 足 元 を 見 つ め 直 す」 宣 言 で あ っ た。
</p>

<p>
そ し て こ の 発 見 が、 後 の 日 本 文 学 に 巨 大 な 影 響 を 与 え た:
</p>

<ul>
<li>志 賀 直 哉 ─ 「日 常 の 中 の 美」 を 描 く 自 然 主 義</li>
<li>島 崎 藤 村 ─ 信 州 の 田 園 風 景 を 文 学 化</li>
<li>柳 田 国 男 ─ 民 俗 学 と し て の 故 郷 研 究</li>
<li>武 者 小 路 実 篤 ─ 「新 し き 村」 の 試 み</li>
</ul>

<p>
「<strong>足 元 を 見 つ め る</strong>」 と い う こ の 視 線 が、 明 治・大 正・昭 和 を 通 じ て、 日 本 文 学 の 一 つ の 大 き な 流 れ と な っ た の で あ る。
</p>

<h3>1898 年 の 武 蔵 野、 2026 年 の 武 蔵 野</h3>

<p>
独 歩 が 書 い た 1898 年 の 武 蔵 野 は、 今 の 東 京 都 武 蔵 野 市・三 鷹 市・小 金 井 市・国 立 市・西 多 摩 地 区、 そ し て 埼 玉 県 入 間 市・所 沢 市 を 含 む 広 大 な 地 域 だ っ た。<br>
そ の ほ と ん ど が、 雑 木 林 と 草 原 で あ っ た。
</p>

<p>
そ の 後 130 年。 武 蔵 野 は、 ど う 変 わ っ た か。
</p>

<p>
独 歩 が 書 い た 風 景 は、 今 は <strong>ご く 一 部 の 公 園 と 緑 地 に し か 残 ら な い</strong>。<br>
だ が、 そ れ で も、 井 の 頭 公 園 や 小 金 井 公 園 を 歩 く と、 独 歩 の 描 い た 「俤 (お も か げ)」 を、 か す か に 感 じ る こ と が で き る。
</p>

<h3>あ な た 自 身 の 「武 蔵 野」 を 探 し て</h3>

<p>
国 木 田 独 歩 が 教 え て く れ た こ と ─ そ れ は、 「<strong>誰 も 美 し い と 言 っ て い な い 風 景 を、 自 分 が 美 し い と 思 う こ と</strong>」 の 大 切 さ。
</p>

<p>
あ な た の 家 の 近 く に も、 ど こ か に <strong>「雑 木 林 の よ う な 場 所」</strong> が あ る は ず:
</p>
<ul>
<li>近 所 の 神 社 の 裏 山</li>
<li>通 学 路 の 並 木 道</li>
<li>マ ン シ ョ ン と マ ン シ ョ ン の 間 の 小 さ な 緑 地</li>
<li>住 宅 街 の 中 の 古 い 庭 の あ る 家</li>
</ul>

<p>
そ こ を、 一 度 立 ち 止 ま っ て、 じ っ く り 見 て み よ う。<br>
ま だ 誰 も 文 章 に し て い な い、 君 だ け の 「武 蔵 野」 を、 発 見 で き る か も し れ な い。
</p>

<p>
独 歩 が 130 年 前 に 教 え て く れ た の は、 「美 は ど こ に で も あ る ─ そ れ を 見 つ け る 眼 が あ る か ど う か」 と い う こ と な の だ。
</p>
