<h3>『道 程』 ─ 全 文 (9 行)</h3>

<blockquote>
<p>
僕 の 前 に 道 は な い<br>
僕 の 後 ろ に 道 は 出 来 る<br>
あ あ、 自 然 よ<br>
父 よ<br>
僕 を 一 人 立 ち に さ せ た 広 大 な 父 よ<br>
僕 か ら 目 を 離 さ な い で 守 る 事 を せ よ<br>
常 に 父 の 気 魄 (き は く) を 僕 に 充 (み た) せ よ<br>
こ の 遠 い 道 程 の た め<br>
こ の 遠 い 道 程 の た め
</p>
</blockquote>

<h3>1914 年 ─ 高 村 31 歳、 結 婚 の 年</h3>

<p>
大 正 3 年 (1914)、 高 村 光 太 郎 31 歳。 こ の 年、 二 つ の 大 き な 出 来 事 が あ り ま し た ─
</p>

<ul>
  <li>10 月 ─ 第 一 詩 集『道 程』 出 版</li>
  <li>12 月 22 日 ─ 長 沼 智 恵 子 (28 歳) と 結 婚</li>
</ul>

<p>
詩『道 程』 は、 こ の 結 婚 と 新 し い 人 生 へ の 旅 立 ち の 決 意 が 込 め ら れ た 詩 で す。
</p>

<h3>「僕 の 前 に 道 は な い」 ─ 創 造 の 詩</h3>

<p>
詩 の 第 一 行 ─ 「僕 の 前 に 道 は な い」。 こ れ は、 既 に あ る 道 (前 例 ・ 伝 統 ・ 模 範) を 歩 む の で
は な く、 自 分 で 新 し い 道 を 創 る 決 意 を 表 し て い ま す。
</p>

<p>
高 村 光 太 郎 は、 父 が 仏 師 (仏 像 彫 刻 家) の 高 村 光 雲 で し た。 普 通 な ら、 父 の 仏 師 を 受 け 継 ぐ 道 が
用 意 さ れ て い ま し た。 し か し 光 太 郎 は、 米 国・英 国・仏 国 へ 留 学 し て 西 洋 美 術 を 学 び、 詩 人 ・
彫 刻 家 と し て 父 と は 違 う 道 を 選 び ま し た。
</p>

<p>
「父 の 道」 で な く、 「自 分 の 道」 を 創 る ─ こ の 決 意 が 詩 に 込 め ら れ て い ま す。
</p>

<h3>「僕 の 後 ろ に 道 は 出 来 る」 ─ 歩 ん だ 跡 が 道 に な る</h3>

<p>
第 二 行 ─ 「僕 の 後 ろ に 道 は 出 来 る」。 前 に 道 は な い が、 自 分 が 歩 い た 跡 が 道 に な る、 と い う 意 味。
</p>

<p>
道 は 最 初 か ら あ る も の で は な く、 誰 か が 歩 い て、 そ の 跡 が 道 に な る ─ こ れ は、 新 し い 何 か を
始 め る 全 て の 人 へ の メ ッ セ ー ジ で す。 君 が 創 ろ う と し て い る 何 か (将 来 の 夢 ・ 新 し い 趣 味 ・
学 校 で の チ ャ レ ン ジ) は、 君 が 歩 む こ と で 初 め て「道」 に な る の で す。
</p>

<h3>「父 よ」 ─ 大 自 然 へ の 呼 び か け</h3>

<p>
詩 の 後 半、 「父 よ」 と い う 言 葉 が 繰 り 返 さ れ ま す。 こ こ で の 「父」 は、 高 村 光 雲 の こ と で は
あ り ま せ ん。 「自 然 よ / 父 よ」 と あ る か ら、 大 自 然 ・ 宇 宙 を「父」 と 呼 ん で い る の で す。
</p>

<p>
「広 大 な 父 よ、 僕 か ら 目 を 離 さ な い で 守 る 事 を せ よ」 ─ 新 し い 道 を 歩 む 不 安 の 中、 自 然 ・
宇 宙 の 大 き な 力 に 守 ら れ た い と 願 う 心。
</p>

<h3>智 恵 子 と の 純 愛 ─ こ の 詩 の 背 景</h3>

<p>
詩『道 程』 を 書 い た 時 期、 高 村 は 長 沼 智 恵 子 と 出 会 い、 愛 し 合 い、 結 婚 し ま し た。 智 恵 子 は
日 本 女 子 大 校 卒 の 画 家。 雑 誌『青 鞜』 (せ い と う ・ 平 塚 ら い て う ら が 創 刊 し た 女 性 運 動 雑 誌)
の 表 紙 を 描 い た 才 媛 で し た。
</p>

<p>
二 人 の 純 愛 は、 後 に 高 村 が 出 版 し た 詩 集『智 恵 子 抄』(1941) に 結 晶 し ま す。 し か し 智 恵 子 は
精 神 を 患 い、 1938 年 53 歳 で 死 去。 そ の 悲 し み の 中 で 高 村 は『智 恵 子 抄』 を 編 ん だ の で す。
</p>

<p>
『道 程』 の「広 い 道 程 の た め」 ─ こ の 詩 で 願 っ た 二 人 の 旅 路 は、 24 年 で 終 わ っ て し ま い ま し た。
</p>

<h3>結 び ─ 小 6 か ら 中 学 へ、 君 の 道 程</h3>

<p>
小 6 の 君 は、 ま も な く 中 学 校 に 進 学 し ま す。 中 学 校 ・ 高 校 ・ 大 学 ・ 就 職 ・ 結 婚 ・ 子 育 て ─
こ れ か ら 続 く 君 の 道 程 (人 生 の 旅 路) は、 ま だ 誰 も 歩 い て い な い 道 で す。
</p>

<p>
高 村 光 太 郎 が 110 年 前 に 詠 ん だ 詩 の 言 葉 を 心 に 留 め ま し ょ う ─
</p>

<blockquote>
<p>
「僕 の 前 に 道 は な い<br>
僕 の 後 ろ に 道 は 出 来 る」
</p>
</blockquote>

<p>
君 自 身 が 歩 む こ と で、 道 が で き る。 こ の 言 葉 を 持 っ て、 中 学 へ の 一 歩 を 踏 み 出 し て く だ さ い。
</p>
