<h3>1771 年 ・ 千 住 骨 ヶ 原 ─ 一 つ の 解 剖 が 歴 史 を 変 え る</h3>

<p>
明 和 8 年 (1771) 3 月 4 日、 江 戸 ・ 千 住 骨 ヶ 原 (現 ・ 南 千 住 駅 近 く)。 罪 人 の 死 体 が 一 体、
解 剖 の た め に 用 意 さ れ た。
</p>

<p>
集 ま っ た の は 江 戸 の 蘭 学 医 ─ 若 狭 小 浜 藩 医 ・ 杉 田 玄 白 (39 歳)、 中 津 藩 医 ・ 前 野 良 沢 (49 歳)、
中 川 淳 庵、 桂 川 甫 周 ら。 そ れ ぞ れ の 手 元 に は、 当 時 最 新 の 西 洋 解 剖 学 書 ─ ド イ ツ 人 ク ル ム ス の
『解 剖 学 表』 の オ ラ ン ダ 語 訳 版 が あ っ た。
</p>

<p>
解 剖 が 進 む に つ れ、 杉 田 ら は 衝 撃 を 受 け た。 ク ル ム ス の 図 と、 実 際 の 人 体 が 全 て 一 致 す る。
心 臓 の 形、 肺 の 分 葉、 内 臓 の 配 置 ─ 全 て が 西 洋 解 剖 学 書 通 り だ っ た。
</p>

<h3>「日 本 の 医 学 は 何 を し て き た の か」</h3>

<p>
当 時、 日 本 の 医 学 は 中 国 古 典 (五 行 説・経 絡 説 等) に 基 づ い て お り、 内 臓 の 位 置 や 形 が
西 洋 医 学 と 異 な っ て い た。 解 剖 を 直 接 見 た 杉 田 ら は、 衝 撃 で 帰 路 言 葉 が 出 な か っ た と い う。
</p>

<p>
そ の 夜、 杉 田 と 前 野 は 決 意 し た ─ 「ク ル ム ス を 日 本 語 に 訳 し て、 日 本 の 医 学 を 西 洋 並 み に
し よ う」。 翌 日 か ら、 三 年 越 し の 翻 訳 作 業 が 始 ま っ た。
</p>

<h3>翻 訳 の 苦 闘 ─ 辞 書 な し で、 一 行 ず つ</h3>

<p>
し か し、 翻 訳 は 困 難 を 極 め た。 当 時 の 日 本 に は ─
</p>

<ul>
  <li>オ ラ ン ダ 語 ─ 日 本 語 辞 書 が な か っ た</li>
  <li>医 学・解 剖 学 用 語 の 日 本 語 が 存 在 し な か っ た</li>
  <li>前 野 良 沢 だ け が 多 少 の オ ラ ン ダ 語 を 読 め た</li>
</ul>

<p>
彼 ら は「一 行 を 一 月 か け て 訳 す」 こ と も あ っ た と い う。 単 語 が わ か ら な い 時 は、 ク ル ム ス
の 図 と 文 を 何 度 も 照 合 し、 文 脈 か ら 推 測。 オ ラ ン ダ 商 館 員 か ら 単 語 を 聞 き 出 す こ と も あ っ た。
</p>

<h3>「神 経」 と い う 言 葉 を 創 る</h3>

<p>
翻 訳 で 一 番 困 難 だ っ た の は、 「日 本 語 に な い 概 念」 を 訳 す こ と だ っ た。
</p>

<p>
オ ラ ン ダ 語「zenuw (ゼ ー ヌ ウ)」 ─ 体 中 に 走 る 細 い 線 維 で、 感 覚 を 伝 え る も の。 中 国 医 学 の
「経 絡 (け い ら く)」 と は 違 う。 ど う 訳 す か?
</p>

<p>
杉 田・前 野 は 議 論 を 重 ね、 こ う し た ─ 「神 (し ん) ─ 知 覚・霊 妙 な 働 き」 + 「経 (け い)
─ 経 路・道 筋」 = 「神 経 (し ん け い)」。 こ の 新 し い 言 葉 が、 今 も 医 学 用 語 と し て 使 わ れ て い る。
</p>

<p>
同 様 に、 「動 脈」「静 脈」「軟 骨」「軸」 等 ─ 多 く の 解 剖 学 用 語 が、『解 体 新 書』 翻 訳 過 程 で
創 ら れ た。 こ れ ら 全 て が、 現 代 医 学 ・ 生 物 学 で 使 わ れ て い る。
</p>

<h3>『解 体 新 書』 出 版 ─ 安 永 3 年 (1774)</h3>

<p>
3 年 半 の 苦 闘 の 末、 安 永 3 年 (1774)、『解 体 新 書』 全 5 巻 が 出 版 さ れ た。 杉 田 玄 白 41 歳、
前 野 良 沢 51 歳。
</p>

<p>
実 は、 翻 訳 を 主 導 し た の は 前 野 良 沢 だ っ た が、 「ま だ 完 璧 で な い」 と 出 版 時 に 名 前 を
載 せ る こ と を 拒 否。 杉 田 玄 白 が「天 真 楼 (て ん し ん ろ う)」 の 名 で 出 版 し た。
</p>

<h3>後 世 へ の 影 響 ─ 蘭 学 → 洋 学 → 福 沢 諭 吉</h3>

<p>
『解 体 新 書』 は、 多 く の 蘭 学 者 を 育 て た。 杉 田 の 弟 子 ・ 大 槻 玄 沢 (お お つ き げ ん た く) が
『蘭 学 階 梯』 を 著 し、 大 槻 の 弟 子 ・ 緒 方 洪 庵 (お が た こ う あ ん) が 大 坂 の 適 塾 を 開 い た。
</p>

<p>
そ し て 適 塾 で 学 ん だ 一 人 が、 福 沢 諭 吉 (本 サ イ ト chugakukomin3/fukuzawa-gakumon 既 出)。
慶 應 義 塾 を 設 立 し、 近 代 日 本 の 礎 を 築 い た。 杉 田・前 野 か ら 福 沢 へ の 系 譜 ─ こ れ が 蘭 学
→ 洋 学 → 近 代 日 本 学 問 の 流 れ だ。
</p>

<h3>結 び ─ 翻 訳 と は、 文 化 を 創 る こ と</h3>

<p>
「翻 訳」 と は、 単 に 言 葉 を 変 換 す る こ と で は な い。 新 し い 概 念 を 受 け 入 れ、 新 し い 言 葉 を
創 り、 自 分 達 の 文 化 に 組 み 込 む こ と。 杉 田・前 野 の 苦 闘 が 教 え る の は こ の 真 理 だ。
</p>

<p>
現 代 の 君 が 教 科 書 で「神 経」 を 学 ぶ 時、 250 年 前 の 二 人 の 蘭 学 者 が 一 つ の 言 葉 を 創 り 出 す
た め に 費 や し た 何 ヶ 月 を、 思 い 出 し て み よ う。
</p>
