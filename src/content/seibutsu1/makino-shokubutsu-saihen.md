<h3>中 学 → 高 校 ─ 牧 野 を 再 読</h3>

<p>
中 学 で、 君 は 牧 野 富 太 郎 (1862–1957) を 学 ん だ (本 サ イ ト chugakurika1/makino-shokubutsu)。<br>
「日 本 植 物 学 の 父」、 ヤ マ ト グ サ の 発 見、 約 1500 種 の 命 名、 NHK 朝 ド ラ 「ら ん ま ん」 の モ デ ル。
</p>

<p>
こ こ で は、 同 じ 牧 野 を ─ <strong>高 校 生 物 「分 類 学」</strong> の 視 点 で、 一 歩 深 め て 学 ぼ う。
</p>

<h3>リ ン ネ の 二 名 法 ─ 牧 野 が 使 っ た「世 界 共 通 語」</h3>

<p>
牧 野 が 約 1500 種 の 植 物 に 与 え た「学 名」 ─ そ の 仕 組 み を 確 認 し よ う。
</p>

<p>
学 名 は、 ス ウ ェ ー デ ン の 植 物 学 者 <strong>カ ー ル・リ ン ネ</strong> (Carl Linnaeus, 1707–1778) が 1753 年 に 確 立 し た「<strong>二 名 法</strong>」 で 書 か れ る。
</p>

<blockquote>
学 名 = 属 名 (大 き い 分 類) + 種 小 名 (具 体 的 な 種)
</blockquote>

<p>
例:
</p>

<ul>
<li>ヤ マ ザ ク ラ ─ <em>Cerasus jamasakura</em></li>
<li>ソ メ イ ヨ シ ノ ─ <em>Cerasus × yedoensis</em></li>
<li>ヤ マ ト グ サ ─ <em>Theligonum japonicum</em></li>
<li>ヒ ト ─ <em>Homo sapiens</em></li>
</ul>

<p>
こ の 2 語 の ラ テ ン 語 で、 世 界 中 の 生 物 学 者 が 同 じ 植 物 を 指 す こ と が で き る。<br>
日 本 人 で も、 ア フ リ カ 人 で も、 ブ ラ ジ ル 人 で も、 「<em>Cerasus jamasakura</em>」 と 書 け ば 同 じ「ヤ マ ザ ク ラ」 を 指 す。<br>
こ れ こ そ が、 牧 野 が 1500 種 に 与 え た「世 界 共 通 言 語」 で あ る。
</p>

生 物 分 類 学 で は、 種 を こ う 階 層 化 す る:

<p>

</p>

<h3>ヤ マ ト グ サ 1889 ─ 改 め て の 意 味</h3>

<p>
1889 年、 22 歳 の 牧 野 が ヤ マ ト グ サ (<em>Theligonum japonicum</em>) を 命 名・発 表 し た。<br>
こ の 業 績 を、 分 類 学 の 視 点 で 再 評 価 す る と:
</p>

<ul>
<li>「<em>japonicum</em>」 ─ ラ テ ン 語 で「日 本 の」 の 意。 種 小 名 に 国 名 を 入 れ た こ と で「こ の 種 が 日 本 で 発 見 さ れ た」 こ と を 国 際 的 に 記 録</li>
<li>「<em>Theligonum</em>」 ─ こ の 属 自 体 は 既 知 だ っ た が、 そ こ に「日 本 産」 の 新 種 を 加 え た こ と が 国 際 学 界 で 認 め ら れ た</li>
<li>こ の 発 表 を 国 際 植 物 学 雑 誌 で 行 う こ と で、 「日 本 人 も 国 際 分 類 学 に 貢 献 で き る」 こ と を 証 明 し た</li>
</ul>

<p>
こ れ ま で 日 本 の 植 物 は、 シ ー ボ ル ト (オ ラ ン ダ) や ツ ン ベ ル グ (ス ウ ェ ー デ ン) な ど 外 国 人 に よ っ て 命 名 さ れ て き た。<br>
牧 野 の ヤ マ ト グ サ 発 表 は、 「日 本 人 自 身 が 自 国 の 自 然 を 命 名 す る」 時 代 の 出 発 点 だ っ た。
</p>

<h3>国 際 植 物 命 名 規 約 と 現 代</h3>

<p>
リ ン ネ の 二 名 法 を 国 際 的 に 制 度 化 し た「<strong>国 際 植 物 命 名 規 約</strong>」(現 在 は「藻 類・菌 類・植 物 国 際 命 名 規 約」 ICN) は、 1867 年 か ら 整 備 さ れ、 数 年 ご と に 改 訂 さ れ て い る。
</p>

<p>
こ の 規 約 に よ り、 全 世 界 の 約 <strong>40 万 種</strong> の 植 物 が 統 一 さ れ た 学 名 で 整 理 さ れ て い る。<br>
そ し て、 毎 年 約 2000 種 の 新 種 が 命 名 さ れ 続 け て い る。
</p>

<p>
牧 野 が 1889 年 に 始 め た「日 本 人 が 自 国 植 物 を 命 名 す る」 仕 事 は、 今 も 多 く の 日 本 人 植 物 学 者 が 受 け 継 い で い る。
</p>

<h3>牧 野 の 標 本 と デ ジ タ ル 化</h3>

<p>
牧 野 が 残 し た 約 <strong>40 万 点</strong> の 標 本 は、 現 在、 高 知 県 立 牧 野 植 物 園・東 京 大 学 総 合 研 究 博 物 館・国 立 科 学 博 物 館 な ど に 所 蔵 さ れ て い る。
</p>

<p>
そ し て 21 世 紀 に 入 り、 こ れ ら の 標 本 が <strong>デ ジ タ ル 化</strong> さ れ、 世 界 中 の 研 究 者 が オ ン ラ イ ン で ア ク セ ス で き る よ う に な っ た (例: GBIF、 高 知 県 立 牧 野 植 物 園 デ ジ タ ル ア ー カ イ ブ)。
</p>

<p>
160 年 前 に 牧 野 が 山 で 採 集 し た 一 枚 の 押 し 葉 が、 今 や ク リ ッ ク 一 つ で 世 界 中 か ら 見 ら れ る。<br>
こ れ も、 リ ン ネ の 二 名 法 と い う「世 界 共 通 語」 の 上 に 築 か れ た、 21 世 紀 の 学 問 の 形 で あ る。
</p>

<h3>君 自 身 の 学 名 探 し</h3>

<p>
家 の 近 く の 公 園 で、 一 種 の 植 物 を 選 び、 学 名 を 調 べ て み よ う。
</p>

<p>
ス マ ホ の 植 物 同 定 ア プ リ (Pl@ntNet、 PictureThis 等) を 使 え ば、 写 真 か ら 即 座 に 種 と 学 名 が 判 別 で き る。
</p>

<p>
そ し て そ の 学 名 を Google Scholar で 検 索 す る と ─ 世 界 中 の 研 究 者 が そ の 植 物 に つ い て 何 を 書 い て き た か が、 一 気 に 開 け る。
</p>

<p>
こ れ こ そ が、 リ ン ネ と 牧 野 が 君 に 残 し た「国 際 学 問 共 同 体」 へ の パ ス ポ ー ト で あ る。
</p>
