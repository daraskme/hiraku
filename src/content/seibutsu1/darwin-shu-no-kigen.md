<h3>1859 年 11 月 24 日、 ロ ン ド ン</h3>

<p>
チャールズ・ダーウィン (Charles Darwin, 1809–1882) の『種 の 起 源』が、 1859 年 11 月 24 日、 ロ ン ド ン で 出 版 さ れ た。
</p>

<p>
初 版 1250 部 ─ <strong>即 日 完 売</strong>。 こ の 1 冊 の 本 が、 19 世 紀 の 生 物 学・思 想・宗 教 観 を 根 底 か ら 揺 さ ぶ る こ と に な る。
</p>

<h3>ダ ー ウ ィ ン の 50 年 の 旅</h3>

<p>
ダ ー ウ ィ ン の『種 の 起 源』 ま で の 道 の り は 長 か っ た:
</p>

<ul>
<li>1809 ─ 英 国 シ ュ ル ー ズ ベ リ ー に 生 ま れ る</li>
<li>1831 ─ ケ ン ブ リ ッ ジ 大 学 卒。 22 歳 で <strong>ビ ー グ ル 号</strong> に 博 物 学 者 と し て 乗 船</li>
<li>1831–1836 ─ 5 年 間 の 世 界 一 周 航 海。 南 米・ガ ラ パ ゴ ス・オ ー ス ト ラ リ ア</li>
<li>1837 ─ 帰 国 後、「進 化」 に つ い て の 「秘 密 の ノ ー ト」 を 書 き 始 め る</li>
<li>1858 ─ ウ ォ レ ス か ら の 手 紙 で 自 然 選 択 の 同 時 発 見 を 知 り、 急 ぐ</li>
<li><strong>1859 ─『種 の 起 源』 出 版</strong></li>
<li>1871 ─『人 間 の 由 来』 で 人 類 進 化 を 論 ず</li>
<li>1882 ─ ウ エ ス ト ミ ン ス タ ー 寺 院 に 国 葬</li>
</ul>

<h3>本 文 抜 粋 (試 訳)</h3>

<blockquote>
<p>
ビ ー グ ル 号 で 南 米 を 航 海 中、 私 は い く つ も の 事 実 に 強 く 心 を 動 か さ れ た。<br>
パ タ ゴ ニ ア の 砂 漠 で 発 見 さ れ る 絶 滅 し た 大 型 哺 乳 類 の 化 石。 そ の 地 で 今 も 生 き て い る ア ル マ ジ ロ と の 不 思 議 な 形 態 的 類 似。
</p>

<p>
そ し て、 ガ ラ パ ゴ ス 諸 島 で 観 察 し た 動 植 物 の 多 様 性。<br>
同 じ 島 群 の 中 で、 島 か ら 島 へ 渡 る だ け で、 フ ィ ン チ (鳥) の 嘴 の 形 が 違 う。 種 子 を 食 べ る フ ィ ン チ、 昆 虫 を 食 べ る フ ィ ン チ、 サ ボ テ ン の 花 を 食 べ る フ ィ ン チ ─ そ れ ぞ れ が 食 物 に 適 応 し た 嘴 を 持 っ て い る。
</p>

<p>
こ れ ら の 事 実 を 説 明 す る た め に は、 一 つ の 仮 説 し か な い と 私 は 考 え た:
</p>

<blockquote>
「<strong>す べ て の 生 物 種 は、 共 通 の 祖 先 か ら、 長 い 時 間 を か け て 変 化 し て き た</strong>」
</blockquote>

<p>
そ し て そ の 「変 化 の メ カ ニ ズ ム」 と し て、 私 は「<strong>自 然 選 択</strong>」(natural selection) を 提 唱 す る。
</p>

<p>
自 然 選 択 の 原 理 は 単 純 で あ る:
</p>

<ol>
<li>同 じ 種 の 個 体 で も、 わ ず か な 違 い (変 異) が あ る</li>
<li>そ の 違 い の う ち、 環 境 に 適 し た も の を 持 つ 個 体 が、 よ り 多 く の 子 孫 を 残 す</li>
<li>こ れ が 何 千、 何 万 世 代 と 繰 り 返 さ れ る と、 種 全 体 が 徐 々 に 変 化 し て い く</li>
<li>こ う し て、 共 通 祖 先 か ら 多 様 な 種 が 分 化 し て い く</li>
</ol>

<p>
こ れ が、 私 が「<strong>進 化 (evolution)</strong>」 と 呼 ぶ も の の 本 質 で あ る。
</p>
</blockquote>

<h3>な ぜ こ の 本 は 衝 撃 だ っ た か</h3>

<p>
19 世 紀 中 葉 の 西 洋 で は、 「<strong>創 造 論</strong>」 が 一 般 的 だ っ た:
</p>

<blockquote>
「神 は 6000 年 前 に、 各 種 の 動 植 物 を 個 別 に、 完 璧 な 形 で 創 造 し た」
</blockquote>

<p>
ダ ー ウ ィ ン の 進 化 論 は、 こ の 創 造 論 を 真 っ 向 か ら 否 定 す る:
</p>

<ul>
<li>種 は「個 別 に 創 造 さ れ た」 の で は な く、 共 通 祖 先 か ら 分 化 し た</li>
<li>地 球 の 歴 史 は 6000 年 で は な く、 何 億 年 と い う 単 位</li>
<li>人 間 も 他 の 動 物 と 同 じ「進 化 の 産 物」 で あ り、 サ ル と 共 通 祖 先 を 持 つ</li>
</ul>

<p>
こ れ は、 当 時 の 宗 教 観・人 間 観 を 根 底 か ら 覆 す も の だ っ た。<br>
大 論 争 が 巻 き 起 こ り、 ダ ー ウ ィ ン と そ の 支 持 者 は 教 会・保 守 派 か ら 激 し く 攻 撃 さ れ た。
</p>

<h3>ウ ォ レ ス と の 「同 時 発 見」</h3>

<p>
ダ ー ウ ィ ン は 自 然 選 択 の ア イ デ ア を 1837 年 頃 か ら 持 っ て い た が、 20 年 以 上 公 表 を た め ら っ て い た。<br>
教 会 と の 衝 突 を 恐 れ、 ま た 自 分 の 理 論 を 完 璧 に 仕 上 げ た か っ た か ら だ。
</p>

<p>
だ が 1858 年、 ダ ー ウ ィ ン は 衝 撃 を 受 け る。<br>
東 南 ア ジ ア で 博 物 学 研 究 を し て い た <strong>ウ ォ レ ス</strong> (Alfred Russel Wallace, 1823–1913) か ら 手 紙 が 届 い た。 彼 が <strong>独 立 に 同 じ 「自 然 選 択」 説 に 到 達 し て い た</strong> こ と が わ か っ た。
</p>

<p>
そ の 1858 年 7 月、 リ ン ネ 学 会 で ダ ー ウ ィ ン と ウ ォ レ ス の 論 文 が 共 同 発 表 さ れ た。<br>
そ し て 翌 年 1859 年 11 月、 ダ ー ウ ィ ン は『種 の 起 源』 を 出 版。 急 い で の 出 版 だ っ た。
</p>

<h3>進 化 論 の そ の 後</h3>

<p>
1859 年 か ら 165 年。 進 化 論 は、 現 代 生 物 学 の <strong>根 本 原 理</strong> と な っ た。
</p>

<ul>
<li>1865 ─ メ ン デ ル の 遺 伝 法 則 (進 化 の メ カ ニ ズ ム の 鍵)</li>
<li>1953 ─ ワ ト ソ ン と ク リ ッ ク の DNA 二 重 螺 旋 構 造 発 見</li>
<li>1970s 以 降 ─ 分 子 生 物 学 ・ 遺 伝 子 解 析 が 進 化 論 を 完 全 に 裏 付 け る</li>
<li>2003 ─ ヒ ト ゲ ノ ム 計 画 完 了</li>
</ul>

<p>
こ れ ら の 発 見 全 て が、 ダ ー ウ ィ ン の 仮 説 ─ 「<strong>す べ て の 生 物 は 共 通 祖 先 か ら 進 化 し た</strong>」 ─ を 裏 付 け て い る。
</p>

<h3>「進 化」 と い う 視 点</h3>

<p>
ダ ー ウ ィ ン が 残 し た 最 大 の 贈 り 物 は、 単 な る「進 化 論」 と い う 仮 説 で は な い。<br>
そ れ は ─「<strong>す べ て の 生 命 は つ な が っ て い る</strong>」 と い う 視 点 で あ る。
</p>

<p>
今 君 の 周 り に い る 全 て の 生 物 ─ 蝶 も、 雀 も、 桜 も、 君 自 身 も ─ 同 じ 一 つ の 祖 先 (35 億 年 前 の 原 始 細 胞) か ら、 何 十 億 年 と い う 時 間 を か け て 分 化 し て き た 「親 戚」 で あ る。
</p>

<p>
こ の 「<strong>生 命 の 大 き な つ な が り</strong>」 を 知 る こ と が、 ダ ー ウ ィ ン が 私 た ち に 残 し た 一 番 大 切 な こ と で あ る。
</p>
