## 野 口 英 世 ─ ア フ リ カ で 倒 れ た 微 生 物 学 者

### 1. 福 島 の 火 傷 の 少 年

1876 年 11 月、 福 島 県 翁 島 村 (現 在 の 猪 苗 代 町) の 貧 し い 農 家 で、 一 人 の 男 児 が 生 ま れ た。 名 を「清 作 (せ い さ く)」 と い う。 (後 に「英 世」 と 改 名 す る。)

清 作 が 1 歳 半 の と き、 母 シ カ が 留 守 中 に 囲 炉 裏 (い ろ り) に 転 落 し、 左 手 を 大 火 傷 し た。 当 時 の 田 舎 で は 手 術 を 受 け ら れ ず、 清 作 の 左 手 は 不 自 由 な ま ま 育 っ た。

し か し、 こ の 火 傷 の こ と が、 後 に 清 作 の 人 生 を 決 定 す る こ と に な っ た。

***

### 2. 渡 部 鼎 と 医 学 へ の 道

15 歳 の と き、 高 等 小 学 校 を 卒 業 し た 清 作 は、 偶 然「ア メ リ カ 帰 り の 医 師」 と 出 会 う。 そ の 名 を 渡 部 鼎 (わ た な べ か な え)。 渡 部 は、 清 作 の 左 手 を 診 て、 手 術 を 行 い、 さ ら に 清 作 の 才 能 と 努 力 を 認 め て、 自 ら 主 宰 す る 会 陽 医 院 で 書 生 と し て 育 て た。

清 作 は 努 力 を 重 ね、 1896 年 上 京 し て 医 術 開 業 試 験 (現 在 の 医 師 国 家 試 験) に 合 格。 20 歳 で 医 師 と な っ た。

***

### 3. 1900 年 ─ ア メ リ カ 渡 航

医 師 と な っ た 清 作 (こ の 頃 か ら「英 世」 と 名 乗 る) は、 北 里 柴 三 郎 (1853-1931、 細 菌 学 者、 日 本 細 菌 学 の 父) の 伝 染 病 研 究 所 で 助 手 と な る が、 国 内 で の 機 会 が 少 な い と 感 じ、 1900 年 ア メ リ カ 渡 航 を 決 意。

ペ ン シ ル ベ ニ ア 大 学 の フ レ ク シ ャ ー (S. Flexner) 教 授 の 下 で 研 究 を 開 始、 1904 年 か ら 新 設 さ れ た ロ ッ ク フ ェ ラ ー 医 学 研 究 所 (ニ ュ ー ヨ ー ク) に 移 籍 し、 後 に 主 任 研 究 員 と な る。

***

### 4. 1911-1913 年 ─ 梅 毒 ス ピ ロ ヘ ー タ の 業 績

野 口 英 世 の 最 初 の 国 際 的 業 績 は、 梅 毒 ス ピ ロ ヘ ー タ (Treponema pallidum、 梅 毒 の 病 原 体) の 研 究 で あ っ た。

1911 年、 野 口 は ロ ッ ク フ ェ ラ ー 研 究 所 で 梅 毒 ス ピ ロ ヘ ー タ の 純 粋 培 養 (= 他 の 微 生 物 が 混 ざ ら な い 形 で 研 究 室 で 培 養 す る こ と) に 成 功。 こ れ は 1905 年 ド イ ツ で の ス ピ ロ ヘ ー タ 発 見 以 来 の 重 要 課 題 で あ り、 国 際 的 評 価 を 受 け た。

1913 年、 野 口 は さ ら に「進 行 麻 痺 (神 経 梅 毒 の 最 終 段 階) の 患 者 脳 か ら 梅 毒 ス ピ ロ ヘ ー タ そ の も の を 検 出」 す る こ と に 成 功。 こ れ で「進 行 麻 痺 = 梅 毒 末 期」 と い う 19 世 紀 以 来 の 推 測 を 病 原 学 的 に 確 認 し た。

こ の 二 つ の 業 績 で、 野 口 は ノ ー ベ ル 医 学 生 理 学 賞 候 補 と し て 国 際 的 に 認 知 さ れ る よ う に な っ た。

***

### 5. 1918 年 以 降 ─ 黄 熱 病 と の 闘 い

野 口 の 次 の 大 き な 課 題 は、 黄 熱 病 (お う ね つ び ょ う、 Yellow Fever) で あ っ た。 ア フ リ カ・中 南 米 の 熱 帯 で 流 行 す る 急 性 ウ イ ル ス 感 染 症 で、 高 熱・黄 疸・出 血 で 死 亡 す る 患 者 が 多 か っ た。

1918 年 か ら、 野 口 は 南 米 エ ク ア ド ル・ペ ル ー・メ キ シ コ 等 で 黄 熱 病 の 病 原 体 を 探 し、 1919 年「黄 熱 病 の 病 原 体 は『Leptospira icteroides』 と い う 細 菌 で あ る」 と 報 告 し た。

し か し、 こ の 報 告 は 後 に 誤 り で あ っ た こ と が 判 明 し た。 1927 年、 ロ ッ ク フ ェ ラ ー 研 究 所 の 同 僚 ス ト ー ク ス ら が、 ア フ リ カ で の 観 察 か ら「黄 熱 病 の 本 当 の 病 原 体 は ウ イ ル ス で あ る」 と 結 論 し た (当 時 は 電 子 顕 微 鏡 が な く ウ イ ル ス を 直 接 観 察 で き な か っ た た め、 「ろ 過 性 病 原 体」 と し て 推 測 さ れ た)。

***

### 6. ア フ リ カ・ガ ー ナ へ

野 口 は 自 ら の 誤 り を 確 認 し、 真 の 病 原 体 を 突 き 止 め る た め に、 1927 年 12 月 ア フ リ カ・ゴ ー ル ド コ ー ス ト (現 ガ ー ナ) の ア ク ラ に 渡 航 し た。 当 時 51 歳。

ア ク ラ の 医 学 研 究 所 で、 野 口 は 必 死 に 黄 熱 病 患 者 か ら の 検 体 を 集 め、 研 究 を 続 け た。 し か し、 彼 自 身 が 黄 熱 病 に 感 染 し た。

1928 年 5 月 21 日、 ア ク ラ で 野 口 英 世 は 死 亡。 享 年 51 歳。

死 の 直 前、 野 口 は こ う 呟 い た と 伝 え ら れ る。

「I do not understand. (私 に は 理 解 で き な い)」

「私 に は な ぜ 自 分 が 倒 れ た の か 理 解 で き な い」 と も、 「私 の 仮 説 が 間 違 っ て い た こ と が 理 解 で き な い」 と も 解 釈 さ れ る、 苦 し い 言 葉 で あ っ た。

***

### 7. 結 び ─ 千 円 札 の 顔

野 口 英 世 の 業 績 の 一 部 (黄 熱 病 細 菌 説) は 後 に 否 定 さ れ た。 し か し、 梅 毒 研 究 で の 業 績 は 確 か な も の で あ り、 そ し て 何 よ り、 福 島 の 火 傷 の 貧 し い 農 家 の 子 が、 ア メ リ カ で 世 界 的 研 究 者 と な り、 ア フ リ カ で 自 ら の 仮 説 を 確 認 す る た め に 命 を 落 と し た と い う 生 涯 が、 日 本 人 が 世 界 の 医 学 共 同 体 に 加 わ っ た 半 世 紀 そ の も の を 象 徴 し た。

2004 年 11 月 か ら 2024 年 7 月 ま で、 日 本 の 1000 円 札 の 肖 像 は 野 口 英 世 で あ っ た。 大 人 も 子 ど も も、 毎 日 そ の 顔 を 見 て き た。 2024 年 7 月 か ら、 紙 幣 の 肖 像 は 北 里 柴 三 郎 ─ 野 口 の 師 ─ に 変 わ っ た。

師 弟 二 代 が、 続 け て 日 本 の 紙 幣 を 飾 っ た。
