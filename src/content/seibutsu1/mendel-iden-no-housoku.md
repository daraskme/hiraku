<h3>ブ ル ノ の 修 道 院 の 庭</h3>

<p>
1856 年、 オ ー ス ト リ ア の ブ リ ュ ン (現 チ ェ コ・ブ ル ノ) の 修 道 院。<br>
34 歳 の 修 道 士 <strong>グ レ ゴ ー ル・メ ン デ ル</strong> (Gregor Mendel, 1822–1884) が、 静 か な 実 験 を 始 め た。
</p>

<p>
実 験 材 料 は ─ エ ン ド ウ 豆。<br>
メ ン デ ル は、 修 道 院 の 庭 で、 8 年 間 (1856–1863) に わ た り、 約 <strong>28,000 株</strong> の エ ン ド ウ 豆 を 育 て、 交 配 し、 結 果 を 統 計 的 に 記 録 し た。
</p>

<p>
こ の 地 道 な 仕 事 か ら、 「遺 伝 の 法 則」 が 解 き 明 か さ れ た。
</p>

<h3>な ぜ エ ン ド ウ 豆 か</h3>

<p>
メ ン デ ル が エ ン ド ウ 豆 を 選 ん だ の は、 偶 然 で は な か っ た:
</p>

<ul>
<li><strong>明 確 な 対 立 形 質</strong> ─ 丸 vs し わ、 黄 色 vs 緑、 高 い vs 低 い な ど、 ど ち ら か は っ き り 分 か れ る 形 質 が 多 い</li>
<li><strong>1 年 で 何 世 代 も</strong> ─ 春 蒔 い て 夏 に 収 穫、 す ぐ 次 の 世 代 を 観 察 で き る</li>
<li><strong>自 家 受 粉 と 他 家 受 粉 の 両 方 が 可 能</strong> ─ 純 系 (じ ゅ ん け い) を 維 持 し つ つ、 意 図 的 交 配 が で き る</li>
<li><strong>大 量 に 育 て や す い</strong> ─ 統 計 的 検 証 が 可 能</li>
</ul>

<h3>三 つ の 法 則</h3>

<h4>① 優 性 の 法 則 (第 一 法 則)</h4>

<p>
丸 い エ ン ド ウ (純 系) と し わ の エ ン ド ウ (純 系) を 交 配 し て み る:
</p>

<blockquote>
P (親 世 代): 丸 ×  し わ → F1 (子 世 代): <strong>全 部 丸</strong>
</blockquote>

<p>
し わ が 「消 え た」 よ う に 見 え る。<br>
こ れ を、 メ ン デ ル は こ う 解 釈 し た: 「丸 い」 形 質 が 「優 性」 で、 「し わ」 が 「劣 性」 だ と。
</p>

<h4>② 分 離 の 法 則 (第 二 法 則)</h4>

<p>
F1 (全 部 丸) の 子 ど も 同 士 (自 家 受 粉) を 交 配 す る:
</p>

<blockquote>
F1 × F1 → F2: <strong>丸 と し わ が 3:1 の 比 率 で 出 現</strong>
</blockquote>

<p>
F1 で 消 え た は ず の「し わ」 が、 F2 で 復 活 す る。 し か も、 ぴ っ た り 3:1 の 統 計 比 率 で。
</p>

<p>
メ ン デ ル は こ う 推 論 し た:
</p>
<ul>
<li>F1 の エ ン ド ウ は、 内 部 に 「丸 の 因 子」 と 「し わ の 因 子」 の 両 方 を 持 つ</li>
<li>配 偶 子 (花 粉・胚 珠) を 作 る と き、 こ の 二 つ の 因 子 が 「分 離」 し て 別 々 の 配 偶 子 に 入 る</li>
<li>こ れ が F2 で 3:1 を 生 む</li>
</ul>

<h4>③ 独 立 の 法 則 (第 三 法 則)</h4>

<p>
二 つ の 形 質 を 同 時 に 追 う:
</p>

<blockquote>
丸 ・ 黄 × し わ ・ 緑 → F1: 全 部 丸 ・ 黄 → F2: <strong>9 : 3 : 3 : 1 の 比 率</strong>
</blockquote>

<p>
こ の「9:3:3:1」 は、「丸 と し わ」「黄 と 緑」 が 独 立 に 3:1 の 法 則 に 従 う こ と か ら 導 か れ る (3:1 × 3:1 = 9:3:3:1)。
</p>

<h3>科 学 の 方 法 と し て の 美 し さ</h3>

<p>
メ ン デ ル の 仕 事 の 美 し さ は ─
</p>

<ol>
<li><strong>具 体 的 な 実 験</strong> (エ ン ド ウ 豆 を 育 て る)</li>
<li><strong>大 量 の デ ー タ</strong> (28,000 株)</li>
<li><strong>統 計 的 解 析</strong> (3:1、 9:3:3:1 の 比 率 発 見)</li>
<li><strong>抽 象 的 仮 説</strong> (「遺 伝 因 子」 と い う 不 可 視 の 存 在 を 推 論)</li>
</ol>

<p>
こ の 流 れ は、 現 代 の 科 学 研 究 の 教 科 書 的 な モ デ ル と な っ て い る。
</p>

<h3>1865 年 ─ 注 目 さ れ な い 論 文</h3>

<p>
1865 年 2 月、 メ ン デ ル は ブ リ ュ ン 自 然 科 学 会 で 自 分 の 結 果 を 発 表 し、 翌 年『植 物 雑 種 に 関 す る 実 験』 と し て 論 文 出 版。
</p>

<p>
だ が ─ 当 時 の 学 界 は、 こ の 論 文 を <strong>ほ と ん ど 注 目 し な か っ た</strong>。<br>
理 由 は い く つ か 考 え ら れ る:
</p>

<ul>
<li>メ ン デ ル が 「無 名 の 修 道 士」 だ っ た こ と</li>
<li>論 文 が 地 方 の 雑 誌 に 載 っ た こ と</li>
<li>当 時、 ダ ー ウ ィ ン の 進 化 論 (1859) が 話 題 を 独 占 し て い た こ と</li>
<li>数 学 的・統 計 的 ア プ ロ ー チ が 当 時 の 生 物 学 で 異 質 だ っ た こ と</li>
</ul>

<p>
メ ン デ ル は そ の 後、 修 道 院 長 と な り、 行 政 業 務 に 追 わ れ、 研 究 を 続 け ら れ な か っ た。<br>
1884 年、 自 分 の 仕 事 が 認 め ら れ る こ と な く、 62 歳 で 静 か に 没 し た。
</p>

<h3>1900 年 ─ 35 年 後 の 「再 発 見」</h3>

<p>
1900 年 ─ メ ン デ ル 没 後 16 年 ─ 三 人 の 別 々 の 研 究 者 が、 独 立 に 同 じ 法 則 に 到 達 し た:
</p>

<ul>
<li><strong>ド・フ リ ー ス</strong> (オ ラ ン ダ、 ユ ー ゴ・ド・フ リ ー ス)</li>
<li><strong>コ レ ン ス</strong> (ド イ ツ、 カ ー ル・コ レ ン ス)</li>
<li><strong>チ ェ ル マ ク</strong> (オ ー ス ト リ ア、 エ リ ッ ヒ ・ チ ェ ル マ ク)</li>
</ul>

<p>
三 人 と も、 先 行 研 究 を 調 べ て い て ─ <strong>メ ン デ ル の 35 年 前 の 論 文</strong> を 発 見 し、 全 て の 法 則 が す で に そ こ に あ っ た こ と を 知 っ た。
</p>

<p>
こ れ が「<strong>メ ン デ ル の 再 発 見</strong>」 と 呼 ば れ、 20 世 紀 遺 伝 学 の 出 発 点 と な っ た。
</p>

<h3>メ ン デ ル → DNA → ゲ ノ ム へ</h3>

<p>
メ ン デ ル の「遺 伝 因 子」 が、 後 の 科 学 史 で 何 と 対 応 す る か を 整 理:
</p>

<ul>
<li>1865 ─ メ ン デ ル「遺 伝 因 子」 (factor)</li>
<li>1909 ─ ヨ ハ ン セ ン「遺 伝 子」 (gene) と い う 言 葉 を 提 唱</li>
<li>1953 ─ ワ ト ソ ン と ク リ ッ ク、 DNA の 二 重 螺 旋 構 造 を 発 見</li>
<li>1953 以 降 ─ 遺 伝 子 が DNA の 一 部 で あ る こ と が 確 立</li>
<li>2003 ─ ヒ ト ゲ ノ ム 計 画 完 了 (人 間 の 全 遺 伝 情 報 解 読)</li>
</ul>

<p>
メ ン デ ル が「不 可 視 の 因 子」 と し て 推 論 し た も の は、 90 年 後 に「分 子 (DNA)」 と し て 可 視 化 さ れ、 140 年 後 に「全 ゲ ノ ム」 と し て 完 全 に 解 読 さ れ た。
</p>

<h3>君 自 身 の 中 の メ ン デ ル の 法 則</h3>

<p>
今 君 自 身 の 体 に、 メ ン デ ル の 三 法 則 は 静 か に 働 い て い る。
</p>

<ul>
<li>君 の 目 の 色、 髪 の 色、 耳 の 形、 え く ぼ ─ こ れ ら 全 て、 親 か ら 受 け 継 い だ 遺 伝 子 (DNA) の 表 現</li>
<li>君 の 兄 弟 と 似 て い る 部 分 と 違 う 部 分 ─ 分 離 の 法 則 が 働 い た 結 果</li>
<li>君 が 持 つ「ま だ 現 れ て い な い 形 質」 (劣 性 遺 伝) ─ 将 来 君 の 子 ど も で 現 れ る か も し れ な い</li>
</ul>

<p>
160 年 前 の ブ ル ノ の 修 道 院 の 庭 で、 メ ン デ ル が エ ン ド ウ 豆 を 数 え な が ら 発 見 し た 三 法 則 が ─ 今 の 君 自 身 の 中 に、 生 き て い る の だ。
</p>
