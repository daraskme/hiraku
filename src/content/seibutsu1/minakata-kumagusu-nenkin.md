## 南 方 熊 楠 ─ 粘 菌 と 民 俗 学 を つ な げ た 知 の 巨 人

### 1. 紀 州 田 辺 の 顕 微 鏡 の 前 で

1900 年 (明 治 33 年)、 14 年 間 の ロ ン ド ン 滞 在 を 終 え て 33 歳 の 南 方 熊 楠 が 帰 国 し た と き、 彼 が 選 ん だ 居 場 所 は 紀 州 田 辺 (現 在 の 和 歌 山 県 田 辺 市) の 田 舎 で あ っ た。

東 京 で も、 京 都 で も な く。 大 英 博 物 館 で 14 年 を 過 ご し、 雑 誌『Nature』 に 50 編 以 上 を 投 稿 し て き た 知 識 人 が、 紀 州 の 山 林 に 隠 棲 し た。

そ の 山 林 こ そ が、 彼 の「研 究 所」 で あ っ た。

毎 日、 南 方 は 山 に 入 り、 朽 木 の 表 面、 落 葉 の 裏 側、 湿 っ た 土 の 中 か ら、 「粘 菌 (ね ん き ん)」 を 採 集 し て き た。 そ し て、 自 宅 の 書 斎 で 顕 微 鏡 を 覗 き 続 け た。

***

### 2. 粘 菌 と い う 不 思 議 な 生 物

粘 菌 と は、 単 細 胞 で あ り な が ら、 必 要 な と き に は 集 合 し て 多 細 胞 の「変 形 体」 を 形 成 す る 不 思 議 な 生 物 で あ る。 動 物 で も、 植 物 で も、 菌 類 (キ ノ コ・カ ビ) で も な い。 「な ん で も な い」 生 物 で あ る。

朽 木 の 上 で、 黄 色 や ピ ン ク の ゼ リ ー の よ う な 粘 菌 が、 ゆ っ く り と 這 う よ う に 形 を 変 え な が ら 動 く。 餌 (バ ク テ リ ア) を 見 つ け れ ば、 そ こ に 集 合 し て 食 べ る。 環 境 が 悪 化 す れ ば、 子 実 体 と 呼 ば れ る キ ノ コ の よ う な 形 を 作 り、 胞 子 を 飛 ば し て 増 え る。

南 方 は 紀 州 の 山 林 で、 こ の 不 思 議 な 生 物 を 何 千 種 も 採 集・分 類 し、 国 内 外 の 研 究 者 と 文 通 し て、 新 種「ミ ナ カ タ ホ コ リ」 な ど を 発 見 し た。

***

### 3. 同 時 に 民 俗 学 を 著 述

し か し、 南 方 の 仕 事 は 粘 菌 だ け に は と ど ま ら な か っ た。

彼 は 同 時 期 に『十 二 支 考』(1914-1923) を 雑 誌『太 陽』 に 連 載 し、 世 界 中 の 民 俗・神 話・宗 教 に お け る「十 二 支 (鼠・牛・虎・…)」 の 比 較 研 究 を 行 っ た。 ま た、 風 邪 の 民 間 療 法、 死 体 の 処 理、 火 葬 と 土 葬 の 違 い、 性 と 婚 姻 の 習 俗 ─ 古 今 東 西 の あ ら ゆ る「人 間 の 暮 ら し」 を 比 較 し、 集 大 成 し た。

不 思 議 な こ と は、 南 方 が「粘 菌 を 観 察 す る 視 線」 と「民 俗 を 研 究 す る 視 線」 が、 彼 の 中 で は 全 く 同 じ も の で あ っ た こ と で あ る。 「な ぜ 紀 州 の 山 に は こ れ ほ ど の 多 様 性 が あ る か」「な ぜ 各 地 に 同 じ 物 語 が あ る か」 ─ 二 つ の 問 い は、 彼 の 中 で 一 つ で あ っ た。

***

### 4. 南 方 マ ン ダ ラ ─ 物・心・事 の 縁 起

1903 年、 南 方 は 友 人 の 真 言 宗 僧 侶・土 宜 法 龍 へ の 書 簡 で、 一 つ の 不 思 議 な 図 を 描 い た。 後 世「南 方 マ ン ダ ラ」 と 呼 ば れ る 思 考 図 で あ る。

そ こ に は「物 (も の、 物 質)」「心 (こ こ ろ、 認 識)」「事 (こ と、 出 来 事)」 と い う 三 つ の 領 域 が 縁 起 (= 仏 教 の 因 果) で 結 ば れ て い た。 西 洋 哲 学 の「主 観 と 客 観」 と い う 二 元 論 を 超 え て、 「物 と 心 と 事」 の 三 元 的 関 係 で 世 界 を 捉 え る 試 み で あ っ た。

粘 菌 と 民 俗 学 が 一 つ に な る 視 線 ─ そ れ が、 南 方 マ ン ダ ラ で あ っ た。

***

### 5. 神 社 の 鎮 守 の 森 を 守 る 闘 い

1906 年、 政 府 が「一 町 村 一 社」 の 方 針 で 全 国 の 神 社 を 統 合・廃 止 す る「神 社 合 祀 」 を 進 め 始 め た と き、 南 方 は 紀 州 で 反 対 運 動 を 展 開 し た。

「神 社 を 廃 す と、 そ の 鎮 守 の 森 も 失 わ れ る。 鎮 守 の 森 に は、 紀 州 独 自 の 多 様 な 生 物 が 生 き て い る。 神 社 と は、 生 物 多 様 性 を 守 る 装 置 で あ る」

こ れ は 1906 年 の 主 張 で あ る。 「環 境 保 護」「生 物 多 様 性」 と い う 言 葉 が ま だ な か っ た 時 代 に、 そ の 概 念 を 明 確 に 述 べ て い た。 1910 年 に は 自 ら 田 辺 警 察 署 に 拘 引 さ れ る 騒 動 も あ っ た が、 紀 州 で の 神 社 統 合 は 大 幅 に 縮 小 さ れ、 多 く の 鎮 守 の 森 が 守 ら れ た。

***

### 6. 在 野 の 巨 人

1929 年 6 月、 昭 和 天 皇 が 紀 州 田 辺 を 行 幸 し、 行 在 所 で 南 方 は 粘 菌 の 進 講 を 行 い、 110 種 の 粘 菌 標 本 を 桐 箱 で 献 上 し た。 「在 野 の 学 者 が 天 皇 に 進 講 す る」 と い う 異 例 の 出 来 事 で、 日 本 の 学 問 史 に 一 つ の 象 徴 と な っ た。

1941 年 12 月、 南 方 熊 楠 は 紀 州 田 辺 で 没。 享 年 74 歳。

そ の 生 涯 は、 規 格 外 で あ っ た。

- 14 ヶ 国 語 を 操 り、
- 大 英 博 物 館 で 14 年 を 過 ご し、
- 紀 州 の 田 舎 で 顕 微 鏡 と 著 述 を 続 け、
- 粘 菌 と 民 俗 学 を 一 つ の 視 野 に 収 め、
- 環 境 保 護 運 動 の 先 駆 者 と な り、
- 在 野 の ま ま、 天 皇 に 進 講 し た。

「学 ぶ」 と は、 大 学 に 属 す る こ と で も、 都 会 に 住 む こ と で も、 一 つ の 専 門 に 閉 じ こ も る こ と で も な い。 ─ 南 方 熊 楠 の 生 涯 は、 そ う 私 た ち に 語 り か け て く る。

***

(注：本 文 は、 南 方 熊 楠 の『十 二 支 考』 ・ 書 簡 集 ・ 進 講 録 等 の 趣 旨 を、 中 学・高 校 の 生 物 教 材 と し て 編 集 部 が 再 構 成 し た 紹 介 文 で あ る。 原 著 は『南 方 熊 楠 全 集』 (平 凡 社) 等 で 閲 覧 で き る。)
