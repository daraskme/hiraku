glossary:
  - term: 与謝野晶子 (よさの あきこ)
    meaning: 1878–1942。大阪府堺市生まれ。本名 鳳 志やう (ほう しょう)。明治37年に第一歌集『みだれ髪』を刊行し、近代日本の女性詩人として最高峰となった。夫は詩人の与謝野鉄幹 (1873–1935)。12 人の子を産み、平和論・女性解放論でも活躍した。
  - term: 君死にたまふことなかれ
    meaning: 「君よ、死んではいけない」。「君」は晶子の実の弟・鳳籌三郎 (ほう ちゅうざぶろう)。日露戦争 (1904–1905) で陸軍第二軍に従軍し、旅順攻囲戦に参加していた。副題に「旅順口包囲軍の中に在る弟宗七を歎きて」とある (「宗七」は通称)。
  - term: 旅順 (りょじゅん)

  - term: 鳳籌三郎 (ほう ちゅうざぶろう)
    meaning: 与謝野晶子の実の弟。本詩発表時、日露戦争に陸軍兵士として従軍中。詩の力もあってか無事に帰国した。
  - term: 明星 (みょうじょう)
    meaning: 与謝野鉄幹・晶子を中心とした浪漫主義詩歌雑誌 (1900–1908)。北原白秋・石川啄木・木下杢太郎など、多くの近代詩人を輩出した。本詩は明治37年9月号に掲載された。
  - term: 大みこゝろ (おおみこころ)
    meaning: 天皇のお心。「天皇のお心は戦を望んでいない」と詩は主張する。
  - term: けものの道 (けものの みち)
    meaning: 「獣 (けもの) のする所業」「人としての道理に反すること」。「人を殺せと教えることが、けものの道だろうか」と詩は問う。
  - term: 旧家 (きうか / きゅうか)
    meaning: 古くから続く家柄。「あきびとの家」(商人の家) は、武家ではないこと、戦に出ねばならない家柄ではないことを示す。
  - term: 大阪堺 (おおさか さかい)
    meaning: 大阪の堺市。晶子の実家「鳳家」は堺の老舗の和菓子店「駿河屋」だった。晶子は鳳家の三女として生まれた。
  - term: 暖簾 (のれん)
    meaning: 商家の入口にかかる布。家業の象徴。「親のなさけは増さりしも」と詩は、戦に出る息子を残された母の悲しみを描く。
  - term: 反戦詩 (はんせん し)
    meaning: 戦争に反対する詩。本作は近代日本文学最初の本格的反戦詩とされ、現在の高校国語教科書にもよく載っている。

<h3>現代語訳</h3>

<p>
ああ 弟 よ、私 は 君 の こと を 泣 い て いる。<br>
君 よ、死 ん で は い け な い。<br>
末 っ 子 と し て 生 ま れ た 君 だ か ら、両親 の 愛情 は 一 層 深 か っ た は ず。<br>
で も、両親 は 君 に 刃 を 握 ら せ て、「人 を 殺 せ」 と 教 え た だ ろ う か。<br>
「人 を 殺 し て、自分 も 死 ね」 と 言 う ため に、24 歳 ま で 育 て た わ け で は な い だ ろ う。
</p>

<p>
堺 の 街 の 商人 と し て、由緒 あ る 旧 家 を 誇 る 主 (あるじ) と し て、<br>
親 の 名 を 継 ぐ 君 だ か ら ─ 死 ん で は な ら な い。<br>
旅順 の 要塞 が 落 ち よ う と、落 ち な か ろ う と、何 の 関係 が あ ろ う か。<br>
君 は 知 ら な い だ ろ う が、商人 の 家 の 掟 (おきて) に は、戦 に 行 か ね ば な ら な い と は 書 い て な い。
</p>

<p>
君 よ、死 ん で は な ら な い。<br>
天皇 (すめらみこと) は、戦 に 自 ら 出 ら れ る こ と は な い。<br>
互 い に 人 の 血 を 流 し、獣 の 道 に 死 ね、と ─<br>
死 ぬ こ と が 人 の 誉 れ だ、 な ど と、<br>
天皇 の お心 が 深 け れ ば 深 い ほ ど、け っ し て お 思 い に は な ら な い は ず だ。
</p>

<p>
ああ 弟 よ、戦 で 死 ん で は な ら な い。<br>
昨年 の 秋、父 を 亡 く さ れ た 母 ぎ み は、<br>
そ の 悲 し み の 中 で、痛 ま し く も、我 が 子 を 召 さ れ、家 を 守 り、<br>
平和 と 聞 か さ れ る 御代 (天皇 の 治 世) の 中 で も、母 の 白髪 は 増 え る ばか り だ。
</p>

<p>
そ し て、ま だ 若 く て 可 憐 な 新妻 を、君 は 忘 れ た か、 そ れ と も 思 い 続 け て い る か。<br>
結婚 し て わ ず か 10 ヶ月 で 別 れ て 戦 地 に 行 っ た、君 の 妻 の 心 を 思 い み よ。<br>
こ の 世 で 頼 れ る の は 君 だ け な の だ。<br>
君 よ、死 ん で は な ら な い。
</p>

<h3>解説 ─ 明治 37 年 9 月、衝撃 の 発表</h3>

<p>
明治 37 年 (1904) 2 月、日 露 戦争 が 始 ま っ た。<br>
ロシア の 旅順 要塞 を 攻略 す る た め、日本 軍 は 第 三 軍 (司令官 乃木 希典) を 派遣 し、激戦 を 繰 り 広 げ た。203 高地 の 戦 い だけ で、日本 軍 約 1 万 5 千 人 が 戦死 し た。
</p>

<p>
そ う し た 戦時 下 の 雰囲気 の 中、9 月、雑誌 『明星』 に 与謝野晶子 の こ の 詩 が 掲載 さ れ た。<br>
そ し て、瞬 く 間 に、文学 界 ─ いや、社会 全体 ─ を 揺 さ ぶ っ た。
</p>

<h3>「天皇 は 戦 に 出 て たまふ な」 ── 反逆 の 一句</h3>

<p>
詩 の 第 3 連 に 出 て く る 一句 ──「<strong>すめらみこと は、戦 に おほみづから は 出 で ま さ ね</strong>」<br>
(天皇 は、戦 に 自 ら 出 ら れ な い)。
</p>

<p>
こ の 表現 は、当時 の 社会 で、極 め て 危 険 な も の だ っ た。<br>
明治 22 年 (1889) の 大日本帝国 憲法 で 「天皇 は 神聖 に し て 侵 す べ か ら ず」 と さ れ、新聞 や 雑誌 で 天皇 に 触 れ る だ け で 不敬罪 に 問 わ れ る 時代 だ っ た か ら で あ る。
</p>

<p>
晶子 は し か し、こ の 表現 で、こ う 言 い た か っ た の だ ─ <br>
「お 心 の 深 い 天皇 で あ れ ば、人々 を 戦 に 出 し て 殺 す こ と を 望 む は ず が な い」 と。<br>
こ れ は、天皇 を 「侮辱」 し た の で は な く、「<strong>戦争 を 推 進 し て い る の は 天皇 で は な い</strong>」 と 訴 え た の だ。
</p>

<p>
だ が こ の 微 妙 な 区別 は、当時 の 多 く の 読者 に は 伝 わ ら な か っ た。
</p>

<h3>大町桂月 の 「乱臣 賊子」 批判 と 晶子 の 反論</h3>

<p>
発表 後 す ぐ、評論 家 の <strong>大町桂月</strong> (おおまち けいげつ、1869–1925) が、雑誌 『太陽』 で 晶子 を 激 し く 批判 し た。
</p>

<blockquote>
「与謝野晶子 の 詩 は、皇室 中心 の 我 が 国 体 を 危 う く す る、乱臣 賊子 の 言 で あ る」 (大町桂月)
</blockquote>

<p>
晶子 は こ の 批判 に 対 し、雑誌 『明星』 で 反論 し た。
</p>

<blockquote>
「歌 は ま こ と の 心 を 歌 う も の。 戦時 で あ ろ う と、ま こ と を 歌 う こ と は、け っ し て 不忠 で は な い」
</blockquote>

<p>
晶子 は 屈 し な か っ た。<br>
そ し て、こ の 詩 と そ の 論争 は、日本 文学 史 上、<strong>「文学 と 国家」</strong> を 巡 る 最大 級 の 事件 と し て、後世 に 記憶 さ れ る こ と と な っ た。
</p>

<h3>5 連 の リフレイン 構造</h3>

<p>

そ し て 最後 の 第 5 連、新妻 へ の 思 い で 終 わ る ── こ こ に 晶子 の 詩 人 と し て の 力量 が 凝縮 さ れ て い る。
</p>

<h3>そ の 後 ─ 弟 は 生 き て 帰 っ た</h3>

<p>
詩 の 力 も あ っ た か、晶子 の 弟・鳳籌三郎 は、日露戦争 を 生 き 延 び て 帰国 し た。<br>
そ し て、堺 の 老舗 駿河屋 (す る が や) を 継 い だ。
</p>

<p>
晶子 は、そ の 後 も 多 く の 詩 を 書 き、12 人 の 子 を 育 て、女性 解放 運動・平和 運動 で も 活躍 し た。<br>
そ し て、太平洋 戦争 の 真 っ た だ 中、昭和 17 年 (1942)、東京 で 64 歳 で 亡 く な っ た。
</p>

<h3>1904 年 か ら 2026 年 へ</h3>

<p>
本詩 が 発表 さ れ て か ら、122 年 が 経 つ。<br>
そ の 間、人類 は 二 度 の 世界 大戦、冷戦、地域 紛争、テロ、そ し て 2020 年代 に 入 っ て も な お 続 く 戦争 を 経験 し て き た。
</p>

<p>

</p>

<p>
