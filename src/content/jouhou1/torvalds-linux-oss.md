## Linux ─ み ん な で 作 る ソ フ ト ウ ェ ア

### 1. 1991 年 8 月 25 日、 一 通 の 投 稿

時 は 1991 年 8 月 25 日。 フ ィ ン ラ ン ド・ヘ ル シ ン キ 大 学 の 21 歳 の 学 生 リ ー ナ ス・ト ー バ ル ズ が、 イ ン タ ー ネ ッ ト 上 の ニ ュ ー ス グ ル ー プ (現 在 の 掲 示 板 の よ う な も の) に、 一 通 の メ ッ セ ー ジ を 投 稿 し た。

「み な さ ん、 こ ん に ち は。 私 は (フ リ ー の) OS を 作 っ て い ま す。 た だ の 趣 味 で、 GNU の よ う に 大 き く て プ ロ な も の に は な り ま せ ん…」

こ の 謙 遜 し た 投 稿 が、 後 に 世 界 を 動 か す こ と に な る OS「Linux」 の 出 発 点 で あ っ た。

***

### 2. OS と は 何 か

そ も そ も「OS (オ ペ レ ー テ ィ ン グ シ ス テ ム、 基 本 ソ フ ト)」 と は 何 か。

コ ン ピ ュ ー タ は、 ハ ー ド ウ ェ ア (CPU・メ モ リ・デ ィ ス ク・画 面 等 の 物 理 的 な 部 品) と、 ア プ リ (ゲ ー ム・ブ ラ ウ ザ・文 書 作 成 等 の ソ フ ト) で 動 い て い る。

し か し、 ア プ リ が 直 接 ハ ー ド ウ ェ ア を 操 作 す る の は 大 変 だ。 そ こ で、 そ の 間 を つ な ぐ「OS」 が 必 要 と な る。 OS は、 「画 面 に 文 字 を 出 す」「フ ァ イ ル を 保 存 す る」「複 数 の ア プ リ を 同 時 に 動 か す」 等 の 共 通 の 仕 組 み を 提 供 す る。

Windows、 macOS、 そ し て Linux ─ こ れ ら は す べ て OS で あ る。

***

### 3. 「一 人 で 作 る」 か「み ん な で 作 る」 か

ト ー バ ル ズ が 取 っ た 道 が、 後 の Linux の 運 命 を 決 め た。 彼 は、 自 分 の OS の「設 計 図 (ソ ー ス コ ー ド)」 を、 イ ン タ ー ネ ッ ト 上 で 全 て 公 開 し た。 そ し て、 「誰 で も 自 由 に 見 て、 改 良 し て、 再 配 布 し て い い」 と し た。

こ れ を「オ ー プ ン ソ ー ス (Open Source)」 と 呼 ぶ。

当 時、 ソ フ ト ウ ェ ア は 企 業 が 秘 密 に し て 売 る「商 品」 で あ る の が 常 識 だ っ た。 設 計 図 は 企 業 秘 密。 し か し ト ー バ ル ズ は、 「世 界 中 の プ ロ グ ラ マ ー が 少 し ず つ 改 良 す れ ば、 一 人 で 作 る よ り 良 い も の が で き る」 と 考 え た。

そ の 結 果 は、 ど う だ っ た か。

***

### 4. 世 界 中 の プ ロ グ ラ マ ー が 集 ま っ た

Linux の 設 計 図 が 公 開 さ れ る と、 世 界 中 の プ ロ グ ラ マ ー が「こ こ を 直 し た」「こ ん な 機 能 を 足 し た」 と、 改 良 を 持 ち 寄 り 始 め た。

10 人、 100 人、 1000 人。 や が て 何 万 人 も の 開 発 者 が、 ボ ラ ン テ ィ ア で、 あ る い は 企 業 の 仕 事 と し て、 Linux の 改 良 に 参 加 す る よ う に な っ た。

「一 人 の 天 才 が 完 成 さ せ た ソ フ ト」 で は な く、 「世 界 中 の 人 が 少 し ず つ 育 て た ソ フ ト」。 こ れ が Linux で あ る。

***

### 5. 30 年 後 ─ 世 界 を 動 か す OS へ

そ し て 30 年 が 過 ぎ た 今、 Linux は ど こ で 動 い て い る か。

- **Web サ ー バ** ─ 世 界 の Web サ イ ト の 大 半 は Linux サ ー バ で 動 く
- **ス ー パ ー コ ン ピ ュ ー タ** ─ 世 界 の 高 速 計 算 機 TOP500 の 全 機 が Linux
- **ス マ ー ト フ ォ ン** ─ Android (世 界 シ ェ ア 約 7 割) は Linux カ ー ネ ル が 基 盤
- **家 電・自 動 車** ─ テ レ ビ・冷 蔵 庫・カ ー ナ ビ 等 の 組 み 込 み シ ス テ ム
- **宇 宙** ─ 国 際 宇 宙 ス テ ー シ ョ ン の PC も Linux

1991 年 に フ ィ ン ラ ン ド の 大 学 生 が「た だ の 趣 味」 と し て 始 め た 小 さ な OS が、 今 や 世 界 数 十 億 台 の 機 器 の 心 臓 部 で 動 い て い る。

***

### 6. Git ─ も う 一 つ の 贈 り 物

ト ー バ ル ズ は 2005 年、 も う 一 つ 世 界 を 変 え る ツ ー ル を 作 っ た。 「Git (ギ ッ ト)」 で あ る。

Git は、 大 勢 の プ ロ グ ラ マ ー が 同 じ ソ フ ト ウ ェ ア を 同 時 に 編 集 し て も 混 乱 し な い よ う、 「誰 が・い つ・ど こ を 変 え た か」 を 全 て 記 録・管 理 す る シ ス テ ム で あ る。 現 在 世 界 中 の プ ロ グ ラ マ ー が 使 い、 「GitHub」 と い う サ ー ビ ス の 基 盤 と も な っ て い る。

実 は、 こ の 学 習 サ イ ト「ひ ら く」 も、 Git で 管 理 さ れ、 GitHub で 公 開 さ れ た オ ー プ ン ソ ー ス で あ る。 君 が 今 読 ん で い る こ の 教 材 も、 ト ー バ ル ズ が 作 っ た 仕 組 み の 上 で 動 い て い る の だ。

***

### 7. 結 び ─ 「み ん な で 作 る」 と い う 思 想

Linux の 物 語 が 教 え て く れ る の は、 「み ん な で 作 る」 と い う 思 想 の 力 で あ る。

一 人 の 力 に は 限 り が あ る。 し か し、 設 計 図 を 公 開 し、 世 界 中 の 人 が 少 し ず つ 力 を 持 ち 寄 れ ば、 一 人 で は 決 し て 作 れ な い 巨 大 で 高 品 質 な も の が で き る。

こ れ は ソ フ ト ウ ェ ア だ け の 話 で は な い。 知 識 (Wikipedia)、 地 図 (OpenStreetMap)、 そ し て 教 材 (こ の「ひ ら く」)── あ ら ゆ る 分 野 で、「み ん な で 作 る」 オ ー プ ン な 営 み が 広 が っ て い る。

1991 年 の 一 通 の 謙 遜 し た 投 稿 が、 そ の 大 き な 流 れ の 一 つ の 源 流 と な っ た。
