<h3>「計 算 と は 何 か」 ─ 1936 年 の 問 い</h3>

<p>
1936 年、 ケ ン ブ リ ッ ジ 大 学 の 24 歳 の 学 生 ・ <strong>ア ラ ン ・ チ ュ ー リ ン グ</strong> (Alan Turing, 1912–1954) が、 一 篇 の 論 文 を 書 い た。
</p>

<p>
タ イ ト ル ─『計 算 可 能 数 に つ い て、 と そ の 決 定 問 題 へ の 応 用』(On Computable Numbers, with an Application to the Entscheidungsproblem)。
</p>

<p>
こ の 論 文 で 彼 は、 一 つ の 想 像 上 の 装 置 を 構 想 し た。<br>
そ れ が ─ <strong>「チ ュ ー リ ン グ 機 械」</strong>(Turing Machine) で あ る。
</p>

<h3>チ ュ ー リ ン グ 機 械 の 仕 組 み</h3>

<p>
チ ュ ー リ ン グ 機 械 は、 驚 く ほ ど 単 純 な 装 置 で あ る:
</p>

<ul>
<li><strong>テ ー プ</strong> ─ 無 限 に 長 い、 マ ス 目 の 紙 テ ー プ。 各 マ ス に「0」 か「1」 を 書 け る</li>
<li><strong>ヘ ッ ド</strong> ─ テ ー プ の 一 つ の マ ス を 読 ん だ り 書 き 換 え た り す る 部 分</li>
<li><strong>状 態</strong> ─ 機 械 の 内 部 の「現 在 の 状 態」 (有 限 の 種 類)</li>
<li><strong>規 則 (プ ロ グ ラ ム)</strong> ─ 「今 の 状 態 ＋ 今 読 ん だ 値 → 何 を 書 く ・ 次 の 状 態 ・ ヘ ッ ド を 左 右 に 動 か す」 の 表</li>
</ul>

<p>
こ の シ ン プ ル な 装 置 が、 <strong>あ ら ゆ る 計 算</strong> ─ 算 数 か ら 微 積 分 か ら 巨 大 デ ー タ 処 理 ま で ─ を 実 行 で き る こ と を、 チ ュ ー リ ン グ は 数 学 的 に 証 明 し た。
</p>

<h3>「想 像 上 の 装 置」 が「現 実 の コ ン ピ ュ ー タ」 に</h3>

<p>
1936 年 当 時、 こ の 装 置 は 純 粋 に「想 像 上 の も の」 だ っ た。<br>
チ ュ ー リ ン グ は 物 理 的 な 機 械 を 作 ろ う と は し て い な か っ た。 「計 算 と は 何 か を 数 学 的 に 定 義 す る」 の が 目 的 だ っ た。
</p>

<p>
だ が ─ 戦 後、 ジ ョ ン ・ フ ォ ン ・ ノ イ マ ン ら が こ の 概 念 を 元 に、 物 理 的 な 電 子 計 算 機 を 設 計 し た。<br>
そ の 設 計 思 想 が、 1940s 末 〜 1950s に「コ ン ピ ュ ー タ」 の 形 で 実 現 し、 現 代 ま で 続 い て い る。
</p>

<p>
今 君 が 持 っ て い る ス マ ホ ─ そ れ は、 1936 年 の チ ュ ー リ ン グ の 想 像 上 の 装 置 を、 現 代 技 術 で 実 現 し た も の だ。
</p>

<h3>第 二 次 大 戦 ─ エ ニ グ マ 解 読</h3>

<p>
1939 年、 第 二 次 大 戦 開 戦。 27 歳 の チ ュ ー リ ン グ は、 イ ギ リ ス 政 府 か ら 招 集 を 受 け、 <strong>ブ レ ッ チ リ ー パ ー ク</strong> (Bletchley Park) に 配 属 さ れ た。
</p>

<p>
そ こ は、 イ ギ リ ス の 秘 密 暗 号 解 読 セ ン タ ー。<br>
目 標 は、 ナ チ ス ・ ド イ ツ が 使 用 す る 暗 号 機 「<strong>エ ニ グ マ</strong>」(Enigma) の 解 読 で あ っ た。
</p>

<p>
エ ニ グ マ は、 1 日 に 数 億 通 り の 設 定 を 持 ち、 「絶 対 に 解 け な い」 と 信 じ ら れ て い た。<br>
だ が チ ュ ー リ ン グ ら は、 「<strong>ボ ン ベ</strong>」(Bombe) と い う 機 械 を 設 計・開 発 し、 エ ニ グ マ を 解 読 す る こ と に 成 功 し た。
</p>

<p>
こ の 解 読 が ─ 戦 争 を <strong>2 ~ 4 年 短 縮 し、 約 1400 万 人 ~ 2100 万 人 の 命 を 救 っ た</strong> と 推 計 さ れ る (歴 史 家 Harry Hinsley)。
</p>

<p>
だ が こ の 業 績 は、 戦 後 数 十 年 間 軍 事 機 密 と さ れ、 チ ュ ー リ ン グ 本 人 も 公 に 称 賛 さ れ る こ と が な か っ た。<br>
彼 の 仕 事 が 一 般 に 知 ら れ る よ う に な っ た の は、 1974 年 以 降 で あ る。
</p>

<h3>1950 年 ─ チ ュ ー リ ン グ テ ス ト</h3>

<p>
戦 後 の 1950 年、 38 歳 の チ ュ ー リ ン グ は、 哲 学 雑 誌『Mind』 に も う 一 つ の 革 命 的 論 文 を 発 表:
</p>

<blockquote>
『計 算 機 械 と 知 性』(Computing Machinery and Intelligence)
</blockquote>

<p>
こ の 論 文 で、 彼 は こ う 問 う た:<br>
「<strong>機 械 は 考 え る こ と が で き る か?</strong>」 (Can machines think?)
</p>

<p>
そ し て、 こ の 抽 象 的 な 問 い を 具 体 的 に 検 証 可 能 な 形 に 変 換 し た:
</p>

<blockquote>
「<strong>人 間 の 審 査 員 が、 隠 れ た 相 手 と 文 字 だ け で 会 話 し、 そ の 相 手 が 機 械 か 人 間 か を 判 別 で き な け れ ば、 そ の 機 械 は 知 能 を 持 つ と み な し て よ い</strong>」
</blockquote>

<p>
こ れ が、 <strong>チ ュ ー リ ン グ テ ス ト</strong> で あ る。<br>
今 日 の AI (ChatGPT、 Claude な ど) の 研 究 ・ 評 価 は、 こ の チ ュ ー リ ン グ テ ス ト か ら 始 ま っ て い る。
</p>

<h3>悲 劇 と 名 誉 回 復</h3>

<p>
だ が、 チ ュ ー リ ン グ の 生 涯 は 悲 劇 で 終 わ る。
</p>

<p>
1952 年、 当 時 の イ ギ リ ス で 違 法 と さ れ て い た「同 性 愛」 で 有 罪 判 決 を 受 け た。<br>
科 学 者 と し て の 地 位 ・ 機 密 取 扱 許 可 を 失 い、 ホ ル モ ン 療 法 (化 学 的 去 勢) を 強 い ら れ た。
</p>

<p>
そ し て 1954 年 6 月 7 日、 自 宅 で 青 酸 カ リ を 含 む リ ン ゴ の そ ば で 死 ん で い る の が 発 見 さ れ た。 41 歳。<br>
自 死 と 公 式 に 認 定 さ れ た (但 し、 別 説 も あ る)。
</p>

<p>
55 年 後 の <strong>2009 年</strong>、 イ ギ リ ス 政 府 (ゴ ー ド ン・ブ ラ ウ ン 首 相) が チ ュ ー リ ン グ へ の 「謝 罪」 を 公 式 表 明。<br>
2013 年、 エ リ ザ ベ ス 女 王 が「王 室 の 恩 赦」 を 与 え、 名 誉 回 復。<br>
2017 年、 同 様 の 罪 で 有 罪 判 決 を 受 け た 全 員 を 一 括 恩 赦 す る「<strong>チ ュ ー リ ン グ 法</strong>」 が 制 定 さ れ た。
</p>

<h3>「コ ン ピ ュ ー タ の 父」 と し て の 不 滅 の 業 績</h3>

<p>
チ ュ ー リ ン グ の 没 後 70 年。 彼 の 業 績 は、 21 世 紀 の 私 た ち の 生 活 を 根 本 的 に 規 定 し て い る:
</p>

<ul>
<li><strong>コ ン ピ ュ ー タ</strong> ─ 1936 年 の チ ュ ー リ ン グ 機 械 が 全 て の コ ン ピ ュ ー タ の 理 論 的 基 礎</li>
<li><strong>ス マ ホ</strong> ─ チ ュ ー リ ン グ 機 械 の 一 つ の 物 理 的 実 現</li>
<li><strong>AI ・ ChatGPT</strong> ─ チ ュ ー リ ン グ テ ス ト の 思 想 か ら 出 発</li>
<li><strong>暗 号 解 読 ・ サ イ バ ー セ キ ュ リ テ ィ</strong> ─ エ ニ グ マ 解 読 か ら の 系 譜</li>
<li><strong>チ ュ ー リ ン グ 賞</strong> ─ 計 算 機 科 学 の ノ ー ベ ル 賞 と 呼 ば れ る 国 際 的 最 高 賞</li>
</ul>

<p>
2021 年 か ら、 イ ギ リ ス の 50 ポ ン ド 紙 幣 の 肖 像 が ─ <strong>ア ラ ン・チ ュ ー リ ン グ</strong> と な っ た。<br>
イ ギ リ ス の 紙 幣 が 偉 人 を 顕 彰 す る 中、 こ の 紙 幣 は チ ュ ー リ ン グ の 業 績 を、 過 去 の 不 名 誉 と と も に、 国 民 が 改 め て 受 け 入 れ た こ と を 示 す。
</p>

<h3>君 の 手 の 中 の チ ュ ー リ ン グ</h3>

<p>
今 君 の 手 の 中 の ス マ ホ。 そ れ は、 1936 年 の チ ュ ー リ ン グ の 想 像 上 の 装 置 を、 現 代 の 半 導 体 技 術 で 実 現 し た も の だ。
