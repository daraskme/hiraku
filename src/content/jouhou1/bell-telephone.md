<h3>1876 年 3 月 10 日 ─ 史 上 初 の 電 話</h3>

<p>
1876 年 3 月 10 日、 米 国 マ サ チ ュ ー セ ッ ツ 州 ボ ス ト ン。 5 番 街 の 賃 貸 住 宅 で、 二 人 の 男 が
歴 史 を 作 っ た 瞬 間 が あ り ま し た。
</p>

<p>
二 階 の 部 屋 に は ア レ ク サ ン ダ ー・グ ラ ハ ム・ベ ル (29 歳)。 隣 の 部 屋 に は 助 手 ト ー マ ス・
ア ウ グ ス タ ス・ワ ト ソ ン (22 歳)。 二 部 屋 を つ な ぐ 電 線 と、 ベ ル が 設 計 し た「電 話 機」 が
あ り ま し た。
</p>

<p>
ベ ル は 自 ら の 装 置 に 向 か っ て 話 し か け ま し た ─
</p>

<blockquote>
<p>
"Mr. Watson, come here. I want to see you."<br>
(ワ ト ソ ン さ ん、 こ ち ら に 来 て く だ さ い。 会 い た い の で す)
</p>
</blockquote>

<p>
隣 室 の ワ ト ソ ン の 装 置 か ら、 ベ ル の 声 が 聞 こ え て き ま し た。 ワ ト ソ ン は 思 わ ず ベ ル の 部 屋 に
駆 け 込 み、 「聞 こ え ま し た!」 と 叫 び ま し た。
</p>

<p>
こ れ が、 史 上 初 め て 電 線 越 し に 人 間 の 声 が 伝 わ っ た 瞬 間 で す。
</p>

<h3>ベ ル の 出 自 ─ 音 声 学 者 の 家 系</h3>

<p>
ア レ ク サ ン ダ ー ・ グ ラ ハ ム ・ ベ ル は、 1847 年 ス コ ッ ト ラ ン ド・エ ジ ン バ ラ 生 ま れ。 父 メ ル ヴ ィ ル・
ベ ル は 音 声 学 ・ 発 声 法 の 研 究 者 で、 聾 唖 者 向 け の「視 話 法 (Visible Speech)」 を 開 発 し ま し た。
</p>

<p>
ベ ル 自 身 も 若 い 頃 か ら、 「音 を ど う 視 覚 化 す る か」 を 研 究。 妻 メ イ ベ ル (Mabel Hubbard) も
聾 唖 者 で し た。 つ ま り、 電 話 の 発 明 の 出 発 点 は、 「聾 唖 者 が 音 を 理 解 で き る 装 置 を 作 る」 と い う
動 機 だ っ た の で す。
</p>

<h3>電 話 の 仕 組 み ─ 音 を 電 気 に、 電 気 を 音 に</h3>

<p>
電 話 の 原 理 は シ ン プ ル で す ─
</p>

<ol>
  <li><strong>送 信 側</strong> ─ 人 の 声 (空 気 の 振 動) が マ イ ク (音 声 ↔ 電 気 変 換 装 置) に 当 た る</li>
  <li>マ イ ク が 音 声 を 電 気 信 号 に 変 換</li>
  <li>電 気 信 号 が 電 線 を 通 っ て 受 信 側 へ</li>
  <li><strong>受 信 側</strong> ─ ス ピ ー カ ー (電 気 ↔ 音 声 変 換 装 置) が 電 気 信 号 を 音 に 戻 す</li>
  <li>人 が 元 の 声 を 聞 く</li>
</ol>

<p>
こ の シ ン プ ル な 原 理 が、 現 代 ス マ ホ ・ Wi-Fi ・ Bluetooth ま で 全 て の 通 信 技 術 の 基 礎 で す。
</p>

<h3>特 許 戦 争 ─ エ リ シ ャ・グ レ イ と の 競 争</h3>

<p>
電 話 の 発 明 は、 ベ ル だ け の 業 績 で は あ り ま せ ん で し た。 同 じ 時 代、 米 国 の エ リ シ ャ・グ レ イ
(Elisha Gray, 1835–1901) も 似 た 装 置 を 開 発 し て い ま し た。
</p>

<p>
1876 年 2 月 14 日、 ベ ル と グ レ イ が 同 じ 日 に 米 国 特 許 庁 に 申 請 を 出 し ま し た。 ベ ル が 数 時 間
早 か っ た た め、 ベ ル に 特 許 が 認 め ら れ ま し た (米 国 特 許 174,465)。 後 に 法 廷 争 い が 続 き ま し た が、
最 終 的 に ベ ル の 特 許 が 維 持 さ れ ま し た。
</p>

<p>
こ の 特 許 は、 「19 世 紀 最 大 の 価 値 を 持 つ 特 許」 と さ れ ま す。
</p>

<h3>Bell Telephone Company ─ 巨 大 通 信 帝 国 の 始 ま り</h3>

<p>
1877 年、 ベ ル と 義 父 ガ ー デ ィ ナ ー・ハ バ ー ド ら が「Bell Telephone Company」 を 設 立。 こ れ が
後 に AT&T (American Telephone and Telegraph) と な り、 20 世 紀 の 米 国 通 信 を 独 占 す る 巨 大 企 業
に な り ま し た。
</p>

<p>
日 本 で は 1890 年 (明 治 23 年)、 東 京 ・ 横 浜 間 で 電 話 サ ー ビ ス が 開 始 さ れ ま し た。 ベ ル 発 明 か ら
わ ず か 14 年 後 で す。
</p>

</p>

<h3>結 び ─ 「離 れ た 人 と 話 す」 の 始 ま り</h3>

<p>
LINE で 友 達 と チ ャ ッ ト す る 時、 家 族 に ビ デ オ 通 話 す る 時 ─ そ の 全 て の 出 発 点 は、 1876 年
ボ ス ト ン の あ の 一 言「Mr. Watson, come here. I want to see you」 で し た。
</p>

<p>
通 信 技 術 を 学 ぶ 時、 こ の 言 葉 を 心 に 留 め て く だ さ い。 君 の ス マ ホ も、 ベ ル の 想 像 力 の 子 孫 な の で す。
</p>
