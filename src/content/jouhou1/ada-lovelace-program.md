## 世 界 最 初 の プ ロ グ ラ マ ー

### エ イ ダ・ラ ブ レ ス、 1843 年

「未 来 の あ る 機 械 ─ そ れ は『解 析 機 関 (Analytical Engine)』 と 呼 ば れ て い る ─ は、 単 に 数 を 計 算 で き る だ け で は な い。 数 で 表 現 で き る あ ら ゆ る も の ─ 音 楽、 文 字、 画 像 ─ を 操 作 で き る だ ろ う。 こ れ は 単 な る 計 算 機 械 で は な く、 思 考 を 自 動 化 す る 機 械 で あ る。」

こ れ を 書 い た の は、 1843 年、 27 歳 の イ ギ リ ス 人 女 性 エ イ ダ・ラ ブ レ ス で あ る。

***

### バ ベ ッ ジ と の 出 会 い

エ イ ダ・ラ ブ レ ス は 1815 年、 詩 人 バ イ ロ ン の 一 人 娘 と し て ロ ン ド ン に 生 ま れ た。 父 は エ イ ダ が 1 歳 の と き 母 と 別 居 し、 二 度 と 会 う こ と は な か っ た。 母 ア ン・イ サ ベ ラ は、 娘 が 父 親 の「狂 気 と 情 熱」 を 受 け 継 ぐ こ と を 恐 れ、 「数 学」 を 教 育 の 中 心 に 据 え た。

そ の お か げ で、 エ イ ダ は 当 時 の 女 性 と し て は 例 外 的 に 高 度 な 数 学 教 育 を 受 け た。 17 歳 の と き、 数 学 者 で あ り 発 明 家 で あ る チ ャ ー ル ズ・バ ベ ッ ジ と 知 り 合 い、 彼 が 設 計 し て い た「階 差 機 関 (Difference Engine)」 を 見 学 す る 機 会 を 得 た。

エ イ ダ は バ ベ ッ ジ の 構 想 に 心 を 奪 わ れ た。

***

### 解 析 機 関 と 注 釈 G

そ の 後、 バ ベ ッ ジ は さ ら に 高 度 な「解 析 機 関 (Analytical Engine)」 を 構 想 し た。 蒸 気 で 動 く、 歯 車 数 千 個 の 機 械 式 汎 用 計 算 機。 そ れ は、 後 の デ ジ タ ル コ ン ピ ュ ー タ と 同 じ 基 本 構 造 ─ 記 憶 装 置・演 算 装 置・制 御 装 置 ─ を 持 つ 構 想 で あ っ た。

1840 年、 ト リ ノ で イ タ リ ア 人 数 学 者 メ ナ ブ レ ア が バ ベ ッ ジ の 解 析 機 関 を イ タ リ ア 語 で 紹 介 す る 論 文 を 書 い た。 1843 年、 エ イ ダ は そ の 論 文 を 英 訳 し、 そ し て 英 訳 の 末 尾 に、 自 ら の「7 つ の 注 釈 (A-G)」 を 加 え た。

特 に「注 釈 G」 で 彼 女 は、 ベ ル ヌ ー イ 数 と い う 数 学 上 の 数 値 列 を 解 析 機 関 で 計 算 す る た め の 手 順 を、 操 作 (operations) の 順 序 と し て 詳 細 に 記 述 し た。 25 個 の 操 作、 そ の 順 序、 デ ー タ の 流 れ ─ こ れ は 後 世 の 言 葉 で 言 え ば、 紛 れ も な く「世 界 最 初 の コ ン ピ ュ ー タ・プ ロ グ ラ ム」 で あ っ た。

***

### 「思 考 を 自 動 化 す る」 と い う 予 言

エ イ ダ の 注 釈 で 最 も 引 用 さ れ る の は、 機 械 の 可 能 性 に つ い て の 一 文 で あ る。

「解 析 機 関 は 単 に 数 を 計 算 で き る だ け で な く、 数 で 表 現 さ れ う る あ ら ゆ る も の を 操 作 で き る だ ろ う。」

1843 年 の こ の 一 文 が、 100 年 後 の デ ジ タ ル コ ン ピ ュ ー タ の マ ル チ メ デ ィ ア (音 楽 ・ 文 字 ・ 画 像) を 予 言 し た と さ れ る ゆ え ん で あ る。

***

### そ の 後 ─ 36 歳 で の 死、 100 年 後 の 再 発 見

し か し 残 念 な が ら、 バ ベ ッ ジ の 解 析 機 関 は 1871 年 の 彼 の 死 ま で に 完 成 し な か っ た。 エ イ ダ も 1852 年、 36 歳 の 若 さ で 子 宮 癌 で 亡 く な っ た。

二 人 の 仕 事 は 100 年 近 く 忘 れ ら れ た。 1947 年、 ENIAC が 動 き 始 め、 1948 年 に ノ イ マ ン 型 コ ン ピ ュ ー タ が 登 場 し、 そ の あ と か ら ─ 100 年 越 し に ─ ラ ブ レ ス と バ ベ ッ ジ の 仕 事 が 再 評 価 さ れ た。

1980 年、 米 国 国 防 総 省 は 軍 事 用 の プ ロ グ ラ ミ ン グ 言 語 を「Ada」 と 命 名 し、 エ イ ダ・ラ ブ レ ス を 顕 彰 し た。 現 在 で も、 ボ ー イ ン グ・787、 国 際 宇 宙 ス テ ー シ ョ ン、 原 子 力 発 電 所 の 制 御 シ ス テ ム は「Ada」 で 書 か れ て い る。

世 界 最 初 の プ ロ グ ラ マ ー の 名 前 は、 今 も 飛 行 機 と 宇 宙 船 と と も に 飛 ん で い る。
