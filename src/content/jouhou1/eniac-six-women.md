## ENIAC ─ 世 界 初 の 電 子 計 算 機 と そ の プ ロ グ ラ マ ー た ち

### 1. 1945 年、 ペ ン シ ル ベ ニ ア 大 学

時 は 第 二 次 世 界 大 戦 末 期、 1945 年。 米 国 ペ ン シ ル ベ ニ ア 大 学 ム ー ア 工 学 部 の 地 下 室 で、 一 つ の 巨 大 な 機 械 が 完 成 し つ つ あ っ た。

ENIAC (Electronic Numerical Integrator and Computer) ─ 真 空 管 17,468 本、 重 量 30 ト ン、 設 置 面 積 約 167 平 方 メ ー ト ル。 1 秒 間 に 5,000 回 の 加 算 が で き る、 世 界 初 の 汎 用 電 子 計 算 機。

設 計 者 は、 物 理 学 者 ジ ョ ン・モ ー ク リ ー と 電 子 工 学 者 J・プ レ ス パ ー・エ ッ カ ー ト。 開 発 資 金 は 米 国 陸 軍 か ら 約 50 万 ド ル (当 時 の 通 貨 価 値)。

そ の 主 目 的 は、 砲 弾 の 弾 道 計 算。 大 砲 か ら 発 射 さ れ た 砲 弾 が、 風 速・気 圧・砲 身 仰 角・砲 弾 重 量 等 を 考 慮 し て、 何 秒 後 に ど こ に 着 弾 す る か を、 高 速 で 計 算 す る 機 械 だ っ た。

***

### 2. 6 人 の 「人 間 計 算 機」

ENIAC が 完 成 し た と し て も、 そ れ を「動 か す」 ─ つ ま り 個 々 の 計 算 を プ ロ グ ラ ミ ン グ す る ─ 仕 事 は、 別 に 必 要 で あ っ た。

そ の 仕 事 は、 当 時 ペ ン 大 学 で「人 間 計 算 機 (Human Computers)」 と し て 機 械 式 計 算 機 で 弾 道 計 算 を 行 っ て い た 6 人 の 若 い 女 性 に 任 さ れ た。

- **ケ イ・マ ク ナ ル テ ィ (Kay McNulty, 1921-2006)** ─ ア イ ル ラ ン ド 出 身、 ペ ン 大 学 数 学 部 卒。
- **ベ テ ィ・ス ナ イ ダ ー (Betty Snyder, 1917-2001)** ─ ペ ン シ ル ベ ニ ア 大 卒、 後 に COBOL 開 発 に も 関 与。
- **マ ー リ ン・ウ ェ ス コ フ (Marlyn Wescoff, 1922-2008)** ─ ペ ン 大 学 卒、 ユ ダ ヤ 系 ア メ リ カ 人。
- **ル ー ス・リ ク タ ー マ ン (Ruth Lichterman, 1924-1986)** ─ ハ ン タ ー 大 学 数 学 部 卒。
- **フ ラ ン・ビ ラ ス (Fran Bilas, 1922-2012)** ─ チ ェ ス ナ ッ ト・ヒ ル 大 学 数 学 部 卒。
- **ベ テ ィ・ジ ェ ニ ン グ ス・バ ー テ ィ ク (Betty Jennings Bartik, 1924-2011)** ─ ノ ー ス ウ ェ ス ト・ミ ズ ー リ 州 立 大 学 数 学 部 卒。

全 員 が 数 学 を 学 び、 機 械 式 計 算 機 で 何 千 回 と 弾 道 計 算 を 行 っ て き た、 当 時 と し て は 例 外 的 に 高 い 数 学 的 能 力 を 持 つ 女 性 達 で あ っ た。

***

### 3. 配 線 板 と 何 千 本 の ケ ー ブ ル

ENIAC の プ ロ グ ラ ミ ン グ は、 現 代 の そ れ と は 全 く 違 っ た。 当 時 は「ソ フ ト ウ ェ ア (ス ト ア ー ド プ ロ グ ラ ム)」 と い う 概 念 が ま だ な い。

代 わ り に、 個 々 の 計 算 ご と に、 ENIAC の 巨 大 な 機 体 の 周 り を 取 り 囲 む「配 線 板 (plugboards)」 に、 何 千 本 の ケ ー ブ ル を 物 理 的 に 差 し 込 み、 ス イ ッ チ を 手 作 業 で 切 り 替 え、 機 械 そ の も の の 回 路 を 一 回 一 回 組 み 替 え る 必 要 が あ っ た。

砲 弾 の 一 つ の 弾 道 計 算 プ ロ グ ラ ム を 動 か す た め に、 配 線 だ け で 数 日 を 要 す る こ と も あ っ た。

6 人 は、 ENIAC の 各 ユ ニ ッ ト の 配 線 図 を 完 全 に 頭 に 入 れ、 ど の ケ ー ブ ル を ど こ に 差 す か を 紙 に 設 計 し、 物 理 的 に 配 線 し、 そ し て エ ラ ー が 出 る た び に ENIAC の 内 部 に 入 っ て 真 空 管 や 配 線 を デ バ ッ グ し た。

「私 た ち は ENIAC の 中 を 知 り 尽 く し て い た」 と 後 に ベ テ ィ・ジ ェ ニ ン グ ス は 回 想 し て い る。

***

### 4. 1946 年 2 月 15 日 ─ 公 開 デ モ

1946 年 2 月 15 日、 ENIAC の 公 開 デ モ が 行 わ れ た。 6 人 の プ ロ グ ラ マ ー が 数 日 か け て 配 線 し た プ ロ グ ラ ム が、 報 道 記 者 達 の 目 の 前 で、 砲 弾 の 60 秒 間 の 弾 道 を 60 秒 以 内 に 計 算 し て み せ た。

「砲 弾 の 弾 道 そ の も の よ り 短 い 時 間 で 計 算 で き る 機 械」 ─ こ れ は 当 時 と し て は 衝 撃 的 な 速 さ で、 翌 日 の 新 聞 で 大 ニ ュ ー ス と な っ た。

し か し、 そ の 後 の レ セ プ シ ョ ン で 男 性 エ ン ジ ニ ア だ け が 招 待 さ れ、 6 人 の 女 性 プ ロ グ ラ マ ー は「機 械 を 説 明 し に 来 た モ デ ル の よ う な 女 性」 と 紹 介 さ れ る だ け で あ っ た。

新 聞 写 真 で も、 6 人 は し ば し ば「ENIAC の 横 に 立 つ 美 し い 女 性 た ち」 と キ ャ プ シ ョ ン さ れ、 そ の 名 前 が 記 さ れ る こ と は 稀 で あ っ た。

***

### 5. 40 年 の 沈 黙 と 1980 年 代 か ら の 再 評 価

6 人 の 業 績 は、 そ の 後 約 40 年 間 ほ と ん ど 知 ら れ な か っ た。 1980 年 代、 大 学 院 生 の キ ャ シ ー・ク ラ イ マ ン (Kathy Kleiman) が 古 い ENIAC 写 真 を 大 学 で 偶 然 見 つ け、 写 真 に 写 る 女 性 達 の 身 元 を 大 学 関 係 者 に 尋 ね た。

「あ あ、 彼 女 た ち は 計 算 助 手 (Refrigerator Ladies) で、 単 な る モ デ ル で す」 と い う 答 え が 返 っ た。

し か し ク ラ イ マ ン は 諦 め ず、 6 人 の 内 ま だ 生 存 し て い た メ ン バ ー を 探 し 出 し、 イ ン タ ビ ュ ー を 重 ね、 そ の 業 績 を 文 書 化 し た。

1997 年、 6 人 全 員 が WITI (Women in Technology International) Hall of Fame に 認 定。 2010 年 ド キ ュ メ ン タ リ ー 映 画『The Computers』 が 公 開 さ れ、 国 際 的 に 知 ら れ る よ う に な っ た。

ベ テ ィ・ホ ル バ ー ト ン (旧 姓 ス ナ イ ダ ー) は、 ENIAC 以 降 も 活 躍 し、 1959 年 に は COBOL プ ロ グ ラ ミ ン グ 言 語 の 開 発 に 関 与。 1997 年 ア ダ・ラ ブ レ ス 賞 受 章 (女 性 と し て の 計 算 機 科 学 へ の 貢 献 を 顕 彰)。 2001 年 没。

***

### 6. 影 の 主 役 た ち の 教 え

ENIAC の 物 語 は、 単 な る「最 初 の コ ン ピ ュ ー タ」 の 物 語 で は な い。

そ れ は、 「歴 史 に 残 ら な か っ た 貢 献」 が、 ど の よ う に 発 見 さ れ、 顕 彰 さ れ る か の 物 語 で も あ る。

技 術 史 と は、 し ば し ば「天 才 個 人」 で 語 ら れ る。 「ジ ョ ン・モ ー ク リ ー と J・プ レ ス パ ー・エ ッ カ ー ト が ENIAC を 発 明 し た」 と。
