## 76 年 後 の 帰 還 を 予 言 し た 男

### 1. 1682 年、 ロ ン ド ン 上 空 の 彗 星

1682 年 夏、 ロ ン ド ン の 空 に 一 つ の 彗 星 が 現 れ た。 26 歳 の 若 い 天 文 学 者 エ ド モ ン ド・ハ レ ー は、 こ の 彗 星 を 観 測 し、 軌 道 の 計 算 を 始 め た。

し か し、 彗 星 と い う 天 体 は 当 時 ま だ「不 規 則 な 訪 問 者」 と し て の 理 解 で あ っ た。 「い つ 来 る か 分 か ら な い、 い つ 去 る か 分 か ら な い、 不 吉 な 前 兆」 と 思 わ れ て き た。

ハ レ ー の 心 に、 一 つ の 疑 問 が 生 ま れ た。 「彗 星 は 本 当 に 不 規 則 な の か? も し か し て、 そ れ は 太 陽 系 の 一 員 と し て、 定 期 的 に 戻 っ て く る の で は な い か?」

***

### 2. ニ ュ ー ト ン と の 出 会 い

ハ レ ー の 疑 問 に 答 え る 鍵 は、 同 時 代 の ケ ン ブ リ ッ ジ 大 学 の 数 学 教 授、 ア イ ザ ッ ク・ニ ュ ー ト ン (1643-1727) の 物 理 学 で あ っ た。

1684 年、 28 歳 の ハ レ ー は ケ ン ブ リ ッ ジ に ニ ュ ー ト ン を 訪 ね、 こ う 尋 ね た。

「太 陽 か ら の 距 離 の 2 乗 に 反 比 例 す る 力 を 受 け る と き、 惑 星 は ど の よ う な 軌 道 を 描 く で し ょ う か?」

ニ ュ ー ト ン は 即 答 し た。

「楕 円 で す」

「ど う し て そ う 言 え る の で す か?」

「私 が 計 算 し ま し た」

ハ レ ー は ニ ュ ー ト ン に「そ の 計 算 を 文 章 に し て 公 表 し て ほ し い」 と 強 く 求 め た。 こ の 訪 問 が、 3 年 後 の 1687 年『自 然 哲 学 の 数 学 的 諸 原 理 (プ リ ン キ ピ ア)』 出 版 の き っ か け と な っ た。 ハ レ ー は 出 版 編 集 を 引 き 受 け、 さ ら に 自 費 で 出 版 費 用 を 全 額 負 担 し た。 「ニ ュ ー ト ン 力 学 の 公 表 そ の も の が、 ハ レ ー の お か げ で 実 現 し た」 と い わ れ る ゆ え ん で あ る。

***

### 3. 1705 年『彗 星 天 文 学 要 論』

ニ ュ ー ト ン 力 学 を 手 に 入 れ た ハ レ ー は、 こ れ を 彗 星 に 応 用 し 始 め た。 過 去 数 百 年 の 彗 星 観 測 記 録 を 集 め、 そ れ ぞ れ の 軌 道 を ニ ュ ー ト ン 力 学 で 計 算 し て い っ た。

そ し て 1705 年、 49 歳 の ハ レ ー は『彗 星 天 文 学 要 論』(Synopsis Astronomiae Cometicae) と い う 短 い 論 文 を 発 表 し た。

そ こ で 彼 は こ う 書 い た。

「1531 年 ア ピ ア ヌ ス が 観 測 し た 彗 星、 1607 年 ケ プ ラ ー が 観 測 し た 彗 星、 そ し て 1682 年 私 が 観 測 し た 彗 星 ─ こ の 三 つ は、 軌 道 要 素 が ほ ぼ 一 致 し て い る。 す な わ ち、 こ れ は 同 一 の 彗 星 が 約 76 年 周 期 で 戻 っ て く る も の で あ る。」

そ し て、 さ ら に 大 胆 な 予 言 を 行 っ た。

「し た が っ て、 こ の 彗 星 は 1758 年 末 か 1759 年 初 に 再 び 現 れ る で あ ろ う。 私 自 身 は そ れ を 見 る こ と は で き ま い が、 後 世 の 天 文 学 者 が 確 認 し て く れ る で あ ろ う。」

***

### 4. 1742 年 ─ ハ レ ー の 死

ハ レ ー は 1720 年 グ リ ニ ッ ジ 天 文 台 第 2 代 王 室 天 文 官 に 任 命 さ れ、 多 く の 天 文 学 上 の 業 績 を 残 し た。 し か し 1742 年 1 月、 85 歳 で 没 し た。 自 ら の 彗 星 予 言 の 帰 還 を 見 る こ と は な か っ た。

死 の 床 で、 ハ レ ー は こ う 言 っ た と 伝 え ら れ る (こ の 言 葉 自 体 は 後 世 の 創 作 の 可 能 性 も あ る)。

「私 の 彗 星 は、 1758 年 に 帰 っ て く る だ ろ う。 そ れ を 私 自 身 が 見 る こ と は で き な い。 し か し、 そ の 帰 還 が、 ニ ュ ー ト ン 力 学 の 正 し さ を 証 明 し、 私 の 仕 事 の 結 晶 と な っ て く れ る だ ろ う。」

***

### 5. 1758 年 ク リ ス マ ス の 夜 ─ 予 言 の 的 中

1758 年 12 月 25 日 (ク リ ス マ ス) の 夜、 ド イ ツ・ザ ク セ ン の ア マ チ ュ ア 天 文 家 ヨ ハ ン・ゲ オ ル ク・パ リ ッ チ ュ (Johann Georg Palitzsch, 1723-1788) が、 自 ら の 観 測 望 遠 鏡 で 夜 空 を 眺 め て い た。

そ し て、 ハ レ ー が 予 言 し た 位 置 に、 微 か な 光 点 を 発 見 し た。

彗 星 の 帰 還 で あ っ た。

そ の 後 1759 年 春 に か け て、 彗 星 は 明 る さ を 増 し、 ヨ ー ロ ッ パ 中 で 観 測 さ れ た。 ハ レ ー の 53 年 前 の 予 言 が、 完 全 に 的 中 し た 瞬 間 で あ っ た。

後 世、 こ の 彗 星 は「ハ レ ー 彗 星 (Halley's Comet)」 と 命 名 さ れ た。

***

### 6. 2061 年 7 月 ─ 君 と ハ レ ー 彗 星

ハ レ ー 彗 星 は そ の 後 も 約 76 年 ご と に 帰 っ て き た。

- **1835 年** ─ 19 世 紀 の 大 観 測
- **1910 年** ─ 大 接 近、 尾 が 地 球 を か す め た
- **1986 年** ─ 6 機 の 探 査 機 (欧 ・ ソ ・ 米 ・ 日 の「国 際 ハ レ ー 艦 隊」) が 接 近 観 測

そ し て、 次 の 帰 還 は…

**2061 年 7 月 28 日 頃**

中 学 生・高 校 生 の 君 た ち は、 そ の と き 50 代 前 後 だ。 ハ レ ー が 1705 年 に 立 て た 予 言 を、 君 が 自 分 の 目 で 確 か め る こ と が で き る の だ。

ハ レ ー が 1682 年 に 26 歳 で 観 測 し た 彗 星 を、 約 380 年 後 の 君 が、 同 じ 太 陽 系 内 で 観 測 す る。 そ の と き、 君 は こ の 教 材 を 思 い 出 す か も し れ な い。

「あ あ、 こ の 彗 星 は、 76 年 ご と に 帰 っ て く る、 ハ レ ー が 予 言 し た 訪 問 者 だ」 と。

科 学 と は、 過 去 と 未 来 を つ な ぐ 言 葉 で あ る。 ハ レ ー の 76 年 後 の 予 言 は、 380 年 後 の 君 ま で 届 い て い る。

***
