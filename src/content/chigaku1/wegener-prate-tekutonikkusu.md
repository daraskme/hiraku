<h3>「ジ グ ソ ー パ ズ ル の よ う に 合 う」</h3>

<p>
世 界 地 図 を 見 て、 南 米 大 陸 の 東 海 岸 と ア フ リ カ 大 陸 の 西 海 岸 を 切 り 抜 い て み よ う。<br>
そ し て 二 つ を 並 べ る と ─ <strong>ジ グ ソ ー パ ズ ル の よ う に ぴ っ た り 合 う</strong>。
</p>

<p>
こ の 不 思 議 な 事 実 に 着 目 し た の が、 ド イ ツ の 気 象 学 者 <strong>ア ル フ レ ッ ド・ヴ ェ ー ゲ ナ ー</strong> (Alfred Wegener, 1880–1930) で あ っ た。
</p>

<h3>1915 年『大 陸 と 海 洋 の 起 源』</h3>

<p>
1915 年、 35 歳 の ヴ ェ ー ゲ ナ ー が、 著 書『<strong>大 陸 と 海 洋 の 起 源</strong>』(Die Entstehung der Kontinente und Ozeane) で 衝 撃 的 な 仮 説 を 提 唱 し た。
</p>

<blockquote>
「地 球 の 大 陸 は、 過 去 に は 一 つ に つ な が っ て お り (超 大 陸 <strong>パ ン ゲ ア</strong>)、 約 2 億 年 か け て 移 動 し、 現 在 の 配 置 に な っ た」
</blockquote>

<p>
こ の 仮 説 を、 「<strong>大 陸 移 動 説</strong>」(Continental Drift) と 呼 ぶ。
</p>

<h3>ヴ ェ ー ゲ ナ ー の 証 拠</h3>

<p>
ヴ ェ ー ゲ ナ ー は、 単 な る 海 岸 線 の 形 だ け で な く、 多 数 の 証 拠 を 集 め た:
</p>

<ol>
<li><strong>大 陸 の 形</strong> ─ 南 米 と ア フ リ カ、 イ ン ド と マ ダ ガ ス カ ル、 オ ー ス ト ラ リ ア と 南 極 が ぴ っ た り 合 う</li>
<li><strong>共 通 化 石</strong> ─ メ ソ サ ウ ル ス (淡 水 性 爬 虫 類) の 化 石 が、 南 米 と ア フ リ カ で 同 時 期 の 地 層 か ら 発 見</li>
<li><strong>共 通 植 物 化 石</strong> ─ グ ロ ッ ソ プ テ リ ス (古 生 代 植 物) の 化 石 が、 南 米・ア フ リ カ・イ ン ド・オ ー ス ト ラ リ ア・南 極 の 5 大 陸 で 共 通</li>
<li><strong>地 質・岩 石 の 一 致</strong> ─ ブ ラ ジ ル と ア フ リ カ で、 同 じ 種 類 の 古 い 山 脈 が 海 を 越 え て 連 続</li>
<li><strong>古 い 氷 河 痕</strong> ─ 約 3 億 年 前 の 氷 河 の 跡 が、 5 大 陸 で 連 続 (現 在 の 配 置 で は 説 明 不 能)</li>
</ol>

<p>
こ れ ら の 証 拠 か ら、 ヴ ェ ー ゲ ナ ー は 確 信 を 持 っ て 主 張 し た: 「<strong>大 陸 は 動 い て い る</strong>」。
</p>

<h3>50 年 の 否 定</h3>

<p>
だ が、 当 時 の 学 界 は ヴ ェ ー ゲ ナ ー の 説 を <strong>嘲 笑 と と も に 否 定</strong> し た。
</p>

<p>
理 由 は 主 に こ う だ っ た:
</p>

<ul>
<li>「<strong>大 陸 を 動 か す 力 を 説 明 で き な い</strong>」 ─ ヴ ェ ー ゲ ナ ー は、 大 陸 が 動 く メ カ ニ ズ ム を 明 確 に 示 せ な か っ た</li>
<li>「<strong>気 象 学 者 が 地 質 学 に 口 を 出 す な</strong>」 ─ 専 門 領 域 を 超 え た 越 境 へ の 反 発</li>
<li>「<strong>固 体 の 大 陸 が 動 く な ど 物 理 的 に 不 可 能</strong>」 ─ 当 時 の 物 理 学 で は こ う 思 わ れ た</li>
</ul>

<p>
ヴ ェ ー ゲ ナ ー は 著 書 の 改 訂 を 重 ね、 反 論 を 続 け た が、 1920s–1950s に か け て 大 陸 移 動 説 は ほ と ん ど 完 全 に 否 定 さ れ た。
</p>

<p>
そ の 間、 ヴ ェ ー ゲ ナ ー 本 人 は 1930 年 11 月、 4 度 目 の グ リ ー ン ラ ン ド 探 検 中 に 客 死。 50 歳 だ っ た。<br>
彼 が 確 信 し て い た 真 理 が「正 し い」 と 認 め ら れ る 30 年 前 の 死 だ っ た。
</p>

<h3>1960 年 代 ─ プ レ ー ト テ ク ト ニ ク ス と し て の 復 活</h3>

<p>
1950–60 年 代、 第 二 次 大 戦 後 の 海 洋 調 査 で 革 命 的 発 見 が 続 い た:
</p>

<ul>
<li><strong>海 底 の 中 央 海 嶺 (ち ゅ う お う か い れ い)</strong> ─ 大 洋 の 中 央 に 連 な る 巨 大 な 海 底 山 脈</li>
<li><strong>海 底 の 地 磁 気 縞 状 配 列</strong> ─ 中 央 海 嶺 を 中 心 に、 同 じ 縞 状 模 様 が 対 称 に 並 ぶ</li>
<li><strong>海 洋 底 の 若 さ</strong> ─ 海 底 の 岩 石 は ど こ も 2 億 年 以 内、 中 央 海 嶺 か ら 離 れ る ほ ど 古 い</li>
</ul>

<p>
こ れ ら の 観 測 結 果 か ら、 1960 年 代 に「<strong>海 底 拡 大 説</strong>」(Sea-floor spreading) が 確 立。 さ ら に 大 陸 移 動 説 と 統 合 さ れ て、 1968 年 頃 に は「<strong>プ レ ー ト テ ク ト ニ ク ス</strong>」(Plate Tectonics) と し て 体 系 化 さ れ た。
</p>

<p>
ヴ ェ ー ゲ ナ ー の 大 陸 移 動 説 ─ 1915 年 に 提 唱、 否 定 さ れ、 1930 年 に 提 唱 者 客 死、 そ し て <strong>50 年 後 に 復 活</strong>。<br>
地 球 科 学 史 で 最 も ド ラ マ チ ッ ク な「名 誉 回 復」 の 一 つ で あ る。
</p>

<h3>プ レ ー ト テ ク ト ニ ク ス の 全 容</h3>

<p>
現 代 の プ レ ー ト テ ク ト ニ ク ス は こ う 説 明 す る:
</p>

<ol>
<li>地 球 の 表 面 は 十 数 枚 の「プ レ ー ト」 (厚 さ 約 100 km の 岩 盤) で 構 成</li>
<li>地 球 内 部 (マ ン ト ル) の 対 流 で、 プ レ ー ト が 年 数 cm ~ 10 cm の 速 度 で 動 く</li>
<li>プ レ ー ト 同 士 の 境 界 で、 地 震・火 山・造 山 運 動 が 起 こ る</li>
<li>長 い 時 間 (数 千 万 年 ~ 数 億 年) で、 大 陸 が 移 動・分 裂・合 体 す る</li>
</ol>

<p>
こ れ で、 ヴ ェ ー ゲ ナ ー が 1915 年 に 提 唱 し た「大 陸 が 動 く メ カ ニ ズ ム」 が、 つ い に 解 明 さ れ た。
</p>

<h3>日 本 列 島 と プ レ ー ト テ ク ト ニ ク ス</h3>

<p>
日 本 列 島 は、 <strong>4 つ の プ レ ー ト</strong> が 衝 突 す る、 世 界 で も 稀 な 場 所 に あ る (本 サ イ ト chigaku1/terada-jishin-no-hanashi):
</p>

<ul>
<li>太 平 洋 プ レ ー ト</li>
<li>フ ィ リ ピ ン 海 プ レ ー ト</li>
<li>ユ ー ラ シ ア プ レ ー ト</li>
<li>北 米 プ レ ー ト</li>
</ul>

<p>
こ の「4 プ レ ー ト 衝 突」 が、 日 本 で 地 震・火 山・温 泉・山 脈 が 多 い 根 本 原 因 で あ る。<br>
そ し て こ の 認 識 自 体、 ヴ ェ ー ゲ ナ ー の 大 陸 移 動 説 か ら 始 ま っ た 50 年 余 り の 探 究 の 結 果 で あ る。
</p>

<h3>科 学 史 の 教 訓</h3>

<p>
ヴ ェ ー ゲ ナ ー の 物 語 は、 い く つ か の 大 切 な 教 訓 を 残 す:
</p>

<ol>
<li><strong>多 数 派 が 否 定 す る 仮 説 も、 後 に 真 理 と わ か る こ と が あ る</strong></li>
<li><strong>専 門 領 域 を 超 え た 越 境 (気 象 学 者 が 地 質 学 へ) が、 大 き な 発 見 を 生 む こ と が あ る</strong></li>
<li><strong>「メ カ ニ ズ ム を 説 明 で き な い」 と し て 否 定 さ れ た 観 察 事 実 が、 後 に メ カ ニ ズ ム が わ か っ て 復 活 す る</strong></li>
<li><strong>科 学 者 が 自 分 の 主 張 を 守 り 続 け る 勇 気 が、 50 年 後 の 名 誉 回 復 に つ な が る</strong></li>
</ol>

<p>
ヴ ェ ー ゲ ナ ー は、 自 分 の 仮 説 が 正 し い と 認 め ら れ る 前 に 死 ん だ。<br>
だ が 彼 が 蒔 い た 種 は、 今 や プ レ ー ト テ ク ト ニ ク ス と し て、 地 球 科 学 の 中 心 に 育 っ て い る。
</p>

<p>
