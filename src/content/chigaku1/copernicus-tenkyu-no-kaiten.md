## 地 動 説 の 誕 生 ─ 1500 年 続 い た 天 動 説 を 覆 す

### 1. プ ト レ マ イ オ ス か ら コ ペ ル ニ ク ス ま で の 1500 年

紀 元 2 世 紀、 ア レ ク サ ン ド リ ア の 天 文 学 者 プ ト レ マ イ オ ス が 著 書『ア ル マ ゲ ス ト』 で、 古 代 ギ リ シ ア の 天 動 説 を 数 学 的 に 完 成 さ せ た。

「地 球 が 宇 宙 の 中 心 で あ り、 太 陽・月・惑 星・恒 星 が そ の 周 り を 回 っ て い る」

こ の 宇 宙 観 は、 古 代 ロ ー マ・中 世 ヨ ー ロ ッ パ・イ ス ラ ム 世 界・中 世 末 期 の キ リ ス ト 教 会 に 至 る ま で、 約 1500 年 間 標 準 教 義 で あ っ た。 そ し て そ れ は、 単 な る 天 文 学 上 の 説 で は な か っ た。 「我 々 人 間 が 宇 宙 の 中 心 で あ る」 と い う 思 想 と 結 び つ い て い た。

***

### 2. コ ペ ル ニ ク ス の 半 世 紀 の 観 測 と 計 算

1473 年、 王 国 プ ロ イ セ ン (現 ポ ー ラ ン ド) の ト ル ニ に 生 ま れ た ニ コ ラ ウ ス・コ ペ ル ニ ク ス は、 イ タ リ ア の ボ ロ ー ニ ャ 大 学 で 天 文 学 を 学 び、 帰 国 後 フ ロ ム ボ ル ク 大 聖 堂 の 司 教 座 聖 堂 参 事 会 員 (現 代 の「カ ト リ ッ ク 大 聖 堂 の 法 律 顧 問」 の よ う な 職) と な っ た。

彼 は こ の 仕 事 の 傍 ら、 半 世 紀 以 上 に わ た っ て 天 体 観 測 と 数 学 計 算 を 続 け た。 そ し て 気 づ い た。

「プ ト レ マ イ オ ス の 天 動 説 は、 惑 星 の 不 規 則 な 運 動 を 説 明 す る た め に『周 転 円』『離 心 円』 と い っ た 複 雑 な 仮 説 を 重 ね 続 け、 計 算 が 醜 い ほ ど 複 雑 で あ る。 し か し、 も し『太 陽 を 中 心 に 置 き、 地 球 を 太 陽 の 周 り を 回 る 一 惑 星 と み な す』 な ら ば、 計 算 は 驚 く ほ ど 簡 潔 に な る。」

***

### 3. 1543 年 5 月 24 日 ─ 死 の 床 で の 受 け 取 り

し か し、 コ ペ ル ニ ク ス は こ の 結 論 を、 容 易 に は 公 表 し な か っ た。 1530 年 代 か ら 草 稿 を 用 意 し て い た が、 当 時 の カ ト リ ッ ク 教 会 が「人 間 中 心 説」 を 信 仰 と 結 び つ け て い た た め、 異 端 と さ れ る こ と を 恐 れ た。

1542 年、 70 歳 と な っ た コ ペ ル ニ ク ス が 病 で 床 に 伏 し た と き、 弟 子 の レ テ ィ ク ス と ニ ュ ル ン ベ ル ク の 出 版 業 者 が、 や っ と『天 球 の 回 転 に つ い て』 を 印 刷 に か け た。

1543 年 5 月 24 日、 コ ペ ル ニ ク ス は 死 の 床 で 印 刷 さ れ た 一 部 を 手 渡 さ れ た と 伝 え ら れ る。 そ の 数 時 間 後、 彼 は 静 か に 息 を 引 き 取 っ た。

70 歳 で、 半 世 紀 の 仕 事 を 世 に 残 し、 そ し て 異 端 審 問 に 直 接 さ ら さ れ る こ と も な く、 静 か に 旅 立 っ た。

***

### 4. ガ リ レ オ・ケ プ ラ ー・ニ ュ ー ト ン へ

コ ペ ル ニ ク ス 没 後、 こ の「太 陽 中 心 説」 を 引 き 継 い だ の が、 三 人 の 巨 人 で あ っ た。

**1610 年、 ガ リ レ オ・ガ リ レ イ (1564-1642)**。 自 ら 改 良 し た 望 遠 鏡 で、 木 星 の 4 つ の 衛 星 (= 木 星 を 中 心 に 衛 星 が 回 っ て お り、 必 ず し も 地 球 が 中 心 で は な い 証 拠)、 金 星 の 満 ち 欠 け (= 金 星 が 太 陽 の 周 り を 回 っ て い る 観 測 的 証 拠) な ど を 観 測。 地 動 説 を 強 力 に 支 持。 1633 年 異 端 審 問 で 撤 回 強 制、 終 身 軟 禁。

**1619 年、 ヨ ハ ネ ス・ケ プ ラ ー (1571-1630)**。 コ ペ ル ニ ク ス の「円 軌 道」 を、 観 測 デ ー タ か ら「楕 円 軌 道」 に 修 正。 ケ プ ラ ー の 3 法 則 と し て 完 成。

**1687 年、 ア イ ザ ッ ク・ニ ュ ー ト ン (1643-1727)**。『プ リ ン キ ピ ア (自 然 哲 学 の 数 学 的 諸 原 理)』 で 万 有 引 力 を 提 案 し、 ケ プ ラ ー の 法 則 (惑 星 が 太 陽 の 周 り を 回 る) の 理 由 を 説 明。 こ の 1687 年 で、 約 150 年 続 い た「天 文 学 革 命」 が 完 成 し た。

***

### 5. 「コ ペ ル ニ ク ス 的 転 回」 と い う 言 葉

コ ペ ル ニ ク ス の 業 績 が 哲 学 史 に 残 し た 最 大 の 言 葉 は、 「コ ペ ル ニ ク ス 的 転 回 (Copernican Revolution)」 で あ る。

18 世 紀、 ド イ ツ の 哲 学 者 イ マ ヌ エ ル・カ ン ト (1724-1804) は『純 粋 理 性 批 判』 (1781) で、 自 ら の 認 識 論 を コ ペ ル ニ ク ス の 業 績 に な ぞ ら え て こ う 述 べ た。

「私 の 思 想 は、 認 識 論 上 の コ ペ ル ニ ク ス 的 転 回 で あ る。 こ れ ま で 人 は『対 象 が 認 識 を 決 定 す る』 と 考 え て き た が、 私 は『認 識 が 対 象 を 決 定 す る』 と 提 案 す る。」

以 来、 「コ ペ ル ニ ク ス 的 転 回」 と い う 言 葉 は、 哲 学・思 想・科 学 史 で「視 点 を 180 度 変 え る 革 命 的 転 回」 を 意 味 す る 比 喩 と し て 使 わ れ 続 け て い る。

***

### 6. 結 び

1543 年、 死 の 床 で コ ペ ル ニ ク ス が 受 け 取 っ た 一 冊。 70 年 後 の ガ リ レ オ。 さ ら に 70 年 後 の ケ プ ラ ー。 そ し て さ ら に 70 年 後 の ニ ュ ー ト ン。

約 200 年 の リ レ ー で、 人 類 の 宇 宙 観 は「地 球 中 心」 か ら「太 陽 中 心」 へ、 そ し て「太 陽 も 銀 河 系 の 一 つ の 恒 星 に 過 ぎ な い」 と い う さ ら に 広 い 視 野 へ と 進 化 し た。

中 学 校 で「地 球 は 太 陽 の 周 り を 1 年 で 回 る」 と 学 ぶ 一 つ の 事 実 は、 こ の 1500 年 の 天 動 説 を 覆 し た リ レ ー の 結 晶 で あ る。

***

(注：本 文 は、 コ ペ ル ニ ク ス『天 球 の 回 転 に つ い て』 (1543) の 序 文 と 第 1 巻 の 趣 旨 を、 中 学・高 校 の 地 学 教 材 と し て 編 集 部 が 再 構 成 し た 解 説 で あ る。 原 文 は ラ テ ン 語 で 書 か れ、 現 代 で は 多 く の 言 語 に 翻 訳 さ れ て い る。)
