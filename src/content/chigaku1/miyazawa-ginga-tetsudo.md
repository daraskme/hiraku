<h3>「ジ ョ バ ン ニ、 君 は 何 を 知 っ て い ま す か」</h3>

<p>
小 学 校 の 教 室。 先 生 が 黒 板 に 大 き な 銀 河 系 の 図 を 描 い て、 ジ ョ バ ン ニ に 尋 ね る ─
「皆 さ ん、 そ れ で は こ の よ う に 川 の よ う に 見 え る 銀 河 が、 ほ ん と う は ど ん な も の か
ご 承 知 で す か」。
</p>

<p>
宮 沢 賢 治『銀 河 鉄 道 の 夜』 は、 こ の 場 面 か ら 始 ま る。 文 学 で あ り な が ら、 冒 頭 か ら 天 文 学 の
講 義 で あ る。 賢 治 が 描 こ う と し た の は、 単 な る 童 話 で は な い ─ 文 学 と 科 学 が 融 合 し た 宇 宙 旅 行
だ っ た の だ。
</p>

<h3>賢 治 の も う 一 つ の 顔 ─ 科 学 者 と し て の 賢 治</h3>

<p>
本 サ イ ト の 小 4 国 語 で、 君 は『や ま な し』 や『雪 渡 り』 を 通 し て 詩 人・童 話 作 家 と し て の
賢 治 に 出 会 っ た。 し か し、 賢 治 に は も う 一 つ の 顔 が あ る ─ 科 学 者 と し て の 賢 治 だ。
</p>

<p>
賢 治 は 盛 岡 高 等 農 林 学 校 (現 岩 手 大 学 農 学 部) で 地 質・土 壌・気 象 を 学 び、 卒 業 後 も 趣 味 で
天 文 学・地 学 ・ 鉱 物 学 を 研 究 し 続 け た。 故 郷 花 巻 で「岩 手 国 民 高 等 学 校」 を 開 き、 農 民 に
科 学 を 教 え た。
</p>

<h3>『銀 河 鉄 道 の 夜』 の 銀 河 ─ 大 正 期 の 天 文 学 知 識</h3>

<p>
賢 治 が こ の 物 語 を 書 い た の は 1924 年 頃 か ら。 当 時、 ま だ「銀 河 系 の 外 に も 銀 河 が あ る」
こ と さ え 確 認 さ れ て い な か っ た (ハ ッ ブ ル が ア ン ド ロ メ ダ を「銀 河 外 銀 河」 と 確 認 す る の は
1925 年)。
</p>

<p>
し か し 賢 治 は、 当 時 の 最 新 天 文 学 を 学 び、 物 語 に 取 り 入 れ た ─
</p>

<ul>
  <li>銀 河 ・ 天 の 川 が「無 数 の 星 の 集 ま り」 で あ る こ と</li>
  <li>白 鳥 座・ケ ン タ ウ ル ス 座・南 十 字 星 等 の 星 座</li>
  <li>「星 雲」「巨 大 望 遠 鏡」 等 の 当 時 の 最 新 概 念</li>
</ul>

<h3>「銀 河 鉄 道」 が 旅 す る ル ー ト</h3>

<p>
ジ ョ バ ン ニ と カ ン パ ネ ル ラ が 銀 河 鉄 道 で 旅 す る ル ー ト を、 現 代 天 文 学 で 追 っ て み よ う ─
</p>

<ul>
  <li><strong>白 鳥 座</strong> ─ 夏 の 夜 空 で、 天 の 川 の 中 を 飛 ぶ 白 鳥 の 形。 物 語 で は「白 鳥 の 停 車 場」。 デ ネ ブ ま で 2615 光 年。</li>
  <li><strong>鷲 座</strong> ─ 隣 の 星 座。 ア ル タ イ ル (七 夕 の 彦 星) は 16.7 光 年。</li>
  <li><strong>ケ ン タ ウ ル ス 座</strong> ─ 南 天 の 大 星 座。 物 語 の「ケ ン タ ウ ル ス 祭 り」。 ア ル フ ァ 星 は 太 陽 系 最 寄 り の 恒 星 (4.4 光 年)。</li>
  <li><strong>南 十 字 星</strong> ─ 物 語 の 終 着 駅。 4 つ の 明 る い 星 が 十 字 を 描 く 南 天 の 星 座。</li>
</ul>

<p>
ル ー ト 全 体 を 通 し て、 賢 治 は「天 の 川 の 西 端 か ら 南 へ」 と い う 銀 河 系 の 構 造 を 反 映 し た
旅 行 路 を 描 い て い る。
</p>

<h3>銀 河 系 の 実 像 ─ 高 1 地 学 で 学 ぶ こ と</h3>

<p>
現 代 天 文 学 で 分 か っ て い る 銀 河 系 の 構 造 ─
</p>

<ul>
  <li>形 ─ 円 盤 状 (棒 渦 巻 銀 河)。 中 心 に「バ ル ジ」 (球 状)、 周 囲 に「ハ ロ ー」 (球 状)。</li>
  <li>直 径 ─ 約 10 万 光 年 (10⁵ 光 年)。 光 で 横 切 る の に 10 万 年。</li>
  <li>厚 さ ─ 円 盤 部 約 1 千 光 年 (10³ 光 年)。</li>
  <li>恒 星 数 ─ 約 2000 億 ~ 4000 億 個。</li>
  <li>太 陽 系 の 位 置 ─ 中 心 か ら 約 2.6 万 光 年 の 外 れ。 銀 河 中 心 を 約 2.4 億 年 で 公 転。</li>
</ul>

<p>
こ れ が 賢 治 の「銀 河 鉄 道」 が 旅 す る 舞 台 で あ る。 物 語 を 読 む 時 に、 こ の マ ッ プ を 頭 に
入 れ て 読 む と、 詩 と 科 学 が 一 体 と な っ た 体 験 が で き る。
</p>

<h3>結 び ─ 賢 治 の 視 点 で 夜 空 を 見 上 げ よ う</h3>

<p>
夏 の 夜、 街 を 離 れ て 暗 い と こ ろ で 夜 空 を 見 上 げ よ う。 そ こ に 帯 状 に 流 れ る 天 の 川 が 見 え る。
そ れ が 銀 河 系 を「内 側 か ら」 見 た 姿 で あ る。
</p>

<p>
そ し て 賢 治 の 物 語 を 思 い 出 そ う ─ そ の 帯 の 中 に「銀 河 鉄 道」 が 走 り、 ジ ョ バ ン ニ と
カ ン パ ネ ル ラ が 旅 し て い る か も し れ な い。 文 学 と 科 学 が 一 体 と な っ た 賢 治 の 想 像 力 を、
君 も 受 け 継 い で 欲 し い。
</p>
