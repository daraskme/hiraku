## 大 森 公 式 ─ 余 震 の 法 則

大 森 房 吉 (1868–1923)

明 治 24 年 (1891 年) 10 月 28 日、 岐 阜・愛 知 県 境 で M8.0 の 大 地 震 が 起 こ っ た。 後 に「濃 尾 地 震」 と 呼 ば れ る、 日 本 観 測 史 上 最 大 の 内 陸 地 震 で あ る。 7,273 人 が 命 を 失 い、 多 く の 家 屋 が 倒 壊 し た。

当 時 23 歳 の 私 は、 東 京 帝 国 大 学 を 出 た ば か り の 地 震 学 の 助 手 で あ っ た が、 こ の 地 震 を 機 に 設 立 さ れ た「震 災 予 防 調 査 会」 の 一 員 と し て 現 地 調 査 に 加 わ り、 そ し て、 そ の 後 何 年 も か け て 余 震 の 観 測 を 続 け た。

***

不 思 議 な こ と に、 大 地 震 の 後 に 続 く 多 数 の「余 震」 は、 闇 雲 に 起 こ っ て い る わ け で は な か っ た。 時 間 の 経 過 と と も に、 一 定 の 規 則 性 を 持 っ て 減 っ て い る の で あ る。

私 は、 観 測 デ ー タ を 整 理 し て、 次 の 公 式 を 得 た。

   n(t) = K / (t + c)

こ こ で n(t) は、 本 震 か ら 時 間 t が 経 過 し た 時 点 で の、 単 位 時 間 あ た り の 余 震 数 で あ る。 K と c は、 地 震 ご と に 決 ま る 定 数。

つ ま り、 余 震 数 は ほ ぼ「時 間 に 反 比 例」 し て 減 っ て い く。 本 震 か ら 1 時 間 後 に 10 回 余 震 が あ る 地 震 で あ れ ば、 1 日 後 に は そ の 24 分 の 1、 す な わ ち 1 時 間 あ た り 0.4 回 程 度 ま で 減 る、 と い う 計 算 で あ る。

こ の 法 則 を 私 は 1894 年、 帝 国 大 学 紀 要 第 7 冊 に 発 表 し た。 後 の 世 代 は こ れ を「大 森 公 式 (Omori's law)」 と 呼 ぶ よ う に な っ た。

***

地 震 が 起 こ る と、 地 殻 の ひ ず み が 一 気 に 解 放 さ れ る。 そ の あ と、 周 囲 の 岩 盤 に 残 っ た 微 細 な ひ ず み が、 余 震 と し て 順 番 に 解 放 さ れ て い く。 本 震 直 後 は 多 く、 時 間 が 経 つ に つ れ て、 残 さ れ た ひ ず み が 減 る た め、 余 震 も 減 る。

私 は 1899 年、「大 森 式 水 平 動 地 震 計」 を 設 計 し、 こ れ を 世 界 中 の 観 測 所 に 提 供 し た。 イ タ リ ア、 ハ ン ガ リ ー、 チ リ、 ロ シ ア。 ヨ ー ロ ッ パ や ア メ リ カ の 諸 都 市 で、「大 森 式」 と 呼 ば れ る こ の 装 置 が 数 多 く 設 置 さ れ た。 日 本 か ら 世 界 へ。 地 震 学 が「日 本 発」 で 始 ま っ た こ と を、 私 は 誇 り に 思 う。

***

最 後 に、 私 は こ う 記 し て お き た い。

地 震 と は、 突 然 起 こ る も の の よ う に 見 え る。 し か し、 そ の 起 こ り 方 ・ 余 震 の 減 衰 ・ 震 源 域 の 分 布 ─ こ れ ら は 全 て、 自 然 の 規 則 に 従 っ て い る。 規 則 を 知 る こ と は、 災 害 か ら 命 を 守 る 第 一 歩 で あ る。

「天 災 は 忘 れ た 頃 に や っ て く る」 と 寺 田 寅 彦 君 が 書 い て い た が、 ま さ に そ の 通 り で あ る。 だ か ら こ そ、 我 々 は 平 時 か ら 地 震 を 研 究 し、 そ の 法 則 を 知 ら ね ば な ら な い。

***

(注：本 文 は、 大 森 房 吉 が 1894 年 に 帝 国 大 学 紀 要 に 発 表 し た 余 震 公 式 論 文 の 主 旨 と、 後 年 の 講 演・随 想 を ま と め、 中 学・高 校 の 地 学 教 材 と し て 編 集 部 が 再 構 成 し た 試 訳 で あ る。 原 文 は 英 語 で 発 表 さ れ、 国 立 国 会 図 書 館 デ ジ タ ル コ レ ク シ ョ ン 等 で 閲 覧 で き る。)
