<h3>「雪 は 天 か ら 送 ら れ た 手 紙」 ─ 高 校 地 学 で の 再 読</h3>

<p>
中 学 で 君 は、 中 谷 宇 吉 郎 の 雪 研 究 を 既 に 学 ん だ (本 サ イ ト chugakurika1/yuki、 chugakurika1/nakaya-yuki-wo-tsukuru-hanashi)。<br>
こ こ で は、 同 じ テ ー マ を <strong>高 校 地 学 ・ 気 象 学 の 視 点</strong> で、 一 歩 進 め て 学 ぼ う。
</p>

<h3>な ぜ 雪 結 晶 か ら 気 象 が 読 め る か</h3>

<p>
中 谷 宇 吉 郎 の 最 大 の 業 績 は、 「<strong>雪 結 晶 の 形 と、 そ れ が 形 成 さ れ た 上 空 の 気 象 条 件 (気 温・湿 度) の 対 応 関 係</strong>」 を 体 系 化 し た こ と で あ る。
</p>

<p>
こ れ を 図 に し た の が「<strong>中 谷 ダ イ ヤ グ ラ ム</strong>」(Nakaya diagram) で あ る:
</p>

<ul>
<li><strong>横 軸</strong> ─ 気 温 (0 °C ~ −30 °C)</li>
<li><strong>縦 軸</strong> ─ 湿 度 (水 蒸 気 量)</li>
<li><strong>図 の 中</strong> ─ そ の 条 件 で で き る 結 晶 の 形 (板 状、 柱 状、 樹 枝 状 等) が 描 か れ る</li>
</ul>

<p>
こ の 図 を 使 え ば ─
</p>

<ol>
<li>地 上 で 雪 を 顕 微 鏡 で 観 察</li>
<li>結 晶 形 を 判 別</li>
<li>中 谷 ダ イ ヤ グ ラ ム と 照 合</li>
<li>上 空 数 千 m の 気 温・湿 度 が 推 定 で き る</li>
</ol>

<p>
こ れ が、 「<strong>雪 は 天 か ら 送 ら れ た 手 紙</strong>」 と い う 中 谷 の 名 言 の 厳 密 な 意 味 で あ る。
</p>

<h3>地 球 物 理 学 と し て の 中 谷 の 仕 事</h3>

<p>
中 谷 の 仕 事 は、 単 な る「雪 の 観 察」 で は な い。 地 球 物 理 学 (geophysics) の 重 要 な 一 領 域 で あ る。
</p>

<p>
地 球 物 理 学 = 地 球 を 物 理 学 の 手 法 で 研 究 す る 学 問。 領 域 は 多 様:
</p>

<ul>
<li>地 震 学 ─ 寺 田 寅 彦 (本 サ イ ト)</li>
<li>気 象 学 ─ 中 谷 宇 吉 郎</li>
<li>火 山 学</li>
<li>海 洋 物 理 学</li>
<li>大 気 物 理 学</li>
<li>古 気 候 学</li>
</ul>

<p>
中 谷 の 雪 研 究 は、 こ の う ち の「気 象 学」 と「大 気 物 理 学」 に 跨 が る。
</p>

<h3>七 種 類 の 雪 結 晶</h3>

<p>
中 谷 の 分 類 で は、 雪 結 晶 は <strong>7 大 種 類・約 80 種</strong> に 分 け ら れ る:
</p>

<table>
<thead>
<tr><th>大 種 類</th><th>気 温</th><th>湿 度</th></tr>
</thead>
<tbody>
<tr><td>板 状 (い た じ ょ う)</td><td>−10 ~ −12 °C, −18 °C 以 下</td><td>中 程 度</td></tr>
<tr><td>柱 状 (ち ゅ う じ ょ う)</td><td>−5 ~ −8 °C, −22 °C 以 下</td><td>低 〜 中</td></tr>
<tr><td>樹 枝 状 (じ ゅ し じ ょ う)</td><td>−13 ~ −16 °C</td><td>高 い</td></tr>
<tr><td>不 規 則 樹 枝 状</td><td>−13 ~ −16 °C</td><td>非 常 に 高 い</td></tr>
<tr><td>角 板 と 樹 枝 の 混 合</td><td>−10 ~ −15 °C</td><td>中 〜 高</td></tr>
<tr><td>砲 弾 状</td><td>−6 ~ −20 °C</td><td>中</td></tr>
<tr><td>あ ら れ ・ ひ ょ う ・ 雪 つ ぶ</td><td>0 °C 付 近</td><td>過 飽 和</td></tr>
</tbody>
</table>

<p>
こ の 表 を 使 え ば、 君 が 庭 で 拾 っ た 一 枚 の 雪 か ら、 数 千 m 上 空 の 気 象 を 読 み 解 け る。
</p>

<h3>現 代 の リ モ ー ト セ ン シ ン グ と の 関 連</h3>

<p>
21 世 紀 の 気 象 学 で は、 気 象 衛 星 ・ ラ ジ オ ゾ ン デ ・ ド ッ プ ラ ー レ ー ダ ー な ど で 上 空 の 気 象 を 直 接 観 測 で き る。<br>
だ が、 中 谷 の 「雪 結 晶 か ら 気 象 を 読 む」 方 法 は、 今 も <strong>気 候 史 研 究</strong> や <strong>古 気 候 学</strong> で 重 要 な 役 割 を 果 た し て い る。
</p>

<p>
例 え ば、 南 極 ・ グ リ ー ン ラ ン ド の 氷 床 か ら 採 取 し た 氷 サ ン プ ル (ア イ ス コ ア) を 顕 微 鏡 で 見 れ ば、 何 万 年 も 前 の 上 空 大 気 の 状 態 が 読 み 取 れ る。<br>
こ れ は、 中 谷 の 1936 年 の 研 究 が 開 い た 道 で あ る。
</p>

<h3>師 弟 二 代 の 物 理 学 ─ 寺 田・中 谷</h3>

<p>
寺 田 寅 彦 (1878–1935) → 中 谷 宇 吉 郎 (1900–1962) と い う 師 弟 関 係 は、 日 本 地 球 物 理 学 の 重 要 な 系 譜 で あ る。
</p>

<ul>
<li>寺 田 ─ 地 震・流 体・身 近 な 物 理 (本 サ イ ト chigaku1/terada-jishin-no-hanashi、 chugakurika1 多 数)</li>
<li>中 谷 ─ 雪 結 晶・霜 柱・気 象 (本 サ イ ト chugakurika1/yuki 等)</li>
</ul>

<p>
二 人 は、 日 本 の 自 然 を「物 理 学 の 眼」 で 観 察 し、 同 時 に そ れ を「美 し い 随 筆」 と し て 書 き 残 し た 名 手 で あ っ た。<br>
彼 ら の 仕 事 は、 現 代 の 地 球 物 理 学・気 象 学・防 災 科 学 の 礎 と な っ て い る。
</p>

<h3>君 自 身 の 雪 観 察 を</h3>

<p>
こ の 冬、 雪 が 降 っ た ら、 黒 い 紙 と 虫 眼 鏡 を 持 っ て 外 へ。<br>
雪 結 晶 を じ っ く り 観 察 し、 中 谷 の 7 種 分 類 と 照 合 し、 上 空 数 千 m の 気 象 を 推 定 し て み よ う。
</p>

<p>
「<strong>雪 は 天 か ら 送 ら れ た 手 紙</strong>」 ─ そ の 手 紙 を、 君 自 身 が 読 む の だ。
</p>

<p>
こ れ が、 中 谷 宇 吉 郎 が 90 年 前 に 開 い た 地 球 物 理 学 の 入 り 口 で あ る。
</p>
