<h3>本 文 (牧 野 富 太 郎 の 言 葉)</h3>

<blockquote>
<p>
私 は 植 物 の 精 (せ い) で あ る。 植 物 が 大 好 き だ。<br>
朝 起 き る か ら、 夜 寝 る ま で、 私 の 頭 の 中 に は い つ も 植 物 が あ る。<br>
食 事 を し て い て も、 そ の 食 卓 の 上 に 並 ぶ 野 菜 を 見 て、 「こ の キ ュ ウ リ は ど の 産 地 の も の だ ろ う」「こ の レ タ ス の 品 種 は」 と、 観 察 を 始 め て し ま う。
</p>

<p>
私 が 植 物 を 愛 す る よ う に な っ た の は、 子 ど も の 頃、 故 郷 の 高 知 県 佐 川 (さ か わ) の 山 を 歩 い た こ と か ら 始 ま る。<br>
山 に は、 名 も 知 ら ぬ 美 し い 花 が、 数 え 切 れ な い ほ ど 咲 い て い た。<br>
そ れ ぞ れ が、 違 う 顔 を 持 ち、 違 う 色 を 持 ち、 違 う 香 り を 持 つ。<br>
私 は そ の 一 つ 一 つ に 心 を 動 か さ れ、 名 前 を 知 り た く な っ た。
</p>

<p>
だ が、 当 時 の 日 本 に は、 植 物 の 名 前 を き ち ん と 教 え て く れ る 図 鑑 も 学 者 も、 ほ と ん ど な か っ た。<br>
そ こ で 私 は 決 心 し た: <strong>「自 分 で、 日 本 の 植 物 す べ て に、 名 前 を つ け よ う」</strong> と。
</p>

<p>
こ の 決 心 か ら、 私 の 60 年 以 上 の 植 物 学 者 と し て の 仕 事 が 始 ま っ た。<br>
山 に 入 っ て 植 物 を 採 集 し、 家 に 持 ち 帰 り、 顕 微 鏡 で 観 察 し、 細 部 を 絵 に 描 き、 既 存 の 文 献 と 照 合 し、 新 種 と わ か れ ば 学 名 を つ け る。<br>
こ の 地 道 な 作 業 を、 毎 日、 毎 日 続 け た。
</p>

<p>
1884 年、 22 歳 の 私 は、 故 郷 高 知 で 一 つ の 小 さ な 草 を 発 見 し た。 ナ デ シ コ 目 の 多 年 草 で、 既 存 の ど の 文 献 に も 載 っ て い な い 新 種 で あ っ た。<br>
私 は そ れ に 「<strong>ヤ マ ト グ サ</strong>」 (Theligonum japonicum) と い う 学 名 を つ け、 1889 年 に 発 表 し た。
</p>

<p>
こ れ は、 <strong>日 本 人 が 自 国 の 植 物 に、 国 際 学 名 を つ け て 発 表 し た 最 初 の 例</strong> と な っ た。<br>
そ れ ま で、 日 本 の 植 物 は す べ て、 外 国 人 (主 に シ ー ボ ル ト ら ド イ ツ 人) に よ っ て 命 名 さ れ て き た。 私 の こ の 発 表 で、 「日 本 人 自 身 が、 自 国 の 自 然 を 命 名 す る」 時 代 が 始 ま っ た の だ。
</p>

<p>
そ の 後 私 は、 約 1500 種 の 植 物 に 学 名 を 与 え、 約 5000 枚 の 植 物 画 を 描 き、 大 図 鑑『<strong>牧 野 日 本 植 物 図 鑑</strong>』を 78 歳 で 完 成 さ せ た。
</p>

<p>
私 に は 小 学 校 中 退 の 学 歴 し か な い。 大 学 教 授 に な れ た わ け で も、 大 き な 研 究 所 に 所 属 で き た わ け で も な い。<br>
だ が、 一 つ の こ と だ け は 譲 ら な か っ た: <strong>「自 然 を 真 摯 に 観 察 す る」</strong> と い う こ と。<br>
こ の 姿 勢 が あ れ ば、 学 歴 や 肩 書 き が な く て も、 学 問 は で き る。
</p>

<p>
若 い 諸 君 へ。<br>
身 の 回 り の 自 然 を、 一 度、 立 ち 止 ま っ て じ っ く り 見 て み た ま え。<br>
そ こ に は、 君 が 知 ら な か っ た 美 し さ と 驚 き が、 必 ず あ る。<br>
そ し て、 そ の 驚 き を 大 切 に し 続 け る な ら ば、 君 も ま た、 自 然 の 「友」 と し て 生 き ら れ る だ ろ う。
</p>
</blockquote>

<h3>解 説 ─ 「日 本 の 植 物 学 の 父」</h3>

<p>
牧 野 富 太 郎 (1862–1957) は、 95 歳 と い う 長 命 を 全 う し た、 日 本 が 誇 る 在 野 の 植 物 学 者 で あ る。
</p>

<p>
彼 の 業 績 は 驚 異 的 で あ る:
</p>

<ul>
<li><strong>命 名 し た 植 物</strong> ─ 約 <strong>1500 種</strong></li>
<li><strong>植 物 採 集 ・ 標 本</strong> ─ 約 <strong>40 万 点</strong></li>
<li><strong>緻 密 な 植 物 画</strong> ─ 約 <strong>5000 図</strong></li>
<li><strong>主 要 著 書</strong> ─ 『日 本 植 物 志 図 篇』『牧 野 日 本 植 物 図 鑑』『随 筆 植 物 一 日 一 題』 な ど</li>
</ul>

<p>
こ れ ら の 業 績 は、 一 人 の 人 間 の 生 涯 の 仕 事 と し て 信 じ ら れ な い ほ ど の 規 模 で あ る。
</p>

<h3>小 学 校 中 退 ・ 在 野 の 学 者</h3>

<p>
牧 野 は、 高 知 県 佐 川 町 (さ か わ ち ょ う) の 雑 貨 商 の 家 に 生 ま れ た。<br>
小 学 校 に 入 学 し た が、 2 年 で 中 退。 そ れ で も 独 学 で 植 物 学 を 学 び 続 け た。
</p>

<p>
26 歳 の 1888 年、 高 知 か ら 上 京 し、 東 京 帝 国 大 学 植 物 学 教 室 に 出 入 り す る よ う に な る。<br>
だ が、 「学 歴 が な い」 と い う 理 由 で、 何 度 も 教 室 か ら 排 除 さ れ そ う に な っ た。<br>
そ れ で も、 彼 の 研 究 の 実 力 を 認 め た 一 部 の 教 授 が 庇 護 し、 や が て 1912 年 (50 歳)、 東 京 帝 大 講 師 と な っ た。
</p>

<p>
そ し て 1939 年 (77 歳)、 つ い に <strong>東 京 帝 国 大 学 か ら 理 学 博 士 の 学 位 を 授 与</strong> さ れ た。<br>
小 学 校 中 退 で、 帝 国 大 学 の 理 学 博 士 ─ こ れ は 日 本 学 問 史 上 で も 稀 有 な 例 で あ る。
</p>

<h3>ヤ マ ト グ サ ─ 日 本 人 初 の 学 名 命 名</h3>

<p>
1884 年 (22 歳)、 牧 野 は 故 郷 高 知 の 山 で、 一 つ の 小 さ な 植 物 を 発 見 し た。<br>
帰 京 後 5 年 間、 文 献 と 標 本 を 照 合 し、 既 存 の ど の 植 物 と も 違 う 新 種 だ と 確 信。
</p>

<p>
1889 年、 彼 は こ れ を 雑 誌『植 物 学 雑 誌』 に 発 表 し、 学 名 を 「<strong>Theligonum japonicum</strong>」 (テ リ ゴ ヌ ム ・ ヤ ポ ニ ク ム)、 和 名 を 「<strong>ヤ マ ト グ サ</strong>」 (大 和 草) と し た。
</p>

<p>
こ れ が、 <strong>日 本 人 が 自 国 の 植 物 に 学 名 を つ け て 国 際 学 会 に 発 表 し た 最 初 の 例</strong> で あ る。<br>
そ れ ま で、 日 本 の 植 物 は す べ て、 シ ー ボ ル ト・ツ ン ベ ル グ ら 西 洋 人 に よ っ て 命 名 さ れ て き た。<br>
牧 野 の こ の 業 績 は、 「<strong>日 本 人 自 身 が、 自 国 の 自 然 を 学 問 す る</strong>」 時 代 を 開 い た 記 念 碑 と さ れ る。
</p>

<h3>「学 名」 と は 何 か</h3>

<p>
植 物・動 物 の 「学 名」 (が く め い) は、 ス ウ ェ ー デ ン の 植 物 学 者 <strong>カ ー ル・リ ン ネ</strong> (Carl Linnaeus, 1707–1778) が 確 立 し た 「<strong>二 名 法</strong>」 で 書 か れ る。
</p>

<p>
2 つ の ラ テ ン 語 で 構 成 さ れ る:
</p>
<ul>
<li><strong>属 名</strong> ─ 大 き な 分 類 (例: <em>Cerasus</em> = サ ク ラ 属)</li>
<li><strong>種 小 名</strong> ─ そ の 中 の 具 体 的 な 種 (例: <em>jamasakura</em> = ヤ マ ザ ク ラ)</li>
</ul>

<p>
こ う し て、 「<em>Cerasus jamasakura</em>」 が ヤ マ ザ ク ラ の 学 名。<br>
世 界 中 の 学 者 が、 言 語 を 超 え て こ の 名 で 同 じ 植 物 を 指 す こ と が で き る。
</p>

<p>
牧 野 富 太 郎 は、 こ の 学 名 を 約 1500 種 の 日 本 産 植 物 に 与 え、 日 本 の 自 然 を 国 際 学 界 に 紹 介 し た。
</p>

<h3>観 察 と 命 名 ─ 科 学 の 基 礎</h3>

<p>
牧 野 の 仕 事 か ら、 私 た ち は 「自 然 科 学 の 基 礎」 を 学 べ る。
</p>

<ol>
<li><strong>観 察</strong> ─ 自 然 を 真 摯 に、 細 部 ま で 見 る</li>
<li><strong>記 録</strong> ─ 見 た こ と を 絵 や 言 葉 で 残 す</li>
<li><strong>分 類</strong> ─ 既 知 の も の と 比 較 し、 似 て い る か 違 う か を 判 断</li>
<li><strong>命 名</strong> ─ 新 種 と わ か れ ば、 学 名 を つ け て 共 有 す る</li>
</ol>

<p>
こ の 4 つ の 作 業 が、 自 然 科 学 の <strong>「博 物 学」</strong> の 基 本 で あ る。<br>
そ し て、 こ の 基 本 を 一 人 の 人 間 が 60 年 以 上 続 け れ ば ─ 牧 野 の よ う な 業 績 が 生 ま れ る。
</p>

<h3>NHK 朝 ド ラ 「ら ん ま ん」</h3>

<p>
2023 年 4 月 ~ 9 月、 NHK 連 続 テ レ ビ 小 説 「<strong>ら ん ま ん</strong>」 が 放 送 さ れ、 牧 野 富 太 郎 が モ デ ル と さ れ た。<br>
主 演 は 神 木 隆 之 介。 放 送 期 間 中、 牧 野 の 業 績 が 改 め て 注 目 さ れ、 高 知 県 立 牧 野 植 物 園 や 練 馬 区 の 牧 野 記 念 庭 園 に は 多 く の 人 が 訪 れ た。
</p>

<p>
160 年 前 に 生 ま れ た 一 人 の 植 物 愛 好 家 の 生 涯 が、 2020 年 代 の 私 た ち に 改 め て 響 い た の は、 ど う し て だ ろ う か?<br>
そ れ は、 「自 分 の 好 き な こ と を、 学 歴 や 肩 書 き に 関 係 な く、 生 涯 追 い 続 け る」 と い う 生 き 方 が、 現 代 の 私 た ち に も 強 い 憧 れ を 抱 か せ る か ら か も し れ な い。
</p>

<h3>あ な た の 植 物 観 察 を 始 め よ う</h3>

<p>
今 度、 学 校 の 帰 り 道 で、 道 端 の 一 つ の 植 物 を、 じ っ く り 観 察 し て み よ う。
</p>

<ol>
<li><strong>葉 の 形</strong> ─ 細 長 い? 丸 い? ギ ザ ギ ザ?</li>
<li><strong>葉 の つ き 方</strong> ─ 互 い 違 い (互 生)? 対 (対 生)?</li>
<li><strong>花 の 数</strong> ─ 一 つ? い く つ も?</li>
<li><strong>花 の 色 と 形</strong></li>
<li><strong>茎 の 色 と 太 さ</strong></li>
</ol>

<p>
そ の 観 察 を ノ ー ト に 描 き、 名 前 を 図 鑑 (ま た は ス マ ホ の 植 物 同 定 ア プ リ) で 調 べ て み よ う。
</p>

<p>
牧 野 富 太 郎 が 22 歳 で ヤ マ ト グ サ を 見 つ け た よ う に、 君 も、 自 分 の 周 り の 自 然 か ら、 新 し い 発 見 が で き る か も し れ な い。
</p>
