<h3>本 文 (『雪 を 作 る 話』 中 谷 宇 吉 郎・抜 粋)</h3>

<blockquote>
<p>
昭 和 11 年 3 月 12 日。 こ の 日 は、 私 に と っ て 一 生 忘 れ ら れ な い 日 で あ っ た。
</p>

<p>
北 海 道 大 学 の 低 温 実 験 室 の 中、 兎 (う さ ぎ) の 毛 の 先 に、 小 さ な 雪 の 結 晶 が 育 っ て い く の を、 私 た ち は 食 い 入 る よ う に 見 つ め て い た。
</p>

<p>
こ れ は、 世 界 で 初 め て の 「人 造 雪」 の 結 晶 で あ っ た。<br>
雪 は、 自 然 が 上 空 で 作 る も の ─ そ う 信 じ ら れ て き た。 だ が、 私 た ち は そ れ を、 こ の 実 験 室 の 中 で、 人 間 の 手 で 作 り 出 し た の で あ る。
</p>

<p>
雪 の 結 晶 の 形 は、 実 に 多 様 で あ る。<br>
六 角 板 状 の も の、 六 角 柱 状 の も の、 樹 枝 状 (じ ゅ し じ ょ う ─ 木 の 枝 の よ う な) の も の、 そ し て 中 央 に 星 を 持 ち 周 り に 美 し い 装 飾 を 持 つ も の ─。<br>
こ の 違 い は、 ど こ か ら く る の か?
</p>

<p>
長 年 の 観 察 と 実 験 か ら、 私 は 次 の こ と を 確 信 し て い る:<br>
<strong>雪 の 結 晶 の 形 は、 そ の 雪 が 上 空 で 生 ま れ た と き の 「気 温」 と 「湿 度」 に よ っ て、 決 ま る。</strong>
</p>

<p>
例 え ば、 摂 氏 マ イ ナ ス 15 度 前 後 で、 湿 度 が 高 い と、 美 し い 樹 枝 状 結 晶 が 育 つ。<br>
マ イ ナ ス 5 度 前 後 で は、 板 状 の 結 晶 に な り や す い。<br>
温 度 と 湿 度 と が、 こ う し て 結 晶 の 形 を 「書 く」 の で あ る。
</p>

<p>
私 た ち の 実 験 室 で、 温 度 と 湿 度 を 様 々 に 変 え て み た。<br>
す る と、 自 然 の 雪 と 同 じ よ う な、 様 々 な 形 の 結 晶 が 育 っ た。<br>
こ れ で 私 た ち の 仮 説 は、 実 験 的 に 確 か め ら れ た こ と に な る。
</p>

<p>
そ し て、 こ こ か ら 重 要 な こ と が 言 え る。<br>
今 度、 空 か ら 落 ち て く る 雪 を 見 つ け た ら、 そ の 形 を よ く 観 察 し て ほ し い。<br>
そ の 形 か ら、 そ の 雪 が 上 空 数 千 メ ー ト ル で、 ど ん な 気 温 ・ 湿 度 の 中 で 生 ま れ た か が、 ほ ぼ わ か る の で あ る。
</p>

<p>
こ う 言 い 換 え て も よ い:<br>
<strong>雪 は 天 か ら 送 ら れ た 手 紙 で あ る。</strong>
</p>

<p>
私 た ち は 今、 そ の 手 紙 を 「読 む」 こ と が で き る よ う に な っ た の だ。
</p>
</blockquote>

<h3>解 説 ─ 「雪 は 天 か ら 送 ら れ た 手 紙」 と は</h3>

<p>
中 谷 宇 吉 郎 の こ の 名 言 は、 単 な る 比 喩 で は な い。<br>
そ れ は、 中 谷 が 何 年 も の 観 察 と 実 験 か ら 引 き 出 し た、 厳 密 な 科 学 的 真 理 で あ る。
</p>

<p>
雪 の 結 晶 の 形 は、 上 空 の 「気 温」 と 「湿 度」 で 決 ま る。<br>
だ か ら、 地 上 に 降 っ て く る 雪 の 形 を 見 れ ば、 そ の 雪 が 「ど ん な 上 空 で 生 ま れ た か」 が 読 め る。
</p>

<p>
雪 一 つ 一 つ が、 上 空 か ら 私 た ち に 「私 は こ う い う 場 所 で 生 ま れ ま し た」 と 知 ら せ る、 小 さ な 手 紙 だ と い う わ け で あ る。
</p>

<h3>世 界 初 の 人 造 雪 ─ 1936 年 3 月 12 日</h3>

<p>
中 谷 宇 吉 郎 は、 1932 年 に 北 海 道 大 学 の 教 授 と な り、 そ こ で 雪 の 研 究 を 始 め た。<br>
彼 は ま ず、 北 海 道 の 雪 を 顕 微 鏡 で 観 察 し、 膨 大 な 数 の 結 晶 を 写 真 に 撮 影 ・ 分 類 し た。 約 6000 枚 の 雪 結 晶 写 真 を 残 し た と 言 わ れ る。
</p>

<p>
そ し て 観 察 か ら の 仮 説 ─「結 晶 の 形 は 気 温 と 湿 度 で 決 ま る」 を 確 か め る た め、 低 温 実 験 室 で 「人 造 雪」 を 作 る こ と に 取 り 組 ん だ。
</p>

<p>
1936 年 3 月 12 日、 兎 の 毛 を 「核」 と し て 用 い る ア イ デ ア が 当 た り、 ね ら い 通 り の 結 晶 を 育 て る こ と に 成 功 し た。<br>
<strong>世 界 で 初 め て の 「人 造 雪 の 結 晶」</strong> ─ 中 谷 と 助 手 た ち は、 顕 微 鏡 を 覗 き 込 み な が ら、 ど ん な に 感 動 し た こ と だ ろ う。
</p>

<h3>雪 の 結 晶 の 多 様 性</h3>

<p>
中 谷 の 分 類 で は、 雪 の 結 晶 は 7 大 種 類 ・ 約 80 種 に 分 け ら れ る。
</p>

<ul>
<li><strong>板 状</strong> (い た じ ょ う) ─ 平 ら な 六 角 形</li>
<li><strong>柱 状</strong> (ち ゅ う じ ょ う) ─ 六 角 形 の 鉛 筆 の よ う</li>
<li><strong>樹 枝 状</strong> (じ ゅ し じ ょ う) ─ 木 の 枝 の よ う に 分 か れ た 美 し い 形</li>
<li><strong>不 規 則 樹 枝 状</strong></li>
<li><strong>角 板</strong> (か く ば ん) と <strong>樹 枝</strong> の 混 合</li>
<li><strong>砲 弾</strong> (ほ う だ ん) ─ 弾 丸 状</li>
<li><strong>あ ら れ ・ ひ ょ う ・ 雪 つ ぶ</strong></li>
</ul>

<p>
そ し て、 こ れ ら の 形 が、 そ れ ぞ れ ど ん な 気 温 ・ 湿 度 で 生 ま れ る か ─ こ れ を 図 に し た の が、 後 に 「<strong>中 谷 ダ イ ヤ グ ラ ム</strong>」 と 呼 ば れ る 有 名 な 図 で あ る。
</p>

<h3>科 学 の 道 筋 ─ 観 察 → 仮 説 → 実 験</h3>

<p>
中 谷 の 雪 の 研 究 は、 中 学 校 で 学 ぶ 「<strong>科 学 の 方 法</strong>」 の 教 科 書 の よ う な 流 れ で あ る。
</p>

<ol>
<li><strong>観 察</strong> ─ 自 然 の 雪 を 何 千 枚 も 撮 影 ・ 分 類</li>
<li><strong>仮 説</strong> ─ 「結 晶 の 形 は 気 温 と 湿 度 で 決 ま る」 と 推 測</li>
<li><strong>実 験</strong> ─ 低 温 実 験 室 で 人 造 雪 を 作 り、 条 件 を 変 え て 結 晶 を 観 察</li>
<li><strong>結 論</strong> ─ 仮 説 が 確 か め ら れ、 「雪 = 上 空 の 気 象 情 報」 が 言 え る</li>
</ol>

<p>
こ の 流 れ が、 ま さ に <strong>科 学 そ の も の</strong> で あ る。
</p>

<h3>師 ・ 寺 田 寅 彦 と の つ な が り</h3>

<p>
中 谷 宇 吉 郎 は、 物 理 学 者・随 筆 家 の <strong>寺 田 寅 彦</strong> (1878–1935) の 弟 子 で あ る。<br>
寺 田 は、 「茶 碗 の 湯」 「線 香 花 火」 な ど、 日 常 の 中 に 物 理 を 見 出 す 随 筆 を 書 い た 名 手 (本 サ イ ト chugakurika1 で 教 材 化)。
</p>

<p>
中 谷 は 寺 田 か ら、 「<strong>日 常 の 小 さ な 現 象 を、 科 学 の 目 で 観 察 す る</strong>」 と い う 姿 勢 を 受 け 継 い だ。<br>
