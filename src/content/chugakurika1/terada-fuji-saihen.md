<h3>1933 年 ・ ベ ラ ン ダ の 出 来 事</h3>

<p>
昭 和 8 年 (1933)、 寺 田 寅 彦 55 歳。 12 月、 自 宅 の ベ ラ ン ダ で、 突 然 ポ ン と い う 音 が し て、
何 か が 飛 ん で 来 た。 床 に 落 ち た の は ─ 藤 (フ ジ) の 種 だ っ た。
</p>

<p>
そ し て し ば ら く す る と、 ま た ポ ン、 ポ ン と 音 が 続 く。 ベ ラ ン ダ の 藤 棚 の 莢 (さ や) が、
一 斉 に は じ け て 種 を 飛 ば し て い る の だ。
</p>

<p>
寺 田 寅 彦 ─ 東 京 帝 大 物 理 学 教 授、 夏 目 漱 石 門 下 の 随 筆 家 ─ は、 こ の 観 察 を『藤 の 実』 と
し て 中 央 公 論 に 発 表 し た。 物 理 学 と 随 筆、 自 然 観 察 と 文 学 が 融 合 し た 名 篇 の 一 つ で あ る。
</p>

<h3>な ぜ 一 斉 に は じ け る の か</h3>

<p>
寺 田 が 観 察 し た の は、 「同 じ 藤 棚 の、 い く つ も の 莢 が、 偶 然 に も 同 じ 時 間 に 一 斉 に は じ け る」
現 象 だ っ た。 こ れ は な ぜ か?
</p>

<p>
莢 (さ や) は、 種 子 を 包 む マ メ 科 の 殻 で あ る。 秋 ~ 冬 に 乾 燥 す る と、 殻 の 二 枚 が 異 な る
方 向 に ね じ れ よ う と し て、 強 い 内 部 応 力 を 生 む。 一 定 の 限 界 を 超 え る と、 突 然 一 気 に
ね じ れ 切 っ て、 種 子 を 数 メ ー ト ル 弾 き 飛 ば す。
</p>

<p>
寺 田 は、 こ の 一 斉 性 を「気 温・湿 度 が 同 じ 棚 の 莢 を 同 時 に 限 界 へ 追 い 込 む か ら」 と 推 測 し た。
</p>

<h3>植 物 の「自 力 散 布」 ─ 種 を 飛 ば す 工 夫</h3>

<p>
植 物 は、 自 分 で 動 け な い 為、 種 を 親 か ら 離 す 工 夫 が 必 要 で あ る。 散 布 の 方 法 は い ろ い ろ ─
</p>

<ul>
  <li><strong>風 散 布</strong> ─ タ ン ポ ポ の 綿 毛、 カ エ デ の 翼</li>
  <li><strong>水 散 布</strong> ─ ヤ シ の 実、 蓮 の 種</li>
  <li><strong>動 物 散 布</strong> ─ ド ン グ リ (リ ス が 隠 す)、 葛 (く ず)・ヒ ッ ツ キ ム シ (動 物 に つ く)</li>
  <li><strong>自 力 散 布</strong> ─ フ ジ・ホ ウ セ ン カ・カ タ バ ミ (自 ら 弾 け る)</li>
</ul>

<p>
フ ジ の 自 力 散 布 は、 乾 燥 と 弾 力 を 利 用 し た「物 理 学 的 」 な 工 夫 で あ る ─ こ の 視 点 が、
物 理 学 者 寺 田 の 文 章 の 特 徴。
</p>

<h3>日 常 の 不 思 議 ─ 寺 田 寅 彦 の 視 点</h3>

<p>
寺 田 寅 彦 の 随 筆 の 魅 力 は、 「日 常 の 些 細 な 出 来 事 か ら、 科 学 の 大 き な 問 題 に 入 っ て い く」
と こ ろ に あ る。
</p>

<p>
本 サ イ ト で 既 出 の 寺 田 教 材 ─
</p>
<ul>
  <li>『茶 碗 の 湯』 ─ 茶 碗 の 湯 気 か ら 雲 ・ 気 象 へ</li>
  <li>『線 香 花 火』 ─ 火 花 か ら 化 学 ・ 物 理 へ</li>
  <li>『金 平 糖』 ─ 角 が 生 え る 不 思 議 か ら 結 晶 成 長 へ</li>
</ul>

<p>
全 て に 共 通 す る の は、 「身 の 回 り の 何 か が、 実 は 不 思 議 で あ る こ と に 気 付 く」 観 察 眼 だ。
</p>

<h3>結 び ─ 君 も「藤 の 実」 を 見 つ け よ う</h3>

<p>
寺 田 寅 彦 か ら 学 ぶ こ と は、 高 度 な 知 識 で は な く、 「日 常 を 注 意 深 く 観 察 す る 姿 勢」 で あ る。
</p>

<p>
今 日 学 校 か ら 帰 る 道 で、 何 か 一 つ「不 思 議」 を 見 つ け て み よ う。 そ し て「な ぜ?」 と 問 う て み よ う。
そ れ が 科 学 の 入 り 口 で あ る。
</p>
