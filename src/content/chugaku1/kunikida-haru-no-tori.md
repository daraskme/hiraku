<h3>あらすじ</h3>

<p>
明 治 末 期、 九 州 の あ る 田 舎 町。「私」 は 中 学 校 の 教 員 と し て 赴 任 し、 城 跡 の 古 い 家 に 下 宿 し て い た。
</p>

<p>
あ る 春 の 日、 城 跡 を 散 歩 し て い る と、 一 人 の 少 年 と 出 会 う。<br>
名 前 は <strong>六 蔵</strong> (ろ く ぞ う)。 12〜13 歳 ぐ ら い の 少 年 だ が、 言 葉 を う ま く 話 す こ と が で き ず、 周 り の 大 人 か ら は 「魯 鈍 (ろ ど ん)」 と 呼 ば れ て い た ─ 今 で 言 う 「知 的 障 害」 を 持 つ 子 ど も で あ っ た。
</p>

<p>
だ が 六 蔵 に は、 一 つ 不 思 議 な 才 能 が あ っ た ─ <strong>鳥 の 鳴 き 真 似 が 非 常 に 上 手 い</strong> こ と。<br>
雀、 鶯 (う ぐ い す)、 雲 雀 (ひ ば り)、 山 鳩 (や ま ば と) ─ あ ら ゆ る 鳥 の 声 を、 ま っ た く そ っ く り に 真 似 で き る。
</p>

<p>
「私」 は、 こ の 不 思 議 な 少 年 と 仲 良 く な っ た。<br>
散 歩 の た び に、 六 蔵 と 城 跡 で 出 会 い、 鳥 の 真 似 を 聞 か せ て も ら う。
</p>

<p>
あ る 日、 六 蔵 が こ う 言 っ た ─<br>
「<strong>お い ら も い つ か 鳥 に な っ て、 空 を 飛 び た い</strong>」 と。
</p>

<p>
「私」 は 笑 い な が ら、 「人 間 は 鳥 に な れ な い よ」 と 答 え た。<br>
だ が 六 蔵 は、 真 面 目 な 顔 で、 こ う 言 い 続 け た:<br>
「いや、 お い ら、 い つ か 必 ず 鳥 に な る ん だ」 と。
</p>

<p>
そ し て、 ま た 別 の 日 ─ 「私」 が 城 跡 を 訪 れ る と、 六 蔵 は い な か っ た。<br>
村 人 が 騒 い で い た。「六 蔵 が 城 跡 か ら 飛 び 降 り て 死 ん だ」 と。<br>
彼 は、 城 跡 の 一 番 高 い 石 垣 の 上 か ら、 両 手 を 鳥 の よ う に 広 げ て、 飛 び 降 り た の だ っ た。
</p>

<p>
「私」 は 城 跡 に 立 ち、 し ば ら く 動 け な か っ た。<br>
そ し て 思 っ た:「<strong>六 蔵 は、 本 当 に 鳥 に な っ た の か も し れ な い</strong>」 と。
</p>

<h3>解 説 ─ 静 か な る 抒 情</h3>

<p>
「春 の 鳥」 は、 国 木 田 独 歩 が 33 歳 で 書 い た 短 編 で あ る。<br>
彼 が こ の 物 語 の 着 想 を 得 た の は、 22 歳 の 頃 (1893–1894 年)、 大 分 県 佐 伯 (さ え き) で 英 語 教 師 を し て い た 経 験 か ら と さ れ る。
</p>

<p>
独 歩 の 文 体 の 特 徴 は、 「<strong>静 か さ</strong>」 で あ る。<br>
激 し い 感 情 を 直 接 描 か ず、 自 然 の 風 景 と 人 物 の 行 動 を 淡 々 と 描 く。<br>
そ し て そ の 「静 か さ」 の 中 に、 深 い 哀 し み と 美 し さ が 浮 か び 上 が る。
</p>

<p>
「春 の 鳥」 の 結 末 ─ 六 蔵 の 死 ─ も、 ド ラ マ チ ッ ク に は 描 か れ な い。<br>
村 人 の 騒 ぎ を 聞 い て、 「私」 は 立 ち 尽 く す。 そ し て 「六 蔵 は 本 当 に 鳥 に な っ た の か も し れ な い」 と 思 う ─ そ れ だ け で あ る。<br>
だ が こ の 抑 制 さ れ た 文 体 が、 か え っ て 読 者 の 心 に 深 く 響 く。
</p>

<h3>「鳥 に な り た い」 と い う 願 い</h3>

<p>
六 蔵 の 「鳥 に な り た い」 と い う 願 い は、 何 を 象 徴 し て い る の か?
</p>

<p>
言 葉 で の コ ミ ュ ニ ケ ー シ ョ ン が 難 し い 六 蔵 に と っ て、 鳥 の 声 だ け が、 彼 が 「上 手 く 出 来 る こ と」 だ っ た。<br>
そ し て 鳥 は、 言 葉 を 持 た ず、 自 由 に 空 を 飛 ぶ。 ─ 六 蔵 が 望 ん だ 「自 由」 が、 そ こ に あ っ た。
</p>

<p>
彼 の 願 い は、 「<strong>こ の 世 界 で『普 通』 と 言 わ れ る 人 々 の 仲 間 に 入 り た い</strong>」 で は な か っ た。<br>
そ う で は な く、 「<strong>自 分 が 自 分 の ま ま で 自 由 に 生 き ら れ る、 別 の 存 在 に な り た い</strong>」 だ っ た の だ。
</p>

<p>
こ の 願 い の 切 な さ が、 物 語 全 体 を 包 ん で い る。
</p>

<h3>明 治 37 年 と 現 代 の 「違 い」</h3>

<p>
明 治 37 年 (1904) ─ こ の 時 代、 知 的 障 害 や 発 達 障 害 と い う 概 念 は ま だ な か っ た。<br>
六 蔵 の よ う な 子 は、 「魯 鈍」 「白 痴」 と 呼 ば れ、 多 く は 偏 見 と 無 理 解 の 中 で 生 き た。
</p>

<p>
だ が 国 木 田 独 歩 は、 こ の 物 語 で、 六 蔵 を そ う い う 偏 見 の 視 線 で は な く、 <strong>一 人 の 「鳥 を 愛 す る 子 ど も」</strong> と し て 描 い た。<br>
彼 の 不 思 議 な 才 能、 純 粋 な 心、 そ し て 切 な い 願 い を ─ 静 か に、 だ が 確 か に 描 い た の で あ る。
</p>

<p>
2026 年 ─ 現 代 の 私 た ち は、 障 害 を 持 つ 人 へ の 理 解 を 深 め つ つ あ る。<br>
だ が、 「人 と 違 う」 こ と へ の 偏 見 が、 す べ て な く な っ た わ け で は な い。
</p>

<p>
独 歩 が 122 年 前 に 描 い た 六 蔵 の 物 語 は、 今 で も、 「<strong>違 い を 見 る 眼 差 し</strong>」 を 私 た ち に 静 か に 問 い か け て く る。
</p>

<h3>全 文 を 読 ん で み よ う</h3>

<p>
「春 の 鳥」 は、 青 空 文 庫 で 全 文 が 読 め る、 4000 字 程 度 の 短 編 で あ る。<br>
ぜ ひ、 一 度 通 読 し、 六 蔵 と 「私」 の 静 か な 春 の 物 語 を、 君 自 身 の 心 で 受 け 止 め て み て ほ し い。
</p>

<p>
そ し て、 六 蔵 が 鳥 に な っ て 飛 び 立 つ 瞬 間 を、 君 の 想 像 で 描 い て み て ほ し い。
</p>
