<h3>「土」 を 読 む</h3>

<p>
た っ た 4 行 の 詩。 そ の 中 に、世界 が 広 が る。
</p>

<p>
1 行目 ─ <strong>「蟻 が」</strong><br>
あ る、小 さ な、地 を 這 う 一匹 の 蟻。
</p>

<p>
2 行目 ─ <strong>「蝶 の 羽 を ひ い て 行 く」</strong><br>
そ の 蟻 が、自分 よ り も ず っ と 大 き い、薄 く 透 け た 蝶 の 羽 を、ひ き ず り な が ら 運 ん で い る。
</p>

<p>
3 行目 ─ <strong>「あ あ」</strong><br>
詩 人 の、思 わ ず 漏 れ た 感 嘆。
</p>

<p>
4 行目 ─ <strong>「ヨ ッ ト の や う だ」</strong><br>
そ し て、そ の 姿 を、ヨ ッ ト に 喩 え た。
</p>

<p>
こ の 喩 え は 何 を 意 味 す る か?<br>
「蝶 の 羽」 は、ヨ ッ ト の 「白 い 帆 (ほ)」 に 似 て い る。 風 を は ら ん で、ふ ん わ り と 立 ち 上 が る。<br>
「蟻」 は、ヨ ッ ト の 「船 体 (せ ん た い)」 に 当 た る。<br>
そ し て、蟻 が 地面 を 動 く 様子 が、ヨ ッ ト が 海 を 滑 る 様子 と 重 な る。
</p>

<p>
だ が こ の 喩 え の 凄 さ は、 そ れ だ け で は な い。
</p>

<ul>
<li><strong>大 と 小 の 反転</strong> ─ 蟻 は 1 cm に 満 た な い 微小 な 存在。 ヨ ッ ト は 何 メ ー ト ル も の 大 き さ。 だ が 詩 は、こ の 二 つ を 重 ね る こ と で、<strong>「小 さ な 蟻 の 世界 が、海 の 広 が り を 抱 い て い る」</strong> こ と を 示 す。</li>
<li><strong>地 と 海 の 反転</strong> ─ 蟻 が 動 く の は 地面。 ヨ ッ ト が 動 く の は 海。 詩 は、地面 を 「海」 に 変 え て し ま う。</li>
<li><strong>生 と 死 の 重 な り</strong> ─ 蝶 の 羽 は、 死 ん だ 蝶 か ら 取 れ た も の。 蟻 は そ れ を 運 ぶ。 死 ん だ も の (蝶 の 羽) が、生 き た も の (蟻) に よ っ て、新 し い 命 (帆) を 与 え ら れ る。</li>
</ul>

<p>
た っ た 4 行 で、こ れ ほ ど 多 く の こ と を 言 え る ─ そ れ が 「短詩」 の 力 で あ る。
</p>

<h3>「春 の 岬」 を 読 む</h3>

<p>
「<strong>春 の 岬、旅 の を は り の 鴎 ど り、浮 き つ つ 遠 く な り に け る か も</strong>」
</p>

<p>
こ ち ら は 3 行 の 短詩。 短歌 の 5・7・5・7・7 の リ ズ ム を 思 わ せ る 部分 も あ る が、 完全 な 短歌 で は な い。
</p>

<p>
情景 ─<br>
春 の あ る 日、私 は 海 に 突 き 出 た 岬 に 立 っ て い る。<br>
は る か な 旅 の 「は て」 ─ 終 着 点 に 来 た よ う な 感覚。<br>
頭 上 に は 鴎 (か も め) が 飛 ん で い る。 そ の 鴎 が、 海 の 上 を 浮 か び な が ら、 だ ん だ ん と 遠 ざ か っ て 行 く。
</p>

<p>
「<strong>な り に け る か も</strong>」 は、 古典 の 感 嘆 ・ 詠嘆 の 表現。「な っ て し ま っ た こ と よ」 と い う、 切 な さ を 含 ん だ 響 き。
</p>

<p>
こ の 詩 は、 旅 と、 別 れ と、 春 の 海 の 静 け さ を、 わ ず か 3 行 で 描 き き っ て い る。
</p>

<h3>三 好 達 治 と い う 詩 人</h3>

<p>
三 好 達 治 (1900–1964) は、 大 阪 で 生 ま れ、 東京 帝 国 大学 で フ ラ ン ス 文学 を 学 ん だ。<br>
ボ ー ド レ ー ル ・ ヴ ェ ル レ ー ヌ ら の フ ラ ン ス 象 徴 派 の 詩 か ら 強 い 影響 を 受 け な が ら、<strong>日本 の 季 節 感 ・ 自然 観</strong> を 短 い 詩形 で 表現 す る 独自 の 詩 を 開 拓 し た。
</p>

<p>
彼 の 詩 は、北 原 白 秋 や 萩 原 朔 太 郎 の よ う な 重 厚 さ よ り も、<br>
立 原 道 造 (たちはら みちぞう) や 中 原 中 也 と 共 通 す る <strong>「軽 や か な 抒情」</strong> が 特 徴 で あ る。
</p>

<p>
『測量船』 か ら 始 ま り、 戦中・戦後 を 経 て、 数 多 く の 詩集 を 残 し た。<br>
本詩 「土」 「春 の 岬」 は、 い ま も 中学・高校 の 教 科書 に 載 る、 三 好 達 治 の 代 表作 で あ る。
</p>

<h3>自分 で も 書 い て み よ う</h3>

<p>
4 行 詩 を、自分 で 書 い て み よ う。<br>
ル ー ル は こ う:
</p>

<ol>
<li>4 行 以 内 (3 行 で も OK)</li>
<li>「~ の よ う だ」 と い う 直喩 を 1 つ 入 れ る</li>
<li>身 の 回 り の 小 さ な 出 来 事 を 取 り 上 げ る</li>
</ol>

<p>
例:<br>
「鉛筆 が / ノ ー ト の 上 を 走 っ て 行 く / ま る で / 雪 の 上 の ス キ ー の よ う だ」
</p>

<p>
あ な た に も、 三 好 達 治 の よ う な 詩 が 書 け る。<br>
「あ あ」 と 思 わ ず 声 が 出 る 瞬間 を、 4 行 に 閉 じ 込 め て み よ う。
</p>
