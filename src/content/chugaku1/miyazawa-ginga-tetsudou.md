<h3>本 文 (『銀 河 鉄 道 の 夜』 宮 沢 賢 治・冒 頭 抜 粋)</h3>

<blockquote>
<p>
「で は み な さ ん は、 そ う い う ふ う に 川 だ と 云 わ れ た り、 乳 の 流 れ た あ と だ と 云 わ れ た り し て い た こ の ぼ ん や り と 白 い も の が ほ ん と う は 何 か ご 承 知 で す か」 ─ 先 生 は、 黒 板 に 吊 し た 大 き な 黒 い 星 座 の 図 の、 上 か ら 下 へ 白 く け ぶ っ た 銀 河 帯 の よ う な と こ ろ を 指 し な が ら、 み ん な に 問 い を か け ま し た。
</p>

<p>
カ ム パ ネ ル ラ が 手 を 挙 げ ま し た。 そ れ か ら 四 五 人 手 を 挙 げ ま し た。 ジ ョ バ ン ニ も 手 を 挙 げ よ う と し て、 急 い で そ の ま ま や め ま し た。 た し か に あ れ が み ん な 星 だ と、 い つ か 雑 誌 で 読 ん だ の で し た が、 こ の ご ろ は ジ ョ バ ン ニ は ま る で 毎 日 教 室 で も 朝 で も 夢 う つ つ で 本 さ え よ く 読 め な い の で、 何 で も か ん で も は っ き り し な い の で し た。
</p>

<p>
と こ ろ が 先 生 は は や く そ れ を 見 つ け た の で し た。
</p>

<p>
「ジ ョ バ ン ニ さ ん。 あ な た は 分 か っ て い る の で し ょ う」
</p>

<p>
ジ ョ バ ン ニ は 勢 い よ く 立 ち 上 が り ま し た が、 立 っ て み る と も う は っ き り と そ れ を 答 え る こ と が で き な い の で し た。 ザ ネ リ が 前 の 席 か ら ふ り 返 っ て、 ジ ョ バ ン ニ を 見 て く す っ と 笑 い ま し た。 ジ ョ バ ン ニ は な ぜ か 胸 が 一 杯 に な っ て、 な ん に も 云 え ず に 立 っ て ゐ ま し た。
</p>

<p>
先 生 が ま た 云 い ま し た。
</p>

<p>
「大 き な 望 遠 鏡 で 銀 河 を よ っ く 調 べ る と、 銀 河 は だ い た い 何 で し ょ う」
</p>

<p>
や っ ぱ り 星 だ と ジ ョ バ ン ニ は 思 い ま し た が、 こ ん ど も や は り す ぐ に 答 え る こ と が で き ま せ ん で し た。
</p>

<p>
先 生 は し ば ら く 困 っ た よ う す で し た が、 眼 を カ ム パ ネ ル ラ の 方 へ 向 け て、
</p>

<p>
「で は カ ム パ ネ ル ラ さ ん」 と 名 指 し ま し た。 す る と あ ん な に 元 気 に 手 を 挙 げ た カ ム パ ネ ル ラ が、 や は り も じ も じ 立 ち 上 が っ た ま ま、 や は り 答 え が で き ま せ ん で し た。
</p>
</blockquote>

<h3>解 説 ─ 「未 完 の 名 作」</h3>

<p>
「銀 河 鉄 道 の 夜」 は、 宮 沢 賢 治 が 1924 年 (大 正 13 年) 頃 か ら 書 き 始 め、 1933 年 (昭 和 8 年) に 結 核 で 亡 く な る ま で、 何 度 も 推 敲 を 重 ね た 物 語 で あ る。
</p>

<p>
で も、 賢 治 は 結 局、 こ の 物 語 を <strong>完 成 さ せ る こ と が で き ず に 死 ん だ</strong>。<br>
現 在 読 ま れ て い る の は、 死 後 に 弟 の 清 六 が 整 理 し た 第 四 次 稿 (推 定 1931 年 頃) で あ る。
</p>

<p>
そ し て こ の 物 語 は、 賢 治 没 後、 1934 年 に 『宮 沢 賢 治 全 集』 で 初 め て 世 に 出 た。<br>
生 前 の 賢 治 は、 ほ と ん ど 無 名 の 詩 人・童 話 作 家 だ っ た。 だ が 死 後、 そ の 仕 事 が 日 本 文 学 史 上 で 最 高 評 価 を 受 け る こ と に な る。
</p>

<h3>あ ら す じ (全 編 を 通 し て)</h3>

<p>
ジ ョ バ ン ニ は、 父 が 漁 に 出 て 行 方 不 明、 母 は 病 気 と い う 家 庭 の 中、 活 字 拾 い の 仕 事 を し な が ら 学 校 に 通 う 少 年 で あ る。<br>
ク ラ ス で は ザ ネ リ ら か ら い じ め ら れ、 唯 一 の 親 友 カ ム パ ネ ル ラ と だ け が 心 を 通 わ せ る。
</p>

<p>
あ る 夏 の 夜、 銀 河 祭 り の 日。<br>
ジ ョ バ ン ニ は 病 気 の 母 の た め に 牛 乳 を 取 り に 行 こ う と し て 街 に 出 た。<br>
途 中、 丘 の 上 で 疲 れ て 横 に な っ た 彼 は、 突 然 「<strong>銀 河 ス テ ー シ ョ ン!</strong>」 と い う 声 を 聞 き、 不 思 議 な 銀 河 鉄 道 に 乗 り 込 ん で い た。
</p>

<p>
車 内 に は、 親 友 カ ム パ ネ ル ラ も 乗 っ て い た。<br>
二 人 は 銀 河 を 走 る 列 車 の 中 で、 北 十 字、 鳥 捕 り、 タ イ タ ニ ッ ク 号 の 遭 難 者 ら、 様 々 な 乗 客 と 出 会 う。<br>
そ し て、 銀 河 の 美 し く 不 思 議 な 風 景 を 目 撃 し て い く。
</p>

<p>
や が て カ ム パ ネ ル ラ は、 「お か あ さ ん が こ ち ら に い る」 と 言 っ て、 列 車 か ら 消 え る。<br>
ジ ョ バ ン ニ が 目 を 覚 ま す と、 そ こ は 元 の 丘 の 上。<br>
急 い で 街 に 戻 る と、 川 で 「カ ム パ ネ ル ラ が ザ ネ リ を 助 け よ う と し て 溺 死 し た」 と い う 知 ら せ が 伝 え ら れ て い た。
</p>

<p>
─ つ ま り、 ジ ョ バ ン ニ が 乗 っ た の は、 <strong>死 後 の カ ム パ ネ ル ラ を 黄 泉 (よ み) へ 送 る 列 車</strong> だ っ た の で あ る。
</p>

<h3>「銀 河 鉄 道」 と い う 象 徴</h3>

<p>
「銀 河 鉄 道」 と は、 何 を 象 徴 し て い る の か?
</p>

<ul>
<li><strong>死 と 旅 立 ち</strong> ─ カ ム パ ネ ル ラ を、 こ の 世 か ら あ の 世 へ 運 ぶ 列 車</li>
<li><strong>友 情 の 別 れ</strong> ─ ジ ョ バ ン ニ と カ ム パ ネ ル ラ が 最 後 に 共 に 過 ご す 時 間</li>
<li><strong>宇 宙 へ の 憧 れ</strong> ─ 銀 河 と い う 広 大 な 宇 宙 へ の 賢 治 の 想 像 力</li>
<li><strong>「ほ ん と う の 幸 い」</strong> ─ 物 語 の 中 で ジ ョ バ ン ニ が 求 め る 究 極 の テ ー マ</li>
</ul>

<p>
賢 治 自 身、 25 歳 の と き に 最 愛 の 妹 ト シ を 病 で 亡 く し て い る。<br>
そ の 喪 失 体 験 が、 こ の 物 語 の 「友 を 失 う 痛 み」 の 背 後 に あ る と 言 わ れ る。
</p>

<h3>賢 治 の 言 葉 「ほ ん と う の 幸 い」</h3>

<p>
物 語 の 中 で 何 度 も 繰 り 返 さ れ る フ レ ー ズ ─ そ れ が 「<strong>ほ ん と う の 幸 い</strong>」 で あ る。
</p>

<blockquote>
ジ ョ バ ン ニ「け れ ど も、 ほ ん と う の 幸 い っ て、 一 体 何 だ ろ う」<br>
カ ム パ ネ ル ラ「僕 わ か ら な い」<br>
ジ ョ バ ン ニ「僕 た ち し っ か り や ら う ね え」<br>
カ ム パ ネ ル ラ「あ あ。 き っ と や る よ」
</blockquote>

<p>
こ れ が、 二 人 の 最 後 の 会 話 と な る。<br>
や が て カ ム パ ネ ル ラ は 消 え、 ジ ョ バ ン ニ は 一 人 残 さ れ る。<br>

<h3>映 画 化 と そ の 後</h3>

<p>

<h3>全 編 を 読 ん で み よ う</h3>

<p>
今 回 は 冒 頭 だ け を 紹 介 し た。<br>
ぜ ひ、 青 空 文 庫 で 全 編 を 読 ん で ほ し い。
</p>

<p>
銀 河 鉄 道 に 乗 り、 ジ ョ バ ン ニ と カ ム パ ネ ル ラ の 旅 を 共 に し、 そ し て 最 後 に、 「ほ ん と う の 幸 い」 と は 何 か を、 君 自 身 の 心 で 問 う て み よ う。
</p>
