## メ ー ト ル 法 ─ 世 界 共 通 の も の さ し

### 1. 単 位 が バ ラ バ ラ だ っ た 時 代

今 か ら 250 年 ほ ど 前、 ヨ ー ロ ッ パ で は、 長 さ や 重 さ の 単 位 が、 国 や 町 ご と に バ ラ バ ラ だ っ た。

例 え ば、 当 時 の フ ラ ン ス で は、 「1 ピエ (足 の 長 さ が も と)」 と い う 単 位 が あ っ た が、 そ の 長 さ は 町 に よ っ て 違 っ て い た。 パ リ の 1 ピエ と、 と な り の 町 の 1 ピエ は、 長 さ が 違 う。 単 位 の 種 類 は、 フ ラ ン ス だ け で 25 万 種 類 も あ っ た と 言 わ れ る。

こ れ で は、 と て も 困 る。

「と な り の 町 で『布 を 10 ピエ』 買 っ た ら、 自 分 の 町 に 帰 る と 長 さ が 違 う」「商 人 が、 単 位 の 違 い を 利 用 し て ご ま か す」 ─ そ ん な こ と が 日 常 的 に 起 こ っ て い た。

***

### 2. フ ラ ン ス 革 命 ─ 「み ん な に 平 等 な も の さ し を」

1789 年、 フ ラ ン ス 革 命 が 起 こ っ た。 「す べ て の 人 は 平 等 だ」 と い う 理 念 を か か げ た 革 命 の 中 で、 人 々 は こ う 考 え た。

「単 位 も、 す べ て の 人 に 平 等 で あ る べ き だ。 王 様 や 特 定 の 町 で は な く、 だ れ に と っ て も 同 じ、 ひ と つ の も の さ し を つ く ろ う。」

で は、 何 を 基 準 に す れ ば、 全 人 類 に 公 平 な の か。 科 学 者 た ち は、 思 い 切 っ た 答 え を 出 し た。

「地 球 そ の も の を、 も の さ し に し よ う。」

***

### 3. 地 球 を も の さ し に す る

1791 年、 フ ラ ン ス の 科 学 者 た ち は、 1 メ ー ト ル を こ う 決 め た。

「地 球 の 北 極 か ら 赤 道 ま で の 距 離 (子 午 線) の、 1000 万 分 の 1 を、 1 メ ー ト ル と す る。」

地 球 の 大 き さ な ら、 ど の 国 の 人 に と っ て も 同 じ だ。 だ れ も 文 句 を 言 え な い、 公 平 な 基 準 で あ る。

し か し、 そ の た め に は、 地 球 の 子 午 線 を 正 確 に 測 ら な け れ ば な ら な い。 二 人 の 科 学 者、 デ ュ ラ ン ブ ル と メ シ ャ ン が、 革 命 の 混 乱 の 中、 7 年 を か け て フ ラ ン ス を 縦 断 し、 子 午 線 を 測 量 し た。

そ し て 1799 年、 そ の 結 果 を も と に、 白 金 で「1 メ ー ト ル の も の さ し (メ ー ト ル 原 器)」 が つ く ら れ た。 こ れ が「世 界 の 1 メ ー ト ル の 基 準」 と な っ た。

***

### 4. 十 進 法 ─ 計 算 し や す い し く み

メ ー ト ル 法 の も う 一 つ の 大 発 明 は、 「す べ て 10 倍 ご と」 に し た こ と だ。

   1 km = 1000 m
   1 m = 100 cm
   1 cm = 10 mm

10 倍、 100 倍、 1000 倍 ─ す べ て が 十 進 法 (10 を も と に す る 数 え 方) で つ な が っ て い る。 だ か ら、 単 位 を 変 え る 計 算 が、 位 を ず ら す だ け で で き る。 と て も 簡 単 だ。

昔 の 単 位 は こ う で は な か っ た。 例 え ば イ ギ リ ス の 単 位 で は、 「1 フ ィ ー ト = 12 イ ン チ」「1 ヤ ー ド = 3 フ ィ ー ト」「1 マ イ ル = 1760 ヤ ー ド」 と、 数 が バ ラ バ ラ で、 計 算 が と て も 面 倒 だ っ た。

***

### 5. 世 界 へ、 そ し て 日 本 へ

メ ー ト ル 法 は、 そ の 公 平 さ と 便 利 さ か ら、 し だ い に 世 界 中 に 広 ま っ た。 1875 年、 17 か 国 が「メ ー ト ル 条 約」 を 結 び、 国 際 的 な 単 位 と し て 認 め ら れ た (日 本 も 1885 年 に 加 盟)。

日 本 で は 昔、 長 さ を「尺 (し ゃ く)」、 重 さ を「貫 (か ん)」 で 表 す「尺 貫 法」 を 使 っ て い た。 明 治 以 降 に メ ー ト ル 法 を 導 入 し、 1959 年 (昭 和 34 年) に、 取 引 や 証 明 に は メ ー ト ル 法 を 使 う こ と が 義 務 づ け ら れ た。

(今 で も「畳 一 畳」「一 坪」「着 物 の 反 物」 な ど に、 昔 の 尺 貫 法 の 名 残 が 残 っ て い る。)

***

### 6. 君 の 定 規 の 中 の 地 球

現 在 で は、 1 メ ー ト ル は さ ら に 精 密 に、 「光 が 真 空 中 を、 1 秒 の 約 3 億 分 の 1 の 時 間 に 進 む 距 離」 と 定 義 さ れ て い る。 子 午 線 の 測 量 よ り も、 ず っ と 正 確 だ。

し か し、 そ の 出 発 点 は、 フ ラ ン ス 革 命 の 中 で 人 々 が 夢 見 た「全 人 類 に 公 平 な、 地 球 を 基 準 と し た も の さ し」 で あ っ た。

君 が 算 数 の 時 間 に 使 う 30 cm の 定 規。 そ の 1 cm、 1 mm の 中 に、 「地 球 の 大 き さ」 と「す べ て の 人 を 平 等 に」 と い う 250 年 前 の 願 い が、 こ め ら れ て い る の で あ る。
