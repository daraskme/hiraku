<h3>一・十・百・千 ─ 君 が も う 知 っ て い る 数</h3>

<p>
小 学 1 年 で、 君 は 1 ~ 100 の 数 を 学 ん だ。 小 学 2 年 で、 100 ~ 10000 の 数 を 学 ん だ。
そ し て 小 学 4 年 で、 「億 (お く)」 と い う 大 き な 数 が 登 場 す る。
</p>

<p>
「一・十・百・千・万・億・兆 …」 ─ こ の 階 段 を、 君 は ど こ ま で 知 っ て い る?
今 日 は 江 戸 期 の ベ ス ト セ ラ ー『塵 劫 記』 (じ ん こ う き) の 助 け を 借 り て、
「無 量 大 数」 と い う 信 じ ら れ な い ほ ど 大 き な 数 ま で 旅 を し よ う。
</p>

<h3>『塵 劫 記』 ─ 江 戸 の ベ ス ト セ ラ ー 算 数 書</h3>

<p>
寛 永 4 年 (1627)、 京 都 の 算 学 者 吉 田 光 由 (よ し だ み つ よ し ・ 1598–1672) が、 一 冊 の
算 数 入 門 書 を 出 版 し た。 題 名 は『塵 劫 記』。
</p>

<p>
こ の 本 は 江 戸 期 中 ・ 大 ベ ス ト セ ラ ー に な り、 寺 子 屋・家 庭 で「子 ど も の 算 数 教 科 書」 と し て
使 わ れ た。 鶴 亀 算・旅 人 算・鼠 算 等、 本 サ イ ト sansu5 で 紹 介 し て き た 教 材 も 全 て こ の 本 か ら。
</p>

<p>
『塵 劫 記』 の 冒 頭 に は、 数 の 単 位 表 が 載 っ て い る ─ 一 か ら 始 ま っ て、 無 量 大 数 ま で 80 段 階。
</p>

<h3>1 億 と は ど の く ら い?</h3>

<p>
「1 億」 と は、 想 像 し に く い 数。 イ メ ー ジ し や す く す る た め に ─
</p>

<ul>
  <li>日 本 の 人 口 ≈ 1 億 2300 万 人</li>
  <li>1 億 秒 = 約 3 年 2 ヶ 月</li>
  <li>1 億 円 ─ 1 万 円 札 で 1 万 枚 (積 む と 1 m)</li>
  <li>地 球 の 体 積 = 約 1 兆 km³ (1 兆 = 1 万 億)</li>
</ul>

<h3>1 兆 と は ど の く ら い?</h3>

<p>
「1 兆」 = 1 億 × 1 万 = 1,000,000,000,000 (0 が 12 個)。 さ ら に 大 き い ─
</p>

<ul>
  <li>日 本 の 国 家 予 算 ≈ 110 兆 円 (2024 年)</li>
  <li>1 兆 円 を 1 円 玉 で 積 む と ─ 1 兆 × 1.5 mm = 1.5 億 m = 15 万 km (地 球 か ら 月 ま で の 距 離 38 万 km の 約 4 割)</li>
  <li>銀 河 系 の 星 の 数 ≈ 2000 億 ~ 4000 億</li>
</ul>

<h3>無 量 大 数 ─ 想 像 を 絶 す る 大 き さ</h3>

<p>
『塵 劫 記』 の 最 大 単 位 は「無 量 大 数 (む り ょ う た い す う)」 = 10⁶⁸ (0 が 68 個)。 仏 教 の
経 典 か ら 取 ら れ た 名 前 で、 「無 量」(計 り 知 れ な い) + 「大 数」 を 意 味 す る。
</p>

<p>
ど の く ら い 大 き い か? 宇 宙 全 体 の 原 子 の 数 ≈ 10⁸⁰ と 推 定 さ れ る。 無 量 大 数 = 10⁶⁸ は、
ほ ぼ 宇 宙 全 体 を 数 え 上 げ ら れ る 大 き さ で あ る。
</p>

<p>
こ ん な 数 を 日 常 で 使 う 機 会 は な い。 で も、 江 戸 期 の 数 学 者 が こ の 数 を「想 像 し て 名 前 を 付 け た」
こ と そ の も の が、 人 間 の 知 性 の 美 し さ で あ る。
</p>

<h3>結 び ─ 数 の 世 界 を 旅 し よ う</h3>

<p>
1・10・100・1000 ─ 君 の 身 近 な 数 か ら 始 ま っ て、 万・億・兆 ─ 国 の 予 算 や 銀 河 の 星 の 数 ─
京・垓 ─ そ し て 無 量 大 数 ─ 宇 宙 の 全 て ─ ま で。
</p>

<p>
数 学 の 楽 し さ の 一 つ は、 「想 像 を 超 え る 大 き さ ・ 小 さ さ」 と 出 会 う こ と。 江 戸 の 子 ど も た ち も、
こ の 数 の 階 段 を 楽 し ん で 覚 え た。 君 も、 こ の 楽 し み を 体 験 し て 欲 し い。
</p>
