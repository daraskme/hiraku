## ハ ノ イ の 塔 ─ 1883 年 か ら の パ ズ ル

### 1. 3 本 の 柱 と 円 盤

机 の 上 に、 3 本 の 柱 が 立 っ て い る と 想 像 し て み よ う。 名 前 を A、 B、 C と し よ う。

A の 柱 に は、 大 き さ の 違 う 円 盤 が、 大 き い も の が 下 に な っ て 重 な っ て い る。 例 え ば、 3 枚 の 円 盤 が 大 ・ 中 ・ 小 の 順 に 積 ま れ て い る と し よ う。

```
    A         B         C
   ===       
   ====      
  ======     
  ||         ||        ||
```

(=== は 円 盤、 || は 柱 を 表 す。 横 幅 が 円 盤 の 大 き さ を 示 す。)

問 題 は こ う だ。

「こ の 3 枚 の 円 盤 を、 全 部 C の 柱 に 移 し て、 元 の よ う に 重 ね な さ い」

***

### 2. 二 つ の ル ー ル

た だ し、 二 つ の ル ー ル を 守 ら な け れ ば な ら な い。

(1) 一 度 に、 一 枚 だ け 動 か せ る。
(2) 大 き な 円 盤 の 上 に、 小 さ な 円 盤 を 置 か な け れ ば な ら な い。 (大 き な 円 盤 を 小 さ な 円 盤 の 上 に 置 い て は い け な い。)

***

### 3. 3 枚 の 円 盤 を 解 い て み よ う

実 際 に や っ て み よ う。

**1 手 目**: A の 一 番 上 の 小 さ な 円 盤 を、 C の 柱 に 移 す。
**2 手 目**: A の 中 円 盤 を、 B の 柱 に 移 す。
**3 手 目**: C の 小 円 盤 を、 B の 柱 に 移 す (中 円 盤 の 上 に)。
**4 手 目**: A の 大 円 盤 を、 C の 柱 に 移 す。
**5 手 目**: B の 小 円 盤 を、 A の 柱 に 移 す。
**6 手 目**: B の 中 円 盤 を、 C の 柱 に 移 す (大 円 盤 の 上 に)。
**7 手 目**: A の 小 円 盤 を、 C の 柱 に 移 す (中 円 盤 の 上 に)。

完 成! 3 枚 の 円 盤 が C の 柱 に 元 通 り 積 ま れ た。 手 数 は 7 手 で あ っ た。

***

### 4. 不 思 議 な 法 則 ─ 2 倍 + 1

い く つ か の 円 盤 数 で 試 し て み る と、 次 の よ う な 表 が 得 ら れ る。

| 円 盤 の 数 | 最 少 手 数 |
| --------- | --------- |
| 1 枚      | 1 手     |
| 2 枚      | 3 手     |
| 3 枚      | 7 手     |
| 4 枚      | 15 手    |
| 5 枚      | 31 手    |
| 6 枚      | 63 手    |
| 7 枚      | 127 手   |
| 8 枚      | 255 手   |

よ く 見 て み よ う。 1 → 3 → 7 → 15 → 31 → …

どこ か 不 思 議 な 法 則 が あ る よ う だ。

そ う、 「次 の 数 = 前 の 数 × 2 + 1」 で あ る!

  1 × 2 + 1 = 3
  3 × 2 + 1 = 7
  7 × 2 + 1 = 15
  15 × 2 + 1 = 31
  31 × 2 + 1 = 63
  ...

つ ま り、 円 盤 を 1 枚 増 や す と、 手 数 は「前 の 2 倍 + 1」 に な る の だ。

***

### 5. な ぜ そ う な る の か?

理 由 は こ う で あ る。

円 盤 が n 枚 と し よ う。 こ の n 枚 を A か ら C に 移 す に は、 次 の 三 段 階 が 必 要 だ。

(1) 上 の n-1 枚 を、 一 旦 B に 移 す (= n-1 枚 を 移 す 手 数)
(2) 一 番 大 き な 円 盤 を、 A か ら C に 移 す (= 1 手)
(3) B に 移 し た n-1 枚 を、 C に 戻 す (= n-1 枚 を 移 す 手 数)

つ ま り、 「n 枚 の 手 数 = (n-1 枚 の 手 数) × 2 + 1」 と い う 関 係 に な る の だ。

***

### 6. 1883 年、 フ ラ ン ス の 数 学 者 リ ュ カ

こ の パ ズ ル を 発 表 し た の は、 フ ラ ン ス の 数 学 者 エ ド ゥ ア ー ル・リ ュ カ (1842-1891)。 1883 年 の こ と で あ っ た。

リ ュ カ は こ の パ ズ ル に、 一 つ の 不 思 議 な 伝 説 を 付 け 加 え た。

「ベ ト ナ ム の 古 い 寺 院 で、 64 枚 の 黄 金 の 円 盤 が ブ ラ フ マ の 創 造 以 来 僧 侶 た ち に よ っ て 動 か さ れ 続 け て い る。 全 て 移 し 終 わ っ た と き、 世 界 が 終 わ る。」

(も ち ろ ん こ れ は リ ュ カ が 考 え た お 話 で あ る。)

64 枚 の 円 盤 を 1 秒 に 1 枚 ず つ 動 か す と、 約 5,800 億 年 か か る。 こ れ は 宇 宙 の 年 齢 (138 億 年) の 約 420 倍。 つ ま り「世 界 の 終 わ り」 は、 永 遠 に 訪 れ な い ─ と い う 数 学 的 な ジ ョ ー ク に な っ て い る。

***

### 7. 自 分 で 作 っ て み よ う

ハ ノ イ の 塔 は、 紙 や 厚 紙 で 簡 単 に 作 る こ と が で き る。

- 紙 を 切 っ て、 大 き さ の 違 う 円 を 5 つ ほ ど 作 る (大 ・ 中 大 ・ 中 ・ 中 小 ・ 小)
- 鉛 筆 や ペ ン を 3 本 並 べ て「柱」 と す る
- 円 を 通 す 穴 を 開 け、 重 ね る

そ し て、 3 枚 ・ 4 枚 ・ 5 枚 と 増 や し て、 手 数 を 数 え て み よ う。 7 手、 15 手、 31 手 ─ 法 則 が 本 当 か ど う か、 自 分 で 確 か め る こ と が で き る。

数 学 の 美 し さ は、 単 に 公 式 を 知 る こ と で は な い。 自 分 で 試 し て、 法 則 に「気 づ く」 こ と に あ る。 リ ュ カ が 100 年 以 上 前 に 残 し た パ ズ ル は、 今 で も 君 を そ の 喜 び に 招 い て く れ る。
