## ピ タ ゴ ラ ス と 直 角 三 角 形

### 1. 大 工 さ ん の 知 恵

今 か ら 4000 年 前 の こ と で す。 古 代 エ ジ プ ト の 大 工 さ ん た ち は、 大 き な 建 物 を つ く る と き、 「直 角」 を つ く る た め に 不 思 議 な 道 具 を 使 っ て い ま し た。

そ の 道 具 と は、 1 本 の な が い ロ ー プ で す。

そ の ロ ー プ に は、 13 こ の 結 び 目 が、 等 し い 間 隔 で つ い て い ま し た。 結 び 目 と 結 び 目 の あ い だ は、 ぜ ん ぶ で 12 こ。

大 工 さ ん は、 3 人 が か り で こ の ロ ー プ の 両 は し と、 4 番 め の 結 び 目、 7 番 め の 結 び 目 を 持 っ て、 ピ ー ン と 張 り ま す。 す る と…

ふ し ぎ! 3 本 の ロ ー プ で、 ぴ っ た り「直 角」 を つ く る 三 角 形 が で き る の で す。

辺 の 長 さ は「3 と 4 と 5」。 こ の 3 つ の 数 で つ く ら れ た 三 角 形 だ け が、 直 角 を 持 っ た 三 角 形 に な る。

***

### 2. ピ ラ ミ ッ ド を つ く っ た 直 角

エ ジ プ ト の 大 工 さ ん は、 こ の「3 : 4 : 5 ロ ー プ」 を 使 っ て、 大 き な 建 物 の か ど を 正 確 に 直 角 に し ま し た。

そ う し て つ く ら れ た の が、 あ の 有 名 な「ピ ラ ミ ッ ド」 で す。 4500 年 前 に つ く ら れ た ピ ラ ミ ッ ド は、 い ま で も 立 派 に そ び え 立 っ て い ま す。

ピ ラ ミ ッ ド の 角 が、 ぴ っ た り 直 角 で あ る こ と は、 こ の「3 : 4 : 5 ロ ー プ」 の 知 恵 の お か げ で す。

***

### 3. ピ タ ゴ ラ ス の 発 見

時 は 流 れ て、 紀 元 前 6 世 紀 (今 か ら 2500 年 前) の こ と で す。 ギ リ シ ア の サ モ ス 島 に「ピ タ ゴ ラ ス」 と い う 数 学 者 が い ま し た。

ピ タ ゴ ラ ス は、 旅 の と ち ゅ う で エ ジ プ ト に 立 ち 寄 り、 大 工 さ ん の「3 : 4 : 5 ロ ー プ の 直 角」 を 知 り ま し た。 そ し て、 不 思 議 に 思 っ た の で す。

「な ぜ、 3 と 4 と 5 と い う 数 だ け が、 こ ん な に き れ い に 直 角 を つ く る の だ ろ う?」

ピ タ ゴ ラ ス は、 こ の 不 思 議 を 何 年 も 考 え 続 け、 つ い に 一 つ の 法 則 を 見 つ け 出 し ま し た。

   「直 角 三 角 形 の 三 つ の 辺 の あ い だ に は、 a × a + b × b = c × c と い う 関 係 が あ る」

3 × 3 + 4 × 4 = 9 + 16 = 25 = 5 × 5。

確 か に! 3 と 4 と 5 だ け で な く、 ほ か の 直 角 三 角 形 に も、 ぜ ん ぶ こ の 法 則 が 当 て は ま る の で す。

こ の 法 則 を、 い ま で は「ピ タ ゴ ラ ス の 定 理」 と 呼 ん で い ま す。 中 学 校 3 年 生 で く わ し く 学 び ま す。

***

### 4. 4000 年 前 か ら 続 く 算 数

エ ジ プ ト の 大 工 さ ん が 知 っ て い た「直 角 ロ ー プ」 の 知 恵。 そ れ を ピ タ ゴ ラ ス が 数 学 の 法 則 に し た こ と。 そ し て、 今 で も 私 た ち が 学 校 で「直 角 三 角 形」 を 学 ぶ こ と。

算 数 と は、 4000 年 前 か ら 続 く、 と て も 古 い 学 問 な の で す。 そ し て、 算 数 は 「家 を つ く る」「土 地 を は か る」 と い っ た、 と て も 身 近 な こ と か ら 始 ま っ た 学 問 で す。

み な さ ん が 今 日 ノ ー ト に 描 く「直 角」 も、 4000 年 前 の 大 工 さ ん が 描 い た「直 角」 と、 同 じ 直 角 で す。
