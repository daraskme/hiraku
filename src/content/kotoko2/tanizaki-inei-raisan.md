<h3>本文（『陰翳礼讃』 谷崎潤一郎・冒頭 抜粋）</h3>

<blockquote>
<p>
今 日 の 日本 の 家屋 は、西洋 の 建築 の 影響 を 受けて、可 能 な 限り 多く の 光 を 取り 入 れ よう と する。<br>
ガラス の 窓、白い 壁、電気 の 灯火。── これら は、確かに 便利 で あ り、衛生 的 で も あ る。<br>
だが、私 たち 日本人 が 本来 持って いた 美 の 感覚 と は、まったく 異なる 原理 に 立 つ もの で ある。
</p>

<p>
日本 の 古い 家屋 を 思い 起こ し て み たま え。<br>
深い 軒 (のき) の 影、薄暗い 床の間 (とこのま)、行灯 (あんどん) の ぼんやり と し た 光。── そ こ に 漆器 を 置 い た とき、初め て 漆 の 黒 は 本当 の 美 を 見せる。<br>
明 る い 食卓 の 上 で 見る 漆器 は、ただ の 黒い 椀 に すぎ ない。 だ が、行灯 の もと で 見る とき、それ は 底 知れ ぬ 闇 を 湛 (たた) えた、宇宙 の 一片 と 化す。
</p>

<p>
金屏風 (きん びょうぶ) も、蒔絵 (まきえ) も、同じ で あ る。<br>
それら の 金 が、煌 (きら) めく の は、明 る い 部屋 で は ない。 薄暗い 部屋 の 中、ろうそく の 揺 ら ぐ 光 の もと で、金 は 不意 に 浮 か び 上 が り、闇 と 戯 (たわむ) れる の だ。
</p>

<p>
ここ に、日本 の 美 の 本質 が ある。<br>
<strong>美 は、物体 そ の もの に は な い。 物体 と 物体 と の 作 り 出 す 陰翳 の 綾 (あや)、明 と 暗 の 戯 れ に ある。</strong>
</p>

<p>
ところ が、近代 西洋 の 美学 は、これ と 真逆 で ある。<br>
すべて を 明 る く 照 ら し、すべて を は っ き り と 見せる。 影 を 排 除 し、輪郭 を く っ き り と させる。<br>
これ は 「視覚 の 美学」 と は 言える が、それ で は <strong>陰翳 が 与 え る 「想像 力 の 余地」</strong> を 失 う。
</p>

<p>
さらに 言 え ば、日本 建築 の 中 で、私 が もっとも 詩 的 だ と 感じ る の は、意外 に も 「<strong>厠 (かわや)</strong>」 ─ 便所 ─ で ある。<br>
古 い 日本 家屋 の 厠 は、母屋 か ら 少 し 離 れ た、庭 に 面 し た 一隅 に あ り、薄 暗 く、静か で、外 の 樹 々 の 葉擦 れ や 鳥 の 声 が 聞 こ え る。<br>
こ こ で 用 を 足 す 時 、私 は ふ し ぎ な ほ ど 心 が 落 ち 着 き、思 索 に 沈 む こ と が で き る。<br>
近代 の 白タイル ・ 蛍光灯 ・ 換気扇 の 完備 さ れ た 西洋 式 ト イ レ で、こ う し た 「精神 的 安 ら ぎ」 が 得 ら れ る だ ろ う か。
</p>

<p>
我 が 国 の 美学 は、要 す る に、<strong>「不足 を 楽 し む」</strong> 美学 で あ る。<br>
完璧 に 明 る い 部屋 で は な く、薄 暗 い 部屋。<br>
完璧 に 整 え ら れ た 庭 で は な く、わ ず か に 苔 む し た 石。<br>
完璧 に 直線 の 柱 で は な く、自然 の 木目 を 残 し た 柱。<br>
そ こ に、日本 人 が 千 年 以 上 か け て 磨 い て き た、独自 の 美 が あ る。
</p>
</blockquote>

<h3>解説 ─ 「陰翳」 を 失 う 近代</h3>

<p>
谷崎 が 本書 を 書 い た 1933 年 (昭和 8 年) ─ 日本 は 急速 に 近代 化 し、 都市 に は 電灯 が 普及 し、 西洋 式 ホテル ・ 病院 ・ 学校 が 建設 さ れ て い た。
</p>

<p>
谷崎 は、こ の 「近代 化 の 速度」 に、強 い 違 和 感 を 抱 い た。<br>
便利 さ や 衛生 と 引 き 換 え に、日本 人 が <strong>千 年 以 上 か け て 磨 い て き た 美 の 感覚</strong> が、急速 に 失 わ れ つ つ あ る と。
</p>

<p>
本書 は、決 し て 「近代 化 す べ き で な い」 と 主張 し て い る わ け で は な い。<br>
谷崎 が 訴 え た の は、「<strong>近代 化 す る な ら、何 を 失 う か を 自覚 し ろ</strong>」 と い う こ と だ っ た。
</p>

<p>
日本 美 の 核心 は、こ の 三 つ の 角度 に お い て、<strong>西洋 近代 の 原理 と 逆 を 行 く</strong> こ と に あ る。
</p>

<h3>「厠 の 美学」 ─ 谷崎 ら し い 視点</h3>

<p>
本書 の 中 で 特 に 有名 な の が、「<strong>厠 (便所) の 美</strong>」 を 論 じ た 一節 で あ る。
</p>

<p>
谷崎 は、近代 西洋 式 の 「白 タ イ ル ・ 蛍光灯」 の ト イ レ を 否定 し、古 い 日本 家屋 の 「薄 暗 く、庭 に 面 し、静寂 の 中 で 思 索 で き る」 厠 を 礼讃 し た。<br>
彼 に と っ て、厠 は 単 な る 「排泄 の 場」 で は な く、 <strong>日本 建築 の 美 学 が 最 も 純粋 に 現 れ る 詩 的 空間</strong> だ っ た の だ。
</p>

<p>
こ の 視点 は、現代 の 「ミ ニ マ リ ズ ム」 「ウ ェ ル ネ ス」 「マ イ ン ド フ ル ネ ス」 と も 通 じ る 部分 が あ る。<br>
「最 も 機能 的 な 空間 が、最 も 美 し い と は 限 ら な い」 ─ こ れ は 21 世紀 の 設計 者 ・ デ ザ イ ナ ー に も 響 く 言葉 で あ る。
</p>

<h3>世界 で 読 ま れ る 『陰翳 礼讃』</h3>

<p>
本書 は、1977 年 に 米国 で 英訳 が 出 版 さ れ (Thomas J. Harper・Edward G. Seidensticker 訳 "In Praise of Shadows")、欧米 の 建築 家 ・ デ ザ イ ナ ー ・ 思想 家 に 強 い 影響 を 与 え た。
</p>

<ul>
<li>建築 家 <strong>Frank Lloyd Wright</strong> は、本書 の 思想 と 共通 す る 部分 を 自分 の 「Falling Water (落水 荘)」 設計 に 反映 し た と 言 わ れ る。</li>
<li>デ ザ イ ナ ー <strong>原研哉</strong> (はら けんや) は、無印 良品 の 「白」 の 哲学 を、谷崎 の 「陰翳」 と 対 比 し て 論 じ た。</li>
<li>建築 家 <strong>Peter Zumthor</strong> (スイス) の 暗 い 光 を 用 い た 建築 (Thermal Baths Vals 等) は、谷崎 の 美学 と 響 き 合 っ て い る。</li>
</ul>

<p>
昭和 8 年 に 日本 で 書 か れ た 一篇 が、半 世紀 後 に 翻訳 さ れ、世界 の 建築 ・ デ ザ イ ン の 言説 を 変 え た。
</p>

<h3>あ な た の 「陰翳」 を 探 し て</h3>

<p>
谷崎 の 美学 は、行灯 や 漆器 や 古い 家 だ け の 話 で は な い。<br>
今 こ こ に、君 の 周 り に も、「陰翳」 は あ る。
</p>

<ul>
<li>夕暮 れ の 教室 の、誰 も い な い 黒板 の 表面</li>
<li>夜 の 街灯 の 下 に 浮 か ぶ、 古 い 銀杏 の 木 の 影</li>
<li>雨 の 後 の 石畳 に 映 る、 街 の 灯</li>
<li>カ フ ェ の 木目 の テー ブ ル に 落 ち る、ペ ン ダ ン ト ラ イ ト の 影</li>
