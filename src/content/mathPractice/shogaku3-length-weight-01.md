description: cm、m、g、kgの関係を、生活の場面に合わせて変換する練習プリント。

problems:
  - prompt: 1mは何cmですか。
    answer: "100cm"
    explanation: 1m = 100cmです。
    difficulty: basic
    type: conversion
    hints:
      - mをcmにすると100倍です。
  - prompt: 250cmは何m何cmですか。
    answer: 2m50cm
    explanation: 200cmが2mで、残りが50cmです。
    difficulty: basic
    type: conversion
    hints:
      - 100cmごとに1mです。
  - prompt: 3m20cmは何cmですか。
    answer: "320cm"
    explanation: 3mは300cmなので、300cm + 20cm = 320cmです。
    difficulty: basic
    type: conversion
    hints:
      - mの部分を先にcmへ直します。
  - prompt: 1kgは何gですか。
    answer: "1000g"
    explanation: 1kg = 1000gです。
    difficulty: basic
    type: conversion
    hints:
      - kgをgにすると1000倍です。
  - prompt: 1800gは何kg何gですか。
    answer: 1kg800g
    explanation: 1000gが1kgで、残りが800gです。
    difficulty: standard
    type: conversion
    hints:
      - 1000gごとに1kgです。
  - prompt: 2kg300gは何gですか。
    answer: "2300g"
    explanation: 2kgは2000gなので、2000g + 300g = 2300gです。
    difficulty: standard
    type: conversion
    hints:
      - kgの部分をgへ直します。
  - prompt: 85cmのリボンと1m20cmのリボンをつなぐと、全部で何cmですか。
    answer: "205cm"
    explanation: 1m20cmは120cmです。85cm + 120cm = 205cmです。
    difficulty: challenge
    type: word-problem
    hints:
      - まず同じ単位にそろえます。
  - prompt: 750gの箱と1kg200gの箱を合わせると、全部で何gですか。
    answer: "1950g"
    explanation: 1kg200gは1200gです。750g + 1200g = 1950gです。
    difficulty: challenge
    type: word-problem
    hints:
      - kgをgに直してからたします。
