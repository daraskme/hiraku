problems:
  - prompt: りんごが 24こ あります。そこへ 18こ もらいました。ぜんぶで 何こ ありますか。
    answer: 42こ
    explanation: 24 + 18 = 42。
    difficulty: basic
    type: word-problem
    hints:
      - ふえたので、たし算にします。
  - prompt: 色紙が 53まい あります。19まい つかいました。のこりは 何まいですか。
    answer: 34まい
    explanation: 53 - 19 = 34。
    difficulty: basic
    type: word-problem
    hints:
      - つかった分をひきます。
  - prompt: 赤い花が 28本、白い花が 37本 あります。あわせて 何本ですか。
    answer: 65本
    explanation: 28 + 37 = 65。
    difficulty: basic
    type: word-problem
    hints:
      - 赤と白をあわせます。
  - prompt: えんぴつが 80本 あります。36本を くばりました。のこりは 何本ですか。
    answer: 44本
    explanation: 80 - 36 = 44。
    difficulty: basic
    type: word-problem
    hints:
      - 80から36をひきます。
  - prompt: 本を 17ページ 読みました。つぎの日に 26ページ 読みました。2日で 何ページ 読みましたか。
    answer: 43ページ
    explanation: 17 + 26 = 43。
    difficulty: basic
    type: word-problem
    hints:
      - 2日分をたします。
  - prompt: かごに 64こ のみかんがあります。27こ 食べました。のこりは 何こですか。
    answer: 37こ
    explanation: 64 - 27 = 37。
    difficulty: standard
    type: word-problem
    hints:
      - 一の位はくり下がりがあります。
  - prompt: 1組に 35人、2組に 38人 います。2つの組で 何人いますか。
    answer: 73人
    explanation: 35 + 38 = 73。
    difficulty: standard
    type: word-problem
    hints:
      - 十の位と一の位をそろえて計算します。
  - prompt: 92円 もっています。47円のけしゴムを買いました。何円 のこりますか。
    answer: 45円
    explanation: 92 - 47 = 45。
    difficulty: standard
    type: word-problem
    hints:
      - お金を使ったので、ひき算です。
  - prompt: バスに 26人 のっていました。バス停で 14人 のり、9人 おりました。いま何人 のっていますか。
    answer: 31人
    explanation: 26 + 14 - 9 = 31。
    difficulty: challenge
    type: word-problem
    hints:
      - のった人はたし、おりた人はひきます。
  - prompt: シールを 75まい もっています。友だちに 18まい あげたあと、23まい もらいました。いま何まい ありますか。
    answer: 80まい
    explanation: 75 - 18 + 23 = 80。
    difficulty: challenge
    type: word-problem
    hints:
      - 先にあげた分をひいてから、もらった分をたします。
