problems:
  - prompt: x + 7 = 15 を解きましょう。
    answer: x = 8
    explanation: 両辺から7をひくと、x = 8。
    difficulty: basic
    type: equation
    hints:
      - +7 を消すには、両辺から7をひきます。
  - prompt: x - 9 = 4 を解きましょう。
    answer: x = 13
    explanation: 両辺に9をたすと、x = 13。
    difficulty: basic
    type: equation
    hints:
      - -9 を消すには、両辺に9をたします。
  - prompt: 3x = 21 を解きましょう。
    answer: x = 7
    explanation: 両辺を3でわると、x = 7。
    difficulty: basic
    type: equation
    hints:
      - 3倍を消すには、3でわります。
  - prompt: 5x + 2 = 17 を解きましょう。
    answer: x = 3
    explanation: 5x = 15、x = 3。
    difficulty: basic
    type: equation
    hints:
      - まず両辺から2をひきます。
  - prompt: 4x - 6 = 18 を解きましょう。
    answer: x = 6
    explanation: 4x = 24、x = 6。
    difficulty: basic
    type: equation
    hints:
      - -6 を右へ移すと +6 になります。
  - prompt: 2x + 5 = x + 12 を解きましょう。
    answer: x = 7
    explanation: 2x - x = 12 - 5 より、x = 7。
    difficulty: standard
    type: equation
    hints:
      - x の項を左、数の項を右に集めます。
  - prompt: 7x - 3 = 4x + 12 を解きましょう。
    answer: x = 5
    explanation: 7x - 4x = 12 + 3、3x = 15、x = 5。
    difficulty: standard
    type: equation
    hints:
      - 文字と数を別々の側に集めます。
  - prompt: 3(x + 2) = 21 を解きましょう。
    answer: x = 5
    explanation: x + 2 = 7、x = 5。分配すると 3x + 6 = 21 でもよいです。
    difficulty: standard
    type: equation
    hints:
      - 先に両辺を3でわれます。
  - prompt: 2(x - 4) + 5 = 13 を解きましょう。
    answer: x = 8
    explanation: 2x - 8 + 5 = 13、2x - 3 = 13、2x = 16、x = 8。
    difficulty: challenge
    type: equation
    hints:
      - かっこを外してから整理します。
  - prompt: 0.5x + 3 = 8 を解きましょう。
    answer: x = 10
    explanation: 0.5x = 5、x = 10。
    difficulty: challenge
    type: equation
    hints:
      - 0.5 は 1/2 です。
