description: 一次関数 y = ax + b の値、変化の割合、式の決定を練習するプリント。
tags:

problems:
  - prompt: y = 2x + 3 で、x = 4 のとき y の値を求めましょう。
    answer: y = 11
    explanation: y = 2 × 4 + 3 = 11。
    difficulty: basic
    type: linear-function
    hints:
      - x に4を代入します。
  - prompt: y = -3x + 5 で、x = 2 のとき y の値を求めましょう。
    answer: y = -1
    explanation: y = -3 × 2 + 5 = -6 + 5 = -1。
    difficulty: basic
    type: linear-function
    hints:
      - 負の数の計算に注意します。
  - prompt: 一次関数 y = 4x - 1 の変化の割合を答えましょう。
    answer: "4"
    explanation: y = ax + b の a が変化の割合なので、4。
    difficulty: basic
    type: linear-function
    hints:
      - x の係数を見ます。
  - prompt: 一次関数 y = -2x + 7 の切片を答えましょう。
    answer: "7"
    explanation: y = ax + b の b が切片なので、7。
    difficulty: basic
    type: linear-function
    hints:
      - x が0のときの y の値です。
  - prompt: 変化の割合が3で、x = 0 のとき y = 2 となる一次関数の式を求めましょう。
    answer: y = 3x + 2
    explanation: 変化の割合が3なので a = 3、切片が2なので b = 2。
    difficulty: standard
    type: linear-function
    hints:
      - y = ax + b に当てはめます。
  - prompt: 点 (0, -4) を通り、変化の割合が5の一次関数の式を求めましょう。
    answer: y = 5x - 4
    explanation: 変化の割合が5、切片が-4なので y = 5x - 4。
    difficulty: standard
    type: linear-function
    hints:
      - x=0 の点は切片を表します。
  - prompt: 点 (1, 3) と点 (3, 7) を通る直線の変化の割合を求めましょう。
    answer: "2"
    explanation: xは2増え、yは4増えるので、4 ÷ 2 = 2。
    difficulty: standard
    type: linear-function
    hints:
      - y の増加量 ÷ x の増加量。
  - prompt: 点 (2, 5) と点 (4, 1) を通る直線の変化の割合を求めましょう。
    answer: "-2"
    explanation: xは2増え、yは4減るので、-4 ÷ 2 = -2。
    difficulty: standard
    type: linear-function
    hints:
      - y が減ると変化の割合は負になります。
  - prompt: 変化の割合が2で、点 (3, 10) を通る一次関数の式を求めましょう。
    answer: y = 2x + 4
    explanation: y = 2x + b に (3,10) を代入すると、10 = 6 + b、b = 4。
    difficulty: challenge
    type: linear-function
    hints:
      - まず y = 2x + b とおきます。
  - prompt: 点 (1, 1) と点 (4, 10) を通る一次関数の式を求めましょう。
    answer: y = 3x - 2
    explanation: 変化の割合は (10 - 1) ÷ (4 - 1) = 3。y = 3x + b に (1,1) を代入して b = -2。
    difficulty: challenge
    type: linear-function
    hints:
      - 先に変化の割合を求めます。
