problems:
  - prompt: x + y = 7、x - y = 1 を解きましょう。
    answer: x = 4, y = 3
    explanation: 2つの式を足すと2x = 8、x = 4。x + y = 7に入れてy = 3です。
    difficulty: basic
    type: simultaneous-equations
    hints:
      - 2つの式を足すとyが消えます。
  - prompt: x + y = 9、x = y + 3 を解きましょう。
    answer: x = 6, y = 3
    explanation: x = y + 3をx + y = 9に代入すると2y + 3 = 9、y = 3、x = 6です。
    difficulty: basic
    type: simultaneous-equations
    hints:
      - 代入法で解きます。
  - prompt: 2x + y = 11、x + y = 7 を解きましょう。
    answer: x = 4, y = 3
    explanation: 上の式から下の式を引くとx = 4。x + y = 7に入れてy = 3です。
    difficulty: basic
    type: simultaneous-equations
    hints:
      - 式を引くとyが消えます。
  - prompt: 3x + y = 13、x + y = 7 を解きましょう。
    answer: x = 3, y = 4
    explanation: 2つの式を引くと2x = 6、x = 3。x + y = 7に入れてy = 4です。
    difficulty: standard
    type: simultaneous-equations
    hints:
      - yの係数が同じです。
  - prompt: 2x + 3y = 16、x + 3y = 11 を解きましょう。
    answer: x = 5, y = 2
    explanation: 上の式から下の式を引くとx = 5。x + 3y = 11に入れて3y = 6、y = 2です。
    difficulty: standard
    type: simultaneous-equations
    hints:
      - 3yを消します。
  - prompt: x + 2y = 10、x - y = 1 を解きましょう。
    answer: x = 4, y = 3
    explanation: 上の式から下の式を引くと3y = 9、y = 3。x - y = 1に入れてx = 4です。
    difficulty: standard
    type: simultaneous-equations
    hints:
      - xの係数が同じです。
  - prompt: りんご2個とみかん3個で330円、りんご2個とみかん1個で230円です。りんご1個とみかん1個の値段を求めましょう。
    answer: りんご90円、みかん50円
    explanation: みかん2個の差が100円なので、みかん1個は50円。2x + 50 = 230より、りんご1個は90円です。
    difficulty: challenge
    type: word-problem
    hints:
      - みかんの個数だけが2個違います。
  - prompt: 2x + y = 230、2x + 3y = 330 を解きましょう。
    answer: x = 90, y = 50
    explanation: 下の式から上の式を引くと2y = 100、y = 50。2x + 50 = 230よりx = 90です。
    difficulty: challenge
    type: simultaneous-equations
    hints:
      - 文章題を式だけにしたものです。
