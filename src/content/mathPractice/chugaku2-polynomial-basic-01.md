description: 単項式・多項式の加減と乗除を、途中式を書きながら練習するプリント。
tags:

problems:
  - prompt: 3x + 5x を計算しなさい。
    answer: 8x
    explanation: 同じ文字の項どうしなので、係数をたして3 + 5 = 8です。
    difficulty: basic
    type: calculation
    hints:
      - xの係数だけを見ます。
  - prompt: 7a - 2a を計算しなさい。
    answer: 5a
    explanation: 7 - 2 = 5なので、5aです。
    difficulty: basic
    type: calculation
    hints:
      - aはそのまま残します。
  - prompt: 4x + 3y - x + 2y を簡単にしなさい。
    answer: 3x + 5y
    explanation: xの項は4x - x = 3x、yの項は3y + 2y = 5yです。
    difficulty: basic
    type: calculation
    hints:
      - 同類項をまとめます。
  - prompt: 2(3x + 4) を展開しなさい。
    answer: 6x + 8
    explanation: 2をかっこの中の3xと4の両方にかけます。
    difficulty: standard
    type: calculation
    hints:
      - 分配法則を使います。
  - prompt: 5(a - 2b) を展開しなさい。
    answer: 5a - 10b
    explanation: 5 × a = 5a、5 × (-2b) = -10bです。
    difficulty: standard
    type: calculation
    hints:
      - 符号に注意します。
  - prompt: 6x ÷ 3 を計算しなさい。
    answer: 2x
    explanation: 係数の6を3で割り、xは残ります。
    difficulty: standard
    type: calculation
    hints:
      - 6 ÷ 3を先に考えます。
  - prompt: 12ab ÷ 4a を計算しなさい。
    answer: 3b
    explanation: 12 ÷ 4 = 3で、aは約分され、bが残ります。
    difficulty: challenge
    type: calculation
    hints:
      - 数字と文字を分けて約分します。
  - prompt: 3(2x - 1) - 2(x + 4) を簡単にしなさい。
    answer: 4x - 11
    explanation: 6x - 3 - 2x - 8 = 4x - 11です。
    difficulty: challenge
    type: calculation
    hints:
      - まずそれぞれのかっこを展開します。
