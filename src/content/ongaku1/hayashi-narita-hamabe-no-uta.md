## 浜 辺 の 歌

林 古 渓 作 詞 / 成 田 為 三 作 曲 (1916)

### 一

あ し た 浜 辺 を さ ま よ え ば、
昔 の こ と ぞ 偲 ば る る。
風 の 音 よ、 雲 の さ ま よ、
寄 す る 波 も 貝 の 色 も。

### 二

ゆ う べ 浜 辺 を も と お れ ば、
昔 の 人 ぞ 偲 ば る る。
寄 す る 波 よ、 か え す 波 よ、
月 の 色 も 星 の 影 も。

### 三

は や も 涙 の 一 雫
一 雫
病 み し 我 は す で に 癒 え て、
浜 の 真 砂 ま な ご い ま は と て、
帰 ら ぬ 波 か な、 月 の 色 か な。

(戦 後 の 教 科 書 で は 一 番・二 番 の み の 場 合 が 多 い)

***

## 教 材 解 説

### 朝 と 夕 の 時 間 構 成

『浜 辺 の 歌』 は、 朝 と 夕 を 対 称 的 に 描 く 構 成 を 持 つ。

**一 番**「あ し た 浜 辺 を…」 ─ 古 語 の「あ し た」 は「朝」 の 意。 朝 の 浜 辺 を 歩 き な が ら、 昔 の こ と (= 過 去 の 出 来 事) を 偲 ぶ。 朝 の 風 ・ 雲 ・ 波 ・ 貝 の 色 ─ す べ て が 過 去 を 呼 び 起 こ す 装 置 と な る。

**二 番**「ゆ う べ 浜 辺 を…」 ─ 夕 方 の 浜 辺。 朝 の「昔 の こ と (= 出 来 事)」 か ら、 夕 の「昔 の 人 (= 思 い 出 の 中 の 人)」 へ と 視 線 が 深 化 す る。 月 ・ 星 が 加 わ り、 回 想 が 深 く な る。

**三 番** ─ 病 か ら 癒 え た「我」 の 視 点。 病 床 か ら の 回 復 を、 浜 辺 の 風 景 と 重 ね 合 わ せ る。 (こ の 番 は 戦 後 教 科 書 に 載 ら な い こ と が 多 い)

### 旋 律 ─ ヘ 長 調 8 分 の 6 拍 子

成 田 為 三 の 旋 律 は、 ヘ 長 調 8 分 の 6 拍 子。 「タ ン・タ・タ / タ ン・タ・タ」 と 一 拍 を 三 つ に 分 け る ゆ っ た り と し た リ ズ ム が、 砂 浜 を 歩 く 足 取 り、 寄 せ 引 く 波、 そ し て 回 想 の 揺 ら ぎ を 表 現 す る。

22 歳 の 成 田 が、 こ の 揺 れ る リ ズ ム を 選 び 取 っ た こ と は、 後 の 日 本 唱 歌 史 に 一 つ の 名 旋 律 を 加 え た。

### 信 州 か ら 秋 田 へ ─ 二 人 の 出 会 い

林 古 渓 は 信 州 永 江 村 (現 中 野 市) の 出 身、 成 田 為 三 は 秋 田 県 鷹 巣 (現 北 秋 田 市) の 出 身。 二 人 は 東 京 音 楽 学 校 で 出 会 い、 林 古 渓 が 教 員、 成 田 為 三 が 学 生 と し て の 関 係 で あ っ た。

林 古 渓 の 詩 が、 成 田 為 三 の 22 歳 の 旋 律 に 出 会 い、 大 正 5 年 (1916) に『浜 辺 の 歌』 が 誕 生 し た。 信 州 と 秋 田 と い う 二 つ の 地 方 出 身 の 詩 人 と 作 曲 家 が、 東 京 で 出 会 い、 海 の 歌 を 残 し た 不 思 議 な 偶 然。

### 100 年 後 も

1916 年 か ら 100 年 を 超 え て、『浜 辺 の 歌』 は 中 学 校 共 通 教 材 と し て 全 国 で 歌 わ れ 続 け て い る。

成 田 為 三 は 1945 年 8 月 12 日、 終 戦 三 日 前 に 病 没 し た。 「平 和」 と「浜 辺 の 歌」 を 同 時 に 見 る こ と は な か っ た。 し か し、 そ の 旋 律 は、 戦 後 の 日 本 が 平 和 と 共 に 歌 い 継 い だ。

林 古 渓 は 1947 年 8 月、 戦 後 の 困 窮 の 中 で 没 し た。 享 年 72 歳。 戦 後 の 教 科 書 に 自 ら の 詞 が 残 さ れ た こ と を 知 っ て の 死 で あ っ た。

100 年 を 超 え る 名 曲 ─ そ れ は、 二 人 の 信 州 と 秋 田 の 男 が 残 し た、 海 の 揺 れ る 遺 産 で あ る。
