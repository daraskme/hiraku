<h3>解 説 ─ 23 歳 の 名 曲</h3>

<p>
1901 年 (明 治 34 年)、 滝 廉 太 郎 は <strong>23 歳</strong> で「荒 城 の 月」 を 作 曲 し た。
</p>

<p>
東 京 音 楽 学 校 (現 在 の 東 京 藝 術 大 学) を 首 席 で 卒 業 し、 翌 1901 年 4 月 か ら ド イ ツ・ラ イ プ ツ ィ ヒ 音 楽 院 に 留 学 す る 直 前 の こ と。<br>
若 き 滝 廉 太 郎 は、 文 部 省 の 『中 学 唱 歌』 編 集 委 員 と し て、 こ の 曲 を 書 い た。
</p>

<p>
詞 は、 仙 台 出 身 の 詩 人 ・ <strong>土 井 晩 翠</strong> (1871–1952) が、 仙 台 の 青 葉 城 跡 を イ メ ー ジ し て 書 い た も の。<br>
滝 廉 太 郎 は、 福 島 県 会 津 若 松 の 鶴 ヶ 城 跡 (戊 辰 戦 争 の 激 戦 地) を イ メ ー ジ し て 作 曲 し た と 伝 え ら れ る。
</p>

<p>
こ う し て、 仙 台 と 会 津 ─ 二 つ の 廃 城 ─ が 一 つ の 名 曲 に 結 晶 し た の だ。
</p>

<h3>明 治 期 の 音 楽 史 と「荒 城 の 月」</h3>

<p>
明 治 維 新 (1868) 以 降、 日 本 は 西 洋 音 楽 を 急 速 に 受 容 し た。
</p>

<ul>
<li>1879 ─ 文 部 省 「音 楽 取 調 掛 (が か り)」 設 置</li>
<li>1881 ─ 『小 学 唱 歌 集』 出 版 (西 洋 メ ロ デ ィ ー に 日 本 語 歌 詞)</li>
<li>1887 ─ 東 京 音 楽 学 校 設 立</li>
<li>1898 ─ 滝 廉 太 郎、 東 京 音 楽 学 校 入 学 (19 歳)</li>
<li><strong>1901 ─ 滝 廉 太 郎「荒 城 の 月」 作 曲</strong></li>
<li>1903 ─ 滝 廉 太 郎、 ド イ ツ 留 学 か ら 帰 国 後、 大 分 で 24 歳 没</li>
<li>1929 ─ 山 田 耕 筰、「荒 城 の 月」 を 編 曲 (現 在 一 般 に 歌 わ れ る 版)</li>
</ul>

<p>
滝 廉 太 郎 は、 「西 洋 音 楽 の 様 式 で、 日 本 の 詩 心 を 表 現 す る」 と い う 道 を 開 い た 草 創 期 の 作 曲 家。<br>
彼 の 後 を 受 け 継 ぐ の が、 山 田 耕 筰・中 山 晋 平・本 居 長 世 ら 大 正 期 の 童 謡 作 曲 家 た ち で あ っ た。
</p>

<h3>「諸 行 無 常」 と い う 主 題</h3>

<p>
「荒 城 の 月」 の 中 心 主 題 は、 <strong>諸 行 無 常 (し ょ ぎ ょ う む じ ょ う)</strong> ─ あ ら ゆ る も の は 移 ろ う ─ で あ る。
</p>

<p>
こ れ は、 『平 家 物 語』 (本 サ イ ト kotoko1) の 冒 頭 「祇 園 精 舎 の 鐘 の 声、 諸 行 無 常 の 響 あ り」 と 同 じ 主 題。<br>
日 本 中 世 か ら 受 け 継 が れ た「無 常 観」 を、 明 治 の 西 洋 音 楽 で 表 現 し た の が「荒 城 の 月」 で あ る。
</p>
