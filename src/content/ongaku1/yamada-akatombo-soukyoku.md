<h3>山 田 耕 筰 と い う 作 曲 家</h3>

<p>
山 田 耕 筰 (1886–1965) は、 日 本 近 代 音 楽 史 で 最 も 重 要 な 作 曲 家 の 一 人 で あ る。
</p>

<p>
彼 の 生 涯 と 業 績:
</p>

<ul>
<li>1886 ─ 東 京 に 生 ま れ る</li>
<li>1908 ─ 東 京 音 楽 学 校 (現 東 京 藝 大) 卒</li>
<li><strong>1910–1913 ─ ベ ル リ ン に 留 学</strong> (滝 廉 太 郎 の 死 後 8 年 後)</li>
<li>1914 ─ 帰 国 後、 日 本 初 の 本 格 的 交 響 楽 団「東 京 フ ィ ル ハ ー モ ニ ー」 設 立</li>
<li><strong>1921 ─ 北 原 白 秋 と 共 同 で 「か ら た ち の 花」 作 曲</strong> (本 サ イ ト kokugo2)</li>
<li>1923 ─ 「赤 い 鳥 小 鳥」 作 曲</li>
<li><strong>1927 ─ 三 木 露 風 詞 に「赤 と ん ぼ」 作 曲</strong></li>
<li>1929 ─ 滝 廉 太 郎「荒 城 の 月」 を 編 曲 (現 在 一 般 に 歌 わ れ る 版)</li>
<li>1940 ─ オ ペ ラ「黒 船」 完 成</li>
<li>1956 ─ 文 化 勲 章 受 章</li>
<li>1965 ─ 没。 78 歳</li>
</ul>

<p>
山 田 耕 筰 は、 童 謡 だ け で な く、 オ ペ ラ・交 響 曲・映 画 音 楽 ま で、 日 本 音 楽 の あ ら ゆ る ジ ャ ン ル を 切 り 開 い た。<br>
「日 本 近 代 音 楽 の 父」 と 呼 ば れ る。
</p>

</p>

<p>
大 正 童 謡 運 動 の 中 心 と な っ た の は、 鈴 木 三 重 吉 が 1918 年 に 創 刊 し た 児 童 文 学 雑 誌『<strong>赤 い 鳥</strong>』 で あ っ た (本 サ イ ト で 多 数 関 連 教 材)。<br>
「子 ど も に も 一 流 の 芸 術 を」 と い う 理 念 の 下、 文 学 と 音 楽 が 結 合 し た。
</p>

<h3>「赤 と ん ぼ」 が 国 民 的 童 謡 に</h3>

<p>
三 木 露 風 の 詞 (1921) に 山 田 耕 筰 が 曲 を 付 け た の は、 1927 年 (昭 和 2 年)。<br>
当 初 は そ れ ほ ど 注 目 さ れ な か っ た が、 戦 後、 NHK の 「ラ ジ オ 歌 謡」 や「み ん な の う た」 で 広 く 紹 介 さ れ、 国 民 的 童 謡 と な っ た。
</p>

<p>
2007 年、 文 化 庁 が 実 施 し た「日 本 の う た 100 選」 で、 「赤 と ん ぼ」 は 選 出 さ れ た。<br>
明 治 ・ 大 正 期 の 童 謡 の 中 で、 今 も 一 番 親 し ま れ て い る 曲 の 一 つ で あ る。
</p>

<h3>詞 と 曲 が 結 び つ く と き</h3>

<p>
三 木 露 風 が「赤 と ん ぼ」 の 詞 を 書 い た の は 1921 年。 山 田 耕 筰 が 曲 を 付 け た の は 1927 年。<br>
6 年 の 時 を 経 て、 詞 と 曲 が 出 会 い、 今 私 た ち の 知 る「赤 と ん ぼ」 が 完 成 し た。
</p>

<p>
詞 だ け で も 美 し い。 曲 だ け で も 美 し い。<br>
だ が、 そ の 二 つ が 結 び つ い た と き ─ ど ち ら が 主 と も 言 え な い、 一 つ の 完 璧 な 「歌」 が 生 ま れ た の だ。
</p>

<p>
こ の「<strong>詞 と 曲 の 結 合</strong>」 こ そ、 山 田 耕 筰 が 日 本 童 謡 史 に 残 し た 最 大 の 業 績 で あ る。
</p>

<h3>歌 っ て み よ う</h3>

<p>
「赤 と ん ぼ」 を、 ぜ ひ 声 に 出 し て 歌 っ て み よ う。
</p>

<p>
三 木 露 風 が 100 年 前 に 詞 で 描 い た 風 景、 山 田 耕 筰 が 95 年 前 に 曲 で 結 晶 化 し た 想 い ─ そ の 全 て が、 君 の 声 を 通 し て、 今 こ こ に 蘇 る。
</p>

<p>
そ し て、 君 が 大 人 に な っ て、 自 分 の 子 ど も や 孫 と 一 緒 に「赤 と ん ぼ」 を 歌 う 日 が 来 た と き ─ 100 年 前 か ら 続 く こ の 歌 が、 さ ら に 未 来 へ と 受 け 継 が れ て い く だ ろ う。
</p>
