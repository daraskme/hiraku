<h3>「春 が 来 た 春 が 来 た ど こ に 来 た」</h3>

<p>
日 本 で 育 っ た ほ ぼ 全 て の 人 が、 一 度 は 歌 っ た こ と が あ る だ ろ う 唱 歌『春 が 来 た』。
明 治 43 年 (1910) に 発 表 さ れ て か ら、 115 年 ─ 三 世 代 を 超 え て 歌 い 継 が れ て き た。
</p>

<p>
シ ン プ ル で 覚 え や す い 歌 詞、 美 し く 静 か な メ ロ デ ィ ー、 そ し て 日 本 人 の 春 の 風 景 そ の も の。
中 1 音 楽 で、 こ の 名 唱 歌 を 深 く 味 わ お う。
</p>

<h3>高 野 辰 之 ─ 信 州 の 国 文 学 者</h3>

<p>
作 詞 者 ・ 高 野 辰 之 (た か の た つ ゆ き ・ 1876–1947) は、 長 野 県 中 野 市 (現 ・ 旧 永 江 村)
の 山 村 で 生 ま れ た。 信 州 の 山 と 田 園 の 風 景 が、 後 の 唱 歌 の 多 く の 原 風 景 と な っ た。
</p>

<p>
東 京 帝 国 大 学 で 国 文 学 を 学 び、 文 部 省 国 語 教 科 書 編 纂 委 員 と し て 活 躍。 国 語 と 唱 歌 の
両 方 を 通 し て、 日 本 の 子 ど も の 教 育 に 大 き く 貢 献 し た。
</p>

<p>
代 表 作 ─ 『故 郷』『朧 月 夜』『紅 葉』『春 が 来 た』。 い ず れ も 四 季 ・ 故 郷 を 歌 っ た 名 唱 歌。
</p>

<h3>岡 野 貞 一 ─ 鳥 取 か ら 東 京 音 楽 学 校 へ</h3>

<p>
作 曲 者 ・ 岡 野 貞 一 (お か の て い い ち ・ 1878–1941) は、 鳥 取 市 生 ま れ。 東 京 音 楽 学 校
(現 東 京 芸 術 大 学) で 西 洋 音 楽 を 学 び、 後 に 同 校 助 教 授。
</p>

<p>
日 本 古 来 の 旋 律 美 と 西 洋 音 楽 の 構 造 を 融 合 し た、 美 し い 唱 歌 を 多 数 作 曲。 高 野 辰 之 と
組 ん で 数 多 く の 名 曲 を 生 ん だ。
</p>

<h3>な ぜ 作 詞・作 曲 者 名 が 公 表 さ れ な か っ た の か</h3>

<p>
明 治 ~ 大 正 ~ 昭 和 戦 前 ま で、 文 部 省 唱 歌 は 作 詞・作 曲 者 名 を 公 表 し な か っ た。 理 由 は ─
</p>

<ul>
  <li>「国 民 共 通 の 文 化」 と し て、 個 人 著 作 物 を 超 え た 公 共 財 と し て 扱 う 為</li>
  <li>歌 詞 や 旋 律 を 必 要 に 応 じ て 改 訂 す る 為 (作 者 が い る と 変 え に く い)</li>
</ul>

<p>
こ の 為、 戦 後 ま で『春 が 来 た』『故 郷』『紅 葉』 等 の 作 者 が 不 明 と さ れ て き た。 戦 後 の 研 究 で、
高 野 辰 之 ・ 岡 野 貞 一 コ ン ビ の 業 績 で あ る こ と が 判 明 し た。
</p>

<h3>『春 が 来 た』 の 歌 詞 構 造 ─ 三 番 の 場 面 展 開</h3>

<p>
『春 が 来 た』 の 歌 詞 は、 三 番 構 成 で 場 面 が 移 り 変 わ る ─
</p>

<ul>
  <li><strong>一 番</strong> ─ 「山」 に 春 が 来 た</li>
  <li><strong>二 番</strong> ─ 「里」 に 春 が 来 た</li>
  <li><strong>三 番</strong> ─ 「野」 に 春 が 来 た</li>
</ul>

<p>
山 → 里 → 野 ─ 高 い と こ ろ か ら 低 い と こ ろ へ、 視 線 が 下 り て く る。 春 は、 山 か ら 始 ま っ て、
里 を 通 り、 野 に 降 り る ─ 日 本 人 の 自 然 観 が 凝 縮 さ れ て い る。
</p>

<h3>「ど こ に 来 た」 の 反 復 ─ シ ン プ ル の 美</h3>

<p>
「春 が 来 た 春 が 来 た ど こ に 来 た」 ─ 同 じ 言 葉 が 繰 り 返 さ れ る。 子 ど も に も 覚 え や す く、
リ ズ ム も 良 い。
</p>

<p>
そ し て 「山 に 来 た 里 に 来 た 野 に も 来 た」 ─ こ の シ ン プ ル さ が、 名 唱 歌 と し て 100 年
歌 い 継 が れ る 力 と な っ て い る。
</p>

<h3>音 楽 的 特 徴 ─ 二 部 形 式・四 拍 子・ハ 長 調</h3>

<p>
『春 が 来 た』 は、 二 部 形 式 (A-A')、 四 拍 子、 ハ 長 調 で 書 か れ て い る。 シ ン プ ル で、
子 ど も で も 覚 え や す く、 歌 い や す い 構 造。
</p>

<p>
メ ロ デ ィ ー の 上 下 が ゆ る や か で、 山 か ら 里 へ 降 り る 風 景 と 重 な る。 詞 と 曲 が 一 体 と な っ た
名 曲。
</p>

<h3>結 び ─ 君 の「春 が 来 た」 を 探 そ う</h3>

<p>
115 年 前、 高 野 辰 之 が 信 州 の 山 村 で 見 た 春 ─ 雪 解 け、 桜、 鳥 の 声、 田 植 え の 準 備 ─ は、
君 が 今 見 て い る 春 と 同 じ だ ろ う か、 違 う だ ろ う か。
</p>

<p>
教 室 で『春 が 来 た』 を 唄 う 時、 100 年 前 の 日 本 の 子 ど も た ち と、 同 じ 旋 律 を 共 有 し て い る
こ と を 思 い 出 そ う。 唱 歌 は、 時 代 を 超 え て 心 を つ な ぐ 文 化 で あ る。
</p>
