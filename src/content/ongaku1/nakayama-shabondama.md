<h3>歌 詞 (野 口 雨 情、 1922)</h3>

<blockquote class="song">
<p>
シ ャ ボ ン 玉 飛 ん だ<br>
屋 根 ま で 飛 ん だ<br>
屋 根 ま で 飛 ん で<br>
こ わ れ て 消 え た
</p>

<p>
シ ャ ボ ン 玉 消 え た<br>
飛 ば ず に 消 え た<br>
生 ま れ て す ぐ に<br>

</p>

<p>

</p>

<p>
詞 の 詳 し い 解 説 は 本 サ イ ト <strong>kokugo2/noguchi-shabondama</strong> で 既 に 学 ん だ (野 口 雨 情 の 亡 き 娘・み ど り を 偲 ぶ 詞)。<br>
こ こ で は、 中 山 晋 平 が こ の 詞 に 付 け た 曲 と、 中 山 と い う 作 曲 家 を 学 ぶ。
</p>

<h3>中 山 晋 平 ─ 大 正 童 謡 三 大 作 曲 家 の 一 人</h3>

<p>
中 山 晋 平 (1887–1952) は、 長 野 県 中 野 市 (旧 ・ 日 野 村) の 農 家 の 出 身。<br>
苦 学 し て 東 京 音 楽 学 校 (現 ・ 東 京 藝 大) を 卒 業 し、 そ の 後 多 数 の 童 謡 ・ 流 行 歌 を 作 曲 し た。
</p>

<p>
彼 の 代 表 作:
</p>

<ul>
<li><strong>童 謡</strong> ─ 「シ ャ ボ ン 玉」「証 城 寺 の 狸 囃 子」「て る て る ぼ う ず」「肩 た た き」「ア メ フ リ」</li>
<li><strong>流 行 歌</strong> ─ 「カ チ ュ ー シ ャ の 唄」(1914)、「東 京 行 進 曲」(1929)、「東 京 音 頭」(1933)</li>
</ul>

<p>
生 涯 で 作 曲 し た 曲 は 約 <strong>3000 曲</strong>。 大 正・昭 和 期 の 日 本 を 音 楽 で 彩 っ た 巨 匠。
</p>

</p>

</p>

<ul>

<li>本 居 長 世 は <strong>物 語 性 ・ 抒 情</strong> ─ 詞 の 物 語 を 丁 寧 に 音 で 描 く</li>
<li><strong>中 山 晋 平 は 親 し み や す さ ・ 庶 民 性</strong> ─ 誰 で も す ぐ 口 ず さ め る シ ン プ ル な メ ロ デ ィ ー</li>
</ul>

<h3>「シ ャ ボ ン 玉」 の メ ロ デ ィ ー</h3>

<p>
中 山 晋 平 の「シ ャ ボ ン 玉」 の メ ロ デ ィ ー は、 と て も シ ン プ ル で、 子 ど も で も す ぐ 覚 え ら れ る。
</p>

<p>
で も、 そ の シ ン プ ル さ の 中 に ─<br>
最 初 の「シ ャ ボ ン 玉 飛 ん だ」 の 軽 や か な 上 昇、<br>
「こ わ れ て 消 え た」 の 静 か な 下 降、<br>
そ し て 最 後 の「飛 ば そ」 の 切 な い 願 い ─
</p>

<p>
詞 の 物 語 を、 メ ロ デ ィ ー が ぴ た り と 表 現 し て い る。
</p>

<h3>中 山 晋 平 の そ の 後</h3>

<p>
中 山 晋 平 は、 童 謡 だ け で な く 大 正・昭 和 の 流 行 歌 を 多 数 作 曲。<br>
「東 京 行 進 曲」(1929)、「東 京 音 頭」(1933) な ど、 当 時 の 日 本 中 で 歌 わ れ た 国 民 的 ヒ ッ ト 曲 を 生 ん だ。
</p>

<p>
庶 民 の 心 を つ か む メ ロ デ ィ ー ─ そ れ が 中 山 晋 平 の 持 ち 味 だ っ た。<br>
1952 年、 65 歳 で 没。
</p>

<h3>歌 い 継 が れ る「シ ャ ボ ン 玉」</h3>

<p>
野 口 雨 情 の 詞 (1922) と 中 山 晋 平 の 曲 (1923) で 完 成 し た「シ ャ ボ ン 玉」 は、 100 年 経 っ た 今 も、 日 本 中 の 子 ど も た ち が 歌 う 童 謡 と し て、 静 か に 生 き 続 け て い る。
</p>

<p>
シ ャ ボ ン 玉 を 吹 い て、 屋 根 ま で 飛 ぶ 瞬 間 ─ そ の 一 瞬 に 100 年 前 の 二 人 の 仕 事 が、 蘇 る の だ。
</p>
