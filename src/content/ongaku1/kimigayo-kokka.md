## 君 が 代 ─ 1000 年 の 旅

### 1. 平 安 時 代 905 年 の 一 首

時 は 平 安 前 期、 905 年 (延 喜 5 年)。 醍 醐 天 皇 の 勅 命 で 紀 貫 之 (き の つ ら ゆ き) ら が、 日 本 最 初 の 勅 撰 和 歌 集『古 今 和 歌 集』 を 編 ん だ。 全 20 巻、 約 1100 首。

そ の 巻 七「賀 歌 (が の う た)」 に、 詠 み 人 知 ら ず の 一 首 が 載 っ た。

「わ が 君 は 千 代 に 八 千 代 に さ ざ れ 石 の 巌 と な り て 苔 の む す ま で」

「あ な た の 命 が、 千 代 ・ 八 千 代 と い う 長 い 時 間、 小 石 (さ ざ れ 石) が 大 き な 岩 (巌) と な り、 そ の 岩 に 苔 が 生 え る ま で、 続 き ま す よ う に」 と い う 長 寿 の 祝 歌 で あ っ た。

そ の 後、 平 安 後 期 の『和 漢 朗 詠 集』 (1018 年 頃) で、 「わ が 君 は」 が「君 が 代 は」 に 改 変 さ れ た。 こ れ が 現 代 ま で 続 く 形 で あ る。

***

### 2. 1880 年 ─ 国 歌 へ

時 は 流 れ て、 明 治 13 年 (1880 年)。 文 明 開 化 の 中、 西 洋 諸 国 と の 外 交 で「日 本 の 国 歌」 が 必 要 と さ れ た。

1869 年 に 海 軍 軍 楽 隊 教 師 ジ ョ ン・ウ ィ リ ア ム・フ ェ ン ト ン (英 国 人) が 一 度 旋 律 を 作 っ た が、 「日 本 的 で な い」 と し て 改 訂 が 求 め ら れ た。

そ こ で 1880 年、 宮 内 省 雅 楽 課 長 林 広 守 (1831-1896) と そ の 部 下 奥 好 義 (1858-1933) ら 数 名 が、 平 安 時 代 か ら 続 く「君 が 代 は…」 の 歌 詞 に、 雅 楽 (古 代 宮 廷 音 楽) の 律 旋 法 (り っ せ ん ぽ う) で 旋 律 を 作 曲 し た。

そ れ を、 当 時 来 日 し て い た ド イ ツ 人 音 楽 顧 問 フ ラ ン ツ・エ ッ ケ ル ト (1852-1916) が、 西 洋 音 楽 風 の 四 声 (ソ プ ラ ノ・ア ル ト・テ ノ ー ル・バ ス) の 合 唱 と 軍 楽 隊 演 奏 用 に 編 曲 し た。

歌 詞 は 平 安 時 代 905 年。 旋 律 は 1880 年。 編 曲 は 1880 年 (ド イ ツ 人 編 曲)。 三 つ の 層 が 1000 年 の 時 を 越 え て 重 な っ た 国 歌 が、 こ こ に 完 成 し た。

***

### 3. 1888 年 ─ 海 軍 で の 演 奏 と 普 及

1888 年 (明 治 21 年)、 エ ッ ケ ル ト 編 曲 の『君 が 代』 が、 海 軍 軍 楽 隊 で 演 奏 さ れ、 同 年 各 国 公 使 館 に 配 布 さ れ た。 こ れ で 国 際 的 に も「日 本 の 国 歌」 と し て 認 知 さ れ 始 め た。

1893 年 に は 文 部 省 が 小 学 校 唱 歌 と し て 学 校 で 教 え る よ う 通 達。 以 来、 戦 前・戦 中・戦 後 を 通 じ て、 日 本 の 学 校 と 公 的 行 事 で 演 奏 さ れ 続 け た。

***

### 4. 1999 年 ─ 国 旗 国 歌 法 で 正 式 化

ただ、 興 味 深 い こ と に、 こ の 約 120 年 間、『君 が 代』 は 法 律 上 の「国 歌」 で は な か っ た。 「事 実 上 の 国 歌」 と し て 演 奏 さ れ 続 け て い た だ け で、 法 的 根 拠 が な か っ た。

1999 年 (平 成 11 年) 8 月 9 日、 国 会 で『国 旗 国 歌 法』 が 制 定。 「日 章 旗」 を 国 旗、「君 が 代」 を 国 歌 と 正 式 に 法 定 化。 1880 年 か ら 約 120 年 を 経 て、 法 的 に 国 歌 と な っ た。

こ の 法 制 化 に は、 戦 中 の 軍 国 主 義 と の 連 想 か ら 反 対 意 見 も 強 く、 国 会 で 多 くの 議 論 を 経 て 成 立 し た。 現 在 で も、 国 歌・国 旗 に つ い て は 様 々 な 立 場 が あ る。

***

### 5. 1000 年 の 旅 と 静 か な 響 き

『君 が 代』 の 旋 律 は、 ヨ ー ロ ッ パ 諸 国 の 力 強 い 行 進 曲 風 国 歌 (フ ラ ン ス「ラ・マ ル セ イ エ ー ズ」、 英 国「ゴ ッ ド・セ イ ブ・ザ・キ ン グ」 等) と は 異 な り、 雅 楽 の 律 旋 法 を 用 い た 静 か で 厳 か な 響 き で あ る。

そ し て 歌 詞 は、 軍 事 行 進 曲 で は な く、 平 安 時 代 の 雅 な 賀 歌。 「あ な た の 命 が 永 く 続 き ま す よ う に」 と い う 静 か な 祝 い の 言 葉。

歌 詞 905 年。 旋 律 1880 年。 編 曲 1880 年。 法 定 化 1999 年。 約 1100 年 の 旅 を 経 て、 31 文 字 の 短 歌 が 今 も 学 校 と 公 的 行 事 で 歌 わ れ て い る。

世 界 で も 珍 し い、 古 代 ・ 近 世 ・ 近 代 の 三 文 化 が 一 つ の 国 歌 に 重 な っ た 稀 有 な 例 で あ る。
