## 故 郷

高 野 辰 之 作 詞 / 岡 野 貞 一 作 曲 (1914)

### 一

兎 (う さ ぎ) 追 ひ し か の 山、
小 鮒 (こ ぶ な) 釣 り し か の 川。
夢 は 今 も め ぐ り て、
忘 れ が た き 故 郷 (ふ る さ と)。

### 二

如 何 (い か) に い ま す 父 母 (ち ち は は)、
恙 (つ つ が) な し や 友 が き。
雨 に 風 に つ け て も、
思 ひ 出 (い) づ る 故 郷。

### 三

志 (こ こ ろ ざ し) を 果 た し て、
い つ の 日 に か 帰 ら ん。
山 は 青 き 故 郷、
水 は 清 き 故 郷。

***

## 教 材 解 説

### 詩 の 構 成 ─ 過 去・現 在・未 来

『故 郷』 は 三 番 で 完 結 す る 構 成 を 持 つ。 一 番 は「過 去」、 二 番 は「現 在」、 三 番 は「未 来」 を 歌 う。

**一 番** で は、 兎 を 追 い、 小 鮒 を 釣 っ た 幼 少 期 の 風 景 が、 今 も 夢 の 中 で 蘇 る と 歌 う。 過 去 の 記 憶 と し て の 故 郷。

**二 番** で は、 都 で 暮 ら す「私」 が、 故 郷 に 残 し て き た 父 母・友 を 「今、 ど う 過 ご し て い ま す か」 と 案 じ る。 現 在 の 思 慕 と し て の 故 郷。

**三 番** で は、 い つ か 自 分 の 志 を 果 た し て、 故 郷 へ 帰 り た い と 願 う。 未 来 の 約 束 と し て の 故 郷。 「山 は 青 き、 水 は 清 き」 と 締 め く く る 美 し い 反 復 リ ズ ム が、 故 郷 の 永 遠 性 を 印 象 づ け る。

### 旋 律 ─ ヘ 長 調 4 分 の 3 拍 子

岡 野 貞 一 の 旋 律 は、 ヘ 長 調 4 分 の 3 拍 子。 16 小 節 の 簡 潔 な 構 造 で、 子 ど も で も 歌 い や す く、 か つ 大 人 の 心 に も 深 く 染 み 入 る。 一 番 か ら 三 番 ま で 同 じ 旋 律 を 繰 り 返 す が、 歌 詞 が「過 去 → 現 在 → 未 来」 と 進 む た め、 同 じ 旋 律 が 聞 こ え 方 を 変 え て い く。

### 高 野 辰 之 の 故 郷・信 州

高 野 辰 之 自 身 が 長 野 県 永 江 村 (現 中 野 市) の 出 身 で あ り、 こ の 詞 の「か の 山」「か の 川」 に は、 信 州 の 山 河 が 投 影 さ れ て い る と 言 わ れ る。 し か し、 詩 が 普 遍 性 を 持 ち、 日 本 中 の 人 が 自 分 の 故 郷 を 思 い 浮 か べ ら れ る の は、 高 野 が 具 体 的 な 地 名 を 出 さ ず、 普 遍 的 な 言 葉 で 描 い た か ら だ。

### 100 年 を 超 え る 名 曲

1914 年 の 発 表 以 来、 戦 中・戦 後 を 通 じ て 教 科 書 か ら 一 度 も 消 え ず、 今 も 中 学 校 共 通 教 材 (歌 唱 共 通 教 材) と し て 全 国 の 中 学 生 が 歌 う。 2 0 1 1 年 の 東 日 本 大 震 災 の あ と、 被 災 地 の 復 興 イ ベ ン ト で『故 郷』 が 歌 わ れ る こ と が 多 く、 「失 わ れ た 故 郷 を 思 う 歌」 と し て 新 し い 意 味 を 帯 び 始 め て い る。
