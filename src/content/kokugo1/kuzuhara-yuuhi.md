<h3>「ぎ ん ぎ ん ぎ ら ぎ ら」 と い う 発 明</h3>

<p>
こ の 詩 の 命 は、 「<strong>ぎ ん ぎ ん ぎ ら ぎ ら</strong>」 と い う 一 言 で あ る。
</p>

<p>
こ れ は、 葛 原 し げ る が 作 っ た、 ま っ た く 新 し い 言 葉 で あ る。<br>
夕 日 が、 ま ぶ し く 強 く 輝 き な が ら 沈 ん で い く 様 子 を、 「ぎ ん」 (銀 色 の 光) と 「ぎ ら ぎ ら」 (強 く 光 る) を 組 み 合 わ せ て 表 現 し た。
</p>

<p>
こ の 言 葉 を 声 に 出 す と、 ど う し て だ ろ う、 体 が 自 然 に 動 き 出 す。<br>
リ ズ ム が と て も 楽 し く、 一 度 聞 い た ら 忘 れ ら れ な い。
</p>

<p>
そ し て、 こ の リ ズ ム を 通 し て、 夕 日 の <strong>「強 く・大 き く・赤 く」</strong> と い う 印 象 が、 子 ど も の 心 に 焼 き 付 く の だ。
</p>

<h3>葛 原 し げ る と い う 詩 人</h3>

<p>
葛 原 し げ る (1886–1961) は、 広 島 県 福 山 市 出 身 の 童 謡 詩 人 で あ る。
</p>

<p>
彼 は 北 原 白 秋・野 口 雨 情・西 條 八 十 ・ 三 木 露 風 ら と 並 ぶ、 大 正 期 の 童 謡 作 家 の 一 人。<br>
「夕 日」 (1921) の ほ か、「と ん び」「お べ ん と う」「と ん か ら り」 な ど 多 く の 童 謡 を 残 し た。
</p>

<p>
特 に 「夕 日」 は、 1923 年 (大 正 12 年) に 作 曲 家 ・ 室 崎 琴 月 (む ろ ざ き き ん げ つ) が 曲 を 付 け、 全 国 の 幼 稚 園 ・ 小 学 校 で 歌 わ れ る よ う に な っ た。<br>
100 年 後 の 今 も、 日 本 人 が 親 し む 国 民 的 童 謡 と し て 生 き 続 け て い る。
</p>

<h3>「烏 よ、 お う ち は あ ち ら だ よ」</h3>

<p>
詩 の 後 半、 詩 人 は 「<strong>烏 (か ら す) よ、 お う ち は あ ち ら だ よ</strong>」 と 烏 に 呼 び か け る。
</p>

<p>
日 が 沈 む 夕 暮 れ。 空 を 飛 ぶ 烏 た ち も、 自 分 の 巣 (=「お う ち」) に 帰 る 時 間 で あ る。<br>
詩 人 は、 「夕 焼 け 小 焼 け の あ ち ら」 ─ 西 の 空 ─ を 指 し て、 烏 に 「家 は そ こ だ よ」 と 教 え て あ げ る。
</p>

<p>
こ こ に、 葛 原 が 描 く <strong>自 然 へ の や さ し い 眼 差 し</strong> が あ る。<br>
人 間 だ け で な く、 烏 も ま た 「お う ち」 を 持 ち、 そ こ へ 帰 っ て い く。<br>
そ ん な 自 然 の 営 み を、 子 ど も と 共 有 し よ う と す る 詩 で あ る。
</p>

<h3>歌 っ て み よ う</h3>

<p>
こ の 詩 は、 室 崎 琴 月 の メ ロ デ ィ ー で 歌 わ れ る こ と が 多 い。<br>
お う ち の 人 と 一 緒 に、 夕 日 を 見 な が ら、 こ の 歌 を 歌 っ て み よ う。
</p>

<p>
「ぎ ん ぎ ん ぎ ら ぎ ら、 夕 日 が 沈 む ─」<br>
そ の 瞬 間、 100 年 前 の 葛 原 し げ る が 見 た 夕 日 と、 君 が 今 見 て い る 夕 日 が、 つ な が る の だ。
</p>
