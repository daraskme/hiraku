<h3>朝 焼 け の 浜 か ら ─ 全 8 行 の 詩</h3>

<blockquote>
<p>
朝 焼 小 焼 だ<br>
大 漁 だ<br>
大 ば い わ し の<br>
大 漁 だ。
</p>
<p>
浜 は ま つ り の<br>
よ う だ け ど<br>
海 の な か で は<br>
何 万 の<br>
鰯 の と む ら い<br>
す る だ ろ う。
</p>
</blockquote>

<h3>仙 崎 の 浜 ─ み す ゞ が 育 っ た 場 所</h3>

<p>
金 子 み す ゞ (1903–1930) は、 山 口 県 仙 崎 (現 長 門 市) と い う 港 町 で 生 ま れ 育 ち ま し た。
仙 崎 は 日 本 海 に 面 し た 小 さ な 漁 港 で、 鰯 (い わ し) や 鯨 (く じ ら) の 漁 で 賑 わ う 町 で し た。
</p>

<p>
み す ゞ は 子 ど も の 頃、 漁 師 さ ん た ち が「大 漁 だ!」 と 喜 ぶ 場 面 を 何 度 も 見 た で し ょ う。
浜 で は 大 漁 旗 が ひ ら め き、 漁 師 さ ん や 家 族 が 嬉 し そ う に 魚 を 数 え て い ま す。
</p>

<h3>で も、 海 の 中 は ど う だ ろ う か?</h3>

<p>
詩 の 後 半 で、 み す ゞ は 視 点 を 変 え ま す ─ 海 の 中 で は、 ど う な っ て い る だ ろ う か?
</p>

<p>
鰯 の 群 れ は、 仲 間 が 何 万 匹 も 突 然 い な く な る。 残 さ れ た 鰯 た ち は、 仲 間 の こ と を 悲 し ん で
「弔 い (と む ら い ・ 葬 式)」 を し て い る か も し れ な い。
</p>

<p>
「浜 の 喜 び」 と「海 の 悲 し み」 ─ 同 じ 出 来 事 が、 二 つ の 異 な る 顔 を 見 せ る。 こ れ が み す ゞ の
教 え る「視 点 を 変 え る」 こ と の 意 味 で す。
</p>

<h3>「い た だ き ま す」 の 意 味</h3>

<p>
給 食 を 食 べ る 前 に「い た だ き ま す」 と 言 い ま す ね。 こ れ は 単 な る 挨 拶 で は な く、
「命 を い た だ き ま す」 と い う 意 味 で す。
</p>

<p>
魚 ・ 肉 ・ 野 菜 ─ 君 が 食 べ る も の は 全 て、 か つ て 生 き て い た 命 で す。 金 子 み す ゞ の『大 漁』
を 読 ん だ 後、 「い た だ き ま す」 の 言 葉 が、 少 し 重 く 心 に 響 く か も し れ ま せ ん。
</p>

<h3>結 び ─ 優 し い 詩 人 か ら の 贈 り も の</h3>

<p>
26 歳 で 亡 く な っ た 詩 人 金 子 み す ゞ。 短 い 生 涯 で 約 512 編 の 童 謡 を 残 し ま し た。 そ の 全 て
に 共 通 す る の は、 「弱 い も の・小 さ な も の・見 え な い も の」 へ の 優 し い 眼 差 し で す。
</p>

<p>
『大 漁』 と い う わ ず か 8 行 の 詩 が、 100 年 経 っ た 今 も 国 語 の 教 科 書 で 読 ま れ 続 け て い る の は、
こ の 優 し さ が 時 代 を 越 え て 心 に 響 く か ら で す。
</p>
