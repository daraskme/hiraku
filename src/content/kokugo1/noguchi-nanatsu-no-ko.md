<h3>げ ん だ い ご に な お す と</h3>

<p>
烏 (か ら す) は、 ど う し て 「カ ア カ ア」 と 啼 (な) い て い る の で し ょ う か。<br>
そ れ は ね、 山 (や ま) の 巣 (す) の 中 に、 か わ い い 七 つ の 子 ど も (子 烏) が い る か ら な の で す。
</p>

<p>
「か わ い い、 か わ い い」 と、 母 烏 は 啼 い て い ま す。<br>
「私 の 子 ど も は、 か わ い い、 か わ い い」 と、 啼 い て い る の で す よ。
</p>

<p>
山 の 古 (ふ る) い 巣 へ、 行 っ て 見 て ご ら ん。<br>
丸 (ま る) い、 つ ぶ ら な 目 を し た、 可 愛 い 子 烏 が、 親 を 待 っ て い ま す よ。
</p>

<h3>野口雨情 の こ こ ろ</h3>

<p>
野口雨情 (の ぐ ち う じ ょ う) は、 茨 城 県 磯 原 (い そ は ら) の、 海 の そ ば の 町 で 生 ま れ ま し た。<br>
若 い と き 上 京 し、 早 稲 田 大 学 で 学 び、 詩 人 と し て の 道 を 歩 み 始 め ま し た。
</p>

<p>
し か し 雨 情 の 人 生 は、 必 ず し も 順 調 で は あ り ま せ ん で し た。<br>
最 初 の 結 婚 は う ま く い か ず、 経 済 的 に も 苦 し い 時 期 が 続 き ま し た。<br>
そ う し た 苦 し み の 中 で、 彼 は <strong>こ の 詩 を 書 い た</strong> と 伝 え ら れ て い ま す。
</p>

<p>
「カ ア カ ア」 と 啼 く 烏 の 声 ─ そ れ を 雨 情 は、 「子 ど も を 思 う 親 の 声」 と し て 聴 い た の で す。<br>
そ し て こ う 思 い ま し た: <strong>「あ の 黒 い、 怖 い と 思 わ れ て い る 烏 も、 自 分 の 子 ど も を 大 切 に 思 う、 や さ し い 親 な の だ」</strong> と。
</p>

<h3>「七 つ の 子」 と は</h3>

<p>
さ て、 こ の 「七 つ の 子」 と い う 言 葉 は、 い ま で も 議 論 さ れ る な ぞ で す。
</p>

<ul>
<li><strong>説 1: 7 歳 の 子 ど も</strong> ─ 「7 歳 の 子 ど も が 一 人 い る」 と い う 意 味。</li>
<li><strong>説 2: 7 羽 の 子 烏</strong> ─ 「7 羽 の 子 ど も 烏 が い る」 と い う 意 味。</li>
</ul>

<p>
野 口 雨 情 本 人 は、 ど ち ら と も は っ き り 答 え ま せ ん で し た。<br>
だ か ら 100 年 以 上 経 っ た 今 も、 こ の 詩 は <strong>あ な た 自 身 の 想 像 で 読 む</strong> こ と が で き る の で す。<br>
さ あ、 君 は ど ち ら だ と 思 い ま す か?
</p>

<h3>歌 と な っ て、 日 本 中 で</h3>

<p>
こ の 詩 は、 1921 年 (大 正 10 年) に 雑 誌 『金 の 船』 で 発 表 さ れ ま し た。<br>
同 じ 年、 作 曲 家 の <strong>本 居 長 世</strong> (も と お り な が よ) が、 や さ し く て 美 し い メ ロ デ ィ ー を 付 け ま し た。
</p>

<p>
そ し て、 「七 つ の 子」 は 日 本 中 の 子 ど も た ち の 歌 と な り ま し た。<br>
2007 年 に 文 化 庁 が 行 っ た 「日 本 の う た 100 選」 で も 選 ば れ た、 国 民 的 童 謡 で す。
</p>

<p>
100 年 以 上 前 に、 自 分 の 苦 し み の 中 で 雨 情 が 書 い た 一 篇 が、 今 も 日 本 中 の 母 と 子 を 結 ぶ 歌 と し て 響 い て い る ─ 詩 の 力 は、 こ う し て 時 代 を 越 え て 伝 わ っ て い く の で す。
</p>
