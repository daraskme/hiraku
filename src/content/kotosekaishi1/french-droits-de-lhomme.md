<h3>原 文 (抜 粋・1789 年 8 月 26 日)</h3>

<blockquote>
<p>
<strong>第 1 条</strong> ─ 人 は 生 ま れ な が ら に し て 自 由 で あ り、 権 利 に お い て 平 等 で あ る。 社 会 的 区 別 は、 公 共 の 利 益 に 基 づ い て の み 設 け ら れ る こ と が で き る。
</p>

<p>
<strong>第 2 条</strong> ─ あ ら ゆ る 政 治 的 結 合 の 目 的 は、 人 の 持 つ 自 然 的 か つ 不 可 侵 の 権 利 を 守 る こ と に あ る。 そ の 権 利 と は、 自 由・所 有・安 全・圧 政 へ の 抵 抗 で あ る。
</p>

<p>
<strong>第 3 条</strong> ─ あ ら ゆ る 主 権 の 源 泉 は、 本 質 的 に 国 民 に 由 来 す る。 い か な る 集 団 ・ 個 人 も、 国 民 か ら 明 示 的 に 由 来 し な い 権 力 を 行 使 す る こ と は で き な い。
</p>

<p>
<strong>第 4 条</strong> ─ 自 由 と は、 他 者 を 害 さ な い 全 て を 行 う こ と が で き る こ と で あ る。
</p>

<p>
<strong>第 5 条</strong> ─ 法 律 は、 社 会 に 有 害 な 行 為 の み を 禁 止 す る こ と が で き る。 法 律 で 禁 止 さ れ て い な い こ と は 妨 げ ら れ ず、 法 律 で 命 じ ら れ て い な い こ と を 強 制 さ れ る こ と は な い。
</p>

<p>
<strong>第 6 条</strong> ─ 法 律 は 一 般 意 思 の 表 明 で あ る。 全 て の 市 民 は、 自 ら ま た は 代 表 者 を 通 じ て、 そ の 形 成 に 参 加 す る 権 利 が あ る。 (中 略)
</p>

<p>
<strong>第 10 条</strong> ─ 何 人 も、 そ の 意 見 ─ 宗 教 上 の 意 見 を 含 め ─ ゆ え に 不 安 を 抱 か せ ら れ る こ と が あ っ て は な ら な い。 た だ し、 そ の 表 明 が 法 律 に 定 め ら れ た 公 共 の 秩 序 を 乱 さ な い 限 り に お い て。
</p>

<p>
<strong>第 11 条</strong> ─ 思 想 と 意 見 の 自 由 な 伝 達 は、 人 の 最 も 貴 重 な 権 利 の 一 つ で あ る。 (中 略)
</p>

<p>
<strong>第 16 条</strong> ─ 権 利 の 保 障 が 確 保 さ れ ず、 権 力 分 立 が 規 定 さ れ て い な い 社 会 は、 憲 法 を 持 つ も の で は な い。
</p>

<p>
<strong>第 17 条</strong> ─ 所 有 権 は、 神 聖 に し て 不 可 侵 で あ る。 (以 下 略)
</p>
</blockquote>

<h3>未 完 の 約 束 ─ 「人」 と 「市 民」 の 区 別</h3>

<p>
こ の 宣 言 の 正 式 名 は 「<strong>人 及 び 市 民 の 権 利 宣 言</strong>」 で あ る。
</p>

<p>
「人」 (Homme) と 「市 民」 (Citoyen) の 区 別 ─ こ こ に、 後 の 多 く の 議 論 が 隠 さ れ て い る。
</p>

<ul>
<li>「人」 ─ 全 て の 人 間 が 持 つ 自 然 権 (生 ま れ な が ら の 自 由・平 等)</li>
<li>「市 民」 ─ 政 治 共 同 体 に 参 加 す る 者 (選 挙 権・公 職 就 任 権)</li>
</ul>

<p>
だ が、 1789 年 当 時、 「市 民」 に 含 ま れ た の は <strong>白 人 男 性 の 一 部</strong> だ け で あ っ た。
</p>

<ul>
<li>女 性 ─「市 民」 で は な い (フ ラ ン ス で 女 性 参 政 権 は 1944 年 ま で 待 つ)</li>
<li>植 民 地 の 黒 人 ─「市 民」 で は な い (奴 隷 制 は 1848 年 ま で 続 く)</li>
<li>無 産 階 級 ─ 選 挙 権 に 財 産 制 限 (1848 年 の 普 通 選 挙 ま で 待 つ)</li>
</ul>

<h3>オ ラ ン プ・ド・グ ー ジ ュ の 反 論</h3>

<p>
1791 年、 フ ラ ン ス の 女 性 思 想 家 <strong>オ ラ ン プ・ド・グ ー ジ ュ</strong> (Olympe de Gouges, 1748–1793) は、 こ の 「人 権 宣 言」 が 男 性 だ け の 権 利 を 主 張 し て い る こ と に 抗 議 し、 「<strong>女 性 と 女 性 市 民 の 権 利 宣 言</strong>」 を 起 草 し た。
</p>

<p>
彼 女 は、 オ リ ジ ナ ル の 17 条 を そ の ま ま 「女 性」 に 置 き 換 え る 形 で、 こ う 書 い た:
</p>

<blockquote>
「女 性 は、 生 ま れ な が ら に し て 自 由 で あ り、 男 性 と 権 利 に お い て 平 等 で あ る」
</blockquote>

<p>
し か し グ ー ジ ュ は、 1793 年、 革 命 政 府 (ジ ャ コ バ ン 派 の 恐 怖 政 治) に よ っ て ギ ロ チ ン で 処 刑 さ れ た。<br>
「女 性 の 権 利」 が フ ラ ン ス で 法 的 に 認 め ら れ る ま で、 そ こ か ら さ ら に 150 年 か か る こ と に な る。
</p>

<h3>世 界 へ の 波 及</h3>

<p>
フ ラ ン ス 人 権 宣 言 の 影 響 は、 そ の 後 の 世 界 史 に 巨 大 な 波 紋 を 広 げ た。
</p>

<ul>
<li>1791 ─ ハ イ チ 革 命 (黒 人 奴 隷 が 「人 権」 を 武 器 に 反 乱)</li>
<li>1810s ─ ラ テ ン ア メ リ カ 諸 国 の 独 立</li>
<li>1848 ─ フ ラ ン ス 第 二 共 和 政 で 男 子 普 通 選 挙</li>
<li>1889 ─ 大 日 本 帝 国 憲 法 (こ の 思 想 を 一 部 受 容)</li>
<li>1948 ─ 国 連 「世 界 人 権 宣 言」</li>
</ul>

<p>
国 連 世 界 人 権 宣 言 (1948) の 第 1 条 は ─<br>
<strong>「す べ て の 人 間 は、 生 ま れ な が ら に し て 自 由 で あ り、 か つ、 尊 厳 と 権 利 と に つ い て 平 等 で あ る」</strong>。<br>
こ れ は、 フ ラ ン ス 人 権 宣 言 第 1 条 の 直 接 の 子 孫 で あ る。
</p>

<h3>日 本 と の つ な が り</h3>

<p>
日 本 で は、 明 治 初 期 の 自 由 民 権 運 動 (1874 年 民 撰 議 院 設 立 建 白 書 ─ 本 サ イ ト kotorekishi1) が、 フ ラ ン ス 人 権 宣 言 の 思 想 か ら 影 響 を 受 け た と さ れ る。
</p>

<p>
そ し て 1947 年 の 日 本 国 憲 法 ─ 「侵 す こ と の で き な い 永 久 の 権 利 と し て、 現 在 及 び 将 来 の 国 民 に 与 へ ら れ る」 (第 11 条) ─ も、 そ の 思 想 的 系 譜 上 に あ る。
</p>

<h3>237 年 後 の 私 た ち へ</h3>

<p>
1789 年 か ら 237 年。<br>
「自 由」「平 等」「人 権」 ─ こ れ ら の 言 葉 を、 私 た ち は 当 た り 前 と 思 っ て い る。
</p>

<p>
だ が、 こ の 「当 た り 前」 を 人 類 が 手 に 入 れ る ま で に、 ど れ ほ ど の 革 命 と、 ど れ ほ ど の 流 血 と、 ど れ ほ ど の 時 間 が か か っ た か。<br>
フ ラ ン ス 人 権 宣 言 を 読 む と き、 私 た ち は そ の 重 さ を 改 め て 知 る。
</p>

<p>
そ し て、 こ の 宣 言 が 今 も <strong>「未 完 の 約 束」</strong> で あ り 続 け て い る こ と も、 心 に 留 め て お き た い。<br>
人 種 ・ ジ ェ ン ダ ー ・ 経 済 格 差 ─ 21 世 紀 の 世 界 に は、 ま だ 「人 権」 の 実 現 を 待 っ て い る 場 所 が、 数 多 く あ る。
</p>

<p>
1789 年 の パ リ で 響 い た 「自 由」 の 声 は、 今 も、 世 界 中 で 響 き 続 け て い る。
</p>
