<h3>「証 明 し ろ」 と 教 科 書 が 言 う 理 由</h3>

<p>
中 2 数 学 で「証 明 せ よ」 と い う 問 題 が 増 え る。 「三 角 形 ABC と 三 角 形 DEF が 合 同 で あ る こ と
を 証 明 せ よ」 ─ こ う い う 問 題 だ。
</p>

<p>
な ぜ「証 明」 が 必 要 な の か。 答 え は、 2300 年 前 の 古 代 ギ リ シ ャ に あ る。 ユ ー ク リ ッ ド
(Euclid) と い う 数 学 者 が、 全 て の 数 学 を「公 理 か ら の 証 明」 で 組 み 立 て た の だ。
</p>

<h3>ユ ー ク リ ッ ド ─ ア レ ク サ ン ド リ ア の 数 学 者</h3>

<p>
紀 元 前 4 ~ 3 世 紀、 ア レ ク サ ン ド ロ ス 大 王 の 死 後、 エ ジ プ ト の ア レ ク サ ン ド リ ア は
ギ リ シ ャ 文 化 の 中 心 だ っ た。 大 王 が 建 て た 大 図 書 館 (ム ー セ イ オ ン) に は、 全 世 界 の
知 識 が 集 め ら れ た。
</p>

<p>
そ こ で 教 え て い た の が ユ ー ク リ ッ ド で あ る。 彼 の 生 涯 は ほ と ん ど 不 明 だ が、 紀 元 前 300 年
頃 ま と め た 教 科 書『原 論』(ス ト イ ケ イ ア、 Elements) が、 そ の 後 2300 年 ・ 世 界 中 で
読 ま れ 続 け る こ と に な る。
</p>

<h3>『原 論』 13 巻 の 構 造 ─ 5 公 理 か ら 465 の 定 理 を 導 く</h3>

<p>
『原 論』 は、 極 め て 単 純 な「5 つ の 公 理」 か ら 始 ま る ─
</p>

<ol>
  <li>任 意 の 二 点 を 結 ぶ 直 線 が 引 け る</li>
  <li>任 意 の 直 線 は 無 限 に 延 ば せ る</li>
  <li>任 意 の 中 心 ・ 半 径 で 円 が 描 け る</li>
  <li>全 て の 直 角 は 等 し い</li>
  <li>(平 行 線 公 理) 一 つ の 直 線 と そ の 上 に な い 点 を 通 り、 そ の 直 線 と 平 行 な 直 線 は 一 本</li>
</ol>

<p>
こ れ だ け を 出 発 点 と し て、 465 個 も の「定 理」 を 論 理 的 に 導 い て い く。 三 角 形 の 性 質、
平 行 線、 円 の 性 質、 立 体 幾 何 ─ 君 が 中 学 ~ 高 校 で 学 ぶ 図 形 の 知 識 の ほ ぼ 全 て が、
こ の 13 巻 に 含 ま れ て い る。
</p>

<h3>命 題 4 ─ 三 角 形 の 合 同 (SAS)</h3>

<p>
中 2 で 学 ぶ「三 角 形 の 合 同 条 件」 の 一 つ、 「二 辺 と そ の 間 の 角 が 等 し い」 (SAS) は、
『原 論』 第 1 巻 命 題 4 で 証 明 さ れ て い る。 古 代 ギ リ シ ャ 人 が 既 に 2300 年 前 に 厳 密 に
証 明 し て い た の だ。
</p>

<p>
ユ ー ク リ ッ ド の 証 明 法 は「重 ね 合 わ せ 法」 ─ 二 つ の 三 角 形 を 重 ね て、 一 致 す る こ と を 示 す。
今 君 が 学 ぶ 教 科 書 の 証 明 も、 こ の 思 想 を 受 け 継 い で い る。
</p>

<h3>命 題 47 ─ ピ タ ゴ ラ ス の 定 理 の 証 明</h3>

<p>
『原 論』 第 1 巻 の 最 後、 命 題 47 で、 ユ ー ク リ ッ ド は ピ タ ゴ ラ ス の 定 理 (中 2 既 出 ・
pythagoras-no-teiri) を 厳 密 に 証 明 し て い る。 「直 角 三 角 形 の 斜 辺 の 上 の 正 方 形 は、
他 二 辺 の 上 の 正 方 形 の 和 に 等 し い」 ─ 図 形 を 切 り 貼 り す る エ レ ガ ン ト な 証 明 だ。
</p>

<p>
ピ タ ゴ ラ ス 本 人 (紀 元 前 6 世 紀) は 直 観 的 に 知 っ て い た が、 数 学 的 に 厳 密 に 証 明 し た の は
ユ ー ク リ ッ ド で あ る。
</p>

<h3>第 5 公 理 ─ 数 千 年 を 揺 さ ぶ っ た 一 行</h3>

<p>
公 理 の 中 で、 第 5 だ け が 何 だ か「自 明 で な い」。 多 く の 数 学 者 が、 こ れ を 他 の 公 理 か ら
証 明 し よ う と し て 失 敗 し て き た。
</p>

<p>
そ し て 19 世 紀、 ロ シ ア の ロ バ チ ェ フ ス キ ー、 ハ ン ガ リ ー の ボ ヤ イ、 ド イ ツ の リ ー マ ン ら が、
「第 5 公 理 を 否 定 し て も 矛 盾 な い 幾 何 学 が 構 築 で き る」 こ と を 発 見 ─ 非 ユ ー ク リ ッ ド 幾 何 学
で あ る。 こ れ が 後 に ア イ ン シ ュ タ イ ン の 相 対 性 理 論 の 数 学 的 基 礎 と な る。
</p>

<h3>結 び ─ 君 の「証 明」 が 受 け 継 ぐ 2300 年</h3>

<p>
教 科 書 の「証 明 せ よ」 と い う 問 題 を 解 く 時、 君 は ユ ー ク リ ッ ド の 2300 年 の 伝 統 を 受 け 継 い で
い る。 公 理 か ら 論 理 で 真 理 を 導 く ─ こ れ が「数 学 的 思 考」 の 核 心 で あ る。
</p>

<p>
証 明 が 苦 手 だ と 感 じ る 時 は、 ユ ー ク リ ッ ド を 思 い 出 そ う ─ 君 は 古 代 ギ リ シ ャ の 知 恵 と
対 話 し て い る の だ。
</p>
