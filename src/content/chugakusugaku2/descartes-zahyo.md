<h3>座 標 と は 何 か</h3>

<p>
平 面 (例 え ば 教 室 の 黒 板 や 紙) の 上 で、 ど こ か の 点 の 位 置 を 表 し た い と し よ う。
</p>

<p>
中 学 で 習 う 方 法 は こ う だ ─<br>
平 面 に 「<strong>x 軸 (横)</strong>」 と 「<strong>y 軸 (縦)</strong>」 を 引 き、 そ の 交 点 を 「原 点 (O)」 と す る。<br>
そ し て、 任 意 の 点 を 「(x, y)」 と い う 二 つ の 数 で 表 す。
</p>

<p>
例: 「原 点 か ら 右 へ 3、 上 へ 2」 の 点 = (3, 2)
</p>

<p>
こ の 方 法 を、 中 学 生 の 君 は 当 た り 前 の よ う に 使 う。<br>
だ が、 こ の「座 標」 は、 人 類 が <strong>1637 年</strong> に 一 人 の フ ラ ン ス 人 哲 学 者 か ら 贈 ら れ た 革 命 的 発 明 で あ る。
</p>

<h3>ル ネ・デ カ ル ト ─ 「我 思 う、 ゆ え に 我 あ り」</h3>

<p>
そ の フ ラ ン ス 人 哲 学 者 と は ─ <strong>ル ネ・デ カ ル ト</strong> (René Descartes, 1596–1650)。
</p>

<p>
彼 は 「<strong>我 思 う、 ゆ え に 我 あ り</strong>」 (Cogito, ergo sum) と い う 言 葉 で 有 名 な、 近 代 哲 学 の 父 で あ る。<br>
だ が、 彼 は 同 時 に 第 一 級 の 数 学 者 で も あ っ た。
</p>

<p>
1637 年、 デ カ ル ト は『<strong>方 法 序 説</strong>』 と い う 哲 学 書 を 出 版 し た。 そ の 付 録 と し て『幾 何 学』 が 付 い て い た。<br>
こ の 『幾 何 学』 の 中 で、 彼 は <strong>「座 標 系」</strong> を 体 系 化 し、 数 学 史 を 永 遠 に 変 え た。
</p>

<h3>「代 数 と 幾 何 の 統 一」 と い う 大 革 命</h3>

<p>
デ カ ル ト 以 前、 数 学 は 二 つ の 「別 々 の 世 界」 に 分 か れ て い た。
</p>

<table>
<thead>
<tr><th>世 界</th><th>扱 う も の</th><th>例</th></tr>
</thead>
<tbody>
<tr><td><strong>代 数</strong> (al-jabr)</td><td>式・数 を 扱 う</td><td>x + 5 = 10、 x² = 4</td></tr>
<tr><td><strong>幾 何</strong> (geometry)</td><td>図 形 を 扱 う</td><td>直 線、 円、 三 角 形</td></tr>
</tbody>
</table>

<p>
古 代 ギ リ シ ア か ら 中 世 ま で、 こ の 二 つ は <strong>「別 の 学 問」</strong> だ っ た。<br>
ピ タ ゴ ラ ス の 定 理 (本 サ イ ト) は 幾 何 学。 ヴ ィ エ ー タ の 文 字 式 (本 サ イ ト chugakusugaku1) は 代 数。 別 の 道 具 を 使 い、 別 の 方 法 で 問 題 を 解 い て い た。
</p>

<p>
デ カ ル ト は、 座 標 系 を 使 う こ と で、 こ の 二 つ を <strong>同 じ も の</strong> と し て 扱 え る よ う に し た:
</p>

<ul>
<li>直 線 ─ y = ax + b と い う <strong>一 次 式</strong> で 表 せ る</li>
<li>円 ─ x² + y² = r² と い う <strong>二 次 式</strong> で 表 せ る</li>
<li>放 物 線 ─ y = x² と い う <strong>二 次 式</strong> で 表 せ る</li>
</ul>

<p>
「<strong>図 形 = 式</strong>」「<strong>式 = 図 形</strong>」 ─ こ の 一 元 化 が、 数 学 を 一 気 に 強 力 に し た。
</p>

<h3>一 次 関 数 y = ax + b</h3>

<p>
中 2 数 学 で 学 ぶ 「<strong>一 次 関 数</strong>」 ─ y = ax + b ─ は、 デ カ ル ト の 座 標 系 の 上 で 直 接 「直 線」 と し て 可 視 化 で き る。
</p>

<p>
具 体 例 で 見 て み よ う:
</p>

<ul>
<li><strong>y = x</strong> ─ 原 点 を 通 り、 右 上 が り の 直 線 (傾 き 1)</li>
<li><strong>y = 2x</strong> ─ 原 点 を 通 り、 急 な 直 線 (傾 き 2)</li>
<li><strong>y = 2x + 3</strong> ─ y 切 片 が 3、 傾 き が 2 の 直 線</li>
<li><strong>y = -x + 5</strong> ─ y 切 片 が 5、 右 下 が り の 直 線</li>
</ul>

<p>
こ う し て、 数 式 を 見 れ ば 直 線 の 様 子 が 想 像 で き、 直 線 を 見 れ ば 数 式 が わ か る ─ こ れ が デ カ ル ト の 革 命 で あ る。
</p>

<h3>デ カ ル ト 以 降 の 数 学</h3>

<p>
デ カ ル ト の 座 標 系 (1637) が 出 て か ら、 ヨ ー ロ ッ パ の 数 学 は 急 速 に 発 展 し た:
</p>

<ul>
<li>1666 ─ ニ ュ ー ト ン、 微 分 積 分 学 を 発 明</li>
<li>1684 ─ ラ イ プ ニ ッ ツ、 微 分 積 分 学 を 独 立 発 表</li>
<li>1748 ─ オ イ ラ ー『無 限 解 析』</li>
<li>19 世 紀 ─ 関 数 論・ベ ク ト ル 解 析・線 形 代 数 の 発 展</li>
</ul>

<p>
こ れ ら す べ て の 礎 が、 デ カ ル ト の 座 標 系 で あ る。
</p>

<h3>身 の 回 り の 座 標</h3>

<p>
2026 年 の 君 の 周 り に は、 デ カ ル ト 座 標 が 至 る と こ ろ に 隠 れ て い る:
</p>

<ul>
<li><strong>GPS</strong> ─ 経 度 と 緯 度 で 位 置 を 表 す ─ こ れ は 球 面 座 標 (デ カ ル ト の 拡 張)</li>
<li><strong>地 図 ア プ リ</strong> ─ 画 面 上 の 場 所 を (x, y) で 管 理</li>
<li><strong>ゲ ー ム の 画 面</strong> ─ キ ャ ラ ク タ ー の 位 置 を (x, y) で 表 現</li>
<li><strong>エ ク セ ル の セ ル</strong> ─ A1、 B3 と い う「列・行」 は 座 標 と 同 じ</li>
</ul>

<p>
中 2 で 君 が 学 ぶ 一 次 関 数 ─ そ の 背 後 に は、 1637 年 の フ ラ ン ス の 哲 学 者 が 開 い た、 数 学 史 上 最 大 級 の 革 命 が あ る。
</p>

<p>
「<strong>図 形 と 数 式 が 同 じ も の</strong>」 ─ こ の 不 思 議 で 美 し い 真 理 を、 こ れ か ら 何 度 も 体 験 し て い く こ と に な る だ ろ う。
</p>
