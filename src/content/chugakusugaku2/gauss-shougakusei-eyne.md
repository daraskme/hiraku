## 10 歳 の 天 才 と 1+2+…+100

### 1. ブ ラ ウ ン シ ュ ヴ ァ イ ク の 小 学 校 ─ 1787 年

時 は 1787 年、 ド イ ツ 北 部 の 街・ブ ラ ウ ン シ ュ ヴ ァ イ ク。 9 月 の あ る 日、 地 元 の 小 学 校 で 算 数 の 授 業 が 始 ま っ て い た。

教 師 の ビ ュ ッ ト ナ ー 先 生 は、 い つ も の よ う に 生 徒 た ち に 厳 し い 課 題 を 出 し た。 い や、 実 は 厳 し い と い う よ り、 生 徒 た ち を し ば ら く 静 か に さ せ る た め の 「時 間 つ ぶ し」 課 題 で あ っ た。

「い い か、 今 か ら こ の 問 題 を 解 け。 解 き 終 わ っ た ら 静 か に 待 て。」

そ し て、 黒 板 に こ う 書 い た。

   1 + 2 + 3 + … + 99 + 100 = ?

生 徒 た ち は、 紙 と 鉛 筆 を 取 り 出 し、 黙 々 と 計 算 を 始 め た。
1 + 2 = 3、 3 + 3 = 6、 6 + 4 = 10、 …
こ の ペ ー ス で い く と、 全 部 で 99 回 の 足 し 算 を し な け れ ば な ら な い。 30 分 は か か る だ ろ う。

***

### 2. 数 秒 後 ─ ガ ウ ス の 答 え

し か し、 教 室 の 一 番 後 ろ の 席 に い た 10 歳 の カ ー ル・ガ ウ ス は、 数 秒 だ け 黒 板 を 見 つ め、 そ し て 自 分 の 石 板 に 何 か を 書 き、 そ れ を 教 卓 の 上 に そ っ と 置 い た。

「先 生、 で き ま し た」

ビ ュ ッ ト ナ ー 先 生 は 信 じ ら れ な か っ た。 が、 ガ ウ ス の 石 板 を 見 る と、 そ こ に は こ う 書 い て あ っ た。

   5050

正 解 で あ っ た。

***

### 3. ガ ウ ス は ど う や っ て 解 い た の か

ガ ウ ス は、 普 通 の 子 ど も の よ う に 「1+2+3+…」 と 順 に 足 し て い っ た わ け で は な か っ た。 彼 は、 次 の よ う に 考 え た の で あ る。

数 を 並 べ て み る。

  1 + 2 + 3 + 4 + … + 97 + 98 + 99 + 100

両 端 か ら ペ ア を つ く る。 1 と 100 を ペ ア、 2 と 99 を ペ ア、 3 と 98 を ペ ア、 …、 50 と 51 を ペ ア。

  1 + 100 = 101
  2 + 99  = 101
  3 + 98  = 101
  ...
  50 + 51 = 101

ど の ペ ア の 和 も、 全 部 101。 そ し て、 ペ ア は 全 部 で 50 個 あ る。 し た が っ て、

  101 × 50 = 5050

答 え は 5050 で あ る。 30 分 の 足 し 算 が、 こ の「ペ ア に 分 け る」 と い う 一 つ の 発 想 で、 一 個 の 掛 け 算 に な っ た の だ。

***

### 4. 等 差 数 列 の 和 の 公 式

ガ ウ ス が 直 観 で 見 抜 い た こ の 発 想 は、 後 に「等 差 数 列 の 和 の 公 式」 と し て 数 学 に 取 り 込 ま れ た。

初 項 a、 末 項 l、 項 数 n の 等 差 数 列 (例 え ば 1, 2, 3, …, 100 の よ う な、 隣 り 同 士 の 差 が 一 定 の 数 列) の 和 を S_n と す る と、

   S_n = n × (a + l) ÷ 2

つ ま り「ペ ア の 和 (a + l)」 ×「ペ ア の 個 数 (n ÷ 2)」 で あ る。

1+2+…+100 に 当 て は め る と、

   S_100 = 100 × (1 + 100) ÷ 2 = 100 × 101 ÷ 2 = 5050

確 か に 5050 で あ る。

***

### 5. 「発 想 の 飛 躍」 と し て の 数 学

こ の 逸 話 が、 200 年 以 上 経 っ た 今 も 語 り 継 が れ る の は、 そ れ が 数 学 の 本 質 を 示 し て い る か ら で あ る。

数 学 と は、 計 算 力 だ け の 問 題 で は な い。 「ど う 見 れ ば、 問 題 が 簡 単 に な る か」 と い う、 発 想 の 飛 躍 こ そ が 数 学 の 真 髄 で あ る。

10 歳 の ガ ウ ス は、 99 回 の 足 し 算 を 数 学 的 な 視 線 で 眺 め 直 し、 そ こ に「対 称 性」 を 見 出 し た。 こ の「対 称 性 に 気 づ く」 と い う 発 想 は、 後 に ガ ウ ス が 19 世 紀 数 学 を 一 人 で 半 世 紀 リ ー ド し、「数 学 王」 と 呼 ば れ る こ と に な る 才 能 の 原 点 で あ っ た。

***

### 6. 練 習 問 題

(1) 1+2+3+…+50 を、 ガ ウ ス の 方 法 で 計 算 し な さ い。
   ヒ ン ト: ペ ア の 和 は (1+50)、 ペ ア の 個 数 は 50÷2 = 25 個。
   答 え: 51 × 25 = 1275

(2) 5+10+15+…+100 を 計 算 し な さ い。 (5 か ら 100 ま で、 5 ご と に 飛 ぶ 等 差 数 列)
   初 項 5、 末 項 100、 項 数 20。
   S_20 = 20 × (5+100) ÷ 2 = 20 × 105 ÷ 2 = 1050

(3) 1 か ら 1000 ま で の 自 然 数 の 総 和 を 求 め な さ い。
   S_1000 = 1000 × (1+1000) ÷ 2 = 1000 × 1001 ÷ 2 = 500,500

10 歳 の ガ ウ ス は、 こ れ ら も、 数 秒 で 解 い た だ ろ う。
