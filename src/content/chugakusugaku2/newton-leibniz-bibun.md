## ニ ュ ー ト ン と ラ イ プ ニ ッ ツ ─ 微 分 積 分 の 同 時 発 見

### 1. 二 つ の 問 い

数 学 に は、 古 代 ギ リ シ ア の 時 代 か ら 解 け な か っ た 二 つ の 大 き な 問 い が あ っ た。

一 つ め ─ 「曲 線 の、 あ る 点 で の 傾 き (変 化 の 割 合) を、 ど う 求 め る か?」 直 線 の 傾 き は 簡 単 だ。 し か し、 曲 が っ た 線 の「そ の 瞬 間 の 傾 き」 を 求 め る の は 難 し い。

二 つ め ─ 「曲 線 で 囲 ま れ た 図 形 の 面 積 を、 ど う 求 め る か?」 四 角 形 や 三 角 形 の 面 積 は 簡 単 だ。 し か し、 曲 線 で 囲 ま れ た 複 雑 な 図 形 の 面 積 は 難 し い。

こ の 二 つ の 問 い に、 17 世 紀 後 半、 二 人 の 天 才 が、 ほ ぼ 同 時 に、 別 々 の 国 で、 答 え を 出 し た。

***

### 2. ニ ュ ー ト ン ─ 驚 異 の 年

一 人 め は、 イ ギ リ ス の ア イ ザ ッ ク・ニ ュ ー ト ン (1643-1727)。

1665 年 か ら 1666 年 に か け て、 ロ ン ド ン で ペ ス ト が 大 流 行 し、 ケ ン ブ リ ッ ジ 大 学 が 休 校 に な っ た。 ニ ュ ー ト ン は 故 郷 ウ ー ル ズ ソ ー プ に 戻 り、 一 人 で 思 索 に ふ け っ た。

こ の わ ず か 1 年 半 の 間 に、 ニ ュ ー ト ン は 万 有 引 力、 光 の 性 質、 そ し て 微 積 分 (彼 は「流 率 法 (り ゅ う り つ ほ う)」 と 呼 ん だ) を 着 想 し た。 後 世「驚 異 の 年 (annus mirabilis)」 と 呼 ば れ る、 人 類 史 上 最 も 創 造 的 な 期 間 の 一 つ で あ る。

し か し、 ニ ュ ー ト ン は 慎 重 で 内 向 的 な 性 格 で、 自 分 の 発 見 を 公 表 す る こ と を 長 く た め ら っ た。

***

### 3. ラ イ プ ニ ッ ツ ─ 先 に 発 表

二 人 め は、 ド イ ツ の ゴ ッ ト フ リ ー ト・ラ イ プ ニ ッ ツ (1646-1716)。 数 学・哲 学・法 学・外 交 ─ あ ら ゆ る 分 野 で 活 躍 し た「万 能 の 知 識 人」 で あ っ た。

ラ イ プ ニ ッ ツ は 1670 年 代 に 独 自 に 微 積 分 を 研 究 し、 1684 年、 微 分 法 の 論 文 を 学 術 誌『ア ク タ・エ ル デ ィ ト ル ム』 に 発 表 し た。 こ れ が、 微 積 分 の 最 初 の「公 式 発 表」 で あ っ た。

ラ イ プ ニ ッ ツ の 偉 大 な 貢 献 の 一 つ は、 「記 号」 で あ る。 現 在 私 た ち が 使 う 微 分 の 記 号「dy/dx」 や、 積 分 の 記 号「∫ (イ ン テ グ ラ ル)」 は、 す べ て ラ イ プ ニ ッ ツ が 考 案 し た も の だ。 こ の 分 か り や す い 記 号 の お か げ で、 ヨ ー ロ ッ パ 大 陸 の 数 学 は 大 き く 発 展 し た。

***

### 4. 微 分 と 積 分 は「逆」 で あ る

ニ ュ ー ト ン と ラ イ プ ニ ッ ツ が、 そ れ ぞ れ 独 立 に 見 抜 い た 最 も 重 要 な 発 見 ─ そ れ は、 「微 分 と 積 分 は、 互 い に 逆 の 操 作 で あ る」 と い う こ と だ っ た。

例 え ば、 車 の 運 動 を 考 え よ う。

- 車 の「距 離」 を 微 分 す る と、 「速 さ」 が 求 ま る。
- 車 の「速 さ」 を 積 分 す る と、 「距 離」 が 求 ま る。

距 離 → (微 分) → 速 さ → (積 分) → 距 離。 行 っ て 帰 っ て く る。 こ の「逆 の 関 係」 を「微 積 分 学 の 基 本 定 理」 と 呼 ぶ。

こ の 発 見 に よ っ て、 「変 化 を 求 め る 微 分」 と「積 み 重 ね を 求 め る 積 分」 が、 一 つ の 強 力 な 数 学 ─ 微 積 分 ─ に ま と ま っ た の で あ る。

***

### 5. 優 先 権 論 争

し か し、 二 人 の 同 時 発 見 は、 数 学 史 上 最 大 の 論 争 を 生 ん だ。

「微 積 分 を 先 に 発 見 し た の は、 ニ ュ ー ト ン か、 ラ イ プ ニ ッ ツ か?」

事 実 を 整 理 す る と:
- 着 想 は ニ ュ ー ト ン が 早 い (1665-66 年)
- 公 表 は ラ イ プ ニ ッ ツ が 早 い (1684 年)

ニ ュ ー ト ン 派 (イ ギ リ ス) は「ニ ュ ー ト ン が 先 だ。 ラ イ プ ニ ッ ツ は 盗 用 し た」 と 主 張。 ラ イ プ ニ ッ ツ 派 (大 陸) は「ラ イ プ ニ ッ ツ が 独 立 に 発 見 し、 先 に 発 表 し た」 と 反 論。 両 国 の 学 界 を 巻 き 込 む 泥 沼 の 論 争 と な っ た。

現 在 で は、 「両 者 が 独 立 に 発 見 し た」 と 考 え ら れ て い る。 し か し、 こ の 論 争 の せ い で、 18 世 紀 の イ ギ リ ス 数 学 は 大 陸 と 没 交 渉 に な り、 ラ イ プ ニ ッ ツ の 優 れ た 記 号 を 使 わ ず、 停 滞 し て し ま っ た と も 言 わ れ る。 ─ 論 争 の 代 償 は 大 き か っ た。

***

### 6. 「同 時 発 見」 と い う 不 思 議

微 積 分 の よ う に、 異 な る 人 物 が ほ ぼ 同 時 に・独 立 に 同 じ 発 見 を す る こ と を、 科 学 史 で は「同 時 発 見 (multiple discovery)」 と 呼 ぶ。

- 微 積 分 ─ ニ ュ ー ト ン と ラ イ プ ニ ッ ツ
- 進 化 論 ─ ダ ー ウ ィ ン と ウ ォ レ ス
- 電 話 ─ ベ ル と グ レ イ (特 許 出 願 が 同 じ 日)

「時 代 が 熟 す と、 同 じ 発 見 が 複 数 の 場 所 で 必 然 的 に 生 ま れ る」。 こ れ は、 科 学 が「一 人 の 天 才 の 気 ま ぐ れ」 で は な く、 「人 類 全 体 の 知 の 積 み 重 ね」 の 上 に あ る こ と を 示 し て い る。

***

### 7. 現 代 を 動 か す 微 積 分

微 積 分 は、 現 代 の あ ら ゆ る 科 学 技 術 の 言 葉 と な っ て い る。

ロ ケ ッ ト の 軌 道 計 算、 天 気 予 報、 経 済 の 予 測、 人 工 知 能 の 学 習 (機 械 学 習 の「勾 配 降 下 法」 は 微 分 そ の も の)、 ス マ ホ の 画 像 処 理 ─ す べ て 微 積 分 で 動 い て い る。

中 学 2 年 で「一 次 関 数 の 変 化 の 割 合 (傾 き)」 を 学 ぶ 君 は、 ニ ュ ー ト ン と ラ イ プ ニ ッ ツ が 切 り 開 い た 微 積 分 へ の、 最 初 の 一 歩 を 踏 み 出 し て い る。 高 校 で「微 分・積 分」 を 学 ぶ と き、 こ の 二 人 の 天 才 の 同 時 発 見 を 思 い 出 し て ほ し い。
