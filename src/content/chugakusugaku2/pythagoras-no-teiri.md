<h3>三 平 方 の 定 理 と は</h3>

<p>
直 角 三 角 形 を 描 い て み よ う。<br>
直 角 を 挟 む 二 辺 の 長 さ を a、 b、 直 角 の 反 対 側 の 辺 (斜 辺) を c と す る と:
</p>

<blockquote class="song">
<p>a² + b² = c²</p>
</blockquote>

<p>
こ れ が 「<strong>三 平 方 の 定 理</strong>」、 別 名 「<strong>ピ タ ゴ ラ ス の 定 理</strong>」 だ。
</p>

<p>
例 え ば 「3-4-5」 の 直 角 三 角 形:<br>
3² + 4² = 9 + 16 = 25 = 5²。 ぴ っ た り 成 り 立 つ。
</p>

<p>
「5-12-13」 の 直 角 三 角 形:<br>
5² + 12² = 25 + 144 = 169 = 13²。 こ れ も 成 り 立 つ。
</p>

<h3>古 代 三 文 明 の 並 行 発 見</h3>

<p>
こ の 定 理 は、 ピ タ ゴ ラ ス が 紀 元 前 6 世 紀 に 「発 見」 し た と さ れ る。<br>
だ が 実 は、 ピ タ ゴ ラ ス よ り も <strong>1300 年 早 く</strong>、 同 じ 定 理 が 知 ら れ て い た。
</p>

<tr><th>文 明</th><th>時 代</th><th>記 録</th></tr>

<tr><td><strong>古 代 中 国</strong></td><td>紀 元 前 1 世 紀 以 前</td><td>『周 髀 算 経』 (し ゅ ひ さ ん け い)</td></tr>
<tr><td><strong>古 代 ギ リ シ ア</strong></td><td>紀 元 前 6 世 紀 (ピ タ ゴ ラ ス)</td><td>体 系 的 な 証 明</td></tr>
</tbody>

<p>
バ ビ ロ ニ ア・中 国・ギ リ シ ア の 三 文 明 が、 互 い に 接 触 し て い な か っ た 時 代 に、 全 く 同 じ 数 学 的 真 理 に 到 達 し て い た。<br>
こ れ は 「<strong>数 学 は 文 化 を 超 え た 普 遍 的 言 語</strong>」 で あ る こ と の 美 し い 証 拠 で あ る。
</p>

<h3>ピ タ ゴ ラ ス の 貢 献 ─ 「証 明」 の 発 明</h3>

<p>
バ ビ ロ ニ ア や 中 国 で は、 こ の 定 理 を 「<strong>事 実 と し て 知 っ て い た</strong>」 が、 「<strong>な ぜ 成 り 立 つ か</strong>」 の 証 明 は し て い な か っ た。
</p>

<p>
ピ タ ゴ ラ ス と そ の 学 派 が 西 洋 数 学 に 残 し た 偉 業 は、 「<strong>証 明</strong>」 と い う 概 念 を 確 立 し た こ と で あ る。
</p>

<p>
事 実 を 観 察 す る だ け で な く、 論 理 に よ っ て 「な ぜ そ う な る か」 を 示 す ─ こ の 姿 勢 が、 西 洋 数 学・西 洋 科 学 の 根 本 と な っ た。
</p>

<h3>身 の 回 り の ピ タ ゴ ラ ス</h3>

<p>
こ の 定 理 は、 中 学 ・ 高 校 数 学 の 基 礎 中 の 基 礎。 だ が 実 は、 日 常 生 活 で も 使 わ れ て い る:
</p>

<ul>
<li><strong>大 工 さ ん の 直 角 確 認</strong> ─ 3-4-5 の 比 で 紐 を 張 る と 直 角 が 作 れ る</li>
<li><strong>テ レ ビ の サ イ ズ</strong> ─ 「50 イ ン チ」 は 画 面 の 対 角 線 = ピ タ ゴ ラ ス の 定 理 で 計 算</li>
<li><strong>GPS の 距 離 計 算</strong> ─ 二 点 間 の 直 線 距 離 = ピ タ ゴ ラ ス</li>
<li><strong>建 築 の 屋 根 の 長 さ</strong> ─ 斜 面 の 長 さ = ピ タ ゴ ラ ス</li>
</ul>

<p>
2500 年 前 の ギ リ シ ア で 確 立 さ れ た 定 理 が、 現 代 の 君 の 生 活 に 直 結 し て い る。
</p>

<h3>数 学 の 美 し い 真 理</h3>

<p>
a² + b² = c² ─ た っ た 一 行 の 式 が、 三 つ の 古 代 文 明 を 結 び、 25 世 紀 を 超 え て 今 も 真 実 で あ り 続 け る。
</p>

<p>
こ れ こ そ が、 数 学 が 持 つ 「<strong>時 代 を 超 え た 真 理</strong>」 の 美 し さ で あ る。<br>
ピ タ ゴ ラ ス が 君 に 残 し た 贈 り 物 だ。
</p>
