glossary:
  - term: 第1文型（SV）
    meaning: 主語 + 動詞のみで完結する文。「動く・存在する・現れる・消える」など、自動詞のみで意味が決まる動詞が中心。例：The sun rises. / He arrived. / She smiled.
  - term: 第2文型（SVC）
    meaning: 主語 + 動詞 + 補語。動詞は「である・になる・見える・思える」など、主語の状態や変化を述べる。be 動詞のほか、become / get / grow / turn / look / seem / appear / feel / sound / taste / smell / remain / keep / stay。
  - term: 第3文型（SVO）
    meaning: 主語 + 動詞 + 目的語。最も基本的な「〜を…する」の形。動詞は他動詞。英語の文の大半がこの形をとる。
  - term: 第4文型（SVOO）
    meaning: 主語 + 動詞 + 間接目的語（人）+ 直接目的語（もの）。「人にものを与える/伝える」型。give / send / show / tell / teach / lend / pay / offer / write / bring / buy / make / cook / find / get など。「give 人 もの」≒「give もの to 人」。
  - term: 第5文型（SVOC）
    meaning: 主語 + 動詞 + 目的語 + 補語。「O を C にする/と思う/と呼ぶ/させる」型。make / call / name / find / think / consider / keep / leave / let / have / get / help などの動詞。「I found the book interesting.」のように、O と C のあいだに「O is C」の関係がある。
  - term: 自動詞（vi.）
    meaning: 目的語を取らない動詞。go / come / arrive / live / die / sleep / smile / cry など。第1文型・第2文型を作る。
  - term: 他動詞（vt.）
    meaning: 目的語を取る動詞。eat / read / make / find / give など。第3文型・第4文型・第5文型を作る。
  - term: 補語（Complement, C）
    meaning: 動詞だけでは意味が完結しないとき、主語や目的語を「説明する」要素。名詞・形容詞・分詞などが補語になる。
  - term: 目的語（Object, O）
    meaning: 動作の対象。「〜を」「〜に」にあたる。名詞・代名詞・名詞句・名詞節などがなる。
  - term: 句動詞（phrasal verb）
    meaning: 動詞 + 副詞（または前置詞）の組み合わせで、ひとつの意味を持つもの。例：look up（調べる）、put off（延期する）、give in（屈する）、carry out（実行する）。
  - term: イディオム（idiom）
    meaning: 単語の字義通りでは意味が分からない慣用表現。例：raining cats and dogs（土砂降り）、let the cat out of the bag（秘密を漏らす）、a piece of cake（とても簡単）。
  - term: 第3文型 vs 第4文型 — 二重目的語
    meaning: 「give him a book」（4文型）と「give a book to him」（3文型）は意味は同じだが構造が違う。to を取るのは「移動・到達」を含意する動詞（give, send, show, tell, write）。for を取るのは「人のために」を含意する動詞（buy, make, cook, find, get）。例：buy her flowers ≒ buy flowers FOR her。
  - term: 自動詞と思いがちな他動詞 (受験頻出)
    meaning: discuss / attend / approach / enter / mention / marry / resemble / oppose / reach / answer。これらは前置詞を伴わずに直接目的語をとる。×discuss about → ○discuss / ×marry with → ○marry。受験で頻出のミスポイント。
  - term: 他動詞と思いがちな自動詞 (受験頻出)
    meaning: arrive at / look at / wait for / agree with / apologize to / listen to / look for / hope for / object to / belong to / refer to / consist of。これらは前置詞が必要。×Wait me → ○Wait for me。

<p class="lead">英語のすべての文は、たった<strong>5つの「文型」</strong>に整理することができます。これは1920年代に日本人の英語学者 <strong>細江逸記（ほそえ いっき、1884–1947）</strong>が確立した、日本の英語教育の伝統的な分析法です。各文型を「動詞の使い方の型」として理解すると、英文を読む速度が劇的に上がります。本教材では、5文型を順に解説しながら、各文型に対応する<strong>大学受験頻出の動詞・イディオム・句動詞</strong>を整理して提示します。1コマで全体像をつかみ、そこから各単元を深めていくための「英語の地図」となる教材です。</p>

<h3>第1文型 — S + V「主語が動く・存在する」</h3>

<p>主語と動詞だけで意味が完結する型。<strong>自動詞</strong>（目的語を取らない動詞）が中心で、後ろに前置詞句や副詞句が付いて場所・時を補うことが多いものの、それは <strong>修飾語（M, modifier）</strong>であって文型の本体には数えません。</p>

<blockquote class="song">
<p><strong>The sun</strong> [S] <strong>rises</strong> [V] in the east [M].<br>
（太陽は東から昇る。）</p>

<p><strong>She</strong> [S] <strong>lives</strong> [V] in Tokyo [M].<br>
（彼女は東京に住んでいる。）</p>

<p><strong>He</strong> [S] <strong>arrived</strong> [V] at the station [M] at noon [M].<br>
（彼は正午に駅に着いた。）</p>
</blockquote>

<h4>第1文型に伴う頻出イディオム（前置詞句）</h4>

<ul>
  <li><strong>arrive at</strong> + 比較的狭い場所 / <strong>arrive in</strong> + 広い場所　「〜に到着する」</li>
  <li><strong>get to</strong>「〜に到着する」（より口語的）</li>
  <li><strong>graduate from</strong>「〜を卒業する」（× graduate）</li>
  <li><strong>belong to</strong>「〜に属する」</li>
  <li><strong>consist of</strong>「〜から成る」</li>
  <li><strong>result in</strong> 〜「〜という結果になる」 / <strong>result from</strong> 〜「〜から生じる」</li>
  <li><strong>depend on</strong>「〜に依存する / 〜次第である」</li>
  <li><strong>look at</strong>「〜を見る」 / <strong>listen to</strong>「〜を聴く」 / <strong>wait for</strong>「〜を待つ」</li>
  <li><strong>apologize to</strong> 人 <strong>for</strong> 事「事のことを人に謝る」</li>
  <li><strong>agree with</strong> 人 / <strong>agree to</strong> 提案・計画「同意する」</li>
</ul>

<div class="callout">
<strong>受験頻出ミス</strong>　日本語で「〜を待つ」「〜に着く」「〜を見る」と訳すために、つい目的語を直接付けてしまうことが多い動詞群です。<br>
× <em>I waited him.</em> → ○ <em>I waited <strong>for</strong> him.</em><br>
× <em>I arrived Tokyo.</em> → ○ <em>I arrived <strong>at/in</strong> Tokyo.</em><br>
× <em>I look the picture.</em> → ○ <em>I look <strong>at</strong> the picture.</em>
</div>

<h3>第2文型 — S + V + C「S は C である / になる」</h3>

<p>主語と補語のあいだに<strong>イコール関係</strong>がある型。be 動詞が代表ですが、「〜になる」「〜のままでいる」「〜に見える/思える」など、状態や変化を表す動詞が幅広く第2文型を作ります。</p>

<blockquote class="song">
<p><strong>He</strong> [S] <strong>is</strong> [V] <strong>a doctor</strong> [C].<br>
（彼は医者である。）<em>← 彼 = 医者</em></p>

<h3>第4文型 — S + V + O₁ + O₂「人にものを与える」</h3>

</blockquote>

<h4>SVOO ⇔ SVO + to / for の書き換え</h4>

<blockquote class="song">
<p><strong>SVOO:</strong> She gave <strong>him</strong> <strong>a present</strong>.<br>
<strong>SVO + to:</strong> She gave <strong>a present</strong> to <strong>him</strong>.</p>

<p><strong>SVOO:</strong> He bought <strong>her</strong> <strong>flowers</strong>.<br>
<strong>SVO + for:</strong> He bought <strong>flowers</strong> for <strong>her</strong>.</p>
</blockquote>

<p>受験では「与える系の動詞 → to」「人のために系の動詞 → for」と覚えるのが原則です。例外は <strong>ask</strong>（of を取る）。</p>

<h3>第5文型 — S + V + O + C「O を C にする / と思う」</h3>

<p>もっとも応用範囲の広い、英語の「<strong>表現力の核心</strong>」となる文型。O と C のあいだに<strong>「O is C」の関係</strong>があるのがポイントです。</p>

<p><strong>I</strong> [S] <strong>found</strong> [V] <strong>the book</strong> [O] <strong>interesting</strong> [C].<br>
（私はその本を面白いと思った。）<em>← the book is interesting の関係</em></p>

<p><strong>She</strong> [S] <strong>kept</strong> [V] <strong>the room</strong> [O] <strong>clean</strong> [C].<br>
（彼女は部屋をきれいに保った。）</p>
</blockquote>

<h3>使い方の心得 — この教材をどう使うか</h3>

<h4>① レンズとして使う</h4>

<p>文型は「正解」ではなく、英文を読む<strong>レンズ</strong>です。すべての英文を「これは何文型？」と分析する必要はありません。ただ、複雑な英文に出会ったとき、「S は何か」「V は何か」「O はあるか」「C はあるか」と骨格を取る習慣をつけると、長文読解が格段に楽になります。</p>

<h4>② 動詞を「形」とともに覚える</h4>

<p>新しい動詞を学ぶときは、<strong>必ず一緒に取る前置詞や文型を含めて</strong>覚えてください。「remind」ではなく「<strong>remind O of N / remind O to V</strong>」、「prevent」ではなく「<strong>prevent O from V-ing</strong>」、というように。動詞単体ではなく、<strong>動詞+型のチャンク（かたまり）</strong>で覚えるのが、英語上達の最短ルートです。</p>

<h4>③ 文型ごとの動詞リストを定着させる</h4>

<p>本教材で挙げた約 80 個の動詞は、大学受験の英語長文で頻出のものです。それぞれの「文型」と「典型例」をセットで暗記してください。受験英語の語彙力の<strong>核</strong>になります。</p>

<h4>④ 「英作文」と「読解」の両方に効く</h4>

<p>文型は、英作文を書くときの<strong>設計図</strong>でもあります。日本語で言いたい内容を、まず「これは何文型で表現できるか？」と考えてから、動詞を選んでいく。これが英作文の正攻法です。</p>
