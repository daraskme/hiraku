problems:
  - prompt: 縄文時代のくらしで使われた、縄目のもようがある土器を何といいますか。
    answer: 縄文土器
    explanation: 縄文土器は、表面に縄目のようなもようがついた土器です。
    difficulty: basic
    type: history
    hints:
      - 時代名と同じ言葉が入ります。
  - prompt: 米づくりが広がった時代を何時代といいますか。
    answer: 弥生時代
    explanation: 弥生時代には水田での米づくりが広まりました。
    difficulty: basic
    type: history
    hints:
      - 縄文時代の次です。
  - prompt: 大きな古墳がつくられた時代を何時代といいますか。
    answer: 古墳時代
    explanation: 権力を持つ人の墓として、大きな古墳がつくられました。
    difficulty: basic
    type: history
    hints:
      - 名前に「古墳」が入ります。
  - prompt: 聖徳太子が定めたとされる、役人の心構えを示したものを何といいますか。
    answer: 十七条の憲法
    explanation: 十七条の憲法は、役人のあるべき姿を示したものとされます。
    difficulty: standard
    type: history
    hints:
      - 「和をもって貴しとなす」で知られます。
  - prompt: 奈良に都が置かれていた時代を何時代といいますか。
    answer: 奈良時代
    explanation: 平城京を中心に政治が行われた時代です。
    difficulty: basic
    type: history
    hints:
      - 都の場所が時代名になっています。
  - prompt: 源頼朝が開いた武士の政権を何といいますか。
    answer: 鎌倉幕府
    explanation: 源頼朝は鎌倉を中心に幕府を開き、武士による政治が始まりました。
    difficulty: standard
    type: history
    hints:
      - 地名と「幕府」を組み合わせます。
  - prompt: 徳川家康が開いた幕府を何といいますか。
    answer: 江戸幕府
    explanation: 徳川家康は江戸に幕府を開き、江戸時代が始まりました。
    difficulty: standard
    type: history
    hints:
      - 今の東京にあたる場所です。
  - prompt: 縄文、弥生、古墳、奈良、鎌倉、江戸を古い順に並べましょう。
    answer: 縄文、弥生、古墳、奈良、鎌倉、江戸
    explanation: くらしの変化、都の場所、武士の政治の始まりを時代順で整理します。
    difficulty: challenge
    type: timeline
    hints:
      - 米づくりは縄文のあと、武士の幕府は奈良のあとです。
