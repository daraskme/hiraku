problems:
  - prompt: 米を育てる田を何といいますか。
    answer: 水田
    explanation: 水をためて稲を育てる田を水田といいます。
    difficulty: basic
    type: term
    hints:
      - 水を使う田です。
  - prompt: 米のもとになる植物を何といいますか。
    answer: 稲
    explanation: 稲が成長し、実ったもみから米がとれます。
    difficulty: basic
    type: term
    hints:
      - 田に植えられている植物です。
  - prompt: 苗を田に植える作業を何といいますか。
    answer: 田植え
    explanation: 育てた苗を水田に植える作業を田植えといいます。
    difficulty: basic
    type: term
    hints:
      - 春から初夏に行われることが多い作業です。
  - prompt: 米づくりに水の管理が大切な理由を書きましょう。
    answer: 稲の成長に必要で、水の量によって育ち方が変わるから
    explanation: 水が多すぎても少なすぎても稲の成長に影響が出るため、農家は水の量を調整します。
    difficulty: standard
    type: explanation
    hints:
      - 稲は水田で育ちます。
  - prompt: 米づくりで機械が使われる理由を一つ書きましょう。
    answer: 作業を早く正確に行い、人手を減らすため
    explanation: 田植え機やコンバインなどを使うことで、広い田でも効率よく作業できます。
    difficulty: standard
    type: explanation
    hints:
      - 広い田を少ない人数で管理します。
  - prompt: 収穫した稲から、もみを取る作業を何といいますか。
    answer: 脱穀
    explanation: 稲の穂からもみを外す作業を脱穀といいます。
    difficulty: standard
    type: term
    hints:
      - 穂から実を外します。
  - prompt: 米の産地では、気候や地形のどんな条件が役立ちますか。
    answer: 水が得やすい、平らな土地がある、夏に気温が上がるなど
    explanation: 水田を作りやすく、稲が育ちやすい条件がそろうと米づくりに向いています。
    difficulty: challenge
    type: geography
    hints:
      - 水、土地、気温を考えます。
  - prompt: 農家が安全でおいしい米を届けるためにしている工夫を一つ書きましょう。
    answer: 品種を選ぶ、肥料や水を管理する、病気や害虫を防ぐ、品質を検査するなど
    explanation: 自然条件に合わせた管理と、収穫後の検査や保管によって、品質を保ちます。
    difficulty: challenge
    type: explanation
    hints:
      - 育てる前、育てている間、収穫後の工夫を考えます。
---
