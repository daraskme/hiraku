problems:
  - prompt: 世界の六大陸を一つ答えましょう。
    answer: ユーラシア大陸、アフリカ大陸、北アメリカ大陸、南アメリカ大陸、南極大陸、オーストラリア大陸 など
    explanation: 世界は大きな陸地のまとまりとして六大陸に分けられます。
    difficulty: basic
    type: geography
    hints:
      - 日本がある大陸も含まれます。
  - prompt: 世界の三大洋を一つ答えましょう。
    answer: 太平洋、大西洋、インド洋
    explanation: 三大洋は世界の大きな海洋のまとまりです。
    difficulty: basic
    type: geography
    hints:
      - 日本の東に広がる大きな海も含まれます。
  - prompt: 赤道から北または南へ何度かを表すものを何といいますか。
    answer: 緯度
    explanation: 緯度は赤道を基準に、南北の位置を表します。
    difficulty: basic
    type: term
    hints:
      - 北緯、南緯といいます。
  - prompt: 本初子午線から東または西へ何度かを表すものを何といいますか。
    answer: 経度
    explanation: 経度は本初子午線を基準に、東西の位置を表します。
    difficulty: basic
    type: term
    hints:
      - 東経、西経といいます。
  - prompt: 赤道に近い地域に多い、年間を通して暑い気候帯を何といいますか。
    answer: 熱帯
    explanation: 熱帯は赤道付近に広がり、気温が高い地域が多いです。
    difficulty: standard
    type: climate
    hints:
      - 「熱」という字が入ります。
  - prompt: 砂漠が広がりやすい、雨が少ない気候帯を何といいますか。
    answer: 乾燥帯
    explanation: 乾燥帯は降水量が少なく、砂漠や草原が広がる地域があります。
    difficulty: standard
    type: climate
    hints:
      - 水分が少ない地域です。
  - prompt: 地球儀と世界地図を比べたとき、地球儀のよい点を一つ書きましょう。
    answer: 地球の形や距離・方位のゆがみが少ない
    explanation: 地球儀は球体なので、地球全体の形に近く表せます。
    difficulty: challenge
    type: explanation
    hints:
      - 地球は平面ではなく球に近い形です。
  - prompt: 世界地図のよい点を一つ書きましょう。
    answer: 一度に広い範囲を見たり、持ち運んだりしやすい
    explanation: 世界地図は平面なので、全体を見渡したり書き込んだりしやすいです。
    difficulty: challenge
    type: explanation
    hints:
      - 使いやすさを考えます。
