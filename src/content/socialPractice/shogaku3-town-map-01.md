problems:
  - prompt: 地図で、上が北のとき、右はどの方位ですか。
    answer: 東
    explanation: 北を上にすると、右が東、左が西、下が南です。
    difficulty: basic
    type: direction
    hints:
      - 東西南北の並びを思い出しましょう。
  - prompt: 学校、図書館、市役所のように、みんなが使う建物を何といいますか。
    answer: 公共施設
    explanation: 多くの人が利用する施設を公共施設といいます。
    difficulty: basic
    type: term
    hints:
      - 公共は「みんなの」という意味です。
  - prompt: 地図で田や畑が多いところは、どんな仕事がさかんだと考えられますか。
    answer: 農業
    explanation: 田や畑が多い地域では、作物を育てる仕事がさかんです。
    difficulty: basic
    type: inference
    hints:
      - 米や野菜を作ります。
  - prompt: 地図記号を使うと、地図はどんなよいところがありますか。
    answer: 施設や土地のようすを小さくわかりやすく表せる
    explanation: 地図記号は、限られた場所に多くの情報を入れるために使われます。
    difficulty: standard
    type: explanation
    hints:
      - 長い文字を書かずにすみます。
  - prompt: まちの中心に駅や商店が集まりやすい理由を一つ書きましょう。
    answer: 人が集まりやすく、買い物や移動に便利だから
    explanation: 交通が便利な場所には店や施設が集まりやすいです。
    difficulty: standard
    type: explanation
    hints:
      - 人の移動を考えます。
  - prompt: 川の近くに田が広がることが多い理由を一つ書きましょう。
    answer: 水を使いやすいから
    explanation: 米づくりには多くの水が必要なので、川や用水路が大切です。
    difficulty: standard
    type: geography
    hints:
      - 米づくりと水の関係です。
  - prompt: 交通量の多い道路の近くで気をつけることを一つ書きましょう。
    answer: 横断歩道を使い、左右をよく見る
    explanation: まちのようすを調べるときは、安全にも注意します。
    difficulty: challenge
    type: safety
    hints:
      - 調査中の安全です。
  - prompt: 自分の家の近くを調べるとき、地図に書きこむとよい情報を三つ書きましょう。
    answer: 道路、公共施設、店、公園、田畑、川など
    explanation: まちの特徴がわかるものを書きこむと、地域のくらしを考えやすくなります。
    difficulty: challenge
    type: map-work
    hints:
      - 人が使う場所、自然、交通を分けて考えます。
