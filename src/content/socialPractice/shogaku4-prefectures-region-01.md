  - prompt: 日本の都道府県は全部でいくつありますか。
    answer: "47"
    explanation: 日本は1都1道2府43県、合わせて47都道府県です。
    difficulty: basic
    type: term
    hints:
      - 1都1道2府43県です。
  - prompt: 北海道地方にある都道府県を一つ答えましょう。
    answer: 北海道
    explanation: 北海道地方は北海道だけで一つの地方です。
    difficulty: basic
    type: geography
    hints:
      - 地方名と同じ名前です。
  - prompt: 東京都の県庁所在地にあたる都市名を答えましょう。
    answer: 東京
    explanation: 東京都の行政の中心は東京です。
    difficulty: basic
    type: geography
    hints:
      - 都の名前と同じです。
  - prompt: 大阪府の県庁所在地にあたる都市名を答えましょう。
    answer: 大阪市
    explanation: 大阪府の府庁所在地は大阪市です。
    difficulty: basic
    type: geography
    hints:
      - 府の名前と同じ都市です。
  - prompt: 中部地方にふくまれる県を一つ答えましょう。
    answer: 愛知県、静岡県、長野県、新潟県、富山県、石川県、福井県、山梨県、岐阜県 など
    explanation: 中部地方は日本列島の中央部に広がる地方です。
    difficulty: standard
    type: geography
    hints:
      - 名古屋市がある県も中部地方です。
  - prompt: 九州地方にふくまれる県を一つ答えましょう。
    answer: 福岡県、佐賀県、長崎県、熊本県、大分県、宮崎県、鹿児島県 など
    explanation: 九州地方は九州本島と周辺の島々を中心とする地方です。
    difficulty: standard
    type: geography
    hints:
      - 沖縄県を九州・沖縄地方として扱うこともあります。
  - prompt: 県庁所在地とは何ですか。
    answer: 都道府県の役所がある中心の都市
    explanation: 県庁、府庁、都庁、道庁が置かれている都市を県庁所在地と呼びます。
    difficulty: standard
    type: explanation
    hints:
      - 行政の中心です。
  - prompt: 地方区分を覚えると、都道府県を学ぶときに何が便利ですか。
    answer: 場所やまとまりで整理しやすくなる
    explanation: 県名をばらばらに覚えるより、地方ごとのまとまりで見ると位置関係を理解しやすくなります。
    difficulty: challenge
    type: explanation
    hints:
      - 地図の中で近い県をまとめられます。
