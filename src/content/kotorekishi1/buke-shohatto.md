<h3>1615 年 7 月 ─ 大 坂 夏 の 陣 直 後 の 法 整 備</h3>

<p>
慶 長 20 年 (1615) 5 月、 大 坂 夏 の 陣 で 徳 川 家 康 が 豊 臣 家 を 滅 ぼ し た。 戦 国 時 代 が 完 全 終 結。
そ の 二 ヶ 月 後 の 7 月、 家 康・秀 忠 (二 代 将 軍) は 全 国 の 大 名 を 京 都 伏 見 城 に 集 め、
一 つ の 法 を 申 し 渡 し た ─ 『武 家 諸 法 度』 で あ る。
</p>

<p>
全 13 条 の こ の 法 が、 そ の 後 260 年 続 く 江 戸 幕 府 の 大 名 統 制 の 基 礎 と な っ た。
</p>

<h3>1615 年 法 度 ─ 主 要 条 文</h3>

<p>
1615 年『武 家 諸 法 度』 の 主 要 な 条 文 ─
</p>

<ol>
  <li><strong>第 一 条</strong> ─ 「文 武 弓 馬 の 道、 専 ら 相 嗜 む べ き 事」 (学 問 と 武 芸 を 励 め)</li>
  <li><strong>第 二 条</strong> ─ 「群 飲 佚 遊 を 制 す べ き 事」 (酒 や 遊 興 に 浸 る な)</li>
  <li><strong>第 五 条</strong> ─ 「諸 国 の 居 城、 修 補 を 為 す と 雖 も、 必 ず 言 上 す べ し」 (城 の 修 理 も 幕 府 に 届 け 出 よ)</li>
  <li><strong>第 六 条</strong> ─ 「隣 国 に 於 い て 新 儀 を 企 て、 徒 党 を 結 ぶ 者、 之 れ あ ら ば、 早 く 言 上 す べ き 事」 (新 し い 動 き を 起 こ す 者 が い れ ば 通 報 せ よ)</li>
  <li><strong>第 八 条</strong> ─ 「私 (ひ そ か) に 婚 姻 を 締 ぶ べ か ら ざ る 事」 (大 名 間 の 婚 姻 は 幕 府 の 許 可 を 得 よ)</li>
</ol>

<p>
全 体 と し て の 趣 旨 は ─ 「武 力 で 反 乱 す る な、 城 や 結 婚 で 同 盟 す る な、 文 化 ・ 学 問 に 励 め」。
</p>

<h3>第 一 条 の 革 命 ─ 「文」 が 「武」 の 前 に</h3>

<p>
注 目 す べ き は 第 一 条 の 「文 武」 ─ 文 (学 問) が 武 (武 芸) の 前 に 書 か れ て い る。
戦 国 期 ま で の 武 士 は「武 一 辺 倒」 だ っ た が、 江 戸 期 か ら は「学 問 と 武 芸 の 両 立」 が 求 め ら れ る。
</p>

<p>
こ の 一 文 が、 江 戸 期 の 武 士 文 化 を 大 き く 変 え た。 全 国 の 藩 校 が 設 立 さ れ、 武 士 が 儒 学・歴 史・
和 漢 学 を 学 ぶ よ う に な っ た。 後 の 福 沢 諭 吉・新 渡 戸 稲 造 ら の 学 問 文 化 は、 こ の 流 れ か ら 生 ま れ た。
</p>

<h3>1635 年 ─ 徳 川 家 光 の 大 改 訂</h3>

<p>
寛 永 12 年 (1635)、 三 代 将 軍 家 光 が 武 家 諸 法 度 を 大 改 訂 し た。 17 条 に 拡 大。 最 大 の 変 更 は
「参 勤 交 代」 の 制 度 化 だ っ た。
</p>

<p>
1635 年 法 度 第 二 条 ─
</p>

<blockquote>
<p>
「大 名・小 名、 在 江 戸 (え ど に あ る) と 在 国 (こ く に あ る) の 交 代、 相 定 む る 所 也 (な り)。
毎 年 四 月 中、 参 覲 (さ ん き ん) を 致 す べ し」
</p>
</blockquote>

<p>
意 味 ─ 「大 名 は 毎 年 4 月 中、 江 戸 に 参 覲 (出 仕) せ よ。 江 戸 と 領 国 を 交 代 で 滞 在 せ よ」。
</p>

<h3>参 勤 交 代 ─ 三 つ の 効 果</h3>

<p>
参 勤 交 代 制 度 は、 江 戸 幕 府 が 260 年 続 い た 秘 密 と 言 わ れ る ─ 三 つ の 効 果 が あ っ た。
</p>

<ol>
  <li><strong>反 乱 の 抑 止</strong> ─ 大 名 の 妻 子 は 江 戸 に 在 住 (人 質)。 大 名 自 身 も 1 年 ご と に 江 戸 に 来 る た め、 反 乱 を 起 こ し に く い。</li>
  <li><strong>諸 藩 の 財 政 圧 迫</strong> ─ 大 名 行 列 の 費 用 (家 臣 数 百 ~ 千 人 が 数 ヶ 月 旅 行) は 大 名 持 ち。 強 い 藩 ほ ど 出 費 大 で、 軍 事 力 強 化 に 使 え な い。</li>
  <li><strong>商 業・宿 場・街 道 の 発 展</strong> ─ 大 名 行 列 が 通 る 街 道 (東 海 道 ・ 中 山 道 等) と 宿 場 町 が 発 展。 江 戸 が 人 口 100 万 人 の 大 都 市 と な る。</li>
</ol>

<h3>違 反 の 処 罰 ─ 改 易 ・ 転 封</h3>

<p>
武 家 諸 法 度 違 反 は 厳 し く 罰 せ ら れ た ─
</p>

<ul>
  <li><strong>改 易 (か い え き)</strong> ─ 領 地 没 収 ・ 家 取 り 潰 し。 福 島 正 則 (1619, 城 修 理 無 届 け), 本 多 正 純 (1622, 失 政) 等</li>
  <li><strong>転 封 (て ん ぽ う)</strong> ─ 領 地 を 別 の 場 所 に 替 え る。 反 抗 的 大 名 を 不 便 な 地 域 へ 移 動</li>
  <li><strong>減 封 (げ ん ぽ う)</strong> ─ 領 地 を 減 ら す。 反 抗 的 大 名 へ の 警 告</li>
</ul>

<p>
江 戸 期 を 通 し て、 改 易 さ れ た 大 名 は 約 240 家 に 上 る。 武 家 諸 法 度 の 抑 止 力 が 強 力 だ っ た こ と が
わ か る。
</p>

<h3>260 年 の 平 和 ─ パ ク ス・ト ク ガ ワ ー ナ</h3>

<p>
武 家 諸 法 度 が 機 能 し た 結 果、 江 戸 幕 府 は 1603 年 か ら 1868 年 ま で の 約 260 年 間、 大 規 模 な
内 戦 を 起 こ さ ず に 統 治 を 続 け た (一 揆 ・ 反 乱 は あ っ た が 小 規 模)。
</p>

<p>
こ の 長 い 平 和 を「パ ク ス ・ ト ク ガ ワ ー ナ」(Pax Tokugawana ・ 徳 川 の 平 和) と 呼 ぶ こ と が あ る。
同 時 期 の ヨ ー ロ ッ パ で は 三 十 年 戦 争 ・ ナ ポ レ オ ン 戦 争 等 が 続 き、 こ の 長 期 平 和 は 世 界 史 的 に も
珍 し い。
</p>

<h3>結 び ─ 法 で 治 め る 思 想</h3>

<p>
武 家 諸 法 度 が 教 え る の は、 「力 だ け で な く、 法 で 統 治 す る」 と い う 思 想 で あ る。 武 力 で 倒 す
こ と は で き る が、 統 治 す る に は 法 が 必 要 ─ こ の 認 識 が、 江 戸 期 の 安 定 を 生 ん だ。
</p>

<p>
明 治 期 の 帝 国 憲 法 (1889)、 戦 後 の 日 本 国 憲 法 (1947) ─ 全 て、 「法 で 治 め る」 と い う 系 譜 の
上 に あ る。 武 家 諸 法 度 か ら 現 代 日 本 国 憲 法 ま で、 法 の 連 続 性 を 心 に 留 め て、 高 校 日 本 史 を
学 ん で ほ し い。
</p>
