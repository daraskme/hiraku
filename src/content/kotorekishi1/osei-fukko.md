glossary:
  - term: 王政復古 (おうせい ふっこ)
    meaning: 天皇親政の政治体制を復活させること。具体的には、鎌倉幕府以来約 700 年続いた武家政権 (鎌倉・室町・江戸) を終わらせ、天皇を中心とする新政府を樹立すること。
  - term: 大号令 (だい ごうれい)
    meaning: 天皇 が 発する 重大 な 命令。明治 維新 期 で は、王政復古 を 宣言 した 慶応 3 年 12 月 9 日 の もの が 「王政復古 の 大号令」 と 呼ばれる。
  - term: 大政奉還 (たいせい ほうかん)
    meaning: 1867 年 10 月 14 日、第 15 代 将軍 徳川慶喜 (とくがわ よしのぶ) が、政権 を 朝廷 に 返上 する こと を 願い出た 政治 行動。これ により、形式 的 に 江戸 幕府 の 政治 権限 は 朝廷 に 戻った。だが、徳川 家 の 実質 的 権力 (領地・軍事 力) は そのまま 残った。
  - term: 王政復古 と 大政奉還 の 関係
    meaning: 大政奉還 (1867年10月) は 「政治 権限 の 返上 」 を 表明 した のみ で、徳川 家 は 依然 と して 巨大 な 領地 と 軍事 力 を 保持 して いた。王政復古 の 大号令 (1867年12月) は、その 徳川 家 を 完全 に 政治 体制 から 排除 した、決定的 な クーデター だった。
  - term: 岩倉具視 (いわくら ともみ)
    meaning: 1825–1883。明治 維新 の 中心 人物 の 一人。 公家 出身 の 政治家。 王政復古 の 大号令 を 起草・実現 する 中核 と なった。 後 に 岩倉 使節団 (1871–1873) の 全権 大使 として 欧米 を 巡視 した。
  - term: 御所 (ごしょ)
    meaning: 天皇 の 居所。京都 御所 が 中心。 1868 年 の 王政復古 は、京都 御所 内 の 小御所 で 宣言 された。
  - term: 小御所 会議 (こ ごしょ かいぎ)
    meaning: 王政復古 の 大号令 発布 の 同日 夜、京都 御所 の 小御所 で 開催 された 緊急 会議。徳川 慶喜 の 処分 を 議論 した。最終 的 に 「辞官 納地」 (官職 と 領地 の 返上) が 決議 された。
  - term: 辞官 納地 (じかん のうち)
    meaning: 「官職 を 辞し、領地 を 朝廷 に 納める」 の 意。小御所 会議 で 徳川 慶喜 に 課された 処分。これ が 後 の 戊辰 戦争 (1868–1869) の 火種 と なった。
  - term: 戊辰 戦争 (ぼしん せんそう)
    meaning: 1868–1869。王政復古 の 大号令 後、徳川 派 (旧 幕府 軍・会津 藩・東北 諸藩) と 新政府 (薩摩・長州・土佐) の 内戦。鳥羽・伏見 の 戦い (1868年1月) から 五稜郭 の 戦い (1869年5月) まで。新政府 側 の 勝利 で 明治 維新 が 完成 した。
  - term: 関白 (かんぱく)・将軍 (しょうぐん)・摂政 (せっしょう)
    meaning: 王政復古 で 「廃止」 された 三 つ の 主要 政治 職。 関白 は 公家 政治 の 最高 職、将軍 は 武家 政治 の 最高 職、摂政 は 天皇 幼少 期 の 代理 職。これら を 一掃 した の が 王政復古 の 大号令。

<h3>現代語訳</h3>

<p>
徳川 内大臣 (慶喜) が、先 ごろ 朝廷 に 「大政 (政治 の 権限) を 返上 する」 と 申し出 たこと、また 「将軍 職 を 辞退 する」 という 両 件 を、今回、断然、お聞き 入れ に なら れた。
</p>

<p>
そもそも 嘉永 6 年 (1853 ─ ペリー 来航) 以来 の 未 曾有 の 国難 (黒船 来航・開国 を めぐる 動揺) に より、先帝 (孝明 天皇) は 長年 心 を 痛められて こられた。 これ は、多くの 人 が 知る ところ で ある。
</p>

<p>
こうした 状況 を 受けて、(明治 天皇 が) ご 決断 を なされ、<strong>王政復古</strong>、すなわち 天皇 親政 を 復活 さ せ、国威 を 取り戻す 基礎 を お立て に なる こと と なった。<br>
これ に より、これ から は、<strong>摂政・関白</strong>、<strong>幕府</strong> など は すべて 廃止 さ れる。<br>
そして 当面、仮 に、<strong>総裁・議定・参与</strong> の 三 職 を 設置 し、すべて の 国政 を、ここ で 裁断 さ れる こと と なった。
</p>

<h3>解説 ─ 「たった 一夜」 の クーデター</h3>

<p>
慶応 3 年 12 月 9 日 (新暦 1868 年 1 月 3 日)、夜 明け 前。<br>
京都 御所 の 周辺 を、薩摩・長州・土佐・芸州・尾張・越前 ── 六 藩 の 兵 が ひそかに 占拠 した。<br>
そして 夜明け と 同時 に、岩倉具視 ら 公家 が 朝廷 内 で 一気に 動き、上記 の 大号令 を 宣言 さ せた。
</p>

<p>
これ は、合法 的 な 政治 過程 を 装い ながら、実 体 と して は <strong>軍事 占拠 を 伴う クーデター</strong> で あった。<br>
260 年 続いた 江戸 幕府 の 終焉 は、こうして <strong>たった 一 晩</strong> で 宣言 された の で ある。
</p>

<p>
慶喜 が 大政 奉還 で 「政治 から 退きます」 と 言った だけ で は、徳川 家 は <strong>巨大 な 領地 と 軍事 力 を 保持</strong> した まま。<br>
これ で は 「名 ばかり の 維新」 で 終わる ─ そう 判断 した 岩倉具視・西郷隆盛・大久保利通 たち は、わずか 2 ヶ月 後、徳川 を <strong>完全 に 政治 体制 から 排除</strong> する 王政復古 を 決行 した の で ある。
</p>

<h3>「三 職」 ─ 新しい 政治 機構</h3>

<p>
大号令 で は、廃止 さ れた もの と、新設 された もの が ある。
</p>

<table>
<thead>
<tr><th>廃止</th><th>新設</th></tr>
</thead>
<tbody>
<tr><td>摂政・関白 (公家 政治 の 頂点)</td><td>総裁 (議長 ・ 有栖川宮 熾仁 親王)</td></tr>
<tr><td>幕府・将軍 職 (武家 政治 の 頂点)</td><td>議定 (公家・諸侯 から 10 名)</td></tr>
<tr><td>京都 守護職・京都 所司代</td><td>参与 (実務 担当・薩摩 長州 土佐 など から)</td></tr>
</tbody>
</table>

<p>
ここで 注目 す べき は、新設 された 「三 職」 が、<strong>公家・諸侯・下級 武士 を 横断 した 政府</strong> だった こと だ。<br>
特 に 「参与」 に は、西郷 隆盛・大久保 利通・木戸 孝允 など の 下級 武士 が 抜擢 された。 これ が、後 の 明治 政府 の 中核 と なる。
</p>

<h3>小御所 会議 と 戊辰 戦争 の 火種</h3>

<p>
大号令 発布 当日 の 夜、京都 御所 の <strong>小御所</strong> で 緊急 会議 が 開かれた。<br>
出席 した の は、新 「議定」 ・ 「参与」 ら 約 20 名。
</p>

<p>
議題 は ─ <strong>「徳川慶喜 を どう 処分 する か」</strong>。
</p>

<p>
強硬 派 (岩倉・大久保・西郷) ─「将軍 職 だけ で なく、官職 (内大臣) も 辞退 さ せ、領地 も 返上 さ せ よ」<br>
穏健 派 (山内 容堂・松平 春嶽) ─ 「慶喜 も 議定 に 加える べき で ある。 そもそも 彼 抜き で 新 政府 は 不安定 だ」
</p>

<p>
激論 の 末、岩倉 が 「迷う ような 公家 は 御簾 (みす) の 外 で 切る」 と すごみ、強硬 派 の 主張 が 通った。<br>
慶喜 は 「辞官 納地」 (官職 と 領地 の 返上) を 命じられた。
</p>

<p>
これ が、約 一 ヶ月 後 の <strong>鳥羽・伏見 の 戦い</strong> (1868年1月3日 ─ 偶然 にも 王政復古 の 新暦 と 同日!) に つながる。<br>
戊辰 戦争 (1868–1869) は 約 一年 半 続き、新 政府 軍 が 旧 幕府 軍 を 各地 で 撃破。1869 年 5 月 の 五稜郭 (函館) の 戦い で、ついに 旧 幕府 派 の 抵抗 は 終わった。
</p>

<h3>「復古」 か 「革命」 か</h3>

<p>
「王政復古」 と いう 名称 は、興味深い。
</p>

<p>
表 看板 は 「<strong>復古</strong>」 ─ 天皇 親政 が 復活 する。<br>
中身 は しかし、まったく 新しい 政治 体制 ─ 西洋 を モデル と した <strong>近代 国民 国家</strong> の 樹立。
</p>

<p>
この 「<strong>復古 を 装った 革命</strong>」 と いう 戦略 が、明治 維新 を 比較 的 平和 的 に 進める 鍵 と なった。<br>
「私 たち は 新しい 思想 を 押し付けて いる の で は ない。古 の 天皇 親政 に 戻る だけ だ」 ── そう 主張 する こと で、保守 派 の 反発 を 和らげた の で ある。
</p>

<p>
これ は、フランス 革命 (1789) や ロシア 革命 (1917) の よう に 「新しい 体制」 を 真正面 から 打ち出した 革命 と は 対照 的 な、日本 独特 の 政治 手法 と 言える。
</p>

<h3>三 部 作 と して の 読み</h3>

<p>
本サイト の kotorekishi1 で は、明治 維新 の 三 大 政治 文書 が 揃った。
</p>

</ol>

<p>
わずか 4 年 弱 の 間 に、260 年 続いた 江戸 体制 が 完全 に 解体 され、近代 国民 国家 の 骨格 が 形成 された。<br>
世界 史 上 で も きわめて 急速 な 構造 変革 で あった。<br>
そして その 第一 歩 が、慶応 3 年 12 月 9 日 の 早朝、京都 御所 で 響いた 一 句 ── 「<strong>王政復古、国威 挽回 ノ 御 基 立 テラセラレ 候</strong>」 だった の で ある。
</p>
