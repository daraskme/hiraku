<h3>あらすじ</h3>

<p>
真 夏 の 暑 い 日 の こ と。 田 舎 の 宿 場 に は、 一 台 の 古 い 馬 車 が 停 ま っ て い た。<br>
馬 車 の 御 者 (ぎ ょ し ゃ) は、 出 発 の 時 間 を 待 ち な が ら 居 眠 り を し て い る。
</p>

<p>
馬 車 の 中 に は、 一 匹 の 蠅 が 止 ま っ て い る。<br>
こ の <strong>蠅</strong> こ そ が、 こ の 物 語 の 「観 察 者」 で あ る。
</p>

<p>
や が て、 馬 車 に 七 人 の 乗 客 が 集 ま り 始 め た。<br>
そ れ ぞ れ が、 そ れ ぞ れ の 事 情 を 抱 え て い る:
</p>

<ul>
<li><strong>母 と 子</strong> ─ 重 病 の 父 に 一 目 会 い に 行 こ う と し て い る</li>
<li><strong>若 い 男 女</strong> ─ 駆 け 落 ち の 途 中</li>
<li><strong>田 舎 紳 士</strong> ─ 商 売 で 大 金 を 持 ち、 急 い で 取 引 に 行 く 途 中</li>
<li><strong>農 夫 親 子</strong> ─ 都 会 で 一 旗 揚 げ よ う と し て い る</li>
</ul>

<p>
そ れ ぞ れ が「<strong>今 す ぐ 出 発 し て く れ</strong>」 と 御 者 を せ か す。 だ が 御 者 は な か な か 起 き な い。<br>
や っ と 出 発 し た 馬 車 は、 急 ぎ 足 で 山 道 を 進 ん だ。
</p>

<p>
だ が ─ 御 者 は、 山 道 の 途 中 で <strong>再 び 居 眠 り を 始 め て し ま う</strong>。<br>
馬 は 自 分 の 判 断 で 進 む。 そ し て、 道 を 曲 が り そ こ ね、 馬 車 は 山 道 の 崖 (が け) か ら 一 気 に <strong>転 落</strong> し た。
</p>

<p>
七 人 の 乗 客 と 御 者、 そ し て 馬 ─ す べ て が、 谷 底 に 落 ち て 死 ん で し ま っ た。
</p>

<p>
そ の 瞬 間 ─ 蠅 だ け が、 馬 車 か ら ふ わ り と 飛 び 立 ち、 真 夏 の 青 空 の 中 へ と 消 え て い っ た。
</p>

<h3>解 説 ─ 革 命 的 な 「視 点」</h3>

<p>
「蠅」 の 文 学 史 上 の 衝 撃 は、 こ の <strong>「視 点」</strong> に あ る。
</p>

<p>
普 通 の 小 説 は、 主 人 公 (人 間) か、 神 の 視 点 (作 者) か ら 描 か れ る。<br>
だ が 横 光 利 一 は ─ <strong>一 匹 の 蠅 を 「観 察 者」</strong> と し た。
</p>

<p>
こ の 視 点 か ら 物 語 を 描 く と、 こ う な る:
</p>

<ul>
<li>人 間 た ち の 「事 情」 は、 蠅 に は 関 係 が な い</li>
<li>母 子 の 願 い も、 駆 け 落 ち の 恋 も、 紳 士 の 商 売 も、 す べ て 「小 さ な 出 来 事」 と 化 す</li>
<li>馬 車 の 転 落 と い う 大 惨 事 が、 蠅 の 「飛 び 立 ち」 と 同 じ 重 さ で 描 か れ る</li>
</ul>

<p>
こ の 「<strong>人 間 中 心 的 視 点 か ら の 解 放</strong>」 が、 「蠅」 の 革 命 的 な 部 分 で あ る。
</p>

<h3>新 感 覚 派 と い う 文 学 運 動</h3>

<p>
横 光 利 一 は、 翌 1924 年 (大 正 13 年)、 川 端 康 成 ら と 同 人 誌『<strong>文 芸 時 代</strong>』 を 創 刊 し た。<br>
こ の 同 人 か ら 生 ま れ た 文 学 運 動 が、 「<strong>新 感 覚 派</strong>」 で あ る。
</p>

<p>
新 感 覚 派 の 特 徴:
</p>
<ul>
<li><strong>比 喩 の 革 新</strong> ─ 「真 昼 の 海 は は が ね の よ う だ」 と い う 鮮 や か な 直 喩</li>
<li><strong>視 点 の 実 験</strong> ─ 「蠅」 の よ う な 非 人 間 視 点</li>
<li><strong>感 覚 の 直 接 性</strong> ─ 説 明 で は な く 感 覚 の 提 示</li>
<li><strong>近 代 性 へ の 関 心</strong> ─ 機 械 ・ 都 市 ・ ス ピ ー ド の 主 題</li>
</ul>

<p>
横 光 と と も に こ の 運 動 を 牽 引 し た 川 端 康 成 は、 後 に 1968 年 に ノ ー ベ ル 文 学 賞 を 受 賞 し た。<br>
日 本 近 代 文 学 の 大 き な 一 つ の 系 譜 が、 こ こ か ら 始 ま っ て い る。
</p>

<h3>「蠅」 と い う タ イ ト ル</h3>

<p>
タ イ ト ル が 「七 人 の 乗 客」 で も「馬 車 の 転 落」 で も な く、 「<strong>蠅</strong>」 で あ る こ と が、 こ の 物 語 の 核 心 を 示 す。
</p>

<p>
人 間 た ち が 必 死 に 抱 え て い る 「事 情」「希 望」「悲 嘆」 ─ そ れ ら は、 蠅 か ら 見 れ ば、 何 の 意 味 も な い。<br>
そ し て、 大 惨 事 の 後 も、 蠅 は ふ わ り と 飛 び 立 つ。 人 間 の 生 と 死 と は 関 係 な く、 自 然 の 営 み と し て。
</p>

<p>
こ の 視 点 が、 読 者 に <strong>「人 間 中 心 的 思 考 へ の 揺 さ ぶ り」</strong> を 与 え る。
</p>

<h3>あ な た 自 身 で 全 文 を</h3>

<p>
「蠅」 は 2000 字 程 度 の 短 い 短 編。<br>
青 空 文 庫 で 全 文 が 読 め る。 ぜ ひ、 一 度 読 ん で、 「蠅 の 視 点」 を 体 験 し て み て ほ し い。
</p>

<p>
そ し て、 自 分 が「人 間 中 心」 で 物 事 を 見 て い る こ と に 気 づ い た と き、 こ の 一 篇 の 革 命 性 が、 君 に も 伝 わ る は ず だ。
</p>
