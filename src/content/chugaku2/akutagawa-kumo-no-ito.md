<h3>あらすじ</h3>

<p>
あ る 日、 お 釈 迦 様 は 極 楽 の 蓮 池 の ふ ち を 散 歩 し て い た。<br>
ふ と 蓮 池 の 底 を 覗 く と、 地 獄 で 苦 し む 罪 人 た ち の 姿 が 見 え た。<br>
そ の 中 に <strong>カ ン ダ タ</strong> と い う 大 盗 賊 を 見 つ け る。 生 前 は 人 殺 し や 放 火 を 繰 り 返 し た 極 悪 人 だ。
</p>

<p>
だ が お 釈 迦 様 は 思 い 出 す ─ <strong>カ ン ダ タ は 一 度 だ け、 道 端 の 小 さ な 蜘 蛛 を 踏 み 殺 さ ず に 助 け た こ と が あ る</strong>。<br>
そ の 小 さ な 善 行 に 応 え て、 お 釈 迦 様 は 一 本 の 蜘 蛛 の 糸 を 地 獄 ま で 垂 ら し て や ろ う と 考 え た。
</p>

<p>
血 の 池 地 獄 で 苦 し む カ ン ダ タ の 前 に、 銀 色 に 輝 く 蜘 蛛 の 糸 が す っ と 降 り て き た。<br>
「こ れ を 登 れ ば 極 楽 に 行 け る か も し れ な い」 と、 カ ン ダ タ は 必 死 に 糸 を 登 り 始 め た。
</p>

<p>
し ば ら く 登 っ て、 ふ と 下 を 見 る と ─ な ん と、 自 分 の 後 ろ に も、 数 限 り な い 罪 人 た ち が、 同 じ 糸 に 取 り す が っ て 登 っ て き て い る で は な い か!
</p>

<p>
「<strong>こ の 細 い 糸 で、 こ ん な に 大 勢 が 登 れ る か! 糸 が 切 れ て 自 分 ま で 落 ち て し ま う!</strong>」<br>
カ ン ダ タ は 思 わ ず 大 声 で 怒 鳴 っ た:
</p>

<blockquote>
「こ ら、 罪 人 ど も。 こ の 蜘 蛛 の 糸 は <strong>俺 の 物</strong> だ ぞ。 お 前 た ち は い っ た い 誰 に 尋 ね て、 登 っ て 来 た。 降 り ろ。 降 り ろ」
</blockquote>

<p>
そ の 瞬 間 ─ <strong>蜘 蛛 の 糸 は カ ン ダ タ の ぶ ら 下 が る 真 上 で プ ツ リ と 切 れ た</strong>。<br>
カ ン ダ タ は、 他 の 罪 人 た ち と 共 に、 ま っ さ か さ ま に 地 獄 へ と 落 ち て い っ た。
</p>

<p>
お 釈 迦 様 は そ の 様 子 を 蓮 池 の ふ ち か ら 見 て い た。<br>
カ ン ダ タ の <strong>「自 分 だ け 助 か り た い」</strong> と い う 無 慈 悲 な 心 が、 結 局 自 分 自 身 を 滅 ぼ し た こ と を、 悲 し ま し く 思 い、 ま た 静 か に 蓮 池 の 周 り の 散 歩 を 続 け る の だ っ た。
</p>

<h3>解 説 ─ 「赤 い 鳥」 創 刊 号 の 物 語</h3>

<p>
「蜘 蛛 の 糸」 は、 1918 年 (大 正 7 年) 7 月、 鈴 木 三 重 吉 (す ず き み え き ち) が 創 刊 し た 児 童 文 学 雑 誌『<strong>赤 い 鳥</strong>』の <strong>創 刊 号</strong> を 飾 っ た 作 品 で あ る。
</p>

<p>
当 時、 日 本 の 子 ど も 向 け の 本 は 道 徳 的・教 訓 的 な も の が 多 か っ た。<br>
鈴 木 三 重 吉 は こ う 考 え た:「子 ど も に も 一 流 の 文 学 を」 と。<br>
そ し て、 当 時 26 歳・既 に 注 目 さ れ て い た 新 進 作 家 ・ 芥 川 龍 之 介 に、 創 刊 号 の 童 話 を 依 頼 し た。
</p>

<p>
芥 川 が 書 き 上 げ た の が、 こ の 「蜘 蛛 の 糸」 だ っ た。<br>
原 型 は、 ド イ ツ 系 ア メ リ カ 人 の 思 想 家 <strong>パ ウ ル・カ ー ラ ス</strong> (Paul Carus) が 1894 年 に 編 ん だ 仏 教 説 話 集 『カ ル マ』 (Karma) と さ れ る。<br>
芥 川 は こ の 西 洋 経 由 の 仏 教 説 話 を、 日 本 語 の 美 し い 文 体 で 書 き 直 し た の で あ る。
</p>

<h3>主 題 ─ 「自 分 だ け が」 の 危 険</h3>

<p>
こ の 物 語 の 中 心 主 題 は、 シ ン プ ル で 痛 烈 で あ る。
</p>

<p>
カ ン ダ タ は、 蜘 蛛 を 一 匹 助 け た 小 さ な 善 行 に よ っ て、 お 釈 迦 様 か ら 救 い の 糸 を 与 え ら れ た。<br>
だ が、 そ の 糸 を 登 る 途 中、 「<strong>俺 の 物</strong>」 と 叫 ん だ 瞬 間、 糸 は 切 れ た。
</p>

<p>
こ の 物 語 が 伝 え る の は ─<br>
「<strong>『自 分 だ け が 救 わ れ た い』 と 思 う 心 は、 結 局、 自 分 自 身 を 滅 ぼ す</strong>」<br>
と い う 仏 教 的 真 理 で あ る。
</p>

<p>
カ ン ダ タ が も し 「み ん な も 一 緒 に」 と 言 っ て い た ら、 糸 は 切 れ な か っ た か も し れ な い。<br>
だ が 彼 は、 千 載 一 遇 の 救 い の 機 会 に、 一 番 大 切 な 「他 者 へ の 思 い や り」 を 失 っ て し ま っ た。
</p>

<h3>108 年 後 の 私 た ち へ</h3>

<p>
1918 年 か ら 108 年 ─ こ の 物 語 が 描 く 心 理 は、 今 も 私 た ち の 周 り に 溢 れ て い る。
</p>

<ul>
<li>災 害 時 に 食 料 を 買 い 占 め る 人</li>
<li>SNS で 「い い ね」 を 独 占 し よ う と す る 人</li>
<li>受 験 戦 争 で 友 達 を 蹴 落 と そ う と す る 人</li>
<li>限 ら れ た 資 源 を 「自 分 だ け」 で 使 お う と す る 国 家</li>
</ul>

<p>
カ ン ダ タ の 「俺 の 物 だ ぞ」 と い う 一 言 は、 形 を 変 え て、 今 も 私 た ち の 心 の 中 で 囁 か れ て い る。<br>
そ の 瞬 間、 私 た ち の 「蜘 蛛 の 糸」 も ま た、 プ ツ リ と 切 れ て し ま う か も し れ な い。
</p>

<p>
全 文 は 青 空 文 庫 で 読 め る。 短 い 物 語 だ が、 一 度 は 全 編 を 通 読 し、 そ の 痛 切 な 結 末 を 心 に 刻 ん で ほ し い。
</p>
