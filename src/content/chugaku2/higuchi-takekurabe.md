<h3>本 文 (『た け く ら べ』 樋 口 一 葉 ・ 冒 頭)</h3>

<blockquote>
<p>
廻 (まわ) れ ば 大 門 (お お も ん) の 見 返 り 柳 (や な ぎ) い と 長 (な が) け れ ど、 お 歯 ぐ ろ 溝 (どぶ) に 燈 火 (と も し び) 映 (うつ) る 三 階 の 騒 ぎ も 手 に 取 る 如 (ご と) く、 明 (あ) け 暮 (く) れ な し の 車 の 行 (ゆ) き 来 (き) に は か り 知 ら れ ぬ 全 盛 (ぜ ん せ い) を う か が わ れ て、 大 音 寺 前 (だ い お ん じ ま え) と 名 は 仏 く さ け れ ど、 さ り と は 陽 気 の 町 と 住 (す) む 人 の 申 (も う) し き。
</p>

<p>
三 嶋 神 社 (み し ま じ ん じ ゃ) の 角 を 曲 が り て、 こ れ よ り は 樋 (とい) の 口 と 言 う ま で の 細 道 (ほ そ み ち)、 軒 (の き) の 端 (は) 並 び に 立 て し 「お 引 き 摺 り (ひ き ず り)」 の 提 灯 (ち ょ う ち ん) ち ら り ち ら り 風 に 揺 (ゆ) れ つ つ、 子 ど も の 声 さ え 賑 (に ぎ) わ し く、 大 黒 屋 の 美 登 利 (み ど り) と 言 う 評 判 の 娘 子 (む す め こ) こ こ に 住 む。
</p>
</blockquote>

<h3>現代語訳</h3>

<p>
歩 い て 大 門 を 出 て 振 り 返 る と、 見 返 り 柳 の 木 が と て も 長 く 垂 れ 下 が っ て い る。<br>
お 歯 ぐ ろ 溝 (川) に は 三 階 建 て の 茶 屋 の 灯 (あ か り) が 映 り、 賑 や か な 騒 ぎ が、 ま る で 手 に 取 る よ う に 見 え る。<br>
朝 か ら 晩 ま で、 客 の 人 力 車 が 絶 え 間 な く 行 き 交 う の で、 こ の 町 が 計 り 知 れ な い 賑 わ い に 満 ち て い る こ と が わ か る。
</p>

<p>
こ こ は 「大 音 寺 前」 と 言 う 町 だ。 名 前 こ そ 仏 教 っ ぽ い け れ ど、 実 は と て も 陽 気 な 町 だ と、 住 む 人 た ち は 言 う。
</p>

<p>
三 嶋 神 社 の 角 を 曲 が る と、 そ こ か ら 樋 の 口 (と い の く ち) ま で の 細 い 道 が あ る。<br>
そ の 軒 (の き) 並 み に 並 ぶ 「お 引 き 摺 り」 の 提 灯 が、 ち ら ち ら と 風 に 揺 れ て い る。<br>
子 ど も の 声 も 賑 や か に 響 い て い る。<br>
こ こ に、 大 黒 屋 の 美 登 利 (み ど り) と い う 評 判 の 美 し い 娘 が、 住 ん で い る。
</p>

<h3>解 説 ─ 23 歳 の 名 作</h3>

<p>
樋 口 一 葉 (ひ ぐ ち い ち よ う) は、 明 治 5 年 (1872) に 東 京 で 生 ま れ、 明 治 29 年 (1896)、 わ ず か <strong>24 歳</strong> で 結 核 の た め に 亡 く な っ た。
</p>

<p>
父 を 早 く に 亡 く し、 母 と 妹 と の 三 人 暮 ら し を 支 え る た め、 一 葉 は 雑 貨 屋 を 営 み な が ら 小 説 を 書 い た。<br>
吉 原 遊 郭 の 近 く の 龍 泉 寺 町 で の 経 験 が、 「た け く ら べ」 の 舞 台 と な っ た。
</p>

<p>
「た け く ら べ」 は 23 歳 の と き に 雑 誌 『文 学 界』 に 連 載 さ れ、 森 鴎 外・幸 田 露 伴 ら か ら 絶 賛 さ れ た。<br>
明 治 文 学 史 上 で 最 も 高 い 評 価 を 得 た 中 編 小 説 の 一 つ で あ る。
</p>

<h3>雅 文 体 (が ぶ ん た い) と い う 美</h3>

<p>
こ の 冒 頭 を 読 む と、 ま ず 「読 み に く い」 と 感 じ る か も し れ な い。<br>
そ れ は、 一 葉 が <strong>「雅 文 体 (擬 古 文)」</strong> と い う、 古 典 の 文 体 を 真 似 た 美 し い 古 風 な 書 き 方 を 選 ん だ か ら だ。
</p>

<p>
明 治 後 半 の 一 般 の 文 章 は、 す で に 口 語 (話 し 言 葉) に 近 く な っ て い た。<br>
だ が 一 葉 は、 あ え て 千 年 前 の 『源 氏 物 語』 を 思 わ せ る 文 体 を 使 い、 そ の リ ズ ム の 中 で 明 治 の 子 ど も た ち を 描 い た。
</p>

<p>
こ の 「<strong>古 い 文 体 + 同 時 代 の 子 ど も</strong>」 と い う 組 み 合 わ せ が、 不 思 議 な 詩 情 を 生 ん で い る。
</p>

<h3>「た け く ら べ」 と い う 題</h3>

<p>
「た け く ら べ」 と は、 「丈 (た け)」 を 「比 べ る」 の 意。<br>
子 ど も た ち が 「ど ち ら が 背 が 高 い か」 と 競 い 合 う、 そ ん な 場 面 を 示 す と 共 に、 子 ど も た ち が <strong>「子 ど も か ら 大 人 へ の 境 目」</strong> に い る こ と も 表 し て い る。
</p>

<p>
物 語 の 主 人 公 た ち ─ 美 登 利 (14 歳)、 信 如 (15 歳) ─ は、 ま さ に そ の 境 目 に い る。<br>
や が て 美 登 利 は 姉 と 同 じ く 遊 女 (ゆ う じ ょ) に な る 運 命 が 待 ち、 信 如 は 寺 に 入 っ て 僧 侶 に な る。<br>
別 々 の 大 人 の 道 を 進 む 二 人 の、 ほ ん の 一 瞬 の 淡 い 恋 心 が、 こ の 物 語 の 核 で あ る。
</p>

<h3>五 千 円 札 の 肖 像</h3>

<p>
樋 口 一 葉 は、 2004 年 か ら 2024 年 ま で の 五 千 円 札 の 肖 像 で あ っ た。<br>
明 治 か ら 100 年 以 上 経 っ て、 24 歳 で 亡 く な っ た 女 性 作 家 が、 日 本 の 紙 幣 を 飾 っ た。<br>
そ れ ほ ど、 彼 女 の 作 品 と 生 涯 は、 日 本 人 の 心 に 深 く 刻 ま れ て い る の だ。
</p>

<p>
全 編 を 読 ん で み た い 人 は、 青 空 文 庫 で 無 料 で 読 め る。<br>
雅 文 体 の リ ズ ム に 慣 れ る と、 明 治 の 子 ど も た ち の 世 界 が、 鮮 や か に 心 に 浮 か び 上 が っ て く る は ず だ。
</p>
