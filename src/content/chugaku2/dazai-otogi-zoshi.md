<h3>本文（『御伽草紙・浦島さん』冒頭・太宰治）</h3>

<blockquote>
<p>
むかしむかし、丹後 (たんご) の 国 の 水の江 (みずのえ) という ところに、浦島 という 男 が 住んで おりました。
</p>

<p>
あの 物語 で 有名な、浦島太郎 (うらしま たろう) の こと で ございます。<br>
ただし、その 浦島 さん の こと を、私 たち は あまりに も よく 知って いる つもり で います が、実 は 細かい こと は、よく わかって いない の で は ない でしょうか。
</p>

<p>
そこで、ここ に、その 浦島 さん の 物語 を、もう 一度、ゆっくり と お話 して みたい と 思います。── と 申しまして も、これ が なかなか、私 自身 に も、まだ よく わかって いない の で ありまして。
</p>

<p>
浦島 さん は、地元 で は、決して 評判 の よい 青年 で は ありませんでした。<br>
「変わり者」「気取り屋」「世間知らず」── そんな 陰口 を、村人 たち は しばしば 叩いて いた よう です。
</p>

<p>
というのも、浦島 さん は、海辺 を 散歩 し ながら、しきりに 海 を ながめ、しきりに 空 を ながめ、しきりに ため息 を ついて いる。── 漁師 の 仕事 を きちんと 学べば よい もの を、何 を 物思い (ものおもい) に 沈んで いる の だ ろう か、と。
</p>

<p>
しかし、浦島 さん 本人 に 言わせれば、それ は 物思い で は ない。<br>
「私 は、ただ、ものごと を 美しく 見たい だけ なのです」 と、彼 は 内心 で 思って いた の でした。
</p>

<p>
そして、ある 日 の こと、彼 が いつも の よう に 海辺 を 散歩 して いる と、村 の 子ども たち が 浜辺 で 何 か を いじめて いる の が 目 に 入った。
</p>

<p>
近づいて よく 見て みる と、それ は 大きな 亀 で した ── ── ──
</p>
</blockquote>

<h3>太宰治の語り直し</h3>

<p>
私たちが 知っている 「浦島太郎」 は、こういう ふうに 始まる はず だ：
</p>

<blockquote>
むかしむかし、浦島太郎 という 漁師が おりました。ある日、浜辺で 子どもたち が 亀を いじめて いる の を 見て、お金を 払って 助けて あげました。
</blockquote>

<p>
ところが 太宰治 の 浦島 は、「<strong>地元では 変人 扱い されて いる青年</strong>」 だ。「ものごと を 美しく 見たい だけ なのです」 と 内心で 思って いる、内省的 で 物思いがちな 男 として 描かれる。
</p>

<p>
これが、太宰治 の 翻案文学 の 鮮やかな ところ だ。<br>
有名な 昔話 の キャラクター に、<strong>近代人 の 内面</strong> を 与え、新しい 物語 として 蘇らせる。<br>
「お話 を、もう 一度、ゆっくり と」 と 言い ながら、語り手 (太宰 自身) は、いつのまにか 読者 と 語り 合う 友人 の よう に 振る舞う。
</p>

<h3>『御伽草紙』 が 書かれた 時 と 場所</h3>

<p>
本作 『御伽草紙』 は、<strong>1945 年 (昭和 20 年) 10 月</strong>、戦争 が 終わった 直後 に 出版 された。<br>
だが、太宰 が これ を 書き 始めた の は、まだ 戦争 の さなか だった。
</p>

<p>
本作 の 序文 で、太宰 は こう 書いて いる ──
</p>

<blockquote>
私たちは 防空壕 (ぼうくうごう) の 中 で、子ども に 絵本 を 読んで 聞かせ ながら、暮らして いた。<br>
ある とき、ふと 思った。── これら の 昔話 を、もう少し 大人 の 心 で 読み 直して みたら、どう なる だろう か、と。
</blockquote>

<p>
1945 年 3 月 9 日 の <strong>東京 大 空襲</strong> で、太宰 の 家 は 焼け 落ちた。<br>
妻 と 三 人 の 子 を 連れて、彼 は 甲府 へ、そして 故郷 の 津軽 へ と 疎開 (そかい) し た。<br>
そうした 移動 の さなか、防空壕 の 中 で、太宰 は 子ども に 絵本 を 読み 聞かせ ながら、その 余白 に 大人 向け の 「翻案 御伽草紙」 を 書き 継いだ。
</p>

<h3>戦時下に「昔話」を書くということ</h3>

<p>
「<strong>なぜ 戦争 の さなか に、昔話 を 書いた の か?</strong>」
</p>

<p>
それは、<strong>現実 逃避</strong> でも あった だろう。爆弾 が 落ち、人 が 死に、国 が 滅び ようと して いる 時 ── 「むかしむかし」 と 始まる 物語 は、ひとり の 大人 を 一時 (いっとき) でも 守って くれた。
</p>

<p>
しかし 同時 に、それ は <strong>現実 への 抵抗</strong> でも あった。<br>
時代 が 「忠君愛国」 「滅私奉公」 「神国 日本」 と 叫んで いる さなか、太宰 は ── あえて、「変人」「気取り屋」「世間知らず」 と 呼ばれた 浦島 を 主人公 に した。<br>
「ものごと を 美しく 見たい だけ なのです」 と、内心 で つぶやく ような 青年 を。
</p>

<p>
これ は、戦時 の 大文字 (「英雄」 「祖国」 「忠義」) に 対する、<strong>小文字 の 抵抗</strong> だった の かも しれない。
</p>

<h3>翻案文学 という ジャンル</h3>

<p>
太宰治 の 『御伽草紙』 は、日本 近代 の <strong>翻案文学</strong> の 代表作 の 一つ だ。<br>
他 に も：
</p>

<ul>
<li><strong>芥川龍之介</strong> 『藪の中』 『羅生門』 『地獄変』 ── 平安・室町期 の 説話 を 近代心理小説 に。</li>
<li><strong>中島敦</strong> 『山月記』 『李陵』 『弟子』 ── 中国 古典 を 翻案 (本サイト の kotoko1 で 読める)。</li>
<li><strong>森鴎外</strong> 『高瀬舟』 ── 江戸 期 の 説話 を 近代倫理小説 に (本サイト の chugaku2 で 読める)。</li>
</ul>

<p>
古い 物語 を、新しい 心 で 読み 直す ── これ は、文学 が ずっと 続けて きた 営み で ある。<br>
そして 君 が 今 「浦島太郎」 を 思い 出す とき、 そこ に は <strong>太宰治 の 浦島</strong> も、すでに 重なって いる。
</p>

<h3>結び</h3>

<p>
原文 の 続き ── 浦島 が 亀 を 助け、竜宮城 へ 招かれ、乙姫様 と 出会い、戻って 来た ら 何百年 も 経って いて 玉手箱 を 開けて しまう ── は、青空文庫 で 全文 が 読める。<br>
太宰 の 語り直し は、知って いる はず の 物語 を、まったく 違う 表情 で 君 に 見せて くれる はずだ。
</p>
