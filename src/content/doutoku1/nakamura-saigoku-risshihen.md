## 西 国 立 志 編 ─ 明 治 を 鼓 舞 し た 自 助 論

### 1. 1871 年 ─ 静 岡 学 問 所 で の 訳 刊

明 治 4 年 (1871 年)、 静 岡 の 駿 府 城 下 に、 一 冊 の 翻 訳 書 が 上 梓 (じ ょ う し) さ れ た。 中 村 正 直 (1832-1891) 訳『西 国 立 志 編』 で あ る。

訳 者 中 村 正 直 は、 江 戸 出 身 の 旧 幕 臣。 1866 年 (慶 応 2 年) に 幕 府 留 学 生 と し て イ ギ リ ス に 渡 り、 1868 年 (明 治 元 年) 戊 辰 戦 争 の 敗 戦 と 共 に 帰 国。 旧 幕 臣 と し て 静 岡 に 移 り、 静 岡 学 問 所 (徳 川 家 が 設 け た 旧 幕 臣 子 弟 の 教 育 機 関) の 教 授 と な っ て い た。

そ の 中 村 が、 イ ギ リ ス 滞 在 中 に 読 ん で 衝 撃 を 受 け た 一 冊 を、 日 本 語 に 訳 し た の が、 こ の『西 国 立 志 編』 で あ っ た。

原 著 は、 サ ミ ュ エ ル・ス マ イ ル ズ『Self-Help』 (1859 年)。 ニ ュ ー ト ン、 シ ェ イ ク ス ピ ア、 ジ ェ ー ム ズ・ワ ッ ト、 ジ ョ ー ジ・ス チ ー ブ ン ソ ン、 ナ ポ レ オ ン、 そ し て 無 数 の イ ギ リ ス の 職 人 ・ 発 明 家 ─ 約 300 人 の「自 ら の 努 力 で 大 業 を 成 し 遂 げ た 人 物」 の 物 語 を 集 め た 自 助 論 で あ っ た。

***

### 2. 「天 は 自 ら 助 く る 者 を 助 く」

『西 国 立 志 編』 の 第 一 章 冒 頭 に、 中 村 が 訳 し た 一 句 が あ る。

「天 は 自 ら 助 く る 者 を 助 く (Heaven helps those who help themselves)」

直 訳 す れ ば「天 は、 自 ら を 助 け よ う と す る 者 を、 助 け る」。 つ ま り、 「他 人 の 助 け を 待 つ な。 ま ず 自 分 で 努 力 せ よ。 そ う し て 初 め て、 天 (= 運 命・神) は 助 け て く れ る」 と い う 思 想 で あ る。

こ の 一 句 が、 明 治 ・ 大 正 ・ 昭 和 を 通 じ て 日 本 人 の 座 右 銘 と し て 引 用 さ れ 続 け、 今 で も 学 校 教 育 や ビ ジ ネ ス 書 で 引 用 さ れ る 名 句 と な っ た。

***

### 3. 偉 人 300 人 の 努 力 の 物 語

『西 国 立 志 編』 の 大 部 分 は、 イ ギ リ ス・ヨ ー ロ ッ パ の 偉 人 た ち の 努 力 の 物 語 で 構 成 さ れ て い る。

- 蒸 気 機 関 を 改 良 し た **ジ ェ ー ム ズ・ワ ッ ト** (1736-1819) は、 元 々 グ ラ ス ゴ ー 大 学 の 計 器 修 理 工 で あ っ た。
- **ジ ョ ー ジ・ス チ ー ブ ン ソ ン** (1781-1848) は、 鉱 山 の 機 関 士 の 子 と し て 生 ま れ、 自 ら 読 み 書 き を 学 び、 後 に 鉄 道 機 関 車 を 発 明。
- **ア イ ザ ッ ク・ニ ュ ー ト ン** (1643-1727) は、 早 朝 か ら 深 夜 ま で 物 理 学 の 計 算 を 続 け、 「私 が 偉 い と す れ ば、 そ れ は 努 力 し た か ら だ」 と 述 べ た。
- **シ ェ イ ク ス ピ ア** (1564-1616) は、 田 舎 の 革 手 袋 職 人 の 子 と し て 生 ま れ、 ロ ン ド ン で 劇 場 の 馬 当 番 か ら 出 発 し、 後 に 英 文 学 史 上 最 大 の 作 家 と な っ た。

「生 ま れ は 関 係 な い。 努 力 で 道 は 開 け る」 ─ こ の メ ッ セ ー ジ が、 明 治 維 新 直 後 の 日 本 人 の 心 に 直 接 響 い た。

***

### 4. 福 沢『学 問 の す ゝ め』 と の 双 璧

1872 年 か ら 1876 年 に か け て、 福 沢 諭 吉『学 問 の す ゝ め』 が 累 計 340 万 部 を 売 り 上 げ た。 同 時 期、 中 村 正 直『西 国 立 志 編』 は 100 万 部 を 超 え た。

二 冊 を 比 べ る と、 主 張 は 重 な り つ つ も 微 妙 に 違 っ た。

- 福 沢『学 問 の す ゝ め』 ─ 「天 は 人 の 上 に 人 を 造 ら ず」。 万 人 が 平 等 で あ り、 学 問 に よ っ て 独 立 す る べ し。 平 等・独 立 の 強 調。
- 中 村『西 国 立 志 編』 ─ 「天 は 自 ら 助 く る 者 を 助 く」。 自 助 努 力 こ そ が 道 を 開 く。 偉 人 の 模 範 か ら 学 べ。 努 力・勤 勉 の 強 調。

二 つ の メ ッ セ ー ジ は、 明 治 の 青 年 た ち の 心 を 二 方 向 か ら 鼓 舞 し た。 「平 等 だ か ら こ そ、 努 力 で 道 が 開 け る」 と。

***

### 5. 中 村 正 直 の そ の 後

中 村 正 直 は『西 国 立 志 編』 の 成 功 後、 1873 年 に は 同 時 期 に 帰 国 し た 旧 幕 臣・洋 学 者 (福 沢 諭 吉・森 有 礼・西 周 ら) と 共 に 啓 蒙 結 社「明 六 社 (め い ろ く し ゃ)」 を 結 成。 機 関 誌『明 六 雑 誌』 で 西 洋 思 想 を 紹 介 し 続 け た。

ま た、 同 時 期 に J.S. ミ ル『自 由 之 理』 (On Liberty の 訳) も 訳 刊 し、 明 治 啓 蒙 思 想 の 中 心 人 物 と な っ た。 1875 年 文 部 省 編 書 課 長、 1881 年 元 老 院 議 官、 1889 年 帝 国 大 学 文 科 大 学 教 授 と し て、 明 治 国 家 形 成 に 関 わ り 続 け た。

1891 年 没。 享 年 60 歳。

***

### 6. 明 治 の バ イ ブ ル ─ 今 も 残 る 一 句

中 村 が 訳 し た「天 は 自 ら 助 く る 者 を 助 く」 と い う 一 句 は、 21 世 紀 の 日 本 で も、 学 校 ・ ビ ジ ネ ス 書 ・ ス ポ ー ツ コ ー チ の 言 葉 ─ あ ら ゆ る 場 で 引 用 さ れ る。

150 年 を 超 え て 響 き 続 け る、 明 治 の バ イ ブ ル の 一 句 で あ る。

***

(注：本 文 は、 中 村 正 直 訳『西 国 立 志 編』 (1871) の 第 一 章 と 序 文、 お よ び 関 連 史 料 を、 中 学 道 徳 教 材 と し て 編 集 部 が 再 構 成 し た 解 説 で あ る。 原 書 は 国 立 国 会 図 書 館 デ ジ タ ル コ レ ク シ ョ ン で 全 文 閲 覧 で き る。)
