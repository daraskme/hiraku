<h3>1900 年、 ア メ リ カ で 出 版 さ れ た 日 本 人 論</h3>

<p>
1900 年、 ア メ リ カ・フ ィ ラ デ ル フ ィ ア。<br>
38 歳 の 一 人 の 日 本 人 が、 英 文 で 一 冊 の 本 を 出 版 し た。
</p>

<p>
タ イ ト ル ─『<strong>Bushido: The Soul of Japan</strong>』(武 士 道 ─ 日 本 の 魂)。<br>
著 者 ─ <strong>新 渡 戸 稲 造</strong> (に と べ い な ぞ う、 1862–1933)。
</p>

<p>
こ の 本 が、 世 界 に「日 本 人 の 倫 理」 を 紹 介 し た 最 初 の 名 著 と な っ た。
</p>

<h3>な ぜ 英 文 で 書 い た か</h3>

<p>
新 渡 戸 が こ の 本 を 英 文 で 書 い た 動 機 は、 序 文 に こ う 記 さ れ て い る。
</p>

<blockquote>
ベ ル ギ ー の 法 学 者・ド・ラ ヴ ェ レ 氏 と 話 し て い た 時、 こ う 問 わ れ た:<br>
「日 本 の 学 校 で は 宗 教 教 育 が な い と 聞 い た が、 ど う や っ て 道 徳 を 教 え る の か?」
</blockquote>

<p>
新 渡 戸 は し ば ら く 答 え ら れ な か っ た。<br>
ヨ ー ロ ッ パ で は、 道 徳 = キ リ ス ト 教 の 教 え で あ り、 学 校 で 教 え ら れ る。 日 本 に は そ う し た 制 度 が な い ─ で は、 日 本 人 は ど こ か ら 道 徳 を 学 ん で い る の か?
</p>

<p>
こ の 問 い に 答 え る た め、 新 渡 戸 は 自 ら の 思 索 を ま と め、 1900 年 に『武 士 道』 を 著 し た。<br>
彼 の 答 え は ─ 「日 本 人 は『武 士 道』 と い う 暗 黙 の 倫 理 規 範 に よ っ て 道 徳 を 学 ぶ」 で あ っ た。
</p>

<h3>七 つ の 徳</h3>

<p>
新 渡 戸 は『武 士 道』 で、 武 士 が 重 ん じ た 倫 理 を 七 つ の 徳 (徳 目) と し て 整 理 し た。
</p>

<h4>① 義 (ぎ ・ Rectitude/Justice)</h4>
<p>
「正 し い こ と を 行 う 道 徳 的 勇 気」。 武 士 道 の 根 幹。 「義 を 見 て せ ざ る は 勇 な き な り」 (孔 子)。
</p>

<h4>② 勇 (ゆ う ・ Courage)</h4>
<p>
「正 し い こ と の た め に 危 険 を 冒 す 勇 気」。 単 な る 蛮 勇 で は な く、 義 と 結 び つ い た 勇 気。
</p>

<h4>③ 仁 (じ ん ・ Benevolence/Mercy)</h4>
<p>
「思 い や り、 慈 し み」。 弱 い 者 を 助 け、 敵 に も 情 け を か け る 心。 孔 子 の 「仁」 か ら の 継 承。
</p>

<h4>④ 礼 (れ い ・ Politeness)</h4>
<p>
「礼 儀 正 し さ」。 ル ー ル と 形 式 で は な く、 「他 者 を 思 う 心 が 形 に 現 れ た も の」。
</p>

<h4>⑤ 誠 (せ い ・ Honesty/Sincerity)</h4>
<p>
「真 心、 嘘 を つ か ぬ こ と」。 武 士 は「武 士 に 二 言 は な い」 と し た。 商 人 が 契 約 書 を 必 要 と し た の に 対 し、 武 士 の 約 束 は 言 葉 だ け で 成 立 し た。
</p>

<h4>⑥ 名 誉 (め い よ ・ Honor)</h4>
<p>
「自 尊 心 と 恥 を 知 る 心」。 「名 を 惜 し む」「武 士 の 一 分 (い ち ぶ ん)」 ─ こ れ ら が 武 士 の 行 動 規 範 だ っ た。
</p>

<h4>⑦ 忠 義 (ち ゅ う ぎ ・ Loyalty)</h4>
<p>
「忠 誠 心」。 主 君・組 織・家 族・友 へ の 真 心。 だ が 同 時 に、 「不 義 な 主 君 を 諌 め る」 こ と も 忠 義 の 一 部 で あ る と さ れ た。
</p>

<h3>セ オ ド ア・ル ー ズ ベ ル ト と『武 士 道』</h3>

<p>
『武 士 道』 が 出 版 さ れ た 1900 年。 米 国 で は 副 大 統 領 だ っ た <strong>セ オ ド ア・ル ー ズ ベ ル ト</strong> (1858–1919) が こ の 本 を 読 ん で 感 銘 を 受 け た。
</p>

<p>
伝 え ら れ る と こ ろ で は、 ル ー ズ ベ ル ト は 『武 士 道』 を <strong>5 万 冊 購 入</strong> し、 子 ど も や 友 人、 政 治 家、 国 民 に 配 っ た と い う。
</p>

<p>
彼 は「ア メ リ カ 人 は こ の 本 か ら 学 ぶ べ き で あ る」 と 述 べ た と さ れ、 自 著『The Strenuous Life』 (1900) で も 武 士 道 的 倫 理 を 推 奨 し た。
</p>

<p>
1901 年、 ル ー ズ ベ ル ト は 大 統 領 に 就 任。 1904–05 年 の 日 露 戦 争 の 講 和 (ポ ー ツ マ ス 条 約) を 仲 介 し、 ノ ー ベ ル 平 和 賞 (1906) を 受 賞。<br>
彼 の 親 日 的 姿 勢 の 一 因 が、『武 士 道』 だ っ た と も 言 わ れ る。
</p>

<h3>新 渡 戸 と い う 国 際 人</h3>

<p>
新 渡 戸 稲 造 は、 単 な る『武 士 道』 の 著 者 で は な か っ た。
</p>

<ul>
<li>1862 ─ 岩 手 県 盛 岡 で 生 ま れ る</li>
<li>1881 ─ 札 幌 農 学 校 (現 北 海 道 大) 卒。 同 期 に 内 村 鑑 三 (本 サ イ ト chugakukomin3 既 出)</li>
<li>1884–1891 ─ 米 国 ジ ョ ン ズ ・ ホ プ キ ン ス 大 学・ド イ ツ 留 学。 ク エ ー カ ー 教 に 入 信 し、 ア メ リ カ 人 妻 (メ リ ー) と 結 婚</li>
<li>1891–1901 ─ 札 幌 農 学 校 教 授</li>
<li>1900 ─ 『武 士 道』 出 版</li>
<li>1903–1906 ─ 京 都 帝 大 法 学 部 教 授</li>
<li>1906–1913 ─ 第 一 高 等 学 校 (旧 一 高、 現 東 大 教 養 学 部) 校 長</li>
<li>1913–1918 ─ 東 京 帝 大 法 学 部 教 授</li>
<li>1920–1926 ─ 国 際 連 盟 事 務 次 長 (ジ ュ ネ ー ブ)</li>
<li>1933 ─ カ ナ ダ・バ ン フ で 病 没。 71 歳</li>
</ul>

<p>
新 渡 戸 は、 一 生 を 通 し て 「<strong>太 平 洋 の 架 け 橋 と な ら ん</strong>」 と い う 信 念 を 持 ち、 日 本 と 世 界 を つ な ぐ こ と に 尽 力 し た。
</p>

<h3>『武 士 道』 を 読 む 意 味 ─ 2026 年</h3>

<p>
126 年 前 に 書 か れ た『武 士 道』 を、 今 読 む 意 味 は あ る か?
</p>

<p>
「武 士」 は も う い な い。 「忠 義」 と い う 概 念 は 現 代 で は 揺 ら い で い る。 「名 誉」 を め ぐ る 価 値 観 も 変 化 し た。
</p>

<p>
だ が、 新 渡 戸 が 描 い た 七 つ の 徳 ─ 義・勇・仁・礼・誠・名 誉・忠 義 ─ そ の 多 く は、 形 を 変 え て 今 も 私 た ち の 中 に 生 き て い る。
</p>

<ul>
<li>「義」 ─ 不 正 を 見 過 ご さ な い 心</li>
<li>「勇」 ─ 正 し い と 思 う こ と を 言 う 勇 気</li>
<li>「仁」 ─ 困 っ て い る 人 を 助 け る 心</li>
<li>「礼」 ─ 他 者 を 尊 重 す る 態 度</li>
<li>「誠」 ─ 約 束 を 守 る、 嘘 を つ か な い</li>
</ul>

<p>
こ れ ら は、 「武 士 道」 と い う 形 で 名 前 が 付 い て い な く て も、 君 が 育 つ 過 程 で 親 ・ 先 生 ・ 友 人 か ら 受 け 取 っ て き た 価 値 観 で は な い だ ろ う か。
</p>

<h3>外 国 人 に「日 本 の 倫 理」 を 説 明 す る な ら</h3>

<p>
新 渡 戸 が「外 国 人 か ら の 質 問」 に 答 え る た め に 書 い た『武 士 道』。<br>
今 の 君 が、 同 じ 質 問 を 受 け た ら、 何 と 答 え る だ ろ う か?
</p>

<p>
「君 の 国 の 倫 理 は、 ど こ か ら 来 て い る の か?」<br>
「学 校 で 道 徳 教 育 を 受 け な い と し て も、 君 は 何 が 正 し い か を ど う 判 断 し て い る の か?」
</p>

<p>
こ う し た 問 い に 答 え る た め に、 新 渡 戸『武 士 道』 を 読 ん で み る こ と は、 今 で も 大 き な 意 味 が あ る。<br>
そ し て、 そ の 答 え は ─ 新 渡 戸 の 時 代 と は 違 う 形 で、 君 自 身 の 言 葉 で、 表 現 さ れ る べ き で あ る。
</p>

<p>
こ れ こ そ が、 道 徳 を「学 ぶ」 と い う こ と で あ る。
</p>
