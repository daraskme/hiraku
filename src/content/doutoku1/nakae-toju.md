<h3>1634 年 ─ 26 歳 の 武 士 が 故 郷 に 帰 る</h3>

<p>
寛 永 11 年 (1634)、 伊 予 国 大 洲 藩 (現 愛 媛 県 大 洲 市)。 一 人 の 26 歳 の 武 士 が、 主 君 に 致 仕 (辞 職)
を 願 い 出 ま し た。 名 前 は 中 江 与 右 衛 門 (後 の 藤 樹)。
</p>

<p>
理 由 は ─ 「故 郷 近 江 で 一 人 暮 ら す 母 親 の 老 後 が 心 配 で、 そ ば に 戻 り た い」。 武 士 と し て 出 世 の
道 が 開 け て い た に も か か わ ら ず、 母 へ の 孝 行 を 優 先 し た の で す。
</p>

<p>
故 郷 に 戻 っ た 藤 樹 は、 私 塾 を 開 き、 武 士 ・ 農 民 ・ 商 人 を 問 わ ず 教 え 始 め ま し た。 や が て
彼 は 「近 江 聖 人」 (お う み せ い じ ん) と 呼 ば れ る よ う に な り ま す。
</p>

<h3>な ぜ「聖 人」 と 呼 ば れ た の か</h3>

<p>
藤 樹 が「近 江 聖 人」 と 呼 ば れ た 理 由 は、 単 に 学 問 だ け で は あ り ま せ ん。 彼 の 生 き 方 そ の も の が、
人 々 の 心 を 動 か し た の で す。
</p>

<p>
逸 話 の 一 つ ─ あ る 馬 喰 (ば く ろ う ・ 馬 を 売 買 す る 商 人) が、 藤 樹 の 塾 の 前 で 落 し 物 を 拾 い ま し た。
中 に は 多 額 の 銭 が。 し か し 馬 喰 は、 そ れ を 自 分 の も の に せ ず、 落 し 主 を 探 し て 返 し ま し た。
</p>

<p>
落 し 主 が「謝 礼 を」 と 言 う と、 馬 喰 は「中 江 先 生 の 弟 子 で す か ら、 当 然 の こ と」 と 断 っ た。 ─
こ の 話 が 周 辺 に 広 ま り、 「藤 樹 の 教 え が、 庶 民 の 道 徳 を 変 え る」 と 評 判 に な っ た の で す。
</p>

<h3>『翁 問 答』 ─ 庶 民 で も 読 め る 道 徳 書</h3>

<p>
1641 年 (寛 永 18 年)、 藤 樹 33 歳 の 時、 『翁 問 答 (お き な も ん ど う)』 を 完 成 さ せ ま し た。
全 4 巻。 弟 子 や 庶 民 と の 問 答 形 式 で 書 か れ た 道 徳 書 で す。
</p>

<p>
特 徴 は ─ 漢 文 で な く、 仮 名 交 じ り の 平 易 な 日 本 語 で 書 か れ て い た こ と。 当 時 の 儒 学 書 は
漢 文 が 普 通 で、 武 士 ・ 学 者 し か 読 め ま せ ん で し た。 藤 樹 は、 「農 民 ・ 商 人 ・ 女 性 ・ 子 ど も で も
読 め る 道 徳 書」 を 目 指 し ま し た。
</p>

<h3>『翁 問 答』 の 中 心 ─ 「孝」 と「敬」</h3>

<p>
『翁 問 答』 で 藤 樹 が 説 い た 中 心 思 想 は、 「孝」 (こ う) と「敬」 (け い) で す。
</p>

<p><strong>孝</strong> ─ 親 ・ 先 祖 ・ 目 上 の 人 へ の 敬 愛。 単 に「親 に 従 う」 で は な く、 「親 を
心 か ら 敬 い、 そ の 願 い を 受 け 継 ぐ」 こ と。 藤 樹 が 致 仕 し て 母 と 暮 ら し た 行 為 は、 ま さ に「孝」
の 実 践 で し た。
</p>

<p><strong>敬</strong> ─ 心 を 慎 み、 自 己 を 律 す る こ と。 一 つ の こ と に 集 中 し、 散 漫 に な ら な い
状 態 (「主 一 無 適」)。 「孝」 が 外 (他 者) へ の 倫 理 な ら、 「敬」 は 内 (自 分) へ の 倫 理 で す。
</p>

<h3>日 本 陽 明 学 の 祖 ─ 知 行 合 一</h3>

<p>
藤 樹 は、 中 国 明 代 の 王 陽 明 (1472–1529) が 唱 え た 陽 明 学 を、 日 本 に 紹 介 し ま し た。 中 心 思 想 は
「知 行 合 一 (ち こ う ご う い つ)」 ─ 「知 識 と 行 動 を 一 致 さ せ る」 こ と。
</p>

<p>
当 時 主 流 だ っ た 朱 子 学 (し ゅ し が く) が「ま ず 学 び、 後 に 行 動」 と 考 え る の に 対 し、 陽 明 学 は
「学 ぶ こ と と 実 践 す る こ と は 同 時 で あ る ─ 知 る だ け で 行 動 し な い の は 真 の 知 識 で は な い」 と
主 張 し ま し た。
</p>

<p>
藤 樹 自 身 が「親 へ の 孝 を 知 っ た 」 か ら こ そ「致 仕 し て 母 と 暮 ら す」 行 動 を と っ た ─ こ れ が
知 行 合 一 の 実 践 例 で す。 後 に 大 塩 平 八 郎 (1837 大 阪 商 人 反 乱) や 吉 田 松 陰 (明 治 維 新 の 思 想 家)
ら が 陽 明 学 を 受 け 継 ぎ ま し た。
</p>

<h3>弟 子 ・ 熊 沢 蕃 山 ─ 政 治 へ の 応 用</h3>

<p>
藤 樹 の 私 塾「藤 樹 書 院」 に は、 多 く の 弟 子 が 集 ま り ま し た。 中 で も 最 も 著 名 な の が、
熊 沢 蕃 山 (く ま ざ わ ば ん ざ ん、 1619–1691) で す。
</p>

<p>
蕃 山 は、 岡 山 藩 主・池 田 光 政 (い け だ み つ ま さ) に 仕 え、 藤 樹 の 思 想 を 政 治 に 応 用 し ま し た。
「民 を 大 切 に す る 政 治」 を 主 張 し、 治 水 ・ 教 育 ・ 飢 饉 救 済 等 で 業 績 を 上 げ ま し た。 後 に 幕 府
か ら 危 険 視 さ れ 蟄 居 と な り ま し た が、 「日 本 陽 明 学 の 二 大 巨 匠」 と し て、 藤 樹 と 並 ぶ 名 声 を
得 ま し た。
</p>

<h3>40 歳 で の 早 世 ─ そ し て 影 響 力 の 永 続</h3>

<p>
中 江 藤 樹 は 1648 年 (慶 安 元 年)、 40 歳 で 病 没 し ま し た。 短 い 生 涯 で し た が、 そ の 思 想 と
人 格 は、 江 戸 期 ~ 現 代 ま で 350 年 以 上 受 け 継 が れ て い ま す。
</p>

<p>
故 郷 滋 賀 県 高 島 市 安 曇 川 町 に は、 藤 樹 を 祀 る 「藤 樹 書 院 跡」 と 「中 江 藤 樹 記 念 館」 が あ り、
今 も 多 く の 人 が 訪 れ ま す。
</p>

<h3>結 び ─ 「孝」 と「敬」 を 君 の 生 活 へ</h3>

<p>
中 江 藤 樹 が 教 え る の は、 高 度 な 学 問 で は な く、 「家 族 を 大 切 に、 自 分 を 律 す る」 と い う
基 本 的 な 倫 理 で す。
</p>

<p>
「孝」 ─ 親 ・ 祖 父 母 ・ 兄 弟 を 大 切 に す る。「敬」 ─ ス マ ホ や ゲ ー ム で 心 が 散 漫 に な ら な い よ う、
一 つ の こ と に 集 中 す る。 こ の 二 つ を、 今 日 か ら 始 め て み ま し ょ う。
</p>

<p>
近 江 聖 人 中 江 藤 樹 が、 380 年 前 か ら 君 に 送 る メ ッ セ ー ジ で す。
</p>
