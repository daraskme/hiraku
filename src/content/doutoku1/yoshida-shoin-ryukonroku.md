## 留 魂 録

吉 田 松 陰 (1830–1859)

### 一 (辞 世 の 歌)

身 は た と ひ 武 蔵 の 野 辺 に 朽 ち ぬ と も
留 め 置 か ま し 大 和 魂

### 二 (門 弟 へ の 言)

私 は 三 十 歳 を 一 つ の 区 切 り と し て、 こ こ で 死 ぬ。 し か し、 こ の 死 を 嘆 か な い で ほ し い。 死 と は、 私 に と っ て は、 春 夏 秋 冬 の 四 季 が ひ と め ぐ り す る よ う な も の だ。

人 に は そ れ ぞ れ の 一 生 が あ る。 十 歳 で 死 ぬ 者 に は 十 歳 な り の 春 夏 秋 冬 が あ り、 二 十 歳 で 死 ぬ 者 に は 二 十 歳 な り の 四 季 が あ る。 三 十 歳 で 死 ぬ 私 に も、 三 十 歳 な り の 四 季 が 既 に 過 ぎ た。

百 歳 を 生 き る 者 が ま こ と の 「長 寿」 で あ る か と い う と、 そ う で は な い。 百 歳 で あ っ て も、 そ の 中 身 が 空 し け れ ば、 そ れ は 百 歳 の 短 命 で あ る。 三 十 歳 で あ っ て も、 そ の 中 身 が 充 ち て い れ ば、 そ れ は 三 十 歳 の 長 寿 で あ る。

### 三 (至 誠 を 信 ぜ よ)

私 の 一 生 は、 至 誠 を 貫 こ う と し た 一 生 で あ っ た。 「至 誠 に し て 動 か ざ る も の は あ ら ず」 ─ こ の 一 句 を、 諸 君 は 心 に 留 め て お い て ほ し い。

真 心 を 込 め て や れ ば、 動 か な い も の は な い。 こ の 確 信 が、 私 を こ こ ま で 連 れ て き た。

世 間 に は、 才 知 で 物 事 を 動 か そ う と す る 者 が 多 い。 し か し、 真 心 を 込 め ず に 出 た 言 葉 や 行 動 は、 ど ん な に 巧 妙 に 見 え て も、 結 局 は 人 を 動 か さ な い。 諸 君 は、 才 知 よ り も、 ま ず 至 誠 を 磨 い て く れ。

### 四 (春 夏 秋 冬 の 譬)

百 姓 が 米 を 作 る と き、 春 に 種 を ま き、 夏 に 育 て、 秋 に 刈 り 入 れ、 冬 に 蓄 え る。 そ の 一 年 の 終 わ り に、 米 と い う 実 り が 残 る。 翌 年 に は ま た 新 し い 種 が ま か れ る。

私 の 三 十 年 の 一 生 も、 そ の よ う な も の だ。 種 を ま き、 育 て、 こ こ で 一 年 の 終 わ り を 迎 え る。 し か し、 諸 君 が、 私 の ま い た 種 を 引 き 継 ぎ、 諸 君 自 身 の 春 夏 秋 冬 を 過 ご す な ら、 私 の 一 生 は 終 わ ら な い。

### 五 (親 を 思 う 歌)

親 思 ふ 心 に 勝 る 親 心
け ふ の 音 づ れ 何 と 聞 く ら ん

(子 が 親 を 思 う 心 よ り も、 親 が 子 を 思 う 心 の ほ う が ま さ る だ ろ う。 今 日、 私 の 処 刑 の 知 ら せ を、 親 は ど う 聞 く だ ろ う か。)

### 六 (結 び)

諸 君、 元 気 で あ れ。

至 誠 を 貫 け。 国 の た め、 人 の た め、 そ し て 自 分 の た め に。 こ の 三 つ は、 至 誠 の 中 で 一 つ に な る。

私 は こ れ よ り 武 蔵 の 野 辺 に 旅 立 つ が、 私 の 大 和 魂 は、 諸 君 と 共 に あ る。 諸 君 が 至 誠 を 失 わ な い 限 り、 私 は 死 な な い。

***

(注：本 文 は『留 魂 録』 全 16 節 の 中 か ら、 中 学 道 徳 教 材 と し て 編 集 部 が 抜 粋 し、 現 代 語 に や わ ら か く 直 し た も の で あ る。 原 文 は 漢 文 訓 読 体 で 書 か れ て お り、 国 立 国 会 図 書 館 デ ジ タ ル コ レ ク シ ョ ン 等 で 閲 覧 で き る。)
