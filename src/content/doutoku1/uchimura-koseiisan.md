<h3>1894 年 ・ 箱 根 ─ 33 歳 の 内 村 が 学 生 に 語 る</h3>

<p>
明 治 27 年 (1894) 7 月、 箱 根 の 山 上 で「キ リ ス ト 教 夏 期 学 校」 が 開 か れ た。
集 ま っ た の は 全 国 か ら 来 た 若 い キ リ ス ト 教 徒 ・ 学 生 た ち。
</p>

<p>
講 師 の 一 人 が 内 村 鑑 三 (う ち む ら か ん ぞ う) ─ 当 時 33 歳。 札 幌 農 学 校 で
キ リ ス ト 教 を 学 び、 米 国 留 学 を 経 て、 ジ ャ ー ナ リ ス ト ・ 教 員 と し て 活 動 し て い た。
</p>

<p>
内 村 は 学 生 た ち に 問 い か け た ─ 「我 々 は、 後 世 に 何 を 残 し て 死 ぬ か」。
</p>

<h3>第 一 の 遺 物 ─ 金 銭</h3>

<p>
内 村 は、 後 世 に 残 せ る も の の 候 補 を 順 に 挙 げ て い く。 第 一 は ─ 金 銭 で あ る。
</p>

<p>
「お 金 を 残 す こ と は、 決 し て 卑 し い こ と で は な い」 と 内 村 は 言 う。 ア メ リ カ で 見 た
カ ー ネ ギ ー ら の 慈 善 事 業 ─ お 金 を 公 共 図 書 館 や 大 学 に 寄 付 す る ─ を 例 に 出 し、
「賢 く 集 め、 賢 く 散 ら す」 こ と が で き る 人 は 後 世 に 大 き く 貢 献 で き る。
</p>

<p>
し か し、 内 村 は こ う 続 け る ─ 「お 金 は 誰 で も 残 せ る わ け で は な い」。
</p>

<h3>第 二 の 遺 物 ─ 事 業</h3>

<p>
第 二 は、 事 業 ─ 学 校、 病 院、 鉄 道、 工 場 な ど ─ で あ る。 福 沢 諭 吉 が 慶 應 義 塾 を、
新 島 襄 が 同 志 社 を 設 立 し た よ う に、 後 世 の 人 が 利 用 で き る「実 体」 を 残 す こ と。
</p>

<p>
し か し、 こ れ も 才 能 と 機 会 が 必 要 で あ り、 「誰 も が 残 せ る も の」 で は な い。
</p>

<h3>第 三 の 遺 物 ─ 思 想</h3>

<p>
第 三 は、 思 想・著 作 で あ る。 福 沢 諭 吉 の『学 問 の す ゝ め』、 新 渡 戸 稲 造 の『武 士 道』、
シ ェ ー ク ス ピ ア ・ プ ラ ト ン ・ 孔 子 ─ 思 想 は 数 千 年 残 る。
</p>

<p>
し か し、 こ れ も 知 的 才 能 が な け れ ば 残 せ な い。 「学 問 の な い 人 は、 思 想 を 残 せ な い の か」
─ 内 村 は 問 う。
</p>

<h3>第 四 の 遺 物 ─ 高 尚 な る 生 涯</h3>

<p>
そ し て、 内 村 は 結 論 す る。 ─ 「誰 も が 残 せ る 最 大 遺 物 は、 高 尚 な る 生 涯 で あ る」。
</p>

<p>
お 金 も、 事 業 も、 思 想 も 残 せ な い 人 は い る。 で も、 ど ん な 人 で も「高 尚 な る 生 涯」
─ 誠 実 に 生 き、 困 難 に 屈 せ ず、 人 の 為 を 思 う 一 生 ─ は 残 す こ と が で き る。
</p>

<p>
内 村 は 続 け る ─ 「君 達 が、 苦 し み に 屈 せ ず、 真 心 を 持 っ て 生 き 抜 い た こ と
そ の も の が、 後 世 へ の 最 大 の 贈 り 物 に な る」。
</p>

<h3>道 徳 編 の 視 点 ─ 公 民 編 と の 違 い</h3>

<p>
本 サ イ ト の 公 民 編 (chugakukomin3) で は、 内 村 の 思 想 を 明 治 期 の 公 民 像 と し て 学 ん だ。
道 徳 編 の 今 回 は、 「自 分 の 一 生 を ど う 生 き る か」 と い う 個 人 の 倫 理 と し て 読 む。
</p>

<p>
公 民 (社 会 の 一 員 と し て の 自 分) と 道 徳 (個 人 と し て の 自 分) ─ 両 方 の 視 点 で 読 む と、
内 村 の 言 葉 が 二 重 に 深 ま る。
</p>

<h3>結 び ─ 君 の「最 大 遺 物」 を 探 す</h3>

<p>
13 歳 ・ 14 歳 の 今、 「後 世 に 何 を 残 す か」 を 考 え る の は 早 い と 思 う か も し れ な い。
し か し、 内 村 は こ う 言 う ─ 「人 生 の 大 計 は、 若 い 時 に 立 て よ」。
</p>

<p>
今 日 ・ 明 日 ・ こ の 一 年 ─ 君 が「高 尚 な る 一 日 を 生 き る」 こ と が、 既 に 後 世 へ の 遺 物 で あ る。
始 め て み よ う、 今 こ こ で。
</p>
