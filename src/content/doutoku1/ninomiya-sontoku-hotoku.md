<h3>14 歳 の 少 年 ─ 没 落 し た 一 家</h3>

<p>
天 明 7 年 (1787)、 相 模 国 足 柄 上 郡 栢 山 村 (現 神 奈 川 県 小 田 原 市) で、
二 宮 金 次 郎 (後 の 尊 徳) は 生 ま れ た。
</p>

<p>
裕 福 な 農 家 だ っ た が、 寛 政 3 年 (1791) の 大 雨 で 二 宮 家 の 田 畑 は 流 さ れ た。
父 利 右 衛 門 は 借 金 を 抱 え、 享 和 元 年 (1801)、 金 次 郎 14 歳 の 時 に 病 没。
2 年 後、 母 も 後 を 追 う よ う に 亡 く な っ た。
</p>

<p>
一 家 は 離 散。 金 次 郎 は 伯 父 の 家 に 引 き 取 ら れ た が、 弟 達 は バ ラ バ ラ。
家 は 没 落 し、 田 畑 も 失 っ た。
</p>

<h3>夜 道 を 歩 き な が ら 学 ぶ</h3>

<p>
伯 父 の 家 で、 金 次 郎 は 朝 か ら 晩 ま で 働 い た。 し か し、 夜 に 本 を 読 む と 「油 が 勿 体 な い」
と 叱 ら れ た。
</p>

<p>
そ こ で 金 次 郎 は、 自 分 で 菜 種 を 育 て、 自 分 の 油 で 灯 火 を 灯 し て 読 書 し た と い う。
ま た、 山 か ら 薪 を 運 ぶ 道 中、 歩 き な が ら 本 を 読 ん だ ─ こ れ が 後 の 「二 宮 金 次 郎 像」
の 原 型 で あ る。
</p>

<h3>24 歳 で 家 を 再 興</h3>

<p>
金 次 郎 は 少 し ず つ 荒 地 を 開 墾。 捨 て ら れ た 稲 を 拾 い、 谷 川 端 の 空 地 に 植 え る ─ 翌 年
一 俵 の 米 を 得 る。 「積 小 為 大」 の 原 点 だ。
</p>

<p>
文 化 9 年 (1812)、 24 歳。 つ い に 父 の 借 金 を 返 し、 田 畑 を 取 り 戻 し、 兄 弟 を 集 め て
二 宮 家 を 再 興 し た。
</p>

<h3>600 余 村 を 救 う ─ 報 徳 仕 法</h3>

<p>
小 田 原 藩 主・大 久 保 忠 真 が 金 次 郎 の 才 を 認 め、 文 政 5 年 (1822) に 桜 町 領 (現 栃 木 県
真 岡 市) の 復 興 を 任 せ た。 金 次 郎 35 歳。
</p>

<p>
桜 町 領 は 飢 饉 と 怠 惰 で 衰 退 し て い た が、 金 次 郎 は 約 10 年 で 復 興 さ せ た。 後 に
小 田 原 藩・烏 山 藩・下 館 藩・相 馬 藩 な ど 諸 大 名 に 招 か れ、 生 涯 で 600 余 の 村 を
復 興 し た。 こ の 復 興 法 を 「報 徳 仕 法 (ほ う と く し ほ う)」 と 呼 ぶ。
</p>

<h3>四 徳 ─ 至 誠・勤 労・分 度・推 譲</h3>

<p>
報 徳 思 想 の 中 心 は 「四 徳」 ─
</p>

<ul>
  <li><strong>至 誠 (し せ い)</strong> ─ 誠 実 で あ る こ と。 す べ て の 行 動 の 根 本。</li>
  <li><strong>勤 労 (き ん ろ う)</strong> ─ 怠 け ず 働 く こ と。 額 に 汗 し て 生 き る。</li>
  <li><strong>分 度 (ぶ ん ど)</strong> ─ 身 の 丈 に 合 っ た 生 活 を す る こ と。</li>
  <li><strong>推 譲 (す い じ ょ う)</strong> ─ 自 分 の 余 力 を 他 者 ・ 未 来 に 譲 る こ と。</li>
</ul>

<p>
こ の 四 つ が 揃 え ば、 個 人 も 村 も 国 も 復 興 で き る ─ こ れ が 尊 徳 の 確 信 だ っ た。
</p>

<h3>明 治 ・ 戦 後 ・ 現 代 ─ 受 け 継 が れ る 思 想</h3>

<p>
明 治 政 府 は 二 宮 尊 徳 を 修 身 (戦 前 の 道 徳) の 教 材 と し て 採 用。 全 国 の 小 学 校 に
「金 次 郎 像」 が 立 て ら れ、 子 ど も た ち の 模 範 と さ れ た。
</p>

<p>
戦 後、 「軍 国 主 義 と 結 び つ い た」 と し て 像 が 撤 去 さ れ た 時 期 も あ っ た が、 報 徳 思 想 は
ト ヨ タ ・ パ ナ ソ ニ ッ ク ・ 渋 沢 栄 一 ら 日 本 の 経 営 思 想 に 深 く 受 け 継 が れ て い る。
</p>

<p>
現 代 で は 「循 環 型 社 会」「ESG 経 営」「持 続 可 能 性 (SDGs)」 と い う 言 葉 で 注 目 さ れ る
価 値 観 ─ 「分 度 と 推 譲」 ─ を、 尊 徳 は 200 年 前 か ら 説 い て い た。
</p>

<h3>結 び ─ 「積 小 為 大」 を 君 の 言 葉 に</h3>

<p>
「小 さ な こ と を 積 み 重 ね れ ば、 大 き な こ と が で き る」 ─ こ の 言 葉 は 単 純 だ が、
実 行 す る の は 難 し い。
</p>

<p>
毎 日 10 ペ ー ジ 読 む。 毎 日 30 分 練 習 す る。 毎 日 一 つ 良 い 事 を す る。
─ 君 の「積 小 為 大」 計 画 を、 今 日 か ら 始 め て み よ う。
</p>
