## 南 洲 翁 遺 訓 ─ 西 郷 隆 盛 の 教 え

### 1. 1869 年、 鹿 児 島 へ の 旅

1869 年 (明 治 2 年)、 戊 辰 戦 争 が 終 わ っ た 翌 年 の 春。 山 形 県 鶴 岡 か ら、 旧 庄 内 藩 士 数 名 が 鹿 児 島 へ の 長 い 旅 に 出 た。

目 的 は 一 つ ─ 戊 辰 戦 争 で 自 ら が 敵 と し て 戦 っ た 男、 西 郷 隆 盛 (当 時 41 歳) に 直 接 会 い、 教 え を 聞 く こ と で あ っ た。

***

### 2. 「敵 が 弟 子 と な る」 と い う 逆 説

な ぜ 庄 内 藩 士 が、 か つ て の 敵 で あ る 西 郷 に 教 え を 求 め に 行 っ た の か。

1868 年 9 月、 庄 内 藩 は 奥 羽 越 列 藩 同 盟 の 一 員 と し て 新 政 府 軍 と 戦 い、 つ い に 降 伏 し た。 当 時 の 常 識 で は、 「敵 藩 は 領 地 没 収、 武 士 は 切 腹 か 流 罪」 が 当 然 で あ っ た。

し か し、 西 郷 は こ う 言 っ た。

「庄 内 藩 士 は、 立 派 な 武 士 と し て 戦 っ た。 敵 と し て 立 派 な 振 る 舞 い を し た 者 を、 私 ど も は 罰 す る こ と は で き な い。 領 地 は そ の ま ま、 武 士 の 名 誉 も そ の ま ま で あ る」

こ の 寛 大 な 処 遇 に、 庄 内 藩 士 は 衝 撃 を 受 け た。 「我 々 は 敵 と し て 戦 っ た。 そ れ で も 西 郷 翁 は 我 々 を 武 士 と し て 認 め た」。

そ こ で、 数 名 の 庄 内 藩 士 が「教 え を 聞 き た い」 と 鹿 児 島 を 訪 ね 始 め た の で あ る。

***

### 3. 西 郷 の 教 え ─ 41 か 条

庄 内 藩 士 は、 鹿 児 島 で、 そ し て そ の 後 西 郷 が 上 京 し た 際 に 東 京 で、 西 郷 に 会 っ て は 教 え を 聞 き、 そ れ を メ モ に 残 し た。

そ し て 1877 年 9 月、 西 南 戦 争 で 西 郷 が 城 山 で 自 刃。 そ の 13 年 後、 1890 年 (明 治 23 年)、 旧 庄 内 藩 士 (赤 沢 経 言・三 矢 藤 太 郎 等) が、 そ れ ら の メ モ を 整 理 し て 41 か 条 と し て 編 纂・出 版 し た。

『南 洲 翁 遺 訓 (な ん し ゅ う お う い く ん)』 で あ る。

***

### 4. 第 30 条 ─ 最 も 有 名 な 一 句

『南 洲 翁 遺 訓』 41 か 条 の 中 で 最 も 有 名 な の が、 第 30 条 で あ る。

「命 も い ら ず、 名 も い ら ず、 官 位 も 金 も い ら ぬ 人 は、 始 末 に 困 る も の な り。 然 (し か) し 斯 (か) か る 人 で な け れ ば、 艱 難 (か ん な ん) を 共 に し て 国 家 の 大 業 は 成 し 得 ら れ ぬ な り」

現 代 語 訳:「命 ・ 名 誉 ・ 地 位 ・ 金 銭 ─ ど れ も 欲 し が ら ぬ 人 は、 一 般 に は 扱 い に 困 る 人 で あ る。 し か し、 そ う い う 人 で な け れ ば、 困 難 を 共 に し て 国 の 大 仕 事 は 成 し 遂 げ ら れ な い」

こ の 一 句 が、 西 郷 隆 盛 自 身 の 生 涯 を 凝 縮 し て い る。 自 ら 命 を 賭 け て 維 新 を 成 し 遂 げ、 名 誉 や 官 位 を 求 め ず、 1877 年 西 南 戦 争 で「敗 れ る と 分 か っ て い な が ら」 弟 子 た ち と 共 に 戦 い、 城 山 で 自 刃 し た 西 郷。

「命 も い ら ず…」 と い う 言 葉 は、 西 郷 自 身 の 人 生 そ の も の の 描 写 で あ っ た。

***

### 5. 「敬 天 愛 人」 ─ 西 郷 の 座 右 銘

西 郷 の も う 一 つ の 有 名 な 言 葉 が、 「敬 天 愛 人 (け い て ん あ い じ ん、 天 を 敬 い、 人 を 愛 す)」 で あ る。

西 郷 は こ の 四 文 字 を 多 く の 揮 毫 (き ご う、 書) に 残 し て お り、 後 世「西 郷 を 表 す 言 葉」 と し て 知 ら れ る。

天 を 敬 う ─ 自 然 と 宇 宙 の 摂 理 に 従 う。 人 を 愛 す ─ 敵 で あ っ て も、 武 士 と し て 立 派 で あ れ ば 敬 う。 こ の 二 つ を 自 ら の 行 動 原 理 と し た。

***

### 6. 結 び

『南 洲 翁 遺 訓』 は、 41 か 条 の 短 い メ モ に 過 ぎ な い。 し か し そ の 中 に、 幕 末 か ら 明 治 へ の 革 命 を 駆 け 抜 け た 一 人 の 男 の 思 想 が、 凝 縮 さ れ て い る。

「命 も い ら ず、 名 も い ら ず、 官 位 も 金 も い ら ぬ 人」 と は 誰 の こ と か。 そ れ は、 こ の 教 え を 残 し た 西 郷 自 身 の こ と で あ り、 そ し て 同 時 に、 こ の 教 え を 読 ん で 自 ら そ う 生 き よ う と し た、 多 く の 後 世 の 人 々 の こ と で も あ る。

中 学 ・ 高 校 で「命 も い ら ず、 名 も い ら ず」 の 一 句 を 学 ぶ 君 が、 こ の 言 葉 を 自 分 の 心 の 中 で 反 芻 し て み る と き、 1869 年 の 鹿 児 島 で、 西 郷 と 向 か い 合 っ た 庄 内 藩 士 と 同 じ 体 験 を、 君 は し て い る の だ。
