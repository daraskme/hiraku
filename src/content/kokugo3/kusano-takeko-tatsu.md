<h3>げ ん だ い ご に な お す と</h3>

<p>
日 の 光 で 光 る 地 面 か ら、 竹 が 生 え て く る。<br>
若 く 青 い 竹 が 生 え て く る。
</p>

<p>
地 面 の 下 で は、 竹 の 根 が 生 え て い る。<br>
根 は だ ん だ ん 細 く 細 く な っ て い き、<br>
根 の 先 か ら 繊 毛 (せ ん も う、 細 い 毛) が 生 え て い る。<br>
か す か に 煙 (け む) る よ う に 繊 毛 が 生 え、<br>
か す か に 震 (ふ る) え て い る。
</p>

<p>
硬 (か た) い 地 面 か ら、 竹 が 生 え て く る。<br>
地 面 の 上 に 鋭 (す る ど) く 竹 が 生 え る。<br>
ま っ し ぐ ら に 竹 が 生 え る。<br>
凍 (こ お) っ た よ う に 引 き 締 ま っ た 節 (ふ し) を、 リ ン リ ン と 振 動 さ せ な が ら、<br>
青 空 の も と に 竹 が 生 え る。<br>
竹、 竹、 竹 が、 生 え る!
</p>

<h3>解 説 ─ 「生 え る」 と い う 動 詞 の 連 打</h3>

<p>
こ の 詩 で 最 も 印 象 的 な の は、 「<strong>生 え る</strong>」 と い う 動 詞 が 何 度 も 繰 り 返 さ れ る こ と で あ る。
</p>

<p>
12 行 の 中 で、 「生 え る」 「生 え」 が 合 計 約 10 回 出 て く る。<br>
こ の 連 打 が、 「<strong>竹 が ど ん ど ん 伸 び て い く 生 命 力</strong>」 を、 読 む 者 の 心 に 焼 き 付 け る。
</p>

<h3>地 下 か ら 空 へ ─ 縦 の 構 造</h3>

<p>
詩 は、 縦 方 向 の 視 線 を 描 く。
</p>

<ul>
<li><strong>地 上</strong> ─ 青 竹 が 生 え る</li>
<li><strong>地 下 1 段 目</strong> ─ 竹 の 根 が 生 え る</li>
<li><strong>地 下 2 段 目</strong> ─ 根 の 先 の 繊 毛 が 生 え る</li>
<li><strong>地 上 ・ 再 び</strong> ─ ま っ し ぐ ら に 竹 が 上 へ 伸 び る</li>
<li><strong>青 空 の も と</strong> ─ 竹 が 空 を 突 く</li>
</ul>

<p>
詩 人 の 眼 は、 地 下 の 細 か い 繊 毛 か ら、 地 上 の 鋭 い 青 竹、 そ し て 青 空 ま で を、 一 気 に 縦 に 貫 く。<br>
そ の 動 き が、 詩 全 体 に <strong>「上 へ、 上 へ」</strong> と い う 力 強 い 方 向 性 を 与 え る。
</p>

<h3>萩 原 朔 太 郎 と い う 詩 人</h3>

<p>
萩 原 朔 太 郎 (1886–1942) は、 群 馬 県 前 橋 出 身。<br>
彼 の 第 一 詩 集 『<strong>月 に 吠 え る</strong>』 (1917) は、 日 本 近 代 詩 の 出 発 点 と 言 わ れ る。
</p>

<p>
そ れ ま で の 日 本 の 詩 は、 文 語 体 (古 い 言 葉) で 書 か れ る こ と が 多 か っ た。<br>
朔 太 郎 は、 こ れ を 口 語 (話 し 言 葉) に 切 り 替 え、 自 由 な 形 で 詩 を 書 い た。
</p>

<p>
本 詩 「竹」 も、 口 語 と 古 風 な 語 (「か た き」「こ お れ る」 な ど) を 混 ぜ な が ら、 全 体 と し て 強 い リ ズ ム を 作 る。<br>
こ の 「<strong>新 し い 詩 の 言 葉</strong>」 を 確 立 し た こ と で、 朔 太 郎 は 「日 本 近 代 詩 の 父」 と 呼 ば れ る。
</p>

<h3>「し ん し ん と」 と い う 沈 黙 と 連 動</h3>

<p>
詩 全 体 に は、 「し ん し ん と」 と い う 沈 黙 を 感 じ さ せ る リ ズ ム が あ る。<br>
竹 は、 派 手 な 音 を 立 て て 伸 び る わ け で は な い。 だ が、 確 か に、 静 か に、 だ が 強 く 伸 び て い く。
</p>

<p>
こ の 「<strong>静 か な 強 さ</strong>」 が、 「し ん し ん と」 と い う リ ズ ム に 表 れ る。<br>
そ し て 詩 の 最 後、 「竹、 竹、 竹 が 生 え!」 で、 そ の 沈 黙 が 一 気 に 解 放 さ れ る。
</p>

<h3>君 も 自 然 物 を 詩 に し て み よ う</h3>

<p>
身 近 な 自 然 物 (花・木・草・石・水 ─ 何 で も) を 一 つ 選 ん で、 朔 太 郎 の よ う に 詩 に し て み よ う。
</p>

<ul>
<li>動 詞 を 何 度 も 繰 り 返 す (例 ─ 「咲 く、 咲 く、 咲 く」)</li>
<li>地 下 か ら 空 ま で の 縦 の 動 き を 描 く</li>
<li>オ ノ マ ト ペ を 一 つ 入 れ る</li>
</ul>

<p>
こ の 三 つ を 守 れ ば、 君 だ け の 「竹」 の よ う な 詩 が、 で き あ が る か も し れ な い。
</p>

<p>
朔 太 郎 が 100 年 前 に 教 え て く れ た 詩 の 力 ─ 動 詞 の 連 打、 縦 の 視 線、 オ ノ マ ト ペ の 響 き ─ を、 君 自 身 の 言 葉 で、 試 し て み よ う。
</p>
