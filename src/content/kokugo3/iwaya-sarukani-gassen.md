<h3>む か し む か し ─ 蟹 の 親 切 と 猿 の ず る さ</h3>

<p>
あ る 日、 蟹 (か に) が お む す び を 拾 い ま し た。 そ こ へ ず る い 猿 が や っ て き て、 こ う 言 い ま し た ─
「君 の お む す び と、 僕 の 柿 の 種 を 交 換 し よ う。 種 は 育 て れ ば た く さ ん の 柿 が な る よ!」
</p>

<p>
蟹 は 親 切 で す か ら、 お む す び を 猿 に 渡 し、 柿 の 種 を も ら い ま し た。 そ し て、 家 の 庭 に 種 を 植 え、
こ う 言 い な が ら 育 て ま し た ─ 「早 く 芽 を 出 せ 柿 の 種、 出 さ ぬ と 鋏 (は さ み) で 切 り き ざ む ぞ!」
</p>

<p>
や が て 柿 の 木 は 大 き く 育 ち、 真 っ 赤 な 柿 が た わ わ に な り ま し た。
</p>

<h3>ず る い 猿 が や っ て き た</h3>

<p>
そ こ へ 又、 あ の 猿 が や っ て き ま し た。 蟹 は 「猿 さ ん、 柿 を 取 っ て く だ さ い な」 と お 願 い し ま し た。
</p>

<p>
猿 は 木 に 登 っ て 柿 を 食 べ 始 め ま し た。 で も、 自 分 ば か り 美 味 し い 熟 し た 柿 を 食 べ、 蟹 に は
ま だ 青 く 硬 い 柿 を 投 げ ま し た。 そ し て、 そ の 青 柿 が 蟹 に 当 た っ て、 蟹 は 死 ん で し ま い ま し た。
</p>

<h3>子 蟹 と 仲 間 達 の 仇 討 ち</h3>

<p>
親 蟹 の 死 を 知 っ た 子 蟹 は、 仇 を 討 つ た め に 仲 間 を 集 め ま し た。 仲 間 は 4 人 (匹) ─
</p>

<ul>
  <li><strong>蜂 (は ち)</strong> ─ 飛 ん で 行 っ て、 針 で 刺 す</li>
  <li><strong>栗 (く り)</strong> ─ 囲 炉 裏 で 焼 け て 弾 け る</li>
  <li><strong>臼 (う す)</strong> ─ 棚 か ら 落 ち て、 重 く 押 し つ ぶ す</li>
  <li><strong>牛 糞 (う し の ふ ん)</strong> ─ 庭 に い て、 滑 ら せ る</li>
</ul>

<p>
そ れ ぞ れ が 猿 の 家 の 様 々 な 場 所 に 隠 れ て、 猿 が 帰 っ て く る の を 待 ち ま し た。
</p>

<h3>知 恵 と 協 力 の 仇 討 ち</h3>

<p>
夜、 猿 が 家 に 帰 っ て き ま し た。 寒 い の で 囲 炉 裏 で 火 に 当 た ろ う と し た ら ─
ぱ ち ん! 栗 が 弾 け て、 猿 の 顔 に 当 た り ま し た。
</p>

<p>
猿 は 痛 が っ て 水 を 飲 も う と 台 所 へ。 す る と 蜂 が 飛 ん で き て、 顔 を チ ク リ! と 刺 し ま し た。
</p>

<p>
慌 て て 外 へ 逃 げ よ う と し た 猿 は、 牛 糞 で つ る り! と 滑 っ て 転 び ま し た。 そ こ へ 棚 か ら 臼 が
ド ス ン と 落 ち て き て、 猿 を 押 し つ ぶ し ま し た。
</p>

<p>
こ う し て 子 蟹 と 仲 間 達 は、 知 恵 と 協 力 で 親 の 仇 を 討 ち ま し た。 そ し て み ん な、 仲 良 く
暮 ら し ま し た と さ。
</p>

<h3>物 語 が 教 え る こ と</h3>

<p>
こ の 昔 話 は、 三 つ の 大 切 な こ と を 教 え て く れ ま す ─
</p>

<ol>
  <li><strong>親 切 を 利 用 す る ず る さ は 悪 い</strong> ─ 猿 が 蟹 を 騙 し た こ と</li>
  <li><strong>協 力 す れ ば 強 い 相 手 に も 勝 て る</strong> ─ 子 蟹・蜂・栗・臼・牛 糞 が 一 緒 に な っ て 猿 を 倒 し た こ と</li>
  <li><strong>そ れ ぞ れ の 力 を 生 か す</strong> ─ 蜂 は 刺 す、 栗 は 弾 け る、 臼 は 重 い ─ 個 性 を 役 立 て る こ と</li>
</ol>

<h3>巌 谷 小 波 ─ 日 本 児 童 文 学 の 父</h3>

<p>
こ の『さ る か に 合 戦』 を 児 童 文 学 と し て 整 え た の は、 巌 谷 小 波 (い わ や さ ざ な み・1870–1933)
で す。 滋 賀 県 出 身 の 児 童 文 学 者 で、 「日 本 児 童 文 学 の 父」 と 呼 ば れ ま す。
</p>

<p>
1894 年 か ら 巌 谷 は『日 本 昔 噺』 (に っ ぽ ん む か し ば な し) 全 24 冊 を 出 版。 桃 太 郎・浦 島 太 郎・
カ チ カ チ 山・舌 切 雀・花 咲 か 爺・一 寸 法 師 等、 日 本 の 代 表 的 な 昔 話 を、 子 ど も が 読 め る 形 に 整 え
ま し た。
</p>

<h3>結 び ─ 君 の「仲 間 」 を 大 切 に</h3>

<p>
子 蟹 一 匹 で は 猿 に 勝 て ま せ ん で し た。 で も、 4 つ の 仲 間 が 力 を 合 わ せ て、 困 難 を 乗 り 越 え ま し た。
</p>

<p>
学 校・家・地 域 ─ 君 の 周 り に は、 い ろ ん な 仲 間 が い ま す。 一 人 で で き な い こ と も、 仲 間 と
力 を 合 わ せ れ ば で き る こ と が あ り ま す。 こ の 昔 話 を 心 に 留 め て、 君 も「自 分 の 仲 間」 を 大 切 に
し て く だ さ い。
</p>
