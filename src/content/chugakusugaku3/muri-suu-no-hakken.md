## √2 と 古 代 ギ リ シ ア の 危 機

### 1. ピ タ ゴ ラ ス 学 派 ─ 「万 物 は 数 で あ る」

紀 元 前 6 世 紀、 南 イ タ リ ア の ク ロ ト ン に、 ピ タ ゴ ラ ス と い う 数 学 者・哲 学 者 が 一 つ の 結 社 を 創 設 し た。 「ピ タ ゴ ラ ス 学 派」 で あ る。

学 派 の 中 心 教 義 は、 短 い 一 句 で 言 い 表 せ た。

「万 物 は 数 で あ る (Everything is number)」

そ し て、 こ こ で 言 う「数」 と は、 整 数 と そ の 比 (= 分 数) で あ っ た。 1, 2, 3, …、 そ し て 1/2, 3/4, 7/5 な ど。 こ れ ら の「美 し い 数」 だ け で、 世 界 の あ ら ゆ る も の が 表 現 で き る ─ こ う 信 じ ら れ て い た。

音 楽 の 和 音 も、 弦 の 長 さ の 比 (2:3 が 完 全 5 度、 3:4 が 完 全 4 度) で 表 さ れ た。 天 体 の 運 行 も、 数 の 比 で 表 さ れ る と 考 え ら れ た。 「数 の 調 和」 こ そ が 宇 宙 の 秩 序 で あ っ た。

***

### 2. 一 辺 1 の 正 方 形 の 中 の 謎

し か し、 学 派 の 内 部 で、 と て も 単 純 な 問 題 が 生 ま れ た。

「一 辺 が 1 の 正 方 形 の、 対 角 線 の 長 さ を 求 め よ」

ピ タ ゴ ラ ス の 定 理 (こ の 学 派 が 発 見 し た 定 理) を 使 え ば、 一 辺 1 の 正 方 形 の 対 角 線 の 長 さ x は、 1² + 1² = x²、 す な わ ち x² = 2、 つ ま り x = √2 で あ る。

問 題 は、 「√2 は、 整 数 の 比 (分 数) で 書 け る か?」 と い う 点 で あ っ た。

実 際 に 計 算 し て み る。 √2 ≒ 1.4142135... 1.41 を 約 分 で き る 分 数 で 試 す と… 7/5 = 1.4, 99/70 = 1.4142..., 707/500 = 1.414, …

ど う 試 し て も、 整 数 比 で 書 け な い。 で は、 こ れ は 単 に「ま だ 見 つ か っ て い な い」 だ け な の か。 そ れ と も、 そ も そ も「不 可 能」 な の か。

***

### 3. ヒ ッ パ ソ ス の 発 見 と 学 派 の 危 機

学 派 の 中 で、 一 人 の 若 い 数 学 者 が、 こ の 問 題 に 取 り 組 ん だ。 メ タ ポ ン テ ィ ウ ム 出 身 の ヒ ッ パ ソ ス (Hippasus) で あ る。

ヒ ッ パ ソ ス は、 背 理 法 と い う 古 代 ギ リ シ ア 独 自 の 論 理 で、 こ う 証 明 し た。

「も し √2 が 分 数 p/q (p と q は 互 い に 素 な 整 数) で 書 け る な ら ─

  2 = p² / q²、 し た が っ て p² = 2 q²

  し た が っ て p² は 偶 数。 し た が っ て p も 偶 数。 p = 2k と お く。

  (2k)² = 2 q² ⟹ 4 k² = 2 q² ⟹ q² = 2 k²

  し た が っ て q² も 偶 数。 し た が っ て q も 偶 数。

  し か し こ れ で は p と q が 共 に 偶 数、 つ ま り 共 通 因 数 2 を 持 つ。 こ れ は『p と q は 互 い に 素 で あ る』 と し た 出 発 点 の 仮 定 に 矛 盾 す る。

  ゆ え に、 √2 は 整 数 比 で は 書 け な い。」

つ ま り、 √2 は「無 理 数」 で あ っ た ─ ピ タ ゴ ラ ス 学 派 の「万 物 は 整 数 比 で あ る」 と い う 教 義 を、 内 部 か ら 崩 す 反 例 で あ っ た。

***

### 4. 海 へ の 追 放 の 伝 説

伝 説 に よ れ ば、 ヒ ッ パ ソ ス が こ の 発 見 を 学 派 の 外 に 公 表 し た と き、 学 派 は 激 怒 し た。 学 派 の 教 義 を 内 部 か ら 崩 す 事 実 を、 異 教 徒 (= 学 派 の 外 の 人 間) に 漏 ら し た 罪 は 重 か っ た。

あ る 伝 説 で は、 ヒ ッ パ ソ ス は 学 派 か ら 追 放 さ れ、 別 の 伝 説 で は、 海 で 溺 死 さ せ ら れ た と 伝 え ら れ る。 ヒ ッ パ ソ ス の 没 年 は 正 確 に は 分 か ら な い が、 学 派 が こ の 発 見 を 100 年 ほ ど 秘 匿 し た こ と は、 後 世 の 記 録 か ら 確 認 で き る。

数 学 が「人 の 信 念」 を 揺 る が し た、 数 少 な い 瞬 間 の 一 つ で あ る。

***

### 5. 「実 数」 と い う 概 念 の 誕 生

し か し、 真 実 は 隠 し 通 せ な か っ た。 100 年 後、 前 3 世 紀 の ア レ ク サ ン ド リ ア の ユ ー ク リ ッ ド は、『原 論』 第 10 巻 で「無 理 量 (無 理 数)」 を 体 系 的 に 扱 い、 √2 が 無 理 数 で あ る こ と の 証 明 を 数 学 史 に 永 遠 に 刻 ん だ。

そ し て 19 世 紀、 デ デ キ ン ト (1831-1916) と カ ン ト ー ル (1845-1918) が、 有 理 数 と 無 理 数 を 合 わ せ た「実 数 (real numbers)」 の 概 念 を 厳 密 に 定 義 し、 数 学 の 基 礎 を 完 成 さ せ た。 「実 数」 と は、 ヒ ッ パ ソ ス の 発 見 か ら 2400 年 を 経 て、 人 類 が 数 の 全 体 を 受 け 入 れ る に 至 っ た 概 念 で あ る。

中 学 3 年 で あ な た が 学 ぶ「√2」「平 方 根」「無 理 数」 と い う 言 葉 の 一 つ 一 つ は、 ヒ ッ パ ソ ス の 命 と 引 き 換 え に 数 学 史 に 加 わ っ た、 重 い 言 葉 な の で あ る。
