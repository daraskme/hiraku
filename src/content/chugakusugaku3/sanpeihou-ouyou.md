<h3>中 2 で 学 ん だ 三 平 方 を 復 習</h3>

<p>
中 2 で、 君 は 三 平 方 の 定 理 (ピ タ ゴ ラ ス の 定 理) を 学 ん だ。 直 角 三 角 形 の 三 辺 の 関 係 ─
</p>

<p style="text-align: center; font-size: 1.2em;">
<strong>a² + b² = c²</strong>
</p>

<p>
直 角 を 挟 む 2 辺 を a・b、 斜 辺 を c と す る。 こ の シ ン プ ル な 関 係 が、 古 代 か ら 現 代 ま で
あ ら ゆ る 数 学 ・ 物 理 ・ 工 学 の 基 礎 と な っ て き た。
</p>

<h3>応 用 1 ─ 直 方 体 の 対 角 線</h3>

<p>
中 3 の 新 し い 応 用 ─ 立 体 の 対 角 線 を 計 算 す る。
</p>

<p><strong>問 題</strong> ─ 縦 a・横 b・高 さ c の 直 方 体 の、 中 を 通 る 最 長 の 対 角 線 の 長 さ
を 求 め よ。</p>

<p><strong>解 き 方</strong> ─ 二 段 階 で 考 え る。</p>

<ol>
  <li>底 面 の 対 角 線: √(a² + b²) ─ 中 2 の 三 平 方</li>
  <li>こ れ と 高 さ c で 新 し い 直 角 三 角 形: √(a² + b² + c²)</li>
</ol>

<p>
つ ま り、 縦 a・横 b・高 さ c の 直 方 体 の 対 角 線 = √(a² + b² + c²)。
</p>

<p><strong>例</strong> ─ 縦 3・横 4・高 さ 12 の 直 方 体 ─
√(3² + 4² + 12²) = √(9 + 16 + 144) = √169 = 13。
</p>

<h3>応 用 2 ─ 円 に 内 接 す る 正 多 角 形</h3>

<p>
半 径 r の 円 に 内 接 す る 正 N 角 形 の 一 辺 の 長 さ を、 三 平 方 で 計 算 で き る。
</p>

<p><strong>正 六 角 形</strong> ─ 中 心 角 60 度 で、 各 三 角 形 は 正 三 角 形。 一 辺 = r。
周 = 6r。 円 周 ÷ 直 径 ≈ 6r ÷ 2r = 3 ─ 円 周 率 π の 大 ま か な 近 似 (3 < π)。
</p>

<p><strong>正 八 角 形</strong> ─ 三 平 方 で 計 算 す る と、 一 辺 = r √(2 − √2) ≈ 0.765r。
周 ≈ 6.12r。 π ≈ 3.06 ─ 正 六 角 形 よ り 精 度 向 上。
</p>

<p>
ア ル キ メ デ ス (紀 元 前 3 世 紀) は、 こ の 方 法 で 正 96 角 形 ま で 計 算 し て、 π ≈ 3.1418
ま で 求 め た。 江 戸 期 の 関 孝 和 ・ 建 部 賢 弘 も 同 様 の 方 法 で π を 計 算 し て い る
(本 サ イ ト chugakusugaku1/takebe-katahiro-mugenkyusu 参 照)。
</p>

<h3>応 用 3 ─ 江 戸 期 の 算 額 文 化</h3>

<p>
江 戸 期、 数 学 は 武 士 ・ 僧 侶 ・ 商 人 だ け で な く、 農 民 ・ 庶 民 の 楽 し み で も あ っ た。
人 々 は 神 社 や 仏 閣 に「算 額 (さ ん が く)」 と 呼 ば れ る 木 札 を 奉 納 し た。
</p>

<p>
そ こ に は、 美 し い 図 形 と 共 に 数 学 問 題 ・ 解 答 が 描 か れ て い た ─ 三 平 方 の 定 理 ・ 円 の 性 質 ・
連 立 方 程 式 等 を 駆 使 し た 高 度 な 問 題。 現 代 で も 全 国 約 800 面 の 算 額 が 残 る。
</p>

<p>
「数 学 を 解 い た こ と を 神 仏 に 感 謝 し、 奉 納 す る」 ─ こ の 信 仰 と 学 問 の 結 び つ き は、
日 本 独 特 の 数 学 文 化 で あ る。
</p>

<h3>応 用 4 ─ 野 球 の 一 三 塁 間 距 離</h3>

<p>
野 球 の 内 野 は 一 辺 90 フ ィ ー ト (27.43 m) の 正 方 形。 一 塁 か ら 三 塁 ま で の 直 線 距 離 は?
</p>

<p>
三 平 方 で 計 算 ─ 27.43² + 27.43² = c² よ り、 c = 27.43 × √2 ≈ 38.79 m。
</p>

<p>
プ ロ 野 球 の キ ャ ッ チ ャ ー が 一 塁 に 牽 制 球 を 投 げ る 時、 こ の 38.79 m を 一 瞬 で 投 げ る ─
身 体 能 力 が 三 平 方 の 定 理 で 表 現 さ れ る 美 し い 例 だ。
</p>

<h3>応 用 5 ─ G P S と 三 平 方</h3>

<p>
ス マ ホ の G P S は、 4 つ の 人 工 衛 星 か ら の 信 号 を 元 に、 君 の 位 置 を 三 平 方 で 計 算 す る。
</p>

<p>
各 衛 星 と の 距 離 ─ d₁ = √((x − x₁)² + (y − y₁)² + (z − z₁)²)。 こ れ を 4 つ 連 立 さ せ て
解 け ば、 (x, y, z) ─ 君 の 位 置 が 数 m の 精 度 で 求 ま る。
</p>

<p>
2600 年 前 の ピ タ ゴ ラ ス の 定 理 が、 ス マ ホ の 地 図 機 能 を 支 え て い る。
</p>

<h3>結 び ─ 一 つ の 定 理 が ど こ ま で 行 け る か</h3>

<p>
a² + b² = c² と い う 一 つ の 単 純 な 関 係 が、 江 戸 算 額 か ら G P S・宇 宙 物 理 学 ま で あ ら ゆ る
場 面 で 使 わ れ る。 数 学 の 美 し さ ・ 強 さ は、 こ の「一 般 性」 に あ る。
</p>

<p>
中 3 で 三 平 方 を 完 全 に マ ス タ ー し て、 君 の 数 学 の 道 具 箱 の 中 で も 最 重 要 の 一 つ と し て
活 用 し よ う。
</p>
