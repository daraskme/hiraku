<h3>1 辺 1 の 正 方 形 の 対 角 線</h3>

<p>
紙 に 1 辺 1 cm の 正 方 形 を 描 い て み よ う。<br>
そ し て、 対 角 線 (向 か い 合 う 頂 点 を 結 ぶ 線) の 長 さ を 定 規 で 測 っ て み よ う。
</p>

<p>
だ い た い <strong>1.4 cm</strong> く ら い に な る は ず だ。
</p>

<p>
ピ タ ゴ ラ ス の 定 理 (本 サ イ ト chugakusugaku2) を 使 う と:<br>
対 角 線² = 1² + 1² = 2<br>
対 角 線 = √2 (ル ー ト 2)
</p>

<p>
で は、 √2 と は ど ん な 数 か?
</p>

<h3>√2 の 小 数 展 開</h3>

<p>
電 卓 で 計 算 し て み る と:
</p>

<blockquote>
√2 = 1.41421356237309504880168872420969…
</blockquote>

<p>
こ の 小 数 は ─ <strong>永 遠 に 続 き、 し か も 循 環 し な い</strong>。<br>
「0.333…」 (= 1/3) の よ う に 「3 が 繰 り 返 す」 こ と も な い。<br>
ま っ た く 規 則 性 が な い 形 で、 桁 が 続 く。
</p>

<p>
こ う し た 数 を「<strong>無 理 数</strong>」 と 呼 ぶ。<br>
「無 理」 と は「分 数 で 表 せ な い」 と い う 意 で あ る (英 語 の irrational ─「比 で 表 せ な い」 の 訳)。
</p>

<h3>古 代 ギ リ シ ア で の 衝 撃</h3>

<p>
紀 元 前 6 世 紀、 古 代 ギ リ シ ア の <strong>ピ タ ゴ ラ ス 学 派</strong> は、 こ う 信 じ て い た:
</p>

<blockquote>
「<strong>万 物 は 数 で あ る</strong>」
</blockquote>

<p>
こ こ で 言 う 「数」 と は、 整 数 (1, 2, 3…) と 整 数 の 比 (1/2、 3/7…) ─ つ ま り「有 理 数」 だ け を 指 し て い た。
</p>

<p>
彼 ら に と っ て、 す べ て の 量 は「整 数 の 比」 で 表 せ る は ず だ っ た。<br>
だ か ら こ そ、 ピ タ ゴ ラ ス の 定 理 で 「1 辺 1 の 正 方 形 の 対 角 線 = √2」 と い う 計 算 結 果 が 出 た と き、 彼 ら は「こ の √2 を 何 ら か の 分 数 で 表 そ う」 と し た。
</p>

<p>
だ が、 ピ タ ゴ ラ ス の 弟 子 <strong>ヒ ッ パ ソ ス</strong> (Hippasus, 紀 元 前 5 世 紀) が ─ √2 は <strong>絶 対 に 分 数 で 表 せ な い</strong> こ と を、 厳 密 に 証 明 し て し ま っ た。
</p>

<h3>ヒ ッ パ ソ ス の 証 明 (背 理 法)</h3>

<p>
ヒ ッ パ ソ ス の 証 明 は、 数 学 史 上 最 古 の 「<strong>背 理 法</strong>」(は い り ほ う) の 一 つ で あ る。
</p>

<p>
証 明 の あ ら す じ:
</p>

<ol>
<li>仮 に、 √2 が 分 数 で 表 せ る と し よ う。 √2 = p/q (p と q は 整 数、 既 約 分 数)</li>
<li>両 辺 を 二 乗 す る と: 2 = p²/q²、 つ ま り 2q² = p²</li>
<li>こ れ よ り p² は 偶 数。 ゆ え に p も 偶 数 (奇 数 の 二 乗 は 奇 数 だ か ら)。 p = 2k と お く。</li>
<li>す る と 2q² = 4k²、 つ ま り q² = 2k²。 同 様 に q も 偶 数。</li>
<li>す る と p も q も 偶 数。 だ が p/q は 既 約 分 数 (約 分 で き な い 分 数) と し た。 矛 盾!</li>
<li>ゆ え に、 「√2 が 分 数 で 表 せ る」 と い う 仮 定 が 間 違 い だ っ た。 → √2 は 分 数 で 表 せ な い 「無 理 数」。</li>
</ol>

<p>
こ の 「<strong>仮 定 が 矛 盾 を 生 む か ら、 仮 定 は 間 違 い</strong>」 と い う 論 法 を、 数 学 で「背 理 法」 と 呼 ぶ。<br>
ヒ ッ パ ソ ス は、 こ の 美 し い 論 法 で、 「万 物 は 数 (有 理 数) で あ る」 と い う ピ タ ゴ ラ ス 学 派 の 世 界 観 を 覆 し た。
</p>

<h3>悲 劇 の 伝 説</h3>

<p>
ピ タ ゴ ラ ス 学 派 に と っ て、 「無 理 数 の 存 在」 は <strong>絶 対 に 公 表 し て は な ら な い 秘 密</strong> だ っ た。<br>
そ れ を 公 に す れ ば、 彼 ら の 信 念 が 崩 壊 し て し ま う。
</p>

<p>
伝 説 に よ れ ば ─ ヒ ッ パ ソ ス は こ の 秘 密 を 外 部 に 漏 ら し た た め、 怒 っ た 学 派 の メ ン バ ー に <strong>海 に 突 き 落 と さ れ て 殺 さ れ た</strong> と さ え 伝 え ら れ る (信 憑 性 は 不 明)。
</p>

<p>
真 偽 は と も か く、 こ の 伝 説 は、 「<strong>新 し い 数 学 的 真 理 が 既 存 の 世 界 観 を 揺 さ ぶ る 衝 撃</strong>」 を 物 語 っ て い る。
</p>

<h3>「数」 の 概 念 の 拡 張 史</h3>

<p>
無 理 数 の 発 見 か ら、 「数 」 の 概 念 は 何 度 も 拡 張 さ れ て き た:
</p>

<ol>
<li><strong>自 然 数</strong> (1, 2, 3…) ─ 古 代 か ら</li>
<li><strong>分 数</strong> (1/2、 3/7…) ─ 古 代 か ら</li>
<li><strong>0 (ゼ ロ)</strong> ─ イ ン ド で 5 世 紀 頃 発 見</li>
<li><strong>負 の 数</strong> (-1、 -3…) ─ 中 国・イ ン ド で 古 代、 ヨ ー ロ ッ パ で 15 世 紀 頃 受 容</li>
<li><strong>無 理 数</strong> (√2、 π…) ─ 古 代 ギ リ シ ア で 発 見、 衝 撃</li>
<li><strong>虚 数</strong> (√-1 = i…) ─ 16 世 紀 イ タ リ ア で 発 見</li>
<li><strong>複 素 数</strong> ─ 18 世 紀 オ イ ラ ー ・ ガ ウ ス が 体 系 化</li>
</ol>

<p>
こ れ ら の 拡 張 が、 そ れ ぞ れ の 時 代 で「<strong>不 自 然 だ、 認 め ら れ な い</strong>」 と 抵 抗 さ れ な が ら、 や が て 数 学 の 必 須 部 分 と な っ た。
</p>

<h3>江 戸 日 本 の 平 方 根</h3>

<p>
江 戸 の 関 孝 和 (本 サ イ ト chugakusugaku1) も、 平 方 根 の 計 算 法 を 独 自 に 発 達 さ せ た。
</p>

<p>
関 は 鋭 い 計 算 法 で、 √2、 √3、 √5 を 小 数 点 以 下 何 桁 ま で も 正 確 に 求 め た。<br>
鎖 国 の 日 本 で、 ヨ ー ロ ッ パ と 同 じ 数 学 が 並 行 に 発 達 し た。
</p>

<h3>身 の 回 り の √2</h3>

<p>
√2 ≈ 1.41421… は、 君 の 身 の 回 り に も 隠 れ て い る:
</p>

<ul>
<li><strong>A4 用 紙 の 縦・横 の 比</strong> ─ 1 : √2 (こ れ で 半 分 に 折 っ て も 同 じ 比 に な る)</li>
<li><strong>正 方 形 の 対 角 線 と 辺 の 比</strong> ─ √2 : 1</li>
<li><strong>音 楽 の 12 音 律 の 計 算</strong> ─ 12√2 を 使 う (1 オ ク タ ー ブ を 12 等 分)</li>
</ul>

<p>
2500 年 前 に ヒ ッ パ ソ ス が 命 を 賭 け て 発 見 し た「無 理 数 √2」 が、 今 の 君 の 周 り で、 静 か に 働 い て い る。
</p>

<p>
「数」 と は 何 か ─ こ の 問 い は、 古 代 か ら 続 く、 数 学 の 一 番 深 い 問 い で あ る。
</p>
