## 358 年 の 謎

### 1. 1637 年、 フ ェ ル マ ー の 余 白

時 は 1637 年、 フ ラ ン ス 南 西 部 ト ゥ ー ル ー ズ 高 等 法 院 の 評 議 員 を 務 め る 30 歳 の ピ エ ー ル・ド・フ ェ ル マ ー が、 自 宅 で 古 代 ギ リ シ ア の 数 学 者 デ ィ オ フ ァ ン ト ス『算 術』 を 読 ん で い た。

『算 術』 第 2 巻 第 8 問 で、 デ ィ オ フ ァ ン ト ス は「与 え ら れ た 平 方 数 を、 二 つ の 平 方 数 に 分 け よ」 と 問 い、 ピ タ ゴ ラ ス 数 (例 え ば 25 = 9 + 16、 つ ま り 5² = 3² + 4²) の 求 め 方 を 解 説 し て い た。

そ こ で フ ェ ル マ ー は ふ と 考 え た。

「平 方 数 (n = 2) で は、 こ う し て 二 つ の 平 方 数 に 分 け ら れ る (= ピ タ ゴ ラ ス 数 が 存 在 す る)。 で は、 立 方 数 (n = 3) を 二 つ の 立 方 数 に 分 け る こ と は で き る だ ろ う か? 4 乗 数 を 二 つ の 4 乗 数 に 分 け る こ と は?」

つ ま り、 「xⁿ + yⁿ = zⁿ」 を 満 た す 自 然 数 解 (x, y, z) は、 n = 3, 4, 5, … で も 存 在 す る か。

フ ェ ル マ ー は、 こ の 問 題 に つ い て 何 日 も 考 え た 末、 『算 術』 の 余 白 に こ う 書 き 残 し た。

「n ≥ 3 の と き、 xⁿ + yⁿ = zⁿ を 満 た す 自 然 数 (x, y, z) は 存 在 し な い。 私 は こ の 命 題 に 対 し て、 真 に 驚 嘆 す べ き 証 明 を 発 見 し た が、 こ の 余 白 は 狭 す ぎ て 書 き 切 れ な い。」

***

### 2. 「書 き 切 れ な い」 の 余 白 の 謎

フ ェ ル マ ー の メ モ は、 そ こ で 終 わ っ た。 「証 明」 は 一 度 も 書 か れ ず、 1665 年 フ ェ ル マ ー は 没 し た。

息 子 サ ミ ュ エ ル が 父 の メ モ を 整 理 し て 1670 年 出 版 し た こ と で、 こ の「フ ェ ル マ ー の 最 終 定 理」 が 数 学 界 に 公 表 さ れ た。

そ し て、 358 年 間 続 く 数 学 者 の 闘 い が 始 ま っ た。

「フ ェ ル マ ー は 本 当 に 証 明 を 知 っ て い た の か?」「彼 の 言 う『驚 嘆 す べ き 証 明』 と は ど ん な も の だ っ た の か?」「そ も そ も こ の 命 題 は 正 し い の か?」

***

### 3. 358 年 の 部 分 解 法 の リ レ ー

1670 年 以 来、 多 く の 数 学 者 が 部 分 的 な 結 果 を 残 し た。

- **オ イ ラ ー (1707-1783)** ─ n = 3 の 場 合 を 証 明
- **ル ジ ャ ン ド ル (1752-1833)、 ジ ェ ル メ ー ヌ (1776-1831)** ─ n = 5 の 場 合 を 証 明
- **ラ メ (1795-1870)** ─ n = 7 の 場 合 を 証 明
- **ク ン マ ー (1810-1893)** ─「正 則 素 数」 の 場 合 を 証 明 (大 部 分 の 素 数 を カ バ ー)

し か し、 「全 て の n に 対 し て」 と い う 完 全 な 証 明 は、 こ の 個 別 解 法 の リ レ ー で は 到 達 で き な か っ た。 数 学 界 は こ の 命 題 を「フ ェ ル マ ー の 最 終 定 理 (Fermat's Last Theorem)」 と 呼 び、 「定 理 と 名 乗 る に は ま だ 証 明 さ れ て い な い、 し か し『定 理』 と 呼 ば ざ る を 得 な い 命 題」 と し て 200 年 以 上 残 し た。

***

### 4. 日 本 人 数 学 者 の 影 の 貢 献 ─ 谷 山-志 村 予 想

20 世 紀 半 ば、 一 つ の 全 く 別 の 数 学 領 域 で 重 要 な 動 き が あ っ た。

1955 年、 日 本 人 数 学 者 谷 山 豊 (た に や ま ゆ た か、 1927-1958) と 志 村 五 郎 (し む ら ご ろ う、 1930-2019) が、 「楕 円 曲 線」 と「モ ジ ュ ラ ー 形 式」 と い う 二 つ の 数 学 的 対 象 が 深 く 結 び つ い て い る と い う 予 想 を 提 起 し た。 こ れ を「谷 山-志 村 予 想」 と 呼 ぶ (谷 山 は 1958 年 31 歳 で 自 死、 志 村 が 後 を 引 き 継 ぐ)。

そ し て 1985 年、 ド イ ツ の 数 学 者 ゲ ル ハ ル ト・フ ラ イ (1944-) が、 「フ ェ ル マ ー の 最 終 定 理 の 反 例 (xⁿ + yⁿ = zⁿ を 満 た す 自 然 数 解) が 存 在 す る な ら、 そ こ か ら 谷 山-志 村 予 想 に 反 す る『お か し な 楕 円 曲 線』 が 作 れ る」 と 指 摘。 こ れ を 1986 年 ケ ン・リ ベ ッ ト が 厳 密 に 証 明 し た。

つ ま り、 「谷 山-志 村 予 想 を 証 明 で き れ ば、 フ ェ ル マ ー の 最 終 定 理 も 自 動 的 に 解 け る」 と い う 道 が 開 け た。

日 本 人 数 学 者 の 仕 事 が、 フ ェ ル マ ー の 最 終 定 理 の 鍵 と な っ た 瞬 間 で あ る。

***

### 5. ワ イ ル ズ の 7 年 間

イ ギ リ ス の 数 学 者 ア ン ド リ ュ ー・ワ イ ル ズ (1953-) は、 10 歳 の と き ケ ン ブ リ ッ ジ の 図 書 館 で フ ェ ル マ ー の 最 終 定 理 を 知 り、 「い つ か 解 こ う」 と 決 意 し た と 後 に 語 っ て い る。

1986 年、 33 歳 の ワ イ ル ズ (プ リ ン ス ト ン 大 学 教 授) は、 リ ベ ッ ト の 結 果 を 知 っ て、 谷 山-志 村 予 想 (の 半 安 定 楕 円 曲 線 の 場 合) を 証 明 す る こ と に 取 り 組 む 決 意 を し た。

そ し て、 7 年 間。 ワ イ ル ズ は ほ ぼ 完 全 に 沈 黙 し、 プ リ ン ス ト ン の 自 宅 の 屋 根 裏 部 屋 に 籠 り、 単 独 で 研 究 を 続 け た。 周 囲 に は「別 の 普 通 の 数 学 を し て い る」 と 装 っ て い た が、 実 は 全 て の 時 間 を フ ェ ル マ ー の 最 終 定 理 に 注 い で い た。

1993 年 6 月、 ケ ン ブ リ ッ ジ で の 講 演 で、 ワ イ ル ズ は 一 度「証 明 完 成」 を 発 表 し た。 数 学 界 が 沸 い た。 し か し 数 ヶ 月 後、 査 読 の 過 程 で 重 大 な「ギ ャ ッ プ」 が 発 見 さ れ た。

ワ イ ル ズ は 絶 望 し か け た が、 弟 子 の リ チ ャ ー ド・テ イ ラ ー と 共 に 1 年 半 か け て 修 正 を 続 け た。 1994 年 9 月 19 日、 屋 根 裏 部 屋 で、 つ い に「修 正 が 完 成 し た 瞬 間」 を 迎 え た。 そ の 瞬 間 を ワ イ ル ズ は 後 に こ う 語 っ て い る。

「全 て が ぴ た り と は ま っ た。 私 は 部 屋 を 行 っ た り 来 た り し て、 数 分 間 そ れ を 見 つ め て い た。 そ し て、 涙 が 出 た。」

### 6. 1995 年 ─ 358 年 ぶ り の「定 理」

1995 年 5 月、 ワ イ ル ズ と テ イ ラ ー の 共 著 論 文「半 安 定 楕 円 曲 線 の モ ジ ュ ラ ー 性」 が、 学 術 誌『Annals of Mathematics』 に 掲 載 さ れ た。 本 文 109 ペ ー ジ。

こ の 論 文 で、 谷 山-志 村 予 想 の 半 安 定 楕 円 曲 線 の 場 合 が 証 明 さ れ、 そ の 帰 結 と し て、 358 年 間「定 理」 と 呼 ば れ つ つ 証 明 さ れ て い な か っ た「フ ェ ル マ ー の 最 終 定 理」 が、 つ い に 本 当 の「定 理」 と な っ た。

1995 年、 ワ イ ル ズ 42 歳。 1637 年 フ ェ ル マ ー が 30 歳 で 余 白 に 書 き 残 し た 一 句 の、 358 年 後 で あ っ た。

### 7. 余 白 の 詩

「真 に 驚 嘆 す べ き 証 明 を 発 見 し た が、 こ の 余 白 は 狭 す ぎ て 書 き 切 れ な い」

こ の フ ェ ル マ ー の 一 句 が、 358 年 間 数 学 者 を 動 か し 続 け た。 「フ ェ ル マ ー は 本 当 に 証 明 を 知 っ て い た の か」 と い う 疑 問 は、 現 在 で も 解 け な い。 (現 代 の 数 学 者 の 多 く は、 フ ェ ル マ ー が 知 っ て い た「証 明」 は お そ ら く 不 完 全 で あ っ た と 考 え て い る。)

し か し、 そ の 一 句 が な け れ ば、 358 年 間 の 数 学 史 は 違 う も の に な っ て い た。

数 学 の 余 白 に 書 き 残 さ れ た た っ た 一 句 が、 358 年 後 の イ ギ リ ス の 屋 根 裏 部 屋 で 涙 を 生 ん だ。 そ の 涙 の 中 に、 谷 山 豊 の 1958 年 の 死、 志 村 五 郎 の 引 き 継 ぎ、 リ ベ ッ ト の 飛 躍、 ワ イ ル ズ の 7 年 の 沈 黙 ─ 全 て の 数 学 者 た ち の 仕 事 が 流 れ 込 ん で い る。

中 学 3 年 で「三 平 方 の 定 理」 を 学 ぶ 君 は、 「x² + y² = z² に は 整 数 解 が 無 限 に あ る」 と い う 事 実 か ら、 こ の 358 年 の 物 語 へ の 一 歩 を 踏 み 始 め て い る。
