<h3>二 次 方 程 式 と 解 の 公 式</h3>

<p>
中 3 数 学 で 学 ぶ「<strong>二 次 方 程 式</strong>」 ─ ax² + bx + c = 0。
</p>

<p>
そ し て、 こ の 方 程 式 の 全 て の 解 を 求 め る「<strong>解 の 公 式</strong>」:
</p>

<blockquote class="song">
<p>
x = (-b ± √(b² - 4ac)) / 2a
</p>
</blockquote>

<p>
こ の 公 式 ─ 一 見 複 雑 だ が、 こ の 短 い 式 が、 ax² + bx + c = 0 と い う 全 て の 二 次 方 程 式 の 解 を 一 気 に 与 え て く れ る。<br>
こ の 公 式 が 完 成 す る ま で に、 人 類 は <strong>約 4000 年</strong> か か っ た。
</p>

<h3>4000 年 の 旅</h3>

<h4>① 古 代 バ ビ ロ ニ ア (紀 元 前 1800 年 頃)</h4>

<p>
現 在 の イ ラ ク 地 方 で 栄 え た 古 代 バ ビ ロ ニ ア。 こ こ の 粘 土 板 に は、 既 に 二 次 方 程 式 の 解 法 が 記 さ れ て い た。
</p>

<p>
例 え ば こ ん な 問 題:
</p>

<blockquote>
「正 方 形 の 面 積 と 一 辺 の 長 さ の 和 が 0.75 で あ る。 一 辺 の 長 さ は?」<br>
→ 数 式 で 書 け ば x² + x = 3/4
</blockquote>

<p>
バ ビ ロ ニ ア の 神 官 数 学 者 は、 言 葉 で「こ う 計 算 す れ ば 解 が 出 る」 と い う 手 順 を 記 録 し て い た。<br>
そ れ は、 現 代 の「平 方 完 成」 に 相 当 す る 手 順 だ っ た。
</p>

<p>
4000 年 前 ─ 文 字 式 も、 「+」「=」 の 記 号 も 無 か っ た 時 代 に、 二 次 方 程 式 は す で に 解 け て い た の だ。
</p>

<h4>② 古 代 ギ リ シ ア (紀 元 前 4–3 世 紀)</h4>

<p>
古 代 ギ リ シ ア の 数 学 者 (エ ウ ク レ イ デ ス、 ア ル キ メ デ ス ら) は、 二 次 方 程 式 を <strong>幾 何 学 的</strong> に 解 い た。<br>
「面 積 = ?」 の 形 で 問 題 を 立 て、 図 形 の 切 り 貼 り で 解 を 求 め た。
</p>

<p>
「代 数 と 幾 何 が 分 離 し て い た」 古 代 ギ リ シ ア で は、 こ う し た 幾 何 学 的 解 法 が 主 流 だ っ た。
</p>

<h4>③ イ ス ラ ム 黄 金 期 ─ ア ル・フ ワ ー リ ズ ミ ー (9 世 紀)</h4>

<p>
8–13 世 紀、 イ ス ラ ム 世 界 は <strong>数 学 の 黄 金 期</strong> を 迎 え て い た。<br>
バ グ ダ ー ド の 「知 恵 の 館」 (バ イ ト ・ ア ル ・ ヒ ク マ) で、 ペ ル シ ャ・ア ラ ブ・イ ン ド・ギ リ シ ア の 数 学 者 が 集 ま り、 古 代 の 知 を 統 合 し て い た。
</p>

<p>
そ の 中 心 人 物 が <strong>ア ル・フ ワ ー リ ズ ミ ー</strong> (al-Khwārizmī, 780 頃 –850 頃) で あ る。<br>
彼 は『代 数 学 の 書』(<em>Kitab al-jabr wal-muqabala</em>) で、 二 次 方 程 式 の 体 系 的 解 法 を ま と め た。
</p>

<p>
彼 が 用 い た 言 葉 「<strong>al-jabr</strong>」 (ア ル ・ ジ ャ ブ ル ─「移 項」 の 意) が、 12 世 紀 に ヨ ー ロ ッ パ に 伝 わ り、 「<strong>algebra</strong>」 (代 数) と な っ た。
</p>

<p>
さ ら に、 彼 の 名 前「al-Khwārizmī」 の ラ テ ン 語 訳「Algoritmi」 が、 「<strong>algorithm</strong>」 (ア ル ゴ リ ズ ム = 計 算 手 順) の 語 源 と な っ た。
</p>

<p>
中 3 で 君 が 学 ぶ「代 数」 と、 プ ロ グ ラ マ ー が 使 う「ア ル ゴ リ ズ ム」 ─ ど ち ら も、 1200 年 前 の ペ ル シ ャ の 数 学 者 の 名 前 を 残 し て い る の だ。
</p>

<h4>④ ル ネ サ ン ス イ タ リ ア ─ カ ル ダ ー ノ (16 世 紀)</h4>

<p>
16 世 紀、 ル ネ サ ン ス イ タ リ ア で、 数 学 は さ ら に 発 展 し た。<br>
<strong>ジ ェ ロ ー ラ モ ・ カ ル ダ ー ノ</strong> (Gerolamo Cardano, 1501–1576) は、 1545 年 『大 法 (Ars Magna)』 で、 二 次 方 程 式・三 次 方 程 式・四 次 方 程 式 の 解 法 を 体 系 化 し た。
</p>

<p>
こ の 時 代 に、 「+」「-」「=」「√」 と い っ た 数 学 記 号 が 整 備 さ れ、 解 の 公 式 が 現 代 と 同 じ 形 で 表 現 で き る よ う に な っ た。
</p>

<h4>⑤ デ カ ル ト (1637) ─ 現 代 的 表 記</h4>

<p>
1637 年、 デ カ ル ト (本 サ イ ト chugakusugaku2) の『幾 何 学』 で、 現 代 と ほ ぼ 同 じ 「x、 y」 の 文 字 と 指 数 (x²) の 表 記 が 確 立。<br>
解 の 公 式 が、 現 在 の 形 で 完 成 し た。
</p>

<h3>解 の 公 式 ─ 4000 年 の 結 晶</h3>

<p>
こ う し て 現 在 の 解 の 公 式:
</p>

<blockquote>
x = (-b ± √(b² - 4ac)) / 2a
</blockquote>

<p>
こ の 一 つ の 公 式 に、 4000 年 の 数 学 の 歴 史 が 結 晶 し て い る。<br>
バ ビ ロ ニ ア の 神 官 → ギ リ シ ア の 幾 何 学 者 → イ ス ラ ム の 数 学 者 → イ タ リ ア の カ ル ダ ー ノ → デ カ ル ト → そ し て 中 3 の 君。
</p>

<h3>「algebra」 と「algorithm」 ─ ペ ル シ ャ の 遺 産</h3>

<p>
ア ル・フ ワ ー リ ズ ミ ー が 9 世 紀 に 残 し た 二 つ の 言 葉:
</p>

<ul>
<li><strong>al-jabr (ア ル ・ ジ ャ ブ ル)</strong> → <strong>algebra (代 数)</strong></li>
<li><strong>al-Khwārizmī (ア ル ・ フ ワ ー リ ズ ミ ー、 人 名)</strong> → <strong>algorithm (ア ル ゴ リ ズ ム)</strong></li>
</ul>

<p>
こ の 二 つ の 言 葉 が、 今 や 世 界 中 の 中 学・高 校 数 学、 そ し て プ ロ グ ラ ミ ン グ・AI の 基 礎 概 念 と し て 使 わ れ て い る。
</p>

<p>
1200 年 前 の 一 人 の ペ ル シ ャ 人 の 数 学 者 が、 こ う し て 21 世 紀 の 君 の 教 室 に 生 き 続 け て い る。
</p>

<h3>二 次 方 程 式 を 解 く こ と</h3>

<p>
中 3 で 二 次 方 程 式 を 解 く と き ─ 因 数 分 解 で も、 平 方 完 成 で も、 解 の 公 式 で も ─ 君 は、 4000 年 に わ た っ て 数 学 者 た ち が 磨 い て き た 一 つ の 知 恵 を、 使 っ て い る の だ。
</p>

<p>
そ し て、 中 3 で 二 次 方 程 式 を マ ス タ ー し た 君 は ─ 高 校 で「指 数 関 数」「対 数」「微 分 積 分」 と い う、 17 世 紀 以 降 の 新 し い 数 学 へ と 進 む。
</p>

<p>
4000 年 の 旅 は、 君 か ら 始 ま り、 ま だ 続 く。
</p>
