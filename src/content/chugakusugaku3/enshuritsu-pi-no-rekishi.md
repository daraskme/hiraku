## 円 周 率 π ─ 4000 年 の 計 算 の 旅

### 1. ど ん な 円 で も 同 じ 数

円 を 一 つ 描 い て み よ う。 大 き な 円 で も、 小 さ な 円 で も い い。 そ の「円 周 の 長 さ」 を「直 径」 で 割 っ て み る。

す る と、 ど ん な 大 き さ の 円 で も、 必 ず 同 じ 数 に な る。 約 3.14159…。

こ の 不 思 議 な 数 を、 「円 周 率 π (パ イ)」 と 呼 ぶ。 円 周 = 直 径 × π。 円 の 面 積 = 半 径 × 半 径 × π。 円 に 関 す る あ ら ゆ る 計 算 に、 こ の π が 登 場 す る。

問 題 は ─ こ の π の 正 確 な 値 は、 い っ た い い く つ な の か? そ し て、 そ れ は ど こ ま で 続 く の か?

人 類 は、 4000 年 か け て、 こ の 問 い を 追 い 続 け て き た。

***

### 2. 古 代 ─ 「だ い た い 3 よ り 少 し 大 き い」

最 古 の 記 録 は 約 4000 年 前。 古 代 バ ビ ロ ニ ア で は π ≒ 3.125 (3 と 8 分 の 1)、 古 代 エ ジ プ ト の「リ ン ド・パ ピ ル ス」 で は π ≒ 3.16 と し て い た。 旧 約 聖 書 に は π ≒ 3 と 読 め る 記 述 も あ る。

ど れ も「3 よ り 少 し 大 き い」 と い う 大 ざ っ ぱ な 段 階 だ っ た。 し か し、 「円 周 と 直 径 の 比 は 一 定 だ」 と 気 づ い た こ と 自 体 が、 大 き な 一 歩 で あ っ た。

***

### 3. ア ル キ メ デ ス ─ は さ み う ち の 天 才

円 周 率 を「計 算 で 求 め る」 方 法 を 初 め て 確 立 し た の は、 古 代 ギ リ シ ア の ア ル キ メ デ ス (前 287～前 212) で あ っ た。

彼 の 発 想 は こ う だ。 円 の 内 側 に ぴ っ た り 入 る 正 多 角 形 (内 接 多 角 形) と、 円 の 外 側 を ぴ っ た り 囲 む 正 多 角 形 (外 接 多 角 形) を 描 く。

円 周 は、 内 接 多 角 形 の 周 よ り は 長 く、 外 接 多 角 形 の 周 よ り は 短 い。 つ ま り、 円 周 率 は こ の 二 つ の 間 に「は さ ま れ て」 い る。

多 角 形 の 角 の 数 を 増 や せ ば 増 や す ほ ど、 多 角 形 は 円 に 近 づ き、 は さ む 範 囲 が 狭 ま っ て、 π を 正 確 に 求 め ら れ る。

ア ル キ メ デ ス は、 正 96 角 形 ま で 計 算 し て、 こ う 結 論 し た。

   3.1408… < π < 3.1429…

(正 確 に は 3 + 10/71 < π < 3 + 1/7。) こ の「は さ み う ち」 の 発 想 は、 約 2000 年 後 の「極 限」「積 分」 の 考 え 方 の 先 駆 で あ っ た。

***

### 4. 祖 沖 之 ─ 1000 年 間 世 界 一

5 世 紀、 中 国・南 北 朝 時 代 の 数 学 者 祖 沖 之 (そ ち ゅ う し、 429-500) が、 円 周 率 の 歴 史 に 輝 か し い 記 録 を 残 し た。

彼 は、 アル キ メ デ ス の 方 法 を 推 し 進 め (正 12288 角 形 ま で 計 算 し た と さ れ る)、 円 周 率 を

   3.1415926 < π < 3.1415927

と 求 め た。 小 数 第 7 位 ま で 正 確。 さ ら に、 簡 便 な 近 似 分 数 と し て 355/113 (= 3.14159292…、 「密 率」 と 呼 ぶ) を 示 し た。

こ の 7 桁 の 精 度 は、 そ の 後 約 1000 年 間、 世 界 で 最 も 正 確 な 円 周 率 で あ り 続 け た。 5 世 紀 の 中 国 数 学 の、 驚 異 的 な 水 準 を 示 す。

***

### 5. 日 本 の 和 算 ─ 関 孝 和 と 建 部 賢 弘

江 戸 時 代 の 日 本 に も、 π を 追 究 し た 和 算 家 が い た。

関 孝 和 (1642 頃-1708) は、 多 角 形 法 を 工 夫 し て π を 約 11 桁 求 め た。 そ の 弟 子 建 部 賢 弘 (た け べ か た ひ ろ、 1664-1739) は、 1722 年、 独 自 の「級 数」 (無 限 に 続 く 足 し 算 で π に 近 づ く 方 法) を 用 い て、 π を 41 桁 ま で 計 算 し た。

驚 く べ き は、 建 部 が ヨ ー ロ ッ パ の ニ ュ ー ト ン・ラ イ プ ニ ッ ツ の 微 積 分 と は 全 く 独 立 に、 無 限 級 数 の 手 法 に た ど り 着 い て い た こ と で あ る。 日 本 の 和 算 は、 世 界 水 準 の 数 学 で あ っ た。

***

### 6. ヨ ー ロ ッ パ と 現 代

ヨ ー ロ ッ パ で は、 オ ラ ン ダ の ル ド ル フ・フ ァ ン・コ ー レ ン (1540-1610) が 生 涯 を か け て π を 35 桁 ま で 計 算 し た。 ド イ ツ で は 今 も π を「ル ド ル フ 数」 と 呼 ぶ。

18 世 紀 以 降、 「無 限 級 数」 の 公 式 (マ チ ン の 公 式 等) を 使 っ て、 手 計 算 で 数 百 桁 ま で 求 め ら れ た。 そ し て 1761 年、 ラ ン ベ ル ト が「π は 無 理 数 (分 数 で 表 せ ず、 小 数 が 無 限 に 続 く) で あ る」 こ と を 証 明 し た。

20 世 紀、 コ ン ピ ュ ー タ の 登 場 で 桁 数 は 爆 発 し た。 2024 年 時 点 で、 π は 約 105 兆 桁 ま で 計 算 さ れ て い る。 π の 計 算 は、 今 で は コ ン ピ ュ ー タ の 性 能 を 測 る テ ス ト と し て も 使 わ れ る。

***

### 7. な ぜ 計 算 し 続 け る の か

日 常 生 活 で は、 π は 3.14 で 十 分 で あ る。 ロ ケ ッ ト の 軌 道 計 算 で も、 15 桁 も あ れ ば 太 陽 系 の 端 ま で 誤 差 数 cm で 計 算 で き る。

そ れ で も 人 類 は、 100 兆 桁 を 超 え て π を 計 算 し 続 け て い る。 な ぜ か。

そ れ は、 π が「ど こ ま で も 続 く、 規 則 の な い 無 限 の 数」 だ か ら だ。 終 わ り の な い も の に 挑 み 続 け る ─ そ こ に、 古 代 バ ビ ロ ニ ア か ら 現 代 の コ ン ピ ュ ー タ ま で、 4000 年 続 く 人 類 の 知 的 好 奇 心 が あ る。

中 学 3 年 で「円 周 = 直 径 × 3.14」 と 計 算 す る 君 は、 こ の 4000 年 の 旅 の、 最 新 の 一 歩 に 立 っ て い る の で あ る。
