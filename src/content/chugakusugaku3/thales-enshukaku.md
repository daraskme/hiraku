<h3>紀 元 前 6 世 紀 ─ 古 代 ギ リ シ ャ の 哲 学 者 が 円 を 見 つ め る</h3>

<p>
紀 元 前 6 世 紀、 ギ リ シ ャ の 植 民 都 市 ミ レ ト ス (現 ト ル コ 西 海 岸)。 タ レ ス
(Θαλῆς ・ 紀 元 前 624 頃 ─ 546 頃) と い う 哲 学 者 が い た。
</p>

<p>
彼 は「万 物 の 根 源 は 水 で あ る」 と 唱 え、 西 洋 哲 学 ・ 科 学 の 祖 と さ れ る。 ギ リ シ ャ 七 賢 人
(Seven Sages of Greece) の 一 人 で あ り、 紀 元 前 585 年 の 日 食 を 予 言 し た と も 伝 え ら れ る。
</p>

<p>
そ し て タ レ ス は、 数 学 の 歴 史 上 初 め て「定 理 を 証 明 し た 人」 と さ れ る。 そ の 第 一 が
「半 円 に 内 接 す る 角 は 直 角」 と い う 命 題 だ っ た。
</p>

<h3>タ レ ス の 定 理 ─ 半 円 の 内 接 角 は 直 角</h3>

<p>
紙 と コ ン パ ス を 用 意 し て、 試 し て み よ う。
</p>

<ol>
  <li>円 を 一 つ 描 く</li>
  <li>直 径 (中 心 を 通 る 線) を 一 本 引 く</li>
  <li>直 径 の 両 端 を A と B と す る</li>
  <li>円 周 上 の ど こ か に 点 C を 置 く</li>
  <li>三 角 形 ABC を 描 く</li>
</ol>

<p>
∠ACB (頂 点 C の 角) を 分 度 器 で 測 っ て み よ う ─ 必 ず 90 度 に な る。 C を 円 周 上 の ど こ に
動 か し て も、 直 角 の ま ま で あ る。
</p>

<p>
こ れ が タ レ ス の 定 理 ─ 「直 径 を 一 辺 と す る 円 内 接 三 角 形 は、 必 ず 直 角 三 角 形」。
</p>

<h3>な ぜ 直 角 に な る の か ─ 中 3 円 周 角 の 定 理 で 説 明</h3>

<p>
中 3 で 学 ぶ「円 周 角 の 定 理」 を 使 え ば、 タ レ ス の 定 理 は 簡 単 に 説 明 で き る。
</p>

<p><strong>円 周 角 の 定 理</strong> ─ 「同 じ 弧 に 対 す る 円 周 角 は、 中 心 角 の 半 分」。</p>

<p>
直 径 の 弧 (半 円 弧) の 中 心 角 は 180 度。 そ の 半 分 は 90 度。 だ か ら、 半 円 弧 に 対 す る
円 周 角 は 必 ず 90 度 ─ こ れ が タ レ ス の 定 理 で あ る。
</p>

<p>
タ レ ス は 2600 年 前 に こ の 真 理 を 発 見 し た。 一 般 化 し た「円 周 角 の 定 理」 は、 後 に 別 の
数 学 者 (ヒ ッ ポ ク ラ テ ス、 ユ ー ク リ ッ ド ら) が 確 立 し て い く こ と に な る。
</p>

<h3>円 周 角 の 定 理 ─ 中 3 で 学 ぶ 一 般 形</h3>

<p>
中 3 で 学 ぶ 円 周 角 の 定 理 の 一 般 形 を 確 認 ─
</p>

<ul>
  <li><strong>定 理 1</strong> ─ 同 じ 弧 に 対 す る 円 周 角 は、 ど こ か ら 見 て も 等 し い。</li>
  <li><strong>定 理 2</strong> ─ 中 心 角 = 2 × 円 周 角 (同 じ 弧 に 対 し て)。</li>
  <li><strong>系</strong> ─ 直 径 に 対 す る 円 周 角 は 90 度 (タ レ ス の 定 理)。</li>
</ul>

<p>
ど の 教 科 書 に も 載 っ て い る こ の 定 理 が、 紀 元 前 6 世 紀 の ギ リ シ ャ で 始 ま っ た こ と を 知 ろ う。
</p>

<h3>応 用 ─ 直 角 三 角 形 を 作 る 道 具</h3>

<p>
タ レ ス の 定 理 は、 直 角 三 角 形 を 作 る「道 具」 と し て 応 用 で き る。
</p>

<p><strong>例</strong> ─ あ る 点 A か ら 別 の 点 B ま で の 距 離 を 直 接 測 れ な い 時 (川 を 隔 て る 等)、
コ ン パ ス で 円 を 描 い て A と B が 円 周 上 に な る よ う に し、 直 径 の 反 対 側 の 点 C を 取 れ ば、
直 角 三 角 形 が で き る。 三 平 方 の 定 理 で 距 離 を 計 算 で き る。
</p>

<p>
古 代 ギ リ シ ャ の 測 量・建 築 で、 こ の よ う な 方 法 が 使 わ れ た と さ れ る。
</p>

<h3>応 用 ─ 野 球 の ダ イ ヤ モ ン ド</h3>

<p>
野 球 場 の 一 塁 ・ 三 塁 を 結 ぶ 直 線 を 直 径 と す る 円 上 に、 ピ ッ チ ャ ー が 立 て ば、
∠一 塁 ・ ピ ッ チ ャ ー ・ 三 塁 = 90 度。 こ れ が タ レ ス の 定 理 だ。
</p>

<p>
野 球 の ダ イ ヤ モ ン ド は 正 方 形 で あ り、 そ の 対 角 線 を 直 径 と す る 円 上 に 四 つ の ベ ー ス が 全 て
あ る (正 方 形 の 外 接 円)。 ピ ッ チ ャ ー マ ウ ン ド は 正 方 形 の 中 心 で あ り、 ど の 二 つ の 隣 接 ベ ー ス
か ら 見 て も、 直 角 で あ る ─ 円 周 角 ・ タ レ ス の 定 理 の 美 し い 応 用 例。
</p>

<h3>結 び ─ 2600 年 前 の 知 恵 を 君 が 使 う</h3>

<p>
紀 元 前 6 世 紀 の ギ リ シ ャ 七 賢 人 が 発 見 し た 真 理 を、 21 世 紀 の 中 3 が 教 科 書 で 学 ぶ。
数 学 と は、 時 代 と 文 化 を 超 え て 受 け 継 が れ る 普 遍 の 知 恵 で あ る。
</p>

<p>
円 を 一 つ 描 く と、 そ こ に は タ レ ス・ピ タ ゴ ラ ス・ユ ー ク リ ッ ド・ニ ュ ー ト ン ・ そ し て 君 の
全 員 が 集 う ─ こ の 不 思 議 を 心 に 留 め て、 円 の 問 題 を 解 い て み よ う。
</p>
