<h3>原文 (漢文)</h3>

<blockquote class="song">
<p>
項王軍壁垓下、兵少食尽。<br>
漢軍及諸侯兵、囲之数重。<br>
夜聞漢軍四面皆楚歌、項王乃大驚曰、<br>
「漢皆已得楚乎。是何楚人之多也。」<br>
項王則夜起、飲帳中。<br>
有美人名虞、常幸従。<br>
駿馬名騅、常騎之。<br>
於是項王乃悲歌慷慨、自為詩曰、<br>
「力抜山兮気蓋世、<br>
　時不利兮騅不逝、<br>
　騅不逝兮可奈何、<br>
　虞兮虞兮奈若何。」<br>
歌数闋、美人和之。<br>
項王泣数行下、左右皆泣、莫能仰視。
</p>
</blockquote>

<h3>現代語訳</h3>

<p>
項王 (項羽) の 軍 は、 垓下 に 城壁 を 築 い て 立 て 籠 も っ た が、 兵 は 少 な く、 食料 も 尽 き か け て い た。<br>
漢 軍 と 諸侯 の 軍 が、 何 重 に も 包 囲 し て い た。
</p>

<p>
夜、 漢 軍 の 四 方 か ら、 楚 (項羽 の 故 郷) の 歌 が 聞 こ え て く る の を 聞 い て、 項王 は 大 い に 驚 い て 言 っ た:
</p>

<blockquote>
「漢 は も う 楚 を 全 部 攻 め 取 っ て し ま っ た の か? な ぜ こ ん な に 楚 の 兵 が 多 い の だ ?」
</blockquote>

<p>
そ し て 項王 は、 夜 中 に 起 き 上 が り、 陣 中 の 幕 の 中 で 酒 を 飲 ん だ。<br>
彼 に は、 虞 (ぐ) と い う 名 の 寵 妃 が お り、 常 に 寵 愛 し 連 れ て い た。<br>
ま た、 騅 (すい) と い う 名 の 駿 馬 が お り、 常 に そ れ に 乗 っ て い た。
</p>

<p>
こ こ で 項王 は、 悲 し み に 沈 み、 慷 慨 (こ う が い ・ 憤 慨) し な が ら、 自 ら 詩 を 作 っ て 詠 ん だ:
</p>

<blockquote class="song">
力 山 を 抜 き、 気 世 を 蓋 ふ (お お う)<br>
時 利 あ ら ず し て、 騅 行 か ず<br>
騅 行 か ざ る を 奈 何 (いかん) す べ き<br>
虞 や 虞 や、 若 (なんじ) を 奈 何 せ ん
</blockquote>

<p>
(私 の 力 は 山 を 抜 き 取 る ほ ど、 気 概 は 世 界 を 覆 う ほ ど 大 き か っ た。<br>
　 だ が、 時 の 運 は 私 に 利 し て い な い。 愛 馬 騅 も 走 ろ う と し な い。<br>
　 騅 が 走 ら な い の を、 ど う し て や れ ば よ い の か。<br>
　 虞 よ、 虞 よ、 そ な た を、 私 は ど う し て や れ ば よ い の か。)
</p>

<p>
項王 は こ の 詩 を 数 回 繰 り 返 し て 歌 い、 美 人 虞 も 唱 和 し た。<br>
項王 の 涙 が 何 筋 も 流 れ 落 ち、 周 り の 部 下 た ち も 皆 泣 い て、 顔 を 上 げ て 主 を 見 る こ と が で き る 者 は い な か っ た。
</p>

<h3>解説 ─ 史記 の な か の 最 大 の 名 場 面</h3>

<p>
こ の 場 面 は、 中 国 文 学 史 上、 最 も 有 名 で 感 動 的 な 場 面 の 一 つ で あ る。
</p>

<p>
紀 元 前 202 年、 楚 漢 戦 争 の 最 終 局 面。<br>
4 年 間 戦 い 続 け た 項 羽 は、 つ い に 漢 軍 (劉 邦 軍) に 垓 下 で 完 全 包 囲 さ れ た。<br>
兵 は 約 10 万 か ら 数 千 に 減 り、 食 料 も 尽 き か け、 援 軍 も 来 ぬ 絶 望 的 状 況。
</p>

<p>
そ の 夜、 漢 軍 の 陣 営 か ら、 突 然 ─ <strong>項 羽 の 故 郷 「楚」 の 民 謡</strong> が 四 方 か ら 響 い て き た。<br>
こ れ は 漢 の 軍 師 ・ 張 良 ら の 戦 略 で あ っ た:<br>
楚 の 兵 出 身 の 漢 軍 兵 た ち に、 楚 の 歌 を 歌 わ せ た の だ。
</p>

<p>
こ の 「四面 楚歌」 を 聞 い た 瞬 間、 項 羽 は こ う 考 え た ─<br>
「漢 は も う 楚 全 土 を 制 圧 し て し ま っ た の か。 だ か ら こ そ、 こ れ ほ ど 多 く の 楚 の 兵 が、 敵 側 に い る の だ」<br>
心 理 戦 と し て、 こ の 戦 術 は 完 璧 で あ っ た。
</p>

<h3>「四面 楚歌」 ─ 故事 成 語 の 誕 生</h3>

<p>
こ の 場 面 か ら、 中 国 ・ 日 本 で 共 通 し て 使 わ れ る <strong>故 事 成 語</strong> が 生 ま れ た:
</p>

<blockquote>
<strong>四面 楚歌</strong> ─ 周 囲 を 完 全 に 敵 に 囲 ま れ、 助 け の 望 み が 絶 え た 状 況
</blockquote>

<p>
現 代 日 本 で も、 政 治 ・ ビ ジ ネ ス ・ 個 人 の 困 難 な 場 面 で よ く 使 わ れ る 表 現 で あ る:
</p>

<ul>
<li>「政 治 家 が 不 祥 事 で 四面 楚歌 の 状 況 に 陥 っ た」</li>
<li>「会 議 で 自 分 の 提 案 が 全 員 か ら 反 対 さ れ、 四面 楚歌 だ っ た」</li>
<li>「家 庭 内 で の 意 見 対 立 で 四面 楚歌 だ」</li>
</ul>

<p>
2200 年 前 の 中 国 の 一 場 面 が、 今 も な お 日 本 語 の 中 で 生 き 続 け て い る の だ。
</p>

<h3>項 羽 の 詩 ─ 中 国 文 学 史 の 名 篇</h3>

<p>
項 羽 が こ の 絶 望 的 状 況 で 詠 ん だ 詩 「<strong>力 抜 山 兮 気 蓋 世</strong>」 は、 中 国 文 学 史 で も 屈 指 の 名 篇 と さ れ る。
</p>

<table>
<thead>
<tr><th>句</th><th>意 味</th></tr>
</thead>
<tbody>
<tr><td><strong>力 抜 山 兮 気 蓋 世</strong></td><td>私 の 力 は 山 を 抜 き、 気 概 は 世 界 を 覆 う</td></tr>
<tr><td><strong>時 不 利 兮 騅 不 逝</strong></td><td>時 の 運 は 私 に 利 さ ず、 愛 馬 騅 も 走 ら な い</td></tr>

<tr><td><strong>虞 兮 虞 兮 奈 若 何</strong></td><td>虞 よ、 虞 よ、 そ な た を ど う し て や れ ば よ い の か</td></tr>
</tbody>

<p>
こ の 4 行 詩 ─ 中 国 で 「<strong>垓 下 歌</strong>」 (が い か の う た) と 呼 ば れ る ─ に は、 項 羽 と い う 一 人 の 男 の 全 て が 凝 縮 さ れ て い る。<br>
彼 の 武 勇 (力 抜 山)、 そ の 武 勇 が 通 用 し な く な っ た 嘆 き (時 不 利)、 愛 馬 へ の 思 い (騅)、 そ し て 何 よ り も、 愛 す る 女 性 虞 (ぐ) を 残 し て 死 な ね ば な ら な い 痛 切 さ (虞 兮 虞 兮)。
</p>

<h3>司 馬 遷 と 『史 記』</h3>

<p>
『史 記』 の 著 者 ・ 司 馬 遷 (し ば せ ん) も、 ま た 悲 劇 の 人 で あ っ た。
</p>

<p>
父 ・ 司 馬 談 の 遺 志 を 継 い で 『史 記』 編 纂 を 始 め た 司 馬 遷 だ が、 紀 元 前 99 年、 匈 奴 と の 戦 い で 降 伏 し た 名 将 ・ 李 陵 (り り ょ う) を 弁 護 し た た め、 武 帝 の 怒 り を 買 い、<strong>宮 刑</strong> (去 勢) に 処 さ れ た。
</p>

<p>
当 時 の 中 国 で、 宮 刑 を 受 け る こ と は 死 ぬ こ と よ り も 屈 辱 と さ れ た。<br>
だ が 司 馬 遷 は、 こ の 屈 辱 を 耐 え て、 父 の 遺 志 で あ る 『史 記』 を 完 成 さ せ る こ と を 選 ん だ。<br>
そ し て、 約 14 年 か け て、 全 130 巻 ・ 約 52 万 字 の 大 著 を 完 成 さ せ た の で あ る。
</p>

<p>
彼 が 描 い た 項 羽 像 ─ 名 将 で あ り な が ら 最 後 に 敗 れ、 愛 す る 女 性 と 別 れ ね ば な ら な か っ た 悲 劇 の 英 雄 ─ に は、 司 馬 遷 自 身 の <strong>挫 折 と 屈 辱 を 知 る 者 の 共 感</strong> が、 ど こ か に 滲 ん で い る か も し れ な い。
</p>

<h3>日 本 へ の 影 響</h3>

<p>
『史 記』 は、 5 世 紀 頃 に 日 本 に 伝 来 し、 以 後 千 数 百 年 に わ た り、 日 本 人 の 教 養 の 根 幹 を な し て き た。
</p>

<p>
特 に 「項 羽 と 劉 邦」 の 物 語 は、 日 本 文 学 ・ 演 劇 ・ 漫 画 で 数 多 く 翻 案 さ れ て き た:
</p>

<ul>
<li>司 馬 遼 太 郎 『項 羽 と 劉 邦』 (1980 年 代 の ベ ス ト セ ラ ー)</li>
<li>横 山 光 輝 『項 羽 と 劉 邦』 (漫 画)</li>
<li>歌 舞 伎 『四面 楚歌』 (江 戸 期 の 演 目)</li>
<li>中 島 敦 『李 陵』 (司 馬 遷 が 弁 護 し た 李 陵 が 主 人 公)</li>
</ul>

<p>
2200 年 前 の 中 国 の 一 場 面 が、 今 も 日 本 人 の 想 像 力 を 刺 激 し 続 け て い る。
</p>

<h3>項 羽 の 最 期</h3>

<p>
こ の 「四面 楚歌」 の 後、 項 羽 は 800 の 騎 馬 を 率 い て 包 囲 を 突 破 し、 烏 江 (う こ う) ま で 落 ち 延 び る。<br>
そ こ で、 故 郷 楚 へ 渡 る 船 が 用 意 さ れ て い た が、 項 羽 は こ う 言 っ て 拒 む:
</p>

<blockquote>
「江 東 (こ う と う) の 子 弟 8 千 人 を 率 い て 西 へ 進 ん だ 私 が、 今 一 人 で 帰 っ て、 ど の 顔 で 江 東 の 父 老 (古 老) に 会 え ば よ い の か。 た と え 彼 ら が 何 も 言 わ な く て も、 私 自 身 の 心 が、 私 を 許 さ な い」
</blockquote>

<p>
そ し て、 項 羽 は 自 ら 首 を 切 っ て 自 害 し た。 30 歳 で あ っ た。
</p>

<p>
中 国 史 上 で も っ と も 強 大 な 武 将 の 一 人 で あ り な が ら、 結 局 政 治 家 と し て の 劉 邦 に 敗 れ た 悲 劇 の 英 雄 ─ そ れ が 項 羽 で あ り、 「四面 楚歌」 と い う 故 事 成 語 の 源 で あ る。
</p>
