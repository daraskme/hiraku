<h3>現 代 語 訳</h3>

<p>
国 (唐) は 反 乱 軍 に よ っ て 破 滅 し て し ま っ た が、 山 河 (自 然) は 変 わ ら ず そ こ に あ る。<br>
こ の 長 安 の 城 (み や こ) に も 春 が 訪 れ、 草 木 は 深 く 茂 っ て い る。
</p>

<p>
こ の 戦 乱 の 時 代 を 思 う と、 花 を 見 て さ え も 涙 が こ ぼ れ、<br>
妻 子 と の 別 れ を 思 う と、 鳥 の 声 を 聞 い て さ え も 心 が 驚 い て し ま う。
</p>

<p>
戦 い の 狼 煙 (の ろ し) は も う 三 ヶ 月 も 続 き、<br>
妻 か ら の 一 通 の 手 紙 が、 万 金 ほ ど の 価 値 を 持 つ。
</p>

<p>
白 髪 の 頭 を か く と、 髪 は さ ら に 短 く な っ て し ま い、<br>
も は や 冠 を 留 め る 簪 (か ん ざ し) も 挿 せ な い ほ ど に な っ て し ま っ た。
</p>

<h3>解 説 ─ 757 年 春、 戦 火 の 長 安</h3>

<p>
こ の 詩 が 書 か れ た 757 年 春 ─ 中 国 史 上 最 大 の 内 乱 「安 史 の 乱」 (755–763) の 真 っ た だ 中 で あ っ た。
</p>

<p>
755 年 11 月、 唐 の 節 度 使 ・ 安 禄 山 (あ ん ろ く ざ ん) が 反 乱 を 起 こ し、 翌 756 年 6 月 に 長 安 を 占 領。<br>
玄 宗 皇 帝 は 西 へ 逃 れ、 楊 貴 妃 (よ う き ひ) は 殺 害 さ れ た。
</p>

<p>
杜 甫 は こ の 戦 乱 の 中、 妻 子 を 田 舎 に 預 け て 単 身、 新 し い 皇 帝 (粛 宗) に 仕 え よ う と し て 動 い て い た。<br>
だ が 反 乱 軍 に 捕 ら え ら れ、 長 安 に 抑 留 さ れ て し ま う。
</p>

<p>
そ の 抑 留 中 の 757 年 春、 長 安 の 廃 墟 の 中 で 詠 ん だ の が、 こ の 「春 望」 で あ る。
</p>

<h3>「国 破 れ て 山 河 あ り」 ─ 一 句 の 重 み</h3>

<p>
こ の 詩 の 第 1 句 ─ 「<strong>国 破 山 河 在</strong>」 ─ は、 千 年 以 上 に わ た り、 東 ア ジ ア の 文 学 で 最 も 多 く 引 用 さ れ て き た 一 句 の 一 つ で あ る。
</p>

<p>
わ ず か 5 文 字 の 中 に、 こ う 凝 縮 さ れ て い る:
</p>
<ul>
<li><strong>国 (人 間 の 営 み)</strong> の は か な さ ─ 一 朝 に し て 滅 び る</li>
<li><strong>山 河 (自 然)</strong> の 永 遠 性 ─ 何 が 起 こ っ て も 変 わ ら ず 残 る</li>
<li>そ の 対 比 か ら 浮 か び 上 が る、 人 間 存 在 の 痛 切 な は か な さ</li>
</ul>

<p>
こ の 「人 と 自 然 の 対 比 か ら 来 る 哀 し み」 は、 そ の 後 の 中 国・日 本 文 学 の 一 大 主 題 と な っ た。
</p>

<h3>五 言 律 詩 と 対 句 の 美</h3>

<p>
「春 望」 は、 唐 代 に 完 成 し た <strong>五 言 律 詩</strong> (1 句 5 字、 8 句) の 厳 密 な 形 式 で 書 か れ て い る。
</p>

<p>
特 に 美 し い の は、 第 3-4 句 と 第 5-6 句 の 「<strong>対 句</strong>」 で あ る。
</p>

<p>
「感 時 - 恨 別」「花 - 鳥」「濺 涙 - 驚 心」 ─ そ れ ぞ れ の 字 が 完 璧 に 対 応 す る。<br>
こ の 対 句 の 美 し さ が、 詩 全 体 に <strong>整 然 と し た リ ズ ム と 音 楽 性</strong> を 与 え て い る。
</p>

<h3>松 尾 芭 蕉 と 「国 破 れ て」</h3>

<p>
日 本 で 「国 破 れ て 山 河 あ り」 を 最 も 深 く 受 け 止 め た 一 人 は、 江 戸 時 代 の 俳 人 ・ <strong>松 尾 芭 蕉</strong> (1644–1694) で あ る。
</p>

<p>
1689 年 (元 禄 2 年)、 芭 蕉 は 『奥 の 細 道』 の 旅 の 途 中、 平 泉 (現 在 の 岩 手 県) を 訪 れ た。<br>
そ こ は、 か つ て 奥 州 藤 原 氏 の 栄 華 を 誇 っ た 場 所 だ が、 既 に 400 年 以 上 前 に 滅 び、 今 は 草 が 茂 る だ け の 廃 墟 で あ っ た。
</p>

<p>
そ こ で 芭 蕉 は こ う 書 き 出 す ─<br>
「三 代 の 栄 耀 (え い よ う) 一 睡 の 中 (う ち) に し て、 大 門 (だ い も ん) の 跡 は 一 里 こ な た に あ り。 秀 衡 (ひ で ひ ら) が 跡 は 田 野 (で ん や) に な り て、 金 鶏 山 (き ん け い ざ ん) 飾 (の み) 形 (か た ち) を 残 す。 ま づ 高 館 (た か だ ち) に 登 れ ば、 (中 略) <strong>『国 破 れ て 山 河 あ り、 城 春 に し て 草 青 み た り』</strong> と、 笠 う ち し き て、 時 の 移 る ま で 涙 を 落 と し は べ り ぬ」
</p>

<p>
そ し て 芭 蕉 は、 こ こ で あ の 名 句 を 詠 ん だ:
</p>

<blockquote class="song">
夏 草 や 兵 (つ わ も の) ど も が 夢 の 跡
</blockquote>

<p>
中 国 ・ 杜 甫 (757 年) → 日 本 ・ 松 尾 芭 蕉 (1689 年) → 私 た ち (2026 年)。<br>
「国 破 れ て 山 河 あ り」 と い う 一 句 が、 1200 年 以 上 に わ た り、 詩 人 か ら 詩 人 へ と 受 け 継 が れ て き た の だ。
</p>

<h3>戦 後 日 本 と 「国 破 れ て」</h3>

<p>
1945 年 8 月 15 日、 日 本 は 太 平 洋 戦 争 に 敗 れ た。<br>
東 京・大 阪・名 古 屋・広 島・長 崎 ─ 主 要 都 市 は 空 襲 と 原 爆 で 廃 墟 と 化 し て い た。
</p>

<p>
そ う し た 焼 け 跡 の 中 で、 多 く の 日 本 人 が 思 い 出 し た の が、 杜 甫 の こ の 一 句 で あ っ た:<br>
「<strong>国 破 れ て 山 河 あ り</strong>」
</p>

<p>
国 (戦 前 の 大 日 本 帝 国) は 滅 ん だ。 だ が、 山 と 河 (日 本 の 自 然) は、 変 わ ら ず こ こ に あ る。<br>
そ の 自 然 を 拠 り 所 に、 私 た ち は 再 び 生 き 直 す こ と が で き る ─ そ ん な 慰 め と 励 ま し を、 こ の 一 句 か ら 多 く の 人 が 受 け 取 っ た。
</p>

<p>
川 端 康 成 ら 戦 後 文 学 者 も、 こ の 一 句 を し ば し ば 引 用 し た。<br>
1200 年 前 の 中 国 の 詩 が、 1945 年 の 日 本 に 改 め て 響 い た の で あ る。
</p>

<h3>詩 聖 ・ 杜 甫</h3>

<p>
杜 甫 (712–770) は、 李 白 (701–762) と 並 ぶ 中 国 唐 代 最 大 の 詩 人 で あ る。
</p>

<p>
だ が 二 人 の 詩 風 は 対 照 的 だ っ た:
</p>

<ul>
<li><strong>李 白「詩 仙」</strong> ─ 酒 と 月 を 愛 し、 自 由 奔 放 に 想 像 力 を 飛 翔 さ せ る</li>
<li><strong>杜 甫「詩 聖」</strong> ─ 民 衆 の 苦 し み を 見 つ め、 戦 乱・貧 困・離 別 を 重 々 し く 詠 む</li>
</ul>

<p>
杜 甫 が 「詩 聖」 と 呼 ば れ る 理 由 は、 彼 が <strong>「人 間 の 痛 み」 を 最 も 深 く、 最 も 正 確 に 詩 に し た</strong> か ら で あ る。<br>
「春 望」 は、 そ の 代 表 作 の 一 つ で あ る。
</p>

<h3>千 年 を 超 え る 響 き</h3>

<p>
757 年 春、 戦 火 の 長 安 で 詠 ま れ た 五 言 律 詩。<br>
1269 年 を 経 た 2026 年、 日 本 の 高 校 生 で あ る 君 が こ れ を 読 む。
</p>

<p>
「国 破 れ て 山 河 あ り」<br>
こ の 5 文 字 は、 戦 争 が 続 く 現 代 の 世 界 (ウ ク ラ イ ナ、 中 東 等) に も、 ま た 災 害・パ ン デ ミ ッ ク 後 の 世 界 に も、 ど こ か で 響 き 続 け て い る。
</p>

<p>
詩 が、 千 年 以 上 の 時 を 超 え て、 人 間 の 同 じ 痛 み に 寄 り 添 う ─ そ れ が、 杜 甫 が 私 た ち に 残 し た 最 大 の 贈 り 物 で あ る。
</p>
