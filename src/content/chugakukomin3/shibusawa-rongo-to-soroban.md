<h3>本 文 (現 代 語 抜 粋)</h3>

<blockquote>
<p>
人 は よ く 私 に 尋 ね る。<br>
「あ な た は 一 方 で 論 語 を 学 び、 ま た 一 方 で 算 盤 を 弾 く。 道 徳 と 利 益 と 言 う、 一 見 矛 盾 す る 二 つ の 道 を 同 時 に 歩 む の は、 ど う し て で す か」 と。
</p>

<p>
そ の 質 問 に 対 す る 私 の 答 え は、 こ う で あ る。
</p>

<p>
道 徳 と 経 済 は、 決 し て 矛 盾 す る も の で は な い。<br>
む し ろ、 <strong>両 者 は 一 つ で あ る べ き</strong> で あ る。
</p>

<p>
何 故 か。 商 売 ・ 経 営 と は、 結 局 の と こ ろ 「人 と 人 と の 関 係」 の 上 に 成 り 立 つ も の だ か ら で あ る。<br>
品 物 を 売 る 相 手、 サ ー ビ ス を 提 供 す る 相 手、 共 に 働 く 仲 間、 株 主、 そ し て 社 会 全 体 ─ こ れ ら す べ て の 「人」 と の 信 頼 関 係 が 損 な わ れ た と き、 ど ん な に 巧 妙 な 算 盤 を 弾 い て も、 長 期 的 に は 必 ず 失 敗 す る。
</p>

<p>
こ の 「人 と 人 と の 信 頼 関 係」 を 守 る も の こ そ、 道 徳 で あ る。<br>
正 直 で あ る こ と、 約 束 を 守 る こ と、 相 手 の 立 場 に 立 つ こ と、 公 平 で あ る こ と ─ こ れ ら が な け れ ば、 商 売 そ の も の が 成 り 立 た な い。
</p>

<p>
だ か ら、 道 徳 を 守 る こ と は、 経 済 的 損 失 で は な く、 む し ろ <strong>長 期 的 な 経 済 的 利 益</strong> な の で あ る。
</p>

<p>
私 は こ う い う 立 場 を、 「<strong>道 徳 経 済 合 一 説</strong>」 (ど う と く け い ざ い ご う い つ せ つ) と 呼 ん で い る。
</p>

<p>
具 体 的 に は、 こ う で あ る。<br>
私 は こ れ ま で、 第 一 国 立 銀 行 ・ 王 子 製 紙 ・ 東 京 海 上 火 災 ・ 帝 国 ホ テ ル ・ 東 京 株 式 取 引 所 な ど、 約 500 の 企 業 の 設 立 や 経 営 に 関 わ っ て き た。<br>
だ が、 私 自 身 の 個 人 資 産 を 大 き く 増 や す こ と は 求 め な か っ た。<br>
そ の 代 わ り、 こ れ ら の 企 業 が 日 本 の 産 業 ・ 経 済 を 発 展 さ せ、 多 く の 人 々 に 仕 事 と 富 を 与 え る こ と を 目 指 し た。
</p>

<p>
ま た、 教 育 ・ 福 祉 の 分 野 で も、 約 600 の 事 業 に 関 わ っ た。<br>
東 京 商 法 講 習 所 (現 在 の 一 橋 大 学)、 日 本 女 子 大 学、 早 稲 田 大 学 ─ 多 く の 私 学 の 設 立 に も 力 を 貸 し た。<br>
こ れ ら は 直 接 的 な 利 益 は 生 ま な い が、 国 家 ・ 社 会 の 長 期 的 利 益 と な る。
</p>

<p>
若 い 諸 君 へ。<br>
こ れ か ら 商 売 や 仕 事 に 携 わ ろ う と す る 諸 君 に、 一 つ だ け 申 し 上 げ た い こ と が あ る。
</p>

<blockquote>
<strong>「片 手 に 論 語、 片 手 に 算 盤 を 持 ち な さ い」</strong>
</blockquote>

<p>
道 徳 (論 語) と 経 済 (算 盤) ─ こ の 二 つ を 同 時 に 持 ち 続 け る こ と が、 真 に 豊 か な 人 生 と、 真 に 持 続 す る 事 業 の 基 礎 で あ る。
</p>
</blockquote>

<h3>解 説 ─ 「日 本 資 本 主 義 の 父」</h3>

<p>
渋 沢 栄 一 (1840–1931) は、 91 歳 の 長 寿 を 全 う し た、 日 本 近 代 化 の 巨 人 で あ る。
</p>

<p>
彼 の 業 績 は 驚 異 的:
</p>

<ul>
<li><strong>関 与 し た 企 業 数</strong> ─ 約 <strong>500 社</strong></li>
<li><strong>関 与 し た 社 会 事 業</strong> ─ 約 <strong>600 件</strong> (教 育・福 祉・国 際 親 善)</li>
<li><strong>代 表 的 企 業</strong> ─ 第 一 国 立 銀 行・王 子 製 紙・東 京 海 上 火 災・東 京 ガ ス・東 京 株 式 取 引 所・帝 国 ホ テ ル・サ ッ ポ ロ ビ ー ル の 前 身 な ど</li>
<li><strong>代 表 的 学 校</strong> ─ 東 京 商 法 講 習 所 (一 橋 大 学 前 身)・日 本 女 子 大 学・早 稲 田 大 学 (大 隈 重 信 と 共 同)・東 京 高 等 商 業 学 校</li>
</ul>

<p>
彼 が 日 本 の 近 代 経 済 ・ 教 育 ・ 福 祉 に 与 え た 影 響 の 大 き さ ゆ え に、 「<strong>日 本 資 本 主 義 の 父</strong>」 と 呼 ば れ る。
</p>

<h3>渋 沢 の 生 涯 ─ 農 民 か ら 実 業 家 へ</h3>

<p>
渋 沢 は、 武 蔵 国 血 洗 島 (ち あ ら い じ ま、 現 在 の 埼 玉 県 深 谷 市) の <strong>農 民 の 家</strong> に 生 ま れ た。<br>
20 代 で 一 橋 家 (徳 川 慶 喜 の 家 督) に 仕 え、 27 歳 の 時、 1867 年 の <strong>パ リ 万 博</strong> に 慶 喜 の 弟・徳 川 昭 武 の 随 員 と し て 渡 欧 し た。
</p>

<p>
欧 州 で の 1 年 半 余 り の 滞 在 が、 渋 沢 の 経 済 観 を 一 変 さ せ た。<br>
銀 行 ・ 株 式 会 社 ・ 鉄 道 ・ 公 共 事 業 ─ 西 洋 の 「資 本 主 義 シ ス テ ム」 を 目 の 当 た り に し、 「日 本 で も こ れ を 作 ら ね ば な ら な い」 と 確 信 し た。
</p>

<p>
維 新 後、 大 蔵 省 に 入 省 し て 制 度 設 計 に 携 わ り、 1873 年 (明 治 6 年) に 退 官。<br>
そ の 後 約 60 年 間、 民 間 実 業 家 と し て、 上 記 の 500 社 を 創 設 ・ 支 援 し て い っ た の で あ る。
</p>

<h3>「道 徳 経 済 合 一 説」 ─ 西 洋 と の 違 い</h3>

<p>
渋 沢 の 経 営 哲 学 の 核 は、 「<strong>道 徳 と 経 済 は 一 つ で あ る べ き</strong>」 と い う 主 張 で あ る。
</p>

<p>
こ れ は、 西 洋 の 経 済 学 (ア ダ ム ・ ス ミ ス 以 降) と は 異 な る 立 場 で あ っ た。<br>
西 洋 経 済 学 で は、 「経 済 は 経 済 法 則 に 従 う も の」「道 徳 は 別 の 領 域」 と 分 離 さ れ る こ と が 多 か っ た。
</p>

<p>
そ し て こ の 道 徳 の 基 準 と し て、 彼 は 『<strong>論 語</strong>』 を 採 用 し た。<br>
論 語 = 孔 子 の 言 葉 (本 サ イ ト kotokokanbun1/rongo-gakuji で 既 習) ─ 「仁・義・礼・智・信」 の 五 徳。<br>
こ れ ら を、 西 洋 由 来 の 「算 盤」 (商 売 の 計 算) と 両 立 さ せ る こ と で、 日 本 独 自 の 経 済 倫 理 を 提 唱 し た の だ。
</p>
