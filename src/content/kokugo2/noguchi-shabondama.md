<h3>げ ん だ い ご に な お す と</h3>

<p>
シ ャ ボ ン 玉 が 飛 ん だ。 屋 根 (や ね) の 高 さ ま で 飛 ん だ。<br>
屋 根 ま で 飛 ん で、 そ し て、 こ わ れ て、 消 え て し ま っ た。
</p>

<p>
シ ャ ボ ン 玉 が 消 え た。 飛 ぶ こ と も で き ず に 消 え た。<br>
生 ま れ て、 す ぐ に、 こ わ れ て、 消 え て し ま っ た。
</p>

<p>
風 よ、 風 よ、 吹 か な い で。<br>
シ ャ ボ ン 玉 を、 飛 ば さ せ て あ げ よ う。
</p>

<h3>悲 し み の 中 で 生 ま れ た 詩</h3>

<p>
こ の 詩 に は、 知 ら れ ざ る 悲 し い 背 景 が あ る。
</p>

<p>
野 口 雨 情 (の ぐ ち う じ ょ う) は、 詩 人 で あ る と 同 時 に、 一 人 の 父 で も あ っ た。<br>
彼 に は <strong>「み ど り」</strong> と い う 名 前 の 娘 が い た。 だ が、 生 ま れ て わ ず か 8 日 で、 そ の 子 は 亡 く な っ て し ま っ た。
</p>

<p>
そ の 後 も 雨 情 は、 詩 人 と し て 多 く の 童 謡 を 書 き 続 け た。<br>
そ し て、 1922 年 (大 正 11 年)、 雑 誌 『金 の 塔』 の 11 月 号 に、 彼 は こ の 「シ ャ ボ ン 玉」 を 発 表 し た。
</p>

<p>
「<strong>生 ま れ て す ぐ に、 こ わ れ て 消 え た</strong>」 ─ こ の 一 行 は、 ま さ に、 生 ま れ て 8 日 で 亡 く な っ た 我 が 子 「み ど り」 を 偲 ぶ も の だ っ た と、 伝 え ら れ て い る。
</p>

<p>
だ が 雨 情 は、 そ の 悲 し み を、 子 ど も に も わ か る、 や さ し い 言 葉 で 書 い た。<br>
そ し て、 最 後 に こ う 結 ぶ ─<br>
「<strong>風、 風、 吹 く な、 シ ャ ボ ン 玉 飛 ば そ</strong>」<br>
「シ ャ ボ ン 玉 が、 こ わ れ な い で、 飛 ん で 行 け る よ う に。 ど う か、 風 よ、 吹 か な い で」 と。
</p>

<p>
こ れ は、 雨 情 が、 失 っ た 娘 へ の 想 い を、 「失 わ れ な い で あ っ て ほ し い」 「飛 ん で 行 け る な ら 飛 ん で ほ し い」 と い う 願 い に、 静 か に 込 め た 詩 な の だ。
</p>

<h3>歌 と し て の 「シ ャ ボ ン 玉」</h3>

<p>
1923 年 (大 正 12 年)、 作 曲 家 の <strong>中 山 晋 平</strong> (な か や ま し ん ぺ い) が、 こ の 詩 に や さ し い メ ロ デ ィ ー を 付 け た。
</p>

<p>
中 山 晋 平 は、 「て る て る ぼ う ず」 「証 城 寺 の 狸 囃 子」 「肩 た た き」 な ど、 大 正 ~ 昭 和 期 の 多 く の 童 謡 を 作 曲 し た 名 作 曲 家。<br>
彼 が 付 け た 「シ ャ ボ ン 玉」 の 曲 は、 100 年 経 っ た 今 も、 幼 稚 園 や 小 学 校 で 歌 い 継 が れ て い る。
</p>

<h3>大 人 と 子 ど も、 違 う 読 み 方</h3>

<p>
こ の 詩 は、 不 思 議 な 詩 で あ る。<br>
子 ど も が 読 む と ─「シ ャ ボ ン 玉 で 遊 ん だ こ と が あ る」 と い う 楽 し い 思 い 出 が 浮 か ぶ。<br>
大 人 が 読 む と ─「失 わ れ た 命」 「は か な い も の」 が 心 に 響 く。
</p>

<p>
同 じ 詩 が、 読 む 人 に よ っ て、 読 む 時 期 に よ っ て、 違 う 表 情 を 見 せ る。<br>
そ れ こ そ が、 雨 情 が 100 年 前 に 込 め た、 詩 の 力 で あ る。
</p>

<p>
今 度、 シ ャ ボ ン 玉 を 吹 い て、 屋 根 ま で 飛 ぶ の を 見 た ら、 ふ と、 雨 情 の 悲 し み と や さ し さ を、 思 い 出 し て み て ほ し い。
</p>
