<h3>『星 と た ん ぽ ぽ』 ─ 全 文</h3>

<blockquote>
<p>
青 い お 空 の 底 ふ か く<br>
海 の 小 石 の そ の よ う に<br>
夜 が く る ま で 沈 ん で る<br>
昼 の お 星 は め に 見 え ぬ。<br>
見 え ぬ け れ ど も あ る ん だ よ、<br>
見 え ぬ も の で も あ る ん だ よ。
</p>
<p>
散 っ て す が れ た た ん ぽ ぽ の、<br>
瓦 の す き に、 だ ま っ て、<br>
春 の く る ま で か く れ て る、<br>
つ よ い そ の 根 は め に 見 え ぬ。<br>
見 え ぬ け れ ど も あ る ん だ よ、<br>
見 え ぬ も の で も あ る ん だ よ。
</p>
</blockquote>

<h3>昼 の 星 ─ 見 え な い だ け で、 そ こ に あ る</h3>

<p>
昼 の 空 を 見 上 げ て み ま し ょ う。 何 が 見 え ま す か? 太 陽 と 雲 だ け で す ね。 で も、 本 当 は ?</p>

<p>
昼 間 も 夜 と 同 じ よ う に、 何 万 ・ 何 億 の 星 が 空 に あ り ま す。 た だ、 太 陽 が 明 る す ぎ て、
私 た ち の 目 に は 見 え な い だ け な の で す。
</p>

<p>
み す ゞ は こ れ を、 「海 の 小 石 の そ の よ う に / 夜 が く る ま で 沈 ん で る」 と 詠 い ま し た。
夜 に な っ て 太 陽 が 沈 ん だ 時、 沈 ん で い た 星 た ち が「現 れ る」 の で す。
</p>

<h3>た ん ぽ ぽ の 根 ─ 春 を 待 つ 命</h3>

<p>
春 の た ん ぽ ぽ は 黄 色 い 花 を 咲 か せ ま す。 そ の 後、 白 い 綿 毛 (種 子) を 風 に 飛 ば し ま す。
そ し て、 花 ・ 綿 毛 が 散 っ た 後、 た ん ぽ ぽ は ど こ に 行 く で し ょ う か?
</p>

<p>
答 え は ─ 根 の 中 に 隠 れ て、 来 年 の 春 を 待 っ て い る の で す。 た ん ぽ ぽ の 根 は 地 中 数 メ ー ト ル
ま で 伸 び る 強 い 根。 冬 の 間 も 生 き 続 け、 春 に な れ ば ま た 黄 色 い 花 を 咲 か せ ま す。
</p>

<p>
み す ゞ は こ れ も、 「瓦 の す き に、 だ ま っ て、 / 春 の く る ま で か く れ て る」 と 詠 い ま し た。
</p>

<h3>「見 え な い け れ ど、 あ る」 ─ 大 切 な 言 葉</h3>

<p>
詩 の 中 心 に あ る 言 葉 は、 二 度 繰 り 返 さ れ ま す ─
</p>

<blockquote>
<p>
見 え ぬ け れ ど も あ る ん だ よ、<br>
見 え ぬ も の で も あ る ん だ よ。
</p>
</blockquote>

<p>
こ れ は、 単 に 星 や た ん ぽ ぽ の 話 で は あ り ま せ ん。 私 た ち の 周 り に は、 目 に 見 え な い け れ ど
大 切 な も の が た く さ ん あ り ま す。
</p>

<ul>
  <li>空 気 ─ 見 え な い け れ ど、 私 た ち が 生 き る 為 に 必 要</li>
  <li>友 達 の 気 持 ち ─ 見 え な い け れ ど、 大 切 に し た い</li>
  <li>お 父 さ ん ・ お 母 さ ん の 愛 ─ 見 え な い け れ ど、 い つ も そ こ に あ る</li>
  <li>過 去 の 思 い 出 ─ 見 え な い け れ ど、 心 の 中 に い つ ま で も</li>
</ul>

<h3>結 び ─ 見 え な い も の を 大 切 に</h3>

<p>
小 2 の 君 へ ─ 「見 え る も の だ け が 全 て で は な い」 と い う こ と を、 こ の 詩 が 教 え て く れ ま す。
</p>

<p>
給 食 を 食 べ る 時、 君 を 育 て て く れ た 多 く の 人 (農 家 さ ん ・ 漁 師 さ ん ・ 運 ん だ 人 ・ 料 理 し た 人) が
そ こ に い る ─ 見 え な い け れ ど、 確 か に い る の で す。
</p>

<p>
金 子 み す ゞ が 100 年 前 に 詠 ん だ 12 行 の 詩 が、 今 も 君 の 心 に 響 く の は、 こ の 真 理 を シ ン プ ル に
教 え て く れ る か ら で す。
</p>
