<h3>げんだいご に なおすと</h3>

<p>
からたち の 花 が 咲いた よ。白い、白い 花 が 咲いた よ。<br>
からたち の とげ は 痛い よ。青い、青い 針 (はり) のような とげ だ よ。<br>
からたち は、畑 の まわり の かきね に なっている 木 だ よ。わたし が いつも 通る 道 に 生えている よ。<br>
からたち も、秋 に なると、実 (み) が なる よ。まるい、まるい、金色 の たま (球) のような 実 だ よ。<br>
わたし は、からたち の 木 の そば で、泣いた こと が あった。あの とき、まわりの 人 たち は、みんな、みんな、わたし に やさしくして くれた よ。<br>
ほら、また、からたち の 花 が 咲いた よ。白い、白い 花 が、咲いた よ。
</p>

<h3>北原白秋 の こころ</h3>

<p>
北原白秋 (きたはら はくしゅう) は、ふくおか県 やながわ の 大きな 酒屋 の むすこ として 生まれました。<br>
若い とき から 詩 を 書き、20 さい の とき に 上京 して、与謝野鉄幹 (よさの てっかん)・晶子 (あきこ) の 「明星派」 で 活躍 しました。
</p>

<p>
しかし、白秋 の 生活 は 必ずしも 幸せ では ありませんでした。<br>
家業 が 倒産 し、結婚 も うまくいかず、警察 に 拘束 (こうそく) されたこと も ありました。<br>
そんな 苦い 経験 が、彼 の 詩 に は しずか に 影 を 落として います。
</p>

<p>
「からたちの花」 は、白秋 が 少年 の ころ、ふるさと の 道 で よく 見た からたち の 垣根 を 思い出して 書いた 詩 です。
</p>

<ul>
<li><strong>白い 花</strong> ── 子ども 時代 の 美しい 思い出 (やさしさ)</li>
<li><strong>青い 棘</strong> ── 子ども 時代 に 経験 した 痛み や 悲しさ</li>
<li><strong>金 の たま (実)</strong> ── 季節 が 巡って、美しい もの が また 実る こと</li>
<li><strong>「みんな みんな やさしかった」</strong> ── すべて を 振り返って、それでも 「やさしさ」 を 見いだす こころ</li>
</ul>

<p>
小さな からたち の 花 に、白秋 は <strong>「やさしさ」と「痛み」を 同時 に 抱えて 生きる 人 の 姿</strong> を 見ました。<br>
そして、最後 は ── 「みんな、やさしかった」 と、すべて を ゆるす ような こころ で、この 詩 を 結んだ の です。
</p>

<h3>歌 と なって</h3>

<p>
この 詩 は、1924 年 (大正 13 年) に 雑誌 『赤い鳥 (あかいとり)』 で 発表 されました。<br>
そして、翌年 1925 年 (大正 14 年)、作曲家 の <strong>山田耕筰 (やまだ こうさく)</strong> が、しずか で 美しい メロディー を つけました。<br>
白秋 と 耕筰 の コンビ は、「赤とんぼ」 「待ちぼうけ」 「この道」 など、たくさん の 童謡 を 生み 出しました。
</p>

<p>
百年 ちかく まえ の この 歌 を、いま、あなた も 歌って みて ください。<br>
白い 花 と、青い 棘 が、こころ の 中 に やさしく 浮かび 上がって くる でしょう。
</p>
