<h3>げ ん だ い ご に な お す と</h3>

<p>
お ー い、 雲 よ!<br>
ゆ っ た り、 ゆ っ た り と、<br>
ず い ぶ ん の ん き そ う じ ゃ な い か。<br>
ど こ ま で 行 く ん だ い?<br>
ず っ と、 磐 城 平 (い わ き た い ら、 現 在 の 福 島 県 い わ き 市) の ほ う ま で 行 く の か い?
</p>

<h3>解 説 ─ 病 床 か ら の 呼 び か け</h3>

<p>
こ の 詩 は、 詩 人 が 病 床 に い な が ら、 空 を 流 れ る 雲 に 呼 び か け た 一 篇 で あ る。
</p>

<p>
山 村 暮 鳥 (1884–1924) は、 結 核 (け っ か く) と い う 病 に 苦 し み な が ら、 多 く の 詩 を 残 し た。<br>
40 歳 で 亡 く な っ た 暮 鳥 は、 こ の 詩 を 含 む 詩 集 『雲』 を 自 分 の 手 で は 出 版 で き ず、 死 の 翌 年 1925 年、 友 人 た ち の 手 に よ っ て 出 版 さ れ た。
</p>

<p>
病 で 自 由 に 動 け な い 詩 人 が、 窓 か ら 空 を 見 る と ─ <strong>雲 が 悠 々 と 流 れ て い く</strong>。<br>
詩 人 は、 そ の 雲 に 思 わ ず 声 を か け る:「お う い 雲 よ、 馬 鹿 に の ん き そ う じ ゃ な い か」 と。
</p>

<p>
「馬 鹿 に の ん き」 ─ こ れ は、 う ら や ま し さ と、 親 し み と、 そ し て 少 し の 寂 し さ を、 同 時 に 含 ん だ 言 葉 で あ る。<br>
動 け な い 自 分 と、 自 由 に 流 れ る 雲 ─ そ の 対 比 が、 静 か に 浮 か ぶ。
</p>

<h3>「磐 城 平 (い わ き た い ら) の ほ う ま で」</h3>

<p>
「磐 城 平」 と は、 現 在 の 福 島 県 い わ き 市 (旧・平 市) の こ と。<br>
群 馬 県 出 身 の 暮 鳥 が 茨 城 県 で 司 祭 を 勤 め て い た 時 期 か ら 見 て、 北 の 方 に あ る 街 だ。
</p>

<p>
詩 人 は、 雲 が 流 れ て い く 方 向 を 見 て、「ず っ と 磐 城 平 の ほ う ま で 行 く の か」 と 推 測 す る。<br>
こ の 「具 体 的 な 地 名」 が、 詩 に <strong>確 か な 現 実 感</strong> を 与 え て い る。
</p>

<p>
空 想 の 風 景 で は な く、 詩 人 が 実 際 に 窓 か ら 見 た 雲、 実 際 に 知 っ て い る 街。<br>
こ の 「身 近 さ」 が、 詩 を 読 む 私 た ち の 心 に も 響 く。
</p>

<h3>「呼 び か け」 と い う 詩 の 技 法</h3>

<p>
こ の 詩 で 最 も 印 象 的 な の は、 詩 人 が <strong>雲 に 直 接 話 し か け て い る</strong> こ と で あ る。
</p>

<p>
「お う い 雲 よ」<br>
「ど こ ま で ゆ く ん だ」<br>
「ゆ く ん か」
</p>

<p>
ま る で、 友 達 に 声 を か け る よ う な 親 し い 口 調 で、 雲 に 語 り か け る。<br>
こ の 「呼 び か け」 の 技 法 は、 古 代 の 和 歌 か ら 現 代 詩 ま で、 多 く の 詩 で 使 わ れ て き た。
</p>

<p>
こ の 技 法 の 効 果:
</p>

<ul>
<li><strong>親 密 感</strong> ─ 詩 人 と 雲 が 「友 達」 の よ う に 感 じ ら れ る</li>
<li><strong>動 き</strong> ─ 詩 が 静 止 し た 描 写 で は な く、 「対 話」 と い う 動 き を 持 つ</li>
<li><strong>読 者 の 参 加</strong> ─ 読 者 も 同 じ よ う に 雲 に 呼 び か け た く な る</li>
</ul>

<h3>君 も 自 然 に 呼 び か け て み よ う</h3>

<p>
窓 の 外 を 見 て、 雲・木・鳥・川・風 ─ 何 か を 見 つ け た ら、 詩 人 の よ う に 呼 び か け て み よ う。
</p>

<ul>
<li>「お う い 風 よ、 ど こ か ら 来 た ん だ?」</li>
<li>「鳥 さ ん、 ど こ へ 帰 る ん だ い?」</li>
<li>「お い 雨、 そ ろ そ ろ 止 ん で く れ よ」</li>
</ul>

<p>
そ の 一 言 が、 君 自 身 の 「詩」 の 始 ま り か も し れ な い。
</p>

<p>
病 床 の 山 村 暮 鳥 が、 窓 か ら 空 に 「お う い 雲 よ」 と 呼 び か け た 瞬 間 ─ そ こ に、 100 年 後 の 君 に も 残 る 詩 が、 生 ま れ た の だ か ら。
</p>
