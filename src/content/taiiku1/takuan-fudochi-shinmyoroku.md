## 不 動 智 神 妙 録 (抄)

沢 庵 宗 彭 (1573–1645)

### 諸 仏 不 動 智 の 事

仏 経 に 説 く と こ ろ の 諸 仏 の 不 動 智 と は、 心 が 一 切 の 物 事 に よ っ て 動 か さ れ ぬ こ と を い う。 こ の「動 か さ れ ぬ」 と は、 心 を ど こ か 一 点 に 縛 り 付 け て 動 か な い こ と で は な い。 む し ろ、 ど こ に も 縛 ら れ ず、 自 在 に 動 け る 心 の こ と を「不 動 智」 と 呼 ぶ の で あ る。

人 は「心 を ど こ か に 置 こ う」 と し て、 か え っ て そ こ に 縛 ら れ る。 心 を ど こ に も 置 か ぬ こ そ、 真 の 自 由 で あ り、 真 の 不 動 で あ る。

### 心 ど こ に か 置 か ん の 問

太 刀 を 持 ち、 敵 と 向 か い 合 っ た と き、 心 を ど こ に 置 け ば よ い か。

敵 の 太 刀 に 心 を 置 け ば、 敵 の 太 刀 に 心 が 奪 わ れ る。 敵 の 体 に 置 け ば、 敵 の 体 に 心 が 奪 わ れ る。 敵 の 顔 に 置 け ば、 敵 の 顔 に 心 が 奪 わ れ る。

で は 自 分 の 太 刀 に 置 け ば。 や は り 自 分 の 太 刀 に 心 が 縛 ら れ、 自 由 を 失 う。 自 分 の 構 え に 置 け ば、 構 え に 縛 ら れ る。 自 分 の 足 に 置 け ば、 足 に 縛 ら れ る。

で は、 心 を ど こ に 置 け ば よ い か。 ─ ど こ に も 置 か ぬ。

ど こ に も 置 か ぬ と は、 心 を 一 点 に 留 め ず、 全 体 に 行 き 渡 ら せ る と い う こ と で あ る。 太 刀 に も 自 分 に も 敵 に も 場 に も、 全 体 に 心 が あ り、 同 時 に ど こ に も 縛 ら れ な い。 こ れ が、 兵 法 の 極 意 で あ り、 同 時 に 禅 の 極 意 で あ る。

### 平 常 心 是 道

「平 常 心 こ そ 道 な り」 と 古 人 は 言 う。 兵 法 の 修 行 と は、 平 常 心 を 養 う こ と で あ る。

平 常 心 と は、 試 合 場 で「上 が ら ぬ」 心 の こ と。 敵 を 前 に し て も、 普 段 の 稽 古 場 で の 心 と 変 わ ら ぬ 心 の こ と。

普 段 の 稽 古 で 心 を 練 ら ね ば、 試 合 で 心 は 上 が る。 普 段 の 稽 古 で 心 を 練 れ ば、 試 合 場 で も 動 じ な い。 兵 法 の 修 行 と は、 こ の 「心 を 練 る」 こ と に 他 な ら な い。

### 結 び

剣 を 振 る う と は、 結 局、 心 を 整 え る こ と で あ る。 心 が 整 え ば、 剣 は 自 ず か ら 整 う。 心 が 乱 れ れ ば、 剣 も 乱 れ る。

禅 と 剣 は 一 つ で あ る。 こ れ を 私 は「不 動 智」 と 呼 ぶ。
