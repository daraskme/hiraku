<h3>日 本 最 古 の 相 撲 ─ 1300 年 前 の 力 比 べ</h3>

<p>
720 年、 奈 良 の 都 で、 日 本 最 古 の 正 史 『日 本 書 紀』 が 完 成 し た。 全 30 巻 ・
漢 文 で 書 か れ た 大 著 の 中 に、 一 つ 興 味 深 い 記 述 が あ る。
</p>

<p>
垂 仁 天 皇 七 年 七 月 ─ 「天 下 第 一 の 力 士」 を 自 称 す る 当 麻 蹶 速 (た い ま の け は や)
が 大 和 の 当 麻 邑 (現 奈 良 県 葛 城 市) に い た。 彼 は 言 う ─ 「我 が 力 に 勝 つ 者 が
ど こ に 居 ろ う か」。
</p>

<p>
こ れ を 聞 い た 垂 仁 天 皇 は、 群 臣 に 問 う た。 一 人 が 答 え る ─ 「出 雲 国 (現 島 根 県) に
野 見 宿 禰 (の み の す く ね) と い う 力 士 が お り ま す。 こ の 者 を 召 し 出 し て 勝 負 さ せ て は
い か が」。
</p>

<h3>野 見 宿 禰 ─ 出 雲 か ら 大 和 へ</h3>

<p>
野 見 宿 禰 は 出 雲 か ら 大 和 へ 召 さ れ、 天 皇 の 前 で 当 麻 蹶 速 と 力 比 べ を し た。
『日 本 書 紀』 は こ う 記 す ─
</p>

<blockquote>
<p>
「乃 (す な わ) ち 二 人 相 対 し て 立 ち、 各 (お の お の) 足 を 挙 げ て 相 蹶 (け) れ り。
則 (す な わ) ち 当 麻 蹶 速 の 脇 骨 を 蹈 (ふ) み 折 り て、 復 (ま た) 其 (そ の) 腰 を 蹈 み
折 り て 殺 し つ。」
</p>
</blockquote>

<p>
─ 二 人 は 向 か い 合 っ て 立 ち、 互 い に 足 で 蹴 り 合 っ た。 野 見 宿 禰 は 当 麻 蹶 速 の あ ば ら
骨 を 蹴 り 折 り、 さ ら に 腰 を 折 っ て 殺 し た ─ と。
</p>

<p>
現 代 の 相 撲 と は 違 い、 蹴 り 技 を 含 む 命 が け の 力 比 べ だ っ た こ と が わ か る。 こ れ が
日 本 最 古 の 相 撲 の 記 録 と さ れ、 後 の 相 撲 の 起 源 と 仰 が れ る。
</p>

<h3>埴 輪 の 始 祖 ─ 殉 死 を 廃 止 し た 野 見 宿 禰</h3>

<p>
野 見 宿 禰 は 力 だ け で な く、 知 恵 で も 優 れ て い た。 当 時、 天 皇 や 貴 人 が 亡 く な る と、
家 臣 が 共 に 殉 死 す る 風 習 が あ っ た。
</p>

<p>
垂 仁 天 皇 の 皇 后 が 亡 く な っ た 時、 野 見 宿 禰 は 提 案 す る ─ 「人 の 代 わ り に、 土 で 人 や 馬
を 作 っ て 古 墳 の 周 り に 並 べ れ ば 良 い」。 こ う し て 生 ま れ た の が 埴 輪 (は に わ) で あ り、
野 見 宿 禰 は 「埴 輪 の 始 祖」 と も 呼 ば れ る。
</p>

<h3>相 撲 ─ 神 事 か ら ス ポ ー ツ へ</h3>

<p>
古 代 の 相 撲 は、 神 へ の 奉 納 行 事 (神 事) と し て 行 わ れ た。 神 社 の 境 内 で、 五 穀 豊 穣 を
祈 っ て 力 士 が 取 り 組 む。 現 代 で も、 大 相 撲 の 土 俵 入 り で 四 股 (し こ) を 踏 む の は、
大 地 の 邪 気 を 祓 う 神 事 の 名 残 で あ る。
</p>

<p>
平 安 期 に は 朝 廷 の「節 会 相 撲」(せ ち え ず も う)、 鎌 倉 期 に は 武 士 の 武 術 と し て、
江 戸 期 に は 興 行 と し て 発 達。 そ し て 明 治 期 以 降、 国 技 ・ 大 相 撲 と し て 制 度 化 さ れ た。
</p>

<h3>武 道 文 化 ─ 「礼 に 始 ま り 礼 に 終 わ る」</h3>

<p>
相 撲・柔 道・剣 道・空 手 ─ 日 本 の 武 道 に は 共 通 の 哲 学 が あ る。 「礼 に 始 ま り 礼 に
終 わ る」 ─ 相 手 へ の 敬 意 が 全 て の 基 本 で あ る。
</p>

<p>
野 見 宿 禰 が 当 麻 蹶 速 を 倒 し た 物 語 は、 力 比 べ の 記 録 で あ る と 同 時 に、 武 道 の 起 源 と
礼 法 の 大 切 さ を 教 え る 物 語 で も あ る。
</p>

<h3>結 び ─ 1300 年 前 か ら 続 く 文 化</h3>

<p>
『日 本 書 紀』 か ら 大 相 撲 ま で、 1300 年 以 上 続 く 相 撲 の 歴 史。 国 技 と し て、 神 事 と し て、
そ し て 武 道 文 化 と し て ─ 君 が 学 ぶ 体 育 の 授 業 の 背 後 に も、 こ の 長 い 伝 統 が あ る こ と を
知 っ て お こ う。
</p>
