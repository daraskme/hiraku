<h3>60 余 度 の 勝 負、 一 度 も 負 け ず</h3>

<p>
1584 年 (諸 説 あ り)、 武 蔵 は 播 磨 か 美 作 に 生 ま れ る。 父 も 兵 法 家 だ っ た。
13 歳 の 時、 初 め て の 真 剣 勝 負 で 新 当 流 の 有 馬 喜 兵 衛 を 倒 し て 以 来、 60 余 度 の
勝 負 で 一 度 も 負 け な か っ た と『五 輪 書』 冒 頭 で 自 ら 記 し て い る。
</p>

<p>
29 歳 の 慶 長 17 年 (1612)、 関 門 海 峡 の 巌 流 島 (船 島) で 行 っ た 佐 々 木 小 次 郎 戦 が
最 も 有 名 だ。 約 束 の 時 間 に わ ざ と 遅 れ、 漕 ぎ な が ら 削 っ た 櫂 (か い) の 木 刀 で
小 次 郎 を 倒 し た と 伝 え ら れ る。
</p>

<h3>二 天 一 流 ─ 二 つ の 太 刀 を 同 時 に 持 つ</h3>

<p>
武 蔵 が 開 い た 流 派 「二 天 一 流」 は、 太 刀 と 脇 差 を 同 時 に 持 つ 「二 刀 流」 が 特 徴 だ。
普 通 の 剣 士 が 一 本 で 戦 う 中、 武 蔵 は 二 本 同 時 に 操 る 革 命 的 な 技 法 を 編 み 出 し た。
</p>

<p>
武 蔵 は 言 う ─ 「人 は 二 本 の 手 を 持 つ。 な ぜ 一 本 し か 使 わ な い の か」。 常 識 を 疑 い、
本 質 か ら 考 え 直 す ─ こ れ が 武 蔵 の 武 道 で あ る。
</p>

<h3>『五 輪 書』 ─ 死 の 直 前 に 残 し た 哲 学 書</h3>

<p>
晩 年 の 武 蔵 は、 細 川 家 (熊 本) に 客 分 と し て 招 か れ た。 寛 永 17 年 (1640)、 56 歳。
熊 本 城 下 で 5 年 過 ご し、 死 の 1–2 ヶ 月 前 (正 保 2 年・1645) に 霊 巌 洞 (れ い が ん ど う)
と い う 山 中 の 洞 窟 で『五 輪 書』 を 完 成 さ せ た。
</p>

<p>
五 巻 構 成 ─
</p>
<ul>
  <li><strong>地 之 巻</strong> ─ 兵 法 の 基 礎 (大 工 の 道 の 喩 え)</li>
  <li><strong>水 之 巻</strong> ─ 技 法 (太 刀 の 持 ち 方 ・ 構 え 方)</li>
  <li><strong>火 之 巻</strong> ─ 戦 い 方 (実 戦 の 心 構 え)</li>
  <li><strong>風 之 巻</strong> ─ 他 流 派 へ の 批 評</li>
  <li><strong>空 之 巻</strong> ─ 悟 り (兵 法 の 究 極)</li>
</ul>

<h3>「兵 法 の 道 は 大 工 の 道 に 似 る」</h3>

<p>
地 之 巻 に、 有 名 な 一 節 が あ る ─ 「兵 法 の 道 を 一 道 と 心 得 て、 大 工 の 道 に 寄 す」。
武 道 と 大 工 仕 事 は、 一 見 違 う が、 共 通 の 原 理 で 動 い て い る。
</p>

<p>
棟 梁 (大 工 の リ ー ダ ー) は ─
</p>
<ul>
  <li>建 物 全 体 の 設 計 を 把 握 す る</li>
  <li>木 の 性 質 を 見 極 め、 適 材 適 所 に 配 す る</li>
  <li>道 具 を 大 切 に 手 入 れ す る</li>
  <li>弟 子 の 力 量 に 応 じ て 仕 事 を 与 え る</li>
</ul>

<p>
武 道 で も 同 じ ─ 全 体 を 見、 人 を 活 か し、 道 具 (武 器) を 大 切 に し、 自 ら 修 練 す る。
こ の 原 理 は、 仕 事・部 活・勉 強 ─ 君 が 取 り 組 む 全 て に 通 じ る。
</p>

<h3>「空」 ─ 悟 り の 境 地</h3>

<p>
最 終 巻「空 之 巻」 は、 短 い が 最 も 深 い。 武 蔵 は 言 う ─ 「兵 法 の 道 を 究 め れ ば、
何 物 も 無 い『空』 の 境 地 に 至 る」。
</p>

<p>
真 の 強 さ と は、 力 や 技 で は な く、 心 が 「何 物 に も 動 じ な い」 状 態。 強 い 風 が 吹 い て も
動 か な い 大 木 の よ う に、 戦 い の 中 で も 心 が 澄 ん で い る ─ こ れ が 武 蔵 の 言 う 「空」 で あ る。
</p>

<h3>世 界 が 読 む『五 輪 書』</h3>

<p>
『五 輪 書』 は 現 代、 世 界 中 で 読 ま れ て い る。 ア メ リ カ で は ハ ー バ ー ド 大 学 の M B A
で 経 営 学 の 古 典 と し て 学 ば れ、 ス ポ ー ツ ・ 軍 事 ・ ビ ジ ネ ス 戦 略 書 と し て も 翻 訳 さ れ て い る。
</p>

<p>
400 年 前 の 剣 豪 が 残 し た 哲 学 が、 国 境 と 時 代 を 超 え て 生 き 続 け て い る ─ そ れ が
日 本 武 道 思 想 の 普 遍 性 で あ る。
</p>

<h3>結 び ─ 君 の「兵 法」 を 探 そ う</h3>

<p>
武 蔵 が 教 え る の は、 単 な る 剣 術 で は な い。 「何 か 一 つ の 道 を 究 め よ。 そ う す れ ば、
全 て の 道 に 通 ず る」 ─ こ の 言 葉 を 心 に 留 め て、 君 が 打 ち 込 む 一 つ の こ と
(部 活・勉 強・趣 味) を 究 め て み よ う。
</p>
