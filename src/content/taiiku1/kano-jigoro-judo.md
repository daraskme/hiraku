<h3>22 歳 で 柔 道 を 創 始</h3>

<p>
1882 年 (明 治 15 年)、 東 京 ・ 永 昌 寺 (し ょ う じ) の 一 室。<br>
22 歳 の 一 人 の 青 年 が、 数 人 の 弟 子 と と も に 「<strong>講 道 館</strong>」(こ う ど う か ん) と い う 道 場 を 始 め た。
</p>

<p>
こ の 青 年 が ─ <strong>嘉 納 治 五 郎</strong> (か の う じ ご ろ う、 1860–1938)。 後 に 「日 本 体 育 の 父」「柔 道 の 父」 と 呼 ば れ る 人 物 で あ る。
</p>

<h3>嘉 納 治 五 郎 の 生 涯</h3>

<p>
1860 年、 兵 庫 県 御 影 (現 神 戸 市) の 酒 造 家 に 生 ま れ る。<br>
東 京 帝 国 大 学 文 学 部 で 政 治 ・ 哲 学 を 学 び、 1881 年 (21 歳) 卒 業。
</p>

<p>
学 生 時 代 か ら、 嘉 納 は 江 戸 時 代 か ら 続 く「<strong>柔 術</strong>」(じ ゅ う じ ゅ つ) を 学 ん で い た。<br>
天 神 真 楊 流 (て ん じ ん し ん よ う り ゅ う)、 起 倒 流 (き と う り ゅ う) な ど 複 数 の 流 派。
</p>

<p>
だ が 彼 は こ う 感 じ た:
</p>

<blockquote>
「柔 術 は 素 晴 ら し い 技 を 持 っ て い る。 だ が、 流 派 ご と に バ ラ バ ラ で、 危 険 な 技 も 多 く、 教 育 体 系 が な い。<br>
こ れ を <strong>近 代 的 な 体 育・教 育</strong> と し て 再 構 築 で き な い か」 と。
</blockquote>

<p>
1882 年、 大 学 卒 業 翌 年、 嘉 納 は 講 道 館 を 創 設 し、 こ の 仕 事 に 取 り 組 ん だ。
</p>

<h3>柔 術 か ら 柔 道 へ</h3>

<p>
嘉 納 が 柔 術 か ら 柔 道 へ の 改 革 で 行 っ た こ と:
</p>

<ol>
<li><strong>危 険 な 技 の 除 外</strong> ─ 関 節 技・当 て 身・打 撃 技 を 競 技 か ら 外 し、 「投 げ 技」 と「寝 技」 中 心 に</li>
<li><strong>安 全 な 倒 れ 方 (受 け 身)</strong> ─ 怪 我 を 防 ぐ 訓 練 を 体 系 化</li>
<li><strong>段 級 制 度</strong> ─ 初 段 → 二 段… の 階 級 と、 色 帯 (白・茶・黒) で 達 成 度 を 可 視 化</li>
<li><strong>礼 の 重 視</strong> ─ 「礼 に 始 ま り 礼 に 終 わ る」 ─ 試 合 の 前 後 で 必 ず 礼</li>
<li><strong>競 技 規 則</strong> ─ 公 平 で 明 確 な ル ー ル と 審 判 制 度</li>
</ol>

<p>
こ う し て、 江 戸 の「武 術」 が、 近 代 の「ス ポ ー ツ・教 育」 へ と 生 ま れ 変 わ っ た。
</p>

<h3>「精 力 善 用 ・ 自 他 共 栄」</h3>

<p>
嘉 納 柔 道 の 核 心 は、 二 つ の 原 理 に 凝 縮 さ れ る。
</p>

<h4>① 精 力 善 用 (せ い り ょ く ぜ ん よ う)</h4>

<blockquote>
「自 分 の 持 て る 力 を、 善 い こ と の た め に、 最 大 限 効 率 的 に 用 い る」
</blockquote>

<p>
柔 道 の 技 は、 「<strong>柔 よ く 剛 を 制 す</strong>」 ─ 力 だ け で な く、 相 手 の 力 を 利 用 し て 投 げ る。<br>
こ の 「効 率」 「工 夫」 「知 恵」 が、 単 な る 武 力 を 超 え た 教 育 価 値 を 持 つ と 嘉 納 は 説 い た。
</p>

<h4>② 自 他 共 栄 (じ た き ょ う え い)</h4>

<blockquote>
「自 分 と 相 手 が、 と も に 栄 え る (高 ま る) こ と を 目 指 す」
</blockquote>

<p>
柔 道 は「相 手 を 倒 す」 ス ポ ー ツ だ が、 嘉 納 は そ の 目 的 を 「<strong>共 に 成 長 す る こ と</strong>」 に 置 い た。<br>
道 場 で の 試 合 を 通 し て、 自 分 も 相 手 も 心 身 と も に 高 ま る ─ そ れ こ そ が 柔 道 の 真 の 目 的、 と。
</p>

<p>
こ の「自 他 共 栄」 の 思 想 は、 後 に 国 際 親 善・平 和 思 想 と 結 び つ い て、 オ リ ン ピ ッ ク と 親 和 し て い く。
</p>

<h3>「日 本 体 育 の 父」 ─ そ の 他 の 業 績</h3>

<p>
嘉 納 は 柔 道 だ け の 人 で は な か っ た。 教 育 者 と し て も 巨 人 だ っ た:
</p>

<ul>
<li>1882 ─ 講 道 館 創 設、 同 時 に 学 習 院 教 授</li>
<li>1893 ─ 第 五 高 等 学 校 (熊 本) 校 長 (33 歳)</li>
<li>1893 ─ 東 京 高 等 師 範 学 校 (現 筑 波 大) 校 長 ─ 26 年 間 (90 歳 ま で 連 続 で は な く、 数 度 に 渡 り 校 長 を 務 め た)</li>
<li>1909 ─ ア ジ ア 初 の IOC 委 員 に</li>
<li>1912 ─ 日 本 の 初 オ リ ン ピ ッ ク 参 加 (ス ト ッ ク ホ ル ム) を 実 現</li>
<li>1938 ─ 1940 年 東 京 オ リ ン ピ ッ ク 招 致 に 成 功</li>
</ul>

<p>
嘉 納 は ま た、 中 国 ・ 留 学 生 (魯 迅 ・ 周 恩 来 ら) を 教 え た 経 験 も あ る。<br>
彼 の 弟 子 か ら、 中 国 共 産 党 初 期 の 指 導 者 が 多 数 育 っ た。<br>
日 本 と ア ジ ア の 近 代 化 を、 教 育 を 通 し て 支 え た 巨 人 で あ っ た。
</p>

<h3>1940 → 1964 ─ 念 願 の 東 京 オ リ ン ピ ッ ク</h3>

<p>
嘉 納 の 最 後 の 大 仕 事 は、 「東 京 オ リ ン ピ ッ ク 招 致」 だ っ た。
</p>

<p>
1938 年、 78 歳 の 嘉 納 は、 カ イ ロ で 開 か れ た IOC 総 会 で 1940 年 東 京 オ リ ン ピ ッ ク 招 致 を 成 功 さ せ た。<br>
だ が ─ そ の 帰 国 の 船 中、 1938 年 5 月 4 日、 太 平 洋 上 で 客 死。 78 歳。
</p>

<p>
そ し て 1940 年 東 京 大 会 自 体 も、 日 中 戦 争 の 激 化 で <strong>中 止</strong> に な っ た。
</p>

<p>
嘉 納 の 念 願 は ─ 26 年 を 経 て、 1964 年 東 京 オ リ ン ピ ッ ク で 実 現 し た。<br>
そ し て こ の 1964 年 大 会 で、 <strong>柔 道 が オ リ ン ピ ッ ク 正 式 競 技 と な っ た</strong>。
</p>

<p>
22 歳 の 嘉 納 が 永 昌 寺 の 一 室 で 始 め た 講 道 館 が、 82 年 を 経 て、 世 界 の 国 際 ス ポ ー ツ と な っ た 瞬 間 で あ る。
</p>

<h3>柔 道 の 現 在</h3>

<p>
今 や 柔 道 は、 200 以 上 の 国・地 域 で 約 2000 万 人 が 行 う 国 際 ス ポ ー ツ。<br>
特 に フ ラ ン ス で は 「柔 道 人 口」 が 日 本 よ り も 多 い (約 56 万 人 vs 約 16 万 人) と 言 わ れ る。
</p>

<p>
オ リ ン ピ ッ ク で は、 1964 年 か ら 一 度 (1968 メ キ シ コ) を 除 い て 毎 回 実 施。<br>
そ し て 「精 力 善 用 ・ 自 他 共 栄」 の 思 想 は、 国 際 試 合 で の 「相 手 を 尊 重 す る 礼 」 と し て、 今 も 生 き 続 け て い る。
</p>

<h3>君 へ の メ ッ セ ー ジ</h3>

<p>
嘉 納 治 五 郎 が 144 年 前 に 始 め た 柔 道 が、 今 や 世 界 中 の 子 ど も た ち の 体 育・教 育 を 支 え て い る。
</p>

<p>
そ の 根 底 に あ る「<strong>精 力 善 用 ・ 自 他 共 栄</strong>」 ─ 自 分 の 力 を 善 い こ と に 使 い、 自 分 と 相 手 が と も に 高 ま る ─ は、 ス ポ ー ツ だ け で な く、 君 の 人 生 の あ ら ゆ る 場 面 で 通 用 す る 哲 学 で あ る。
</p>

<p>
22 歳 で 何 か を 始 め よ う と し て い る 若 い 嘉 納 の 姿 を、 心 に 留 め て お い て ほ し い。
</p>
