## ク ー ベ ル タ ン と 近 代 オ リ ン ピ ッ ク

### 1. 途 絶 え た 古 代 の 祭 典

古 代 ギ リ シ ア に は、 紀 元 前 776 年 か ら 約 1200 年 も の 間、 4 年 ご と に 開 か れ る 大 き な 競 技 祭 が あ っ た。 「古 代 オ リ ン ピ ッ ク」 で あ る。

ギ リ シ ア 各 地 の 都 市 国 家 か ら 選 手 が 集 ま り、 か け っ こ・レ ス リ ン グ・戦 車 競 走 な ど で 競 っ た。 大 会 の 期 間 中 は、 都 市 同 士 が 戦 争 を し て い て も 休 戦 す る「オ リ ン ピ ッ ク 休 戦」 が 守 ら れ た。 ス ポ ー ツ が、 平 和 を つ く り 出 し て い た の だ。

し か し、 紀 元 393 年、 ロ ー マ 皇 帝 テ オ ド シ ウ ス が「異 教 の 祭 り」 と し て 禁 止 し、 オ リ ン ピ ッ ク は 途 絶 え た。 そ れ か ら、 約 1500 年 ─ 。

***

### 2. 一 人 の フ ラ ン ス 人 の 夢

19 世 紀 末、 一 人 の フ ラ ン ス 人 が、 こ の 古 代 の 祭 典 に 心 を 奪 わ れ た。 ピ エ ー ル・ド・ク ー ベ ル タ ン (1863-1937) で あ る。

貴 族 の 家 に 生 ま れ た ク ー ベ ル タ ン は、 教 育 に 強 い 関 心 を 持 っ て い た。 彼 は イ ギ リ ス の 学 校 を 訪 れ、 そ こ で「ス ポ ー ツ が 青 少 年 の 教 育 の 中 心 に な っ て い る」 こ と に 感 銘 を 受 け た。 体 を 鍛 え、 ル ー ル を 守 り、 仲 間 と 協 力 し、 フ ェ ア に 競 う ─ ス ポ ー ツ は、 立 派 な 人 間 を 育 て る。

ク ー ベ ル タ ン は 考 え た。

「ス ポ ー ツ に は、 青 少 年 を 教 育 し、 さ ら に は 世 界 を 平 和 に す る 力 が あ る。 古 代 ギ リ シ ア の オ リ ン ピ ッ ク を、 現 代 に 復 活 さ せ よ う。 世 界 中 の 若 者 が 集 ま り、 フ ェ ア に 競 い、 友 情 を 育 む 場 を つ く ろ う。」

***

### 3. 1894 年 IOC、 1896 年 ア テ ネ

1894 年、 ク ー ベ ル タ ン は パ リ の ソ ル ボ ン ヌ 大 学 で 国 際 会 議 を 開 き、 「国 際 オ リ ン ピ ッ ク 委 員 会 (IOC)」 を 設 立 し た。 そ し て、 オ リ ン ピ ッ ク の 復 活 を 正 式 に 決 議 し た。

2 年 後 の 1896 年、 オ リ ン ピ ッ ク 発 祥 の 地 ギ リ シ ア の 首 都 ア テ ネ で、 第 1 回 近 代 オ リ ン ピ ッ ク が 開 催 さ れ た。 14 か 国 か ら 約 240 人 の 選 手 が 参 加 (当 時 は 男 子 の み)。 陸 上・水 泳・体 操・レ ス リ ン グ・フ ェ ン シ ン グ な ど が 行 わ れ た。

約 1500 年 ぶ り に、 オ リ ン ピ ッ ク が よ み が え っ た 瞬 間 で あ っ た。

***

### 4. オ リ ン ピ ズ ム ─ 競 技 を 超 え た 理 念

ク ー ベ ル タ ン に と っ て、 オ リ ン ピ ッ ク は 単 な る ス ポ ー ツ 大 会 で は な か っ た。 彼 が か か げ た 理 念 を「オ リ ン ピ ズ ム」 と 呼 ぶ。

「ス ポ ー ツ を 通 じ て 心 と 体 を 鍛 え、 友 情・連 帯・フ ェ ア プ レ ー の 精 神 を 育 て、 世 界 平 和 に 貢 献 す る。」

ク ー ベ ル タ ン が 引 用 し て 広 め た 有 名 な 言 葉 が あ る。

「オ リ ン ピ ッ ク で 重 要 な の は、 勝 つ こ と で は な く、 参 加 す る こ と で あ る。 人 生 で 大 切 な の は、 成 功 す る こ と で は な く、 努 力 す る こ と で あ る。」

(こ の 言 葉 は、 1908 年 ロ ン ド ン 大 会 の 際 に 主 教 タ ル ボ ッ ト が 述 べ た も の を、 ク ー ベ ル タ ン が 引 用 し た も の。)

***

### 5. 五 つ の 輪

ク ー ベ ル タ ン は 1913 年、 オ リ ン ピ ッ ク の シ ン ボ ル「五 輪 マ ー ク」 を 考 案 し た。

5 つ の 輪 は、 五 大 陸 (ヨ ー ロ ッ パ・ア ジ ア・ア フ リ カ・オ セ ア ニ ア・ア メ リ カ) を 表 し、 そ れ ら が つ な が る こ と で「世 界 の 結 び つ き と 連 帯」 を 示 す。 青・黄・黒・緑・赤 の 5 色 と 白 い 地 を 合 わ せ る と、 世 界 中 の 国 旗 の ほ と ん ど の 色 を 表 せ る よ う に 工 夫 さ れ て い る。

「世 界 は ひ と つ」 ─ こ の 願 い が、 五 つ の 輪 に こ め ら れ て い る。

***

### 6. 日 本 と ク ー ベ ル タ ン ─ 嘉 納 治 五 郎

日 本 が オ リ ン ピ ッ ク に 初 め て 参 加 し た の は、 1912 年 の ス ト ッ ク ホ ル ム 大 会 で あ る。 そ の 中 心 と な っ た の が、 柔 道 の 創 始 者・嘉 納 治 五 郎 (か の う じ ご ろ う) で あ っ た。

嘉 納 は 1909 年、 ア ジ ア 初 の IOC 委 員 に 就 任。 ク ー ベ ル タ ン の「ス ポ ー ツ に よ る 教 育」 と い う 理 念 に 深 く 共 鳴 し た。 嘉 納 が 柔 道 で 説 い た「精 力 善 用・自 他 共 栄」 の 精 神 は、 ク ー ベ ル タ ン の オ リ ン ピ ズ ム と 響 き 合 う も の で あ っ た。

二 人 は、 国 は 違 え ど も、 「ス ポ ー ツ は 人 を 育 て、 世 界 を つ な ぐ」 と い う 同 じ 夢 を 見 て い た の だ。

***

### 7. 受 け 継 が れ る 理 念

ク ー ベ ル タ ン は 1937 年 に 亡 く な っ た。 し か し、 彼 が 復 活 さ せ た オ リ ン ピ ッ ク は、 二 度 の 世 界 大 戦 に よ る 中 断 を 乗 り 越 え、 今 で は 夏・冬 合 わ せ て 4 年 に 一 度、 世 界 中 の 何 千 人 も の 選 手 が 集 う、 地 球 最 大 の ス ポ ー ツ の 祭 典 と な っ た。

そ し て、 現 在 で は、 障 が い の あ る 選 手 が 競 う「パ ラ リ ン ピ ッ ク」 も 並 ん で 開 か れ、 オ リ ン ピ ズ ム は さ ら に 広 が っ て い る。

「ス ポ ー ツ に は、 世 界 を 平 和 に す る 力 が あ る」 ─ 130 年 前 の 一 人 の 教 育 者 の 夢 は、 今 も 世 界 の 中 で 生 き 続 け て い る。
