## 正 岡 子 規 と ベ ー ス ボ ー ル

### 1. 病 床 で の 野 球 へ の 思 い

明 治 29 年 (1896 年) 4 月 か ら 7 月 に か け て、 新 聞『日 本』 で 連 載 さ れ た 随 筆『松 蘿 玉 液』 の 一 節 に、 こ ん な 文 章 が あ る。

「ベ ー ス ボ ー ル と い ふ も の は、 米 国 よ り 来 し ス ポ ー ル に し て、 二 隊 に 分 れ て 球 を 打 ち、 三 角 形 の 走 路 を 巡 り、 本 塁 に 還 る を 一 点 と す る も の な り。」

書 い た の は、 当 時 29 歳 の 正 岡 子 規 (1867-1902)。 帝 国 大 学 で の 学 生 時 代 に ベ ー ス ボ ー ル に 熱 中 し た、 結 核 で 病 床 に 伏 し て い た 俳 人 で あ る。

***

### 2. 「野 球 (の ぼ ー る)」 と い う 雅 号

子 規 が 帝 国 大 学 で ベ ー ス ボ ー ル を プ レ ー し て い た 1889-1890 年 頃、 彼 は 自 ら の 雅 号 と し て「野 球 (の ぼ ー る)」 を 名 乗 っ て い た。

こ れ は、 本 名「常 規 (つ ね の り、 通 称 升 = の ぼ る)」 と、 英 語 の「ball」 を 組 み 合 わ せ た 言 葉 遊 び だ。 「升 (の ぼ る) → の (ぼ ー る) → 野 球」 と い う 訳。 子 規 の ベ ー ス ボ ー ル へ の 熱 中 ぶ り が 伝 わ る 名 前 で あ る。

(よ く 「正 岡 子 規 が『野 球』 と い う 訳 語 を 作 っ た」 と 言 わ れ る が、 こ れ は 不 正 確。 「野 球 (や き ゅ う)」 と い う 訳 語 を 提 案 し た の は 1894 年 の 中 馬 庚 (ち ゅ う ま ん か な え) で あ り、 子 規 の 雅 号「の ぼ ー る」 と は 同 じ 漢 字 で も 読 み が 異 な る。)

***

### 3. 1888 年 の 喀 血 と 大 学 中 退

子 規 の ベ ー ス ボ ー ル 熱 中 と 病 と の 闘 い は、 ほ ぼ 同 時 期 に 始 ま っ た。

1888 年 (21 歳)、 子 規 は 帝 国 大 学 の 寮 で 喀 血 (か っ け つ、 結 核 性 の 血 を 吐 く 症 状) を 起 こ し、 結 核 と 診 断 さ れ た。 「血 を 吐 い て も 鳴 き 続 け る 鳥」 で あ る ホ ト ト ギ ス (= 子 規) を 雅 号 と し、 「子 規」 と 自 ら を 呼 ぶ よ う に な っ た。

し か し 子 規 は 病 を 押 し て ベ ー ス ボ ー ル を 続 け た。 そ し て、 体 が 動 か な く な っ た 1896 年 以 降、 病 床 か ら「ベ ー ス ボ ー ル」 を 文 字 で 蘇 ら せ る 仕 事 を 始 め た。

***

### 4. 「打 者」「走 者」「飛 球」「直 球」

『松 蘿 玉 液』 内「ベ ー ス ボ ー ル」 の 章 で、 子 規 は ベ ー ス ボ ー ル の ル ー ル を 詳 細 に 解 説 し、 米 国 用 語 の そ れ ぞ れ を 漢 語 と し て 訳 出 し た。

- **打 者** (バ ッ タ ー)
- **走 者** (ラ ン ナ ー)
- **四 球** (フ ォ ア ボ ー ル)
- **死 球** (デ ッ ド ボ ー ル)
- **飛 球** (フ ラ イ)
- **直 球** (ス ト レ ー ト)
- **本 塁** (ホ ー ム ベ ー ス)
- **三 振** ・ **空 振 り**

こ れ ら の 用 語 は、 単 な る 翻 訳 と い う よ り「漢 語 と し て の 創 作」 に 近 い。 そ し て、 こ の う ち 多 く が 130 年 後 の 現 代 で も 標 準 用 語 と し て 使 わ れ て い る。

中 学 生 の あ な た が 体 育 の 授 業 で「打 者」「走 者」 と 言 う と き、 そ れ は 1896 年 の 病 床 で 29 歳 の 子 規 が 苦 し み な が ら 訳 出 し た 言 葉 で あ る。

***

### 5. 1902 年 9 月 ─ 35 歳 の 死

子 規 は そ の 後 も 病 床 で 短 歌・俳 句 の 革 新 を 進 め、『歌 よ み に 与 ふ る 書』『獺 祭 書 屋 俳 話』 等 の 重 要 著 作 を 残 し た。

し か し 1902 年 9 月 19 日、 結 核 が 全 身 に 広 が り、 子 規 は 病 床 で 没 し た。 享 年 35 歳。

そ の 短 い 生 涯 で、 子 規 は

- 短 歌・俳 句 の 近 代 化
- 写 生 と い う 概 念 の 確 立
- ベ ー ス ボ ー ル 用 語 の 日 本 語 化

と い う、 そ れ ぞ れ が 一 人 の 仕 事 と し て 充 分 な 業 績 を、 一 人 で 同 時 に 成 し 遂 げ た。

***

### 6. 2002 年 ─ 没 後 100 年 の 野 球 殿 堂 入 り

子 規 没 後 100 年 の 2002 年 12 月、 正 岡 子 規 は 日 本 野 球 殿 堂 入 り を 果 た し た。 「特 別 表 彰」 と し て、 ベ ー ス ボ ー ル の 日 本 紹 介 と 用 語 翻 訳 の 功 績 を 称 え る も の で あ っ た。

俳 人 が 野 球 殿 堂 入 り す る と い う、 一 見 奇 妙 な 出 来 事 で あ る。 し か し、 子 規 の 短 い 生 涯 を 見 れ ば、 そ れ は 当 然 で あ っ た。

文 学 と ス ポ ー ツ。 翻 訳 と 文 化。 病 床 で の 言 葉 と、 校 庭 で の 走 り 込 み。

正 岡 子 規 の 生 涯 は、 こ の 二 つ が 別 で は な い こ と を、 私 た ち に 教 え て く れ る。 中 学 校 の 校 庭 で 「打 者!」 と 叫 ぶ 君 の 声 の 中 に、 130 年 前 の 病 床 の 俳 人 の 仕 事 が、 今 も 生 き て い る。

***

(注：本 文 は、 正 岡 子 規『松 蘿 玉 液』 (1896) の「ベ ー ス ボ ー ル」 の 章、 お よ び 関 連 史 料 を、 中 学 体 育・道 徳 教 材 と し て 編 集 部 が 再 構 成 し た 解 説 で あ る。 原 文 は 青 空 文 庫 で 公 開 さ れ て い る。)
