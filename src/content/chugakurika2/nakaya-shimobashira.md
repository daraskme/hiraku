<h3>本 文 (『霜 柱 の 研 究』 中 谷 宇 吉 郎 ・ 抜 粋)</h3>

<blockquote>
<p>
冬 の 朝、 庭 を 歩 く と、 ザ ク ッ、 ザ ク ッ と 心 地 よ い 音 が す る。<br>
足 元 を 見 る と、 土 の 表 面 か ら、 細 く 透 き 通 っ た 氷 の 柱 が、 何 本 も 立 ち 上 が っ て い る。<br>
こ れ が 「霜 柱 (し も ば し ら)」 で あ る。
</p>

<p>
私 は 子 ど も の 頃、 故 郷 の 石 川 県 片 山 津 で、 こ の 霜 柱 を 踏 む の が 大 好 き だ っ た。<br>
そ し て、 い つ も 不 思 議 に 思 っ て い た: 「<strong>な ぜ、 土 か ら こ ん な 氷 の 柱 が 生 え る の だ ろ う?</strong>」 と。
</p>

<p>
大 人 に な っ て、 物 理 学 者 と な り、 北 海 道 大 学 で 雪 の 結 晶 を 研 究 し て い る 私 が、 改 め て こ の 「霜 柱」 と い う 現 象 を 調 べ て み る と ─ こ れ は、 雪 結 晶 と は ま た 違 う、 興 味 深 い メ カ ニ ズ ム で 形 成 さ れ る こ と が わ か っ た。
</p>

<p>
霜 柱 が 立 つ 条 件 は、 三 つ で あ る:
</p>

<ol>
<li><strong>地 表 面 の 温 度 が 0 °C 以 下 に な る こ と</strong></li>
<li><strong>地 中 の 温 度 が 0 °C よ り 上 で あ る こ と</strong></li>
<li><strong>土 が 適 度 な 水 分 を 含 み、 微 細 な 隙 間 (毛 細 管) を 持 つ こ と</strong></li>
</ol>

<p>
こ の 3 条 件 が 揃 う と、 次 の よ う な こ と が 起 こ る:
</p>

<p>
地 表 面 で 水 分 が 凍 結 す る ─ こ れ で 小 さ な 氷 の 種 が で き る。<br>
そ の 氷 の 種 の 真 下 で は、 地 中 か ら 水 分 が <strong>毛 細 管 現 象</strong> に よ っ て 吸 い 上 げ ら れ る (紙 タ オ ル が 水 を 吸 う の と 同 じ 原 理)。<br>
吸 い 上 げ ら れ た 水 は、 既 存 の 氷 の 下 端 に 達 し た 瞬 間 に 凍 結 し、 氷 を 下 か ら 押 し 上 げ る。<br>
こ う し て、 氷 が 一 晩 か け て 「<strong>下 か ら ど ん ど ん 伸 び る</strong>」 の だ。
</p>

<p>
朝 に な っ て 霜 柱 を 見 る と、 そ の 高 さ は 数 セ ン チ メ ー ト ル に も 達 す る こ と が あ る。<br>
一 本 一 本 が、 透 き 通 っ た 氷 の 結 晶 で あ り、 顕 微 鏡 で 見 る と、 細 い 線 状 結 晶 が 束 に な っ た 構 造 を し て い る。
</p>

<p>
霜 柱 が 立 つ 場 所 と し て、 特 に 有 名 な の が <strong>関 東 平 野</strong> で あ る。<br>
そ の 理 由 は、 こ の 地 の 土 壌 が <strong>関 東 ロ ー ム 層</strong> と 呼 ば れ る、 火 山 灰 性 の 細 か い 土 で あ る か ら。<br>
細 か い 粒 子 が 適 度 な 隙 間 を 作 り、 そ こ を 水 が 毛 細 管 現 象 で 上 が り や す い。<br>
つ ま り、 「霜 柱 が 立 ち や す い 土」 と 「立 ち に く い 土」 が あ る の だ。
</p>

<p>
さ ら に、 霜 柱 と 「雪 の 結 晶」 を 比 べ て み よ う:
</p>

<ul>
<li><strong>雪 の 結 晶</strong> ─ 空 中 で、 水 蒸 気 か ら 直 接 氷 に な る (昇 華)。 六 角 形 の 美 し い 結 晶。</li>
<li><strong>霜 柱</strong> ─ 土 中 の 液 体 の 水 が、 地 表 で 凍 結 し、 下 か ら 押 し 上 げ ら れ て 成 長。 細 長 い 柱 状。</li>
</ul>

<p>
同 じ 「氷」 で あ り な が ら、 で き 方 が 違 え ば 全 く 違 う 形 に な る。 こ れ も、 自 然 の 美 し い 多 様 性 の 一 つ で あ る。
</p>

<p>
朝、 ザ ク ッ、 ザ ク ッ と 霜 柱 を 踏 ん で 学 校 に 行 く 子 ど も た ち よ。<br>
君 た ち が 何 気 な く 踏 ん で い る そ の 霜 柱 の 中 に は、 一 晩 か け て 地 中 か ら 水 が 上 が り 続 け、 凍 り 続 け た、 自 然 の 静 か な 営 み が あ る。<br>
そ の こ と を 知 る だ け で、 冬 の 朝 が、 少 し 違 う 顔 を 見 せ て く れ る は ず だ。
</p>
</blockquote>

<h3>解 説 ─ 霜 柱 の メ カ ニ ズ ム</h3>

<p>
中 谷 宇 吉 郎 が 解 明 し た 霜 柱 の メ カ ニ ズ ム は、 二 つ の 物 理 現 象 の 組 み 合 わ せ で あ る。
</p>

<h4>① 毛 細 管 現 象 ─ 水 が 重 力 に 逆 ら っ て 上 が る</h4>

<p>
細 い 管 (毛 細 管) や 土 の 微 細 な 隙 間 に 水 を 入 れ る と、 水 は <strong>重 力 に 逆 ら っ て</strong> 上 に 上 が る。<br>
こ れ は、 水 の 表 面 張 力 と、 水 と 管 の 壁 の 間 の 吸 着 力 が、 重 力 よ り 大 き く 働 く た め で あ る。
</p>

<p>
日 常 で の 例:
</p>
<ul>
<li><strong>紙 タ オ ル</strong> ─ 水 を 吸 う</li>
<li><strong>植 物 の 根</strong> ─ 根 か ら 茎 の 上 ま で 水 を 上 げ る</li>
<li><strong>蝋 燭 (ろ う そ く)</strong> ─ 芯 を 通 し て 蝋 を 燃 や す</li>
</ul>

<p>
霜 柱 で は、 地 中 の 水 が 土 の 微 細 な 隙 間 を 通 っ て、 地 表 ま で 上 が る。
</p>

<h4>② 凍 結 ─ 0 °C で 液 体 → 固 体</h4>

<p>
地 表 ま で 上 が っ た 水 は、 0 °C 以 下 の 表 面 で 凍 結 し、 氷 と な る。<br>
こ の 「<strong>地 中 か ら の 水 の 供 給</strong>」 と 「<strong>地 表 で の 凍 結</strong>」 が 一 晩 中 続 く こ と で、 細 い 氷 柱 が 数 セ ン チ メ ー ト ル に ま で 成 長 す る。
</p>

<h3>霜 柱 と 雪 結 晶 の 違 い</h3>

<p>
同 じ 「氷」 で あ り な が ら、 こ の よ う に 全 く 違 う 形 で 形 成 さ れ る。<br>
中 谷 宇 吉 郎 は、 「雪 の 結 晶」 と 「霜 柱」 ─ 両 方 を 研 究 す る こ と で、 「氷」 と い う 自 然 現 象 の 多 様 性 を 明 ら か に し た の で あ る。
</p>

<h3>関 東 平 野 と 霜 柱</h3>

<p>
霜 柱 が 立 ち や す い 場 所 と し て、 関 東 平 野 が 有 名 で あ る。<br>
そ の 理 由 は、 こ の 地 の 土 壌 が 「<strong>関 東 ロ ー ム 層</strong>」 と い う 火 山 灰 性 の 細 か い 土 か ら 成 る か ら だ。
</p>

<ul>
<li>富 士 山 ・ 浅 間 山 ・ 箱 根 ・ 赤 城 山 な ど の 火 山 か ら 数 万 年 〜 数 十 万 年 か け て 降 り 積 も っ た 火 山 灰</li>
<li>細 か い 粒 子 が、 適 度 な 「毛 細 管」 を 形 成</li>
<li>水 を よ く 含 む 性 質</li>
</ul>

<p>
こ の 三 つ の 条 件 が、 霜 柱 形 成 に 理 想 的 で あ る。
</p>

<p>
一 方、 砂 地 や 粘 土 質 の 場 所 で は、 毛 細 管 が 大 き す ぎ た り 小 さ す ぎ た り し て、 霜 柱 は 立 ち に く い。
</p>

<h3>都 市 化 と 霜 柱 の 減 少</h3>

<p>
近 年、 都 市 部 で は 霜 柱 を 見 る 機 会 が 減 っ た と 言 わ れ る。<br>
理 由 は い く つ か あ る:
</p>

<ol>
<li><strong>地 表 の 舗 装</strong> ─ ア ス フ ァ ル ト ・ コ ン ク リ ー ト で 土 が 見 え な い</li>
<li><strong>ヒ ー ト ア イ ラ ン ド 現 象</strong> ─ 都 市 部 の 気 温 が 上 昇 し、 0 °C を 下 回 る こ と が 少 な く な っ た</li>
<li><strong>地 中 水 の 減 少</strong> ─ 排 水 シ ス テ ム の 整 備 で、 地 中 の 水 分 が 減 っ た</li>
</ol>

<p>
霜 柱 は、 都 市 化 の 進 行 を 静 か に 教 え て く れ る 「自 然 の 指 標」 で も あ る の だ。
</p>

<h3>あ な た の 霜 柱 観 察 を</h3>

<p>
次 の 冬、 朝 早 く に 庭 や 公 園 を 歩 い て、 霜 柱 を 探 し て み よ う。<br>
そ し て、 一 本 つ ま ん で、 太 陽 に か ざ し て み よ う。 透 き 通 っ た 氷 の 中 に、 細 か い 線 状 の 結 晶 が 見 え る は ず だ。
