<h3>本 文 (『夕 凪 と 夕 風』 寺 田 寅 彦 ・ 抜 粋)</h3>

<blockquote>
<p>
夏 の 海 辺 の 夕 暮 れ ─ 太 陽 が 海 に 沈 み か け る そ の 頃、 不 思 議 な 一 瞬 が あ る。<br>
そ れ ま で 昼 中 ず っ と 吹 い て い た 海 か ら の 涼 し い 風 が、 ぴ た り と 止 ま る の で あ る。<br>
波 も 静 か に な り、 空 気 が 重 く な っ た よ う に 感 じ る。 こ れ を 「夕 凪 (ゆ う な ぎ)」 と 呼 ぶ。
</p>

<p>
し ば ら く そ の 状 態 が 続 い た 後、 ふ と、 反 対 方 向 か ら、 つ ま り 陸 の 方 か ら、 ま た 別 の 風 が 吹 き 始 め る。<br>
こ れ が 「夕 風 (ゆ う か ぜ)」 で あ る。
</p>

<p>
こ の 一 連 の 風 の 変 化 ─ 海 風 → 夕 凪 → 陸 風 ─ は、 海 岸 沿 い の 地 域 で は 毎 日 観 察 で き る、 規 則 正 し い 現 象 で あ る。<br>
特 に、 瀬 戸 内 海 沿 岸 (広 島・岡 山 な ど) で は、 「夕 凪」 と い う 言 葉 が 日 常 語 と し て 使 わ れ る ほ ど、 顕 著 で あ る。
</p>

<p>
で は、 な ぜ こ う し た 現 象 が 起 こ る の か?<br>
そ の 理 由 は、 単 純 で あ る。 <strong>陸 と 海 の 温 ま り 方 と 冷 え 方 の 違 い</strong> に 由 来 す る の だ。
</p>

<p>
昼 間 ─ 太 陽 が 照 る と、 陸 地 は 速 く 温 ま る。 一 方、 海 は 水 が あ る た め、 温 ま り に く い。<br>
す る と、 陸 地 上 の 空 気 が 温 め ら れ て 上 昇 し、 そ こ に 海 か ら 冷 た い 空 気 が 吸 い 込 ま れ る。<br>
こ う し て 「海 風」 が 吹 く。
</p>

<p>
夕 方 ─ 太 陽 が 沈 む と、 陸 地 は 急 速 に 冷 え る。 海 は ま だ 昼 間 の 熱 を 保 っ て い る。<br>
今 度 は、 海 の 上 の 空 気 が 温 か く な り、 上 昇 す る。 そ こ に 陸 地 か ら 冷 た い 空 気 が 流 れ 込 む。<br>
こ う し て 「陸 風」 が 吹 く。
</p>

<p>
さ て、 海 風 か ら 陸 風 へ の 切 り 替 わ り の 瞬 間 ─ ち ょ う ど、 陸 と 海 の 温 度 が 釣 り 合 っ た 時 ─ 風 の 力 が ゼ ロ に な る。<br>
こ れ が 「夕 凪」 で あ る。
</p>

<p>
朝 に も 同 じ 現 象 が 起 こ る。 夜 中 の 陸 風 が、 夜 明 け と と も に 弱 ま り、 や が て 海 風 に 切 り 替 わ る。 そ の 間 に 「朝 凪 (あ さ な ぎ)」 が あ る。
</p>

<p>
こ う し て 私 た ち は、 「夕 凪」 と い う、 一 見 不 思 議 で 詩 的 な 現 象 を、 物 理 の 言 葉 で 説 明 す る こ と が で き る。<br>
だ が こ の こ と は、 「夕 凪」 の 美 し さ を 損 な う も の で は な い。 む し ろ、 そ の 背 後 に 働 い て い る 自 然 の 規 則 性 を 知 る こ と で、 私 た ち は も っ と 深 く 「夕 凪」 を 味 わ う こ と が で き る の で あ る。
</p>
</blockquote>

</p>

<h3>な ぜ 海 は 温 ま り に く い か</h3>

<p>
寺 田 の 説 明 の 根 底 に は、 「<strong>水 は 温 ま り に く く、 冷 め に く い</strong>」 と い う 物 理 学 の 基 本 が あ る。
</p>

<p>
こ れ は、 中 学 校 理 科 で 学 ぶ 「<strong>比 熱</strong>」 の 概 念 で あ る。<br>
水 の 比 熱 は、 約 4.2 J/(g・K)。 一 方、 砂 や 土 の 比 熱 は 約 0.8 J/(g・K) ほ ど。<br>
つ ま り、 同 じ 熱 を 与 え て も、 水 は 砂 の <strong>5 倍 温 度 が 上 が り に く い</strong>。
</p>

<p>
こ の 「温 ま り 方 の 差」 が、 一 日 を 通 し て 海 と 陸 の 間 に 温 度 差 を 生 み、 そ れ が 風 を 生 む。
</p>

<h3>瀬 戸 内 海 の 「夕 凪」 が 有 名 な 理 由</h3>

<p>
瀬 戸 内 海 沿 岸 (広 島・岡 山・愛 媛・香 川 な ど) で は、 「夕 凪」 が 特 に 強 く・長 く 続 く こ と で 知 ら れ る。<br>
真 夏 の 夕 凪 は、 1–2 時 間 ほ ど 風 が 完 全 に 止 む こ と も あ り、 蒸 し 暑 さ で 有 名 で あ る。
</p>

<p>
な ぜ 瀬 戸 内 海 か?<br>
瀬 戸 内 海 は、 <strong>四 国 と 本 州 に 囲 ま れ た 内 海</strong>。 そ し て 周 囲 を 山 が 取 り 囲 む。<br>
こ の 地 形 の た め、 海 風 と 陸 風 の 切 り 替 わ り が、 純 粋 な 形 で 観 察 さ れ、 ま た 周 辺 か ら の 別 の 風 が 入 り 込 ま な い。<br>
結 果、 「夕 凪」 が 長 く・は っ き り と 現 れ る の で あ る。
</p>

<h3>「夕 凪 は 詩 で あ り、 物 理 で あ る」</h3>

<p>
寺 田 寅 彦 の 文 章 の 魅 力 は、 「<strong>美 し さ と 科 学 の 両 立</strong>」 に あ る。
</p>

<p>
彼 は こ う 書 い た:<br>
「夕 凪 を 物 理 で 説 明 し て も、 そ の 美 し さ は 損 な わ れ な い。 む し ろ、 自 然 の 規 則 性 を 知 る こ と で、 も っ と 深 く 味 わ え る」 と。
</p>

<p>
こ の 姿 勢 は、 寺 田 の 弟 子 で あ る 中 谷 宇 吉 郎 の 「<strong>雪 は 天 か ら 送 ら れ た 手 紙 で あ る</strong>」 と い う 名 言 (本 サ イ ト chugakurika1/nakaya-yuki-wo-tsukuru-hanashi) に も 受 け 継 が れ て い る。
</p>

<p>
科 学 と 詩 は、 対 立 す る も の で は な い。<br>
寺 田 寅 彦 ・ 中 谷 宇 吉 郎 の 系 譜 が 教 え て く れ る の は、 「<strong>自 然 を 物 理 の 眼 で 見 て も、 そ の 美 は 失 わ れ な い</strong>」 と い う、 ひ と つ の 大 切 な 真 実 で あ る。
</p>

<h3>あ な た の 「夕 凪」 体 験 を</h3>

<p>
今 度、 海 辺 を 訪 れ た ら、 夕 暮 れ 時 に、 風 の 様 子 を 観 察 し て み よ う。
</p>

<ol>
<li>昼 間 ─ 海 か ら 涼 し い 風 が 吹 い て く る (海 風)</li>
<li>夕 方 ─ 風 が ぴ た り と 止 む 瞬 間 が あ る (夕 凪)</li>
<li>夜 ─ 陸 か ら 反 対 向 き に 風 が 吹 き 始 め る (陸 風)</li>
</ol>

<p>
こ の 一 連 の 変 化 を、 自 分 の 体 で 感 じ た と き、 君 は 寺 田 寅 彦 と 同 じ 体 験 を し て い る。<br>
そ し て、 「夕 凪」 と い う 一 瞬 の 詩 情 の 背 後 に、 ど ん な 物 理 が 働 い て い る か を、 君 は 知 る の で あ る。
</p>
