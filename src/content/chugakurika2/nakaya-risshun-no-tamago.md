<h3>本文（『立春の卵』中谷宇吉郎・抜粋）</h3>

<blockquote>
<p>
昭和 22 年 2 月の 初め頃、新聞 や 雑誌 が、こぞって、奇妙な 報道 を 始めた。
</p>

<p>
「<strong>立春 の 日 に は、卵 が 立つ</strong>」
</p>

<p>
すなわち、毎年 2 月 4 日 ごろ の 立春 の 瞬間、地球 の 自転軸 が ある 特定 の 角度 を 取り、その 影響 で、生卵 が テーブル の 上 で、自分 ひとり で 立つ ─ という の で ある。<br>
新聞 は 写真 入り で、「立つ 卵」 の 様子 を 大きく 報じた。雑誌 は 「立春 の 不思議」 と 題して、特集 を 組んだ。
</p>

<p>
さて、物理学者 と して、私 は この 報道 に 強い 関心 を 抱いた。<br>
というのも、もし 本当 に 「立春 の 日 に だけ 卵 が 立つ」 ので あれば、これ は 物理学 上 の 重大 な 発見 と 言わ ねば ならない。地球 の 自転軸 の 傾き が、卵 の 安定 に 直接 影響 を 及ぼす と すれば、それ は 何 か 新しい 力 が 働いて いる こと に なる。
</p>

<p>
私 は さっそく、自宅 の 台所 で 実験 を 始めた。
</p>

<p>
2 月 4 日 ─ 立春 の 当日。<br>
私 は 卵 を 一個、テーブル の 上 に そっと 立てて みた。<br>
すると ─ ── ─ 立った。
</p>

<p>
私 は 喜んだ。<strong>これ は 本当 だった の か!</strong> と。<br>
だが、待てよ ─ 念 の ため、他 の 日 に も 試して みよう。
</p>

<p>
翌日、2 月 5 日。<br>
私 は ふたたび 卵 を 立てた。<br>
すると ─ また 立った。
</p>

<p>
2 月 6 日、7 日、8 日 ─ 毎日 試した。<br>
そして、その 度 に、卵 は 立った の だ。
</p>

<p>
ここ で、私 は、自分 が <strong>科学者 と して 安易 な 確信 を 抱きそうに なって いた</strong> こと を、反省 し た。<br>
科学 の 検証 は、<strong>再現性</strong> と <strong>反証可能性</strong> の 二 つ を 満たさ ね ば ならない。<br>
「立春 の 日 に 立った」 と いう 報告 だけ で は 足りない。それ が 「<strong>立春 以外 の 日 に は 立たない</strong>」 と いう こと も、確かめ ね ば ならない。
</p>

<p>
そして、立春 以外 の 日 に も 卵 が 立つ こと が 確かめ られた 以上、「立春 だから 立つ」 と いう 主張 は、<strong>科学的 に は 支持 できない</strong>。
</p>

<p>
で は、なぜ 新聞 や 雑誌 は、「立春 に 卵 が 立つ」 と 報じた の か?
</p>

<p>
おそらく、誰か が 立春 の 日 に 卵 を 立たせて 見せた。それ が 写真 と して 新聞 に 出 た。読者 は それ を 「立春 と 関係 が ある」 と 思い、家 で やって みた。<br>
そして、立春 の 日 に やって みた ─ <strong>立った</strong>。<br>
「やはり 立った! 立春 の 力 だ!」 と 喜んで、また 報告 が 増える。<br>
こう して、「立春 の 卵」 の 都市 伝説 が、雪だるま 式 に 大きく なって いった の で ある。
</p>

<p>
卵 の 殻 は、よく 見れば、完全 な 滑らか な 球 で は ない。微 小 な 凹凸 が あり、その 中 の 三 点 で テーブル に 接して、安定 する。重心 が その 三 点 の 真上 に くれば、卵 は 立つ。<br>
立春 とは 関係 ない。じっくり 集中 し、慎重 に 試せば、いつ でも 卵 は 立つ の だ。
</p>

<p>
さて、こう 書く と、私 は 「人々 の 夢 を 壊す 物理学者」 と 思われる かも しれない。<br>
そう では ない。<br>
私 が 言いたい の は、「立春 の 卵 を 立てる」 という 経験 そのもの は、<strong>大変 美しく、楽しい もの</strong> だ と いう こと だ。<br>
ただ、それ を 「立春 の 神秘」 と して 信じ込む と、本当 の 美しさ (慎重 さ と 集中 を 要する 物理 現象 の 美しさ) を 見失って しまう。
</p>

<p>
科学 は、夢 を 壊す もの で は ない。<br>
科学 は、より 正確 な 眼 で、世界 の 真 の 美しさ を 見せて くれる もの で ある。
</p>
</blockquote>

<h3>解説：科学リテラシー の 三 原則</h3>

<p>
中谷宇吉郎 が 「立春 の 卵」 を 検証 する 過程 は、<strong>科学リテラシー</strong> の 教科書 の よう に 整って いる。
</p>

<ol>
<li>
  <strong>仮説 を 立てる</strong> ─ 「立春 の 日 に 卵 が 立つ」 という 主張 を、検証可能 な 形 に する。
</li>
<li>
  <strong>実験 する</strong> ─ 立春 の 日 と、立春 以外 の 日 と、両方 で 試す (これ が 「対照実験 = control experiment」)。
</li>
<li>
  <strong>結論 を 導く</strong> ─ 「いつ でも 立つ」 ので あれば、「立春 と は 関係 ない」 と 結論 する。
</li>
</ol>

<p>
この 三 原則 は、現代 の 私 たち が SNS で 「すごい!」「不思議!」 と 拡散 される 情報 を 受け取る とき に も、まったく 同じ 形 で 使える。
</p>

<h3>立つ 卵 の 物理</h3>

<p>
卵 が テーブル の 上 で 立つ ─ これ は、物理 学 的 に は どう 説明 できる の か?
</p>

<ol>
<li>
  卵 殻 の 表面 に は <strong>微細 な 凹凸</strong> が ある。完全 な 滑らか な 球 で は ない。
</li>
<li>
  卵 が テーブル に 触れる の は、その 凹凸 の うち、<strong>3 つ の 突起</strong> (頂点 三角形 の よう に 三 点 で)。
</li>
<li>
  3 つ の 接 点 が 作る 三角形 の <strong>内部</strong> に、卵 の <strong>重心</strong> が 真上 から 落ちれば、卵 は 倒れない。
</li>
<li>
  ただし、その 接 点 三角形 は とても 小さい ので、重心 を 真上 に 持って くる に は、<strong>慎重 な 微調整</strong> が 必要。集中 と 忍耐 が 要る。
</li>
</ol>

<p>
卵 が 立つ ─ それ は、地球 の 自転軸 の 傾き で も、立春 の 神秘 で も ない。<br>
<strong>君 の 集中力 と 忍耐</strong> が、卵 を 立たせる の で ある。
</p>

<h3>「神秘」 を 信じる 心 ─ 心理学 の 視点</h3>

<p>
「立春 の 卵」 の 都市 伝説 は、なぜ あれほど 広まった の か?
</p>

<p>
ここ に は、人間 の 認知 の 仕組み が 関係 して いる。
</p>

<ul>
<li><strong>確証バイアス</strong> (confirmation bias) ─ 「立春 だ から 立つ」 と 信じて 試すと、立った 時 を 強く 覚え、立たなかった 時 を 忘れる。</li>
<li><strong>パレイドリア</strong> (pareidolia) ─ ランダム な もの の 中 に、意味 や パターン を 見いだそう と する 傾向。</li>
<li><strong>選択 報道</strong> ─ メディア は 「立った」 という 成功 例 だけ を 報じ、「立たなかった」 という 失敗 例 を 報じない。</li>
</ul>

<p>
これら は、現代 の SNS で 拡散 する 「不思議 な 出来事」 にも、まったく 同じ よう に 当て嵌まる。
</p>

<h3>中谷宇吉郎 と 科学随筆 の 伝統</h3>

<p>
中谷宇吉郎 は、師 の 寺田寅彦 と 同じく、<strong>科学 と 随筆 を 結ぶ</strong> 名手 で あった。<br>
雪 の 結晶 の 研究 で 世界 的 に 有名 だが、彼 の もう 一つ の 偉業 は ── 一般 の 人 に 「科学 の 楽しさ」 を 伝える 文章 を 書き 続けた こと だ。
</p>

<p>
本作 『立春 の 卵』 は、その 中 でも 名作 と 言える 一篇 で ある。<br>
「都市伝説 を 鵜呑み に せず、自分 で 確かめる」 という 姿勢 を、楽しく 軽妙 に 伝えて いる。
</p>

<p>
2026 年 の 私 たち も、SNS で 何 か 不思議 な ニュース を 見たら、まず こう 自問 しよう ──<br>
「<strong>これ は 本当 か。自分 で 確かめられる か?</strong>」<br>
それ が、中谷宇吉郎 が 80 年 前 に 教えて くれた、科学 の 第一歩 で ある。
</p>
