<h3>「雪 は 天 か ら 送 ら れ た 手 紙 で あ る」</h3>

<p>
中 谷 宇 吉 郎 (1900–1962) の 最 有 名 な 名 言。 著 書『雪』 (1938 ・ 岩 波 新 書) お よ び 多 数 の
随 筆 で 繰 り 返 し 述 べ ら れ た 言 葉 で あ る。
</p>

<p>
こ れ は 単 な る 詩 的 比 喩 で は な い。 中 谷 が 確 立 し た 科 学 的 事 実 ─ 雪 の 結 晶 の 形 が、
そ れ が 出 来 た 上 空 の 温 度 ・ 湿 度 を 反 映 し て い る ─ を 表 現 し た 言 葉 で あ る。
</p>

<p>
本 サ イ ト の chugakurika1/yuki で、 君 は 中 1 編 の 中 谷『雪』 に 触 れ た。 こ こ で は、
戦 後 の 随 筆『雪 と 人 生』 を 通 し て、 中 谷 の 思 想 を よ り 深 く 学 ぼ う。
</p>

<h3>寺 田 寅 彦 の 弟 子 ─ 「観 察 す る 心」 の 系 譜</h3>

<p>
中 谷 宇 吉 郎 は、 1900 年 石 川 県 加 賀 市 の 旧 片 山 津 町 出 身。 旧 制 第 四 高 等 学 校 を 経 て
東 京 帝 国 大 学 物 理 学 科 に 入 学。 そ こ で 出 会 っ た 恩 師 が、 寺 田 寅 彦 だ っ た。
</p>

<p>
寺 田 か ら 中 谷 が 学 ん だ の は、 「日 常 の 不 思 議 を 観 察 す る」 姿 勢 で あ る。 本 サ イ ト の 寺 田 教 材
(茶 碗 の 湯・線 香 花 火・金 平 糖 等) と 系 譜 関 係 に あ る。
</p>

<h3>1936 年 ・ 世 界 初 ─ 人 工 雪 の 結 晶</h3>

<p>
昭 和 11 年 (1936) 3 月、 北 海 道 大 学 理 学 部 低 温 室。 中 谷 と 弟 子 た ち は、 世 界 で 初 め て
雪 の 結 晶 を 人 工 的 に 作 っ た。
</p>

<p>
方 法 は ─ 低 温 (−15 ℃) の 部 屋 で、 ウ サ ギ の 毛 を 一 本 だ け 垂 ら し、 飽 和 水 蒸 気 を 周 囲 に 流 す。
水 蒸 気 が ウ サ ギ の 毛 を 核 と し て 結 晶 化 ─ 美 し い 6 角 形 の 雪 結 晶 が 成 長 し た。
</p>

<p>
こ れ で、 雪 結 晶 の 形 と 気 象 条 件 (温 度 ・ 水 蒸 気 量) の 関 係 が、 実 験 的 に 完 全 に 解 明 さ れ た。
</p>

<h3>ナ カ ヤ ダ イ ヤ グ ラ ム ─ 雪 結 晶 の「手 紙」 を 読 む 鍵</h3>

<p>
中 谷 が 確 立 し た 雪 結 晶 形 と 気 象 条 件 の 関 係 図 は、 「ナ カ ヤ ダ イ ヤ グ ラ ム」 と し て 国 際
雪 氷 学 界 で 認 め ら れ て い る。
</p>

<ul>
  <li>−5 ℃ 付 近 ・ 高 湿 度 ─ 樹 枝 状 (枝 の よ う な) 結 晶</li>
  <li>−15 ℃ 付 近 ─ 美 し い 6 角 板 状 結 晶</li>
  <li>−20 ℃ 以 下 ─ 角 柱 状 ・ 針 状 結 晶</li>
  <li>低 湿 度 ─ シ ン プ ル な 角 板</li>
  <li>高 湿 度 ─ 複 雑 な 樹 枝 状 装 飾</li>
</ul>

<p>
つ ま り、 地 上 で 雪 結 晶 を 採 集・観 察 す れ ば、 そ の 結 晶 が 形 成 さ れ た 上 空 数 千 m の 温 度・湿 度
が 推 定 で き る ─ こ れ が「天 か ら 送 ら れ た 手 紙」 の 真 意 で あ る。
</p>

<h3>『雪 と 人 生』 ─ 戦 後 の 随 筆 集</h3>

<p>
昭 和 31 年 (1956)、 中 谷 が 戦 後 に 出 版 し た 随 筆 集『雪 と 人 生』。 雪 ・ 氷 の 研 究 と 並 行 し て、
人 間 ・ 社 会 ・ 自 然 と の 関 わ り を 考 え る 文 章 を 集 め た 一 冊。
</p>

<p>
中 谷 は こ う 書 く ─ 「自 然 を 観 察 す る こ と は、 結 局 自 分 を 観 察 す る こ と で あ る。 雪 結 晶 の
形 か ら 上 空 の 気 象 が 読 め る よ う に、 人 の 生 き 方 か ら も そ の 人 の 内 面 が 読 め る」。
</p>

<p>
科 学 と 文 学、 自 然 と 人 間 ─ 中 谷 は そ の 境 界 を 自 由 に 行 き 来 し た。
</p>

<h3>結 び ─ 自 然 か ら 学 ぶ「観 察 す る 心」</h3>

<p>
中 谷 宇 吉 郎 ─ 寺 田 寅 彦 か ら 受 け 継 い だ「観 察 す る 心」 を、 雪 と い う 一 つ の 自 然 現 象 に 注 い だ
科 学 者・随 筆 家。
</p>

<p>
冬、 雪 が 降 っ た 日、 君 も「天 か ら の 手 紙」 を 受 け 取 っ て み よ う。 黒 い 布 や 板 の 上 に 雪 を 受 け、
ル ー ペ で 結 晶 を 観 察 す る。 そ の 形 か ら、 数 千 m 上 空 の 気 象 を 想 像 す る ─ こ れ が 中 谷 の 教 え る
科 学 の 楽 し み で あ る。
</p>
