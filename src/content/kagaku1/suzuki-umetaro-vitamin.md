<h3>「江 戸 患 い」 と 呼 ば れ た 病</h3>

<p>
江 戸 時 代 か ら、 日 本 の 都 市 住 民 を 苦 し め て き た 不 思 議 な 病 気 が あ っ た。<br>
名 前 は「<strong>脚 気</strong>」(か っ け)、 別 名「<strong>江 戸 患 い</strong>」(え ど わ ず ら い)。
</p>

<p>
症 状 は こ う だ:
</p>

<ul>
<li>手 足 の 浮 腫 (む く み)</li>
<li>神 経 麻 痺・歩 け な く な る</li>
<li>心 臓 が 弱 る (心 不 全)</li>
<li>放 置 す れ ば 死 に 至 る</li>
</ul>

<p>
特 徴 的 な の は、 「<strong>白 米 を 主 食 と す る 都 市 住 民</strong>」 に 多 い こ と だ っ た。<br>
地 方 で 麦 や 玄 米 を 食 べ る 人 々 に は ほ と ん ど 出 ず、 都 会 で 精 米 (せ い ま い) し た 白 米 を 食 べ る 人 々 に 多 発。
</p>

<h3>明 治 ・ 大 正 期 ─ 軍 の 大 問 題</h3>

<p>
明 治 維 新 (1868) 後、 日 本 軍 が 拡 大 す る と、 脚 気 は <strong>軍 の 重 大 問 題</strong> と な っ た。<br>
兵 士 に 白 米 を 配 給 し て い た た め、 兵 士 が 大 量 に 脚 気 で 倒 れ た。
</p>

<ul>
<li>1882 年 ─ 海 軍 で 脚 気 死 亡 率 が 約 4%</li>
<li>1894–95 年 (日 清 戦 争) ─ 戦 死 者 約 1500 人 に 対 し、 脚 気 死 者 約 4000 人</li>
<li>1904–05 年 (日 露 戦 争) ─ 陸 軍 の 戦 死 者 約 5 万 人 に 対 し、 脚 気 死 者 約 2 万 人</li>
</ul>

<p>
原 因 は ま だ わ か ら な か っ た。 大 論 争 が 起 こ っ た。
</p>

<h3>脚 気 論 争 ─ 海 軍 vs 陸 軍</h3>

<p>
明 治 期 の 日 本 軍 で、 二 人 の 軍 医 が 真 っ 向 か ら 対 立 し た:
</p>

<h4>海 軍 軍 医・<strong>高 木 兼 寛</strong> (た か き か ね ひ ろ、 1849–1920)</h4>

<p>
英 国 で 医 学 を 学 ん だ 高 木 は、 統 計 か ら こ う 推 論 し た:<br>
「<strong>白 米 食 が 原 因 だ。 食 事 を 変 え れ ば 脚 気 は 治 る</strong>」
</p>

<p>
1884 年、 海 軍 の 食 事 を 「麦 飯 + 肉 + 野 菜」 に 切 り 替 え る と、 脚 気 が <strong>劇 的 に 減 少</strong>。<br>
高 木 の 「食 事 説」 が 実 証 さ れ た。
</p>

<h4>陸 軍 軍 医 ・ <strong>森 林 太 郎</strong> (も り り ん た ろ う ・ 森 鴎 外、 1862–1922)</h4>

<p>
ド イ ツ で 医 学 を 学 ん だ 森 (作 家 ・ 森 鴎 外 と し て も 有 名 ─ 本 サ イ ト で 既 出) は、 こ う 反 論 し た:<br>
「<strong>脚 気 は 細 菌 性 の 感 染 症 だ。 食 事 で は 治 ら な い</strong>」
</p>

<p>
当 時、 コ ッ ホ・パ ス ツ ー ル に よ る 細 菌 学 が 大 流 行 し て お り、 「あ ら ゆ る 病 気 は 細 菌 が 原 因」 と 信 じ ら れ て い た。<br>
森 は こ の 立 場 か ら、 陸 軍 で 麦 飯 導 入 を 阻 止 し た。
</p>

<p>
結 果 ─ 陸 軍 で は 麦 飯 導 入 が 遅 れ、 日 露 戦 争 で <strong>約 2 万 人</strong> の 脚 気 死 者 を 出 し た。<br>
こ の 悲 劇 は、 後 の 森 鴎 外 評 価 に 影 を 落 と し て い る。
</p>

<h3>鈴 木 梅 太 郎 の 発 見 (1910)</h3>

<p>
脚 気 論 争 が 続 く 中、 一 人 の 化 学 者 が <strong>科 学 的 決 着</strong> を 付 け た。
</p>

<p>
東 京 帝 国 大 学 農 学 部 教 授 ・ <strong>鈴 木 梅 太 郎</strong> (1874–1943)。
</p>

<p>
彼 は こ う 考 え た: <strong>「白 米 を 食 べ て も 脚 気 に な る が、 玄 米 や 麦 飯 で は な ら な い ─ つ ま り、 精 米 で 取 り 除 か れ る 部 分 (米 ぬ か) に 何 か 重 要 な 物 質 が あ る は ず」</strong>
</p>

<p>
そ し て 1910 年、 米 ぬ か か ら、 脚 気 を 予 防・治 療 す る 不 思 議 な 物 質 を 単 離 し た。<br>
鈴 木 は こ れ を 「<strong>オ リ ザ ニ ン</strong>」(oryzanin、 oryza = ラ テ ン 語 で 「米」) と 命 名 し、 1910 年 12 月 に 東 京 化 学 会 誌 に 日 本 語 で 発 表 し た。
</p>

<p>
こ れ は、 世 界 で <strong>最 初 の ビ タ ミ ン 単 離</strong> と 言 う べ き 業 績 で あ っ た。
</p>

<h3>「言 語 障 壁」 ─ ノ ー ベ ル 賞 を 逃 し た 理 由</h3>

<p>
だ が、 鈴 木 の 発 表 は 日 本 語 で の み 行 わ れ た た め、 欧 米 の 科 学 界 で は ほ と ん ど 注 目 さ れ な か っ た。
</p>

<p>
2 年 後 の 1912 年、 ポ ー ラ ン ド 出 身 で ロ ン ド ン に 居 た 化 学 者 <strong>カ ジ ミ ー ル・フ ン ク</strong> (Casimir Funk, 1884–1967) が、 同 様 の 物 質 を 単 離・発 表 し、 「<strong>vitamin</strong>」(vita = 生 命 + amine) と 命 名 し た。
</p>

<p>
そ し て 1929 年、 英 国 の 生 化 学 者 フ レ デ リ ッ ク・ホ プ キ ン ス (Frederick Gowland Hopkins, 1861–1947) が 「ビ タ ミ ン 発 見」 で ノ ー ベ ル 賞 を 受 賞 ─ <strong>鈴 木 で は な く</strong>。
</p>

<p>
鈴 木 は 後 に ド イ ツ 語 で 自 分 の 仕 事 を 再 発 表 し た が、 既 に 国 際 名「ビ タ ミ ン」 が 定 着 し て お り、 オ リ ザ ニ ン と の 同 一 性 も 認 め ら れ た が、 「世 界 初 の ビ タ ミ ン 発 見 者」 の 名 誉 は 取 り 戻 せ な か っ た。
</p>

<p>
こ れ は、 「<strong>科 学 と 言 語</strong>」 の 関 係 を 考 え さ せ る 重 要 な 事 例 で あ る。
</p>

<h3>鈴 木 梅 太 郎 と 理 化 学 研 究 所</h3>

<p>
鈴 木 梅 太 郎 は、 オ リ ザ ニ ン 発 見 後 も 多 く の 仕 事 を 残 し た:
</p>

<ul>
<li>1917 ─ <strong>理 化 学 研 究 所</strong> (理 研) 設 立 に 関 与</li>
<li>1923 ─ 関 東 大 震 災 後 の 食 糧 問 題 で 緊 急 食 料 化 学 研 究</li>
<li>多 数 の 弟 子 を 育 成 ─ 日 本 化 学 界 の 礎 を 築 く</li>
</ul>

<p>
1943 年 没。 69 歳。
</p>

<h3>「ビ タ ミ ン」 の そ の 後</h3>

<p>
鈴 木 と フ ン ク の 発 見 か ら 約 110 年。<br>
今 や ビ タ ミ ン は、 化 学 構 造 が 完 全 に わ か り、 13 種 類 に 分 類 さ れ て い る (A、 B1、 B2、 B6、 B12、 C、 D、 E、 K な ど)。
</p>

<p>
そ し て、 鈴 木 梅 太 郎 の オ リ ザ ニ ン = ビ タ ミ ン B1 は ─ 今 も 私 た ち の 体 で、 1 日 に 数 mg 必 要 と さ れ、 神 経 機 能 と 糖 質 代 謝 を 支 え 続 け て い る。
</p>

<h3>君 へ の メ ッ セ ー ジ</h3>

<p>
鈴 木 梅 太 郎 の 物 語 は、 二 つ の 教 訓 を 残 す:
</p>

<ol>
<li><strong>日 本 の 科 学 は、 100 年 以 上 前 か ら 世 界 級 だ っ た</strong> ─ 自 信 を 持 と う</li>
<li><strong>世 界 に 発 信 す る 言 語 と 場 が 必 要 だ</strong> ─ 英 語 力 と 国 際 性 が、 21 世 紀 の 君 に も 求 め ら れ る</li>
</ol>

<p>
脚 気 と い う 病 を 解 明 し、 同 時 に「日 本 か ら 世 界 へ」 と い う 課 題 を 残 し た 鈴 木 梅 太 郎。<br>
彼 の 仕 事 は、 化 学 史 の 教 訓 と し て、 今 も 私 た ち に 響 い て い る。
</p>
