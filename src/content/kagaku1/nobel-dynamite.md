## ノ ー ベ ル と ダ イ ナ マ イ ト

### 1. 危 険 す ぎ る 爆 薬

19 世 紀 半 ば、 鉱 山 や ト ン ネ ル の 工 事 で は、 黒 色 火 薬 が 使 わ れ て い た。 し か し、 黒 色 火 薬 は 威 力 が 弱 く、 硬 い 岩 盤 を 砕 く に は 力 不 足 で あ っ た。

1847 年、 イ タ リ ア の 化 学 者 ソ ブ レ ロ が、 は る か に 強 力 な 液 体 爆 薬「ニ ト ロ グ リ セ リ ン」 を 発 見 し た。 し か し、 こ れ に は 致 命 的 な 問 題 が あ っ た。 衝 撃 に 極 め て 敏 感 で、 少 し の 振 動 で も 爆 発 し て し ま う の だ。 運 搬 中 に 爆 発 す る 事 故 が 各 地 で 多 発 し た。

ス ウ ェ ー デ ン の ノ ー ベ ル 家 も、 こ の ニ ト ロ グ リ セ リ ン を 製 造 し て い た が、 1864 年、 ス ト ッ ク ホ ル ム の 工 場 が 大 爆 発。 ア ル フ レ ッ ド・ノ ー ベ ル の 弟 エ ミ ー ル を 含 む 5 人 が 死 亡 し た。

***

### 2. 1867 年 ─ ダ イ ナ マ イ ト の 発 明

弟 を 失 っ た ア ル フ レ ッ ド・ノ ー ベ ル (1833-1896) は、 「ニ ト ロ グ リ セ リ ン を 安 全 に 扱 う 方 法」 の 研 究 に 没 頭 し た。

そ し て 1867 年、 つ い に 解 決 策 を 見 つ け た。 危 険 な 液 体 ニ ト ロ グ リ セ リ ン を、 「珪 藻 土 (け い そ う ど)」 ─ 珪 藻 と い う 微 生 物 の 殻 が 堆 積 し て で き た、 ス ポ ン ジ の よ う に 穴 だ ら け の 土 ─ に 染 み 込 ま せ た の だ。

す る と、 液 体 が 安 定 し、 衝 撃 で 簡 単 に は 爆 発 し な く な っ た。 そ れ で い て、 「雷 管 (信 管)」 で 意 図 的 に 起 爆 す れ ば、 強 力 に 爆 発 す る。 安 全 に 運 べ て、 必 要 な と き だ け 爆 発 さ せ ら れ る 固 体 爆 薬 ─ こ れ が「ダ イ ナ マ イ ト」 で あ る。

***

### 3. 富 と「死 の 商 人」

ダ イ ナ マ イ ト は、 世 界 を 変 え た。 ト ン ネ ル 工 事、 鉱 山 開 発、 運 河 建 設、 道 路 建 設 ─ 硬 い 岩 盤 を 砕 く 土 木 工 事 が 飛 躍 的 に 進 ん だ。 ス エ ズ 運 河、 ア ル プ ス の ト ン ネ ル、 ア メ リ カ の 大 陸 横 断 鉄 道 ─ 多 く の 大 工 事 が ダ イ ナ マ イ ト で 実 現 し た。

ノ ー ベ ル は 世 界 中 に 爆 薬 工 場 を 持 ち、 巨 万 の 富 を 築 い た。

し か し、 ダ イ ナ マ イ ト は 同 時 に、 戦 争 の 兵 器 と し て も 使 わ れ た。 ノ ー ベ ル は「死 の 商 人」 と も 陰 で 呼 ば れ る よ う に な っ た。

***

### 4. 自 分 の 死 亡 記 事 を 読 む

1888 年、 一 つ の 出 来 事 が ノ ー ベ ル を 大 き く 揺 さ ぶ っ た。

兄 リ ュ ド ビ ッ ヒ が 死 去 し た 際、 フ ラ ン ス の あ る 新 聞 が、 弟 ア ル フ レ ッ ド と 取 り 違 え て、 こ う 報 じ た の だ。

「死 の 商 人、 死 す。 ダ イ ナ マ イ ト で 人 を 殺 す 方 法 を 見 つ け て 財 を 成 し た ア ル フ レ ッ ド・ノ ー ベ ル 氏 が、 昨 日 死 去 し た。」

ま だ 生 き て い る 自 分 の「死 亡 記 事」 を 読 ん だ ノ ー ベ ル は、 衝 撃 を 受 け た。

「自 分 は、 死 ん だ あ と、 こ う 記 憶 さ れ る の か。 ─『死 の 商 人』 と し て。」

(こ の 逸 話 に は 諸 説 あ る が、 ノ ー ベ ル 賞 創 設 の 動 機 の 一 つ と し て 広 く 語 ら れ て い る。)

***

### 5. 1895 年 ─ 遺 言

1895 年、 ノ ー ベ ル は 遺 言 を 書 い た。 そ こ に は、 こ う 記 さ れ て い た。

「私 の 全 財 産 を も と に 基 金 を つ く り、 そ の 利 子 を、 前 年 に 人 類 に 最 大 の 貢 献 を し た 人 々 に、 賞 と し て 分 配 す る こ と。 国 籍 は 一 切 問 わ な い。 部 門 は、 物 理 学・化 学・生 理 学 医 学・文 学・そ し て 平 和、 の 五 つ と す る。」

全 財 産 ─ 当 時 の 約 3,100 万 ス ウ ェ ー デ ン ク ロ ー ネ (現 在 の 価 値 で 数 百 億 円) ─ を、 人 類 の 進 歩 を 称 え る 賞 に 託 し た の で あ る。

1896 年、 ノ ー ベ ル は イ タ リ ア で 没 し た。 享 年 63 歳。 そ し て 1901 年、 第 一 回 ノ ー ベ ル 賞 が 授 与 さ れ た。

***

### 6. な ぜ「平 和 賞」 が あ る の か

ノ ー ベ ル 賞 の 五 部 門 に「平 和」 が 含 ま れ て い る の は、 象 徴 的 で あ る。

自 ら の 発 明 (爆 薬) が 戦 争 に 使 わ れ た こ と へ の、 複 雑 な 思 い。 「科 学 技 術 は、 人 類 を 豊 か に も す る が、 殺 し も す る」 と い う 両 義 性 へ の、 ノ ー ベ ル 自 身 の 自 覚。 平 和 賞 は、 そ の 償 い と 願 い の 表 れ と も 解 釈 さ れ る。

ダ イ ナ マ イ ト を 発 明 し た 化 学 者 が、 自 ら の 富 を、 物 理 学・化 学・医 学 と 並 ん で「平 和」 を 称 え る 賞 に 託 し た。 こ こ に、 科 学 と 倫 理 の 永 遠 の 問 い が 込 め ら れ て い る。

***

### 7. 化 学 と し て の 爆 発

最 後 に、 ダ イ ナ マ イ ト の 化 学 を 確 認 し て お こ う。

爆 発 と は、 化 学 反 応 が 極 め て 短 時 間 で 進 み、 大 量 の 気 体 と 熱 を 一 気 に 放 出 す る 現 象 で あ る。 ニ ト ロ グ リ セ リ ン は 分 解 し て、 二 酸 化 炭 素・水 蒸 気・窒 素・酸 素 と な り、 体 積 が 一 瞬 で 数 千 倍 に 膨 張 す る。 こ の 急 激 な 膨 張 が、 岩 盤 を 砕 く 力 と な る。

中 学 理 科 で 学 ぶ「発 熱 反 応」 の、 最 も 極 端 な 例 で あ る。 同 じ 化 学 反 応 が、 工 事 に も 戦 争 に も 使 わ れ る。 ど う 使 う か は、 化 学 で は な く、 人 間 が 決 め る こ と で あ る。

ノ ー ベ ル の 物 語 は、 そ の こ と を 私 た ち に 問 い か け 続 け て い る。
