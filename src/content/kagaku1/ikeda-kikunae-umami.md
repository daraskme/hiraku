## う ま 味 の 発 見 ─ 新 調 味 料 ニ 就 テ

池 田 菊 苗 (1864–1936)

明 治 41 年 (1908 年) の 春 の 夕 食 の こ と で あ る。 妻 が 用 意 し て く れ た 湯 豆 腐 を 食 べ な が ら、 私 は ふ と 不 思 議 な こ と に 気 が つ い た。

こ の 昆 布 だ し の 湯 豆 腐 は、 確 か に 美 味 し い。 し か し、 そ の 「美 味 し さ」 は、 古 来 か ら 知 ら れ て い る 「甘 い」 「酸 っ ぱ い」 「苦 い」 「し ょ っ ぱ い」 の 四 つ の 味 覚 の ど れ と も 違 う。 そ こ に は 別 の 「第 五 の 味」 が あ る よ う に 感 じ ら れ る。

化 学 者 と し て、 私 は そ の 「第 五 の 味」 の 正 体 を 突 き 止 め た い と 思 っ た。

***

そ こ で 私 は 実 験 を 始 め た。 真 昆 布 12 kg を 大 鍋 で 煮 出 し、 そ の だ し 汁 を 慎 重 に 蒸 発・濃 縮 し、 残 渣 を 結 晶 化 す る。 何 度 も 精 製 を 繰 り 返 し、 つ い に 約 30 g の 白 い 結 晶 を 得 た。

こ の 結 晶 を 分 析 す る と、 そ れ は 1866 年 に ド イ ツ の リ ッ ト ハ ウ ゼ ン が 小 麦 グ ル テ ン か ら 単 離 し た「グ ル タ ミ ン 酸」 と 同 じ 物 質 で あ っ た。 し か し、 グ ル タ ミ ン 酸 そ の も の は 酸 性 が 強 く、 う ま 味 を 感 じ に く い。 そ こ で、 そ れ を ナ ト リ ウ ム で 中 和 し た「グ ル タ ミ ン 酸 ナ ト リ ウ ム (MSG)」 を 作 っ て み た。

舌 に の せ て み る と、 そ こ に は 紛 れ も な く、 あ の 昆 布 だ し の「第 五 の 味」 が あ っ た。

***

私 は こ の 味 覚 を 「う ま 味」 と 命 名 し、 1909 年 4 月、 東 京 化 学 会 誌 第 30 冊 に「新 調 味 料 ニ 就 テ」 と し て 発 表 し た。

「四 つ の 基 本 味 覚 に 加 え、 第 五 の 基 本 味 覚 が 存 在 す る。 そ れ は 昆 布 だ し の よ う な 食 物 の 中 に 含 ま れ、 化 学 的 に は グ ル タ ミ ン 酸 (の ナ ト リ ウ ム 塩) で 表 さ れ る。 こ の 味 覚 を 私 は『う ま 味』 と 名 づ け た い。」

***

科 学 と は、 日 常 の 食 卓 か ら 始 ま る も の で あ る。 私 が こ の 発 見 を し た 出 発 点 は、 妻 の 用 意 し て く れ た 湯 豆 腐 で あ っ た。 「美 味 し い」 と い う 主 観 的 感 覚 を、「グ ル タ ミ ン 酸 ナ ト リ ウ ム」 と い う 客 観 的 化 学 物 質 に 還 元 す る こ と。 こ れ が、 化 学 と い う 学 問 の 仕 事 で あ る と 私 は 考 え る。

そ し て、 こ の 「う ま 味」 が、 鈴 木 三 郎 助 氏 と の 提 携 で 工 業 化 さ れ、 多 く の 家 庭 の 食 卓 を 豊 か に す る な ら ば、 そ れ は 科 学 が 日 常 に 役 立 つ 美 し い 例 と な る で あ ろ う。

***

(注：本 文 は、 池 田 菊 苗 が 1909 年 に 東 京 化 学 会 誌 に 発 表 し た「新 調 味 料 ニ 就 テ」 の 趣 旨 と、 後 年 の 講 演・回 想 を ま と め、 教 材 用 に 編 集 部 が 再 構 成 し た 試 訳 で あ る。 原 文 は 国 立 国 会 図 書 館 デ ジ タ ル コ レ ク シ ョ ン で 閲 覧 で き る。)
