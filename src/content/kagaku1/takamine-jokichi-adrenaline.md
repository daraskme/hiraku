## ア ド レ ナ リ ン ─ 世 界 初 の ホ ル モ ン 純 粋 抽 出

### 1. ニ ュ ー ヨ ー ク の 高 峰 研 究 所、 1900 年

明 治 33 年 (1900 年) の 夏、 ニ ュ ー ヨ ー ク 市 内 に あ っ た「高 峰 研 究 所」 で、 一 人 の 日 本 人 化 学 者 が、 数 ヶ 月 続 け て き た 実 験 の 最 終 段 階 に 取 り 組 ん で い た。

24 歳 の 上 中 啓 三 (う え な か け い ぞ う、 1876-1960)。 東 京 大 学 卒 業 後、 1899 年 か ら 高 峰 譲 吉 の 助 手 と し て ニ ュ ー ヨ ー ク に 渡 っ て い た 若 い 化 学 者 で あ る。

上 中 が 取 り 組 ん で い た の は、 「副 腎 髄 質 か ら 何 か 重 要 な 物 質 を 取 り 出 す」 と い う 研 究 で あ っ た。 副 腎 髄 質 か ら の 抽 出 物 が「心 拍 を 上 昇 さ せ、 血 圧 を 高 め る」 と い う 生 理 作 用 を 持 つ こ と は、 1894 年 オ リ バ ー と シ ェ ー フ ァ ー が 報 告 し て い た。 し か し、 そ の「物 質」 の 正 体 は、 ま だ 純 粋 に 抽 出 さ れ た こ と は な か っ た。

上 中 は、 牛 の 副 腎 を 大 量 に 入 手 し、 ア ル コ ー ル・ア ン モ ニ ア・希 硫 酸 等 で 分 別 抽 出 を 繰 り 返 し、 何 千 回 の 失 敗 を 経 て、 つ い に 1900 年 7 月、 副 腎 抽 出 液 か ら 白 い 結 晶 を 得 る こ と に 成 功 し た。

***

### 2. 1901 年 特 許 と「ア ド レ ナ リ ン」 の 命 名

こ の 結 晶 は、 後 に「ア ド レ ナ リ ン (Adrenaline)」 と 命 名 さ れ た (ラ テ ン 語 ad = に, renal = 腎 の、 つ ま り「腎 の 上 (= 副 腎) の」 と い う 意 味)。

化 学 式 は C₉H₁₃NO₃。 分 子 量 183.2。 体 重 70 kg の 人 間 の 体 内 で、 ご く 微 量 (数 マ イ ク ロ グ ラ ム) で 心 拍 を 倍 に し、 血 圧 を 急 上 昇 さ せ、 気 管 支 を 拡 げ、 瞳 孔 を 開 き、 「闘 争・逃 走 反 応」 を 体 に 引 き 起 こ す 強 力 な ホ ル モ ン で あ っ た。

1901 年、 高 峰 譲 吉 (1854-1922) が こ の 発 見 を 単 独 名 で 米 国 特 許 取 得・American Journal of Pharmacy 誌 に 発 表。 「世 界 初 の ホ ル モ ン 純 粋 抽 出」 と し て 国 際 的 に 認 め ら れ た。

***

### 3. ホ ル モ ン 概 念 よ り 5 年 早 い 発 見

驚 く べ き こ と は、 こ の ア ド レ ナ リ ン 発 見 (1900-1901) が、 「ホ ル モ ン」 と い う 概 念 そ の も の が ま だ 命 名 さ れ て い な か っ た 時 期 に 起 こ っ た こ と で あ る。

「ホ ル モ ン (Hormone)」 と い う 言 葉 を、 イ ギ リ ス の 生 理 学 者 ベ ー リ ス と ス タ ー リ ン グ が 命 名 し た の は 1905 年 で あ っ た。 つ ま り 高 峰 (上 中) は、 「ホ ル モ ン」 と い う 概 念 が ま だ な か っ た 時 期 に、 「ホ ル モ ン」 と い う カ テ ゴ リ ー の 最 初 の 物 質 を、 結 晶 と し て 取 り 出 し て し ま っ た の で あ る。

物 質 の 発 見 が、 概 念 の 命 名 を 5 年 先 行 し た 例 と し て、 化 学 史 ・ 医 学 史 に 残 る。

***

### 4. 救 急 医 療 の 必 須 薬 と し て の ア ド レ ナ リ ン

ア ド レ ナ リ ン は、 そ の 強 力 な 生 理 作 用 ゆ え に、 21 世 紀 の 現 代 で も 救 急 医 療 の 必 須 薬 と し て 使 わ れ て い る。

- **心 停 止 蘇 生** ─ 心 マ ッ サ ー ジ と 共 に ア ド レ ナ リ ン 注 射 で 心 拍 を 再 開 さ せ る。
- **ア ナ フ ィ ラ キ シ ー シ ョ ッ ク** ─ 蜂 刺 さ れ・食 物 ア レ ル ギ ー で 急 性 シ ョ ッ ク を 起 こ し た 患 者 へ。 エ ピ ペ ン (自 己 注 射 器) が こ れ。
- **喘 息 発 作** ─ 重 度 発 作 で 気 管 支 を 拡 げ る 緊 急 薬。
- **局 所 麻 酔** ─ 麻 酔 薬 と 一 緒 に 注 射 し、 血 管 収 縮 で 麻 酔 効 果 を 延 長 さ せ る。

「百 年 以 上 前 の 日 本 人 の 発 見 が、 今 も 毎 日、 世 界 中 で 命 を 救 い 続 け て い る」 と 言 え る。

***

### 5. タ カ ジ ア ス タ ー ゼ ─ も う 一 つ の 大 成 功

高 峰 譲 吉 の 業 績 は、 ア ド レ ナ リ ン だ け で は な か っ た。 彼 は 1894 年、 麹 菌 (こ う じ き ん、 日 本 酒 や 味 噌 の 醸 造 に 使 う 菌) か ら 強 力 な デ ン プ ン 分 解 酵 素 (ジ ア ス タ ー ゼ) を 抽 出 し、 「タ カ ジ ア ス タ ー ゼ (Taka-Diastase)」 と 命 名 し て 商 品 化 し て い た。

消 化 不 良 や 胃 弱 の 薬 と し て 米 国 で 爆 発 的 に 売 れ、 高 峰 は 大 富 豪 と な っ た。 商 標「Taka」 は も ち ろ ん「高 峰」 か ら 取 ら れ て い る。 こ の 富 で、 ニ ュ ー ヨ ー ク に 高 峰 研 究 所 を 設 立 で き た わ け で あ る。

つ ま り、 「日 本 酒 を つ く る 麹 菌 → 米 国 で 大 ヒ ッ ト 商 品 → そ の 利 益 で 研 究 所 設 立 → 世 界 初 の ホ ル モ ン 発 見」 と い う 連 鎖 が 起 こ っ た。 日 本 の 伝 統 (麹 菌) が、 アメ リ カ の 富 と 結 び つ き、 さ ら に 世 界 初 の 医 学 的 発 見 へ と つ な が っ た 不 思 議 な 物 語 で あ る。

***

### 6. 1922 年、 ニ ュ ー ヨ ー ク で の 死

高 峰 譲 吉 は 1922 年 7 月 22 日、 ニ ュ ー ヨ ー ク で 没。 享 年 68 歳。

そ の 1922 年、 高 峰 は ノ ー ベ ル 医 学 生 理 学 賞 候 補 と し て 推 薦 さ れ て い た が、 受 賞 に は 至 ら な か っ た。 し か し、 そ の 後 100 年 を 経 て、 ア ド レ ナ リ ン は 救 急 医 療 の 必 須 薬 と な り、 高 峰 譲 吉 の 名 は 日 本 化 学 史 ・ 医 学 史 に 不 朽 と な っ た。

加 賀 高 岡 の 漁 師 町 に 生 ま れ た 一 人 の 化 学 者 が、 ア メ リ カ で 富 を 築 き、 そ し て 世 界 の 医 療 を 永 遠 に 変 え た。

***
