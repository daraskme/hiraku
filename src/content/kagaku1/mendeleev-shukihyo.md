<h3>1869 年 ・ ロ シ ア ─ 元 素 の 謎 を 解 い た 一 人 の 男</h3>

<p>
1869 年 3 月、 ロ シ ア・サ ン ク ト ペ テ ル ブ ル ク。 サ ン ク ト ペ テ ル ブ ル ク 大 学 化 学 教 授 ・
ド ミ ト リ・メ ン デ レ ー エ フ (35 歳) は、 一 つ の 大 発 見 を ロ シ ア 化 学 学 会 で 発 表 し た。
</p>

<p>
「元 素 を 原 子 量 順 に 並 べ る と、 性 質 が 周 期 的 に 繰 り 返 す」 ─ 後 に「周 期 律」 と 呼 ば れ る、
化 学 の 最 重 要 法 則 の 発 見 で あ る。
</p>

<h3>シ ベ リ ア の 末 っ 子 ─ メ ン デ レ ー エ フ の 生 涯</h3>

<p>
1834 年、 シ ベ リ ア の ト ボ リ ス ク 市。 17 人 兄 弟 の 末 っ 子 と し て メ ン デ レ ー エ フ は 生 ま れ た。
父 は 中 学 校 校 長 で あ っ た が、 メ ン デ レ ー エ フ が 13 歳 の 時 に 死 去。 母 マ リ ア が 一 家 を 支 え た。
</p>

<p>
14 歳 の メ ン デ レ ー エ フ の 才 能 を 確 信 し た 母 マ リ ア は、 末 っ 子 を 連 れ て 4000 km
(東 京 ─ ベ ト ナ ム ホ ー チ ミ ン 程 度) を 馬 車 で 旅 行 ─ サ ン ク ト ペ テ ル ブ ル ク の 名 門 校 に 入 れ た。
そ の 後 マ リ ア は 病 没。 ド ミ ト リ は 母 の 願 い を 受 け て 必 死 に 勉 強 し、 24 歳 で 大 学 助 教 授 と な っ た。
</p>

<h3>1869 年 ・ 教 科 書 の 為 に 元 素 を 並 べ る</h3>

<p>
1868 年、 メ ン デ レ ー エ フ は 大 学 教 科 書『化 学 の 基 礎』 を 執 筆 中 だ っ た。 当 時 知 ら れ て い た
元 素 は 63 種。 ど の 順 序 で 教 科 書 に 並 べ る か、 悩 ん で い た。
</p>

<p>
1869 年 2 月、 彼 は 元 素 の 名 前・原 子 量・性 質 を カ ー ド に 書 き 出 し、 ト ラ ン プ の よ う に
並 べ 替 え て み た。 そ し て、 ふ と 気 付 い た ─ 「原 子 量 順 に 並 べ る と、 性 質 が 周 期 的 に
繰 り 返 し て い る!」
</p>

<h3>周 期 律 ─ 性 質 が 繰 り 返 す</h3>

<p>
メ ン デ レ ー エ フ の 発 見 を 整 理 す る ─
</p>

<ul>
  <li>原 子 量 が 小 さ い 順 に 元 素 を 並 べ る</li>
  <li>あ る 距 離 (周 期) ご と に、 似 た 性 質 の 元 素 が 出 て く る</li>
  <li>水 素 (H) と リ チ ウ ム (Li) と ナ ト リ ウ ム (Na) と カ リ ウ ム (K) は 同 じ 族 (反 応 性 が 似 る)</li>
  <li>ヘ リ ウ ム (He) と ネ オ ン (Ne) と ア ル ゴ ン (Ar) は 同 じ 族 (反 応 し に く い)</li>
</ul>

<p>
こ の「周 期 性」 が、 元 素 の 内 部 構 造 (電 子 配 置) を 反 映 し て い る こ と は、 当 時 は ま だ
わ か ら な か っ た (電 子 の 発 見 は 1897 年・原 子 構 造 解 明 は 20 世 紀 初 頭)。
</p>

<h3>「空 欄」 を 信 じ る ─ 予 言 す る 勇 気</h3>

<p>
メ ン デ レ ー エ フ は こ こ で、 信 じ ら れ な い 決 断 を し た。 周 期 表 を 並 べ る と、 い く つ か の
場 所 が「空 欄」 に な る。
</p>

<p>
彼 は こ う 考 え た ─ 「こ の 場 所 に は、 ま だ 発 見 さ れ て い な い 元 素 が あ る は ず だ。
周 期 律 が 正 し け れ ば、 そ の 元 素 の 性 質 は こ う で あ る」。
</p>

<p>
予 言 し た 元 素 ─
</p>
<ul>
  <li><strong>エ カ ア ル ミ ニ ウ ム</strong> (周 期 表 の Al の 下) ─ 原 子 量 約 68, 融 点 低 い, 密 度 約 6</li>
  <li><strong>エ カ ホ ウ 素</strong> (B の 下) ─ 原 子 量 約 44</li>
  <li><strong>エ カ ケ イ 素</strong> (Si の 下) ─ 原 子 量 約 72, 密 度 約 5.5</li>
</ul>

<p>
「エ カ (eka)」 と は サ ン ス ク リ ッ ト 語 で「一」 を 意 味 ─ 「一 つ 下 の 元 素」 と い う ニ ュ ア ン ス。
</p>

<h3>1875 年 ・ 1879 年 ・ 1886 年 ─ 予 言 が 的 中 す る</h3>

<p>
1875 年、 フ ラ ン ス の 化 学 者 ボ ア ボ ー ド ラ ン が 新 元 素「ガ リ ウ ム」(Ga) を 発 見。 性 質 が、
メ ン デ レ ー エ フ の「エ カ ア ル ミ ニ ウ ム」 予 言 と 一 致 し た。
</p>

<p>
1879 年、 ス ウ ェ ー デ ン の ニ ル ソ ン が「ス カ ン ジ ウ ム」 (Sc) を 発 見。 「エ カ ホ ウ 素」 と 一 致。
</p>

<p>
1886 年、 ド イ ツ の ウ ィ ン ク ラ ー が「ゲ ル マ ニ ウ ム」 (Ge) を 発 見。 「エ カ ケ イ 素」 と 一 致。
</p>

<p>
三 つ の 予 言 が 全 て 的 中 し、 周 期 律 は 確 立 さ れ た。 メ ン デ レ ー エ フ は 世 界 的 名 声 を 得 て、
ノ ー ベ ル 賞 候 補 に も 何 度 も な っ た (受 賞 直 前 で 1907 年 死 去)。
</p>

<h3>現 代 周 期 表 ─ 118 元 素</h3>

<p>
現 代 の 周 期 表 は、 メ ン デ レ ー エ フ の 1869 年 の 表 を 拡 張 し た も の。 118 元 素 が 既 知。
</p>

<ul>
  <li>原 子 番 号 1 ~ 94 ─ 自 然 界 に 存 在 す る</li>
  <li>原 子 番 号 95 ~ 118 ─ 人 工 合 成 さ れ た 元 素</li>
  <li>2016 年、 113 番 元 素 が 日 本 の 理 研 で 発 見 ・ 「ニ ホ ニ ウ ム (Nh)」 と 命 名 さ れ た</li>
</ul>

<p>
周 期 表 は、 化 学 ・ 物 理 学 ・ 生 物 学 ・ 工 学 の あ ら ゆ る 分 野 で 使 わ れ る、 自 然 の「設 計 図」 で
あ る。
</p>

<h3>結 び ─ 一 枚 の 表 が 教 え る こ と</h3>

<p>
メ ン デ レ ー エ フ は、 教 科 書 を 書 こ う と し た だ け だ っ た。 そ の 過 程 で、 自 然 の 深 い 規 則 性 を
発 見 し、 未 発 見 の 元 素 を 予 言 す る ま で に 至 っ た。
</p>

<p>
「整 理 す る こ と が、 発 見 を 生 む」 ─ こ の 教 訓 を、 君 が 高 校 化 学 を 学 ぶ 上 で 心 に 留 め て 欲 し い。
周 期 表 を 暗 記 し て、 元 素 の 物 語 を 自 分 の も の に し て み よ う。
</p>
