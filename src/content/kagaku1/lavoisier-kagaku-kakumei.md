<h3>「化 学 を 科 学 に し た 男」</h3>

<p>
ア ン ト ワ ー ヌ ・ ラ ボ ア ジ エ (Antoine-Laurent de Lavoisier, 1743–1794) は、 「<strong>近 代 化 学 の 父</strong>」 と 呼 ば れ る。
</p>

<p>
彼 が 仕 事 を 始 め る 18 世 紀 半 ば、 化 学 は ま だ「錬 金 術 (al-chemy)」 の 影 を 引 き ず っ て い た。<br>
「物 質 と は 何 か」「燃 焼 と は 何 か」「化 学 反 応 と は 何 か」 ─ こ れ ら の 根 本 問 題 が、 確 か な 答 え を 持 た な か っ た。
</p>

<p>
ラ ボ ア ジ エ は、 厳 密 な 質 量 測 定 と 体 系 的 実 験 で、 こ れ ら の 問 題 に 答 え を 与 え、 化 学 を <strong>近 代 科 学</strong> へ と 変 え た。
</p>

<h3>『化 学 原 論』 (1789)</h3>

<p>
ラ ボ ア ジ エ の 主 著『<strong>化 学 原 論</strong>』(Traité élémentaire de chimie) は、 1789 年 に 出 版 さ れ た。
</p>

<p>
こ の 著 作 で 彼 が 確 立 し た 三 つ の 大 原 則:
</p>

<h4>① 質 量 保 存 の 法 則</h4>

<blockquote>
「<strong>化 学 反 応 で は、 反 応 物 の 総 質 量 と 生 成 物 の 総 質 量 は 等 し い</strong>」
</blockquote>

<p>
当 然 の よ う に 聞 こ え る が、 こ れ を 厳 密 に 確 立 し た の は ラ ボ ア ジ エ の 功 績。
</p>

<p>
彼 は、 密 閉 容 器 の 中 で 化 学 反 応 (燃 焼) を 起 こ し、 反 応 前 と 反 応 後 の 容 器 全 体 の 質 量 を 0.001 g 単 位 で 測 定。<br>
結 果 ─ <strong>ぴ っ た り 同 じ</strong>。 「物 質 は 消 え も せ ず、 生 ま れ も せ ず、 形 を 変 え る だ け」 と 結 論 し た。
</p>

<h4>② 33 元 素 表</h4>

<p>
ラ ボ ア ジ エ は『化 学 原 論』 で、 33 個 の「<strong>元 素</strong>」(element) を リ ス ト ア ッ プ し た。<br>
う ち <strong>23 個</strong> が、 現 代 の 元 素 周 期 表 に も 載 っ て い る。
</p>

<p>
古 代 ギ リ シ ア の「<strong>四 元 素 説</strong>」(水・火・空 気・土) を、 完 全 に 葬 り 去 っ た 瞬 間 で あ る。
</p>

<h4>③ 酸 素 の 命 名 と フ ロ ギ ス ト ン 説 の 否 定</h4>

<p>
17–18 世 紀 の 化 学 で は、 「物 が 燃 え る と き、 物 か ら『<strong>フ ロ ギ ス ト ン (燃 素)</strong>』 と い う 物 質 が 出 て い く」 と 信 じ ら れ て い た。
</p>

<p>
ラ ボ ア ジ エ は、 厳 密 な 質 量 測 定 で こ う 確 か め た:<br>
「燃 焼 す る と、 物 質 の 質 量 は <strong>増 え る</strong>」<br>
こ れ は、 物 質 が「<strong>空 気 中 の 何 か と 結 合 し て い る</strong>」 こ と を 意 味 す る。<br>
そ し て 彼 は、 こ の「結 合 す る 何 か」 を「<strong>酸 素</strong>」(oxygène) と 名 付 け、 フ ロ ギ ス ト ン 説 を 完 全 に 否 定 し た。
</p>

<h3>科 学 の 「言 葉」 の 整 備</h3>

<p>
ラ ボ ア ジ エ の 業 績 は、 こ れ ら の 法 則 だ け で は な い。<br>
彼 は『化 学 原 論』 で、 化 学 の <strong>「言 葉」 そ の も の を 整 備</strong> し た。
</p>

<ul>
<li>「酸 素 (oxygène)」「水 素 (hydrogène)」「炭 素 (carbone)」 ─ 元 素 の 体 系 的 命 名 法</li>
<li>「酸 化 物 (oxide)」「水 酸 化 物 (hydroxide)」「塩 化 物 (chloride)」 ─ 化 合 物 の 命 名 法</li>
<li>化 学 反 応 を 表 す 「等 式」 の 形 式</li>
</ul>

<p>
こ う し て、 化 学 は <strong>万 国 共 通 の 言 語</strong> を 持 ち、 科 学 と し て 国 際 的 に 発 展 で き る よ う に な っ た。
</p>

<h3>ラ ボ ア ジ エ の 悲 劇 (1794)</h3>

<p>
だ が ─ ラ ボ ア ジ エ の 人 生 は、 こ の『化 学 原 論』 出 版 か ら わ ず か 5 年 後 に、 突 然 終 わ る こ と に な る。
</p>

<p>
1789 年 7 月 14 日、 バ ス テ ィ ー ユ 牢 獄 襲 撃 ─ <strong>フ ラ ン ス 革 命</strong> が 始 ま っ た。<br>
ラ ボ ア ジ エ は 貴 族 で あ り、 さ ら に 「徴 税 請 負 人」 (国 か ら 徴 税 を 請 け 負 う 民 間 業 者 ─ 庶 民 か ら の 反 感 が 強 い 職 業) で も あ っ た。
</p>

<p>
革 命 が 過 激 化 す る 1793–1794 年 の 恐 怖 政 治 期、 彼 は 反 革 命 容 疑 で 逮 捕 さ れ、 1794 年 5 月 8 日 に ギ ロ チ ン で 処 刑 さ れ た。 50 歳 だ っ た。
</p>

<p>
処 刑 を 知 っ た 数 学 者 <strong>ジ ョ ゼ フ ・ ル イ ・ ラ グ ラ ン ジ ュ</strong> は、 こ う 嘆 い た:
</p>

<blockquote>
「フ ラ ン ス は ラ ボ ア ジ エ の 頭 を 落 と す の に 一 瞬 で 済 ん だ が、 こ の よ う な 頭 脳 を ま た 持 つ に は、 100 年 か か る か も し れ な い」
</blockquote>

<h3>マ リ ・ ア ン ヌ ─ 妻 と し て の 貢 献</h3>

<p>
ラ ボ ア ジ エ の 仕 事 は、 一 人 で 成 し 遂 げ ら れ た も の で は な か っ た。<br>
妻 <strong>マ リ ・ ア ン ヌ ・ ピ エ レ ッ ト ・ ポ ル ズ</strong> (Marie-Anne Paulze, 1758–1836) が、 重 要 な 役 割 を 果 た し た。
</p>

<p>
彼 女 は 13 歳 で ラ ボ ア ジ エ と 結 婚。 化 学 を 学 び、 夫 の 実 験 を 記 録 し、 化 学 装 置 の 図 を 描 き、 英 国 の 化 学 文 献 を 仏 訳 し た。<br>
『化 学 原 論』 の 美 し い 図 版 は、 彼 女 の 手 に よ る も の だ。
</p>

<p>
夫 の 処 刑 後、 彼 女 は ラ ボ ア ジ エ の ノ ー ト・実 験 記 録 を 守 り 抜 き、 そ の 業 績 を 後 世 に 伝 え た。<br>
近 代 化 学 の 礎 は、 こ の 夫 妻 の 共 同 作 業 で 築 か れ た の だ。
</p>

<h3>ラ ボ ア ジ エ か ら 君 へ</h3>

<p>
中 学・高 校 で 化 学 を 学 ぶ と き、 君 は 必 ず 出 会 う:
</p>

<ul>
<li>「質 量 保 存 の 法 則」 ─ ラ ボ ア ジ エ (1789)</li>
<li>「元 素 周 期 表」 ─ ラ ボ ア ジ エ の 33 元 素 → メ ン デ レ ー エ フ の 周 期 表 (1869)</li>
<li>「酸 素」 と 燃 焼 ─ ラ ボ ア ジ エ</li>
<li>「化 学 式」 ─ ラ ボ ア ジ エ 由 来 の 命 名 法</li>
</ul>

<p>
化 学 の 教 科 書 を 開 い た 君 は、 1789 年 の パ リ で 始 ま っ た 革 命 ─ <strong>政 治 革 命 と 同 じ 時 期 に 起 こ っ た 化 学 革 命</strong> ─ の 遺 産 を、 直 接 受 け 取 っ て い る。
</p>

<p>
ラ ボ ア ジ エ の 頭 脳 は、 ギ ロ チ ン で 落 ち た 後 も、 こ う し て 230 年 を 超 え て、 君 の 教 室 に 生 き 続 け て い る。
</p>
