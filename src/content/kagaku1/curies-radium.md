<h3>パ リ、 ボ ロ ボ ロ の 倉 庫 ─ 1898 年</h3>

<p>
1898 年、 パ リ。<br>
廃 屋 の よ う な 古 い 倉 庫 の 中 で、 31 歳 の 一 人 の 女 性 と そ の 夫 が、 巨 大 な 鍋 で 黒 い 鉱 石 を 煮 続 け て い た。
</p>

<p>
女 性 の 名 は <strong>マ リ ・ キ ュ リ ー</strong> (Marie Curie, 1867–1934)。 ポ ー ラ ン ド か ら パ リ に 移 住 し た 物 理 学 者。<br>
夫 は <strong>ピ エ ー ル ・ キ ュ リ ー</strong> (Pierre Curie, 1859–1906)、 フ ラ ン ス の 物 理 学 者。
</p>

<p>
彼 ら は、 ピ ッ チ ブ レ ン ド (黒 い ウ ラ ン 鉱 石) か ら、 新 し い 元 素 を 単 離 し よ う と し て い た。
</p>

<h3>放 射 能 と い う 新 概 念</h3>

<p>
1896 年、 フ ラ ン ス の 物 理 学 者 ア ン リ ・ ベ ク レ ル (Henri Becquerel, 1852–1908) が、 ウ ラ ン 化 合 物 が 自 発 的 に 「奇 妙 な 光 線」 を 放 つ こ と を 発 見 し た。
</p>

<p>
マ リ ・ キ ュ リ ー は、 こ の 現 象 に 興 味 を 持 ち、 博 士 論 文 の 主 題 と し て 選 ん だ。<br>
彼 女 は こ の 現 象 を 「<strong>radioactivity</strong>」 (放 射 能) と 命 名 し た。 ラ テ ン 語 で「光 線 を 出 す 性 質」 の 意。
</p>

<p>
マ リ は、 ウ ラ ン 以 外 に も 放 射 能 を 持 つ 物 質 が あ る か を 調 べ た。<br>
そ し て、 ピ ッ チ ブ レ ン ド (黒 い ウ ラ ン 鉱 石) が、 ウ ラ ン 単 体 よ り も 強 い 放 射 能 を 持 つ こ と を 発 見 し た。
</p>

<p>
彼 女 は こ う 結 論 し た:<br>
「<strong>ピ ッ チ ブ レ ン ド に は、 ウ ラ ン よ り も 強 力 な 放 射 性 元 素 が 含 ま れ て い る は ず</strong>」
</p>

<h3>1 ト ン か ら 0.1 g ─ 4 年 の 苦 闘</h3>

<p>
新 元 素 を 単 離 す る た め、 キ ュ リ ー 夫 妻 は 巨 大 な ピ ッ チ ブ レ ン ド の 山 を 処 理 し た。
</p>

<ul>
<li>原 料: 約 <strong>1 ト ン</strong> の ピ ッ チ ブ レ ン ド</li>
<li>作 業: 巨 大 な 鉄 鍋 で 化 学 処 理 を 繰 り 返 す</li>
<li>期 間: <strong>4 年 間</strong></li>
<li>結 果: <strong>0.1 g</strong> の 純 粋 な ラ ジ ウ ム を 単 離</li>
</ul>

<p>
作 業 場 所 は、 パ リ の ソ ル ボ ン ヌ 大 学 が 提 供 し た、 ボ ロ ボ ロ の 古 い 解 剖 室 跡 の 倉 庫。<br>
冬 は 寒 く、 雨 漏 り も し た。 マ リ は、 「我 々 の 黄 金 時 代 だ っ た」 と 振 り 返 っ て い る。
</p>

<p>
1898 年 7 月 ─ <strong>ポ ロ ニ ウ ム</strong> (Po) を 発 見 (マ リ の 祖 国「ポ ー ラ ン ド」 に 因 む)。<br>
1898 年 12 月 ─ <strong>ラ ジ ウ ム</strong> (Ra) を 発 見。
</p>

<p>
こ の 一 年 で、 二 つ の 新 元 素 が 周 期 表 に 加 わ っ た。
</p>

<h3>1903 年 ノ ー ベ ル 物 理 学 賞</h3>

<p>
1903 年、 ノ ー ベ ル 物 理 学 賞 が、 キ ュ リ ー 夫 妻 と ベ ク レ ル に 与 え ら れ た。
</p>

<p>
だ が ─ ス ウ ェ ー デ ン 王 立 科 学 ア カ デ ミ ー は、 当 初 「マ リ は 単 な る 夫 の 補 助 」 と し て、 マ リ を 受 賞 者 か ら 外 そ う と し た。<br>
ピ エ ー ル は こ れ に 強 く 抗 議 し、 「妻 の 貢 献 を 認 め ら れ な け れ ば、 私 も 受 賞 を 辞 退 す る」 と 主 張。
</p>

<p>
結 果、 マ リ は <strong>史 上 初 の 女 性 ノ ー ベ ル 賞 受 賞 者</strong> と な っ た。
</p>

<h3>1906 年 ─ ピ エ ー ル の 死</h3>

<p>
1906 年 4 月 19 日、 ピ エ ー ル ・ キ ュ リ ー は パ リ の 雨 の 街 で、 馬 車 に 轢 か れ て 急 死 し た。 46 歳。
</p>

<p>
マ リ は 38 歳 で 未 亡 人 と な っ た。<br>
だ が 彼 女 は 研 究 を 続 け、 ソ ル ボ ン ヌ 大 学 物 理 学 教 授 と な る (フ ラ ン ス の 大 学 で 史 上 初 の 女 性 教 授)。
</p>

<h3>1911 年 ─ 二 度 目 の ノ ー ベ ル 賞</h3>

<p>
1911 年、 マ リ は 単 独 で <strong>ノ ー ベ ル 化 学 賞</strong> を 受 賞。 ラ ジ ウ ム と ポ ロ ニ ウ ム の 発 見、 ラ ジ ウ ム の 単 離 と 化 学 的 性 質 の 研 究 で。
</p>

<p>
こ の 受 賞 で、 マ リ は <strong>「異 な る 二 つ の 科 学 分 野 (物 理・化 学) で ノ ー ベ ル 賞 を 二 度 受 賞 し た 唯 一 の 女 性」</strong>、 そ し て 男 女 を 通 じ て も 数 少 な い 「二 度 受 賞 者」 と な っ た。
</p>

<p>
ノ ー ベ ル 賞 を 二 度 受 賞 し た 人 物 は、 マ リ ・ キ ュ リ ー の 他 に は ─
</p>

<ul>
<li>ラ イ ナ ス ・ ポ ー リ ン グ (化 学 1954 + 平 和 1962)</li>
<li>ジ ョ ン ・ バ ー デ ィ ン (物 理 1956 + 1972 ─ 同 じ 分 野 で 二 度)</li>
<li>フ レ デ リ ッ ク ・ サ ン ガ ー (化 学 1958 + 1980 ─ 同 じ 分 野 で 二 度)</li>
</ul>

<p>
こ れ ら の 中 で、 マ リ だ け が「異 分 野 で 二 度」 か つ「女 性」 で あ る。
</p>

<h3>ラ ジ ウ ム の 影 ─ 自 ら の 命</h3>

<p>
ラ ジ ウ ム の 発 見 当 初、 「奇 跡 の 治 療 物 質」 と 考 え ら れ、 医 療 ・ 化 粧 品 ・ 飲 料 水 ま で に 添 加 さ れ た。
</p>

<p>
だ が 1920s 以 降、 放 射 線 の 強 い 発 が ん 性 が 判 明。<br>
マ リ 自 身 も、 30 年 以 上 ラ ジ ウ ム を 素 手 で 扱 い、 ノ ー ト に 触 れ、 ポ ケ ッ ト に 試 料 を 入 れ て い た。
</p>

<p>
1934 年 7 月 4 日、 マ リ ・ キ ュ リ ー は <strong>再 生 不 良 性 貧 血</strong> (放 射 線 障 害 の 一 つ) で 死 去。 67 歳。
</p>

<p>
彼 女 が 残 し た 研 究 ノ ー ト は、 今 で も 強 い 放 射 能 を 帯 び て お り、 鉛 の 容 器 に 保 管 さ れ て い る。<br>
半 減 期 約 1600 年 の ラ ジ ウ ム の 影 響 で、 こ の ノ ー ト は <strong>あ と 1500 年 ほ ど 安 全 に 触 れ ら れ な い</strong> と さ れ る。
</p>

<h3>女 性 科 学 者 の 先 駆 け と し て</h3>

<p>
1867 年 ポ ー ラ ン ド 生 ま れ の マ リ は、 当 時 の 女 性 と し て は 異 例 の 道 を 歩 ん だ:
</p>

<ul>
<li>当 時 の ポ ー ラ ン ド の 大 学 は 女 性 入 学 禁 止 → 24 歳 で パ リ に 移 住 し ソ ル ボ ン ヌ で 学 ぶ</li>
<li>女 性 と し て 史 上 初 の ノ ー ベ ル 賞 (1903)</li>
<li>女 性 と し て 史 上 初 の ソ ル ボ ン ヌ 大 学 教 授 (1906)</li>
<li>女 性 と し て 唯 一 の「二 度 受 賞」</li>
<li>娘 イ レ ー ヌ ・ ジ ョ リ オ ・ キ ュ リ ー も 1935 年 ノ ー ベ ル 化 学 賞 受 賞 ─ 「ノ ー ベ ル 賞 一 家」</li>
</ul>

<p>
マ リ は、 「女 性 が 科 学 を す る」 こ と が 不 可 能 と さ れ て い た 時 代 に、 そ の 道 を 切 り 開 い た 一 人 で あ る。
</p>

<p>
彼 女 の 言 葉:
</p>

<blockquote>
「人 生 に お い て 恐 れ る べ き こ と は 何 一 つ あ り ま せ ん。 す べ て は た だ 理 解 さ れ る べ き こ と で す」<br>
─ マ リ ・ キ ュ リ ー
</blockquote>

<p>
化 学 を 学 ぶ と き、 こ の「日 本 の 江 戸 の 鈴 木 梅 太 郎 (本 サ イ ト kagaku1) と 同 時 代 を 生 き た 一 人 の 女 性 科 学 者」 の 仕 事 と 生 涯 を、 心 に 留 め て お い て ほ し い。
</p>
