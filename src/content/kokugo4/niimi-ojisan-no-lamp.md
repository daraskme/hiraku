<h3>30 年 後 ─ お じ い さ ん が 孫 に 語 る</h3>

<p>
物 語 は、 お じ い さ ん が 孫 に 語 る 形 で 進 み ま す。 そ の 内 容 は ─ お じ い さ ん が ま だ 若 者 だ っ た 頃、
ラ ン プ 屋 と し て ど う や っ て 暮 ら し て き た か、 と い う 話 で す。
</p>

<p>
若 い 頃 の お じ い さ ん の 名 前 は 巳 之 助 (み の す け)。 貧 し い 少 年 で し た。
</p>

<h3>巳 之 助 ─ ラ ン プ 屋 と し て 成 功 す る</h3>

<p>
巳 之 助 は 一 生 懸 命 働 き、 ラ ン プ を 売 る 商 売 で 成 功 し ま し た。 結 婚 し、 家 族 を 持 ち、
家 を 建 て ま し た。
</p>

<p>
明 治 ~ 大 正 期、 ラ ン プ は 一 家 に 一 つ ・ 二 つ 必 要 な 大 切 な 道 具 で し た。 暗 い 夜 道 を 照 ら し、
家 族 で 夕 食 を 囲 む 灯 り。 巳 之 助 が 売 る ラ ン プ は、 多 く の 家 庭 を 明 る く し ま し た。
</p>

<h3>電 気 が 来 る ─ 文 明 開 化 の 後 期</h3>

<p>
や が て、 巳 之 助 の 町 に 電 気 (電 灯) が 来 ま し た。 電 灯 は ラ ン プ よ り も 明 る く、 安 全 で、
し か も 燃 料 を 補 充 し な く て も 良 い。 一 度 ス イ ッ チ を 入 れ れ ば つ き っ ぱ な し。
</p>

<p>
町 の 人 は み ん な 喜 ん で 電 灯 を 取 り 付 け ま し た。 ラ ン プ は 売 れ な く な り ま し た。
</p>

<h3>怒 り の 巳 之 助</h3>

<p>
巳 之 助 は、 怒 り に 燃 え ま し た。 「電 気 屋 が 来 た せ い で、 俺 の 商 売 が 駄 目 に な っ た!」
彼 は 電 気 屋 を 恨 み、 一 時 は 殺 そ う と さ え 思 い ま し た。
</p>

<p>
火 を 燃 や し、 電 気 屋 の 家 を 焼 こ う か ─ そ こ ま で 思 い 詰 め ま し た。
</p>

<h3>気 付 き ─ 「時 代 を 恨 ん で も 仕 方 が な い」</h3>

<p>
そ の 怒 り の 中、 巳 之 助 は ふ と 気 付 き ま し た ─ 「電 気 屋 が 悪 い わ け で は な い。 時 代 が 変 わ っ た
の だ。 古 い も の は、 新 し い も の に 譲 る の が 道 理 だ」。
</p>

<p>
ラ ン プ も、 か つ て は 行 灯 (あ ん ど ん) を 駆 逐 し た。 そ し て 今、 ラ ン プ が 電 灯 に 駆 逐 さ れ る ─
こ れ が 時 代 の 流 れ。
</p>

<h3>池 で ラ ン プ を 沈 め る ─ 最 後 の 別 れ</h3>

<p>
巳 之 助 は 大 き な 決 心 を し ま し た。 家 中 の ラ ン プ を 集 め、 池 へ 運 ん で い っ た の で す。
</p>

<p>
池 に 着 い た 巳 之 助 は、 ラ ン プ を 一 つ ず つ 灯 し、 そ し て 池 に 沈 め て い き ま し た。 静 か な 夜 の 水 面 に、
最 後 の ラ ン プ の 灯 り が 揺 ら ぎ、 消 え て い く ─ 美 し い、 そ し て 悲 し い 場 面 で す。
</p>

<p>
こ う し て 巳 之 助 は ラ ン プ 屋 を 廃 業 し、 別 の 仕 事 を 始 め ま し た。 そ し て 何 十 年 も 経 っ て、
今、 お じ い さ ん と し て 孫 に そ の 話 を 語 っ て い る の で す。
</p>

<h3>新 美 南 吉 ─ 30 歳 で 早 世 し た 児 童 文 学 の 巨 匠</h3>

<p>
こ の 物 語 を 書 い た 新 美 南 吉 (1913–1943) は、 30 歳 で 結 核 で 早 世 し ま し た。 短 い 生 涯 で、
『ご ん ぎ つ ね』『手 袋 を 買 い に』『お じ い さ ん の ラ ン プ』 等、 多 く の 名 作 を 残 し ま し た。
</p>

<p>
南 吉 の 作 品 に 共 通 す る の は、 「悲 し い 場 面 の 中 に、 命 へ の 優 し さ ・ 受 容 が あ る」 こ と。
『ご ん ぎ つ ね』 で 兵 十 が 加 助 を 銃 で 撃 つ 場 面、『手 袋 を 買 い に』 で 子 ぎ つ ね が 人 間 の 世 界 に
触 れ る 場 面 ─ 全 て に 共 通 す る 優 し さ で す。
</p>

<h3>結 び ─ 時 代 の 変 化 を 受 け 入 れ る</h3>

<p>
現 代 で も、 時 代 は 急 速 に 変 わ り ま す。 ス マ ホ が 出 て、 街 の カ メ ラ 屋 ・ レ ン タ ル ビ デ オ 屋 が
減 り ま し た。 ネ ッ ト 通 販 が 普 及 し て、 商 店 街 が 寂 し く な り ま し た。
</p>

<p>
そ の よ う な 変 化 を、 巳 之 助 の よ う に「怒 る」 の で は な く、「受 け 入 れ て、 次 へ 進 む」 こ と が
で き る か。 新 美 南 吉 が 君 に 投 げ か け る 大 切 な 問 い で す。
</p>
