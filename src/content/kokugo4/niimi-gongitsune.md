<h3>本文 (『ごんぎつね』新美南吉・冒頭)</h3>

<blockquote>
<p>
こ れ は、 私 が 小 さ い と き に、 村 の 茂 平 (も へ い) と い う お じ い さ ん か ら 聞 い た お 話 で す。
</p>

<p>
昔 (む か し)、 私 た ち の 村 の 近 く の、 中 山 (な か や ま) と い う と こ ろ に、 小 さ な お 城 が あ っ て、 中 山 様 (な か や ま さ ま) と い う お 殿 様 (と の さ ま) が お ら れ た そ う で す。
</p>

<p>
そ の 中 山 か ら、 少 し 離 れ た 山 の 中 に、 「ご ん ぎ つ ね」 と い う 狐 (き つ ね) が い ま し た。 ご ん は、 一 人 ぼ っ ち の 小 狐 で、 し だ の 一 ぱ い し げ っ た 森 の 中 に、 穴 を ほ っ て 住 ん で い ま し た。
</p>

<p>
そ し て、 夜 で も 昼 で も、 あ た り の 村 へ 出 て 来 て、 い た ず ら ば か り し ま し た。
</p>

<p>
畑 へ 入 っ て 芋 を 掘 り ち ら し た り、 菜 種 が ら の、 ほ し て あ る の へ 火 を つ け た り、 百 姓 家 の 裏 手 に つ る し て あ る、 と ん が ら し を む し り 取 っ て、 い っ た り、 い ろ ん な こ と を し ま し た。
</p>

<p>
あ る 秋 の こ と で し た。 二、 三 日 雨 (あ め) が ふ り 続 い た そ の あ い だ、 ご ん は、 外 へ も 出 ら れ な く て、 穴 の 中 に し ゃ が ん で い ま し た…
</p>
</blockquote>

<h3>解 説 ─ 「私 が 聞 い た お 話」 と い う 入 り 口</h3>

<p>
「ご ん ぎ つ ね」 の 冒 頭 は、 特 に 印 象 的 で あ る。
</p>

<blockquote>
<strong>「こ れ は、 私 が 小 さ い と き に、 村 の 茂 平 と い う お じ い さ ん か ら 聞 い た お 話 で す」</strong>
</blockquote>

<p>
普 通 の 物 語 な ら、 い き な り 「昔、 ご ん と い う 狐 が い ま し た」 と 始 ま る。<br>
だ が 新 美 南 吉 は、 「<strong>私 が 聞 い た お 話</strong>」 と い う ワ ン ク ッ シ ョ ン を 置 い た。
</p>

<p>
こ れ に は、 大 切 な 効 果 が あ る:
</p>

<ul>
<li><strong>物 語 が 「私」 と い う 人 物 を 通 し て 伝 え ら れ た</strong> ─ 直 接 の 出 来 事 で は な く、 「昔 か ら 伝 え ら れ て き た お 話」 と し て の 重 み が 生 ま れ る。</li>
<li><strong>茂 平 と い う お じ い さ ん の 存 在</strong> ─ 物 語 の 後 ろ に、 こ の 話 を 大 切 に 語 り 継 い で き た 人 が い る こ と が わ か る。</li>
<li><strong>「あ る か も し れ な い」 と い う リ ア リ テ ィ</strong> ─ 「茂 平 さ ん が 知 っ て い る」 と 書 か れ る だ け で、 物 語 が「ど こ か で 本 当 に あ っ た こ と」 の よ う に 感 じ ら れ る。</li>
</ul>

<p>
わ ず か 一 行 の 工 夫 で、 こ の 物 語 は <strong>「お と ぎ 話」 で あ り な が ら「本 当 に あ っ た こ と」</strong> の よ う な、 不 思 議 な 魅 力 を 持 つ。
</p>

<h3>18 歳 の 天 才 ─ 新 美 南 吉</h3>

<p>
こ の 物 語 を 書 い た 時、 新 美 南 吉 は <strong>わ ず か 18 歳</strong> だ っ た。
</p>

<p>
彼 は 愛 知 県 半 田 市 (は ん だ し) で 生 ま れ、 4 歳 で 母 を 亡 く し、 父 が 再 婚 し た 後 も 寂 し い 子 ど も 時 代 を 過 ご し た。<br>
小 学 校 ・ 中 学 校 と 文 学 が 好 き で、 18 歳 で 「ご ん ぎ つ ね」 を 書 き 上 げ、 雑 誌 『赤 い 鳥』 (1932 年 1 月 号) に 採 用 さ れ た。
</p>

<p>
そ の 後、 彼 は 小 学 校 教 員 と し て 働 き な が ら、 「手 袋 を 買 い に」 「お じ い さ ん の ラ ン プ」 な ど、 多 く の 名 作 を 残 し た。<br>
だ が、 29 歳 の 若 さ で <strong>結 核</strong> (け っ か く) の た め 亡 く な っ た。
</p>

<p>
わ ず か 29 年 の 短 い 生 涯 で、 新 美 南 吉 は、 日 本 の 児 童 文 学 史 に 永 遠 に 刻 ま れ る 仕 事 を 残 し た の で あ る。
</p>

<h3>「ご ん ぎ つ ね」 の 主 題 ─ す れ ち が い と 後 悔</h3>

<p>
「ご ん ぎ つ ね」 全 編 を 読 む と、 こ の 物 語 が 描 い て い る の は ─ <strong>「す れ ち が い」 と 「後 悔」</strong> で あ る。
</p>

<ul>
<li>ご ん は、 兵 十 (ひ ょ う じ ゅ う) の 母 親 が 亡 く な っ た こ と を 知 り、 自 分 が 兵 十 に い た ず ら し た こ と を 後 悔 し、 償 い と し て 毎 日 栗 や 松 茸 を こ っ そ り 兵 十 の 家 に 届 け る。</li>
<li>だ が 兵 十 は、 そ れ に 気 づ か ず、 い た ず ら ぎ つ ね だ と 思 っ て、 ご ん を 鉄 砲 で 撃 っ て し ま う。</li>
<li>撃 っ た 後、 兵 十 は 家 の 中 に 栗 を 見 つ け、 ご ん が 償 い に 来 て い た こ と に 気 づ く ─ だ が、 ご ん は も う 死 ん で し ま っ た 後 だ っ た。</li>
</ul>

<p>
「<strong>お い、 ご ん、 お ま え だ っ た の か</strong>」 ─ 兵 十 が こ の 言 葉 を 言 っ た 時、 ご ん は も う 何 も 答 え ら れ な い。
</p>

<p>
こ の 痛 切 な ラ ス ト シ ー ン が、 日 本 の 多 く の 小 学 生 の 心 に、 永 遠 に 残 る の で あ る。
</p>

<h3>『赤 い 鳥』 と 児 童 文 学 の 黄 金 期</h3>

<p>
「ご ん ぎ つ ね」 が 載 っ た 雑 誌 『<strong>赤 い 鳥</strong>』 は、 1918 年 に 鈴 木 三 重 吉 (す ず き み え き ち) が 創 刊 し た 児 童 文 学 雑 誌 で あ る。
</p>

<p>
そ れ ま で の 日 本 の 子 ど も 向 け の 本 は、 道 徳 的 で 教 訓 的 な も の が 多 か っ た。<br>
鈴 木 三 重 吉 は こ う 考 え た:「子 ど も に も、 大 人 の 文 学 と 同 じ よ う に、 本 当 の 芸 術 を 与 え よ う」 と。
</p>

<p>
そ し て、 当 時 の 一 流 の 作 家・詩 人 を 動 員 し て:
</p>
<ul>
<li>芥 川 龍 之 介 「蜘 蛛 の 糸」「杜 子 春」</li>
<li>北 原 白 秋 の 多 く の 童 謡 詩</li>
<li>三 重 吉 自 身 の 翻 案 物 語</li>
<li>そ し て、 新 人 で あ っ た 新 美 南 吉 の 「ご ん ぎ つ ね」</li>
</ul>

<p>
こ う し た 名 作 が、 全 て 『赤 い 鳥』 か ら 生 ま れ た。<br>
日 本 の 児 童 文 学 の 黄 金 期 ─ そ れ は、 こ の 雑 誌 と 共 に あ っ た の で あ る。
</p>

<h3>全 文 を 読 ん で み よ う</h3>

<p>
「ご ん ぎ つ ね」 の 全 文 は、 青 空 文 庫 で 無 料 で 読 め る。<br>
ぜ ひ、 冒 頭 の 続 き ─ ご ん が 兵 十 と 出 会 い、 い た ず ら し、 母 親 の 葬 列 を 見 て 反 省 し、 償 い に 通 う ─ 全 編 を 通 し て 読 ん で ほ し い。
</p>

<p>
そ し て、 ラ ス ト シ ー ン の 「お い、 ご ん、 お ま え だ っ た の か」 を、 君 の 心 の 中 に 大 切 に し ま っ て お い て ほ し い。
</p>
