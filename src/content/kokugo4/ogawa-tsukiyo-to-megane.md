<h3>あらすじ</h3>

<p>
こ ぢ ん ま り と し た 田 舎 の 村 の、 一 軒 家。<br>
そ こ に、 お ば あ さ ん が 一 人 で 住 ん で い た。 細 か い 縫 い 物 を し て、 一 人 で 暮 ら し て い る、 心 や さ し い お ば あ さ ん だ っ た。
</p>

<p>
あ る 月 夜 の 晩 の こ と。<br>
コ ト コ ト と 戸 を 叩 く 音 が し た。<br>
「ど な た で す か」 と お ば あ さ ん が 戸 を 開 け る と、 そ こ に は <strong>め が ね 売 り</strong> の 行 商 人 が 立 っ て い た。
</p>

<p>
「お ば あ さ ん、 め が ね は い か が で す か。 こ れ を か け れ ば、 細 か い 仕 事 が よ く 見 え る よ う に な り ま す よ」
</p>

<p>
お ば あ さ ん は、 最 近 目 が 悪 く な っ て き て、 縫 い 物 が 大 変 だ っ た。<br>
そ こ で め が ね を 一 つ 買 う こ と に し た。
</p>

<p>
め が ね を か け て み る と ─ <strong>細 か い 縫 い 物 が、 ま る で 若 い 頃 の よ う に は っ き り 見 え た</strong>。<br>
お ば あ さ ん は 大 喜 び し て、 め が ね 売 り に お 礼 を 言 い、 そ し て、 し ば ら く 縫 い 物 を し て い た。
</p>

<p>
や が て、 月 が さ ら に 高 く 昇 っ た 頃。<br>
ま た 戸 が コ ト コ ト と 叩 か れ た。<br>
「ど な た で す か」 と 出 て み る と、 そ こ に は <strong>蝶 の よ う に 美 し い 少 女</strong> が 立 っ て い た。
</p>

<p>
少 女 は こ う 言 っ た:<br>
「お ば あ さ ん、 私 は ば ら の 花 で 指 を 刺 し て し ま い ま し た。 そ の 傷 が と て も 痛 い の で す。 ど う か 手 当 て を し て く だ さ い」
</p>

<p>
お ば あ さ ん は や さ し く 少 女 を 中 に 入 れ て、 め が ね を か け て、 そ の 指 を よ く 見 た。<br>
そ こ に は、 ば ら の 花 び ら の よ う な 美 し い 傷 が あ っ た。
</p>

<p>
お ば あ さ ん が 心 を 込 め て 手 当 て を す る と、 少 女 は に っ こ り 笑 っ て、 こ う 言 っ た:<br>
「お ば あ さ ん、 あ り が と う ご ざ い ま す」
</p>

<p>
そ し て、 少 女 は 静 か に 月 夜 の 中 へ と 消 え て い っ た。<br>
お ば あ さ ん は 戸 口 に 立 っ て、 蝶 の よ う に 飛 び 去 る 少 女 を、 い つ ま で も 見 送 っ て い た。
</p>

<p>
そ の 後、 お ば あ さ ん は 思 う:<br>
「あ の 子 は、 本 当 に 人 間 の 子 だ っ た の だ ろ う か。 そ れ と も、 ば ら の 花 か ら 抜 け 出 し て き た、 妖 精 だ っ た の だ ろ う か」
</p>

<p>
め が ね が な け れ ば、 あ ん な に 美 し い 少 女 の 顔 も、 細 か い 傷 も、 見 え な か っ た こ と だ ろ う。<br>
そ し て、 月 夜 が な け れ ば、 あ の 不 思 議 な 出 会 い も、 起 こ ら な か っ た こ と だ ろ う。
</p>

<p>
お ば あ さ ん の 心 に は、 そ の 夜 の 月 と、 蝶 の よ う な 少 女 の 笑 顔 が、 い つ ま で も 残 っ て い た。
</p>

<h3>解 説 ─ 「日 本 の ア ン デ ル セ ン」</h3>

<p>
小 川 未 明 (1882–1961) は、 日 本 児 童 文 学 史 で 最 も 重 要 な 作 家 の 一 人 で あ る。<br>
「<strong>日 本 児 童 文 学 の 父</strong>」「<strong>日 本 の ア ン デ ル セ ン</strong>」 と 呼 ば れ、 短 い 童 話 を 数 多 く 書 い た。
</p>

<p>
代 表 作:
</p>

<ul>
<li>「赤 い ろ う そ く と 人 魚」 (1921) ─ 本 サ イ ト kokugo3 で 既 習</li>
<li>「月 夜 と め が ね」 (1922) ─ 本 教 材</li>
<li>「野 ば ら」 (1920)</li>
<li>「金 の 輪」 (1919)</li>
</ul>

<p>
未 明 の 童 話 の 特 徴 は、 <strong>静 か な 美 し さ と、 ど こ か 寂 し い 結 末</strong> で あ る。<br>
派 手 な 冒 険 や 戦 い は な く、 静 か な 出 会 い と 別 れ。 そ し て、 そ こ に 漂 う 神 秘 と 詩 情。<br>
こ の 雰 囲 気 が、 デ ン マ ー ク の ア ン デ ル セ ン と 似 て い る と 言 わ れ た。
</p>

<h3>「月 夜」 と い う 舞 台</h3>

<p>
こ の 物 語 で、 「月 夜」 が 特 別 な 役 割 を 果 た し て い る。
</p>

<p>
昼 間 で は な く、 月 夜。<br>
雨 の 夜 で も な く、 月 が 明 る く 照 る 夜。
</p>

<p>
こ の 「月 明 か り の 静 け さ」 が、 普 通 で な い 出 来 事 が 起 こ る <strong>境 界 (き ょ う か い) の 時 間</strong> を 作 る。<br>
昼 と 夜、 現 実 と 夢、 人 間 と 妖 精 ─ そ の 境 界 が 一 瞬 だ け 曖 昧 に な る 時 間 が、 月 夜 で あ る。
</p>

<h3>「め が ね」 と い う 象 徴</h3>

<p>
題 名 に も あ る 「<strong>め が ね</strong>」 は、 こ の 物 語 の 重 要 な 道 具 だ。
</p>

<p>
め が ね は、 「<strong>見 え な か っ た も の が 見 え る よ う に な る</strong>」 道 具 で あ る。<br>
お ば あ さ ん は、 め が ね を か け る こ と で、 細 か い 縫 い 物 が 見 え る よ う に な っ た。<br>
そ し て、 蝶 の よ う な 少 女 の 細 か い 傷 も、 見 え る よ う に な っ た。
</p>

<p>
し か し、 め が ね は 物 理 的 な 視 力 だ け で な く、「<strong>心 の 視 力</strong>」 も 象 徴 し て い る か も し れ な い。<br>
普 段 見 過 ご し て い る 美 し い も の、 不 思 議 な も の、 や さ し い も の ─ そ れ ら を 「見 る」 力 を、 め が ね は 与 え て く れ た。
</p>

<h3>「蝶 の よ う な 少 女」 の 正 体</h3>

<p>
お ば あ さ ん の 家 を 訪 れ た 少 女 は、 本 当 に 人 間 だ っ た の だ ろ う か。
</p>

<p>
ば ら の 花 で 指 を 刺 し た こ と、 蝶 の よ う に 美 し い 姿、 月 夜 の 中 へ 消 え て い っ た こ と ─ こ れ ら の 状 況 か ら、 少 女 は <strong>ば ら の 花 か ら 生 ま れ た 妖 精</strong> だ っ た の か も し れ な い。
</p>

<p>
小 川 未 明 は、 答 え を 明 確 に 書 か な い。<br>
「人 間 だ っ た か も し れ な い し、 妖 精 だ っ た か も し れ な い」 と、 読 者 の 想 像 に 委 ね る。
</p>

<p>
こ の <strong>「は っ き り 答 え を 出 さ な い」</strong> 余 韻 こ そ が、 未 明 童 話 の 魅 力 で あ る。
</p>

<h3>『赤 い 鳥』 の 系 譜</h3>

<p>
「月 夜 と め が ね」 は、 雑 誌 『<strong>赤 い 鳥</strong>』 の 大 正 11 年 (1922) 5 月 号 に 掲 載 さ れ た。
</p>

<p>
『赤 い 鳥』 は、 鈴 木 三 重 吉 が 1918 年 に 創 刊 し た 児 童 文 学 雑 誌。<br>
こ の 雑 誌 に 掲 載 さ れ た 名 作 を 並 べ る と:
</p>

<ul>
<li>1918 年 7 月 創 刊 号 ─ 芥 川 龍 之 介 「蜘 蛛 の 糸」 (本 サ イ ト chugaku2)</li>
<li>1919 年 ─ 小 川 未 明 「金 の 輪」</li>
<li>1920 年 ─ 小 川 未 明 「野 ば ら」</li>
<li>1921 年 ─ 小 川 未 明 「赤 い ろ う そ く と 人 魚」 (本 サ イ ト kokugo3)</li>
<li><strong>1922 年 ─ 小 川 未 明 「月 夜 と め が ね」 (本 教 材)</strong></li>
<li>1932 年 ─ 新 美 南 吉 「ご ん ぎ つ ね」 (本 サ イ ト kokugo3)</li>
</ul>

<p>
大 正 ~ 昭 和 初 期 の 日 本 児 童 文 学 の 黄 金 期 ─ そ の 中 心 に い た の が、 小 川 未 明 と『赤 い 鳥』 だ っ た。
</p>

<h3>全 文 を 読 ん で み よ う</h3>

<p>
「月 夜 と め が ね」 の 全 文 は、 青 空 文 庫 で 無 料 で 読 め る。<br>
今 回 の 紹 介 は あ ら す じ ベ ー ス だ が、 ぜ ひ 原 文 で、 小 川 未 明 が 描 い た 月 夜 の 静 け さ と 蝶 の 少 女 の 美 し さ を、 味 わ っ て み て ほ し い。
</p>

<p>
読 み 終 え た 時、 君 の 心 に も、 100 年 前 の あ の 月 夜 の 風 景 が、 静 か に 浮 か び 上 が っ て く る は ず だ。
</p>
