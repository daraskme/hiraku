<h3>現代語訳</h3>

<p>
ふるさと とは、遠く 離れた ところに 居て こそ、想い 出す ものだ。<br>
そして、悲しく 詩 (うた) と して 歌う ものだ。
</p>

<p>
たとえ、私 が 落ちぶれて、見知らぬ 土地 で 乞食 (こじき) に 身 を 落として しまった と しても ─<br>
ふるさと は、私 の 「帰る べき 場所」 で は ない だろう。
</p>

<p>
ひとり、都 (= 東京) の 夕暮れ に、ふるさと を 思って、涙 ぐむ ─<br>
その 気持ち を 抱いた まま、また、遠く 離れた 都 (東京) に 帰ろう。<br>
遠く 離れた 都 に、帰ろう。
</p>

<h3>「ふるさとは 遠きにありて 思ふもの」 という 逆説</h3>

<p>
詩 の 第一行 「<strong>ふるさと は 遠きにありて 思ふもの</strong>」 ─ これ は、日本人 が 「ふるさと」 という 言葉 で 感じる もの の、もっとも 本質 的 な 部分 を 突いて いる。
</p>

<p>
ふるさと は、<strong>遠く 離れて いる から こそ</strong>、思い 出される。<br>
近く に 住んで いる とき、私 たち は 自分 の 故郷 を、ただ 当たり前 の 風景 と して しか 見ない。<br>
だが、ひとたび 都市 (東京 や 大阪) に 出て、何ヶ月、何年 が 経つ と ─ 故郷 は、その 距離 の 分 だけ、<strong>美しく 輝き 始める</strong>。
</p>

<p>
これ は、近代 日本人 の 「故郷喪失」 を 象徴 する。<br>
明治 維新 (1868) 以降、多くの 日本人 が、農村 から 都市 へ と 移動 した。彼ら にとって、「ふるさと」 は ── 帰る べき 実在 の 場所 で は なく、東京 の 夕暮れ で 思い 出す<strong>「心 の 中 の 風景」</strong> に なった。
</p>

<h3>「帰る ところに あるまじや」 ─ 痛み の 中心</h3>

<p>
だが、詩 の 中盤 で、犀星 は 衝撃 的 な こと を 言う ──<br>
「<strong>たとえ 私 が 乞食 に なって しまった と しても、ふるさと は、私 の 帰る べき 場所 で は ない</strong>」。
</p>

<p>
なぜ?<br>
なぜ、犀星 にとって 「ふるさと」 は、思い 出す もの で は あって も、帰る 場所 で は ない の か?
</p>

<p>
ここ に、犀星 自身 の 苦しい 出自 が 影 を 落として いる。
</p>

<h3>室生犀星 という 人</h3>

<p>
室生犀星 (1889 – 1962) は、石川県 金沢 で、加賀 藩士 の 父 と、その 召使 だった 母 の 間 に 生まれた。<br>
だが、生母 と は すぐ に 引き離され、雨宝院 (うほういん) という 寺 の 養子 と なって 育った。<br>
養家 の 暮らし は 厳しく、犀星 は 学校 で も いじめ に 遭い、複雑な コンプレックス を 抱えながら 育った。
</p>

<p>
彼 にとって 「ふるさと 金沢」 は、<strong>美しい 場所</strong> である と 同時 に、<strong>苦しい 思い出 の 場所</strong> でも あった。
</p>

<p>
17 歳 で 東京 に 出て、詩 を 書き、その 才能 を 認められた 犀星 は、北原白秋・萩原朔太郎 と 並ぶ 大正 詩壇 の 中心 的 存在 と なった。<br>
しかし、東京 で 成功 した 後 も、彼 は ふるさと 金沢 と の 距離感 を、生涯 持ち 続けた。
</p>

<p>
「ふるさと は 遠きにありて 思ふもの ─ そして 悲しく うたふ もの」<br>
これは、犀星 の 個人 的 な 真実 で あった と 同時 に、近代 日本人 の <strong>共通 の 心情</strong> で も あった。
</p>

<h3>同時代 の 「ふるさと」 詩</h3>

<p>
室生犀星 「ふるさと は 遠きにありて 思ふもの」 (1918)。<br>
同 時代 の 「ふるさと」 を 歌った 詩 を 並べて みよう。
</p>

<ul>
<li>1909 ─ 北原白秋 「邪宗門」 (異国 趣味)</li>
<li>1914 ─ 高村光太郎 『道程』 (自我 確立)</li>
<li>1917 ─ 萩原朔太郎 『月に吠える』 (鬱屈)</li>
<li>1918 ─ 室生犀星 『抒情小曲集』 (郷愁)</li>
<li>1923 ─ 高村光太郎 「冬が来た」 (本サイト kokugo6)</li>
</ul>

<p>
大正 期 は、日本 近代 詩 の 黄金 期。<br>
それぞれ の 詩人 が、それぞれ の 「ふるさと」 と 「都市」 の 間 の 緊張 を、自分 の 言葉 で 歌った。
</p>

<h3>「遠き みやこ に かへらばや」 ─ 都市 への 帰還</h3>

<p>
最後 の 二行 ─ 「<strong>遠き みやこ に かへらばや、遠き みやこ に かへらばや</strong>」 ─ は、繰り返さ れる こと で、リズム と 余韻 を 残す。
</p>

<p>
犀星 は、結局、ふるさと へ 帰らず、東京 で 暮らし 続ける。<br>
そして、東京 の 夕暮れ で、ふるさと を 思って 涙 ぐむ。<br>
だが その 涙 こそ が、彼 を 詩人 と して 生かす ─ それ こそ が、芸術 家 と して の 彼 の 生き方 だった の だ。
</p>

<p>
「ふるさと は 遠きにありて 思ふもの」 ─ この 一行 は、100 年 経った 今 も、東京・大阪 など の 都市 で 働く 日本人 の 心 の 風景 を、見事 に 言い 当てて いる。
</p>
