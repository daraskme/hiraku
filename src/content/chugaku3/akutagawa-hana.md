<h3>あらすじ</h3>

<p>
京 都 の 山 科 (や ま し な) の 池 の 尾 (い け の お) と い う 寺 に、 <strong>禅 智 内 供</strong> (ぜ ん ち な い ぐ) と い う 高 僧 が い た。<br>
彼 に は 一 つ、 大 き な 悩 み が あ っ た。 そ れ は、 顎 (あ ご) の 下 ま で 垂 れ 下 が る、 五 ・ 六 寸 (約 15–18 cm) も あ る 異 様 に 長 い 鼻 で あ っ た。
</p>

<p>
こ の 鼻 の せ い で、 内 供 は 食 事 を す る の に も 不 自 由 し て い た (鼻 を 弟 子 に 持 ち 上 げ て も ら う 必 要 が あ っ た)。<br>
だ が 何 よ り も 辛 い の は、 鼻 を 他 人 に 笑 わ れ る こ と だ っ た。 表 向 き は 「私 は こ の 鼻 を 気 に し て い な い」 と 装 い な が ら、 内 心 は ず っ と 苦 し み 続 け て き た。
</p>

<p>
あ る 日、 弟 子 が 「鼻 を 短 く す る 方 法」 を 中 国 か ら 持 ち 帰 っ た。<br>
熱 い 湯 で 茹 で、 踏 ま せ て 油 を 絞 り 出 す、 と い う 荒 治 療 で あ る。<br>
内 供 は そ の 方 法 を 試 し、 見 事 に 鼻 を 普 通 の 大 き さ に 縮 め る こ と に 成 功 し た。
</p>

<p>
こ れ で 笑 わ れ な く な る、 と 内 供 は 喜 ん だ。<br>
だ が ─ 鼻 が 短 く な っ た 後、 周 り の 人 々 は <strong>以 前 よ り も っ と 露 骨 に 笑 う</strong> よ う に な っ た の だ。
</p>

<p>
内 供 は ま っ た く 理 解 で き な か っ た。 な ぜ 普 通 の 鼻 に な っ た の に、 か え っ て 笑 わ れ る の か?
</p>

<p>
芥 川 は こ こ で、 鋭 く 人 間 の 心 を 暴 く。
</p>

<blockquote>
「<strong>人 間 の 心 に は、 互 い に 矛 盾 す る 二 つ の 感 情 が あ る。 も と よ り 他 人 の 不 幸 に 同 情 し な い 者 は な い。 し か し、 そ の 人 が、 ど う に か し て そ の 不 幸 を 切 り 抜 け る こ と が で き る と、 こ ち ら は な ん と な く 物 足 り な い 気 持 ち が す る</strong>」
</blockquote>

<p>
内 供 は、 そ の 「物 足 り な い 気 持 ち」 を、 自 分 が 周 り か ら 向 け ら れ て い る こ と に 気 づ い た。<br>
そ し て、 い つ し か 内 供 は ─ <strong>元 の 長 い 鼻 に 戻 り た い</strong> と 思 う よ う に な っ た。
</p>

<p>
あ る 朝、 目 を 覚 ま す と、 不 思 議 な こ と に、 内 供 の 鼻 は <strong>元 の 五 ・ 六 寸 の 長 い 鼻 に 戻 っ て い た</strong>。<br>
内 供 は こ う 呟 い た:「<strong>こ う な れ ば、 も う 誰 も 笑 う ま い</strong>」 と。
</p>

<h3>解 説 ─ 出 世 作 と し て の 「鼻」</h3>

<p>
芥 川 龍 之 介 (あ く た が わ り ゅ う の す け) は、 大 正 5 年 (1916) 2 月、 東 京 帝 国 大 学 英 文 科 4 年 生 (23 歳) の と き、 雑 誌 『新 思 潮』 に こ の 「鼻」 を 発 表 し た。
</p>

<p>
こ れ を 読 ん だ <strong>夏 目 漱 石</strong> は、 芥 川 に 直 接 手 紙 を 書 き、 こ う 絶 賛 し た:
</p>

<blockquote>
「あ な た の も の は 大 変 面 白 い と 思 い ま す ─ 落 ち 着 い て 真 面 目 に 不 真 面 目 を や っ て ゐ る 所 が 大 変 上 等 で す ─ 鼻 一 本 で あ ん な も の を こ し ら え る の は 中 々 普 通 の 人 に 出 来 る 事 で は あ り ま せ ん」<br>
─ 夏 目 漱 石 か ら 芥 川 龍 之 介 へ の 手 紙 (1916 年 2 月 19 日)
</blockquote>

<p>
日 本 文 壇 の 最 高 権 威 だ っ た 漱 石 か ら の こ の 絶 賛 は、 芥 川 の 文 壇 デ ビ ュ ー を 一 気 に 決 定 づ け た。
</p>

<p>
そ し て こ の 年 の 暮 れ、 漱 石 は 49 歳 で 病 没 す る。<br>
芥 川 は、 漱 石 か ら 「次 の 時 代 の 文 学」 を 託 さ れ た 一 人 と し て、 そ の 後 11 年 間、 数 多 く の 名 作 を 残 す こ と と な る。
</p>

<h3>原 典 ─ 『今 昔 物 語 集』 の 一 話</h3>

<p>
「鼻」 の 原 典 は、 平 安 時 代 末 期 の 説 話 集 『今 昔 物 語 集』 巻 28 第 20 話 「池 尾 禅 珍 内 供 鼻 語」 で あ る。
</p>

<p>
原 典 の あ ら す じ は、 ほ ぼ 芥 川 版 と 同 じ。 だ が、 大 き な 違 い が 一 つ あ る。
</p>

<p>
原 典 は、 単 に 「鼻 が 大 き か っ た 僧 が、 こ う や っ て 鼻 を 短 く し た」 と い う、 <strong>面 白 お か し い 笑 い 話</strong> と し て 書 か れ て い る。<br>
原 典 に は、 「人 の 不 幸 を 喜 ぶ 心 理」 と い う 主 題 は、 ほ と ん ど な い。
</p>

<p>
芥 川 は、 こ の 古 い 笑 い 話 に、 <strong>近 代 的 な 心 理 分 析</strong> を 加 え た。<br>
そ し て 平 安 時 代 の 物 語 を、 大 正 時 代 の 心 理 小 説 と し て 蘇 ら せ た。
</p>

<h3>「人 の 不 幸 を 喜 ぶ 心」 ─ 普 遍 的 な テ ー マ</h3>

<p>
芥 川 が 暴 い た 「人 間 の 暗 い 心」 ─ <strong>人 の 不 幸 か ら の 立 ち 直 り を、 か え っ て 不 快 に 思 う 心 理</strong> ─ は、 残 念 な が ら、 100 年 経 っ た 今 も 私 た ち の 周 り に あ る。
</p>

<ul>
<li>SNS で 誰 か が 成 功 す る と、 揚 げ 足 を 取 る コ メ ン ト が 増 え る</li>
<li>同 級 生 が 急 に 成 績 を 上 げ る と、 嫉 妬 す る</li>
<li>テ レ ビ で 「成 功 者」 を 見 る と、 ど こ か 「気 に 食 わ な い」 と 感 じ る</li>
</ul>

<p>
芥 川 は、 こ う し た 心 理 が <strong>「人 間 の 本 質」</strong> の 一 部 だ と 看 破 し た。<br>
そ し て、 そ の こ と を、 「鼻」 と い う 笑 い 話 を 通 し て、 美 し く、 そ し て 残 酷 に 描 い た の で あ る。
</p>

<h3>芥 川 賞 と 現 代</h3>

<p>
芥 川 は そ の 後、 「羅 生 門」 「藪 の 中」 「地 獄 変」 「ト ロ ッ コ」 「杜 子 春」 「蜘 蛛 の 糸」 な ど、 数 多 く の 名 作 を 残 し た。<br>
だ が 1927 年 7 月、 35 歳 で 服 毒 自 殺。「ぼ ん や り し た 不 安」 と 書 き 残 し て の 自 死 だ っ た。
</p>

<p>
友 人 の 菊 池 寛 (き く ち か ん) は、 芥 川 の 業 績 を 永 遠 に 記 念 す る た め に、 1935 年 「<strong>芥 川 賞</strong>」 を 創 設 し た。<br>
以 来 90 年、 日 本 文 学 界 で 最 も 名 誉 あ る 文 学 賞 と し て、 多 く の 新 進 作 家 を 世 に 送 り 出 し て き た。
</p>

<p>
芥 川 が 23 歳 で 書 い た 「鼻」 ─ そ の 短 い 物 語 の 中 に、 100 年 を 超 え て 響 き 続 け る 「人 間 と は 何 か」 へ の 問 い が あ る。<br>
ぜ ひ、 青 空 文 庫 で 全 文 を 読 み、 そ の 問 い を 自 分 の 心 で 受 け 止 め て み て ほ し い。
</p>
