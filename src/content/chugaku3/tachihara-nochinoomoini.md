<h3>現代語 へ の 案 内</h3>

<p>
詩 は ほ ぼ 現 代 仮 名 遣 い に 近 い 形 で 書 か れ て い る が、 旧 仮 名 (歴 史 的 仮 名 遣 い) の リ ズ ム が 美 し い。<br>
「か へ つ て 行 つ た」「し づ ま り か へ つ た」「う た ひ や ま な い」 ─ こ の 古 風 な 響 き が、 詩 全 体 に 透 明 な 抒 情 を 与 え る。
</p>

<p>
詩 の 大 意:<br>
私 の 夢 は い つ も、 山 の 麓 (ふ も と) の 寂 し い 村 に 帰 っ て 行 っ た。<br>
水 引 草 (赤 い 穂 を 持 つ 秋 の 草) に 風 が 立 ち、 草 ひ ば り が ず っ と 歌 い、 ひ っ そ り と し た 午 後 の 林 道。<br>
青 空 に 陽 が 照 っ て い た。 そ し て 私 は 見 て い た ─ 島 々 が 美 し く 夕 陽 に 輝 く の を、 砂 の 窪 み に 落 ち て い た 果 実 の 種 の あ と を。<br>
そ れ は 遠 い 過 ぎ た 日 の 夢。<br>
そ し て ─ ああ、 い ま も な お、 蒼 白 く 美 し く、 後 (の ち) の 思 い に な っ て 帰 っ て 行 く。<br>
水 引 草 に 風 が 立 ち、 草 ひ ば り が 歌 い や ま な い、 後 の 思 い に な っ て 帰 っ て 行 く。
</p>

<h3>解 説 ─ 立 原 道 造 と い う 詩 人 ・ 建 築 家</h3>

<p>
立 原 道 造 (1914–1939) は、 24 歳 と い う 短 い 生 涯 で 多 く の 名 詩 を 残 し た 詩 人 で あ る。<br>
そ し て 同 時 に、 東 京 帝 国 大 学 工 学 部 建 築 学 科 で 学 ん だ <strong>一 級 建 築 士</strong> で も あ っ た。
</p>

<p>
詩 と 建 築 ─ 一 見 関 係 な い 二 つ の 領 域 だ が、 立 原 に と っ て は 両 者 は 深 く つ な が っ て い た。<br>
詩 の <strong>「構 造」</strong> を 大 切 に し、 ソ ネ ッ ト 14 行 の 厳 密 な 形 式 を 日 本 語 で 試 み 続 け た 立 原 の 姿 勢 は、 建 築 家 と し て の 「形 と 機 能 の 完 璧 な 一 致」 を 求 め る 精 神 と も 通 じ て い る。
</p>

<h3>ソ ネ ッ ト と い う 14 行 詩</h3>

<p>
「ソ ネ ッ ト」 は、 13 世 紀 イ タ リ ア に 生 ま れ た 14 行 詩 の 形 式 で あ る。<br>
ペ ト ラ ル カ・ダ ン テ ら の イ タ リ ア 詩 人、 シ ェ イ ク ス ピ ア・ミ ル ト ン ら の 英 国 詩 人 が、 こ の 形 式 で 名 詩 を 残 し た。
</p>

<p>
立 原 道 造 は、 こ の 西 洋 の 形 式 を、 日 本 語 で 試 み た 最 初 期 の 詩 人 の 一 人 で あ る。<br>
彼 の ソ ネ ッ ト は 一 般 に <strong>「4 行 + 4 行 + 3 行 + 3 行」</strong> の 形 を 取 る。「の ち の お も ひ に」 も こ の 形 式 で あ る。
</p>

<p>
こ の 形 式 の 美 し さ ─ 「展 開 → 展 開 → 反 転 → 結 び」 ─ が、 詩 に <strong>音 楽 的 な 完 璧 性</strong> を 与 え て い る。
</p>

<h3>「夢 が 帰 る」 と い う 不 思 議</h3>

<p>
詩 の 冒 頭 ─ 「夢 は い つ も か へ つ て 行 つ た」 ─ こ の 一 句 が、 詩 全 体 の 主 題 を 決 め る。
</p>

<p>
普 通、 私 た ち は 「夢 を 見 る」 と 言 う。 だ が 立 原 は 「夢 が 帰 る」 と 書 い た。<br>
こ の 表 現 が 示 す の は ─ 「夢」 は 自 分 が 創 り 出 す も の で は な く、 ど こ か 別 の 場 所 (山 麓 の 村) か ら 私 の も と に 何 度 も 「帰 っ て 来 る」 何 か ─ と い う 感 覚 で あ る。
</p>

<p>
こ の 「帰 る」 と い う 主 題 は、 詩 の 最 終 部 で 反 復 さ れ る:<br>
「<strong>の ち の お も ひ に か へ つ て ゆ く</strong>」<br>
「夢」 は、 こ の 詩 を 書 い て い る 「い ま」 に お い て も、 ま だ 私 の 中 に 「後 の 思 い」 と し て 帰 っ て き て い る ─ そ う 詩 人 は 言 う の だ。
</p>

<h3>軽 井 沢 と 立 原</h3>

<p>
詩 の 中 の 「山 の 麓 の さ び し い 村」「林 道」「島 々」 ─ こ れ ら は、 立 原 が 愛 し た <strong>軽 井 沢</strong> (か る い ざ わ) の 風 景 を 思 わ せ る。
</p>

<p>
東 京 帝 大 在 学 中、 立 原 は 何 度 も 軽 井 沢 に 通 い、 そ こ の 自 然 の 中 で 詩 を 紡 い だ。<br>
ま た、 建 築 家 と し て の 立 原 は、 軽 井 沢 で 「<strong>ヒ ヤ シ ン ス ハ ウ ス</strong>」 と い う 小 さ な 別 荘 の 設 計 図 を 描 い た。<br>
し か し、 そ れ が 建 て ら れ る こ と な く 立 原 は 24 歳 で 亡 く な っ た。
</p>

<p>
そ し て 2004 年 ─ 立 原 の 没 後 65 年 ─ 彼 の 設 計 図 を 元 に、 浦 和 (現・さ い た ま 市) の 別 所 沼 公 園 に 「<strong>ヒ ヤ シ ン ス ハ ウ ス</strong>」 が 実 際 に 建 設 さ れ た。<br>
詩 人 立 原 の 「未 完 の 夢」 が、 65 年 後 に 形 と な っ た の で あ る。

<h3>結 核 ─ 早 世 の 詩 人 た ち</h3>

<p>
立 原 道 造 は、 1939 年 (昭 和 14 年) 3 月、 24 歳 と い う 若 さ で、 結 核 性 中 耳 炎 に よ り 亡 く な っ た。
</p>

<p>
20 世 紀 前 半、 結 核 (け っ か く) は 「不 治 の 病」 と 呼 ば れ、 多 く の 詩 人 ・ 作 家 を 早 世 さ せ た。
</p>

<ul>
<li>樋 口 一 葉 (1872–1896) ─ 24 歳</li>
<li>宮 沢 賢 治 (1896–1933) ─ 37 歳</li>
<li>梶 井 基 次 郎 (1901–1932) ─ 31 歳</li>
<li>中 原 中 也 (1907–1937) ─ 30 歳</li>
<li><strong>立 原 道 造 (1914–1939) ─ 24 歳</strong></li>
<li>新 美 南 吉 (1913–1943) ─ 29 歳</li>
</ul>

<p>
近 代 日 本 文 学 史 は、 結 核 に よ る 早 世 と 切 り 離 せ な い。<br>
「短 い 命 が 残 し た 名 作」 ─ そ れ が、 こ の 時 代 の 文 学 の 大 き な 特 徴 で あ る。
</p>

<h3>「の ち の お も ひ に」 を 読 む こ と</h3>

<p>
立 原 道 造 が 24 歳 で 残 し た こ の 詩 を、 86 年 後 の 私 た ち は 今 も 読 む こ と が で き る。
</p>

<p>
「<strong>夢 は い つ も か へ つ て 行 つ た</strong>」 ─ こ の 一 句 が 持 つ 透 明 な 美 し さ は、 短 い 命 か ら 生 ま れ た か ら こ そ、 こ ん な に も 切 な く 響 く の か も し れ な い。
</p>

<p>
詩 を 読 み 終 え た 後、 君 の 心 に 何 か が 「帰 っ て 来 て い る」 の を 感 じ た ら ─ そ れ こ そ が、 立 原 道 造 が 君 に 送 っ た 「<strong>の ち の お も ひ</strong>」 で あ る。
</p>
