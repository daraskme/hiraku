<h3>現 代 語 へ の 案 内</h3>

<p>
詩 は ほ ぼ 古 い 仮 名 遣 い (歴 史 的 仮 名 遣 い) で 書 か れ て い る。<br>
「見 き」(見 た)、「さ び し か り け り」(さ び し か っ た)、「け ぶ り 立 つ 見 つ」(煙 が 立 つ の を 見 た) ─ こ う し た 古 風 な 表 現 が、 静 か な 林 の 雰 囲 気 を 醸 す。
</p>

<h3>解 説 ─ 「し み じ み と 見 る」</h3>

<p>
「落 葉 松」 全 8 連 を 通 し て、 詩 人 は 「ひ た す ら 落 葉 松 を 見 て、 落 葉 松 と と も に 歩 く」 と い う 経 験 を 描 く。
</p>

<p>
詩 に は、 派 手 な 出 来 事 は 何 も 起 こ ら な い:
</p>
<ul>
<li>第 1 連 ─ 林 を 過 ぎ、 落 葉 松 を 見 る</li>
<li>第 2 連 ─ 出 で、 入 り、 ま た 細 い 道 が 続 く</li>
<li>第 3 連 ─ 林 の 奥 に も 道 が あ る</li>
<li>第 4 連 ─ 自 分 だ け で な く、 他 の 人 も 通 る 細 い 道</li>
<li>第 5 連 ─ 落 葉 松 と さ さ や く</li>
<li>第 6 連 ─ 浅 間 山 の 噴 煙 を 見 上 げ る</li>
<li>第 7 連 ─ 雨 の 中、 か ん こ 鳥 が 鳴 く</li>
<li>第 8 連 ─ 世 の 中 と 自 然 へ の 静 か な 詠 嘆</li>
</ul>

<p>
こ の 「<strong>何 も 起 こ ら な い こ と</strong>」 こ そ が、 こ の 詩 の 本 質 で あ る。<br>
落 葉 松 林 と 詩 人 の 心 が、 静 か に 共 鳴 し、 そ こ に <strong>「し み じ み と」</strong> と い う 感 情 が 生 ま れ る ─ そ れ を 描 い た 詩 で あ る。
</p>

<h3>リ フ レ イ ン の 美</h3>

<p>
詩 の 各 連 で、 「<strong>か ら ま つ</strong>」 と い う 言 葉 が、 ほ ぼ 毎 行 に 出 て く る。
</p>

<p>
普 通、 詩 で は 同 じ 言 葉 を 繰 り 返 し 使 う こ と は 避 け る べ き と さ れ る。<br>
だ が 白 秋 は、 あ え て こ の リ フ レ イ ン を 大 胆 に 使 っ た。
</p>

<p>
そ し て、 そ の 効 果 は 絶 大 で あ る。<br>
「か ら ま つ」 と い う 言 葉 が 繰 り 返 さ れ る た び に、 読 者 の 心 に 落 葉 松 の イ メ ー ジ が 重 な っ て い く。<br>
や が て、 詩 全 体 が <strong>一 つ の 落 葉 松 林</strong> と な っ て、 読 者 を そ の 中 へ と 招 き 入 れ る。
</p>

<h3>軽 井 沢 と 浅 間 山</h3>

<p>
詩 の 第 6 連 に 「<strong>浅 間 嶺 に け ぶ り 立 つ 見 つ</strong>」 と あ る。<br>
浅 間 山 (あ さ ま や ま) は、 群 馬・長 野 県 境 の 活 火 山。 軽 井 沢 か ら 見 上 げ る と、 と き ど き 噴 煙 が 立 ち 上 る の が 見 え る。
</p>

<p>
こ の 描 写 か ら、 詩 の 舞 台 は <strong>信 州・軽 井 沢</strong> の 落 葉 松 林 で あ る こ と が わ か る。
</p>

<p>
軽 井 沢 は、 明 治 以 来、 多 く の 文 学 者 の 創 作 の 場 と な っ た:
</p>
<ul>
<li>北 原 白 秋 ─ 「落 葉 松」 (1921)</li>
<li>室 生 犀 星 ─ 軽 井 沢 を 第 二 の 故 郷 と し て 多 く の 作 品</li>
<li>堀 辰 雄 ─ 「風 立 ち ぬ」 (1936–1938)</li>
<li>立 原 道 造 ─ ソ ネ ッ ト 「の ち の お も ひ に」 な ど (本 サ イ ト chugaku3)</li>
<li>川 端 康 成 ─ 「雪 国」 構 想 ・ 別 荘 滞 在</li>
</ul>

<p>
軽 井 沢 の 高 原 の 落 葉 松 林 は、 100 年 以 上 に わ た り、 日 本 文 学 の 一 つ の 「聖 地」 と な っ て き た。
</p>

<h3>「世 の 中 よ、 あ は れ な り け り」</h3>

<p>
詩 の 最 後 ─ 第 8 連 ─ で、 詩 人 は 突 然 「世 の 中」 へ の 思 い を 口 に す る。
</p>

<blockquote>
世 の 中 よ、 あ は れ な り け り。<br>
常 な け ど う れ し か り け り。<br>
山 川 に 山 川 の 音、<br>
か ら ま つ に か ら ま つ の 風。
</blockquote>

<p>
「<strong>世 の 中 は あ わ れ で あ る。 だ が 永 遠 で は な い (= 移 ろ う) こ と も 嬉 し い こ と だ。 山 川 に は 山 川 の 音 が あ り、 落 葉 松 に は 落 葉 松 の 風 が あ る</strong>」
</p>

<p>
こ こ で 詩 は、 個 別 の 落 葉 松 林 か ら、 <strong>万 物 の 「あ る が ま ま」 の 美 し さ</strong> へ と 視 野 を 広 げ る。
</p>

<p>
落 葉 松 に 吹 く 風 は、 落 葉 松 だ け の 風 で あ る。<br>
山 川 に 響 く 音 は、 山 川 だ け の 音 で あ る。<br>
そ し て、 そ れ ら す べ て が「あ は れ (し み じ み と し た 美 し さ)」 を 持 つ。
</p>

<p>
こ れ こ そ が、 北 原 白 秋 が 「落 葉 松」 を 通 し て 私 た ち に 伝 え た い、 自 然 へ の 眼 差 し で あ る。
</p>

<h3>声 に 出 し て 読 ん で み よ う</h3>

<p>
「落 葉 松」 は、 黙 読 で は な く、 ぜ ひ <strong>声 に 出 し て</strong> 読 ん で ほ し い。
</p>

<p>
「か ら ま つ の 林 を 過 ぎ て、 か ら ま つ を し み じ み と 見 き」 ─ こ の 一 行 を 声 に 出 し た と き、 君 は 信 州 の 高 原 の、 静 か な 落 葉 松 林 の 中 に 立 っ て い る。<br>
百 年 前 の 白 秋 と、 同 じ 風 景 を、 同 じ 心 で、 「し み じ み と」 見 て い る の で あ る。
</p>
