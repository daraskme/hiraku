<h3>1905 年、 ス イ ス・ベ ル ン の 特 許 局</h3>

<p>
1905 年、 ス イ ス・ベ ル ン の 特 許 局 に、 一 人 の 26 歳 の 三 級 技 官 が い た。<br>
名 前 は <strong>ア ル ベ ル ト・ア イ ン シ ュ タ イ ン</strong> (Albert Einstein, 1879–1955)。
</p>

<p>
彼 は ド イ ツ 生 ま れ で、 ス イ ス 工 科 大 学 (チ ュ ー リ ヒ) を 卒 業 し た が、 大 学 で の ポ ス ト を 得 ら れ ず、 仕 方 な く 特 許 局 で 働 い て い た。<br>
仕 事 は 退 屈 だ っ た が、 そ の 分、 物 理 学 の 思 索 に 時 間 を 使 え た。
</p>

<p>
こ の 退 屈 な 役 所 仕 事 を し な が ら、 26 歳 の ア イ ン シ ュ タ イ ン は ─ <strong>1905 年 の 一 年 間 に 4 つ の 革 命 的 論 文 を 発 表 し た</strong>。
</p>

<h3>「奇 跡 の 年」 1905</h3>

<p>
1905 年 ─ 物 理 学 史 で「<strong>奇 跡 の 年</strong>」(Annus Mirabilis) と 呼 ば れ る 年。
</p>

<p>
こ の 一 年 で、 ア イ ン シ ュ タ イ ン は 以 下 を 発 表:
</p>

<ol>
<li><strong>3 月: 光 量 子 仮 説</strong> ─ 光 が 「粒 子 (光 子)」 で あ る こ と。 量 子 力 学 の 出 発 点。 後 に ノ ー ベ ル 賞 (1921) の 対 象。</li>
<li><strong>5 月: ブ ラ ウ ン 運 動</strong> ─ 水 中 の 微 粒 子 の 不 規 則 運 動 が、 水 分 子 の 衝 突 で 起 こ る こ と を 数 学 的 に 説 明。 原 子 ・ 分 子 の 実 在 を 実 証。</li>
<li><strong>6 月: 特 殊 相 対 性 理 論</strong> ─ 時 間 と 空 間 の 概 念 を 革 命。 ニ ュ ー ト ン 力 学 の 限 界 を 突 破。</li>
<li><strong>9 月: E = mc²</strong> ─ 質 量 と エ ネ ル ギ ー が 等 価 で あ る こ と。 後 の 原 子 物 理 学 の 基 礎。</li>
</ol>

<p>
た っ た 一 年 で、 26 歳 の 一 人 の 特 許 局 員 が、 20 世 紀 物 理 学 の 4 つ の 柱 を 構 築 し た。<br>
1666 年 の 22 歳 の ニ ュ ー ト ン の「奇 跡 の 年」 と 並 ぶ、 物 理 学 史 の ハ イ ラ イ ト で あ る。
</p>

<h3>特 殊 相 対 性 理 論 ─ 「光 速 は 絶 対」</h3>

<p>
ア イ ン シ ュ タ イ ン の 出 発 点 は、 こ の 一 つ の 大 胆 な 仮 説:
</p>

<blockquote>
「<strong>光 の 速 度 は、 観 測 者 が 動 い て い て も 静 止 し て い て も、 必 ず 同 じ (c ≒ 30 万 km/s)</strong>」
</blockquote>

<p>
こ れ は、 ニ ュ ー ト ン 力 学 の「速 度 の 加 算 法 則」 と 真 っ 向 か ら 対 立 す る。
</p>

<p>
普 通 の 感 覚 で は ─ 時 速 100 km の 車 か ら 時 速 50 km の ボ ー ル を 投 げ た ら、 ボ ー ル の 速 度 は 150 km/h。<br>
だ が 光 で は、 光 速 で 動 く 物 体 か ら 光 を 発 し て も、 光 の 速 度 は <strong>同 じ c = 30 万 km/s</strong>。
</p>

<p>
こ の 不 思 議 な 事 実 は、 1887 年 マ イ ケ ル ソ ン・モ ー リ ー の 実 験 で 確 認 さ れ て い た。<br>
だ が「な ぜ そ う な る か」 を 説 明 で き る 人 は い な か っ た。
</p>

<p>
ア イ ン シ ュ タ イ ン は、 こ の 事 実 を <strong>「光 速 不 変 」 と い う 公 理 と し て 受 け 入 れ る</strong> こ と か ら 始 め た。<br>
そ し て、 そ こ か ら 論 理 的 に 何 が 導 か れ る か を 厳 密 に 計 算 し た。
</p>

<h3>驚 く べ き 結 論</h3>

<p>
ア イ ン シ ュ タ イ ン の 計 算 か ら、 次 の 結 論 が 導 か れ た:
</p>

<ol>
<li><strong>時 間 は 相 対 的</strong> ─ 動 い て い る 観 測 者 の 時 計 は、 静 止 し て い る 観 測 者 か ら 見 て 遅 く 進 む</li>
<li><strong>空 間 は 相 対 的</strong> ─ 動 い て い る 物 体 は、 静 止 し て い る 観 測 者 か ら 見 て 縮 ん で 見 え る</li>
<li><strong>同 時 性 は 相 対 的</strong> ─「同 時 に 起 き た」 と い う 判 断 が、 観 測 者 に よ っ て 違 う</li>
<li><strong>質 量 と エ ネ ル ギ ー は 等 価</strong> ─ E = mc²</li>
</ol>

<p>
こ れ ら の 結 論 は、 ニ ュ ー ト ン の「絶 対 時 間 ・ 絶 対 空 間」 を 完 全 に 覆 す。<br>
だ が、 数 学 的 に は 厳 密 で、 後 に 実 験 で 全 て 確 認 さ れ た。
</p>

<h3>E = mc² の 衝 撃</h3>

<p>
最 も 有 名 な 公 式 ─ <strong>E = mc²</strong> ─ は、 質 量 と エ ネ ル ギ ー が 等 価 で あ る こ と を 示 す。
</p>

<p>
例 え ば、 1 g の 物 質 が 完 全 に エ ネ ル ギ ー に 変 換 さ れ た ら ─
</p>

<blockquote>
E = m × c² = 0.001 kg × (3 × 10⁸ m/s)² = 9 × 10¹³ J
</blockquote>

<p>
こ れ は、 約 <strong>2 万 1500 ト ン</strong> の TNT 火 薬 の 爆 発 と 同 じ エ ネ ル ギ ー。<br>
わ ず か 1 g か ら、 巨 大 な 都 市 を 一 つ 吹 き 飛 ば せ る エ ネ ル ギ ー が 出 る ─ こ れ が、 後 に 原 子 爆 弾 (1945)、 原 子 力 発 電 の 物 理 的 基 礎 と な っ た。
</p>

<h3>一 般 相 対 性 理 論 (1915) ─ 重 力 = 時 空 の 曲 が り</h3>

<p>
1915 年、 ア イ ン シ ュ タ イ ン は 36 歳 で <strong>一 般 相 対 性 理 論</strong> を 完 成 さ せ た。
</p>

<p>
こ こ で 彼 は、 重 力 を 全 く 新 し い 形 で 解 釈 し た:
</p>

<blockquote>
「<strong>重 力 と は、 質 量 が 周 り の 時 空 を 曲 げ る こ と で 生 ま れ る</strong>」
</blockquote>

<p>
具 体 的 に は:
</p>

<ul>
<li>太 陽 の よ う な 大 き な 質 量 は、 周 り の 「時 空」 を 大 き く 曲 げ る</li>
<li>地 球 は、 そ の 曲 が り に 沿 っ て「真 っ 直 ぐ」 動 く ─ そ れ が 我 々 か ら 見 れ ば 太 陽 の 周 り の 公 転 に 見 え る</li>
<li>ニ ュ ー ト ン の 「万 有 引 力」 は、 こ の「時 空 の 曲 が り」 の 近 似 だ っ た</li>
</ul>

<p>
こ の 理 論 は、 1919 年 の 日 食 観 測 で 実 証 さ れ た。<br>
皆 既 日 食 の 際、 太 陽 の 近 く を 通 る 星 の 光 が、 ア イ ン シ ュ タ イ ン が 予 言 し た 通 り「太 陽 の 重 力 で 曲 げ ら れ る」 こ と が 確 認 さ れ た。
</p>

<p>
一 夜 に し て、 ア イ ン シ ュ タ イ ン は 世 界 的 名 声 を 得 た。
</p>

<h3>君 の 周 り の 相 対 性 理 論</h3>

<p>
相 対 性 理 論 は、 一 見 「日 常 に 関 係 な い」 と 思 う か も し れ な い。<br>
だ が、 実 は 君 の 生 活 で も 使 わ れ て い る。
</p>

<h4>GPS</h4>

<p>
ス マ ホ の GPS は、 上 空 約 2 万 km の 衛 星 か ら の 信 号 で 位 置 を 計 算 す る。<br>
衛 星 上 で は ─
</p>
<ul>
<li>特 殊 相 対 性 理 論 ─ 衛 星 が 高 速 で 動 い て い る た め、 時 計 が 遅 く な る (1 日 あ た り 7 マ イ ク ロ 秒)</li>
<li>一 般 相 対 性 理 論 ─ 衛 星 は 地 球 の 重 力 が 弱 い 高 高 度 に い る た め、 時 計 が 速 く 進 む (1 日 あ た り 45 マ イ ク ロ 秒)</li>
</ul>

<p>
合 わ せ て 1 日 あ た り 38 マ イ ク ロ 秒 の 時 計 の ず れ。 こ れ を 補 正 し な い と、 GPS は 1 日 で 10 km 以 上 ず れ て し ま う。<br>
君 の ス マ ホ の 地 図 ア プ リ が 正 確 に 動 く の は、 ア イ ン シ ュ タ イ ン の 計 算 の お か げ で あ る。
</p>

<h4>原 子 力</h4>

<p>
E = mc² の 公 式 は、 原 子 力 発 電・原 爆 の 基 礎。 100 年 前 の 公 式 が、 今 の 電 力 供 給 や 国 際 政 治 を 動 か し て い る。
</p>
