<h3>「物 理 学 の 父」 ガ リ レ オ</h3>

<p>
ガ リ レ オ・ガ リ レ イ (1564–1642) は、 「<strong>近 代 物 理 学 の 父</strong>」 と 呼 ば れ る。
</p>

<p>
彼 の 仕 事 が、 中 世 ま で 続 い た「ア リ ス ト テ レ ス 的 自 然 観」 を 打 ち 砕 き、 ニ ュ ー ト ン・ア イ ン シ ュ タ イ ン に つ な が る「近 代 物 理 学」 の 道 を 開 い た。
</p>

<h3>ピ サ の 斜 塔 ─ 落 下 実 験 の 伝 説</h3>

<p>
1589 年 頃 (25 歳)、 ガ リ レ オ は ピ サ 大 学 の 数 学 教 授 だ っ た。
</p>

<p>
伝 説 に よ れ ば ─ 彼 は <strong>ピ サ の 斜 塔</strong> の 上 か ら、 大 小 2 つ の 鉛 玉 を 同 時 に 落 と し た。<br>
古 代 ギ リ シ ア の ア リ ス ト テ レ ス (BC 384–322) は「重 い 物 ほ ど 速 く 落 ち る」 と 説 い た。 だ が ガ リ レ オ は ─ 同 時 に 着 地 す る こ と を 示 し た。
</p>

<p>
こ の「ピ サ の 斜 塔 の 実 験」 が 史 実 か は 不 明 (ガ リ レ オ 自 身 の 著 作 に 記 述 な し)。<br>
だ が、 「<strong>ア リ ス ト テ レ ス の 古 い 物 理 学 を 実 験 で 否 定 し た</strong>」 と い う ガ リ レ オ の 仕 事 の 象 徴 と し て、 今 も 語 ら れ て い る。
</p>

<h3>落 下 の 法 則 ─ 数 学 と 実 験</h3>

<p>
ガ リ レ オ は、 落 下 の 詳 細 な 実 験 を 繰 り 返 し、 こ う 結 論 し た:
</p>

<ol>
<li><strong>真 空 中 で は、 質 量 に か か わ ら ず、 す べ て の 物 体 は 同 じ 加 速 度 で 落 下 す る</strong> (重 力 加 速 度 g ≒ 9.8 m/s²)</li>
<li><strong>落 下 距 離 は 時 間 の 二 乗 に 比 例 す る</strong> (d = ½ g t²)</li>
<li><strong>初 速 度 が あ れ ば、 そ れ に「等 速 直 線 運 動」 と「加 速 度 運 動」 が 重 ね 合 わ さ る</strong> (放 物 運 動)</li>
</ol>

<p>
こ の「<strong>運 動 を 数 式 で 表 現 す る</strong>」 と い う 発 想 は、 当 時 と し て は 革 命 的 だ っ た。<br>
ガ リ レ オ は こ う 書 い た: 「<strong>自 然 の 書 物 は 数 学 で 書 か れ て い る</strong>」(Il libro della natura è scritto in lingua matematica)。
</p>

<h3>望 遠 鏡 と『星 界 の 報 告』(1610)</h3>

<p>
1609 年、 ガ リ レ オ は オ ラ ン ダ で 発 明 さ れ た ば か り の 望 遠 鏡 の 噂 を 聞 い た。<br>
彼 は す ぐ に 自 ら 望 遠 鏡 を 改 良 し (倍 率 約 30 倍)、 史 上 初 め て 天 体 を 望 遠 鏡 で 系 統 的 に 観 察 し た。
</p>

<p>
1610 年『<strong>星 界 の 報 告</strong>』 で 発 表 さ れ た 観 測 結 果:
</p>

<ul>
<li><strong>月 の 山 と クレ ー タ ー</strong> ─ 月 は 「完 全 な 球 」 で は な く、 地 球 と 同 じ よ う に 地 形 が あ る</li>
<li><strong>木 星 の 衛 星 4 個</strong> (ガ リ レ オ 衛 星 ─ イ オ、 エ ウ ロ パ、 ガ ニ メ デ、 カ リ ス ト) ─ 「全 て の 天 体 が 地 球 を 中 心 に 回 っ て い る」 と い う 天 動 説 を 直 接 否 定</li>
<li><strong>金 星 の 満 ち 欠 け</strong> ─ 金 星 が 太 陽 の 周 り を 回 っ て い る こ と を 示 す</li>
<li><strong>太 陽 黒 点</strong> ─ 太 陽 も「完 全 」 で は な く、 シ ミ が あ る</li>
<li><strong>銀 河</strong> ─ 無 数 の 星 の 集 ま り だ と わ か る</li>
</ul>

<p>
こ れ ら の 観 測 が、 コ ペ ル ニ ク ス の <strong>地 動 説</strong> (1543) を 強 力 に 支 持 し た。
</p>

<h3>1633 年 異 端 審 問</h3>

<p>
1633 年 6 月 22 日、 ロ ー マ。<br>
69 歳 の ガ リ レ オ は、 カ ト リ ッ ク 教 会 の <strong>異 端 審 問</strong> で 有 罪 判 決 を 受 け た。
</p>

<p>
罪 状: 「地 動 説 を 唱 え、 聖 書 に 反 し た」。<br>
判 決: 「地 動 説 を 撤 回 す る こ と」、 著 作『天 文 対 話』 (1632) を 発 禁、 終 身 軟 禁。
</p>

<p>
ガ リ レ オ は、 跪 い て 撤 回 文 書 を 読 み 上 げ た。<br>
そ し て 立 ち 上 が る 時 ─ 伝 説 に よ れ ば ─ こ う 小 さ く つ ぶ や い た:
</p>

<blockquote>
「<strong>そ れ で も 地 球 は 動 い て い る</strong>」 (E pur si muove)
</blockquote>

<p>
こ の つ ぶ や き が 実 際 に あ っ た か は 不 明 (1761 年 の 書 物 に 初 め て 記 述)。<br>
だ が、 真 偽 を 超 え て、 「<strong>権 力 に 屈 し て も、 真 理 そ の も の は 変 わ ら な い</strong>」 と い う 科 学 の 精 神 の 象 徴 と し て 語 り 継 が れ て い る。
</p>

<h3>軟 禁 中 の 最 大 の 著 作『新 科 学 対 話』(1638)</h3>

<p>
1633 年 か ら 1642 年 没 ま で、 ガ リ レ オ は ピ サ 近 郊 の 自 宅 に 終 身 軟 禁 さ れ た。<br>
書 物 の 出 版 も 禁 じ ら れ た。
</p>

<p>
だ が、 こ の 軟 禁 中 に、 彼 は 最 大 の 物 理 学 著 作 『<strong>新 科 学 対 話</strong>』 (Discorsi e dimostrazioni matematiche intorno a due nuove scienze) を 完 成 さ せ た。
</p>

<p>
原 稿 は 密 か に オ ラ ン ダ に 送 ら れ、 1638 年 に ラ イ デ ン で 出 版 さ れ た。<br>
こ の 著 作 で、 落 下 の 法 則、 振 り 子 の 法 則、 材 料 力 学 の 基 礎 な ど が 体 系 的 に 説 明 さ れ た。<br>
ニ ュ ー ト ン 力 学 (1687) の 直 接 の 前 提 と な っ た 著 作 で あ る。
</p>

<h3>ガ リ レ オ → ニ ュ ー ト ン</h3>

<p>
1642 年 1 月 8 日、 ガ リ レ オ は 78 歳 で 没 し た。
</p>

<p>
そ の 同 じ 年 ─ 1642 年 ク リ ス マ ス の 日 ─ イ ギ リ ス の ウ ル ス ソ ー プ で、 <strong>ア イ ザ ッ ク ・ ニ ュ ー ト ン</strong> (本 サ イ ト butsuri1/newton-banyuinryoku) が 生 ま れ た。
</p>

<p>
こ の「ガ リ レ オ 没 → ニ ュ ー ト ン 生」 の 一 年 が、 物 理 学 史 上 で 最 も 美 し い「バ ト ン タ ッ チ」 と さ れ る。<br>
ガ リ レ オ が 残 し た「実 験 と 数 式 で 自 然 を 記 述 す る」 方 法 が、 ニ ュ ー ト ン に よ っ て 完 成 形 と な っ た。
</p>

<h3>1992 年 ─ 教 会 の 公 式 謝 罪</h3>

<p>
1633 年 の 異 端 審 問 か ら <strong>359 年</strong> 後 の 1992 年 10 月 31 日 ─<br>
教 皇 ヨ ハ ネ・パ ウ ロ 二 世 が、 ロ ー マ 教 皇 庁 科 学 ア カ デ ミ ー で 公 式 表 明 し た:
</p>

<blockquote>
「ガ リ レ オ ・ ガ リ レ イ に 対 す る 1633 年 の 裁 判 は、 誤 り だ っ た。 教 会 は、 ガ リ レ オ の 名 誉 を 回 復 す る」
</blockquote>

<p>
359 年。 ─ ガ リ レ オ が 跪 い て 地 動 説 を 撤 回 し た 日 か ら、 教 会 が「誤 り だ っ た」 と 認 め る ま で の 時 間。
</p>

<p>
こ の 359 年 が、 「<strong>真 理 と 権 力 の 関 係</strong>」 を 物 語 っ て い る。
</p>

<h3>「自 然 は 数 学 で 書 か れ て い る」</h3>

<p>
ガ リ レ オ が 残 し た 最 大 の 言 葉 を、 改 め て 引 用 す る:
</p>

<blockquote>
「<strong>哲 学 は、 我 々 の 目 の 前 に 開 か れ た 偉 大 な 書 物 (= 宇 宙) の 中 に 書 か れ て い る。 そ の 書 物 は、 数 学 の 言 葉 で 書 か れ て い る</strong>」<br>
─『Il Saggiatore (試 金 石)』 1623
</blockquote>

<p>
こ れ が、 ニ ュ ー ト ン・ア イ ン シ ュ タ イ ン と 続 く 近 代 物 理 学 の 出 発 点 で あ る。<br>
中 学・高 校 で 数 学 と 物 理 を 学 ぶ こ と は、 ガ リ レ オ が 開 い た「自 然 を 数 学 で 読 む」 旅 へ の 第 一 歩 で あ る。
</p>

<p>
ガ リ レ オ・ガ リ レ イ。 460 年 前 の イ タ リ ア に 生 ま れ、 真 理 を 守 ろ う と し て 苦 し み、 そ れ で も 自 然 の 数 学 的 美 し さ を 私 た ち に 教 え て く れ た 巨 人 で あ る。
</p>
