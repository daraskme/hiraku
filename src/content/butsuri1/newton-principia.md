<h3>1665 年 ・ 23 歳 ─ ペ ス ト 流 行 で 田 舎 に 戻 っ た 学 生</h3>

<p>
1665 年、 ロ ン ド ン で ペ ス ト が 大 流 行。 ケ ン ブ リ ッ ジ 大 学 ト リ ニ テ ィ・カ レ ッ ジ も 閉 鎖 さ れ、
23 歳 の 学 生 ア イ ザ ッ ク・ニ ュ ー ト ン は 故 郷 リ ン カ ン シ ャ ー 州 ウ ル ス ソ ー プ 村 に 戻 っ た。
</p>

<p>
こ の 約 2 年 間 (1665–1667、「奇 跡 の 年」 と 呼 ば れ る) に、 ニ ュ ー ト ン は 微 分 積 分・光 の 分 解・
運 動 法 則・万 有 引 力 の 構 想 を 全 て 思 い 付 い た。 一 人 の 若 者 が、 こ ん な 短 期 間 で こ れ ほ ど の
発 見 を し た こ と は、 科 学 史 上 例 が な い。
</p>

<h3>林 檎 の 逸 話 ─ 史 実 だ っ た</h3>

<p>
ウ ル ス ソ ー プ 村 の 庭 で、 ニ ュ ー ト ン は 林 檎 が 木 か ら 落 ち る の を 見 て、 こ う 思 い 付 い た と 言 う ─
「林 檎 を 地 面 に 引 き 寄 せ る 力 と、 月 を 地 球 の 周 り に 引 き 留 め る 力 は、 同 じ 力 な の で は な い か?」
</p>

<p>
こ の 逸 話 は、 後 に ニ ュ ー ト ン 本 人 が 知 人 (ウ ィ リ ア ム・ス タ ッ ク レ イ ら) に 語 っ た 話 で、
史 実 と し て 認 め ら れ て い る。 ニ ュ ー ト ン が 座 っ て い た と さ れ る 林 檎 の 木 (品 種 ─ Flower of Kent)
は 現 在 も 故 郷 に 残 り、 そ の 子 孫 が 世 界 中 の 大 学・研 究 所 に 接 ぎ 木 さ れ て い る (東 大・京 大 等 に も)。
</p>

<h3>20 年 後 ─ ハ レ ー の 説 得 で 出 版 を 決 意</h3>

<p>
ニ ュ ー ト ン は 慎 重 な 性 格 で、 発 見 を す ぐ 出 版 し な か っ た。 1684 年、 友 人 の 天 文 学 者
エ ド モ ン ド・ハ レ ー (後 の ハ レ ー 彗 星 で 有 名) が ケ ン ブ リ ッ ジ を 訪 れ、 こ う 尋 ね た ─
「楕 円 軌 道 を 説 明 す る 力 の 法 則 を ご 存 知 で す か?」
</p>

<p>
ニ ュ ー ト ン は 即 答 し た ─ 「ケ プ ラ ー の 第 三 法 則 か ら、 距 離 の 二 乗 に 反 比 例 す る 引 力 で 説 明 で き る」。
ハ レ ー は 驚 き、 出 版 を 強 く 説 得。 出 版 費 用 も ハ レ ー が 個 人 で 負 担 し、 3 年 後 の 1687 年、
『プ リ ン キ ピ ア』 が 世 に 出 た。
</p>

<h3>運 動 第 一 法 則 ─ 慣 性 の 法 則</h3>

<p>
『プ リ ン キ ピ ア』 で ニ ュ ー ト ン は、 ま ず 三 つ の「運 動 の 法 則」 を 提 示 す る。
</p>

<p><strong>第 一 法 則 (慣 性 の 法 則)</strong> ─ 「外 力 が 働 か な け れ ば、 物 体 は 静 止 し て い る か、
等 速 直 線 運 動 を 続 け る」。
</p>

<p>
ガ リ レ オ の 慣 性 概 念 (本 サ イ ト butsuri1/galileo-rakkataikenshi) を 厳 密 に 定 式 化 し た も の。
バ ス が 急 ブ レ ー キ で 君 が 前 に 倒 れ そ う に な る の は、 君 の 体 が「慣 性」 で 動 き 続 け よ う と す る 為。
</p>

<h3>運 動 第 二 法 則 ─ F = ma</h3>

<p>
<strong>第 二 法 則</strong> ─ 「力 = 質 量 × 加 速 度」 (F = ma)。 物 体 に 加 わ る 力 が、 加 速 度 を 生 む。
</p>

<p>
こ の 一 つ の 式 が、 古 典 力 学 の 中 心 で あ る。 軽 い 物 は 簡 単 に 加 速 で き、 重 い 物 は 大 き な 力 が
必 要。 君 の 自 転 車 を こ ぎ 出 す の が 静 止 ─ 加 速 中 に 一 番 力 が 要 る の は こ の 為。
</p>

<h3>運 動 第 三 法 則 ─ 作 用・反 作 用</h3>

<p>
<strong>第 三 法 則</strong> ─ 「作 用 と 反 作 用 は 大 き さ が 等 し く、 逆 向 き」。
A が B を 押 せ ば、 B も A を 同 じ 強 さ で 押 し 返 す。
</p>

<p>
ロ ケ ッ ト ─ 燃 料 ガ ス を 後 ろ に 噴 出 す る (作 用) → ロ ケ ッ ト が 前 へ 進 む (反 作 用)。
銃 の 反 動 も 同 じ 原 理。 君 が 床 を 押 す と 床 が 君 を 押 し 返 す か ら、 立 っ て い ら れ る。
</p>

<h3>万 有 引 力 の 法 則 ─ 全 て の 物 質 が 引 き 合 う</h3>

<p>
さ ら に、 ニ ュ ー ト ン は 重 力 を 一 般 化 し た ─ 「全 て の 物 質 は、 質 量 の 積 に 比 例 し、
距 離 の 2 乗 に 反 比 例 す る 力 で 引 き 合 う」。
</p>

<p style="text-align: center; font-size: 1.1em;">
<strong>F = G m₁ m₂ / r²</strong>
</p>

<p>
ここで F = 引力, G = 万有引力定数 (約 6.674 × 10⁻¹¹), m₁ m₂ = 二 物 体 の 質 量, r = 距 離。
</p>

<p>
こ の 式 で、 林 檎 (m₁ ≈ 0.1 kg, r ≈ 6378 km = 地 球 半 径) と 地 球 (m₂ ≈ 6 × 10²⁴ kg) の 引 力
を 計 算 す る と、 約 1 N (ニ ュ ー ト ン) ─ こ れ が 林 檎 の「重 さ」 で あ る。
</p>

<p>
同 じ 式 で、 月 と 地 球 の 引 力 を 計 算 す る と、 月 が 楕 円 軌 道 で 地 球 を 周 る 周 期 が 約 27.3 日 に
な る ─ 実 際 の 観 測 値 と 一 致 し た。 ニ ュ ー ト ン の 理 論 が 正 し い こ と が 証 明 さ れ た 瞬 間。
</p>

<h3>太 陽 系・潮 汐・彗 星 ─ 全 て が 説 明 さ れ た</h3>

<p>
ニ ュ ー ト ン の 法 則 で、 こ れ ま で 謎 だ っ た 多 く の 現 象 が 説 明 さ れ た ─
</p>

<ul>
  <li>ケ プ ラ ー の 惑 星 三 法 則 (楕 円 軌 道、 面 積 速 度、 周 期 と 軌 道 長 半 径)</li>
  <li>月 の 引 力 が 起 こ す 潮 の 満 ち 引 き (潮 汐)</li>
  <li>ハ レ ー 彗 星 の 76 年 周 期 (ハ レ ー が ニ ュ ー ト ン 力 学 で 計 算)</li>
</ul>

<p>
天 と 地 が 同 じ 法 則 で 動 い て い る ─ こ れ が ニ ュ ー ト ン 革 命 の 中 心 思 想 で あ る。
</p>

<h3>結 び ─ 240 年 後 の ア イ ン シ ュ タ イ ン ま で 続 い た 理 論</h3>

<p>
ニ ュ ー ト ン の 古 典 力 学 は、 1915 年 ま で の 240 年 間、 物 理 学 の 全 て の 基 礎 だ っ た。 ア イ ン シ ュ タ イ ン の
一 般 相 対 性 理 論 (1915) に よ り、 ニ ュ ー ト ン 力 学 は「強 い 重 力 や 高 速 で 不 正 確 に な る」 こ と が
わ か り、 修 正 が 必 要 と 判 明 し た。
</p>

<p>
し か し、 君 の 身 の 回 り の 物 理 (車 ・ 自 転 車 ・ ボ ー ル ・ 飛 行 機) は 全 て、 ニ ュ ー ト ン 力 学 で
完 璧 に 計 算 で き る。 高 校 物 理 で 学 ぶ「力 学」 は、 こ の 340 年 前 の 理 論 そ の も の で あ る。
</p>
