## 土 星 型 原 子 模 型 ─ 日 本 が 世 界 に 提 案 し た 原 子 の 姿

### 1. 1903 年、 東 京 数 学 物 理 学 会 で の 発 表

明 治 36 年 (1903 年) 12 月、 東 京 数 学 物 理 学 会 の 例 会 で、 38 歳 の 長 岡 半 太 郎 が 一 つ の 理 論 を 発 表 し た。

「原 子 は ど の よ う な 構 造 を 持 つ か」 と い う、 当 時 物 理 学 の 最 大 の 謎 の 一 つ に 対 す る、 日 本 か ら の 答 え で あ っ た。

長 岡 の 提 案 は こ う で あ っ た。

「原 子 の 中 心 に は、 正 電 荷 を 持 っ た 大 き な 粒 が あ る。 そ し て、 そ の 周 り を、 負 電 荷 を 持 っ た 多 数 の 電 子 が、 土 星 の 環 の よ う に 同 心 円 状 に 回 っ て い る。」

こ れ を「土 星 型 原 子 模 型 (Saturnian Model)」 と 呼 ぶ。

翌 1904 年、 長 岡 は こ の 理 論 を 英 文 で 改 め て『Philosophical Magazine』 (英 国 の 物 理 学 専 門 誌) に 投 稿 し、 国 際 的 に 公 表 し た。

***

### 2. 当 時 の ラ イ バ ル ─ ト ム ソ ン の「ブ ド ウ パ ン 模 型」

長 岡 が こ の 模 型 を 発 表 し た 時 期 は、 イ ギ リ ス の J.J. ト ム ソ ン (電 子 の 発 見 者・1906 年 ノ ー ベ ル 賞) も 別 の 原 子 模 型 を 提 案 し て い た。

ト ム ソ ン の 模 型 は、 こ う で あ っ た。

「原 子 全 体 が 正 電 荷 を 持 っ た ふ わ ふ わ し た 球 で、 そ の 中 に 電 子 が ブ ド ウ パ ン の レ ー ズ ン の よ う に 点 在 し て い る」(Plum Pudding Model)

長 岡 の「土 星 模 型」 と ト ム ソ ン の「ブ ド ウ パ ン 模 型」 は、 当 時 競 合 す る 二 大 理 論 で あ っ た。

***

### 3. 1911 年 ─ ラ ザ フ ォ ー ド の 実 験

8 年 後 の 1911 年、 イ ギ リ ス の マ ン チ ェ ス タ ー 大 学 で、 ア ー ネ ス ト・ラ ザ フ ォ ー ド が 弟 子 の ガ イ ガ ー と マ ー ス デ ン と 共 に 行 っ た 実 験 が、 こ の 論 争 に 決 着 を つ け た。

金 箔 に α 粒 子 (ヘ リ ウ ム の 原 子 核) を 当 て て、 ど の よ う に 散 乱 さ れ る か を 観 測 す る 実 験。 ト ム ソ ン の 模 型 (ふ わ ふ わ し た 正 電 荷) が 正 し け れ ば、 α 粒 子 は あ ま り 散 乱 さ れ な い は ず で あ っ た。

し か し、 結 果 は 違 っ た。 ほ と ん ど の α 粒 子 は 金 箔 を す り 抜 け た が、 約 1 万 個 に 1 個 の 割 合 で、 α 粒 子 が ほ ぼ 真 後 ろ に 跳 ね 返 さ れ る こ と が 観 測 さ れ た の で あ る。

「こ れ は、 ま る で 紙 に 銃 弾 を 撃 っ た ら、 そ の 弾 が 真 後 ろ に 跳 ね 返 っ て き た よ う な も の だ」 と ラ ザ フ ォ ー ド は 後 に 述 べ た。

こ の 実 験 か ら、 ラ ザ フ ォ ー ド は こ う 結 論 し た。

「原 子 の ほ と ん ど は 空 っ ぽ で あ る。 し か し、 そ の 中 心 に、 非 常 に 小 さ く、 非 常 に 重 い、 正 電 荷 の 粒 (原 子 核) が あ る。 そ の 周 り を、 電 子 が 回 っ て い る。」

こ れ が「ラ ザ フ ォ ー ド 模 型」 で あ っ た。

***

### 4. 長 岡 模 型 が「事 実 上 正 し か っ た」 こ と

ラ ザ フ ォ ー ド 模 型 と 長 岡 模 型 を 比 べ て み る と、 構 造 は 本 質 的 に 同 じ で あ る。 「中 心 に 大 き な 正 電 荷、 そ の 周 り を 電 子 が 回 る」。

つ ま り、 1903 年 の 長 岡 の 提 案 は、 1911 年 の ラ ザ フ ォ ー ド の 実 験 に よ っ て 確 認 さ れ た の で あ る。 ラ ザ フ ォ ー ド 自 身、 後 に 長 岡 の 業 績 を 称 え て い る。

「日 本 か ら の 提 案 が、 8 年 早 か っ た」 ─ こ れ は 後 の 原 子 物 理 学 史 で 一 つ の 象 徴 的 事 実 と な っ た。

***

### 5. 安 定 性 の 問 題 と 1913 年 ボ ー ア 模 型

し か し、 長 岡 模 型 (お よ び ラ ザ フ ォ ー ド 模 型) に は、 古 典 物 理 学 上 の 問 題 が あ っ た。

「電 子 が 中 心 か ら 引 力 を 受 け る だ け な ら、 電 子 は 螺 旋 を 描 い て 中 心 に 落 下 す る は ず。 原 子 が 安 定 に 存 在 で き る の は な ぜ か」

こ の 問 題 を 解 決 し た の が、 1913 年 ニ ー ル ス・ボ ー ア (デ ン マ ー ク) の 量 子 仮 説 で あ っ た。 「電 子 は 特 定 の 軌 道 (定 常 状 態) で の み 存 在 で き、 古 典 物 理 学 の よ う に 連 続 的 に エ ネ ル ギ ー を 放 出 で き な い」 と ボ ー ア は 提 案 し た。

こ れ で、 長 岡 模 型 → ラ ザ フ ォ ー ド 模 型 → ボ ー ア 模 型 と い う 系 譜 が 完 成 し、 現 代 の 量 子 力 学 的 原 子 観 (1920 年 代 の シ ュ レ ー デ ィ ン ガ ー 方 程 式 と ハ イ ゼ ン ベ ル ク 不 確 定 性 原 理) へ と つ な が る 道 が 開 か れ た。

***

### 6. 日 本 物 理 学 の 出 発 点

長 岡 半 太 郎 が 1903 年 に 残 し た こ の 業 績 は、 日 本 の 近 代 物 理 学 の 国 際 デ ビ ュ ー で あ っ た。

長 岡 の 弟 子 か ら 弟 子 へ と 伝 わ っ た 物 理 学 の 系 譜 は、 後 に 湯 川 秀 樹 (1907-1981、 中 間 子 論 で 1949 年 日 本 人 初 の ノ ー ベ ル 物 理 学 賞)、 朝 永 振 一 郎 (1906-1979、 1965 年 ノ ー ベ ル 物 理 学 賞)、 江 崎 玲 於 奈 (1925-、 1973 年 ノ ー ベ ル 物 理 学 賞) ら を 生 ん だ。

1903 年 の 東 京 数 学 物 理 学 会 で の 一 つ の 発 表 が、 100 年 後 の 日 本 物 理 学 の 礎 と な っ た。

そ し て、 長 岡 半 太 郎 は 1937 年、 文 化 勲 章 第 一 回 受 章 者 と し て 顕 彰 さ れ、 1950 年 に 85 歳 で 没 し た。 そ の 生 涯 は、 日 本 が 世 界 の 科 学 共 同 体 に 加 わ っ た 半 世 紀 と、 ほ ぼ 重 な る も の で あ っ た。
