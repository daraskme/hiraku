## 風 船 玉

寺 田 寅 彦

子 ど も が 縁 日 で 買 っ て き た ゴ ム 風 船 玉 を、 い ち に ち 中 引 き ず り 回 し て 遊 ん で い た。 夕 方 に な っ て、 風 船 は も う 半 分 ほ ど に 萎 ん で い た。 翌 朝 起 き て み る と、 そ れ は ち い さ な 皺 だ ら け の 袋 と な っ て、 畳 の 上 に 転 が っ て い た。

風 船 が 萎 む と い う こ と は、 中 の 気 体 が 抜 け た と い う こ と で あ る。 で は、 ど こ か ら ど の よ う に 抜 け た の か。 ゴ ム の 表 面 は、 目 で 見 る か ぎ り、 ど こ に も 穴 が あ い て い る よ う に 見 え な い。 そ れ で も 気 体 は 抜 け る。

物 理 学 的 に い え ば、 ゴ ム も、 紙 も、 ガ ラ ス で す ら、 分 子 の レ ベ ル で 見 れ ば、 連 続 し た 一 枚 の 板 で は な い。 そ こ に は 無 数 の 隙 間 が あ り、 そ の 隙 間 を、 風 船 の 中 の 気 体 分 子 が、 一 個 ま た 一 個 と、 ゆ っ く り と 通 り 抜 け て ゆ く。 こ の 過 程 が「拡 散」と 呼 ば れ る も の で あ る。

***

気 体 の 分 子 は、 静 止 し て い る わ け で は な い。 室 温 で も、 空 気 中 の 窒 素 分 子 は 一 秒 間 に 数 百 メ ー ト ル と い う 速 さ で 飛 び 回 っ て い る。 そ し て、 互 い に ぶ つ か り 合 い、 容 器 の 壁 に ぶ つ か り、 ま た 跳 ね 返 る。 そ の 衝 突 の 結 果 が、 私 た ち が 感 じ る「圧 力」で あ る。

風 船 が ふ く ら ん で い る の は、 中 の 気 体 分 子 が、 ゴ ム の 内 側 を 絶 え ず 突 い て い る か ら で あ り、 ま た ゴ ム 自 身 の 弾 性 が、 そ の 圧 力 と つ り あ う 形 で 引 き 戻 し て い る か ら で あ る。 そ れ は 動 か な い 静 止 の よ う で い て、 実 は 一 秒 ご と に、 一 兆 兆 個 の 分 子 が、 ゴ ム を 叩 い て い る 動 的 な 均 衡 で あ る。

***

風 船 が ゆ っ く り 萎 む の は、 内 部 の 気 体 分 子 が、 一 個 ま た 一 個 と、 ゴ ム の 微 小 な 隙 間 を 抜 け て 外 へ 出 て ゆ く か ら で あ る。 外 の 空 気 も も ち ろ ん ゴ ム の 隙 間 を 抜 け て 中 に 入 っ て く る。 し か し 一 般 に、 中 の 気 体 (例 え ば ヘ リ ウ ム) の ほ う が 外 の 空 気 よ り も 小 さ な 分 子 で あ る か ら、 出 て 行 く ほ う が 入 っ て く る ほ う よ り 速 い。 そ の 差 が、 風 船 を 萎 ま せ る。

ヘ リ ウ ム を 詰 め た 風 船 が 一 日 で 萎 ん で し ま う の も、 空 気 を 詰 め た 風 船 が 数 日 し か も た な い の も、 す べ て こ の「分 子 の 大 き さ」と「拡 散 速 度」の 関 係 で 説 明 で き る。

***

子 ど も の 縁 日 の 玩 具 の 中 に も、 物 理 学 の 美 し い 法 則 が 隠 れ て い る。 い や、 物 理 学 と い う も の は、 そ も そ も 日 常 の 中 か ら 始 ま っ た 学 問 で あ っ た。 風 船 玉 の よ う な 小 さ な も の の 中 に、 大 気 の 拡 散 や、 浸 透 圧 や、 さ ら に は 宇 宙 ま で の 物 理 が 隠 れ て い る こ と を 知 る と、 子 ど も の 玩 具 が、 ふ し ぎ な 哲 学 の 教 科 書 の よ う に 思 え て く る。

風 船 が 萎 む の を ぼ ん や り 眺 め て い る 子 ど も の 横 で、 私 は し ば し ば、 そ う い う こ と を 考 え る。

***

(注：本 教 材 は、 寺 田 寅 彦 の 随 筆『風 船 玉』 (1922 年 発 表) の 趣 旨 を も と に、 教 材 用 に 編 集 部 が 再 構 成 し た 試 訳 で あ る。 原 文 と は 異 な る 箇 所 が あ る。 原 文 は 青 空 文 庫 で 公 開 さ れ て い る。)
