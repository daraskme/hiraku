## 中 間 子 論 ─ 日 本 人 初 の ノ ー ベ ル 賞

### 1. 原 子 核 の 謎

20 世 紀 初 頭、 物 理 学 者 た ち は 原 子 の 構 造 を 解 明 し つ つ あ っ た。 1911 年 ラ ザ フ ォ ー ド が「原 子 の 中 心 に 小 さ く て 重 い 原 子 核 が あ る」 こ と を 示 し、 1932 年 チ ャ ド ウ ィ ッ ク が「中 性 子」 を 発 見。 原 子 核 は「陽 子」 と「中 性 子」 か ら で き て い る こ と が 分 か っ た。

し か し、 こ こ で 大 き な 謎 が 残 っ た。

陽 子 は、 プ ラ ス の 電 気 を 持 っ て い る。 同 じ プ ラ ス ど う し は、 電 気 的 に 反 発 す る。 原 子 核 の 中 に は 陽 子 が ぎ っ し り 詰 ま っ て い る の だ か ら、 本 来 な ら 反 発 し 合 っ て バ ラ バ ラ に 飛 び 散 る は ず だ。

そ れ な の に、 原 子 核 は 安 定 し て い る。 な ぜ か?

何 か、 電 気 的 な 反 発 力 を 上 回 る「強 い 力」 が、 陽 子 と 中 性 子 を 結 び つ け て い る に 違 い な い。 こ の 力 を「核 力 (か く り ょ く)」 と 呼 ぶ。 し か し、 そ の 正 体 は 誰 に も 分 か ら な か っ た。

***

### 2. 1935 年、 28 歳 の 湯 川 の ひ ら め き

こ の 謎 に 挑 ん だ の が、 当 時 大 阪 帝 国 大 学 講 師 だ っ た 28 歳 の 湯 川 秀 樹 で あ っ た。

湯 川 は こ う 考 え た。

「二 つ の 粒 子 の 間 に 働 く 力 は、 別 の 粒 子 を キ ャ ッ チ ボ ー ル の よ う に や り 取 り す る こ と で 生 ま れ る の で は な い か。」

例 え ば、 電 気 の 力 (電 磁 気 力) は、 「光 子 (フ ォ ト ン)」 と い う 粒 子 を や り 取 り す る こ と で 生 ま れ る ─ こ れ は 当 時 す で に 分 か り 始 め て い た。 で は、 核 力 も、 何 か の 粒 子 を や り 取 り し て い る の で は な い か。

湯 川 は、 さ ら に 重 要 な こ と に 気 づ い た。 「核 力 は、 原 子 核 の 大 き さ 程 度 の 近 距 離 で し か 働 か な い。 力 の 到 達 距 離 は、 媒 介 す る 粒 子 の 質 量 で 決 ま る ─ 重 い 粒 子 ほ ど 近 距 離 し か 届 か な い。」

そ こ で 湯 川 は、 核 力 の 到 達 距 離 か ら 逆 算 し て、 媒 介 粒 子 の 質 量 を 計 算 し た。 答 え は、 電 子 の 約 200 倍。 陽 子 よ り は 軽 く、 電 子 よ り は ず っ と 重 い ─ つ ま り「中 間」 の 質 量 を 持 つ 粒 子 で あ る。

湯 川 は こ れ を「中 間 子 (ち ゅ う か ん し、 メ ソ ン)」 と 名 づ け、 そ の 存 在 を 理 論 的 に 予 言 し た。

***

### 3. 1935 年 の 論 文

1935 年、 湯 川 は『素 粒 子 の 相 互 作 用 に つ い て・第 一 報』 を 日 本 数 学 物 理 学 会 記 事 に 発 表 し た (英 語)。

し か し、 当 時 は ほ と ん ど 注 目 さ れ な か っ た。 「そ ん な 未 知 の 粒 子 が 本 当 に あ る の か」 と 半 信 半 疑 だ っ た の で あ る。 ま だ 観 測 さ れ て い な い 粒 子 を「理 論 だ け で 予 言 す る」 と い う の は、 大 胆 す ぎ る 主 張 で あ っ た。

***

### 4. 1947 年 ─ パ イ 中 間 子 の 発 見

12 年 後 の 1947 年。 イ ギ リ ス の セ シ ル・パ ウ エ ル ら が、 高 い 山 の 上 で「宇 宙 線」 (宇 宙 か ら 降 り 注 ぐ 高 エ ネ ル ギ ー 粒 子) を 写 真 乾 板 で 観 測 し て い た。

そ し て、 そ の 中 か ら、 湯 川 が 予 言 し た と お り の 質 量 (電 子 の 約 273 倍) を 持 つ 粒 子 を 発 見 し た。 「パ イ 中 間 子 (π 中 間 子)」 で あ る。

湯 川 の 1935 年 の 予 言 が、 実 験 で 実 証 さ れ た 瞬 間 で あ っ た。

***

### 5. 1949 年 ─ 日 本 人 初 の ノ ー ベ ル 賞

1949 年 (昭 和 24 年)、 湯 川 秀 樹 は 中 間 子 論 に よ り、 日 本 人 と し て 初 め て ノ ー ベ ル 物 理 学 賞 を 受 賞 し た。

こ の 時 期 の 日 本 は、 第 二 次 世 界 大 戦 の 敗 戦 (1945 年) か ら わ ず か 4 年。 国 土 は 焼 け 野 原 で、 食 糧 難 が 続 き、 国 民 は 自 信 を 失 っ て い た。

そ ん な 中 で の、 日 本 人 初 の ノ ー ベ ル 賞。 日 本 中 が 沸 き 立 っ た。 「日 本 人 も 世 界 に 通 用 す る」「焼 け 跡 か ら で も、 知 の 力 で 世 界 の 頂 点 に 立 て る」 ─ 湯 川 の 受 賞 は、 打 ち ひ し が れ た 国 民 に、 大 き な 希 望 を 与 え た。

***

### 6. 平 和 へ の 願 い

湯 川 は、 そ の 後 も 物 理 学 の 研 究 を 続 け る 一 方 で、 核 兵 器 廃 絶 運 動 に も 力 を 注 い だ。 1955 年、 ア イ ン シ ュ タ イ ン・ラ ッ セ ル ら と 共 に「ラ ッ セ ル・ア イ ン シ ュ タ イ ン 宣 言」 に 署 名 (湯 川 は 日 本 人 唯 一 の 署 名 者)。 核 兵 器 の 廃 絶 と 科 学 者 の 社 会 的 責 任 を 訴 え た。

自 ら の 研 究 (原 子 核 物 理 学) が 原 子 爆 弾 に つ な が り う る こ と を 深 く 自 覚 し て い た 湯 川 は、 「科 学 は 人 類 の 幸 福 の た め に 使 わ れ る べ き」 と い う 信 念 を 生 涯 持 ち 続 け た。

***

### 7. 系 譜

湯 川 秀 樹 の 中 間 子 論 は、 日 本 の 物 理 学 の 系 譜 の 中 に あ る。

1903 年、 長 岡 半 太 郎 が 土 星 型 原 子 模 型 を 提 案 し、 日 本 の 原 子 物 理 学 を 始 め た。 そ の 弟 子 か ら 弟 子 へ と 伝 わ っ た 知 が、 湯 川 秀 樹 (1949 年 ノ ー ベ ル 賞)、 朝 永 振 一 郎 (1965 年 ノ ー ベ ル 賞、 湯 川 の 同 級 生・盟 友) を 生 ん だ。

1903 年 の 長 岡 か ら 1949 年 の 湯 川 ま で、 約 半 世 紀。 日 本 の 物 理 学 が「世 界 に 追 い つ き、 そ し て 世 界 の 頂 点 に 立 つ」 ま で の 物 語 で あ る。

そ し て、 そ の 出 発 点 は、 28 歳 の 青 年 が「原 子 核 の 中 で 陽 子 が 飛 び 散 ら な い の は な ぜ か」 と い う、 素 朴 な 疑 問 に 真 剣 に 向 き 合 っ た こ と で あ っ た。

***
