<h3>1687 年、 ロ ン ド ン</h3>

<p>
1687 年 7 月 5 日、 ロ ン ド ン で 一 冊 の 本 が 出 版 さ れ た。<br>
タ イ ト ル は『<strong>自 然 哲 学 の 数 学 的 原 理</strong>』(Philosophiæ Naturalis Principia Mathematica) ─ 通 称『<strong>プ リ ン キ ピ ア</strong>』。
</p>

<p>
著 者 は ケ ン ブ リ ッ ジ 大 学 教 授 <strong>ア イ ザ ッ ク・ニ ュ ー ト ン</strong> (Isaac Newton, 1643–1727)。 44 歳 だ っ た。
</p>

<p>
こ の 本 が、 物 理 学 と 天 文 学 を 永 遠 に 変 え た。
</p>

<h3>本 文 抜 粋 (試 訳)</h3>

<blockquote>
<p>
全 て の 物 体 は、 互 い に 引 き 合 う 力 を 持 つ。<br>
こ の 力 は ─ 二 つ の 物 体 の 質 量 の 積 に 比 例 し、 距 離 の 2 乗 に 反 比 例 す る。
</p>

<blockquote>
F = G × (m₁ × m₂) / r²
</blockquote>

<p>
こ こ で、 F は 引 力、 m₁ と m₂ は 二 つ の 物 体 の 質 量、 r は 二 つ の 物 体 間 の 距 離、 G は 万 有 引 力 定 数 で あ る。
</p>

<p>
こ の 法 則 は、 宇 宙 の ど こ で も、 ど の 物 体 同 士 で も 成 立 す る。<br>
だ か ら 私 は、 こ れ を 「<strong>万 有 引 力</strong>」(universal gravitation) と 呼 ぶ。
</p>

<p>
こ の 法 則 で、 リ ン ゴ が 木 か ら 落 ち る 理 由 も、 月 が 地 球 の 周 り を 回 る 理 由 も、 地 球 が 太 陽 の 周 り を 回 る 理 由 も、 全 て が 説 明 で き る。<br>
そ し て、 ケ プ ラ ー が 観 測 か ら 導 い た 惑 星 運 動 の 三 法 則 も、 こ の 万 有 引 力 か ら 数 学 的 に 導 出 で き る。
</p>

<p>
天 上 の 世 界 と 地 上 の 世 界 は、 別 々 の 法 則 に 従 う も の で は な い。<br>
両 者 は、 同 じ 一 つ の 法 則 で 統 一 さ れ る の だ。
</p>
</blockquote>

<h3>「天 と 地 の 統 一」 ─ 17 世 紀 の 革 命</h3>

<p>
こ の 一 つ の 法 則 が、 何 世 紀 に も わ た っ て 続 い た 「<strong>天 と 地 は 別 の 世 界</strong>」 と い う 古 い 信 念 を、 完 全 に 葬 り 去 っ た。
</p>

<p>
ニ ュ ー ト ン 以 前 ─
</p>

<ul>
<li>地 上 の 物 体 ─ 落 ち る、 飛 ぶ、 転 が る (地 上 の 力 学)</li>
<li>天 上 の 物 体 ─ 月 ・ 惑 星 ・ 星 は、 神 が 定 め た 道 を 回 る (天 文 学)</li>
</ul>

<p>
こ の 二 つ は、 ま っ た く 別 の 世 界 と 考 え ら れ て い た。<br>
ア リ ス ト テ レ ス 以 来 2000 年 続 い た 世 界 観 で あ る。
</p>

<p>
ニ ュ ー ト ン は、 こ の 二 つ を「<strong>同 じ 一 つ の 法 則 (万 有 引 力)</strong>」 で 説 明 し た。
</p>

<ul>
<li>リ ン ゴ が 落 ち る (地 上) = 地 球 と リ ン ゴ の 万 有 引 力</li>
<li>月 が 地 球 を 回 る (天 上) = 地 球 と 月 の 万 有 引 力</li>
<li>地 球 が 太 陽 を 回 る (天 上) = 太 陽 と 地 球 の 万 有 引 力</li>
</ul>

<p>
全 部 同 じ 法 則 ─ こ れ は、 17 世 紀 の 知 識 人 に と っ て 想 像 を 超 え る 衝 撃 だ っ た。
</p>

<h3>「奇 跡 の 年」 1665–1666</h3>

<p>
ニ ュ ー ト ン が こ れ ら の 発 見 の 多 く を 構 想 し た の は、 22–23 歳 の と き だ っ た。
</p>

<p>
1665 年、 ロ ン ド ン で <strong>ペ ス ト の 大 流 行</strong>。 ケ ン ブ リ ッ ジ 大 学 は 閉 鎖 さ れ、 ニ ュ ー ト ン は 故 郷 リ ン カ ン シ ャ ー の ウ ル ス ソ ー プ に 帰 省 し た。
</p>

<p>
こ の 故 郷 で の 1 年 半 (1665–1666) で、 彼 は 以 下 を ほ ぼ 構 想 し た:
</p>

<ul>
<li><strong>微 積 分 学</strong> ─ 変 化 を 扱 う 新 し い 数 学</li>
<li><strong>光 学</strong> ─ 白 色 光 が 七 色 に 分 か れ る こ と の 発 見 (プ リ ズ ム 実 験)</li>
<li><strong>万 有 引 力 の 概 念</strong> ─ リ ン ゴ の 落 下 と 月 の 運 動 の 統 一</li>
</ul>

<p>
こ の 1 年 半 を、 後 世 は 「<strong>ニ ュ ー ト ン の 奇 跡 の 年</strong>」(Annus Mirabilis) と 呼 ぶ。<br>
22 歳 の 一 人 の 若 者 が、 後 の 300 年 の 物 理 学・数 学 の 基 礎 を 一 人 で 構 想 し た 期 間 で あ る。
</p>

<h3>リ ン ゴ の 逸 話</h3>

<p>
有 名 な「<strong>リ ン ゴ が 木 か ら 落 ち る の を 見 て 万 有 引 力 を 思 い つ い た</strong>」 と い う 話 は、 真 実 か?
</p>

<p>
ニ ュ ー ト ン 自 身 が 晩 年、 知 人 (ウ ィ リ ア ム ・ ス ト ゥ ー ク レ イ) に こ う 語 っ た 記 録 が 残 る:
</p>

<blockquote>
「あ る 日、 庭 で 思 索 に ふ け っ て い た。 リ ン ゴ が 木 か ら 落 ち る の を 見 て、 ふ と 思 っ た ─ 『な ぜ こ の リ ン ゴ は 真 っ 直 ぐ 下 に 落 ち る の か。 横 や 上 で は な く、 必 ず 地 球 の 中 心 に 向 か っ て?』<br>
そ し て 考 え 続 け た: 『同 じ 力 が、 月 に も 働 い て い る な ら ば ─ そ の 力 で 月 が 地 球 を 回 る こ と も 説 明 で き る の で は な い か?』」
</blockquote>

<p>
リ ン ゴ の 木 ─ 今 で も ウ ル ス ソ ー プ に 子 孫 が 育 っ て い る ─ が、 物 理 学 史 を 変 え た の だ。
</p>

<h3>『プ リ ン キ ピ ア』 ま で の 20 年</h3>

<p>
1666 年 に 構 想 し た 万 有 引 力 を、 ニ ュ ー ト ン が 公 表 し た の は <strong>20 年 以 上 後</strong> の 1687 年 だ っ た。
</p>

<p>
な ぜ そ ん な に か か っ た か?
</p>

<ul>
<li>完 璧 な 数 学 的 証 明 を 構 築 す る ま で 公 表 を 控 え た</li>
<li>そ の た め に、 自 ら 微 積 分 学 を 完 成 さ せ る 必 要 が あ っ た</li>
<li>1684 年、 友 人 の 天 文 学 者 ハ レ ー (彗 星 で 有 名) が「あ な た は 答 え を 知 っ て い る は ず」 と 訪 ね て き て、 出 版 を 強 く 勧 め た</li>
<li>ハ レ ー が 自 費 で 出 版 を 援 助 ─ 1687 年 7 月 5 日 出 版</li>
</ul>

<h3>ニ ュ ー ト ン の 運 動 三 法 則</h3>

<p>
『プ リ ン キ ピ ア』 の 冒 頭 に、 三 つ の 運 動 法 則 が 提 示 さ れ る:
</p>

<table>
<thead>
<tr><th>法 則</th><th>内 容</th></tr>
</thead>
<tbody>
<tr><td><strong>第 一 (慣 性 の 法 則)</strong></td><td>外 力 が な け れ ば、 物 体 は 静 止 ま た は 等 速 直 線 運 動 を 続 け る</td></tr>
<tr><td><strong>第 二 (運 動 方 程 式)</strong></td><td>F = ma (力 = 質 量 × 加 速 度)</td></tr>
<tr><td><strong>第 三 (作 用 反 作 用)</strong></td><td>す べ て の 作 用 に は、 同 じ 大 き さ で 反 対 向 き の 反 作 用 が あ る</td></tr>
</tbody>
</table>

<p>
高 校 物 理 で 学 ぶ 古 典 力 学 の <strong>全 て の 基 礎</strong> が、 こ の 三 つ の 法 則 に 含 ま れ て い る。
</p>

<h3>「巨 人 の 肩 の 上 に 立 つ」</h3>

<p>
ニ ュ ー ト ン 自 身 は、 自 分 の 業 績 を こ う 語 っ た:
</p>

<blockquote>
「私 は ど う し て こ ん な に 遠 く ま で 見 え た の か。 そ れ は、 <strong>巨 人 の 肩 の 上 に 立 っ て い た</strong> か ら だ」
</blockquote>

<p>
彼 が 言 う「巨 人」 と は、 ガ リ レ オ・ケ プ ラ ー・デ カ ル ト ら、 先 行 す る 科 学 者 の こ と。<br>
ニ ュ ー ト ン は、 彼 ら の 業 績 を 統 合 し、 さ ら に 高 い 頂 点 に 到 達 し た。
</p>

<h3>ニ ュ ー ト ン 力 学 の 限 界</h3>

<p>
ニ ュ ー ト ン 力 学 は、 約 200 年 間「絶 対 真 理」 と さ れ て き た。<br>
だ が 20 世 紀 に な っ て、 そ の 限 界 が 明 ら か に な っ た:
</p>

<ul>
<li><strong>1905 ・ 1915 ─ ア イ ン シ ュ タ イ ン の 相 対 性 理 論</strong> ─ 光 速 に 近 い 領 域 で は ニ ュ ー ト ン 力 学 が 通 用 し な い</li>
<li><strong>1920s ─ 量 子 力 学</strong> ─ 原 子 の よ う な ミ ク ロ の 世 界 で は ニ ュ ー ト ン 力 学 が 通 用 し な い</li>
</ul>

<p>
だ が、 日 常 ス ケ ー ル ─ 物 が 落 ち る、 車 が 走 る、 ロ ケ ッ ト が 飛 ぶ ─ で は、 今 も ニ ュ ー ト ン 力 学 が 完 全 に 正 し い。<br>
NASA の 宇 宙 船 軌 道 計 算 も、 今 も ニ ュ ー ト ン 力 学 で 行 わ れ て い る。
</p>

<h3>君 の 周 り の ニ ュ ー ト ン</h3>

<p>
今 君 の 周 り の あ ら ゆ る 物 体 は、 互 い に 引 き 合 っ て い る:
</p>

<ul>
<li>君 と 教 室 の 机 ─ 万 有 引 力 が 働 く (が 非 常 に 小 さ い)</li>
<li>君 と 地 球 ─ そ の 力 が 「重 力」(= 君 の 体 重 と し て 体 感)</li>
<li>地 球 と 月 ─ そ の 力 で 月 が 地 球 を 回 る</li>
<li>太 陽 と 地 球 ─ そ の 力 で 一 年 周 期 の 公 転</li>
</ul>

<p>
全 部、 1687 年 の ニ ュ ー ト ン の 方 程 式 F = G × (m₁ × m₂) / r² で 計 算 で き る。
</p>

</p>
