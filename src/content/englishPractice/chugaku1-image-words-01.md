  - term: spring
    kind: word
    meaning_ja: 春、バネ、泉、跳ねる
    part_of_speech: noun / verb
    pronunciation: spring
    core_image:
      summary: 内側にたまった力や命が、外へぱっと跳ね出すイメージ。
      src: /illustrations/englishPractice/spring-core/scene1.webp
      alt: バネ、春の芽、泉が同じ外へ跳ね出す動きでつながっている概念図。
      caption: spring の中心には、押しこめられていたものが外へ勢いよく現れる感覚がある。
      visual_prompt: stored energy suddenly bursting outward, shown as a metal spring, springtime growth, and water from a natural spring.
    senses:
      - label: バネ
        meaning_ja: 押されていた力が、元に戻ろうとして跳ねるもの。
        image_note: ぎゅっと縮んだ金属が、外へ押し返す。
        examples:
          - en: The spring bounced back.
            ja: バネが跳ね返った。
      - label: 春
        meaning_ja: 冬のあいだ隠れていた命が、地面からいっせいに現れる季節。
        image_note: 芽や花が、土の中からぱっと出てくる。
        examples:
          - en: Spring comes after winter.
            ja: 春は冬のあとに来る。
      - label: 泉
        meaning_ja: 地下にあった水が、地面から湧き出る場所。
        image_note: 見えない地下の水が、表面へ押し出される。
        examples:
          - en: Water springs from the ground.
            ja: 水が地面から湧き出る。
      - label: 跳ねる
        meaning_ja: 急に飛び出す、跳び上がる。
        image_note: 体やものが、一瞬で別の位置へ動く。
        examples:
          - en: The cat sprang onto the table.
            ja: その猫はテーブルへぱっと跳び上がった。
    examples:
      - en: I like spring.
        ja: 私は春が好きです。
      - en: This pen has a small spring.
        ja: このペンには小さなバネがあります。
      - en: There is a spring near the mountain.
        ja: 山の近くに泉があります。
    quiz:
      - spring の三つの意味を、同じイメージで説明しましょう。
      - 「水が地面から湧き出る」を spring を使って言いましょう。
      - 「春」と「バネ」に共通する動きを日本語で説明しましょう。
  - term: run

      alt: 走る人、流れる水、動く機械、店の運営が一本の流れでつながっている概念図。
      caption: run の中心には、何かが道筋にそって続いて動いている感覚がある。
      visual_prompt: continuous movement along a path or system, shown as a runner, running water, a running machine, and a smoothly run shop.
    senses:
      - label: 走る
        meaning_ja: 人や動物が、足をすばやく動かして前へ進む。
        image_note: 体が道にそって止まらず進む。
        examples:
          - en: I run every morning.
            ja: 私は毎朝走ります。
      - label: 流れる
        meaning_ja: 水や道などが、ある方向へ続いている。
        image_note: 水の流れが、曲がりながら先へ進む。
        examples:
          - en: The river runs through the town.
            ja: 川が町の中を流れています。
      - label: 動く
        meaning_ja: 機械やシステムが、止まらずに働いている。
        image_note: 歯車や仕組みが、連続して動き続ける。
        examples:
          - en: The machine is running.
            ja: 機械が動いています。
      - label: 運営する
        meaning_ja: 店や計画などを、うまく動くように続けていく。
        image_note: 人やものの流れを整えて、仕事が止まらないようにする。
        examples:
          - en: She runs a small shop.
            ja: 彼女は小さな店を営んでいます。
    examples:
      - en: We run in the park.
        ja: 私たちは公園で走ります。
      - en: This bus runs every ten minutes.
        ja: このバスは10分ごとに運行しています。
      - en: My father runs a restaurant.
        ja: 私の父はレストランを経営しています。
    quiz:
      - run の「走る」と「運営する」に共通するイメージを説明しましょう。
      - 「機械が動いています」を run を使って言いましょう。
      - river runs と bus runs の run は、どんな動きとして似ていますか。

        meaning_ja: 高い位置にあったものが、下へ移動する。
        image_note: 葉やものが重力に引かれて下へ向かう。
        examples:
          - en: Leaves fall in autumn.
            ja: 秋には葉が落ちます。
      - label: 降る
        meaning_ja: 雨や雪が、空から地面へ落ちてくる。
        image_note: 雲の上の水が、下へ細く落ちてくる。

            ja: 雨が雲から降ります。
      - label: 下がる
        meaning_ja: 数や気温などが、高い状態から低い状態へ移る。
        image_note: 目に見えない数値が、段を下りるように低くなる。
        examples:
          - en: Prices fell yesterday.
            ja: 昨日、値段が下がりました。
      - label: 秋
        meaning_ja: 葉が落ち、日が低くなり、夏の勢いが落ち着く季節。
        image_note: 木の葉や日ざしが、ゆっくり下へ向かう。
        examples:
          - en: Fall comes after summer.
            ja: 秋は夏のあとに来ます。
    examples:
      - en: Be careful not to fall.
        ja: 落ちないように気をつけて。
      - en: Snow falls quietly.
        ja: 雪が静かに降ります。
      - en: The temperature fell at night.
        ja: 夜に気温が下がりました。
    quiz:
      - fall の「落ちる」と「秋」に共通するイメージを説明しましょう。
      - 「雨が降る」を fall を使って言いましょう。
      - prices fell の fell は、何がどちら向きに動いたイメージですか。
