<h3>「し た し た し た … 暗 い 闇 を 引 い て 来 る」</h3>

<p>
小 説『死 者 の 書』 (し し ゃ の し ょ) の 冒 頭 ─
</p>

<blockquote>
<p>
し た し た し た。 ……<br>
し た し た し た。 ……<br>
耳 に 聞 え て 来 る の は、 そ う し た 物 音 ば か り で あ る。 ……
</p>
</blockquote>

<p>
死 後 の 闇 の 中、 何 か が 滴 (し た た) る 音 だ け が 響 く。 こ こ で 目 覚 め る の は、 古 代 大 和 で
非 業 の 死 を 遂 げ た 皇 子 ─ 大 津 皇 子 (お お つ の み こ) の 魂 で あ る。
</p>

<p>
折 口 信 夫 (1887–1953) が、 国 文 学 者・民 俗 学 者 と し て の 全 知 識 を 注 ぎ 込 ん で 書 い た 異 色 の 名 作
を、 高 3 国 語 で 読 み 解 こ う。
</p>

<h3>折 口 信 夫 ─ 学 者 で あ り 詩 人</h3>

<p>
折 口 信 夫 は、 1887 年 大 阪 府 生 ま れ。 國 學 院 大 学 で 国 文 学 を 学 び、 後 に 慶 應 義 塾 大 学・
國 學 院 大 学 で 教 授 と な っ た。 柳 田 国 男 と 並 ぶ「日 本 民 俗 学 の 双 璧」 と さ れ る。
</p>

<p>
学 問 と 並 行 し て、 短 歌・小 説 も 書 い た ─ 筆 名 は「釈 迢 空 (し ゃ く ち ょ う く う)」。 仏 教
を 学 び、 古 代 信 仰 を 文 学 に 結 晶 さ せ る 二 面 性 が、 折 口 の 個 性 で あ る。
</p>

<h3>大 津 皇 子 ─ 24 歳 で 死 を 賜 っ た 皇 子</h3>

<p>
『死 者 の 書』 の 主 人 公 は、 古 代 の 実 在 の 人 物 ─ 大 津 皇 子 (663–686)。 天 武 天 皇 と 大 田 皇 女
(姉 の 持 統 天 皇 と 同 じ 母) の 皇 子。 文 武 両 道 ・ 容 姿 端 麗 ・ 詩 歌 に 秀 で た、 当 時 の 天 才 で あ っ た。
</p>

<p>
し か し、 父 天 武 帝 が 686 年 9 月 に 崩 御 す る と、 そ の 翌 月、 大 津 皇 子 は 謀 反 の 罪 で 死 を 賜 る ─
24 歳 だ っ た。 持 統 天 皇 (姑) が、 自 ら の 皇 子 ・ 草 壁 皇 子 を 皇 太 子 に す る 為 だ っ た と さ れ る
(『日 本 書 紀』『万 葉 集』 等 に 記 述)。
</p>

<h3>大 津 皇 子 の 辞 世 歌 ─ 万 葉 集 に 残 る 悲 痛</h3>

<p>
処 刑 直 前、 大 津 皇 子 が 詠 ん だ と さ れ る 辞 世 歌 が、 『万 葉 集』 巻 三・416 に 残 る ─
</p>

<blockquote>
<p>
百 (も も) 伝 (づ た) ふ 磐 余 (い わ れ) の 池 に 鳴 く 鴨 を<br>
今 日 の み 見 て や 雲 隠 り な む
</p>
</blockquote>

<p>
意 味 ─ 「磐 余 の 池 に 鳴 く 鴨 を、 今 日 限 り 見 て、 私 は 雲 の か な た へ 隠 れ る (死 ぬ) の だ ろ う か」。
</p>

<p>
24 歳 の 才 人 の、 静 か で 哀 し い 辞 世。 大 津 皇 子 の 遺 体 は、 大 和 と 河 内 の 境 の 二 上 山
(に じ ょ う さ ん) に 葬 ら れ た。
</p>

<h3>二 上 山 ─ 葬 送 の 道 と「ま れ び と」</h3>

<p>
二 上 山 は、 雌 岳 ・ 雄 岳 の 二 つ の 峰 を 持 つ 山。 古 代 か ら、 大 和 (現 奈 良) と 河 内 (現 大 阪)
を 結 ぶ「葬 送 の 道」 が 通 っ て お り、 多 く の 古 墳 が 周 囲 に 散 在 す る。
</p>

<p>
折 口 は、 こ の 二 上 山 を 物 語 の 中 心 に 据 え た。 大 津 皇 子 の 魂 が、 死 後 数 十 年 の 時 を 経 て、
藤 原 南 家 郎 女 (い ら つ め ・ 藤 原 南 家 ・ 武 智 麻 呂 の 娘) の 前 に 現 れ る。
</p>

<p>
折 口 が 提 唱 し た「ま れ び と」 (外 界 か ら 訪 れ る 神・異 人) の 概 念 が、 文 学 で 表 現 さ れ た 場 面 だ。
</p>

<h3>民 俗 学 リ ア リ ズ ム ─ 学 問 が 文 学 に な る</h3>

<p>
『死 者 の 書』 の 特 異 さ は、 民 俗 学 の 知 見 が「そ の ま ま 文 学 に な っ て い る」 こ と だ。
</p>

<ul>
  <li>古 代 大 和 の 葬 礼・も が り (殯) ─ 民 俗 学 で 研 究 さ れ る 風 習</li>
  <li>女 性 の 機 織 り と 祭 祀 ─ 古 代 信 仰 で の 女 性 の 役 割</li>
  <li>「ま れ び と」 の 訪 れ ─ 折 口 自 身 の 民 俗 学 概 念</li>
  <li>古 代 大 和 言 葉 ─ 万 葉 集 ・ 古 事 記 に 通 じ た 言 語 感 覚</li>
</ul>

<p>
読 者 は 物 語 を 楽 し む と 同 時 に、 古 代 日 本 文 化 を 体 験 す る こ と に な る。
</p>

<h3>結 び ─ 学 問 と 文 学 を 結 ぶ 試 み</h3>

<p>
通 常、 学 問 と 文 学 は 別 の 領 域 で あ る。 し か し 折 口 は、 自 ら の 民 俗 学 ・ 国 文 学 の 知 見 を、
そ の ま ま 文 学 に 結 晶 さ せ た。
</p>

<p>
こ の 試 み は 現 代 で も 受 け 継 が れ て い る。 京 極 夏 彦 ・ 高 田 崇 史 ・ 宮 部 み ゆ き 等、 日 本 文 化 ・
歴 史 を 物 語 に 結 び 付 け る 現 代 作 家 の 系 譜 は、 折 口 信 夫 ま で 遡 る。
</p>

<p>
高 3 で 読 む『死 者 の 書』 ─ 一 度 読 ん で 終 わ り に せ ず、 何 度 も 戻 っ て き て 欲 し い 一 冊 で あ る。
</p>
