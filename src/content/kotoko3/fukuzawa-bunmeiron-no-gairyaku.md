<h3>本文 (第一章 「議論 の 本位 を 定むる 事」 冒頭・現代仮名遣い)</h3>

<blockquote>
<p>
軽重 長短 善悪 是非 等 の 字 は、相対 して 生じたる 考 (かんがえ) なり。<br>
軽き もの なければ、重き もの の 名 も あらず。<br>
善き もの なければ、悪しき もの の 名 も あらず。
</p>

<p>
ゆえ に、軽 と 重 と、その 一 (いつ) を 知らずして、他 を 名 づくべから ず。
</p>

<p>
俗 に 文明 と いい、開化 と いう も の、まさに この 類 に て、その 重 と 軽 と、長 と 短 と、是 と 非 と を 比較 し て 始めて、その 一 を 名 づくる こと を 得 べき なり。
</p>

<p>
然 (しか) ら ば、何 を もって 軽重 を 定 (さだ) めん か。<br>
「議論 の 本位 (ほんい)」 を 定 むる に あり。
</p>

<p>
本位 と は、一 つ の 基 (もとい) に 立ちて、これ を もって 物 を 判ずる の 標準 を いう。<br>
たとえば、目 の 前 に 大 と 小 と の 二 つ の もの あり と せ ん。これ を 比較 して、大 を 大 と し、小 を 小 と する なら ば、まさに 正しき なり。
</p>

<p>
され ど、もし 別 に、この 二 者 より さらに 大 なる 別 の もの あり と すれ ば、先 の 「大」 は、新 たに 「小」 と なる べし。<br>
ゆえ に、「大」 「小」 は、その もの 自身 の 中 に 元来 存在 する の で は なく、比較 する 場 に よって 定まる もの なり。
</p>

<p>
これ を もって 推す に、文明 と 野蛮 も また、絶対 の もの に は あらず。<br>
今 日 の 日本 が 「文明」 と 呼ぶ もの は、欧米 の 比較 に よって 「未だ 至らず」 と 言える。<br>
だが、その 欧米 自身 を、別 の 時代・別 の 文化 と 比較 する なら ば、それ もまた 一 の 「野蛮」 と 見え る かも しれぬ。
</p>

<p>
ゆえ に、私 たち は、議論 の 出発点 で、何 を 「本位」 と する か を、まず 定めね ば ならぬ。<br>
本位 が 異なる なら ば、結論 もまた 異なる。<br>
正しき 議論 と は、本位 の 確かさ から 始まる の で ある。
</p>
</blockquote>

<h3>解説 ─ 「議論 の 出発点 を 問え」</h3>

<p>
『文明論之概略』 の 第一章 で、福澤諭吉 が 説いた こと は、現代 の 言葉 で 言えば こう なる ──
</p>

<blockquote>
あらゆる 議論 は、その 「<strong>枠組み (フレーム)</strong>」 に よって 結論 が 変わる。<br>
だから 議論 を 始める 前 に、「<strong>何 を 基準 と するか</strong>」 を まず 確認 せよ。
</blockquote>

<p>
これ は、20 世紀 後半 に 米国 で 発達 した <strong>「フレーミング 理論」</strong> (Lakoff など) や、<strong>「議論 の 前提 を 問う」</strong> 哲学 的 反省 を、120 年 先取り した もの と 言える。
</p>

<h3>福澤 の 革命 的 主張</h3>

<p>
1875 年 当時 の 日本 で、福澤 が 主張 した の は、以下 の よう な <strong>三 段 階 の 思考</strong> だった。
</p>

<ol>
<li>
  <strong>第一 段 ─ 相対 性 の 認識</strong><br>
  「軽重・長短・善悪・是非」 など、私 たち が 「絶対」 と 思って いる 概念 は、実 は すべて 「比較」 の 産物 で ある。
</li>
<li>
  <strong>第二 段 ─ 本位 (基準点) の 自覚</strong><br>
  ならば、議論 を する とき、まず 「何 を 基準 と するか」 を 明確 に せ ね ば ならない。
</li>
<li>
  <strong>第三 段 ─ 文明 への 適用</strong><br>
  「日本 は 文明 か、それとも 野蛮 か?」 という 問い も、何 を 基準 と する か で 答え が 変わる。 西洋 を 基準 と する なら 日本 は 未開、東洋 古典 を 基準 と する なら 日本 は 文明 で ある。
</li>
</ol>

<h3>明治 8 年 の 文脈</h3>

<p>
本書 が 書かれた 明治 8 年 (1875) は、日本 が 急速 に 西洋化 して いた 時期 で ある。
</p>

<ul>
<li>1871 ─ 廃藩置県 (本サイト kotorekishi1)</li>
<li>1872 ─ 学制 発布、新橋・横浜 間 鉄道 開通</li>
<li>1873 ─ 徴兵令、地租 改正</li>
<li>1874 ─ 民撰議院 設立 建白 (自由民権 運動 開始)</li>
</ul>

<p>
こうした 急速 な 「西洋化 = 文明化」 が 進む 中 で、福澤 は 「<strong>では、その 『文明』 と は そもそも 何 か?</strong>」 と 問い 直した。
</p>

<p>
彼 の 立場 は、単純 な 「西洋 礼賛」 で は なかった。<br>
むしろ、「西洋 を 鵜呑み に せ ず、自分 たち の 頭 で 『文明 と は 何 か』 を 考えよう」 と 訴えた の で ある。<br>
この <strong>主体 的 で 批判 的 な 知性</strong> こそ が、福澤 が 後 進 の 知識 人 に 託した もの だった。
</p>

<h3>『学問 の すゝめ』 から 『文明論』 へ</h3>

<p>
中学 で 既 に 読んだ 『学問 の すゝめ』 (1872–1876) は、福澤 の <strong>啓蒙 書</strong> で あった。<br>
「天 は 人 の 上 に 人 を 造らず」 で 始まる、平易 で 力 強い 文章。
</p>

<p>
それ に 対し、『文明論 之 概略』 (1875) は、より <strong>哲学 的 で 体系 的</strong> な 著作。<br>
「文明 と は 何 か」 「日本 の 進む べき 道 は 何 か」 を、思想 史 の 全体 像 を 描き ながら 論じる。
</p>

<p>
福澤 自身、本書 を 「自分 の 全 著作 の 中 で 最も 重要」 と 位置 づけて いた。<br>
丸山 真男 (本サイト kotoko3 で 既習) は 戦後、本書 の 注釈 書 『「文明論 之 概略」 を 読む』 (1986) を 書き、20 世紀 末 に 福澤 の 思想 を 再 評価 する 動き を 主導 した。
</p>

<h3>大学 入試 評論 と 福澤</h3>

<p>
福澤 諭吉 は、大学 入試 評論 で 最も 頻出 する 明治・大正 期 思想家 の 一人。<br>
本作 『文明論之概略』 から の 出題 は、特に 多い。
</p>

<p>
入試 で 試 される の は、福澤 の 「<strong>相対 性 思考</strong>」 「<strong>枠組み 反省</strong>」 を、読み 取れる か どう か。<br>
彼 の 文章 の <strong>論理 構造</strong> を 解き ほぐす 能力 は、大学 で の 学問 全般 (哲学・社会学・歴史学・経済学) で 必要 と なる。
</p>

<h3>21 世紀 へ の 接続</h3>

<p>
1875 年 の 福澤 の 問い は、150 年 後 の 私 たち に そのまま 突き 出されて いる。
</p>

<ul>
<li><strong>2020 年代 の 「文明」 と は?</strong> ─ AI? インターネット? 民主 主義? 持続 可能性?</li>
<li><strong>「西洋 中心 主義」 から の 解放 は どこ まで 進んだ か?</strong></li>
<li><strong>「議論 の 本位」 を 問わ ぬ ま ま、SNS で の 喧嘩 が 続いて いない か?</strong></li>
</ul>

<p>
福澤 が 教えて くれた の は、「<strong>議論 を 始める 前 に、足元 を 確かめ よ</strong>」 と いう 知恵 で ある。<br>
これ は 1875 年 で も、2026 年 で も、変わら ない 真実 で ある。
</p>
