<h3>第二巻 冒頭（試訳）</h3>

<blockquote>
<p>
夜明けに 目覚めたなら、自分自身に こう 語りかけよ ──<br>
「今日、私は、出しゃばりで、忘恩で、傲慢で、嘘つきで、嫉妬深く、自分勝手な 者 たち に 会う だろう」 と。
</p>

<p>
これら すべて の 悪徳 は、彼ら が <strong>善 と 悪 を 区別 する 知恵 を 欠いて いる</strong> ことから 生じて いる。<br>
だが、私 は、善 の 美しさ と、悪 の 醜さ と、そして、悪 を 為す 者 もまた 私 と 同じ <strong>人間 性</strong> を 持って いる こと ── これら を 理解 して いる。
</p>

<p>
ゆえに、私 は、その 者 たち に <strong>害される こと は ない</strong>。<br>
彼ら が 私 を 醜 い もの に 引き ずり込む こと は できない。<br>
そして 私 は、彼ら に <strong>怒り を 抱く こと も、彼ら を 憎む こと も ない</strong>。
</p>

<p>
なぜ なら、私 たち は、同じ 目的 の ため に 生まれ、共に 働く ため に 生まれた からで ある。<br>
ちょうど、両 足 の よう に、両 手 の よう に、上 下 の まぶた の よう に、上 下 の 歯 の よう に。<br>
互いに 妨げ合う こと は、自然 に 反する。<br>
他者 に 怒り、他者 を 厭う こと は、まさに この 自然 に 反する こと だ。
</p>

<p>
朝、起き上がる の が 辛い 時、こう 思え ──<br>
「私 は、人間 の 業 (わざ) を 為す ため に 起きる の だ。私 が この 世 に 来た 目的、そして 自然 が 私 を 造った 目的 を、今 から 為そう と して いる の だ。それ を 嘆く 必要 が ある か?」 と。
</p>

<p>
あるいは、布団 の 温かさ の 中 に とどまる 方 が 心地 よい か?<br>
「だが、私 は <strong>快楽</strong> の ため に 生まれた の か?<br>
それとも <strong>働き、為す こと</strong> の ため に 生まれた の か?」<br>
小さな 植物 を 見よ、雀 を 見よ、蟻 を 見よ、蜘蛛 を 見よ、蜂 を 見よ ── 彼ら は それぞれ の 仕事 を し、そう する こと で 世界 の 秩序 に 寄与 して いる。<br>
お前 は、人間 と して の 仕事 を 為す こと を 厭う の か?
</p>
</blockquote>

<h3>解説 ─ 戦場 の 天幕 で 書かれた 哲学</h3>

<p>
『自省録』 の これら の 言葉 は、ローマ 皇帝 が、贅沢 な 宮殿 で 書いた もの では ない。<br>
それ は、ドナウ 川 沿い の 野営 地 で、ゲルマン 民族 と の 戦争 の さなか、夜 の 天幕 で、油 ランプ の 灯 り の もと、皇帝 が <strong>誰 に 見せる つもり も なく</strong> 書き 残した、私的 な 覚書 で ある。
</p>

<p>
マルクス・アウレリウス (在位 161–180) は、世界 で 最も 強力 な 帝国 の 頂点 に 立ち ながら、生涯 の 大半 を 戦場 と 疫病 の 中 で 過ごした。
</p>

<ul>
<li>165–180 ─ アントニウス の 疫病 (推定 500 万人 以上 死亡)</li>
<li>166–180 ─ マルコマンニ 戦争 (ゲルマン 民族 と の 長期 戦争)</li>
<li>175 ─ 養子 アヴィディウス・カッシウス の 反乱</li>
<li>175 ─ 妻 ファウスティナ の 死</li>
</ul>

<p>
そんな 苦難 の 中、彼 は <strong>自分自身 を 励まし、整える ため</strong> に、毎日 の よう に メモ を 書き 続けた。<br>
それ が 1800 年 経って も 読まれる 古典 と なった。
</p>

<h3>ストア 派 哲学 の 核心</h3>

<p>
本 章 に は、ストア 派 哲学 の 根本 テーゼ が 凝縮 されて いる。
</p>

<ol>
<li>
  <strong>「悪徳 は 無知 から 来る」</strong> ─ 人 が 悪 を 為す の は、彼 が 「善 と 悪 を 区別 する 知恵」 を 欠いて いる から に すぎ ない。これ は ソクラテス 以来 の 思想。
</li>
<li>
  <strong>「他者 と の 共同 が 人間 の 本性」</strong> ─ 「両足、両手、まぶた、歯 の よう に」 互い に 協働 する ため に 生まれた。
</li>
<li>
  <strong>「快楽 で は なく 働き が 人間 の 目的」</strong> ─ 自然 が 私 たち を 造った 目的 は、それぞれ の 「仕事」 を 為す こと に ある。
</li>
<li>
  <strong>「外的 な もの に 動か される な」</strong> ─ 他者 が 私 を 「醜 い もの に 引き ずり 込む こと は できない」。私 の こころ の 状態 は、私 が 決める。
</li>
</ol>

<h3>「夜明け の 心 構え」 ─ 現代 心理 学 と の 呼応</h3>

<p>
「<strong>今日、私 は 出しゃばり で 嘘つき な 者 たち に 会う だろう</strong>」<br>
── これ は、悲観 主義 では ない。<strong>現実 主義 的 な 心 の 準備</strong> で ある。
</p>

<p>
20 世紀 の <strong>認知 行動 療法</strong> (CBT) の 祖 アーロン・ベック は、ストア 派 から 強く 影響 を 受けた と 明言 して いる。<br>
CBT の 核心 テーゼ は ──「<strong>出来事 が 直接 私 たち を 苦しめる の で は ない。出来事 に 対する 私 たち の 解釈 が、私 たち を 苦しめる</strong>」。
</p>

<p>
これ は マルクス・アウレリウス の 別 の 一節 の 直接 的 な 翻訳 と 見て よい ──<br>
「<strong>外的 な こと が 私 を 煩わせる なら、私 を 煩わせる の は その 外的 な こと 自体 で は なく、私 の それ に 対する 判断 で ある</strong>」(第八巻 47)。
</p>

<p>
2000 年 前 の ストア 派 の 智恵 が、現代 の メンタル ヘルス の 最前線 と 呼応 して いる の だ。
</p>

<h3>皇帝 の 謙虚</h3>

<p>
マルクス・アウレリウス は、世界 最大 の 権力 を 持つ 皇帝 で あった。<br>
だが 彼 は こう 書く ──「<strong>植物 を 見よ、雀 を 見よ、蟻 を 見よ</strong>」。
</p>

<p>
皇帝 で あり ながら、彼 は 自分 を 宇宙 の 中 の 一 点、一つ の 自然 の 部分 と して 捉え る。<br>
彼 は、雀 や 蟻 と 等しく、自然 が 与えた 「仕事」 を 為す ため に 生まれた と 考える。<br>
ここ に、ストア 派 の <strong>「コスモポリタニズム」</strong> (世界 市民 主義) の 原点 が ある。「皇帝 も 奴隷 も、宇宙 の 一部 と して 等しい」。
</p>

<h3>21 世紀 の ストア 派 復興</h3>

<p>
不思議 な こと に、2010 年 代 以降、世界 中 で <strong>ストア 派 の 復興</strong> (Modern Stoicism) が 起こって いる。
</p>

<ul>
<li>毎年 「Stoic Week」 が 開催 され、世界 中 で 何万 人 もの 参加 者 が、ストア 派 の 実践 を 試みて いる。</li>
<li>Ryan Holiday の 著作 『The Obstacle Is the Way』 など、現代 ストア 派 関連 書籍 が ベスト セラー に なって いる。</li>
<li>Tim Ferriss・Jeff Bezos・LeBron James など、現代 の 実業 家・アスリート が ストア 派 から の 影響 を 公言 して いる。</li>
<li>2020 年 の パンデミック で、不安 と 不確実 性 に 直面 した 多く の 人 が、マルクス・アウレリウス を 再 発見 した。</li>
</ul>

<p>
1800 年 前 に 戦場 の 天幕 で 書かれた 言葉 が、なぜ 今 改めて 響く の か?<br>
それ は、人間 が 直面 する 根本 的 な 苦悩 ─ 死、苦難、関係 の 困難、不確実 性 ─ が、時代 を 超えて 変わらない から だろう。<br>
そして、それ に 立ち向 かう 智恵 もまた、根本 的 に は 変わらない。
</p>

<h3>あなた の 「自省 録」 を 書こう</h3>

<p>
マルクス・アウレリウス は、出版 を 意図 せず、ただ 自分 自身 の ため に 書いた。<br>
私 たち も、彼 に ならって、毎晩 一行 で も、その 日 の 自分 を 振り返る 文章 を 書いて み ては どう か。
</p>

<p>
1800 年 後 の 君 が 残す 一文 も、ひょっと したら、その 100 年 後 の 誰か を 励ます かも しれない。
</p>
