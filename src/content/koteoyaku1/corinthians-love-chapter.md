<h3>本 文 (試 訳・本 サ イ ト 編 集 部)</h3>

<h4>第 1 節 ─ 愛 な き 信 仰 の 虚 し さ</h4>

<blockquote>
<p>
た と え 私 が、 人 間 の 言 葉 や 天 使 の 言 葉 を 話 し て も、 愛 が な け れ ば、 私 は 騒 が し い 銅 鑼 や、 や か ま し い 鐃 鈸 (に ょ う は つ) に 過 ぎ な い。
</p>

<p>
た と え 私 が、 預 言 の 賜 物 を 持 ち、 あ ら ゆ る 奥 義 と 知 識 を 知 り、 山 を 動 か す ほ ど の 信 仰 を 持 っ て い て も、 愛 が な け れ ば、 私 は 無 で あ る。
</p>

<p>
た と え 私 が、 全 財 産 を 貧 し い 人 々 に 施 し、 自 ら の 体 を 火 に 焼 か れ る た め に 渡 し て も、 愛 が な け れ ば、 私 に は 何 の 益 も な い。
</p>
</blockquote>

<h4>第 4–7 節 ─ 愛 と は 何 か</h4>

<blockquote>
<p>
<strong>愛 は 寛 容 に し て 慈 悲 あ り、<br>
愛 は 嫉 ま ず、<br>
愛 は 誇 ら ず、<br>
高 ぶ ら ず、<br>
礼 を 失 せ ず、<br>
自 ら の 利 を 求 め ず、<br>
怒 ら ず、<br>
人 の 悪 を 思 わ ず、<br>
不 義 を 喜 ば ず し て、 真 理 を 喜 び、<br>
す べ て を 包 み、 す べ て を 信 じ、 す べ て を 望 み、 す べ て を 忍 ぶ。</strong>
</p>
</blockquote>

<h4>第 8–13 節 ─ 永 遠 に 残 る も の</h4>

<blockquote>
<p>
愛 は 決 し て 滅 び な い。 預 言 は 廃 棄 さ れ、 異 言 は 止 み、 知 識 も 廃 棄 さ れ る。 私 た ち が 知 る こ と は 部 分 的 で あ り、 預 言 す る こ と も 部 分 的 で あ る。 し か し、 完 全 な も の が 来 る 時、 部 分 的 な も の は 廃 棄 さ れ る。
</p>

<p>
私 が 幼 子 で あ っ た 時 は、 幼 子 の よ う に 語 り、 幼 子 の よ う に 考 え、 幼 子 の よ う に 判 断 し て い た。 し か し、 大 人 と な っ た 時、 幼 子 の こ と を 捨 て た。
</p>

<p>
今 私 た ち は、 鏡 を 通 し て お ぼ ろ に 見 て い る。 し か し そ の 時 に は、 顔 と 顔 と を 合 わ せ て 見 る。 今 私 が 知 る の は 部 分 的 だ が、 そ の 時 に は、 私 が 知 ら れ て い る よ う に、 私 自 身 も 知 る。
</p>

<p>
<strong>こ う し て、 信 仰 と、 希 望 と、 愛 と、 こ の 三 つ は い つ ま で も 残 る。 そ の 中 で 最 も 大 き い も の は、 愛 で あ る。</strong>
</p>
</blockquote>

<h3>解 説 ─ パ ウ ロ と コ リ ン ト 教 会</h3>

<p>
本 章 は、 西 暦 55 年 頃、 使 徒 パ ウ ロ が ギ リ シ ア の 港 町 <strong>コ リ ン ト</strong> の 教 会 宛 て に 書 い た 手 紙 の 一 部 で あ る。
</p>

<p>
パ ウ ロ は 西 暦 50–52 年 頃、 コ リ ン ト に 1 年 半 滞 在 し、 こ こ で 教 会 を 設 立 し た。<br>
そ の 後、 別 の 地 (お そ ら く エ フ ェ ソ ス) に 移 っ た 彼 の も と に、 コ リ ン ト 教 会 内 で 起 こ っ て い る 問 題 の 報 告 が 届 い た。
</p>

<ul>
<li>派 閥 争 い ─ 「私 は パ ウ ロ 派」 「私 は ア ポ ロ 派」 と 信 徒 た ち が 分 か れ て い る</li>
<li>道 徳 的 問 題 ─ 性 的 不 品 行 が 教 会 内 で 起 き て い る</li>
<li>礼 拝 の 混 乱 ─ 「異 言 を 話 す」 こ と が 特 別 視 さ れ、 信 徒 た ち の 競 い 合 い が 起 き て い る</li>
<li>聖 餐 (せ い さ ん) の 不 平 等 ─ 共 同 食 事 で 富 め る 者 と 貧 し き 者 が 区 別 さ れ て い る</li>
</ul>

<p>
こ れ ら の 問 題 を 受 け て、 パ ウ ロ は 手 紙 を 書 い た。 そ の 手 紙 の 第 13 章 が、 こ の 「<strong>愛 の 章</strong>」 で あ る。
</p>

<h3>「愛 が な け れ ば 無 で あ る」 ─ 反 転 の 修 辞</h3>

<p>
本 章 第 1–3 節 の 構 造 は、 極 め て 印 象 的 で あ る。
</p>

<p>
パ ウ ロ は、 当 時 の コ リ ン ト 教 会 で 「素 晴 ら し い」 と さ れ た こ と を、 一 つ ず つ 並 べ る:
</p>

<ul>
<li>言 葉 の 賜 物 ─ 「人 間 の 言 葉、 天 使 の 言 葉」</li>
<li>知 識 の 賜 物 ─ 「あ ら ゆ る 奥 義 と 知 識」</li>
<li>信 仰 の 賜 物 ─ 「山 を 動 か す ほ ど の 信 仰」</li>
<li>慈 善 の 賜 物 ─ 「全 財 産 を 貧 し い 人 々 に」</li>
<li>殉 教 の 賜 物 ─ 「自 ら の 体 を 火 に」</li>
</ul>

<p>
そ し て 一 つ 一 つ に、 こ う 付 け 加 え る ─ <strong>「し か し、 愛 が な け れ ば、 私 は 無 で あ る」</strong>。
</p>

<p>
当 時 の コ リ ン ト 教 会 で は、 こ れ ら の 「賜 物」 を 持 つ こ と が、 信 徒 の 価 値 を 決 め る と 思 わ れ て い た。<br>
パ ウ ロ は そ れ を、 真 っ 向 か ら 否 定 し た。「あ な た が 何 を 持 ち、 何 を し て い て も、 愛 が な け れ ば、 す べ て は 虚 し い」 と。
</p>

<h3>ギ リ シ ア 語 の 「愛」 ─ 4 つ の 言 葉</h3>

<p>
パ ウ ロ が 本 章 で 用 い た 「愛」 と い う 語 は、 ギ リ シ ア 語 の <strong>ア ガ ペ ー (agapē)</strong> で あ る。
</p>

<p>
古 代 ギ リ シ ア 語 に は、 日 本 語 の 「愛」 一 語 で は 区 別 で き な い、 4 つ の 「愛」 が あ っ た:

こ れ こ そ が、 キ リ ス ト 教 の 核 心 と な る 「愛」 で あ り、 ま た 西 洋 倫 理 学 の 根 本 概 念 と も な っ た。
</p>

<h3>「信 ・ 望 ・ 愛」 ─ 三 神 徳</h3>

<p>
本 章 最 後 の 一 句 ─ 「<strong>信 仰 と、 希 望 と、 愛 と、 こ の 三 つ は い つ ま で も 残 る。 そ の 中 で 最 も 大 き い も の は、 愛 で あ る</strong>」 ─ は、 西 洋 神 学 で 「<strong>三 神 徳</strong>」 と 呼 ば れ る。
</p>

<ul>
<li><strong>信 仰 (faith / pistis)</strong> ─ 神 を 信 じ る こ と</li>
<li><strong>希 望 (hope / elpis)</strong> ─ 未 来 を 信 じ る こ と</li>
<li><strong>愛 (love / agapē)</strong> ─ 他 者 を 愛 す る こ と</li>
</ul>

<p>
こ の 三 つ は、 中 世 神 学 (ト マ ス・ア ク ィ ナ ス ら) で キ リ ス ト 教 倫 理 の 三 大 徳 と さ れ、 古 代 ギ リ シ ア の 「知 ・ 勇 ・ 節 制 ・ 正 義」 の 四 元 徳 と 並 ぶ 西 洋 倫 理 の 柱 と な っ た。
</p>

<h3>2000 年 を 超 え て 響 く 愛</h3>

<p>
パ ウ ロ が こ の 手 紙 を 書 い て か ら、 約 1970 年。
</p>

<p>
そ の 間、 こ の 「愛 の 章」 は ─<br>
キ リ ス ト 教 の 礼 拝 で 朗 読 さ れ、<br>
中 世 神 学 の 議 論 の 基 礎 と な り、<br>
ダ ン テ 『神 曲』 や ミ ル ト ン 『失 楽 園』 を 形 作 り、<br>
近 代 の 結 婚 式 ・ 葬 式 で 何 万 回 と 朗 読 さ れ、<br>
キ ン グ 牧 師 の 公 民 権 演 説 で 引 用 さ れ、<br>
マ ザ ー ・ テ レ サ の 慈 善 活 動 の 基 盤 と な り、<br>
今 も 世 界 中 で、 愛 と は 何 か を 問 い 続 け る 人 々 の 心 に 響 い て い る。
</p>

<p>
プ ラ ト ン (洞 窟 の 比 喩)、 マ ル ク ス ・ ア ウ レ リ ウ ス (自 省 録)、 そ し て パ ウ ロ (愛 の 章) ─ こ の 三 者 は、 西 洋 古 典 の 「人 間 と は 何 か」 へ の 答 え の、 三 つ の 大 き な 流 れ で あ る。
</p>

<p>
プ ラ ト ン は 「真 理 を 知 れ」 と 言 い、<br>
マ ル ク ス ・ ア ウ レ リ ウ ス は 「自 己 を 整 え よ」 と 言 い、<br>
パ ウ ロ は 「他 者 を 愛 せ よ」 と 言 う。
</p>

<p>
こ の 三 つ が 互 い に 絡 み 合 い、 西 洋 文 明 の 根 を 形 作 っ て き た の で あ る。
</p>
