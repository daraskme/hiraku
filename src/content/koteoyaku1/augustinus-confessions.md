<h3>本 文 (試 訳)</h3>

<h4>『告 白』 第 八 巻 第 十 二 章 「庭 園 の 回 心」</h4>

<blockquote>
<p>
私 は、 友 ア リ ピ ウ ス と と も に ミ ラ ノ の 家 の 庭 園 に い た。 自 分 の 過 去 の 罪 と、 そ れ で も キ リ ス ト 教 に 完 全 に 帰 依 で き な い 自 分 の 弱 さ を、 苦 し み 嘆 い て い た。
</p>

<p>
深 い 涙 が 流 れ た。 私 は 涙 を 友 に 隠 す た め に、 ア リ ピ ウ ス の 元 か ら 離 れ、 庭 園 の 奥 に あ る い ち じ く の 木 の 下 に 身 を 投 げ た。
</p>

<p>
そ し て、 涙 な が ら に 神 に 訴 え た:<br>
「主 よ、 い つ ま で で す か。 い つ ま で 私 の 不 信 を 引 き 延 ば さ れ る の で す か。 ど う か、 こ の 心 を 完 全 に 変 え て く だ さ い。 ど う か、 今 こ の 時 に」 と。
</p>

<p>
こ の よ う に 嘆 き、 涙 を 流 し て い た 時 ─ 突 然、 隣 家 か ら 子 ど も の 歌 声 の よ う な も の が 聞 こ え た。<br>
そ の 声 は、 こ う 繰 り 返 し て い た:
</p>

<blockquote class="song">
<strong>「ト レ・レ ゲ、 ト レ・レ ゲ」</strong> (Tolle, lege; tolle, lege)<br>
─ 「取 れ、 読 め。 取 れ、 読 め」 ─
</blockquote>

<p>
私 は 突 然 涙 を 止 め、 立 ち 上 が っ た。<br>
こ の 声 が 子 ど も た ち の 遊 び 歌 か ど う か は、 思 い 出 せ な か っ た。 だ が、 何 か の 神 託 の よ う に 思 え た。
</p>

<p>
私 は 急 い で ア リ ピ ウ ス が 座 っ て い た 場 所 に 戻 っ た。 そ こ に は、 私 が 読 ん で い た 「使 徒 パ ウ ロ の 書 簡 集」 が 開 か れ て 置 か れ て い た。
</p>

<p>
私 は、 そ の 書 物 を 取 り 上 げ、 開 き、 最 初 に 目 に 入 っ た 箇 所 を 読 ん だ。<br>
そ こ に は こ う 書 か れ て い た ─
</p>

<blockquote>
「酒 宴 や 酔 い と か、 不 道 徳 や 放 縦 と か、 争 い と 妬 み と か、 そ う い う こ と か ら 離 れ て、 主 イ エ ス ・ キ リ ス ト を 着 な さ い。 肉 の 欲 望 を 満 た す た め に、 思 い を 巡 ら す こ と を し て は な ら な い」<br>
─ (ロ ー マ の 信 徒 へ の 手 紙 13 章 13–14 節)
</blockquote>

<p>
こ れ 以 上、 読 む 必 要 は な か っ た。 こ の 言 葉 を 読 ん だ 瞬 間 ─ <strong>突 然、 私 の 心 か ら 一 切 の 疑 い の 闇 が 取 り 払 わ れ た</strong>。<br>
私 の 罪 へ の 執 着 が、 一 気 に 消 え 去 っ た。 そ し て、 私 の 全 存 在 が、 神 の 光 で 満 た さ れ る の を 感 じ た。
</p>

<p>
私 は 書 物 を 閉 じ、 指 で そ の 箇 所 を 押 さ え た ま ま、 ア リ ピ ウ ス に 語 っ た。<br>
ア リ ピ ウ ス も ま た、 自 ら 書 物 を 取 り 上 げ、 私 が 読 ん だ 箇 所 の 後 を 読 み、 そ の 場 で 私 と と も に 信 仰 を 受 け 入 れ た。
</p>

<p>
そ し て 私 た ち は、 家 に 戻 っ て 母 モ ニ カ に 全 て を 話 し た。<br>
モ ニ カ は、 私 の た め に 三 十 年 以 上 涙 を 流 し て 祈 り 続 け て き た の だ っ た。 こ の 知 ら せ を 聞 い た 母 は、 喜 び に あ ふ れ、 神 を 讃 美 し た。
</p>

<p>
こ う し て、 386 年 8 月 ─ 私 は、 32 歳 で キ リ ス ト 教 に 完 全 に 回 心 し た の で あ る。<br>
そ し て こ れ こ そ が、 私 の 人 生 の 真 の 始 ま り で あ っ た。
</p>
</blockquote>

<h3>解 説 ─ 32 歳 の 回 心</h3>

<p>
ア ウ レ リ ウ ス ・ ア ウ グ ス テ ィ ヌ ス (354–430) は、 古 代 末 期 の キ リ ス ト 教 思 想 を 確 立 し、 後 の 西 洋 中 世 神 学 ・ 哲 学 の 礎 を 築 い た 巨 人 で あ る。
</p>

<p>
彼 の 生 涯 は、 二 つ に 分 け ら れ る:
</p>

<ul>
<li><strong>32 歳 ま で</strong> ─ マ ニ 教 ・ 占 星 術 ・ 新 プ ラ ト ン 主 義 を 学 び、 修 辞 学 の 教 授 と し て 名 声 を 求 め た 知 的 探 求 の 時 期</li>
<li><strong>32 歳 以 降</strong> ─ キ リ ス ト 教 に 回 心 し、 司 教 と し て 神 学 著 作 を 残 し た 時 期</li>
</ul>

<p>
こ の 二 つ を 分 け る 決 定 的 瞬 間 が、 386 年 8 月 の <strong>「庭 園 の 回 心」</strong> で あ っ た。
</p>

<h3>「取 れ、 読 め」 (Tolle, lege)</h3>

<p>
回 心 の 直 接 の き っ か け と な っ た の は、 庭 園 で 聞 こ え た 「<strong>Tolle, lege</strong>」 (ト レ、 レ ゲ) と い う 声。<br>
ラ テ ン 語 で「取 れ、 読 め」 の 意。
</p>

<p>
こ の 声 が、 本 当 に 子 ど も た ち の 遊 び 歌 だ っ た か、 あ る い は 神 か ら の 啓 示 だ っ た か ─ ア ウ グ ス テ ィ ヌ ス 自 身 は 「私 に は 思 い 出 せ な い」 と 書 い て い る。
</p>

<p>
だ が、 何 で あ れ、 こ の 声 が ア ウ グ ス テ ィ ヌ ス を 動 か し、 聖 書 を 開 か せ、 そ し て あ の 「ロ ー マ 書 13 章 13–14 節」 を 読 ま せ た。
</p>

<p>
そ の 一 節 を 読 ん だ 瞬 間 ─ <strong>「私 の 全 存 在 が、 神 の 光 で 満 た さ れ た」</strong> と 彼 は 書 く。<br>
こ れ こ そ が、 西 洋 文 学 史 で 最 も 有 名 な 「回 心 (か い し ん)」 の 瞬 間 で あ る。
</p>

<h3>母 モ ニ カ の 30 年 の 祈 り</h3>

<p>
ア ウ グ ス テ ィ ヌ ス の 母 <strong>モ ニ カ</strong> (332–387) は、 敬 虔 な キ リ ス ト 教 徒 だ っ た。<br>
だ が 父 パ ト リ キ ウ ス は 非 キ リ ス ト 教 徒。 ア ウ グ ス テ ィ ヌ ス 自 身 も 若 い 頃 は キ リ ス ト 教 か ら 離 れ て い た。
</p>

<p>
モ ニ カ は、 息 子 の キ リ ス ト 教 へ の 回 心 を、 <strong>30 年 以 上 に わ た っ て 涙 で 祈 り 続 け た</strong> と 伝 え ら れ る。<br>
そ し て、 386 年 の 庭 園 の 回 心 の 翌 年 1387 年、 モ ニ カ は 「私 の 祈 り は す べ て 叶 え ら れ た」 と 言 い 残 し、 オ ス テ ィ ア で 没 し た。
</p>

<p>
モ ニ カ は 後 に 聖 人 と さ れ、 ア メ リ カ ・ カ リ フ ォ ル ニ ア 州 の 都 市 <strong>サ ン タ ・ モ ニ カ</strong> (Santa Monica) の 名 前 の 由 来 と な っ た。
</p>

<h3>『告 白』 と い う 自 伝 文 学</h3>

<p>
ア ウ グ ス テ ィ ヌ ス は、 回 心 か ら 約 10 年 後、 397–400 年 頃 に『告 白』(Confessiones) を 書 い た。<br>
全 13 巻 の 自 伝 的 著 作 で あ る。
</p>

<p>
こ の『告 白』 の 革 命 性 は、 「<strong>自 分 の 過 去 の 罪 と 内 面 を 詳 細 に 神 に 告 白 す る</strong>」 と い う 形 式 に あ っ た。
</p>

<ul>
<li>幼 児 期 の 嫉 妬</li>
<li>少 年 時 代 の 盗 み</li>
<li>青 年 期 の 性 的 放 縦</li>
<li>マ ニ 教 へ の 傾 倒</li>
<li>そ し て 回 心 へ の 長 い 苦 闘</li>
</ul>

<p>
こ れ ら 全 て を、 ア ウ グ ス テ ィ ヌ ス は 包 み 隠 さ ず 書 い た。<br>
そ の 上 で、 「そ れ で も 神 は 私 を 救 っ て く だ さ っ た」 と 結 ぶ。
</p>

<p>
こ の 「<strong>自 分 を 赤 裸 々 に 語 る</strong>」 形 式 は、 西 洋 自 伝 文 学 の 出 発 点 と な っ た。
</p>

<h3>後 世 へ の 影 響 ─ ル ソ ー『告 白』 ま で</h3>

<p>
ア ウ グ ス テ ィ ヌ ス『告 白』 の 後 ─ 1400 年 を 経 て、 同 じ タ イ ト ル の 名 著 が 書 か れ た。
</p>

<p>
ジ ャ ン ・ ジ ャ ッ ク ・ ル ソ ー『<strong>告 白</strong>』 (Les Confessions, 1782 出 版)。<br>
ル ソ ー は こ の 著 作 の 序 で、 ア ウ グ ス テ ィ ヌ ス『告 白』 を 意 識 し て こ う 書 い た:「私 は、 一 度 も 試 み ら れ た こ と の な い、 未 来 永 劫 模 倣 す る 人 も い な い だ ろ う 一 つ の 仕 事 を 企 て る」 と。
</p>

<p>
ア ウ グ ス テ ィ ヌ ス → ル ソ ー → ト ル ス ト イ → 現 代 の 自 伝 文 学 ─ そ の 出 発 点 が、 1600 年 前 の 「庭 園 の 回 心」 だ っ た の で あ る。
</p>
