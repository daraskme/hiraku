<h3>神 社 の 鳥 居 を く ぐ る と、 数 学 が 待 っ て い た</h3>

<p>
江 戸 期、 各 地 の 神 社・仏 閣 を 訪 れ る と、 ど こ で も 不 思 議 な 額 が 飾 ら れ て い た。 美 し い 円 と
三 角 形 の 図、 美 し い 漢 字 で 書 か れ た 文 章 ─ そ れ は 数 学 の 問 題 と 解 答 だ っ た。
</p>

<p>
こ の 額 を「算 額 (さ ん が く)」 と 呼 ぶ。 江 戸 期 か ら 明 治 期 に か け て、 武 士・農 民・商 人・
子 ど も ま で が、 数 学 を 解 い て は 神 様 に 奉 納 し た の だ。
</p>

<h3>な ぜ 数 学 を 神 様 に 奉 納 す る の か</h3>

<p>
日 本 の 神 道 で は、 「優 れ た こ と を 達 成 し た 時、 神 様 に 感 謝 し て 奉 納 す る」 文 化 が あ る。
剣 の 達 人 が 神 社 に 刀 を 奉 納 す る よ う に、 数 学 が 解 け た 人 は そ の 喜 び を 木 札 に 描 い て
神 様 に 捧 げ た。
</p>

<p>
ま た、 神 社 は 公 共 の 場 所 で あ り、 多 く の 人 が 訪 れ る。 算 額 を 奉 納 す る こ と は、 「私 は こ れ
を 解 き ま し た。 皆 さ ん、 検 証 し て く だ さ い」 と い う 発 表 の 場 で も あ っ た。
</p>

<h3>誰 が 算 額 を 奉 納 し た の か</h3>

<p>
算 額 を 奉 納 し た の は、 専 門 の 数 学 者 だ け で は な か っ た ─
</p>

<ul>
  <li><strong>武 士</strong> ─ 関 流 ・ 最 上 流 等 の 学 派 に 所 属</li>
  <li><strong>農 民 ・ 商 人</strong> ─ 寺 子 屋 で 算 術 を 学 ん だ 大 人 達</li>
  <li><strong>僧 侶</strong> ─ 寺 院 教 育 で 数 学 を 学 ん だ 者</li>
  <li><strong>子 ど も</strong> ─ 寺 子 屋 で 数 学 を 学 ん だ 児 童 ・ 少 年</li>
  <li><strong>女 性</strong> ─ 数 は 少 な い が、 算 額 を 奉 納 し た 女 性 も い た</li>
</ul>

<p>
こ れ は 世 界 史 的 に 異 例 で あ る。 同 時 代 の ヨ ー ロ ッ パ で は、 数 学 は 大 学 ・ 王 立 学 会 等 の
エ リ ー ト の も の だ っ た。 江 戸 期 の 日 本 で は、 数 学 が 庶 民 の 文 化 だ っ た。
</p>

<h3>典 型 的 な 算 額 問 題 ─ 円 と 三 角 形</h3>

<p>
算 額 で 最 も 人 気 の 問 題 タ イ プ は、 「円 内 に 円 を 詰 め る」 (Circle Packing) だ っ た。
</p>

<p><strong>例 題</strong> ─ 「半 径 1 の 大 円 の 中 に、 直 径 が 等 し い 3 つ の 円 を 互 い に 接 す る よ う に 詰 め る。
小 円 の 半 径 を 求 め よ」。
</p>

<p>
解 答 ─ 中 心 同 士 を 結 ぶ と 正 三 角 形 が で き、 三 平 方 を 使 う と 小 円 の 半 径 = (2 √3 − 3) / 3
≈ 0.155。 美 し い 図 と 共 に 神 様 に 奉 納。
</p>

<h3>遺 題 継 承 ─ 数 学 が 進 化 す る 動 力</h3>

<p>
和 算 の 慣 習 で「遺 題 継 承 (い だ い け い し ょ う)」 と い う も の が あ っ た。
本 の 末 尾 に「未 解 決 問 題」 を 残 し て お き、 次 の 著 者 が そ れ を 解 い て 新 し い 本 を 出 す。
</p>

<p>
こ の 仕 組 み で、 関 孝 和 →『発 微 算 法』 → 建 部 賢 弘 → … と 数 学 が 集 団 で 発 達 し た。
現 代 の 学 術 論 文 ・ オ ー プ ン サ イ エ ン ス と 似 た 仕 組 み が、 江 戸 期 に 既 に あ っ た の だ。
</p>

<h3>現 在 残 る 約 800 面 の 算 額</h3>

<p>
明 治 期 に 西 洋 数 学 が 導 入 さ れ、 和 算・算 額 の 文 化 は 急 速 に 衰 退 し た。 し か し、 全 国 の
神 社・仏 閣 に 約 800 面 の 算 額 が 残 さ れ、 デ ジ タ ル ア ー カ イ ブ さ れ て い る。
</p>

<p>
特 に 算 額 が 多 い 県 ─ 福 島・宮 城・岩 手・福 井・三 重・奈 良 等。 君 の 住 む 地 域 の 神 社 に も、
算 額 が 残 っ て い る か も し れ な い。 一 度 探 し て み よ う。
</p>

<h3>結 び ─ 数 学 を 楽 し む 心</h3>

<p>
「数 学 は 難 し い、 嫌 い」 と 言 う 君 へ ─ 江 戸 期 の 農 民・商 人・子 ど も た ち は、 数 学 を 解 い て
神 様 に 奉 納 す る ほ ど 楽 し ん で い た。 数 学 は、 元 来 楽 し い も の だ っ た の だ。
</p>

<p>
教 科 書 の 問 題 を 解 く 時、 「神 社 に 奉 納 す る つ も り で、 美 し く 解 こ う」 と 思 う と、 数 学 が
急 に 別 の 顔 を 見 せ て く れ る。
</p>
