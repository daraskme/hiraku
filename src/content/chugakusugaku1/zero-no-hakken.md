## ゼ ロ の 発 見 ─ 「何 も な い」 を 数 に す る

### 1. 「何 も な い」 を、 ど う 書 く か

想 像 し て み よ う。 数 字 は あ る が、 「ゼ ロ」 と い う 記 号 が な い 世 界 を。

「り ん ご が 3 個 あ る」 ─ こ れ は「3」 と 書 け る。 「り ん ご が 1 個 も な い」 ─ こ れ は、 ど う 書 く の か?

古 代 の 多 く の 文 明 で は、 「何 も な い」 は そ も そ も 書 か な か っ た。 「な い も の は、 な い」。 書 く 必 要 が な い と 考 え た の で あ る。

し か し、 こ こ に 大 き な 問 題 が 潜 ん で い た。

***

### 2. 位 取 り の 問 題

数 を「位 取 り」 で 書 く と き、 ゼ ロ が な い と 困 る こ と が 起 こ る。

例 え ば、 「二 百 五 (205)」 を 書 き た い と し よ う。 こ れ は「百 の 位 が 2、 十 の 位 が な い、 一 の 位 が 5」 で あ る。

も し「十 の 位 が な い」 こ と を 示 す 記 号 が な け れ ば、 「2 5」 と 書 く こ と に な る。 し か し「2 5」 は「二 十 五 (25)」 と 区 別 が つ か な い。

そ こ で、 「十 の 位 が 空 で あ る」 こ と を 示 す 記 号 が 必 要 に な っ た。 そ れ が、 ゼ ロ「0」 で あ る。 「205」 ─ 真 ん 中 の 0 が、「十 の 位 は 空 で す」 と 教 え て く れ る。

***

### 3. 「空 位」 か ら「数」 へ

実 は、 こ の「空 位 を 示 す 記 号」 と し て の ゼ ロ は、 古 代 バ ビ ロ ニ ア (紀 元 前 3 世 紀 頃) や 中 米 の マ ヤ 文 明 に も 存 在 し た。 「こ こ の 位 は 空 で す」 と い う 印 で あ る。

し か し、 こ れ ら は「印」 に と ど ま っ て い た。 ゼ ロ を「計 算 の 対 象 と な る 数 そ の も の」 と し て 扱 う ─ つ ま り「5 - 5 = 0」「7 + 0 = 7」「3 × 0 = 0」 の よ う に、 ゼ ロ を 一 つ の 数 と し て 足 し 引 き 掛 け す る ─ と い う 段 階 に は、 ど の 文 明 も 達 し て い な か っ た。

「何 も な い」 を、 一 つ の「数」 と し て 認 め る。 こ れ は、 想 像 以 上 に 難 し い 飛 躍 で あ っ た。

***

### 4. 628 年、 イ ン ド の ブ ラ フ マ グ プ タ

そ の 飛 躍 を 成 し 遂 げ た の が、 古 代 イ ン ド で あ っ た。

7 世 紀 の 数 学 者 ブ ラ フ マ グ プ タ (598-668) は、 628 年 に 著 し た『ブ ラ フ マ・ス プ タ・シ ッ ダ ー ン タ』 で、 ゼ ロ の 計 算 規 則 を 世 界 で 初 め て 体 系 的 に 記 述 し た。

- 「あ る 数 か ら、 そ れ 自 身 を 引 く と、 ゼ ロ に な る」(a - a = 0)
- 「数 に ゼ ロ を 足 し て も、 そ の 数 は 変 わ ら な い」(a + 0 = a)
- 「ど ん な 数 に ゼ ロ を か け て も、 ゼ ロ に な る」(a × 0 = 0)

こ れ ら は、 現 代 の 私 た ち が 当 た り 前 だ と 思 っ て い る ゼ ロ の 規 則 そ の も の で あ る。 ブ ラ フ マ グ プ タ は、 ゼ ロ を「空 位 の 印」 か ら「計 算 で き る 数」 へ と 昇 格 さ せ た の だ。

(同 じ 書 で、 ブ ラ フ マ グ プ タ は 負 の 数 の 規 則 も 確 立 し て い る。 → 「マ イ ナ ス の 数 の 発 見」 参 照。)

***

### 5. ア ラ ビ ア を 経 て 世 界 へ

イ ン ド の ゼ ロ と 位 取 り 記 数 法 は、 9 世 紀 に イ ス ラ ム 世 界 に 伝 わ り、 ア ル・フ ワ ー リ ズ ミ ー ら に よ っ て「ア ラ ビ ア 数 字」 と し て 発 展 し た。

さ ら に、 12～13 世 紀 に、 フ ィ ボ ナ ッ チ の『Liber Abaci』 (1202) 等 を 通 じ て、 ヨ ー ロ ッ パ に 伝 わ っ た。

「ゼ ロ」 と い う 言 葉 の 語 源 を た ど る と、 そ の 旅 が 見 え て く る。 ア ラ ビ ア 語 の「ス ィ フ ル (sifr、『空』 の 意)」 → ラ テ ン 語「ゼ フ ィ ル ム」 → 英 語「zero」。 ち な み に、 同 じ「sifr」 か ら、 「暗 号 (cipher)」 と い う 言 葉 も 生 ま れ た。

***

### 6. ゼ ロ が 支 え る 現 代

ゼ ロ が「計 算 で き る 数」 と し て 確 立 さ れ た こ と は、 1000 年 以 上 後 の 現 代 を 支 え て い る。

現 代 の コ ン ピ ュ ー タ は、 全 て の 情 報 を「0」 と「1」 の 二 進 数 で 表 現 す る。 ス マ ホ も、 イ ン タ ー ネ ッ ト も、 AI も、 す べ て「0 と 1」 の 上 で 動 い て い る。

「何 も な い」 を 数 に す る ─ こ の 古 代 イ ン ド の 思 想 が な け れ ば、 デ ジ タ ル 技 術 は 生 ま れ な か っ た。

中 学 1 年 で「0」 を 当 た り 前 の よ う に 使 う 君 は、 実 は、 人 類 が 数 千 年 か け て た ど り 着 い た、 最 も 深 い 数 学 的 発 見 の 一 つ を、 手 に し て い る の で あ る。

「何 も な い」 こ と に も、 名 前 と 居 場 所 を 与 え る。 ゼ ロ と は、 そ う い う 思 想 の 結 晶 で あ る。
