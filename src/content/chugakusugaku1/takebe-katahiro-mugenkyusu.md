<h3>建 部 賢 弘 ─ 江 戸 数 学 の 頂 点</h3>

<p>
関 孝 和 (1642 頃 –1708) ─ 江 戸 数 学 史 最 大 の 天 才 (本 サ イ ト で 既 習)。<br>
そ の 弟 子 の 中 で、 師 を も 凌 ぐ 仕 事 を し た の が <strong>建 部 賢 弘</strong> (た け べ か た ひ ろ、 1664–1739) で あ る。
</p>

<p>
彼 の 業 績:
</p>

<ul>
<li><strong>円 周 率 を 小 数 点 以 下 42 桁 ま で 計 算</strong> (1722) ─ 当 時 の 世 界 最 高 水 準</li>
<li><strong>無 限 級 数</strong> を 用 い た 計 算 法 を 確 立</li>
<li><strong>ベ ル ヌ ー イ 数</strong> を ヨ ー ロ ッ パ よ り 早 く 発 見 し た と さ れ る</li>
<li><strong>江 戸 幕 府 の 数 学 役</strong> と し て 仕 え る</li>
</ul>

<h3>『綴 術 算 経』 ─ 和 算 の 集 大 成</h3>

<p>
1722 年、 建 部 が 58 歳 の と き 完 成 さ せ た の が 『<strong>綴 術 算 経</strong>』 (て つ じ ゅ つ さ ん け い)。
</p>

<p>
タ イ ト ル の 意 味 を 分 解 す る と:
</p>

<ul>
<li>「<strong>綴 (て つ)</strong>」 ─ 連 な る、 連 ね る</li>
<li>「<strong>術 (じ ゅ つ)</strong>」 ─ 方 法、 計 算 法</li>
<li>「<strong>算 経 (さ ん け い)</strong>」 ─ 数 学 の 書</li>
</ul>

<p>
合 わ せ て「<strong>連 な る (= 無 限 に 続 く) 計 算 法 の 書</strong>」 ─ つ ま り 「無 限 級 数 の 数 学 書」 で あ る。
</p>

<h3>円 周 率 42 桁</h3>

<p>
本 書 で 建 部 は、 円 周 率 (π) を こ の よ う に 計 算 し た:
</p>

<blockquote>
π = 3.14159 26535 89793 23846 26433 83279 50288 4…
</blockquote>

<p>
小 数 点 以 下 <strong>42 桁</strong> ま で 正 確 で あ る。
</p>

<p>
当 時 の ヨ ー ロ ッ パ で の 円 周 率 計 算 と 比 べ て み よ う:
</p>

<p>
1722 年 時 点 で、 建 部 の 42 桁 は <strong>世 界 最 高 水 準</strong> だ っ た。
</p>

<h3>「無 限 級 数」 と い う 道 具</h3>

<p>
建 部 が こ の 高 精 度 を 達 成 で き た の は、 「<strong>無 限 級 数</strong>」 と い う 道 具 を 使 っ た か ら で あ る。
</p>

<p>
無 限 級 数 と は、 「無 限 に 続 く 数 を 足 し 続 け る」 計 算。<br>
不 思 議 な こ と に、 結 果 が 確 定 す る 場 合 が あ る:
</p>

<blockquote>
1 + 1/2 + 1/4 + 1/8 + 1/16 + 1/32 + … = 2
</blockquote>

<p>
こ の 「無 限 に 足 し 続 け る と 2 に 限 り な く 近 づ く」 と い う 計 算 を、 円 周 率 や 平 方 根 な ど の 計 算 に 応 用 す る。
</p>

<p>
ヨ ー ロ ッ パ で は、 ニ ュ ー ト ン (1666 頃) や ラ イ プ ニ ッ ツ (1670s) が 微 積 分 学 と と も に 無 限 級 数 を 体 系 化。<br>
建 部 賢 弘 (1722) は、 鎖 国 の 日 本 で、 こ れ を <strong>独 立 に 達 成</strong> し た。
</p>

<h3>関 孝 和・建 部 賢 弘 の 系 譜</h3>

<p>
関 孝 和 (1642 頃 –1708) は、 弟 子 を 多 数 育 て た。 そ の 中 で 最 も 優 れ た 二 人 が:
</p>

<ul>
<li><strong>建 部 賢 弘</strong> (1664–1739) ─ 円 周 率 42 桁、 無 限 級 数</li>
<li><strong>建 部 賢 明</strong> (1661–1716) ─ 賢 弘 の 兄、 同 じ く 数 学 者</li>
</ul>

<p>
こ の 兄 弟 と 関 孝 和 で、 「<strong>関 派 (せ き は)</strong>」 と 呼 ば れ る 江 戸 和 算 の 中 心 学 派 が 形 成 さ れ た。
</p>

<h3>江 戸 数 学 が 残 し た も の</h3>

<p>
関 孝 和 ・ 建 部 賢 弘 の 仕 事 は、 江 戸 期 を 通 し て 日 本 に 受 け 継 が れ た。<br>
明 治 維 新 (1868) で 西 洋 数 学 が 入 っ て き た と き、 日 本 人 が 急 速 に そ れ を 吸 収 で き た の は、 江 戸 和 算 で 培 わ れ た「数 学 の 土 壌」 が あ っ た か ら と も 言 わ れ る。
</p>

<p>
建 部 賢 弘 の 42 桁 の 円 周 率、 そ し て 無 限 級 数 の 概 念 ─ こ れ ら は、 鎖 国 の 日 本 で 育 っ た、 世 界 級 の 知 的 業 績 で あ る。
</p>

<p>
中 1 で 君 が 学 ぶ「数 と 式」 の 背 後 に、 こ う し た 巨 人 た ち の 営 み が 連 な っ て い る こ と を、 ぜ ひ 覚 え て お い て ほ し い。
</p>
