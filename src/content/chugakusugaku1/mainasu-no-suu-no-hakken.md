## マ イ ナ ス の 数 ─ 人 類 2000 年 の 旅

### 1. 中 国 ─ 紀 元 前 1 世 紀 の 算 木

今 か ら 2000 年 以 上 前、 古 代 中 国 の 漢 の 時 代 に、 す で に「マ イ ナ ス の 数」 は 計 算 さ れ て い た。

『九 章 算 術』 と い う 数 学 書 の 第 8 章「方 程」 で、 こ う 述 べ ら れ て い る。

   「正 の 数 と 負 の 数 を 同 じ 計 算 の 中 に 混 ぜ る と き、
    同 名 (同 じ 符 号) 相 除、 異 名 (違 う 符 号) 相 益 と す る」

つ ま り、 (-3) + (-5) = -8 (同 じ 符 号 な ら そ の ま ま 足 し て 符 号 を 維 持)、 (+3) + (-5) = -2 (違 う 符 号 な ら 差 を と る) ─ 現 代 の 中 学 1 年 で 学 ぶ の と 全 く 同 じ 規 則 で あ る。

古 代 中 国 で は、 「算 木」 と い う 計 算 用 の 棒 を 使 い、 赤 い 棒 で 正 の 数、 黒 い 棒 で 負 の 数 を 表 し た。 こ れ が 世 界 最 古 の「マ イ ナ ス の 数 の 視 覚 化」 で あ る。

***

### 2. イ ン ド ─ 7 世 紀 の ブ ラ フ マ グ プ タ

7 世 紀 の イ ン ド に、 ブ ラ フ マ グ プ タ (598–668) と い う 数 学 者 が い た。 彼 は 628 年、 『ブ ラ フ マ・ス プ タ・シ ッ ダ ー ン タ』 と い う 天 文・数 学 書 を 著 し、 0 と 負 の 数 の 演 算 規 則 を 体 系 的 に ま と め た。

ブ ラ フ マ グ プ タ が 述 べ た 規 則 (現 代 風 に 書 き 直 す と):

   「正 + 正 = 正」(財 産 + 財 産 = 財 産)
   「負 + 負 = 負」(借 金 + 借 金 = 借 金)
   「正 + 負 = 差 を と る」(財 産 と 借 金 を 合 わ せ て 差 を と る)
   「正 × 正 = 正、 負 × 負 = 正、 正 × 負 = 負」
   「0 × ど ん な 数 = 0、 0 + ど ん な 数 = そ の 数」

特 に「負 × 負 = 正」 ─ 「マ イ ナ ス か け る マ イ ナ ス は プ ラ ス」 ─ と い う 規 則 を、 ブ ラ フ マ グ プ タ は 既 に 明 確 に 述 べ て い た。 中 学 1 年 で 出 会 う 不 思 議 な 規 則 で あ る が、 こ れ は 7 世 紀 か ら 知 ら れ て い た の だ。

***

### 3. ヨ ー ロ ッ パ ─ 16 世 紀 ま で の 抵 抗

し か し 不 思 議 な こ と に、 ヨ ー ロ ッ パ で は 負 の 数 は 16 世 紀 ま で 受 け 入 れ ら れ な か っ た。

古 代 ギ リ シ ア の 数 学 (ピ タ ゴ ラ ス、 ユ ー ク リ ッ ド、 ア ル キ メ デ ス) は、 数 を「幾 何 学 的 な 長 さ」 と し て 捉 え て お り、 「長 さ が マ イ ナ ス」 と い う 概 念 を 認 め な か っ た。 中 世 の ヨ ー ロ ッ パ で も、 ア ラ ビ ア 経 由 で イ ン ド の 数 学 が 伝 わ っ た 後 も、 「負 の 数 (numeri absurdi = ば か げ た 数)」 と 呼 ば れ て、 真 剣 に 扱 わ れ な か っ た。

16 世 紀 イ タ リ ア の カ ル ダ ー ノ (1501-1576) が、 三 次 方 程 式 の 解 法 で 負 の 数 (そ し て 虚 数) を 使 い 始 め た。 17 世 紀 フ ラ ン ス の デ カ ル ト (1596-1650) が、 座 標 軸 で「マ イ ナ ス の 数 を、 ゼ ロ よ り 左 の 数 と し て 図 形 的 に 表 す」 こ と を 発 明 し て、 よ う や く 負 の 数 が 普 通 の 数 と し て 受 け 入 れ ら れ た。

中 国 や イ ン ド か ら、 ヨ ー ロ ッ パ ま で、 約 1000 年 〜 2000 年 の 遅 れ で あ っ た。

***

### 4. 数 の 概 念 の 拡 大

人 類 の 数 学 史 は、 「ど ん な 数 を 認 め る か」 の 拡 大 の 歴 史 で あ る。

- 1, 2, 3, ... (自 然 数) → 古 代 か ら
- 0 (ゼ ロ) → 5 世 紀 イ ン ド
- マ イ ナ ス の 数 (負 の 数) → 前 1 世 紀 中 国 / 7 世 紀 イ ン ド
- 分 数 (有 理 数) → 古 代 エ ジ プ ト・ギ リ シ ア
- √2 の よ う な 無 理 数 → ピ タ ゴ ラ ス 学 派 (前 5 世 紀、 し か し 受 容 に 時 間)
- 虚 数 i → 16 世 紀 イ タ リ ア (カ ル ダ ー ノ・ボ ン ベ リ)

新 し い 数 を 認 め る た び に、 数 学 の 世 界 は 広 が っ て き た。 マ イ ナ ス の 数 を 「ば か げ た 数」 と 拒 否 し た ヨ ー ロ ッ パ も、 や が て 受 け 入 れ た。

あ な た が 今 日 学 ぶ「マ イ ナ ス の 数」 は、 2000 年 前 の 中 国 と 1400 年 前 の イ ン ド か ら、 長 い 旅 を し て あ な た の ノ ー ト に 届 い た 数 な の で あ る。
