## 文 字 式 の 発 明 ─ 1500 年 ま で の 数 学 と そ の 後

### 1. 文 字 が な か っ た 時 代

ま ず、 想 像 し て み よ う。

「文 字 式」 が 存 在 し な か っ た 時 代 の 数 学 を。

紀 元 前 の エ ジ プ ト で、 ピ ラ ミ ッ ド を 建 て る 大 工 さ ん が 計 算 を す る と き。 古 代 ギ リ シ ア で、 ユ ー ク リ ッ ド が『原 論』 を 書 く と き。 古 代 中 国 で、『九 章 算 術』 を 編 ま れ る と き。 中 世 イ ス ラ ム 世 界 で、 ア ル・フ ワ ー リ ズ ミ ー が「ア ル ジ ェ ブ ラ」 (= 代 数 学 の 語 源) を 著 し た と き。

そ れ ら は す べ て、 「2 + 3 = 5」「7 × 8 = 56」 と い っ た「具 体 的 な 数 だ け」 で 書 か れ て い た。

「a + b = b + a」 と い う 一 般 的 な 交 換 法 則 を 表 現 す る 手 段 が、 数 学 に は 存 在 し な か っ た の だ。

***

### 2. 法 則 を 文 章 で 説 明 す る 苦 労

例 え ば、 「ど ん な 二 つ の 数 を 取 っ て も、 そ の 和 は 順 番 を 入 れ 替 え て も 同 じ」 と い う 当 た り 前 の 法 則 を、 古 代・中 世 の 数 学 者 は こ う 説 明 し た。

「あ る 二 つ の 数 を 取 る。 そ の 二 つ の 数 を 足 す と、 一 つ の 値 が 得 ら れ る。 同 じ 二 つ の 数 を、 順 番 を 入 れ 替 え て 足 し て も、 同 じ 値 が 得 ら れ る。」

文 章 で は 数 行 だ。 し か し も っ と 複 雑 な 法 則、 例 え ば「(a + b)² = a² + 2ab + b²」 を 文 章 で 書 こ う と す る と、 数 ペ ー ジ に な る。

数 学 が 進 歩 す る た め に は、 「一 般 的 な 数」 を 表 す 記 号 が 必 要 で あ っ た。

***

### 3. 1591 年、 フ ラ ン ス の ウ ィ エ ー ト

そ し て 1591 年、 フ ラ ン ス の 数 学 者 フ ラ ン ソ ワ・ウ ィ エ ー ト (François Viète, 1540-1603) が、 著 書『解 析 術 入 門』(In artem analyticen isagoge) で、 数 学 史 上 の 革 命 を 起 こ し た。

ウ ィ エ ー ト は、 ア ル フ ァ ベ ッ ト の 文 字 を 数 式 に 体 系 的 に 用 い る こ と を 始 め た。 母 音 (A, E, I, O, U) を 未 知 数、 子 音 (B, C, D, F, …) を 既 知 数 と し て 区 別。

例 え ば、 「A + B = B + A」 と 書 け ば、 「ど ん な 二 つ の 数 A と B を 取 っ て も、 そ の 和 は 順 番 を 入 れ 替 え て も 同 じ」 と い う 法 則 を、 一 行 で 表 現 で き る よ う に な っ た。

ウ ィ エ ー ト は こ の 業 績 で「代 数 学 の 父」 と 呼 ば れ る よ う に な っ た。

***

### 4. 1637 年、 デ カ ル ト の『幾 何 学』

ウ ィ エ ー ト の 文 字 式 を、 現 代 と 同 じ 形 に 完 成 さ せ た の が、 フ ラ ン ス の 哲 学 者・数 学 者 ル ネ・デ カ ル ト (René Descartes, 1596-1650) で あ っ た。

1637 年、 デ カ ル ト は『方 法 序 説』(Discours de la méthode) と そ の 付 録『幾 何 学』(La Géométrie) を 著 し た。 そ の 中 で、 現 代 の 数 学 表 記 を ほ ぼ 完 成 さ せ た。

- **x, y, z** = 未 知 数 (ま だ 分 か ら な い 数)
- **a, b, c** = 既 知 数 (す で に 分 か っ て い る 数)
- **上 付 き 指 数** (x², y³, …)
- **+, -** 等 の 演 算 記 号

こ の 約 束 が、 400 年 経 っ た 21 世 紀 で も そ の ま ま 使 わ れ て い る。 君 が ノ ー ト に「3x + 5 = 14」 と 書 く と き の x、 そ し て「a + b = b + a」 と 書 く と き の a と b は、 1591 年 ウ ィ エ ー ト と 1637 年 デ カ ル ト が 数 学 に 贈 っ た 遺 産 で あ る。

***

### 5. 文 字 式 が も た ら し た 革 命

文 字 式 の 発 明 は、 数 学 を「個 別 の 計 算 (3 + 5 = 8 等)」 か ら「一 般 的 法 則 の 学 問 (a + b = b + a 等)」 へ と 変 え た。

数 学 は、 文 字 式 と い う 道 具 を 手 に 入 れ た 17 世 紀 以 降、 爆 発 的 に 進 化 し た。

- 1665 年 頃、 ニ ュ ー ト ン が 微 分 積 分 を 発 明
- 1684 年、 ラ イ プ ニ ッ ツ が 微 分 積 分 の 独 立 発 表
- 1687 年、 ニ ュ ー ト ン『プ リ ン キ ピ ア』 で 万 有 引 力 と 古 典 力 学 を 完 成

こ れ ら す べ て が、 ウ ィ エ ー ト と デ カ ル ト の 文 字 式 を 前 提 と し て い た。 文 字 式 が な け れ ば、 ニ ュ ー ト ン の 法 則「F = ma」 も、 ア イ ン シ ュ タ イ ン の「E = mc²」 も、 書 け な か っ た。

***

### 6. 中 1 で 学 ぶ「文 字 式」 と い う 第 一 歩

中 学 1 年 で 突 然 出 て く る「文 字 を 使 っ た 式」 ─ そ れ は、 1591 年 ウ ィ エ ー ト 以 来 の、 数 学 史 上 最 大 の 革 命 の 入 口 で あ る。

「a × 2 = 2a」 と い う 一 見 当 た り 前 の 約 束 を 学 ぶ こ と は、 「数 学 を、 個 別 の 計 算 か ら 一 般 的 法 則 へ と 拡 張 す る」 と い う 巨 大 な 一 歩 で あ る。

そ の 一 歩 を、 君 は 今、 ノ ー ト の 一 行 で 踏 ん で い る。 ウ ィ エ ー ト と デ カ ル ト が 400 年 前 に 残 し た 道 を、 君 が 歩 い て い る の だ。
