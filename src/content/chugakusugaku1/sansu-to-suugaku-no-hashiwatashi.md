<h3>「算 数」 と 「数 学」 ─ 何 が 違 う か</h3>

<p>
小 学 校 6 年 間、 君 は「算 数」 を 学 ん で き た。<br>
中 学 校 か ら は「数 学」 と 名 前 が 変 わ る。<br>
で は、 こ の 二 つ は ど う 違 う の か?
</p>

<p>
答 え は、 一 つ の 鍵 と な る 概 念 に 集 約 さ れ る ─ <strong>「文 字 式」</strong> で あ る。
</p>

<table>
<thead>
<tr><th></th><th>算 数 (小 学 校)</th><th>数 学 (中 学 校 ~)</th></tr>
</thead>
<tbody>
<tr><td><strong>扱 う 数</strong></td><td>具 体 的 な 数 (3、 5、 10、 100)</td><td>抽 象 的 な 数 (x、 y、 a、 b)</td></tr>
<tr><td><strong>問 題 の 例</strong></td><td>「3 個 の り ん ご」</td><td>「x 個 の り ん ご」</td></tr>
<tr><td><strong>強 さ</strong></td><td>個 別 の 計 算 が で き る</td><td>無 数 の 場 合 を 一 度 に 扱 え る</td></tr>
<tr><td><strong>弱 さ</strong></td><td>個 別 に 計 算 し な け れ ば な ら な い</td><td>抽 象 的 で 慣 れ ま で 難 し い</td></tr>
</tbody>
</table>

<h3>「文 字 式」 と い う 発 明 (1591 年)</h3>

<p>
「文 字 を 使 っ て 計 算 す る」 と い う 考 え 方 は、 い つ 生 ま れ た の か?
</p>

<p>
1591 年、 フ ラ ン ス の 数 学 者 <strong>フ ラ ン ソ ワ ・ ヴ ィ エ ー タ</strong> (François Viète, 1540–1603) が、 著 書『解 析 学 入 門』 で、 数 学 史 上 初 め て 体 系 的 に 「文 字 を 使 っ た 計 算」 を 提 唱 し た。
</p>

<p>
そ れ ま で の 数 学 は、 こ う だ っ た:
</p>

<blockquote>
「ある 数 と 5 の 和 が 10 で あ る。 そ の 数 を 求 め よ」
</blockquote>

<p>
こ れ を、 ヴ ィ エ ー タ は こ う 書 い た:
</p>

<blockquote>
「x + 5 = 10、 x を 求 め よ」
</blockquote>

<p>
こ の <strong>「x」 と い う 一 文 字</strong> が、 数 学 を 一 気 に 強 力 に し た。<br>
こ れ 以 降、 ヨ ー ロ ッ パ の 数 学 は 急 速 に 発 達 し、 ニ ュ ー ト ン (1642–1727) の 微 積 分 学 へ と つ な が っ て い く。
</p>

<h3>同 じ 頃 の 日 本 ─ 関 孝 和 の 「点 竄 術」</h3>

<p>
不 思 議 な こ と に、 同 じ 頃 の 江 戸 日 本 で も、 ヨ ー ロ ッ パ と は 全 く 独 立 に、 同 様 の 「文 字 を 使 っ た 計 算」 が 発 明 さ れ て い た。
</p>

<p>
和 算 家 ・ <strong>関 孝 和</strong> (1642 頃 –1708、 本 サ イ ト sansu5/enseki-ritsu で 既 習) が、 「<strong>点 竄 術 (て ん ざ ん じ ゅ つ)</strong>」 と 呼 ば れ る 代 数 体 系 を 確 立 し た の は、 1683 年 頃。
</p>

<p>
ヴ ィ エ ー タ の 文 字 式 (1591) か ら 約 90 年 後 ─ 鎖 国 の 日 本 で、 関 孝 和 が ま っ た く 独 立 に 同 じ 概 念 に 到 達 し た。
</p>

<p>
こ れ は、 「<strong>数 学 は 世 界 共 通 の 言 葉</strong>」 で あ る こ と の、 美 し い 証 明 で あ る。<br>
文 化 が 違 っ て も、 言 語 が 違 っ て も、 数 学 の 真 理 は 同 じ。 だ か ら 別 々 の 場 所 で、 別 々 の 人 が、 同 じ 答 え に 到 達 す る。
</p>

<h3>具 体 例 ─ 鶴 亀 算 を 文 字 式 で 解 く</h3>

<p>
小 5 で 君 が 学 ん だ <strong>鶴 亀 算</strong> (本 サ イ ト sansu5/tsurukame-zan):
</p>

<blockquote>
鶴 と 亀 が 合 わ せ て 5 匹 い ま す。 足 の 数 は 合 わ せ て 14 本 で す。 鶴 と 亀 は そ れ ぞ れ 何 匹 ず つ?
</blockquote>

<p>
小 学 校 で は、 こ れ を <strong>表 や 図</strong> で 解 い た:
</p>

<table>
<thead>
<tr><th>鶴 の 数</th><th>亀 の 数</th><th>足 の 合 計</th></tr>
</thead>
<tbody>
<tr><td>5</td><td>0</td><td>10 本</td></tr>
<tr><td>4</td><td>1</td><td>12 本</td></tr>
<tr><td><strong>3</strong></td><td><strong>2</strong></td><td><strong>14 本 ✓</strong></td></tr>
</tbody>
</table>

<p>
答 え: 鶴 3 匹、 亀 2 匹。
</p>

<p>
中 1 数 学 で は、 こ の 問 題 を <strong>文 字 式 (連 立 方 程 式)</strong> で 解 く:
</p>

<blockquote>
鶴 の 数 を x、 亀 の 数 を y と お く。<br>
<strong>x + y = 5</strong> ─ (合 計 5 匹)<br>
<strong>2x + 4y = 14</strong> ─ (足 の 合 計 14 本)<br>
<br>
こ の 二 つ の 式 か ら、 x = 3、 y = 2 が 導 か れ る。
</blockquote>

<p>
表 で 5 回 試 す よ り、 文 字 式 で 2 行 書 く 方 が、 圧 倒 的 に 速 い し、 美 し い。<br>
さ ら に、 鶴 と 亀 の 合 計 が 100 匹 で も、 1000 匹 で も、 同 じ 式 で 解 け る。
</p>

<p>
こ れ が「文 字 式」 の 力 で あ る。
</p>

<h3>「al-jabr」 と い う 言 葉</h3>

<p>
「代 数」 の 英 語 「algebra」 (ア ル ジ ェ ブ ラ) は、 ア ラ ビ ア 語 の 「<strong>al-jabr</strong>」 (ア ル ・ ジ ャ ブ ル) か ら 来 て い る。
</p>

<p>
意 味 は 「移 項 (= 式 の 中 で 項 を 反 対 側 に 移 す こ と)」。<br>
9 世 紀 の イ ス ラ ム 数 学 者 <strong>ア ル ・ フ ワ ー リ ズ ミ ー</strong> (al-Khwarizmi, 780 頃 –850) が、 著 書 の タ イ ト ル に こ の 言 葉 を 使 っ た。
</p>

<p>
イ ス ラ ム 数 学 (9 世 紀) → ヨ ー ロ ッ パ 数 学 (12 世 紀 翻 訳 ・ 1591 年 ヴ ィ エ ー タ) → 江 戸 日 本 (1683 年 関 孝 和) と、 「文 字 を 使 う 数 学」 は 世 界 中 で 並 行 に 発 達 し た。
</p>

<h3>君 自 身 で 試 し て み よ う</h3>

<p>
身 近 な 問 題 を、 文 字 式 に し て み よ う。
</p>

<ol>
<li>「コ ン ビ ニ で お や つ を 買 う。 100 円 の チ ョ コ レ ー ト x 個 と 150 円 の ク ッ キ ー y 個 を 買 い、 合 計 600 円 だ っ た」 ─ こ れ を 式 に す る と?</li>
<li>「明 日 学 校 に 行 く ま で の 時 間 を t 時 間 と す る。 そ の 間 に 食 事 0.5 時 間、 勉 強 2 時 間、 ゲ ー ム g 時 間。 残 り の 時 間 は ?」</li>
</ol>

<p>
こ の よ う に、 君 の 日 常 の 中 に、 文 字 式 で 表 せ る も の は い く ら で も あ る。<br>
中 学 数 学 の 旅 は、 こ の 「<strong>文 字 を 使 う 視 点 で 世 界 を 見 る</strong>」 と こ ろ か ら 始 ま る の だ。
</p>
