<h3>関 孝 和 ─ 日 本 数 学 史 最 大 の 天 才</h3>

<p>
関 孝 和 (せ き た か か ず、 1642 頃 –1708) は、 日 本 数 学 史 上 最 大 の 天 才 と 呼 ば れ る。
</p>

<p>
彼 の 主 な 業 績:
</p>

<ul>
<li><strong>点 竄 術 (て ん ざ ん じ ゅ つ)</strong> ─ 文 字 を 使 う 代 数 体 系 の 確 立</li>
<li><strong>行 列 式</strong> ─ ヨ ー ロ ッ パ の ラ イ プ ニ ッ ツ (1693) よ り 約 10 年 早 く 発 見</li>
<li><strong>円 周 率</strong> ─ 小 数 点 以 下 11 桁 ま で 正 確 に 計 算 (本 サ イ ト sansu5/enseki-ritsu)</li>
<li><strong>方 程 式 の 解 法</strong> ─ 高 次 方 程 式 の 解 法 を 体 系 化</li>
<li><strong>級 数 論</strong> ─ 無 限 級 数 を 用 い た 高 度 な 計 算</li>
</ul>

<p>
こ れ ら す べ て を、 関 孝 和 は <strong>ヨ ー ロ ッ パ と 一 切 接 触 せ ず に</strong> 独 自 に 達 成 し た。<br>
鎖 国 の 日 本 で、 中 国 の 数 学 を 出 発 点 と し て、 ヨ ー ロ ッ パ に 匹 敵 す る 数 学 を 築 い た の で あ る。
</p>

<h3>「点 竄 術」 と は 何 か</h3>

<p>
「点 竄 術 (て ん ざ ん じ ゅ つ)」 と い う 言 葉 を、 分 解 し て 見 て み よ う。
</p>

<ul>
<li><strong>点</strong> ─ 紙 の 上 に 書 か れ た 点 (= 文 字 ・ 数 ・ 記 号)</li>
<li><strong>竄 (ざ ん)</strong> ─ 移 す、 並 び 替 え る</li>
<li><strong>術 (じ ゅ つ)</strong> ─ 方 法 ・ 技 術</li>
</ul>

<p>
合 わ せ て、 「紙 上 の 文 字 を 操 作 す る 方 法」 ─ つ ま り <strong>「文 字 式 を 使 っ た 計 算 法」</strong> の 意 で あ る。
</p>

<p>
こ れ は、 中 1 で 君 が 今 学 ん で い る 「文 字 式」 と、 本 質 的 に 同 じ も の だ。
</p>

<h3>江 戸 の 文 字 式 ─ 漢 字 と 算 木 で</h3>

<p>
た だ、 関 孝 和 の 時 代 の 「文 字」 は、 ア ル フ ァ ベ ッ ト (x、 y) で は な か っ た。<br>
彼 ら が 使 っ た の は ─
</p>

<table>
<thead>
<tr><th>現 代 の 表 記</th><th>関 孝 和 時 代 の 表 記</th></tr>
</thead>
<tbody>
<tr><td>x (未 知 数 1)</td><td>「天 元 (て ん げ ん)」 と い う 漢 字</td></tr>
<tr><td>y (未 知 数 2)</td><td>「人 元 (じ ん げ ん)」 と い う 漢 字</td></tr>
<tr><td>z (未 知 数 3)</td><td>「地 元 (ち げ ん)」 と い う 漢 字</td></tr>
<tr><td>+ (足 す)</td><td>言 葉 で 「加 え る」</td></tr>
<tr><td>= (等 し い)</td><td>「等 す」 「等 し」</td></tr>
</tbody>
</table>

<p>
さ ら に、 数 字 そ の も の も、 ア ラ ビ ア 数 字 (1、 2、 3) で は な く、 算 木 (さ ん ぎ ─ 短 い 木 の 棒) を 並 べ た 形 で 紙 に 描 か れ た。
</p>

<p>
表 現 方 法 は 全 く 違 う が ─ <strong>計 算 の 仕 組 み は、 現 代 の 文 字 式 と ま っ た く 同 じ</strong> で あ る。
</p>

<h3>ヴ ィ エ ー タ と 関 孝 和 ─ 並 行 す る 二 つ の 数 学</h3>

<p>
時 系 列 で 並 べ て み よ う:
</p>

<ul>
<li>1591 年 ─ <strong>ヴ ィ エ ー タ</strong>『解 析 学 入 門』(フ ラ ン ス) ─ 文 字 式 革 命</li>
<li>1642 年 頃 ─ <strong>関 孝 和</strong> 生 ま れ る (日 本)</li>
<li>1674 年 ─ <strong>関 孝 和</strong>『発 微 算 法』(日 本) ─ 点 竄 術 の 体 系 化</li>
<li>1683 年 頃 ─ 関 孝 和、 行 列 式 を 発 見</li>
<li>1693 年 ─ <strong>ラ イ プ ニ ッ ツ</strong>、 行 列 式 を 発 表 (ド イ ツ)</li>
<li>1708 年 ─ <strong>関 孝 和</strong> 没 す</li>
</ul>

<p>
ヴ ィ エ ー タ (1591) か ら 関 孝 和 の 仕 事 (1674) ま で、 約 83 年。<br>
そ し て、 関 孝 和 の 行 列 式 (1683 頃) と ラ イ プ ニ ッ ツ の 行 列 式 (1693) ─ <strong>た っ た 10 年 違 い</strong>。
</p>

<p>
鎖 国 の 日 本 と ヨ ー ロ ッ パ が、 ほ ぼ 同 じ 時 期 に 同 じ 数 学 を 独 立 に 発 達 さ せ た ─ こ れ は 数 学 史 上 で 最 も 美 し い 並 行 で あ る。
</p>

<h3>「数 学 は 世 界 共 通 の 言 葉」</h3>

<p>
こ の 並 行 発 達 は、 数 学 に 関 す る 大 切 な 真 理 を 教 え て く れ る:
</p>

<blockquote>
<strong>数 学 は、 文 化・言 語・時 代 を 超 え て、 同 じ 真 理 を 指 し 示 す。</strong>
</blockquote>

<p>
中 1 で 君 が 学 ぶ 「文 字 式」 は、 ヨ ー ロ ッ パ で は ヴ ィ エ ー タ が 1591 年 に 体 系 化 し、 日 本 で は 関 孝 和 が 1674 年 に 確 立 し た。<br>
場 所 も 時 代 も 違 う 二 人 が、 全 く 同 じ 「文 字 を 使 う」 と い う 発 想 に 到 達 し た。
</p>

<p>
だ か ら、 君 が 学 ぶ こ の「文 字 式」 は ─<br>
ア フ リ カ の 数 学 者 も、 南 米 の 数 学 者 も、 ヨ ー ロ ッ パ の 数 学 者 も、 同 じ よ う に 理 解 す る、 <strong>世 界 で 唯 一 の 共 通 言 語</strong> な の だ。
</p>

<h3>関 孝 和 の 時 代 の 算 額 文 化</h3>

<p>
関 孝 和 の 時 代 (江 戸 前 期) は、 数 学 が <strong>庶 民 に も 楽 し ま れ る</strong> 時 代 だ っ た。<br>
本 サ イ ト sansu5/sangaku-bunka で 既 習 の 「算 額 (さ ん が く)」 文 化 が、 ま さ に そ の 時 代 に 始 ま っ て い る。
</p>

<p>
関 孝 和 の 弟 子 た ち ─ 建 部 賢 弘 (た け べ か た ひ ろ、1664–1739) ら ─ も、 神 社 に 算 額 を 奉 納 し、 庶 民 と 数 学 を 楽 し ん で い た。
</p>

<p>
こ の <strong>「文 字 式 を 楽 し む 文 化」</strong> ─ そ れ が、 ヨ ー ロ ッ パ と 並 ぶ 高 度 な 数 学 を、 江 戸 日 本 に 根 付 か せ た 土 壌 で あ っ た。
</p>

<h3>君 へ の メ ッ セ ー ジ</h3>

<p>
中 1 で 「x」「y」 と い う 文 字 を 使 い 始 め た 君 は ─<br>
ヴ ィ エ ー タ (フ ラ ン ス、 1591)、 関 孝 和 (日 本、 1674)、 ラ イ プ ニ ッ ツ (ド イ ツ、 1693) と、 同 じ 系 譜 に 連 な っ て い る。
</p>

<p>
そ し て こ の 文 字 式 と い う 道 具 は、 や が て 高 校 で 学 ぶ <strong>関 数</strong>・<strong>微 分 積 分</strong>・<strong>線 形 代 数</strong> の す べ て の 基 礎 と な る。
</p>

<p>
350 年 前 の 関 孝 和 が 紙 に 書 い た 「天 元」 と、 君 が 今 ノ ー ト に 書 く 「x」 ─ そ れ ら は 同 じ 一 つ の 「未 知 を 見 つ け 出 す 道 具」 で あ る。<br>
こ の 道 具 を 使 い こ な せ る か ど う か が、 中 学 数 学 ・ 高 校 数 学 ・ そ し て そ の 後 の 一 生 を 左 右 す る。
</p>

<p>
関 孝 和 の よ う に、 ヴ ィ エ ー タ の よ う に、 「文 字 を 自 由 に 操 る 力」 を 身 に つ け て ほ し い。
</p>
